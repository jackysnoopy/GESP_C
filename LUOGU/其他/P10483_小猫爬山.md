# 小猫爬山

## 题目描述
N<=18 小猫重量，缆车容量 W，最少车辆数。

## 输入格式
N W，随后 N 行重量。

## 输出格式
最少缆车数。

## 样例
### 样例 1
```
5 1996
1
2
1994
12
29
```
```
2
```

## 解题思路
### 问题分析
将 $N$ 只小猫分配到若干缆车中，每辆缆车容量不超过 $W$，求最少需要多少辆缆车。这是装箱问题（Bin Packing）的精确求解版本。

### 核心思路
1. **搜索 + 剪枝（迭代加深 / DFS）**：逐个小猫分配到已有的缆车或新开一辆。
2. **剪枝策略**：
   - 按重量降序排列，先分配重猫，更容易触发剪枝
   - 若某辆缆车剩余空间恰好等于当前小猫重量，优先放入（最优性剪枝）
   - 若多辆缆车剩余空间相同，只尝试放入第一辆（对称性剪枝）
   - 当前使用缆车数 + 剩余最优情况 $\ge$ 已知最优解时剪枝
3. $N \le 18$，搜索空间可控。

### 算法流程
1. 读入小猫重量和缆车容量 $W$
2. 将重量降序排序
3. DFS 逐个分配小猫：尝试放入每辆已有缆车（不超载），或新开一辆
4. 维护当前最优解，剪枝优化
5. 输出最少缆车数

### 复杂度分析
- 时间复杂度：最坏 $O(K^N)$，$K$ 为当前已用缆车数，实际因剪枝远小于此
- 空间复杂度：$O(N)$
