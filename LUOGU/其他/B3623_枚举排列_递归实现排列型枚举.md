# 枚举排列（递归实现排列型枚举）

## 题目描述
今有 $n$ 名学生，要从中选出 $k$ 人排成一列拍照。

请按字典序输出所有可能的排列方式。

## 输入格式
仅一行，两个正整数 $n, k$。

## 输出格式
若干行，每行 $k$ 个正整数，表示一种可能的队伍顺序。

## 样例
### 样例 1
```
3 2
```
```
1 2
1 3
2 1
2 3
3 1
3 2
```

## 提示
对于 $100\%$ 的数据，$1\leq k\leq n \leq 10$。

## 解题思路

### 问题分析
从 n 个元素中选出 k 个按顺序排列，按字典序输出所有排列。

### 核心思路
使用递归回溯法枚举排列。用 visited 数组标记已选元素。

### 算法流程
1. 定义 dfs(cnt) 表示已选 cnt 个元素
2. 若 cnt == k，输出当前排列
3. 否则遍历 1..n，选择未使用的元素加入排列，递归

### 复杂度分析
- 时间复杂度：O(P(n,k))
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <vector>
using namespace std;

int n, k;
vector<int> perm;
vector<bool> used;

void dfs(int cnt) {
    if (cnt == k) {
        for (int x : perm) cout << x << " ";
        cout << "
";
        return;
    }
    for (int i = 1; i <= n; i++) {
        if (!used[i]) {
            used[i] = true;
            perm.push_back(i);
            dfs(cnt + 1);
            perm.pop_back();
            used[i] = false;
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n >> k;
    used.resize(n + 1, false);
    dfs(0);
    return 0;
}
```
