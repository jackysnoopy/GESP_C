# [Poetize6] IncDec Sequence

## 题目描述

给定一个长度为 $n$ 的数列 ${a_1,a_2,\cdots,a_n}$，每次可以选择一个区间 $[l,r]$，使这个区间内的数都加 $1$ 或者都减 $1$。 
  
请问至少需要多少次操作才能使数列中的所有数都一样，并求出在保证最少次数的前提下，最终得到的数列有多少种。

## 输入格式

第一行一个正整数 $n$。  
接下来 $n$ 行，每行一个整数，第 $i+1$ 行的整数表示 $a_i$。

## 输出格式

第一行输出最少操作次数。  
第二行输出最终能得到多少种结果。

## 样例 #1

**输入**
```
4
1
1
2
2
```

**输出**
```
1
2
```

## 提示

对于 $100\%$ 的数据，$n\le 100000$，$0 \le a_i \le 2^{31}$。

## 解题思路

### 问题分析

给定数列 $a$，每次可以选择一个区间 $[l,r]$ 全加 $1$ 或全减 $1$。求最少操作次数使所有数相等，以及此时最终数列有多少种可能。

### 核心思路

**差分数组性质。** 设差分数组 $d_i = a_i - a_{i-1}$（$i=2..n$）。我们的目标是将所有 $a_i$ 变为相同值，即 $d_2 = d_3 = \cdots = d_n = 0$。

每次区间 $[l,r]$ 加 $1$ 等价于 $d_l += 1$，$d_{r+1} -= 1$。减 $1$ 相反。

**结论：**
- 最少操作次数 $= \max(pos, neg)$，其中 $pos$ 为正差分的和，$neg$ 为负差分绝对值的和。
- 最终数列的可能数 $= |pos - neg| + 1$。

### 算法流程

1. 读入 $n$ 和数组 $a$。
2. 计算正差分和 $pos$ 与负差分绝对值的和 $neg$：
   - 对于 $i=2..n$，$d = a_i - a_{i-1}$。
   - 如果 $d > 0$，$pos += d$；如果 $d < 0$，$neg += -d$。
3. 输出 $\max(pos, neg)$ 和 $|pos - neg| + 1$。

### 复杂度分析

- **时间复杂度：** $O(n)$。
- **空间复杂度：** $O(1)$。

### 正确性证明

每次操作要么增加一对 $(+1, -1)$，要么减少一对 $(+1, -1)$。我们要消除所有非零的差分值。最少操作次数取决于需要消除的正差分和负差分中较大的那个。最终数列的值可以是任意介于 $a_{\min} + neg$ 和 $a_{\max} - pos$ 之间的整数。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    long long pos = 0, neg = 0;
    for (int i = 1; i < n; i++) {
        long long d = a[i] - a[i-1];
        if (d > 0) pos += d;
        else neg -= d;
    }
    cout << max(pos, neg) << "\n" << abs(pos - neg) + 1 << "\n";
    return 0;
}
```
