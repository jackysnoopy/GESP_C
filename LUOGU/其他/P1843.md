# P1843 - 二分时间

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **二分答案** 类问题。

### 核心思路

二分最少时间，验证：自然干燥+机器辅助能否在mid时间内烘干所有。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, a, b;
    cin >> n >> a >> b;
    vector<long long> w(n);
    for (int i = 0; i < n; i++) cin >> w[i];
    long long lo = 0, hi = 5e14;
    while (lo < hi) {
        long long mid = (lo + hi) / 2;
        long long need = 0;
        for (int i = 0; i < n; i++) {
            long long rem = w[i] - mid * a;
            if (rem > 0) {
                need += (rem + b - 1) / b;
            }
        }
        if (need <= mid) hi = mid;
        else lo = mid + 1;
    }
    cout << lo << "\n";
    return 0;
}

```
