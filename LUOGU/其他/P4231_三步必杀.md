# 三步必杀

## 题目描述

$N$ 个柱子排成一排，一开始每个柱子损伤度为 $0$。

接下来勇仪会进行 $M$ 次攻击，每次攻击可以用 $4$ 个参数 $l,r,s,e$ 来描述：

表示这次攻击作用范围为第 $l$ 个到第 $r$ 个之间所有的柱子(包含 $l,r$)，对第一个柱子的伤害为 $s$，对最后一个柱子的伤害为 $e$。

攻击产生的伤害值是一个等差数列。若 $l=1,r=5,s=2,e=10$，则对第 $1 \sim 5$ 个柱子分别产生 $2,4,6,8,10$ 的伤害。

鬼族们需要的是所有攻击完成之后每个柱子的损伤度。

## 输入格式

第一行 $2$ 个整数 $N,M$，用空格隔开，下同。

接下来 $M$ 行，每行 $4$ 个整数 $l,r,s,e$，含义见题目描述。

数据保证对每个柱子产生的每次伤害值都是整数。

## 输出格式

由于输出数据可能过大无法全部输出，为了确保你真的能维护所有柱子的损伤度，只要输出它们的异或和与最大值即可。

（异或和就是所有数字按位异或起来的值。）

（异或运算符在 c++ 里为 `^`。）

## 解题思路

### 问题分析

$N$ 个柱子初始损伤度为 $0$。$M$ 次攻击，每次在 $[l,r]$ 区间内施加等差数列伤害：第 $l$ 个柱子 $s$，第 $r$ 个柱子 $e$，中间线性变化。求最终所有柱子损伤度的异或和与最大值。

### 核心思路

**二阶差分。** 等差数列的差分是等差数列（常数差），所以一次差分还不够，需要二次差分。

对于攻击 $(l,r,s,e)$，公差 $d = \frac{e-s}{r-l}$（整数）。在二阶差分数组上的操作为：
- $d2[l] += s$
- $d2[l+1] += d - s$
- $d2[r+1] -= e + d$
- $d2[r+2] += e$

然后做两次前缀和恢复原数组。

### 算法流程

1. 初始化二阶差分数组 $d2$（大小 $N+5$）。
2. 对于每次攻击，按上述公式在 $d2$ 上标记。
3. 第一次前缀和得到一阶差分，第二次前缀和得到原数组。
4. 遍历原数组，计算最大值和异或和。

### 复杂度分析

- **时间复杂度：** $O(N + M)$。
- **空间复杂度：** $O(N)$。

### 要点

- 公差 $d$ 可以预先计算，因为 $s,e,l,r$ 保证公差为整数。
- 二阶差分的原理：原数组对位置 $i$ 的值 $v_i$，一阶差分 $\Delta v_i = v_i - v_{i-1}$，二阶差分 $\Delta^2 v_i = \Delta v_i - \Delta v_{i-1}$。等差数列的一阶差分为常数，二阶差分只在端点处非零。
- 注意边界情况：$l+1$ 和 $r+2$ 可能超出 $N$，数组需要开大一些。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m;
    cin >> n >> m;
    vector<long long> d2(n + 5, 0);
    while (m--) {
        int l, r;
        long long s, e;
        cin >> l >> r >> s >> e;
        long long d = (e - s) / (r - l);
        d2[l] += s;
        d2[l+1] += d - s;
        d2[r+1] -= e + d;
        d2[r+2] += e;
    }
    for (int i = 1; i <= n; i++) d2[i] += d2[i-1];
    for (int i = 1; i <= n; i++) d2[i] += d2[i-1];
    long long mx = 0, xr = 0;
    for (int i = 1; i <= n; i++) {
        mx = max(mx, d2[i]);
        xr ^= d2[i];
    }
    cout << xr << " " << mx << "\n";
    return 0;
}
```
