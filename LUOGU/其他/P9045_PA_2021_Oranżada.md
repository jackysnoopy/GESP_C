# [PA 2021] Oranżada

## 题目描述

有一排共 $n$ 瓶橙汁，其中第 $i$ 瓶的品牌为 $a_i$。

你可以花费 $1$ 个单位的的代价交换两瓶相邻的橙汁。

求最小代价使得最左边 $k$ 瓶橙汁品牌两两不同。

## 输入格式

第一行，两个整数 $n, k$；

第二行，$n$ 个整数 $a_1, a_2, \cdots, a_n$。

## 输出格式

一行，一个整数，若有解，输出最小代价；否则，输出 $-1$。

## 样例

### 样例输入1
```5 3
3 3 3 1 2
```

### 样例输出1
```4
```

### 样例输入2
```3 2
1 1 1
```

### 样例输出2
```-1
```

## 提示

#### 样例 #1 解释
最优方案为先交换位置 $3$ 和 $4$ 的瓶子、再交换位置 $4$ 和 $5$ 的瓶子，接着交换位置 $2$ 和 $3$ 的瓶子，最后交换位置 $3$ 和 $4$ 的瓶子，共 $4$ 次操作。
#### 样例 #2 解释
显然无解。
#### 数据范围
对于 $100\%$ 的数据，$1 \leq k, a_i \leq n \leq 5 \times 10^5$。

## 解题思路

### 问题分析
交换相邻橙汁，使前k瓶品牌不同。求最小交换次数。

### 核心思路
为前k个位置分配不同的品牌，使用贪心或排序方法计算最小交换距离。

### 算法流程
1. 分析需要的分配
2. 计算最小相邻交换次数
3. 输出结果

### 复杂度分析
时间O(N log N)，空间O(N)。
