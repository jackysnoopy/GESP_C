# P8957 - 「CGOI-3」巫泡弹弹乐

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 个弹力菇有 (a,b) 属性，边权 max(a_i,a_j)+max(b_i,b_j)。求 MST。

### 核心思路

MST + 建图优化。分别按 a 和 b 排序，只连接相邻点对。这样 O(n) 条边即可保证 MST 正确性。

### 复杂度分析

时间复杂度：O(n log n)，空间复杂度：O(n)

### 关键技巧

关键：对于 max(a_i,a_j)+max(b_i,b_j) 形式的边权，排序后相邻点的边足够构建 MST。
