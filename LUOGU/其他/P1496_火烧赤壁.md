# 火烧赤壁

## 题目描述

给定每个起火部分的起点和终点，请你求出燃烧位置的长度之和。

## 输入格式

第一行一个整数，表示起火的信息条数 $n$。  
接下来 $n$ 行，每行两个整数 $a, b$，表示一个着火位置的起点和终点（**注意：左闭右开**）。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

**输入**
```
3
-1 1
5 11
2 9
```

**输出**
```
11
```

## 提示

### 数据规模与约定

对于全部的测试点，保证 $1 \leq n \leq 2 \times 10^4$，$-2^{31} \leq a <  b \lt 2^{31}$，且答案小于 $2^{31}$。

## 解题思路

### 问题分析

给定 $N$ 个区间 $[a_i, b_i)$，求这些区间的并集总长度（被覆盖的总长度）。

### 核心思路

**离散化 + 差分。** 由于坐标范围可达 $\pm 10^9$，不能直接开数组。将坐标离散化后，在每个离散化后的区间上做差分标记。

对每个区间 $[l, r)$，$diff[l] += 1$，$diff[r] -= 1$。扫描时如果当前覆盖数 $> 0$，则累加该段长度。

### 算法流程

1. 读入 $N$ 个区间，收集左右端点到 `vals` 中。
2. 排序去重，得到离散化坐标。
3. 对每个区间，找到 $l$ 和 $r$ 在离散化数组中的下标，在差分数组上标记。
4. 扫描离散化坐标，如果当前覆盖数 $> 0$，加上当前段长度 $vals[i] - vals[i-1]$。

### 复杂度分析

- **时间复杂度：** $O(N \log N)$。
- **空间复杂度：** $O(N)$。

### 要点

- 区间是 $[a_i, b_i)$ 左闭右开，因此差分操作为 $diff[l] += 1$，$diff[r] -= 1$。
- 离散化后扫描时，最后一个坐标点不需要检查（因为区间右边界不包含）。

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int N;
    cin >> N;
    vector<pair<int,int>> seg(N);
    vector<int> val;
    for (int i = 0; i < N; i++) {
        cin >> seg[i].first >> seg[i].second;
        val.push_back(seg[i].first);
        val.push_back(seg[i].second);
    }
    sort(val.begin(), val.end());
    val.erase(unique(val.begin(), val.end()), val.end());
    int sz = val.size();
    vector<int> d(sz + 2, 0);
    for (auto &s : seg) {
        int l = lower_bound(val.begin(), val.end(), s.first) - val.begin() + 1;
        int r = lower_bound(val.begin(), val.end(), s.second) - val.begin() + 1;
        d[l]++; d[r]--;
    }
    long long ans = 0;
    int cur = 0;
    for (int i = 1; i < sz; i++) {
        cur += d[i];
        if (cur > 0) ans += val[i] - val[i-1];
    }
    cout << ans << "\n";
    return 0;
}
```
