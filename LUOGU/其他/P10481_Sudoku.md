# Sudoku

## 题目描述
标准 9x9 数独求解（输入为 0 表示空）。

## 输入格式
第一行测试用例数，每组 9 行各 9 位数字字符串。

## 输出格式
每组输出填好的数独。

## 解题思路
### 问题分析
标准 9×9 数独求解。给定初始盘面（0 表示空），需要填入 1~9 使得每行、每列、每个 3×3 宫格内的数字不重复。

### 核心思路
1. **回溯搜索**：逐行逐列尝试填入合法数字，冲突则回溯。
2. 使用三个布尔数组 `row[9][10]`、`col[9][10]`、`box[9][10]` 分别记录每行、每列、每个宫格已使用的数字。
3. **剪枝优化**：优先选择可填数字最少的空格（MRV），减少搜索分支。
4. 使用位运算（`int` 的二进制位表示可用数字）可进一步加速约束检查和枚举。

### 算法流程
1. 读入数独盘面，初始化行、列、宫的占用标记
2. 找到可填选择最少的空格
3. 枚举该空格所有合法数字，递归填入
4. 若所有空格填满则输出解
5. 否则回溯，尝试下一个数字

### 复杂度分析
- 时间复杂度：最坏 $O(9^{81})$，实际因剪枝极快，通常毫秒级求解
- 空间复杂度：$O(81)$
