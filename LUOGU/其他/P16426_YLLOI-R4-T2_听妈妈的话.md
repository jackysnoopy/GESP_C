# 「YLLOI-R4-T2」听妈妈的话

## 题目描述

小 Y 养了 $n$ 个会孵化出鸡的鸡蛋，这些鸡蛋依次放入 $n$ 个养殖箱，第 $i$ 个养殖箱的鸡蛋会在第 $a_i$ 天孵化出来，小 Y 可以随意决定这些鸡孵出来后的性别，决定后不能再更改。

在每一天结束后：

- 对于每两个相邻的养殖箱，若里面的鸡蛋均已孵化且鸡为一公一母（一母一公），他们便会交配，小 Y 会得到一个新鸡蛋（该鸡蛋不用于孵化）。

第 $t$ 天及其之后所有鸡就不再下蛋，求小 Y 最多能得到多少个新鸡蛋。

## 输入格式

第一行两个正整数 $n,t$。

第二行 $n$ 个正整数 $a_i$。

## 输出格式

一个整数。

## 样例

### 样例输入1
```3 4
1 2 3
```

### 样例输出1
```3
```

### 样例输入2
```5 5
1 1 1 1 1
```

### 样例输出2
```16
```

## 提示

#### 【样例解释#1】

一种可能的情况：

第 $1$ 天，第 $1$ 个养殖箱的鸡孵化出来，小 Y 令其为公鸡。

第 $2$ 天，第 $2$ 个养殖箱的鸡孵化出来，小 Y 令其为母鸡。

第 $2$ 天结束，第 $1$ 个养殖箱的鸡与第 $2$ 个养殖箱的鸡产生一个鸡蛋。

第 $3$ 天，第 $3$ 个养殖箱的鸡孵化出来，小 Y 令其为公鸡。

第 $3$ 天结束，第 $1$ 个养殖箱的鸡与第 $2$ 个养殖箱的鸡产生一个鸡蛋，第 $2$ 个养殖箱的鸡与第 $3$ 个养殖箱的鸡产生一个鸡蛋。

第 $4$ 天及其之后不再下蛋。

共产生 $3$ 个鸡蛋，可以证明，无论小 Y 怎样决定鸡的性别，产生的鸡蛋数都不会超过 $3$。

#### 【数据范围】

**本题采用捆绑测试。**

- Subtask 1（20 pts）：$n,t\le 10$。
- Subtask 2（20 pts）：$\forall a_i=1$。
- Subtask 3（20 pts）：$n=2$。
- Subtask 4（20 pts）：$n,t\le 2000$。
- Subtask 5（20 pts）：无特殊限制。

对于全部数据，保证：$1\le n\le 10^6$，$1\le a_i,t\le 10^9$。

## 解题思路

### 问题分析

n 个蛋在不同天孵化，分配公母使相邻异性对每天产蛋，第 t 天后停止。

### 核心思路

每对相邻 (i,i+1) 在 max(a[i],a[i+1])+1 到 t-1 天产蛋。每对最多产生 max(0, t-max(a[i],a[i+1])) 个蛋。可通过交替公母激活所有正贡献对。

### 算法流程

1. 对每对相邻计算 days = t - max(a[i],a[i+1])
2. 若 days > 0 累加

### 复杂度分析

时间 O(n log n)，空间 O(n)
