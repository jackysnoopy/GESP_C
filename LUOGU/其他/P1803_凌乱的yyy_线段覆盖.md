# 凌乱的yyy / 线段覆盖

## 题目描述

**Python 用户可以尝试使用 pypy3 提交试题。**

快 noip 了，yyy 很紧张！

现在各大 oj 上有 $n$ 个比赛，每个比赛的开始、结束的时间点是知道的。

yyy 认为，参加越多的比赛，noip 就能考的越好（假的）。

所以，他想知道他最多能参加几个比赛。

由于 yyy 是蒟蒻，如果要参加一个比赛必须善始善终，而且不能同时参加 $2$ 个及以上的比赛。

## 输入格式

第一行是一个整数 $n$，接下来 $n$ 行每行是 $2$ 个整数 $a_{i},b_{i}\ (a_{i}<b_{i})$，表示比赛开始、结束的时间。

## 输出格式

一个整数最多参加的比赛数目。

## 样例

### 样例输入1
```3
0 2
2 4
1 3
```

### 样例输出1
```2
```

## 提示

- 对于 $20\%$ 的数据，$n \le 10$；
- 对于 $50\%$ 的数据，$n \le 10^3$；
- 对于 $70\%$ 的数据，$n \le 10^{5}$；
- 对于 $100\%$ 的数据，$1\le n \le 10^{6}$，$0 \le a_{i} < b_{i} \le 10^6$。

## 解题思路

### 问题分析

线段覆盖问题：选最多的不重叠线段。

### 核心思路

贪心：按右端点排序，每次选结束最早的线段。

### 算法流程

1. 按右端点排序
2. 维护上次结束位置，选右端点 >= 上次结束的线段

### 复杂度分析

时间 O(n log n)，空间 O(n)
