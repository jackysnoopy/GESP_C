# 学生分组

## 题目描述

有 $n$ 组学生，给出初始时每组中的学生个数，再给出每组学生人数的上界 $R$ 和下界 $L\ (L \le R)$，每次你可以在某组中选出一个学生把他安排到另外一组中，问最少要多少次才可以使 $N$ 组学生的人数都在 $[L,R]$ 中。

## 输入格式

第一行一个整数 $n$，表示学生组数；

第二行 $n$ 个整数，表示每组的学生个数；

第三行两个整数 $L,R$，表示下界和上界。

## 输出格式

一个数，表示最少的交换次数，如果不能满足题目条件输出 $-1$。

## 样例

### 样例输入
```
2
10 20
10 15
```

### 样例输出
```
5
```

## 提示

### 数据范围及约定

对于全部数据，保证 $1\le n \le 50$。

## 解题思路

### 问题分析

学生分组问题：n 组学生，每组需在 [L, R] 范围内，每次移动一个学生到另一组。求最少移动次数。

### 核心思路

枚举目标值 t（L ≤ t ≤ R），计算使每组恰好 t 人所需的移动次数 = max(excess, deficit)。当 excess = deficit 时可行。

### 算法流程

1. 检查总人数是否在 [L*n, R*n] 范围内
2. 枚举 t 从 L 到 R
3. 计算 excess = sum(max(0, a[i]-t)) 和 deficit = sum(max(0, t-a[i]))
4. 若 excess == deficit，更新最小移动次数
5. 输出最小值或 -1

### 复杂度分析

- 时间复杂度：O(N·(R-L))
- 空间复杂度：O(N)

### 关键技巧

- excess == deficit 时可以完美重分配
- 由于 n ≤ 50，直接枚举即可
