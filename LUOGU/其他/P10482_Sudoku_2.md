# Sudoku 2

## 题目描述
多组 81 字符数独，句点表示空格，输出完整数独。

## 输入格式
多组 81 字符行，末尾 end。

## 输出格式
每组一行完整数独。

## 样例
### 样例 1
```
.2738..1..1...6735.......293.5692.8...........6.1745.364.......9518...7..8..6534.
```
```
527389416819426735436751829375692184194538267268174593643217958951843672782965341
```

## 解题思路
### 问题分析
标准数独求解问题。给定一个 $9 \times 9$ 的数独盘面（`.` 表示空格），需要填入 $1 \sim 9$ 使得每行、每列、每个 $3 \times 3$ 宫格内数字不重复。

### 核心思路
1. **回溯搜索 + 约束传播**：逐个尝试空格填入合法数字，若无合法数字则回溯。
2. 使用三个布尔数组 `row[9][10]`、`col[9][10]`、`box[9][10]` 记录每行/每列/每宫已使用的数字。
3. **优化**：优先选择可填数字最少的空格（MRV 启发式），大幅减少搜索分支。
4. 也可用位运算加速：用整数的二进制位表示可填数字集合，用 `lowbit` 快速枚举。

### 算法流程
1. 读入 81 字符串，初始化行/列/宫的占用标记
2. 选取可填选择最少的空格
3. 枚举该空格所有合法填法，递归搜索
4. 若所有空格填满则输出结果
5. 回溯直到找到解

### 复杂度分析
- 时间复杂度：最坏 $O(9^{81})$，实际因约束剪枝极快
- 空间复杂度：$O(81)$
