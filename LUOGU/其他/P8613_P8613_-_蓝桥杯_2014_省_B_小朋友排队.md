# P8613 - 蓝桥杯 2014 省 B 小朋友排队

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 个小朋友按身高排序，每次只能交换相邻两人。每次交换增加不高兴程度（第 k 次交换加 k）。求最小不高兴程度之和。

### 核心思路

归并排序求逆序对 + BIT。每个小朋友的交换次数等于其逆序对数（前面比他高的 + 后面比他矮的）。总不高兴程度 = Σ k_i(k_i+1)/2。

### 复杂度分析

时间复杂度：O(n log n)，空间复杂度：O(n)

### 关键技巧

使用 BIT 分别计算每个位置的前缀逆序对和后缀逆序对。
