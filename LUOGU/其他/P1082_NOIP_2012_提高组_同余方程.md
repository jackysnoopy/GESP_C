# [NOIP 2012 提高组] 同余方程

## 题目描述

求关于 ` x` 的同余方程 ` a x \equiv 1 \pmod {b}` 的最小正整数解。

## 输入格式

一行，包含两个整数 `a,b`，用一个空格隔开。

## 输出格式

一个整数 `x_0`，即最小正整数解。输入数据保证一定有解。

## 样例

### 样例输入
```
3 10
```

### 样例输出
```
7
```

## 提示

### 数据规模与约定

- 对于 `40\%` 的数据，`2 \le b\le  10^3`；
- 对于 `60\%` 的数据，`2 \le b\le  5	imes 10^7`；
- 对于 `100\%` 的数据，`2 \le a, b\le  2	imes 10^9`。

## 解题思路

### 问题分析

求 ax ≡ 1 (mod b) 的最小正整数解，即求 a 模 b 的乘法逆元。

### 核心思路

1. **扩展欧几里得算法**：求 ax + by = gcd(a,b) 的一组特解。
2. 当 gcd(a,b)=1 时，ax ≡ 1 (mod b) 有唯一解 x = x0 mod b。
3. 将 exgcd 的结果 x0 取模并调整为正数。

### 算法流程

1. 读入 a, b
2. 调用扩展欧几里得求 ax + by = 1 的解
3. 将解取模 b 调整为最小正整数

### 复杂度分析

时间 O(log(min(a,b)))

## 参考代码

```cpp
#include <iostream>
using namespace std;

long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) { x = 1; y = 0; return a; }
    long long g = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return g;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long a, b, x, y;
    cin >> a >> b;
    exgcd(a, b, x, y);
    cout << ((x % b + b) % b) << '\n';
    return 0;
}
```
