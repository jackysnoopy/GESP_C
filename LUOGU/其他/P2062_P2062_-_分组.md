# P2062 - 分组

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **贪心** 类问题。

### 核心思路

排序后从大到小贪心分组，第i个人需要>=a[i]人的队伍。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    sort(a.begin(), a.end(), greater<int>());
    int ans = 0;
    int i = 0;
    while (i < n) {
        // Find team size: first person wants >= a[i], team needs at least a[i] people
        int need = a[i];
        if (i + need <= n) {
            ans++;
            i += need;
        } else break;
    }
    cout << ans << "\n";
    return 0;
}

```
