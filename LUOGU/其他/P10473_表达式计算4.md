# 表达式计算4

## 题目描述
给出一个表达式，其中运算符仅包含 +，-，*，/，^，要求求出表达式的最终值。

数据可能会出现括号情况，还有可能出现多余括号情况。

数据保证不会出现超过 int 范围的数据，数据可能会出现负数情况。

## 输入格式
仅一行一个字符串，即为表达式。

## 输出格式
仅一行，既为表达式算出的结果。

## 样例
### 样例 1
```
(2+2)^(1+1)
```
```
16
```

## 提示
表达式总长度不超过 $30$。

## 解题思路
### 问题分析
表达式求值，支持 +,-,*,/,^ 和括号。可能有括号不匹配的情况。

### 核心思路
中缀转后缀再求值。处理多余括号：在表达式两端加足够多的括号。

### 算法流程
1. 读入表达式
2. 去除多余括号
3. 中缀转后缀（考虑优先级）
4. 计算后缀表达式

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <stack>
#include <string>
#include <cmath>
using namespace std;

int pri(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    if (op == '^') return 3;
    return 0;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s; cin >> s;
    // 补括号处理多余括号
    s = '(' + s + ')';
    
    // 中缀转后缀
    string post;
    stack<char> ops;
    
    for (int i = 0; i < (int)s.size(); i++) {
        char c = s[i];
        if (c == '(') {
            ops.push(c);
        } else if (c == ')') {
            while (!ops.empty() && ops.top() != '(') {
                post += ops.top(); ops.pop();
            }
            if (!ops.empty()) ops.pop();
        } else if (c == '+' || c == '-' || c == '*' || c == '/' || c == '^') {
            // 处理负号
            if (c == '-' && (i == 0 || s[i-1] == '(')) {
                post += '0';
            }
            while (!ops.empty() && pri(ops.top()) >= pri(c)) {
                post += ops.top(); ops.pop();
            }
            ops.push(c);
        } else {
            post += c;
        }
    }
    
    // 计算后缀
    stack<int> nums;
    for (char c : post) {
        if (c >= '0' && c <= '9') {
            nums.push(c - '0');
        } else {
            int b = nums.top(); nums.pop();
            int a = nums.top(); nums.pop();
            switch (c) {
                case '+': nums.push(a + b); break;
                case '-': nums.push(a - b); break;
                case '*': nums.push(a * b); break;
                case '/': nums.push(a / b); break;
                case '^': nums.push((int)pow(a, b)); break;
            }
        }
    }
    
    cout << nums.top() << "\n";
    return 0;
}
```
