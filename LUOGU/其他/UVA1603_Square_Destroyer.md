# Square Destroyer

## 题目描述
最少火柴棍删除使所有正方形消失。

## 输入格式
多组格子规模与初始火柴状态。

## 输出格式
最小删除数。

## 样例
### 样例 1
```
2
1011001
0010010
1010010
0101011
```
```
2
```

## 解题思路
### 问题分析
给定一个由火柴棍组成的网格，部分火柴可能已被删除（用 0 表示已删除，1 表示存在）。目标是删除最少的火柴棍，使得图中不再存在任何正方形。

### 核心思路
1. **IDA* 搜索**：Iterative Deepening A*，逐步增加删除火柴的数量上限进行搜索。
2. **状态表示**：用位运算表示每根火柴的存在状态，预处理所有可能的正方形（1×1, 2×2, ..., n×n）包含哪些火柴。
3. **启发函数**：统计当前剩余的未被破坏的正方形数量，作为下界估计。
4. **剪枝**：
   - 若当前步数 + 启发值 > 上限则剪枝
   - 优先删除被最多正方形共用的火柴

### 算法流程
1. 读入网格大小和火柴状态
2. 预处理所有正方形及其包含的火柴编号
3. IDA* 搜索：从上限 = 0 开始，逐步增加
4. 每层搜索中，选择一个未被破坏的正方形，枚举删除其某根火柴
5. 若所有正方形都被破坏则返回当前步数

### 复杂度分析
- 时间复杂度：IDA* 在小规模下高效，实际接近 $O(b^d)$，$b$ 为分支因子，$d$ 为最优解深度
- 空间复杂度：$O(M)$，$M$ 为火柴总数
