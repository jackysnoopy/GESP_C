# P1036 [NOIP 2002 普及组] 选数

## 题目分析
- 从 $n$ 个整数中选 $k$ 个，求和为质数的方案数。$n\le20$，$k<n$，$x_i\le5\times10^6$。

## 核心思路
$C(n,k)$ 最多 $C(20,10)=184756$，枚举所有 $k$ 元子集求和判质。和最大 $10^8$，判质用试除 $O(\sqrt{\text{sum}})$。

## 算法流程
1. 读 $n,k$ 与数组。
2. DFS/组合枚举选 $k$ 个，累加和。
3. 对每个和判质（$<2$ 非质，否则试除到 $\sqrt{\text{sum}}$），计数。
4. 输出方案数。

## 复杂度分析
- 时间 $O(C(n,k)\cdot\sqrt{\text{sum}})$，可接受。

## 样例验证
$[3,7,12,19], k=3$：和 22,29,38,34，仅 29 为质 → 输出 `1`。

## 解题思路

### 问题分析
从 n 个整数中选 k 个数，求这 k 个数之和为质数的方案数。n ≤ 20，数据规模允许暴力枚举所有组合。

### 核心思路
使用 DFS 枚举所有 C(n,k) 种选法，对每种选法计算 k 个数之和，判断和是否为质数。由于 n ≤ 20，C(20,10) = 184756，暴力枚举完全可行。

### 算法流程
1. 读入 n、k 和 n 个数
2. DFS(start, step) 枚举组合：
   - step == k+1 时，计算已选 k 个数的和 sum
   - 调用 isprime(sum) 判断是否为质数（<2 非质，否则试除到 √sum）
   - 是质数则 cnt++
3. 从 start 位置开始，依次选择每个数进入下一层
4. 输出 cnt

### 复杂度分析
- 时间复杂度：O(C(n,k) × √sum)，sum 最大约 10^8
- 空间复杂度：O(k)，递归栈深度
