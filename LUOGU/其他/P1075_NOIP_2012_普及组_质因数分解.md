# P1075 [NOIP 2012 普及组] 质因数分解

## 题目分析
- $n$ 是两个不同质数 $p<q$ 的乘积，求较大的 $q$。

## 核心思路
从 2 起找 $n$ 的第一个因子 $p$（必为较小质数），答案 $= n/p$。$n\le 2\times10^9$，$p\le\sqrt n\approx44721$，枚举到 $\sqrt n$ 即可。

## 算法流程
1. 读 $n$。
2. $i$ 从 2 到 $\sqrt n$，找第一个 $n\bmod i=0$ 的 $i$。
3. 输出 $n/i$。

## 复杂度分析
- 时间 $O(\sqrt n)$，空间 $O(1)$。

## 样例验证
$n=21$，最小因子 $3$，答案 $21/3=7$，输出 `7`。

## 解题思路

### 问题分析
已知 n = p × q（p < q 均为质数），求较大的质数 q。

### 核心思路
从 2 开始枚举，找到 n 的第一个因子 i（即最小的 p），则 q = n / i。由于 p < q，p 必然 ≤ √n，因此只需枚举到 √n 即可找到 p。

### 算法流程
1. 读入 n
2. i 从 2 枚举到 √n：
   - 若 n % i == 0，说明 i 就是较小的质因子 p
   - 输出 n / i 即为较大的质因子 q
   - 立即返回

### 复杂度分析
- 时间复杂度：O(√n)，枚举到 √n 即可
- 空间复杂度：O(1)，只用常数变量
