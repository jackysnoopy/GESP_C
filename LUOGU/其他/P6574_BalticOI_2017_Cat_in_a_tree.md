# [BalticOI 2017] Cat in a tree

## 题目描述

小猫在一棵有 $n$ 个节点的树上，它通过标记节点来划分领地。  
它标记的节点满足彼此距离不小于 $d$。  
两节点之间的距离指的是两点间路径上边的个数。
求小猫最多能标记多少个节点。

## 样例

--- Sample 1 ---
Input:
4 3
0
0
1
Output:
2
--- Sample 2 ---
Input:
3 1000
0
0
Output:
1
