# P2301 - 排序配对

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **贪心** 类问题。

### 核心思路

排序后相邻配对使差的平方和最小。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int m, n;
    cin >> m >> n;
    vector<int> a(m);
    for (int i = 0; i < m; i++) cin >> a[i];
    sort(a.begin(), a.end());
    // Choose n pairs from m sticks, each person picks 2
    // Minimize sum of (L1-L2)^2
    // Greedy: pair consecutive elements after sorting
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        long long diff = a[2 * i + 1] - a[2 * i];
        ans += diff * diff;
    }
    cout << ans << "\n";
    return 0;
}

```
