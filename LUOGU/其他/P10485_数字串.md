# 数字串

## 题目描述
给定数字串与表达式模板，最大化结果。

## 输入格式
数字串、模板。

## 输出格式
最大结果。

## 样例
### 样例 1
```
72387592
0+0*0*0
```
```
62304
```

## 解题思路
### 问题分析
本题给定一个数字串和一个由 `+`、`*` 和 `0` 组成的表达式模板，需要将数字串中的数字按顺序填入模板中的 `0` 位置，使得表达式的值最大。由于乘法优先级高于加法，需要考虑加法和乘法的组合优化。

### 核心思路
1. 分析模板结构：将模板按 `+` 分割成若干段，每段内是 `*` 连接的乘积。
2. **动态规划**：设 $dp[i][j]$ 表示前 $i$ 个数字填入模板前 $j$ 个位置时的最大值。
3. 关键在于处理乘法段和加法段的分界。对于每段乘法，需要将数字分配到乘法因子中使得乘积最大（贪心：尽量把大数字放在高位）。
4. 可以按 `+` 号将模板分成若干组，每组内是乘法运算。对每组，贪心地分配数字以最大化乘积，然后求和。

### 算法流程
1. 解析模板，按 `+` 分割为若干乘法段
2. 统计每个乘法段需要的数字个数
3. 对每个乘法段，贪心地分配数字使乘积最大
4. 求所有乘法段结果之和

### 复杂度分析
- 时间复杂度：$O(n \log n)$，其中 $n$ 为数字串长度
- 空间复杂度：$O(n)$
