# P7871 - 「Wdoi-4」芙兰？姆Q！贤者与谜题

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 块贤者之石排成一列，需要赋予不同的正整数能级。能量从激发态的石传递给相邻能级较小的石。给定 q 个条件 (s,t)，表示从 s 激发最终经过 t。求字典序最小的能级方案。

### 核心思路

这是一个构造问题。需要分析能量传递的路径约束，将条件转化为能级的偏序关系。利用拓扑排序或贪心构造字典序最小的方案。

### 复杂度分析

时间复杂度：O(n + q log n)，空间复杂度：O(n + q)

### 关键技巧

注意字典序最小的要求，需要从左到右贪心赋值。处理矛盾条件输出 QED。
