# [NOISG 2017] RMQ

## 题目描述

小 K 有 $N$ 头奶牛，编号分别为 $0$ 至 $N-1$。这些奶牛以某种未知的顺序排成了一排。

现在给出 $Q$ 条信息，每条信息包含三个整数 $L_i,R_i,A_i$，表示区间 $[L_i,R_i]$ 内编号最小的奶牛的编号为 $A_i$。请你构造一组奶牛的顺序使得其可以满足所有信息。

## 样例

--- Sample 1 ---
Input:
5 3
0 2 1
1 3 0
1 4 0
Output:
1 4 3 0 2
--- Sample 2 ---
Input:
3 2
0 1 1
1 2 1
Output:
-1 -1 -1
