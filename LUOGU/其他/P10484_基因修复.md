# 基因修复

## 题目描述
给定主串与词典，最少插入字符使主串包含词典所有词。

## 输入格式
主串、词典规模与词典。

## 输出格式
最少插入数。

## 样例
### 样例 1
```
samplefix
1
sample
```
```
4
```

## 解题思路
### 问题分析
给定一个主串和若干模式串（词典），要求在主串中插入最少的字符，使得插入后的字符串包含词典中所有模式串作为子串。注意是插入而非替换。

### 核心思路
1. **AC 自动机 + 状态压缩 DP**：
   - 构建 AC 自动机处理词典，每个节点记录匹配到该节点时已覆盖的模式串集合（用位掩码表示）。
   - $dp[i][j][S]$ 表示处理到主串第 $i$ 个字符、在 AC 自动机节点 $j$、已覆盖模式串集合 $S$ 时的最少插入次数。
2. **转移**：
   - **不插入**：主串第 $i$ 个字符转移到自动机的下一个状态
   - **插入**：尝试插入 $A \sim Z$ 中任意一个字符，转移到自动机对应状态
3. 答案为 $dp[n][j][\text{all}]$ 的最小值。

### 算法流程
1. 读入主串和词典，构建 AC 自动机
2. 初始化 DP 数组
3. 对每个位置 $i$、每个自动机节点 $j$、每个状态 $S$：
   - 尝试不插入（使用主串当前字符）
   - 尝试插入 26 个字母
4. 更新 DP 值
5. 输出覆盖所有模式串的最小插入次数

### 复杂度分析
- 时间复杂度：$O(n \times N \times 2^k \times 26)$，其中 $n$ 为主串长度，$N$ 为自动机节点数，$k$ 为模式串个数
- 空间复杂度：$O(n \times N \times 2^k)$，可用滚动数组优化至 $O(N \times 2^k)$
