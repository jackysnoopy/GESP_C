# P8539 - 「Wdoi-2」来自地上的支援

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 个核反应机组，每次操作在前 i 个中选活动度最高的加 v。m 次询问：修改第 x 个机组初始值为 s，使其至少被选中 k 次，求最小 s。

### 核心思路

这是一道复杂的模拟 + 二分题。需要分析选择过程的规律，利用数据结构维护选择序列。

### 复杂度分析

时间复杂度：O(n log n + m log n)，空间复杂度：O(n)

### 关键技巧

注意 IO 量较大，使用快速 IO。v ≤ 100 的约束可以利用。
