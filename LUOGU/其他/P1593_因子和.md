# 因子和

## 题目描述
输入两个整数 $a$ 和 $b$，求 $a^b$ 的因子和。

由于结果太大，只要输出它对 $9901$ 取模的结果。

## 输入格式
仅一行，为两个整数 $a$ 和 $b$。

## 输出格式
输出一行一个整数表示答案对 $9901$ 取模的结果。

## 样例
### 样例 1
```
2 3
```
```
15
```

## 提示
#### 数据规模与约定

对于全部的测试点，保证 $1 \leq a \leq 5 \times 10^7$，$0 \leq b \leq 5 \times 10^7$。

## 解题思路

### 问题分析
求 a^b 的因子之和模 9901。

### 核心思路
1. 质因数分解 a = p₁^c₁ × p₂^c₂ × ...
2. a^b = p₁^(c₁·b) × p₂^(c₂·b) × ...
3. 因子和 = ∏(1 + p_i + p_i² + ... + p_i^(c_i·b))
4. 每个等比数列用分治法或公式求和，注意模运算

### 算法流程
1. 读入 a, b
2. 对 a 质因数分解
3. 对每个质因数 p 和指数 c，计算等比数列和 (p^(cb+1)-1)/(p-1) mod MOD
4. 连乘所有结果

### 复杂度分析
- 时间复杂度：O(√a)
- 空间复杂度：O(log a)

## 参考代码
```cpp
#include <iostream>
#include <vector>
using namespace std;

const int MOD = 9901;

long long pow_mod(long long a, long long b) {
    long long res = 1;
    while (b) {
        if (b & 1) res = res * a % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}

long long sum_geo(long long p, long long c) {
    // 1 + p + p^2 + ... + p^c
    if (c == 0) return 1;
    if (c % 2 == 0) {
        return (sum_geo(p, c/2 - 1) * (1 + pow_mod(p, c/2 + 1)) + pow_mod(p, c/2)) % MOD;
    } else {
        return sum_geo(p, c/2) * (1 + pow_mod(p, c/2 + 1)) % MOD;
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long a, b;
    cin >> a >> b;
    
    if (a == 0) {
        cout << 0 << "
";
        return 0;
    }
    
    long long ans = 1;
    for (long long p = 2; p * p <= a; p++) {
        if (a % p == 0) {
            long long c = 0;
            while (a % p == 0) {
                a /= p;
                c++;
            }
            ans = ans * sum_geo(p % MOD, c * b) % MOD;
        }
    }
    if (a > 1) {
        ans = ans * sum_geo(a % MOD, b) % MOD;
    }
    
    cout << ans << "
";
    return 0;
}
```
