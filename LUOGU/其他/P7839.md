# P7839 - 「Wdoi-3」夜雀 singing

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 棵树排成一列，m 只夜雀在特定位置。歌会进行 t 时刻，第 i 个时刻只能在高度 > i 的树上唱歌。可以设置飞行点让夜雀瞬间移动。求最少飞行点数量。

### 核心思路

这是一个区间覆盖/贪心问题。需要分析每只夜雀在每个时刻的可达范围，找出必须设置飞行点的位置。核心思路是分析树的高度约束，将问题转化为区间覆盖。

### 复杂度分析

时间复杂度：O(n + m)，空间复杂度：O(n)

### 关键技巧

注意 n,m 可达 5×10^6，需要 O(n) 算法。利用高度约束的单调性进行贪心。
