# Counting Sequences

## 题目描述
对给定 n 找最小长度递增正整数序列使其和为 n。

## 输入格式
若干行 n（0 结束）。

## 输出格式
每行一个序列。

## 样例
### 样例 1
```
21
```
```
1 2 3 4 5 6
```

## 解题思路
### 问题分析
给定一个正整数 $n$，找到一个严格递增的正整数序列，使其和恰好等于 $n$，且序列长度最小。如果长度相同，选择字典序最小的序列。

### 核心思路
1. **搜索（DFS）**：由于序列严格递增且和为 $n$，最大长度不超过 $O(\sqrt{n})$（因为 $1+2+\cdots+k = k(k+1)/2 \le n$）。
2. 从最小长度开始尝试，找到第一个可行解即为最优解。
3. DFS 时维护当前已选数字之和、最后一个数字（下一个必须更大）、剩余需要凑的值。
4. **剪枝**：
   - 若剩余值无法用递增序列凑出（最小可能和 > 剩余值），剪枝
   - 若当前长度已超过目标长度，剪枝

### 算法流程
1. 读入 $n$
2. 从小到大枚举序列长度 $L$（$L$ 从 1 开始）
3. DFS 搜索长度为 $L$ 的递增序列使其和为 $n$
4. 若找到则输出并结束；否则 $L$ 增 1 继续
5. 输出序列

### 复杂度分析
- 时间复杂度：$O(C(n, L))$ 的组合搜索，实际因剪枝很快
- 空间复杂度：$O(L)$
