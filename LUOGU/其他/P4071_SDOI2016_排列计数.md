# [SDOI2016] 排列计数

## 题目描述

求有多少种 $1$ 到 $n$ 的排列 $a$，满足序列恰好有 $m$ 个位置 $i$，使得 $a_i = i$。

答案对 $10^9 + 7$ 取模。

## 输入格式

**本题单测试点内有多组数据**。

输入的第一行是一个整数 $T$，代表测试数据的组数。

以下 $T$ 行，每行描述一组测试数据。

对于每组测试数据，每行输入两个整数，依次代表 $n$ 和 $m$。

## 输出格式

共输出 $T$ 行，对于每组测试数据，输出一行一个整数代表答案。

## 样例

### 样例输入
```
5
1 0
1 1
5 2
100 50
10000 5000
```

### 样例输出
```
0
1
20
578028887
60695423
```

## 提示

#### 数据规模与约定

本题共 20 个测试点，各测试点等分，其数据规模如下表。

| 测试点编号 | $T =$  | $n, m \leq$ |  测试点编号  |      $T =$      | $n, m \leq$ |
| :--------: | :----: | :---------: | :----------: | :-------------: | :---------: |
| $1\sim 3$  | $10^3$ |     $8$     | $10 \sim 12$ |     $10^3$      |   $10^3$    |
| $4 \sim 6$ | $10^3$ |    $12$     | $13 \sim 14$ | $5 	imes 10^5$ |   $10^3$    |
| $7 \sim 9$ | $10^3$ |    $100$    | $15 \sim 20$ | $5 	imes 10^5$ |   $10^6$    |

对于全部的测试点，保证 $1 \leq T \leq 5 	imes 10^5$，$1 \leq n \leq 10^6$，$0 \leq m \leq 10^6$。

## 解题思路

### 问题分析

求 n 的排列中恰好有 m 个位置 a[i]=i 的方案数。

### 核心思路

1. 先选 m 个位置固定为不动点：C(n,m)。
2. 剩余 n-m 个位置全为错排：D(n-m)（错排公式）。
3. 答案 = C(n,m) * D(n-m) mod p。
4. 错排 D(k) = (k-1) * (D(k-1) + D(k-2))。

### 算法流程

1. 预处理阶乘和错排
2. 对每组读入 n,m
3. 输出 C(n,m)*D(n-m)

### 复杂度分析

时间 O(n + q)

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;
const long long MOD = 1000000007;
long long power(long long b, long long e, long long m) {
    long long r = 1; b %= m;
    while (e > 0) { if (e & 1) r = r * b % m; b = b * b % m; e >>= 1; }
    return r;
}
int main() {
    ios::sync_with_stdio(false); cin.tie(nullptr);
    int T; cin >> T;
    int MAX = 1000000;
    vector<long long> fac(MAX + 1, 1), invf(MAX + 1, 1);
    for (int i = 1; i <= MAX; ++i) fac[i] = fac[i - 1] * i % MOD;
    invf[MAX] = power(fac[MAX], MOD - 2, MOD);
    for (int i = MAX - 1; i >= 0; --i) invf[i] = invf[i + 1] * (i + 1) % MOD;
    vector<long long> D(MAX + 1, 0);
    D[0] = 1; D[1] = 0;
    for (int i = 2; i <= MAX; ++i) D[i] = (i - 1) * (D[i - 1] + D[i - 2]) % MOD;
    while (T--) {
        int n, m; cin >> n >> m;
        long long c = fac[n] % MOD * invf[m] % MOD * invf[n - m] % MOD;
        cout << c * D[n - m] % MOD << '\n';
    }
    return 0;
}
```
