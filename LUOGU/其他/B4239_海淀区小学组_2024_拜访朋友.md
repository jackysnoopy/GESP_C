# [海淀区小学组 2024] 拜访朋友

## 题目描述

2024 年海淀区中小学生信息学竞赛小学组复赛题目，数据为洛谷自造。

陶陶有 $n$ 个好朋友，他们的住所都分布在同一条笔直的街道旁，如果把这条街道看作一个数轴，选定一个点作为坐标原点（坐标为 $0$），则这 $n$ 个好朋友的住所的坐标分别为 $x_1, x_2, \dots, x_n$。一天，陶陶要去朋友们的住所去拜访朋友们，陶陶初始位置的坐标为 $x_0$，他希望至少拜访到 $n - 1$ 个好朋友，请你确定陶陶要最少要行走的总距离。

## 输入格式

第一行包含两个整数 $n$ 和 $x_0$，第二行包含 $n$ 个整数 $x_1, x_2, \dots, x_n$。

## 输出格式

仅有一个正整数，表示最小的总行走距离。

## 样例

### 样例输入1
```3 10
1 7 12
```

### 样例输出1
```7
```

### 样例输入2
```2 0
11 -10
```

### 样例输出2
```10
```

### 样例输入3
```5 0
0 0 1000 0 0
```

### 样例输出3
```0
```

## 提示

对于 $100\%$ 的数据：$1 \leq n \leq 10^5$，对于每一个正整数 $i$，$1 \leq i \leq n$，都有 $-10^6 \leq x_i \leq 10^6$。

## 解题思路

要拜访 n-1 个朋友，即跳过一个。总距离 = sum|xi-x0|，跳过距离最大的那个朋友可最小化。

### 算法流程

计算总距离 sum，找最大 |xi-x0|，答案为 sum - max|xi-x0|。

### 复杂度分析

时间复杂度：O(n log n)

空间复杂度：O(n) 或 O(1)
