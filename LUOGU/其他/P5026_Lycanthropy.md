# Lycanthropy

## 题目描述

我们把山顶上的湖泊看作一条长度为 $m$ 的直线，一开始水深都在水平线上，我们视作此时的水深为 $0$。

接下来，在一瞬间，小正方形的"朋友"们跳起并扎入水中，导致在入水点的水降低而远离入水点的水升高，注意两个 "朋友" 可能在同一地点入水。

小正方形的每个朋友有一个体积数值 $v$，当体积为 $v$ 的一个朋友跳入水中，我们设入水点为 $i$，将会导致 $i - v + 1$ 到 $i$ 的水位依次降低 $1,2,\cdots,v$。

同样地，第 $i$ 到 $i + v - 1$ 的水位会依次降低 $v,v - 1,\cdots,1$。

相对应地，$i - v$ 的水位不变， $i - v - 1$ 到 $i - 2 \times v$ 水位依次增加 $1,2,\cdots,v$， $i - 2 \times v$ 到 $i - 3 \times v + 1$ 水位依次增加 $v,v - 1,\cdots,1$。

同样，$i + v$ 水位不变，$i + v + 1$ 到 $i + 2 \times v$ 水位增加 $1,2,\cdots,v$，$i + 2 \times v$ 到 $i + 3 \times v - 1$ 水位依次增加 $v,v - 1,\cdots,1$。

现在小正方形想要穿过这个湖，他想要知道在这 $n$ 个"朋友"跳入水中后湖上每个节点的水位，你能帮帮它吗？

## 输入格式

第一行为两个整数 $n$,$m$，表示"朋友"的数目与湖泊的宽度。

接下来 $n$ 行，一行两个整数 $v,x$，表示第 $i + 1$ 个朋友的体积与入水点。

## 输出格式

一行 $m$ 个整数，第 $i$ 个整数表示 $i$ 号位的水深。

## 样例 #1

**输入**
```
1 10
1 5
```

**输出**
```
0 0 1 0 -1 0 1 0 0 0
```

## 样例 #2

**输入**
```
2 10
2 6
3 1
```

**输出**
```
-2 0 0 0 0 0 2 2 2 2
```

## 提示

对于 $30\%$ 的数据，$n \le 50,m \le 500$。

对于 $70\%$ 的数据，$n \le 10^5,m \le 10^5$。

对于 $100\%$ 的数据，$n \le 10^6,1 \le x \le m \le 10^6,1 \le v \le 10000$。

## 解题思路

### 问题分析

长度为 $m$ 的直线湖泊，初始水深为 $0$。$n$ 个朋友跳入水中，每个人有体积 $v$ 和入水点 $x$，产生对称的波形：入水点水位下降，远离入水点水位上升。求最终水深分布。

### 核心思路

**对称差分。** 每个人的影响是对称的，可以分左右两侧分别用差分数组处理。每个朋友产生的影响可以分解为四个等差数列区间：
- 区间 $[x-3v+1, x-2v]$：斜率 $+1$ 上升
- 区间 $[x-2v+1, x]$：斜率 $-1$ 下降
- 区间 $[x+1, x+2v]$：斜率 $-1$ 下降
- 区间 $[x+2v+1, x+3v-1]$：斜率 $+1$ 上升

用一阶差分处理这些斜率变化。

### 算法流程

1. 初始化差分数组 $diff$（大小 $m+5$）。
2. 对于每个朋友 $(v,x)$：
   - 计算四个影响区间，在 $diff$ 上标记 $+1$ 或 $-1$。
   - 注意边界检查，防止越界。
3. 对 $diff$ 做前缀和得到最终水深。
4. 输出。

### 复杂度分析

- **时间复杂度：** $O(n + m)$。
- **空间复杂度：** $O(m)$。

### 要点

- 每个朋友的影响是左右对称的。以 $x$ 为中心，左右各影响 $3v-1$ 的距离。
- 处理边界：如果区间起点 $< 1$ 或终点 $> m$，需要裁剪。
- 使用 `long long` 避免累加溢出。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m;
    cin >> n >> m;
    vector<long long> dif(m + 5, 0);
    while (n--) {
        int v, x;
        cin >> v >> x;
        int l1 = max(1, x - 3*v + 1), r1 = x - 2*v;
        int l2 = max(1, x - 2*v + 1), r2 = x;
        int l3 = max(1, x + 1), r3 = min(m, x + 2*v);
        int l4 = max(1, x + 2*v + 1), r4 = min(m, x + 3*v - 1);
        if (l1 <= r1) { dif[l1] += 1; dif[r1+1] -= 1; }
        if (l2 <= r2) { dif[l2] -= 1; dif[r2+1] += 1; }
        if (l3 <= r3) { dif[l3] -= 1; dif[r3+1] += 1; }
        if (l4 <= r4) { dif[l4] += 1; dif[r4+1] -= 1; }
    }
    for (int i = 1; i <= m; i++) {
        dif[i] += dif[i-1];
        cout << dif[i] << " \n"[i == m];
    }
    return 0;
}
```
