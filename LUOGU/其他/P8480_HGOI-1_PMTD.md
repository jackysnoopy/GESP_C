# 「HGOI-1」PMTD

## 题目描述

$\text{uuku}$ 在学习[四则运算](https://baike.baidu.com/item/%E5%9B%9B%E5%88%99%E8%BF%90%E7%AE%97/5337481?fr=aladdin)！

为了验证 $\text{uuku}$ 学习成果，$\text{bh1234666}$ 给出一个长为 $n$ 整数序列 $a_i$。并让 $\text{uuku}$ 给这个序列进行 $m$ 次操作。

每次操作可以任意选择序列中一个数 $a_i$，令 $a_i$ 变成 $a_i+2$，$a_i-2$，$a_i\times 2$，$\lfloor\frac{a_i}{2}\rfloor$ 这四个结果中的一个。

$\text{bh1234666}$ 希望 $m$ 次操作后，整个序列的极差（最大值减最小值）最大。

显然 $\text{uuku}$ 没有认真学习，所以他希望你来帮他回答这个问题。

## 输入格式

第一行两个整数 $n$，$m$。

第二行 $n$ 个整数，表示序列 $a_i$。

## 输出格式

共一行一个整数，表示最大的极差。

## 样例

### 样例输入1
```3 2
0 1 0
```

### 样例输出1
```6
```

## 提示

#### 样例解释

第一步操作：将 $1$ 加上 $2$ 得到 $3$。

第二步操作：将 $3$ 乘以 $2$ 得到 $6$。

极差为 $6-0=6$。

#### 数据范围

本题采用**捆绑测试**，共有 $2$ 个 $\text{subtask}$，最终分数为所有 $\text{subtask}$ 分数之和。

$$
\def\arraystretch{1.5}
\begin{array}{|c|c|c|}\hline
\textbf{Task} & \textbf{Score} & \textbf{特殊限制} \cr\hline
1 & 40 & n \le 5,m \le 5 \cr\hline
2 & 60 &  \cr\hline
\end{array}
$$

对于 $100\%$ 的数据，$2 \le n \le 10^6$，$1 \le m \le 10$，$0 \le a_i \le 10^9$。

## 解题思路

### 问题分析

给定长度为 n 的整数序列，每次操作可对任一元素执行 +2、-2、×2、⌊÷2⌋ 四种操作之一，共 m 次操作，求操作后序列极差（最大值减最小值）的最大值。

### 核心思路

关键观察：所有四种操作都是**单调非递减**函数（即若 a < b，则 f(a) ≤ f(b) 对所有操作 f 成立）。因此，对每个元素，其在 k 次操作后能达到的最大值和最小值，可以通过贪心地在每一步选择最优操作得到：
- 最大值：每步取 max(v+2, v-2, 2v, ⌊v/2⌋)
- 最小值：每步取 min(v+2, v-2, 2v, ⌊v/2⌋)

对所有元素分别计算 k=0..m 次操作后的最大/最小值，再对每种操作分配方案（k 次给最大值元素，m-k 次给最小值元素），取全局最优。

### 算法流程

1. 读入序列 a 和操作次数 m
2. 对每个元素，用 DP 递推 k=0..m 步的 maxVal[k] 和 minVal[k]
3. 维护每个 k 值对应的全局最大/第二大 maxVal 和最小/第二小 minVal（及来源元素索引）
4. 枚举操作分配 k（0..m），若最大和最小来自不同元素则直接计算，否则取次优组合
5. 输出最大极差

### 复杂度分析

- 时间复杂度：O(n × m)，n ≤ 10^6，m ≤ 10，约 10^7 次操作
- 空间复杂度：O(m)
