# 醒来之后，我回到了汰换系统更新前

## 题目描述

下楼买烟之后，我重生了，我回到了 CS2 汰换系统更新前一天。

现在我仓库里有 $n$ 个物品，每个物品的购入价为 $v_{i}$，我清楚的记得接下来 $m$ 天里每个物品的价格 $w_{i}$，而我现在所需要做的，就是最大化我的收益，但是我不希望任何一个物品有亏损，如果没办法盈利，我将把这个物品留下来，这个物品不会计入盈亏，我一定要活下去口牙。（在本题中，所有物品只允许卖出。）

## 输入格式

第一行两个正整数 $n,m\ (1\le n\le 100,1\le m\le 100)$，表示物品数量以及天数。

第二行包含 $n$ 个正整数，第 $i$ 个正整数为 $v_{i}\ (1\le v_{i}\le 10^7)$，表示第 $i$ 个物品的购入价格。

接下来 $m$ 行每行 $n$ 个正整数，第 $i$ 行第 $j$ 个正整数为 $w_{i,j}\ (1\le w_{i,j}\le 10^7)$，表示第 $i$ 天第 $j$ 个物品的价格。

## 输出格式

一行一个正整数，表示我最终的最大收益。

## 样例

### 样例输入1
```1 1
1
10
```

### 样例输出1
```9
```

### 样例输入2
```2 3
4 8
2 5
2 99
1 7
```

### 样例输出2
```91
```

## 提示



## 解题思路

### 问题分析

CS2交易系统利润最大化。

### 核心思路

每个物品选最高价卖出，不亏则卖。

### 算法流程

1. 对每个物品找m天最高价
2. 高于买入价则卖出获利

### 复杂度分析

O(n×m)时间，O(n)空间。
