# 阶乘分解

## 题目描述

给定整数 `N (3 \le N \le 10^6)`，试把阶乘 `N!` 分解质因数，按照算术基本定理的形式输出分解结果中的 `p_i` 和 `c_i` 即可。

## 输入格式

一个整数 `N`。

## 输出格式

`N!` 分解质因数后的结果，共若干行，每行一对 `p_i, c_i`，表示含有 `p_i^{c_i}` 项。按照 `p_i` 从小到大的顺序输出。

## 样例

### 样例输入
```
5
```

### 样例输出
```
2 3
3 1
5 1
```

## 提示



## 解题思路

### 问题分析

将 N! 分解质因数。对于每个素数 p，N! 中 p 的指数为 sum(floor(N/p^k))。

### 核心思路

1. **Legendre 公式**：N! 中素数 p 的指数 = floor(N/p) + floor(N/p^2) + floor(N/p^3) + ...
2. 先筛出 [2, N] 的所有素数，再对每个素数用 Legendre 公式求指数。

### 算法流程

1. 读入 N
2. 线性筛出 [2, N] 的素数
3. 对每个素数 p 计算 Legendre 公式
4. 输出 p 和对应指数

### 复杂度分析

时间 O(N)，空间 O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<bool> is_comp(n + 1, false);
    vector<int> primes;
    for (int i = 2; i <= n; ++i) {
        if (!is_comp[i]) primes.push_back(i);
        for (int j = 0; j < (int)primes.size() && i * primes[j] <= n; ++j) {
            is_comp[i * primes[j]] = true;
            if (i % primes[j] == 0) break;
        }
    }
    for (int p : primes) {
        long long cnt = 0;
        long long pk = p;
        while (pk <= n) { cnt += n / pk; pk *= p; }
        cout << p << " " << cnt << '\n';
    }
    return 0;
}
```
