# 【模板】线性筛素数

## 题目描述

如题，给定一个范围 `n`，有 `q` 个询问，每次输出第 `k` 小的素数。

## 输入格式

第一行包含两个正整数 `n,q`，分别表示查询的范围和查询的个数。

接下来 `q` 行每行一个正整数 `k`，表示查询第 `k` 小的素数。

## 输出格式

输出 `q` 行，每行一个正整数表示答案。

## 样例

### 样例输入
```
100 5
1
2
3
4
5
```

### 样例输出
```
2
3
5
7
11
```

## 提示

【数据范围】  
对于 `100\%` 的数据，`n = 10^8`，`1 \le q \le 10^6`，保证查询的素数不大于 `n`。

Data by NaCly_Fish.

## 解题思路

### 问题分析

本题要求线性筛素数。给定范围 n 和 q 个询问，每次输出第 k 小的素数。需要用线性筛（欧拉筛）预处理出所有素数。

### 核心思路

1. **线性筛（欧拉筛）**：每个合数只被其最小质因子筛一次，保证 O(n) 复杂度。
2. 用数组 `is_composite` 标记合数，用 `primes` 数组存储所有素数。
3. 对于每个 i，若未被标记则加入素数列表；然后用已有的素数筛掉 i 的倍数，当 `i % p == 0` 时停止（保证每个合数只被最小质因子筛一次）。
4. 输出第 k 小的素数即为 `primes[k-1]`。

### 算法流程

1. 读入 n, q
2. 线性筛预处理 [1, n] 内所有素数
3. 对每个询问 k，输出 primes[k-1]

### 复杂度分析

时间 O(n)，空间 O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, q;
    cin >> n >> q;
    vector<bool> is_composite(n + 1, false);
    vector<int> primes;
    for (int i = 2; i <= n; ++i) {
        if (!is_composite[i]) primes.push_back(i);
        for (int j = 0; j < (int)primes.size() && i * primes[j] <= n; ++j) {
            is_composite[i * primes[j]] = true;
            if (i % primes[j] == 0) break;
        }
    }
    while (q--) {
        int k;
        cin >> k;
        cout << primes[k - 1] << '\n';
    }
    return 0;
}
```
