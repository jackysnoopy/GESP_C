# [蓝桥杯青少年组国赛 2023] 数学实验

## 题目描述

老师在黑板上写出了一个正整数数列，让所有同学都来做一个数学实验。数列不超过 500000 个数，每个数在 1~80 之间。找出两个相邻且相同的数，删掉其中一个，剩下的一个加 1。重复操作直到无法继续，求能得到的最大数。

## 样例

Input: 6\n1 2 2 2 3 4
Output: 5

## 解题思路

### 问题分析

经典区间合并 DP 问题（P3147 262144）。给定一个数列，合并相邻相同的两个值 v 得到一个 v+1，求最大能得到的值。

### 核心思路

定义 `dp[i][v]`：从位置 i 开始，能否合并出值 v。若能，`dp[i][v]` 存储合并后的区间右端点（即下一个可用位置）。

转移方程：如果 `dp[i][v-1] = k`（即 i 开始合并出 v-1 结束于 k），且 `dp[k+1][v-1]` 存在（即 k+1 开始也能合并出 v-1），则 `dp[i][v] = dp[k+1][v-1]`（两个 v-1 合并为一个 v）。

### 算法流程

1. 初始化：`dp[i][a[i]] = i`（每个位置本身就是一个值为 a[i] 的区间）。
2. 从小到大枚举 v（2 到 80）：
   - 枚举所有位置 i，若 `dp[i][v-1]` 和 `dp[dp[i][v-1]+1][v-1]` 都存在，则 `dp[i][v] = dp[dp[i][v-1]+1][v-1]`。
3. 记录能达到的最大 v。

### 复杂度分析

- **时间复杂度**：$O(n \times 80)$，即 $O(80n)$。
- **空间复杂度**：$O(n \times 80)$，约 40MB，可以接受。

### 关键技巧

- 区间 DP 的经典优化：用 `dp[i][v]` 存储右端点而非 bool 值，可以 O(1) 判断能否继续合并。
- 由于值域只有 80，外层循环只有 80 次，效率很高。
