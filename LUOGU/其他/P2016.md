# P2016 - 最小顶点覆盖

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **树形DP** 类问题。

### 核心思路

dp[u][0/1]表示u选/不选时覆盖子树所需最少士兵。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<vector<int>> adj;
int dp[1505][2];

void dfs(int u, int p) {
    dp[u][0] = 0;
    dp[u][1] = 1;
    for (int v : adj[u]) {
        if (v == p) continue;
        dfs(v, u);
        dp[u][0] += dp[v][1];
        dp[u][1] += min(dp[v][0], dp[v][1]);
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    adj.resize(n);
    for (int i = 0; i < n; i++) {
        int id, k; cin >> id >> k;
        for (int j = 0; j < k; j++) {
            int v; cin >> v;
            adj[id].push_back(v);
            adj[v].push_back(id);
        }
    }
    dfs(0, -1);
    cout << min(dp[0][0], dp[0][1]) << "\n";
    return 0;
}

```
