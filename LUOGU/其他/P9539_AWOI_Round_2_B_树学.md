# 「AWOI Round 2 B」树学

## 题目描述

HR 刚做完一个简单而又靓丽的题目，它想来试试这道题，可是它太蒻了，不会做，您能帮帮它吗？

构造一个长度为 $n$ 的小写字母串，要求和给定的长度为 $n$ 的小写字母串 $s$ 的相似度在 $[l,r]$ 范围内。

定义两个长度都为 $n$ 的字符串 $a,b$ 的相似度为 $\sum^{n}_{i=1}[a_i=b_i]$。

您需要使构造出的字符串的字典序尽量小。

## 输入格式

第一行三个正整数 $n,l,r$。

接下来一行一个字符串，表示给定的小写字母串 $s$。

## 输出格式

输出一行一个字符串，表示您构造的字典序最小的可行字符串。

## 样例

### 样例输入1
```3 1 2
aab
```

### 样例输出1
```aaa
```

### 样例输入2
```4 4 4
awoi
```

### 样例输出2
```awoi
```

## 解题思路

### 问题分析
构造字典序最小的字符串使与s的相似度在[l,r]内。

### 核心思路
先全填'a'，计算与s的匹配数。若在范围内直接输出。若超过r，从后往前将'a'改为'b'减少匹配。若不足l，从后往前将非'a'改为s[i]增加匹配。

### 算法流程
1. 初始化全'a'
2. 计算匹配数
3. 根据需要增减匹配

### 复杂度分析
时间O(N)，空间O(N)。
