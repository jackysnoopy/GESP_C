# 序列合并

## 题目描述
有两个长度为 $N$ 的**单调不降**序列 $A,B$，在 $A,B$ 中各取一个数相加可以得到 $N^2$ 个和，求这 $N^2$ 个和中最小的 $N$ 个。

## 输入格式
第一行一个正整数 $N$；

第二行 $N$ 个整数 $A_{1\dots N}$。

第三行 $N$ 个整数 $B_{1\dots N}$。

## 输出格式
一行 $N$ 个整数，从小到大表示这 $N$ 个最小的和。

## 样例
### 样例 1
```
3
2 6 6
1 4 8
```
```
3 6 7
```

## 提示
对于 $50\%$ 的数据，$N \le 10^3$。

对于 $100\%$ 的数据，$1 \le N \le 10^5$，$1 \le A_i,B_i \le 10^9$。

## 解题思路
### 问题分析
两个升序序列 A,B 各取一个数相加，求最小的 N 个和。

### 核心思路
优先队列。A[0]+B[0] 最小，每次弹出后加入 A[i]+B[j+1]。

### 算法流程
1. 读入 N 和 A, B
2. 将 A[0]+B[i] 入队（记录 A 的下标）
3. 每次取出最小值，将 A[idx+1]+B[i] 入队
4. 输出 N 个结果

### 复杂度分析
- 时间复杂度：O(N log N)
- 空间复杂度：O(N)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n; cin >> n;
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> b[i];
    
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());
    
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
    for (int i = 0; i < n; i++) pq.push({a[0] + b[i], 0});
    
    for (int i = 0; i < n; i++) {
        auto [sum, idx] = pq.top(); pq.pop();
        cout << sum << (i + 1 < n ? " " : "\n");
        if (idx + 1 < n) pq.push({a[idx+1] + sum - a[idx], idx + 1});
    }
    return 0;
}
```
