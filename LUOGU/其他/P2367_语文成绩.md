# 语文成绩

## 题目描述

语文老师总是写错成绩，所以当她修改成绩的时候，总是累得不行。她总是要一遍遍地给某些同学增加分数，又要注意最低分是多少。你能帮帮她吗？

## 输入格式

第一行有两个整数 $n$，$p$，代表学生数与增加分数的次数。

第二行有 $n$ 个数，$a_1 \sim a_n$，代表各个学生的初始成绩。

接下来 $p$ 行，每行有三个数，$x$，$y$，$z$，代表给第 $x$ 个到第 $y$ 个学生每人增加 $z$ 分。

## 输出格式

输出仅一行，代表更改分数后，全班的最低分。

## 样例 #1

**输入**
```
3 2
1 1 1
1 2 1
2 3 1
```

**输出**
```
2
```

## 提示

对于 $40\%$ 的数据，有 $n \le 10^3$。

对于 $60\%$ 的数据，有 $n \le 10^4$。

对于 $80\%$ 的数据，有 $n \le 10^5$。

对于 $100\%$ 的数据，有 $n \le 5\times 10^6$，$p \le n$，学生初始成绩 $ \le 100$，$z \le 100$。

## 解题思路

### 问题分析

有 $n$ 个学生和 $p$ 次加分操作，每次操作给区间 $[x,y]$ 内的所有学生加 $z$ 分。求所有操作结束后全班的最低分。

**数据规模：** $n$ 高达 $5 \times 10^6$，$p \le n$。这意味着 $O(np)$ 的朴素做法会超时，必须使用差分。

### 核心思路

**一维差分数组**是处理区间加法的利器。设原数组为 $a$，差分数组 $d$ 满足 $d[i] = a[i] - a[i-1]$。对原数组区间 $[x,y]$ 加 $z$ 等价于：
- $d[x] += z$
- $d[y+1] -= z$

所有操作完成后，对 $d$ 做一遍前缀和即可还原最终数组，遍历取最小值。

### 算法流程

1. 读入 $n, p$ 和初始成绩数组 $a$。
2. 初始化差分数组 $d$ 长度为 $n+2$（防止 $y+1$ 越界）。
3. 对于每次操作 $(x,y,z)$：$d[x] += z$，$d[y+1] -= z$。
4. 遍历 $i=1..n$，维护当前累加值 $cur$（初始 $0$），每次 $cur += d[i]$，则第 $i$ 个学生的最终成绩为 $a[i] + cur$。更新最小值。
5. 输出最小值。

### 复杂度分析

- **时间复杂度：** $O(n + p)$，每个操作 $O(1)$，每个学生 $O(1)$。
- **空间复杂度：** $O(n)$，存储成绩和差分数组。

### 要点

- 由于 $n$ 最大 $5 \times 10^6$，使用 `vector` 而不是静态数组，注意 $\texttt{cin}$ 需加速（`ios::sync_with_stdio(false)`）。
- 差分数组的下标从 $1$ 开始，注意 $y = n$ 时 $y+1$ 不越界。
- 初始成绩 $\le 100$，$z \le 100$，使用 `int` 足够，但最终答案可能为负数？成绩是分数，加分后不会为负。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <climits>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, p;
    cin >> n >> p;
    vector<int> a(n + 1, 0);
    for (int i = 1; i <= n; i++) cin >> a[i];
    vector<int> diff(n + 2, 0);
    while (p--) {
        int x, y, z;
        cin >> x >> y >> z;
        diff[x] += z;
        diff[y + 1] -= z;
    }
    int ans = INT_MAX, cur = 0;
    for (int i = 1; i <= n; i++) {
        cur += diff[i];
        ans = min(ans, a[i] + cur);
    }
    cout << ans << "\n";
    return 0;
}
```
