# P10991 [蓝桥杯 2023 国 Python A] 选段排序

## 解题思路

排序一个子数组 [L,R] 后，A[p] 和 A[q] 的值取决于它们在排序后数组中的位置。

考虑几种情况：
1. 只排序 [1,q]：A[q] = max(A[1..q])，A[p] 不变，答案 = max(A[1..q]) - A[p]
2. 只排序 [p,n]：A[p] = min(A[p..n])，A[q] 不变，答案 = A[q] - min(A[p..n])
3. 排序 [p,q]：A[q] = max(A[p..q])，A[p] = min(A[p..q])
4. 排序整个数组：A[q]-A[p] = 全局第q小 - 第p小

取所有情况的最大值。
