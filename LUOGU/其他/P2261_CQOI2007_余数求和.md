# [CQOI2007] 余数求和

## 题目描述

给出正整数 `n` 和 `k`，请计算

$`G(n, k) = \sum_{i = 1}^n k \bmod i``

其中 `k\bmod i` 表示 `k` 除以 `i$ 的余数。

## 输入格式

输入只有一行两个整数，分别表示 `n` 和 `k`。

## 输出格式

输出一行一个整数表示答案。

## 样例

### 样例输入
```
10 5
```

### 样例输出
```
29
```

## 提示

#### 样例 1 解释

`G(10, 5)=0+1+2+1+0+5+5+5+5+5=29`。

#### 数据规模与约定

- 对于 `30\%` 的数据，保证 `n , k \leq 10^3`。
- 对于 `60\%` 的数据，保证 `n, k \leq 10^6`。
- 对于 `100\%` 的数据，保证 `1 \leq n, k \leq 10^9`。

## 解题思路

### 问题分析

求 sum(k mod i, i=1..n) = sum(k - i*floor(k/i)) = n*k - sum(i*floor(k/i))。对 floor(k/i) 用整除分块优化。

### 核心思路

1. 变形：G(n,k) = n*k - sum(i * floor(k/i), i=1..n)
2. **整除分块**：floor(k/i) 的值只有 O(sqrt(k)) 种，对于相同的 floor(k/i) 值，i 的范围是连续的。
3. 对每个块 [l, r]，floor(k/i) 相同，求和 = floor(k/l) * sum(i, i=l..r) = floor(k/l) * (l+r)*(r-l+1)/2。

### 算法流程

1. 读入 n, k
2. 整除分块计算 sum(i*floor(k/i))
3. 输出 n*k - sum

### 复杂度分析

时间 O(sqrt(k))

## 参考代码

```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long n, k;
    cin >> n >> k;
    long long ans = n * k;
    for (long long l = 1, r; l <= n && l <= k; l = r + 1) {
        r = min(k / (k / l), n);
        ans -= (k / l) * (l + r) * (r - l + 1) / 2;
    }
    cout << ans << '\n';
    return 0;
}
```
