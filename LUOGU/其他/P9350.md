# P9350 - JOI 2023 Final Advertisement 2

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

N 个居民在数轴上，影响力 E_i。捐赠给 i 会让距离 ≤ E_i-E_j 的居民 j 自动获得书。求最少捐赠数。

### 核心思路

贪心 + 区间覆盖。将条件 |X_i-X_j| ≤ E_i-E_j 转化为区间约束。从影响力最大的居民开始贪心。

### 复杂度分析

时间复杂度：O(N log N)，空间复杂度：O(N)

### 关键技巧

将不等式转化为 X_j-E_j ≤ X_i-E_i 且 X_j+E_j ≥ X_i+E_i，简化为二维偏序问题。
