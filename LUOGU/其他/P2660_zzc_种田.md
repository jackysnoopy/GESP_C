# zzc 种田

## 题目描述

可能以后 zzc 就去种田了。

田地是一个巨大的矩形，然而 zzc 每次只能种一个正方形,而每种一个正方形时 zzc 所花的体力值是正方形的周长，种过的田不可以再种，zzc 很懒还要节约体力去泡妹子，想花最少的体力值去种完这块田地，问最小体力值。

## 输入格式

两个正整数 $x,y$，表示田地的长和宽。

## 输出格式

输出最小体力值。

## 样例

### 样例输入1
```1 10
```

### 样例输出1
```40
```

### 样例输入2
```2 2
```

### 样例输出2
```8
```

## 提示

$1\le x,y\le 10^{16}$。

## 解题思路

### 问题分析

围栏周长 X，围矩形田地，求最大面积。

### 核心思路

2(a+b)=X，a+b=X/2。面积 a*b 在 a=b=X/4 时最大。取整数分配。

### 算法流程

1. s = X/2
2. a = s/2, b = s-a
3. 输出 a*b

### 复杂度分析

时间 O(1)，空间 O(1)
