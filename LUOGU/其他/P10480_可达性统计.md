# 可达性统计

## 题目描述
给定一张 N 个点 M 条边的有向无环图，分别统计从每个点出发能够到达的点的数量。

## 输入格式
第一行两个整数 N,M，接下来 M 行每行两个整数 x,y，表示从 x 到 y 的一条有向边。

## 输出格式
输出共 N 行，表示每个点能够到达的点的数量。

## 解题思路
### 问题分析
本题要求在 DAG（有向无环图）中，统计从每个节点出发能到达的节点数量。由于图是 DAG，可以利用拓扑排序从后往前递推。

### 核心思路
1. **bitset 优化**：为每个节点维护一个 bitset，表示该节点能到达的所有节点集合。
2. 按拓扑序**逆序**处理节点：对于节点 $u$，其可达集合为 $\{u\} \cup \bigcup_{v \in children(u)} reachable(v)$。
3. 由于是 DAG，按逆拓扑序处理可确保处理 $u$ 时其所有后继节点已经处理完毕。
4. 最后统计每个 bitset 中 1 的个数即为可达点数。

### 算法流程
1. 读入图，构建邻接表
2. 计算拓扑序（Kahn 算法或 DFS）
3. 按拓扑序逆序遍历每个节点，将后继节点的 bitset 取并集
4. 统计每个节点 bitset 中 1 的个数并输出

### 复杂度分析
- 时间复杂度：$O(N \times M / 64)$，bitset 并集操作每次 $O(N/64)$
- 空间复杂度：$O(N^2 / 64)$，存储 $N$ 个 bitset
