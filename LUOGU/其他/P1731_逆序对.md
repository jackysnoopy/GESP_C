# 逆序对

## 题目描述
求排列逆序对数量。

## 输入格式
第一行 n，第二行排列。

## 输出格式
逆序对数量。

## 样例
### 样例 1
```
6
2 6 3 5 1 4
```
```
11
```

## 解题思路
### 问题分析
求一个排列中逆序对的数量。逆序对定义为满足 $i < j$ 且 $a[i] > a[j]$ 的数对 $(i, j)$。

### 核心思路
1. **归并排序法**：在归并排序的合并过程中，当左半部分的元素 $a[i]$ 大于右半部分的元素 $a[j]$ 时，$a[i]$ 及其后面的所有左半部分元素都与 $a[j]$ 构成逆序对，数量为 `mid - i + 1`。
2. **树状数组（Fenwick Tree）法**：从左到右遍历数组，对于每个元素 $a[i]$，查询已处理元素中值大于 $a[i]$ 的个数（即 `sum(max_val) - sum(a[i])`），然后将 $a[i]$ 插入树状数组。
3. 归并排序法实现更简洁，树状数组法更适合动态维护。

### 算法流程
1. 读入排列
2. 归并排序法：分治，递归排序左右半部分，在合并时统计逆序对
3. 或树状数组法：从左到右遍历，查询大于当前值的已处理元素个数并累加
4. 输出逆序对总数

### 复杂度分析
- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$
