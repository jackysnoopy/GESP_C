# P1125 [NOIP 2008 提高组] 笨小猴

## 题目分析
- 统计单词各字母频次，$\text{maxn}$ 为最大频次，$\text{minn}$ 为出现字母中最小频次。若 $\text{maxn}-\text{minn}$ 为质数，输出 `Lucky Word` 与差值；否则 `No Answer` 与 `0`。

## 核心思路
统计频次，在出现过的字母中取 max/min，做差判质（0、1 非质）。

## 算法流程
1. 读单词，统计 26 字母频次。
2. 在频次 $>0$ 的字母中取 max 与 min。
3. $d=\text{maxn}-\text{minn}$，判 $d$ 是否为质数（$d<2$ 非质，否则试除到 $\sqrt d$）。
4. 质数 → `Lucky Word` / $d$；否则 `No Answer` / `0`。

## 复杂度分析
- 时间 $O(|s| + \sqrt{26})$，空间 $O(1)$。

## 样例验证
- `error`：r=3，e/o 各 1，maxn=3,minn=1,$d=2$ 质数 → `Lucky Word` / `2`。
- `olympic`：各字母 1 次，$d=0$ → `No Answer` / `0`。

## 解题思路

### 问题分析
给定一个单词，统计各字母出现频次，在出现过的字母中求最大频次与最小频次之差，判断该差值是否为质数。

### 核心思路
1. 用数组统计 26 个字母的出现次数
2. 遍历频次数组，只考虑出现过的字母（频次 > 0），取最大值和最小值
3. 计算差值 d，判断 d 是否为质数（d < 2 非质，否则试除到 √d）

### 算法流程
1. 读入字符串 s
2. 遍历 s，统计各字母频次到数组 a[26]
3. 遍历 a[0..25]，在非零频次中求 max_char 和 min_char
4. 计算 d = max_char - min_char
5. 调用 isprime(d)：若 d < 2 返回 false，否则试除 2..√d
6. 质数 → 输出 "Lucky Word" 和 d；否则输出 "No Answer" 和 0

### 复杂度分析
- 时间复杂度：O(|s| + √max_freq)，|s| 为字符串长度，max_freq ≤ |s|
- 空间复杂度：O(1)，固定大小数组
