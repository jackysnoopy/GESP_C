# 最长上升子序列

## 题目描述

这是一个简单的动规板子题。

给出一个由 $n(n\le 5000)$ 个不超过 $10^6$ 的正整数组成的序列。请输出这个序列的**最长上升子序列**的长度。

最长上升子序列是指，从原序列中**按顺序**取出一些数字排在一起，这些数字是**逐渐增大**的。

## 输入格式

第一行，一个整数 $n$，表示序列长度。

第二行有 $n$ 个整数，表示这个序列。

## 输出格式

一个整数表示答案。

## 样例

### 样例输入1
```6
1 2 4 1 3 4
```

### 样例输出1
```4
```

## 提示

分别取出 $1$、$2$、$3$、$4$ 即可。

## 解题思路

dp[i] 表示以 a[i] 结尾的最长上升子序列长度。对每个 i，枚举 j < i 且 a[j] < a[i]，dp[i] = max(dp[i], dp[j]+1)。

### 算法流程

双层循环，外层枚举 i，内层枚举 j < i，若 a[j] < a[i] 则转移 dp[i] = max(dp[i], dp[j]+1)。答案为 max(dp)。

### 复杂度分析

时间复杂度：O(n²)

空间复杂度：O(n) 或 O(1)
