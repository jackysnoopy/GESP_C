# 中位数

## 题目描述
给定一个长度为 $N$ 的非负整数序列 $A$，对于前奇数项求中位数。

## 输入格式
第一行一个正整数 $N$。

第二行 $N$ 个正整数 $A_{1\dots N}$。

## 输出格式
共 $\lfloor \frac{N + 1}2\rfloor$ 行，第 $i$ 行为 $A_{1\dots 2i - 1}$ 的中位数。

## 样例
### 样例 1
```
7
1 3 5 7 9 11 6
```
```
1
3
5
6
```

### 样例 2
```
7
3 1 5 9 8 7 6
```
```
3
3
5
6
```

## 提示
对于 $20\%$ 的数据，$N \le 100$；

对于 $40\%$ 的数据，$N \le 3000$；

对于 $100\%$ 的数据，$1 \le N ≤ 10^5$，$0 \le A_i \le 10^9$。

## 解题思路

### 问题分析
给定序列，输出前 1,3,5,... 项的中位数。

### 核心思路
对顶堆：用一个大根堆存较小的一半，小根堆存较大的一半，始终保持两堆大小平衡。

### 算法流程
1. 读入 N 和序列
2. 对每个元素，先加入大根堆，再调整
3. 若大根堆比小根堆多超过 1 个，移动
4. 当处理奇数个元素时，输出大根堆顶

### 复杂度分析
- 时间复杂度：O(N log N)
- 空间复杂度：O(N)

## 参考代码
```cpp
#include <iostream>
#include <queue>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N;
    cin >> N;
    
    priority_queue<int> max_heap;  // 大根堆，放较小的一半
    priority_queue<int, vector<int>, greater<int>> min_heap;  // 小根堆
    
    for (int i = 1; i <= N; i++) {
        int x;
        cin >> x;
        
        if (max_heap.empty() || x <= max_heap.top())
            max_heap.push(x);
        else
            min_heap.push(x);
        
        // 调整平衡
        if (max_heap.size() > min_heap.size() + 1) {
            min_heap.push(max_heap.top());
            max_heap.pop();
        }
        if (min_heap.size() > max_heap.size()) {
            max_heap.push(min_heap.top());
            min_heap.pop();
        }
        
        if (i % 2 == 1)
            cout << max_heap.top() << "\n";
    }
    
    return 0;
}
```
