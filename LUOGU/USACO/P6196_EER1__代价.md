# [EER1] 代价

## 题目描述
给出一个长度为 $n+2$ 的序列 $a_i$，其中第 $1$ 个数和第 $n+2$ 个数固定为 $1$。你每次可以选择序列中间的一个数删除（不能是第一个和最后一个），删除位置 $p$ 上的数的代价为 $a_{p-1} \times a_p \times a_{p+1}$。你需要执行这个操作直到无法操作为止。求最小的代价和。

## 输入格式
第一行一个正整数 $n$。

第二行 $n$ 个正整数，第 $i$ 个数表示 $a_{i+1}$。

## 输出格式
一行一个正整数，表示最小的代价和。

## 样例
### 样例 1
```
3
1 2 3
```
```
9
```

### 样例 2
```
4
19 26 8 17
```
```
846
```

### 样例 3
```
6
1 1 1 1 1 1
```
```
6
```
