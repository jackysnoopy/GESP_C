# P2983 - 按价格排序

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **贪心** 类问题。

### 核心思路

排序后贪心买最便宜的。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    long long B;
    cin >> n >> B;
    vector<pair<long long,long long>> choc(n);
    for (int i = 0; i < n; i++) cin >> choc[i].first >> choc[i].second;
    // Sort by price
    sort(choc.begin(), choc.end());
    long long ans = 0, rem = B;
    for (int i = 0; i < n; i++) {
        long long take = min(choc[i].second, rem / choc[i].first);
        ans += take;
        rem -= take * choc[i].first;
    }
    cout << ans << "\n";
    return 0;
}

```
