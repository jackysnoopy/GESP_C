# P2920 - 按截止时间排序

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **贪心** 类问题。

### 核心思路

按deadline排序后贪心调度。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<pair<int,int>> jobs(n);
    for (int i = 0; i < n; i++) cin >> jobs[i].second >> jobs[i].first;
    sort(jobs.begin(), jobs.end());
    int cur = 0;
    for (int i = 0; i < (int)jobs.size(); i++) {
        cur += jobs[i].first;
        if (cur > jobs[i].second) { cout << -1 << "\n"; return 0; }
    }
    cout << cur << "\n";
    return 0;
}

```
