# P8093 - USACO22JAN Searching for Soulmates S

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

给定两个性格值 a,b，通过 *2、/2（偶数时）、+1 操作将 a 变为 b。求最小操作次数。

### 核心思路

分两种情况：a ≤ b 时直接加 1 或先乘后加；a > b 时先除到某个中间值 c，再从 c 乘/加到 b。对每个可能的中间值 c（通过除 2 得到），计算下降到 c 再上升到 b 的总代价。

### 复杂度分析

时间复杂度：O(log a × log b)，空间复杂度：O(1)

### 关键技巧

关键：a > b 时，最优路径可能先降到低于 b 的值再升回来。使用反向计算（从 b 反推到 c）来高效计算上升代价。
