# P8900 - USACO22DEC Barn Tree S

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

树上有 N 个节点，每个有 h_j 个干草捆。移动干草使每个节点相同数量。求最少命令数和命令序列。

### 核心思路

树形 DP + 贪心。从叶子向上计算每个子树的盈余/亏损。盈余的子树向父节点发送干草，亏损的从父节点接收。

### 复杂度分析

时间复杂度：O(N)，空间复杂度：O(N)

### 关键技巧

后序遍历处理子树，确保处理子节点时子树内部已平衡。每条边最多产生一条命令。
