# P3074 - DAG最长路

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

本题属于 **拓扑排序** 类问题。

### 核心思路

拓扑排序求关键路径长度。

### 时间复杂度

取决于具体算法实现。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m;
    cin >> n >> m;
    vector<int> t(n + 1);
    for (int i = 1; i <= n; i++) cin >> t[i];
    vector<vector<int>> adj(n + 1);
    vector<int> indeg(n + 1, 0);
    for (int i = 0; i < m; i++) {
        int a, b; cin >> a >> b;
        adj[a].push_back(b);
        indeg[b]++;
    }
    // Topological sort with earliest start time
    vector<long long> finish(n + 1, 0);
    queue<int> q;
    for (int i = 1; i <= n; i++) if (indeg[i] == 0) q.push(i);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            finish[v] = max(finish[v], finish[u] + t[u]);
            if (--indeg[v] == 0) q.push(v);
        }
    }
    long long ans = 0;
    for (int i = 1; i <= n; i++) ans = max(ans, finish[i] + t[i]);
    cout << ans << "\n";
    return 0;
}
```
