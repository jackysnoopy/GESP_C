# [USACO02FEB] 三角形牧场

## 题目描述

和所有人一样，奶牛喜欢变化。它们正在设想新造型的牧场。奶牛建筑师 Hei 想建造围有漂亮白色栅栏的三角形牧场。她拥有 $n$ 块木板，每块的长度 $l_i$ 都是整数，她想用所有的木板围成一个三角形使得牧场面积最大。

请帮助 Hei 小姐构造这样的牧场，并计算出这个最大牧场的面积。

## 输入格式

第 $1$ 行：一个整数 $n$；

第 $2$ 到第 $(n + 1)$ 行，每行一个整数，第 $(i + 1)$ 行的整数 $l_i$ 表示第 $i$ 块木板的长度。

## 输出格式

仅一个整数：最大牧场面积乘以 $100$ 然后舍尾的结果。如果无法构建，输出 $-1$。

## 样例

### 样例输入
```
5
1
1
3
3
4
```

### 样例输出
```
692
```

## 提示

#### 样例输入输出 1 解释

$692=	ext{舍尾后的}(100	imes	ext{三角形面积})$，此三角形为等边三角形，边长为 $4$。

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $3\le n\le40$，$1\le l_i\le40$。

## 解题思路

### 问题分析

n 根木板（长度 l_i），选 3 根组成三角形使面积最大。输出面积 ×100 取整。

### 核心思路

**折半枚举**：n≤40，总长度≤1600。将木板分为两半，枚举所有 3^{n/2} 种分配（每根分给边 a/b/c），合并两半后枚举合法三角形。

### 算法流程

1. 前半枚举所有 (sumA, sumB, sumC) 三元组
2. 后半枚举所有分配，合并计算三边长度
3. 验证三角不等式，用海伦公式计算面积
4. 取最大值 ×100 取整

### 复杂度分析

- 时间：O(3^{n/2})
- 空间：O(3^{n/2})

### 关键技巧

- 折半枚举将 3^n 降到 3^{n/2}
- 海伦公式直接计算三角形面积
