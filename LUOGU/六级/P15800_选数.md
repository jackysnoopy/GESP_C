# [GESP202603 六级] 选数

## 题目描述
给定两个包含 $n$ 个整数的数组 $a=[a_1,\dots,a_n]$ 与 $b=[b_1,\dots,b_n]$。你需要指定若干下标 $p_1\lt \cdots\lt p_k$（$1\leq k\leq n$）使得以下条件成立：

- $1\leq p_i\leq n$（$1\leq i\leq k$）；
- $p_{i+1}\geq p_i+b_{p_i}$（$1\leq i< k$）。

你需要在满足以上条件的前提下最大化 $\sum_{i=1}^k a_{p_i}$，也即最大化数组 $a$ 对应下标的整数之和。

## 输入格式
第一行，一个正整数 $n$，表示数组长度。

第二行，$n$ 个正整数 $a_1,a_2,\dots,a_n$，表示数组 $a$。

第三行，$n$ 个正整数 $b_1,b_2,\dots,b_n$，表示数组 $b$。

## 输出格式
一行，一个整数，表示在满足下标条件的前提下，数组 $a$ 对应下标的整数之和的最大值。

## 背景
对应的选择、判断题：<https://ti.luogu.com.cn/problemset/1210>

## 提示
对于 $40\%$ 的测试点，保证 $2\leq n\leq 10^3$。

对于所有测试点，保证 $2\leq n\leq 10^5$，$0\leq a_i\leq 10^9$，$0\leq b_i\leq n$。

## 解题思路

### 问题分析
选取递增下标 p₁ < p₂ < ... < pₖ，满足 p_{i+1} ≥ p_i + b[p_i]。求最多选多少个元素。

### 核心思路
动态规划。dp[i] 表示以 i 结尾时最多选多少个。
转移：dp[i] = 1 + max{dp[j] | j < i 且 j + b[j] ≤ i}

### 算法流程
1. 读入 n 和数组 a, b
2. dp[i] = 1（只选自己）
3. 对每个 i，找满足 j + b[j] ≤ i 的最大 dp[j]
4. 答案是 max(dp[i])

### 复杂度分析
- 时间复杂度：O(n²)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> b[i];
    
    vector<int> dp(n, 1);
    int ans = 1;
    
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < i; j++) {
            if (j + b[j] <= i)
                dp[i] = max(dp[i], dp[j] + 1);
        }
        ans = max(ans, dp[i]);
    }
    
    cout << ans << '\n';
    return 0;
}
```
