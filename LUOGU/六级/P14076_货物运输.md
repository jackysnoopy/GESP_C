# [GESP202509 六级] 货物运输

## 题目描述

A 国有 $n$ 座城市，依次以 $1,2,\ldots,n$ 编号，其中 $1$ 号城市为首都。这 $n$ 座城市由 $n-1$ 条双向道路连接，第 $i$ 条道路（$1 \le i < n$）连接编号为 $u_i,v_i$ 的两座城市，道路长度为 $l_i$。任意两座城市间均可通过双向道路到达。

现在 A 国需要从首都向各个城市运送货物。满载货物的车队会从首都开出，经过一座城市时将对应的货物送出，因此车队需要经过所有城市。A 国希望你设计一条路线，在从首都出发经过所有城市的前提下，最小化经过的道路长度总和。注意一座城市可以经过多次，车队最后可以不返回首都。

## 输入格式

第一行，一个正整数 $n$，表示 A 国的城市数量。

接下来 $n-1$ 行，每行三个正整数 $u_i,v_i,l_i$，表示一条双向道路。

## 输出格式

一行，一个整数，表示经过的道路长度总和的最小值。

## 样例

### 样例输入 1
```
4
1 2 6
1 3 1
3 4 5
```

### 样例输出 1
```
18
```

### 样例输入 2
```
7
1 2 1
2 3 1
3 4 1
7 6 1
6 5 1
5 1 1
```

### 样例输出 2
```
9
```

## 提示

对于 $30\%$ 的测试点，保证 $1 \le n \le 8$。

对于另外 $30\%$ 的测试点，保证仅与一条双向道路连接的城市恰有两座。

对于所有测试点，保证 $1 \le n \le 10^5$，$1 \le u_i,v_i \le n$，$1 \le l_i \le 10^9$。

## 解题思路

### 问题分析

$n$ 座城市构成一棵树（$n-1$ 条边且连通），根为首都 $1$。需要从根出发，**经过所有节点**，可以重复经过节点，**不要求返回根**，求最短路径总长。

### 核心思路

这是一个经典的**树遍历最短路**问题。结论是：

$$
\text{最短路径总长} = 2 \times \text{所有边权之和} - \text{从根到最远节点的距离}
$$

**推导**：由于需要经过每个节点，每条边至少需要走两次（去和回），因此基础代价是 $2 \times \text{sum}$。但由于我们可以停在任意节点而无需返回首都，可以省去最后一段返回路的代价。贪心地，我们选择**从根到最远节点的路径**作为最后一段不走回头路，这样节省最多。

### 算法流程

1. 建图（邻接表，存储 `(邻居, 边权)`）。
2. DFS 从 $1$ 出发，计算到每个节点的距离，记录最大值 $\text{maxDist}$。
3. 计算 $\text{sum} = \sum l_i$。
4. 输出 $2 \times \text{sum} - \text{maxDist}$。

### 复杂度分析

- **时间复杂度**：$O(n)$，DFS 遍历全图。
- **空间复杂度**：$O(n)$，邻接表存储。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<vector<pair<int, long long>>> g(n + 1);
    long long sum = 0;
    for (int i = 1; i < n; i++) {
        int u, v;
        long long w;
        cin >> u >> v >> w;
        g[u].push_back({v, w});
        g[v].push_back({u, w});
        sum += w;
    }
    
    // DFS 找最远距离
    long long maxDist = 0;
    auto dfs = [&](auto&& self, int u, int fa, long long dist) -> void {
        maxDist = max(maxDist, dist);
        for (auto [v, w] : g[u]) {
            if (v == fa) continue;
            self(self, v, u, dist + w);
        }
    };
    dfs(dfs, 1, 0, 0);
    
    cout << 2 * sum - maxDist << "\n";
    return 0;
}
```
