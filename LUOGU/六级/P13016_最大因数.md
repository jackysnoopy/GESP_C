# [GESP202506 六级] 最大因数

## 题目描述

给定一棵有 $10^9$ 个结点的有根树，结点依次以 $1,2,\dots,10^9$ 编号，根结点编号为 $1$。对于编号为 $k$（$2\le k\le 10^9$）的结点，其父结点的编号为 $k$ 的因数中除 $k$ 以外最大的因数。

现有 $q$ 组询问，每组询问给定 $x_i,y_i$，求编号分别为 $x_i,y_i$ 的两个结点在这棵树上的距离（连接两结点的简单路径所包含的边数）。

## 输入格式

第一行一个正整数 $q$，表示询问组数。

接下来 $q$ 行，每行两个正整数 $x_i,y_i$。

## 输出格式

输出共 $q$ 行，每行一个整数表示答案。

## 样例

### 样例 1

```
输入：
3
1 3
2 5
4 8

输出：
1
2
1
```

### 样例 2

```
输入：
1
120 650

输出：
9
```

## 提示

- 对于 $60\%$ 的测试点，$1\le x_i,y_i\le 1000$。
- 对于所有测试点，$1\le q\le 1000$，$1\le x_i,y_i\le 10^9$。

## 解题思路

### 问题分析

题中定义了一棵特殊的树：结点 $k$ 的父结点是 $k$ 的最大真因数（即 $k$ 除以它的最小质因子）。这等价于：从 $k$ 出发，不断除以它的最小质因子，最终到达 $1$（根结点）。因此，树的结构完全由数的质因数分解决定。

### 核心思路

1. **树的深度**：结点 $k$ 的深度等于 $k$ 的质因子总数（含重复）。因为每除以一次最小质因子就走一步向根，而除的次数恰好是质因子的重数之和。

2. **求距离**：在树上，两结点 $x,y$ 的距离为：
   \[
   \text{dist}(x,y) = \text{depth}(x) + \text{depth}(y) - 2 \times \text{depth}(\text{LCA}(x,y))
   \]
   其中 LCA 是 $x$ 和 $y$ 的最近公共祖先。

3. **求 LCA**：由于树是自下而上单向的（子结点到父结点），我们可以先将 $x$ 到根的路径上所有结点记录下来（用哈希集合），然后让 $y$ 向上走，碰到的第一个在 $x$ 路径中的结点就是 LCA。

4. **质因数分解**：$x_i,y_i\le 10^9$，预处理 $\sqrt{10^9}\approx 31623$ 以内的质数，用试除法分解。$q\le 1000$，总时间复杂度可以接受。

### 算法流程

1. 线性筛预处理 $[2,31623]$ 内的所有质数。
2. 定义函数 `get_spf(x)`：用质数表试除，返回 $x$ 的最小质因子。
3. 定义函数 `get_path(x)`：反复令 $x\gets x/\text{spf}(x)$，记录路径上的所有结点，同时计算深度。
4. 对每组询问 $(x,y)$：
   - 获取 $x$ 到根的路径及深度 $\text{dep}_x$。
   - 将 $x$ 的路径结点存入集合。
   - 从 $y$ 开始向上走，记录步数 $\text{dep}_y$，直到遇到集合中的结点（即 LCA）。
   - 输出 $\text{dep}_x + \text{dep}_y - 2 \times \text{dep}_{\text{LCA}}$。

### 复杂度分析

- **时间复杂度**：$O(\sqrt{10^9}/\log\sqrt{10^9} + q \cdot (\log x + \log y))$，质数表大小约 $3400$，每次分解 $O(\log x)$。
- **空间复杂度**：$O(\log x + \log y)$，用于存储路径。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> primes;

void sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) primes.push_back(i);
        for (int p : primes) {
            if (i * p > n) break;
            is_prime[i * p] = false;
            if (i % p == 0) break;
        }
    }
}

int spf(int x) {
    for (int p : primes) {
        if (p * p > x) break;
        if (x % p == 0) return p;
    }
    return x; // x itself is prime
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    sieve(31623);
    
    int q;
    cin >> q;
    while (q--) {
        int x, y;
        cin >> x >> y;
        
        // Get path of x to root
        unordered_set<int> path;
        int dep_x = 0, cur = x;
        while (cur > 1) {
            path.insert(cur);
            cur /= spf(cur);
            dep_x++;
        }
        path.insert(1);
        
        // Climb y to find LCA
        int dep_y = 0, lca = y;
        cur = y;
        while (!path.count(cur)) {
            cur /= spf(cur);
            dep_y++;
        }
        lca = cur;
        
        // Compute depth of LCA
        int dep_lca = 0;
        cur = lca;
        while (cur > 1) {
            cur /= spf(cur);
            dep_lca++;
        }
        
        cout << dep_x + dep_y - 2 * dep_lca << '\n';
    }
    
    return 0;
}
```
