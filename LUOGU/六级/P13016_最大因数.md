# [GESP202506 六级] 最大因数

## 题目描述

给定一棵有 $10^9$ 个结点的有根树，结点依次以 $1,2,\dots,10^9$ 编号，根结点编号为 $1$。对于编号为 $k$（$2\le k\le 10^9$）的结点，其父结点的编号为 $k$ 的因数中除 $k$ 以外最大的因数。

现有 $q$ 组询问，每组询问给定 $x_i,y_i$，求编号分别为 $x_i,y_i$ 的两个结点在这棵树上的距离（连接两结点的简单路径所包含的边数）。

## 输入格式

第一行一个正整数 $q$，表示询问组数。

接下来 $q$ 行，每行两个正整数 $x_i,y_i$。

## 输出格式

输出共 $q$ 行，每行一个整数表示答案。

## 样例

### 样例 1

```
输入：
3
1 3
2 5
4 8

输出：
1
2
1
```

### 样例 2

```
输入：
1
120 650

输出：
9
```

## 提示

- 对于 $60\%$ 的测试点，$1\le x_i,y_i\le 1000$。
- 对于所有测试点，$1\le q\le 1000$，$1\le x_i,y_i\le 10^9$。

## 解题思路

### 问题分析

结点 $k$ 的父结点是 $k$ 的最大真因数，即 $k$ 除以它的最小质因子。从 $k$ 出发，不断除以最小质因子，最终到达根结点 $1$。树的结构完全由数的质因数分解决定。

### 核心思路

1. **树的深度**：结点 $k$ 的深度等于 $k$ 的质因子总数（含重复）。每除以一次最小质因子就向根走一步，除的次数恰好是质因子重数之和。

2. **求距离**：两结点 $x,y$ 的距离为：
   \[
   \text{dist}(x,y) = \text{depth}(x) + \text{depth}(y) - 2 \times \text{depth}(\text{LCA}(x,y))
   \]

3. **求 LCA**：将 $x$ 到根的路径上所有结点存入哈希集合，然后让 $y$ 向上走（每次调用 `getParent()`），碰到的第一个在集合中的结点即为 LCA。

4. **质因数分解**：预处理 $\sqrt{10^9} \approx 31623$ 以内的质数表，用试除法求最小质因子。

### 算法流程

1. 线性筛预处理 $[2, 31623]$ 内的所有质数。
2. 定义 `getMinPrimeFactor(n)` 函数：遍历质数表，返回 $n$ 的最小质因子（若 $n$ 是质数则返回 $n$ 本身）。
3. 定义 `getParent(n)` 函数：返回 $n / \text{getMinPrimeFactor}(n)$，即 $n$ 的父结点。
4. 定义 `getDepth(n)` 函数：从 $n$ 出发不断调用 `getParent()` 直到根结点，返回步数（深度）。
5. 对每组询问 $(x, y)$：
   - 从 $x$ 开始，反复调用 `getParent()`，将路径上的所有结点存入集合 `pathX`，并将根结点 $1$ 加入。
   - 从 $y$ 开始，反复调用 `getParent()`，直到遇到 `pathX` 中的结点，该结点即为 LCA。
   - 计算 `depthX`、`depthY`、`depthLCA`。
   - 输出 `depthX + depthY - 2 * depthLCA`。

### 复杂度分析

- **时间复杂度**：$O(31623 + q \cdot (\log x + \log y))$，线性筛 $O(31623)$，每次查询路径构建 $O(\log x)$、爬升找 LCA $O(\log y)$、三次 `getDepth` 各 $O(\log x)$。
- **空间复杂度**：$O(\log x)$，存储路径集合。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const int MAXP = 31623;
vector<int> primes;
bool isComposite[MAXP + 1];

void initPrimes() {
    for (int i = 2; i <= MAXP; ++i) {
        if (!isComposite[i]) primes.push_back(i);
        for (int p : primes) {
            if (i * p > MAXP) break;
            isComposite[i * p] = true;
            if (i % p == 0) break;
        }
    }
}

int getMinPrimeFactor(int n) {
    for (int p : primes) {
        if (p * p > n) break;
        if (n % p == 0) return p;
    }
    return n;
}

int getParent(int n) {
    if (n == 1) return 0;
    return n / getMinPrimeFactor(n);
}

int getDepth(int n) {
    int depth = 0;
    while (n > 1) {
        n = getParent(n);
        depth++;
    }
    return depth;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    initPrimes();
    
    int q;
    cin >> q;
    while (q--) {
        int x, y;
        cin >> x >> y;
        
        unordered_set<int> pathX;
        int temp = x;
        while (temp > 1) {
            pathX.insert(temp);
            temp = getParent(temp);
        }
        pathX.insert(1);
        
        temp = y;
        int lca = 1;
        while (temp >= 1) {
            if (pathX.count(temp)) {
                lca = temp;
                break;
            }
            if (temp == 1) break;
            temp = getParent(temp);
        }
        
        int depthX = getDepth(x);
        int depthY = getDepth(y);
        int depthLCA = getDepth(lca);
        
        cout << depthX + depthY - 2 * depthLCA << "\n";
    }
    
    return 0;
}
```
