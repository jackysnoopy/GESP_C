# [GESP202606 六级] 满二叉树

## 题目描述

给定一棵包含 $n$ 个结点的有根二叉树，结点依次以 $1, 2, \dots, n$ 编号，根结点编号为 $1$。

对于结点 $i$，其左儿子的编号记为 $l_i$，右儿子编号记为 $r_i$。特别地，如果左儿子不存在则 $l_i = 0$，如果右儿子不存在则 $r_i = 0$。

树中每个结点都对应一棵以其为根的子树。请你求出给定有根树的所有 $n$ 棵子树中，有多少棵子树是满二叉树。

满二叉树是指所有叶子深度均相同，且除叶子外均有两个儿子的二叉树，例如以下三棵二叉树均是满二叉树：

```
()   ()        ()
    /  \      /  \
   ()  ()   ()    ()
           /  \  /  \
          ()  ()()  ()
```

在上面这棵有 $3$ 个节点的二叉树中，有 $3$ 个子树是满二叉树（包括整个树本身，及所有的单个叶子节点）；

又例如，

```
   (1)
   / \
 (2) (3)
     / \
   (4) (5)
```

在上面这棵有 $5$ 个节点的二叉树中，有 $4$ 个子树是满二叉树（包括节点 $3$ 的子树，以及所有单个叶子节点）。

## 输入格式

第一行，一个正整数 $n$，表示有根二叉树结点数量。

接下来 $n$ 行，每行两个非负整数 $l_i, r_i$，表示结点 $i$ 的左儿子编号和右儿子编号，整数之间以空格分隔。

## 输出格式

输出一行，一个整数，表示所有子树中满二叉树的数量。

## 样例

### 样例 1

**输入**
```
4
2 3
4 0
0 0
0 0
```

**输出**
```
2
```

### 样例 2

**输入**
```
3
2 3
0 0
0 0
```

**输出**
```
3
```

## 提示

### 数据范围

对于 $40\%$ 的测试点，保证 $1 \le n \le 500$。

对于所有测试点，保证 $1 \le n \le 10^5$。

## 解题思路

### 问题分析

需要统计二叉树中所有子树里满二叉树的数量。满二叉树的定义：所有叶子深度相同，且除叶子外均有两个儿子。

### 核心思路

使用后序遍历（DFS）自底向上判断每棵子树是否为满二叉树。DFS 返回子树深度（$-1$ 表示不是满二叉树）：

1. **空节点**：返回 $-1$。
2. **叶子节点**：没有左右儿子，是满二叉树，返回深度 $0$。
3. **非叶子节点**：必须同时有左右儿子，且左右子树都是满二叉树（返回值 $\ge 0$），**且左右子树深度相同**，才是满二叉树，返回深度 $d+1$。

关键点：满二叉树要求所有叶子深度相同。仅检查"左右子树都是满二叉树"不够，还需检查左右深度相等。例如左子树深度 $1$、右子树深度 $0$，虽然各自是满二叉树，但合并后叶子深度不同，不满足定义。

### 算法流程

1. 读入 $n$ 和每个节点的左右儿子编号。
2. 从根节点 $1$ 开始 DFS，返回子树深度：
   - 若当前节点为空（$u = 0$），返回 $-1$。
   - 递归获取左右子树深度 `leftDep`、`rightDep`。
   - 若当前节点是叶子（左右儿子均为空），$ans++$，返回 $0$。
   - 若当前节点有左右儿子，且 `leftDep >= 0`、`rightDep >= 0`，且 `leftDep == rightDep`，$ans++$，返回 `leftDep + 1`。
   - 否则返回 $-1$（不是满二叉树）。
3. 输出 $ans$。

### 复杂度分析

- **时间复杂度**：$O(n)$，每个节点恰好访问一次。
- **空间复杂度**：$O(n)$，存储左右儿子数组，递归栈深度最坏 $O(n)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

vector<int> lch, rch;
int ans = 0;

int dfs(int u) {
    if (u == 0) return -1;
    
    int leftDep = dfs(lch[u]);
    int rightDep = dfs(rch[u]);
    
    if (lch[u] == 0 && rch[u] == 0) {
        ans++;
        return 0;
    }
    
    if (lch[u] != 0 && rch[u] != 0 && leftDep >= 0 && rightDep >= 0 && leftDep == rightDep) {
        ans++;
        return leftDep + 1;
    }
    
    return -1;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    lch.resize(n + 1);
    rch.resize(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> lch[i] >> rch[i];
    }

    dfs(1);
    cout << ans << "\n";
    return 0;
}
```
