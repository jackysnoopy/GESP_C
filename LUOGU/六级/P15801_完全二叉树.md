# [GESP202603 六级] 完全二叉树

## 题目描述

给定一棵包含 $n$ 个结点的**有根二叉树**，结点依次以 $1,2,\dots,n$ 编号，根结点编号为 $1$。

对于结点 $i$，其左儿子的编号记为 $l_i$，右儿子编号记为 $r_i$。特别地，如果左儿子不存在则 $l_i=0$，如果右儿子不存在则 $r_i=0$。

树中每个结点都对应一棵以其为根的子树。请你求出给定有根树的所有 $n$ 棵子树中，有多少棵子树是完全二叉树。

## 输入格式

第一行，一个正整数 $n$。

接下来 $n$ 行，每行两个正整数 $l_i,r_i$，表示结点 $i$ 的左儿子编号和右儿子编号。

## 输出格式

输出一行一个整数，表示所有子树中完全二叉树的数量。

## 样例

### 样例输入 1
```
4
2 3
4 0
0 0
0 0
```

### 样例输出 1
```
4
```

### 样例输入 2
```
4
2 3
0 0
4 0
0 0
```

### 样例输出 2
```
3
```

## 提示

对于 $40\%$ 的测试点，$1 \le n \le 500$。

对于所有测试点，$1 \le n \le 10^5$。

## 解题思路

### 问题分析

给定一棵二叉树，判断每个节点的子树是否为完全二叉树（所有层从左到右连续填充，最后一层节点尽量靠左），统计个数。

### 核心思路

树形 DP。对每个节点 $u$ 维护三个信息：

- `h[u]`：子树的高度（从 $u$ 到最远叶子的边数 +1）
- `full[u]`：子树是否为**满二叉树**（所有层完全填满）
- `ok[u]`：子树是否为**完全二叉树**

**叶子节点**（无子节点）：$h=1$，$full=true$，$ok=true$。

**非叶子节点**：设左子 $l$，右子 $r$，`hl = h[l]`（无左子则为 0），`hr = h[r]`（无右子则为 0），`fl = full[l]`（无左子视为 true），`fr = full[r]`（无右子视为 true）。

$$
\begin{aligned}
h[u] &= \max(hl, hr) + 1 \\
full[u] &= fl \land fr \land (hl = hr) \\
ok[u] &= \\
& \quad \text{$l$ 和 $r$ 均为完全二叉树，且} \\
& \quad \big((hl = hr \land fl) \lor (hl = hr + 1 \land fr)\big)
\end{aligned}
$$

解释：
- 若 $hl = hr$ 且左子为满 → 左子全部填满，右子最后一层可能有空缺但靠左→完全
- 若 $hl = hr + 1$ 且右子为满 → 右子全部填满，左子最后一层可空缺→完全
- 同时要求两个子树本身是完全二叉树

DFS 后序遍历即可。

### 算法流程

1. 读入 $n$ 及每个节点的左右子。
2. 从根 DFS 后序遍历。
3. 对每个节点按上述公式计算 $h$, $full$, $ok$。
4. 统计 $ok[u] = true$ 的节点数。

### 复杂度分析

- **时间复杂度**：$O(n)$，每个节点访问一次。
- **空间复杂度**：$O(n)$，存储树和 DP 数组。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<int> l(n + 1), r(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> l[i] >> r[i];
    }
    
    vector<int> h(n + 1, 0);
    vector<bool> full(n + 1, false), ok(n + 1, false);
    int ans = 0;
    
    auto dfs = [&](auto&& self, int u) -> void {
        if (u == 0) return;
        self(self, l[u]);
        self(self, r[u]);
        
        int hl = h[l[u]], hr = h[r[u]];
        bool fl = (l[u] == 0) || full[l[u]];
        bool fr = (r[u] == 0) || full[r[u]];
        bool cl = (l[u] == 0) || ok[l[u]];
        bool cr = (r[u] == 0) || ok[r[u]];
        
        h[u] = max(hl, hr) + 1;
        full[u] = fl && fr && (hl == hr);
        ok[u] = cl && cr && 
                ((hl == hr && fl) || (hl == hr + 1 && fr));
        
        if (ok[u]) ans++;
    };
    dfs(dfs, 1);
    
    cout << ans << "\n";
    return 0;
}
```
