# [GESP202309 六级] 小杨的握手问题

## 题目描述

小杨的班级里共有 $N$ 名同学，学号从 $0$ 至 $N-1$。

某节课上，老师安排全班同学进行一次握手游戏，规则如下：老师安排了一个顺序，让全班 $N$ 名同学依次进入教室。每位同学进入教室时，需要和 **已经在教室内** 且 **学号小于自己** 的同学握手。

小杨想知道，整个班级总共会进行多少次握手。

## 输入格式

输入包含 $2$ 行。第一行一个整数 $N$，表示同学的个数；第二行 $N$ 个整数，依次描述同学们进入教室的顺序，每个整数在 $0 \sim N-1$ 之间，表示该同学的学号。

保证每位同学会且只会进入教室一次。

## 输出格式

输出一行一个整数，表示握手的总次数。

## 样例

### 样例输入 1
```
4
2 1 3 0
```

### 样例输出 1
```
2
```

### 样例输入 2
```
6
0 1 2 3 4 5
```

### 样例输出 2
```
15
```

## 提示

对于 $30\%$ 的测试点，$N \le 100$。

对于所有测试点，$2 \le N \le 3\times 10^5$。

## 解题思路

### 问题分析

同学按给定顺序入场，每个新入场的人与已在教室中且学号更小的人握手。等价于：输入数组 $a[0..N-1]$ 为入场顺序，求

$$
\sum_{i=0}^{N-1} \big|\{j < i \mid a[j] < a[i]\}\big|
$$

即对于每个位置 $i$，统计其左侧比它小的元素个数，再求和。这是经典的**逆序对**计数问题（只是符号方向相反，实际上是求顺序对）。

### 核心思路

用 **树状数组（Fenwick Tree）** 维护已入场同学的学号。按顺序处理：

1. 对当前学号 $x$，查询树状数组中学号 $< x$ 的已入场人数（`query(x-1)`），累加。
2. 将 $x$ 加入树状数组（`add(x, 1)`）。

由于学号范围 $0 \sim N-1$，BIT 大小 $N$ 即可，$O(N \log N)$。$N \le 3\times 10^5$ 十分充裕。

### 算法流程

1. 读入 $N$ 和入场顺序数组 $a$。
2. 初始化大小为 $N$ 的 BIT（下标从 $1$ 开始，学号 $+1$ 映射）。
3. 遍历 $a$：
   - `ans += bit.query(x)` （查询学号 $< x$ 的人数）
   - `bit.add(x+1, 1)`
4. 输出 $ans$。

### 复杂度分析

- **时间复杂度**：$O(N \log N)$。
- **空间复杂度**：$O(N)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

struct BIT {
    int n;
    vector<int> tree;
    BIT(int n_) : n(n_), tree(n_ + 1, 0) {}
    void add(int i, int v) {
        for (; i <= n; i += i & -i) tree[i] += v;
    }
    int sum(int i) {
        int res = 0;
        for (; i > 0; i -= i & -i) res += tree[i];
        return res;
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N;
    cin >> N;
    
    BIT bit(N);
    long long ans = 0;
    
    for (int i = 0; i < N; i++) {
        int x;
        cin >> x;
        // 查询学号小于 x 的已入场人数
        ans += bit.sum(x);
        // 标记学号 x 已入场（BIT 下标从 1 开始）
        bit.add(x + 1, 1);
    }
    
    cout << ans << "\n";
    return 0;
}
```
