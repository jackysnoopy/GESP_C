# [GESP202409 六级] 小杨和整数拆分

## 题目描述

小杨有一个正整数 $n$，小杨想将它拆分成若干完全平方数的和，同时小杨希望拆分的数量越少越好。

编程计算总和为 $n$ 的完全平方数的最小数量。

## 输入格式

输入只有一行一个正整数 $n$。

## 输出格式

输出一行一个整数表示答案。

## 样例

### 样例输入
```
18
```

### 样例输出
```
2
```

## 提示

对于全部数据，$1 \leq n \leq 10^5$。

## 解题思路

### 问题分析

将正整数 $n$ 拆成若干个完全平方数（$1^2, 2^2, 3^2, \ldots$）之和，求所需的最少个数。这是经典的**完全背包**问题（每个平方数可以用无限次）。

### 核心思路

定义 $dp[i]$ 表示凑出 $i$ 所需的最少完全平方数个数。初始化 $dp[0] = 0$，其余为无穷大。

对于每个完全平方数 $j^2$（$1 \le j^2 \le n$），做完全背包转移：

$$
dp[i] = \min(dp[i],\; dp[i - j^2] + 1)
$$

最终输出 $dp[n]$。

由于 $n \le 10^5$，平方数最多 $\sqrt{10^5} \approx 316$ 个，总复杂度 $O(n \sqrt{n}) \approx 3 \times 10^7$，在 C++ 中完全可过。

### 算法流程

1. 读入 $n$。
2. 初始化 $dp$ 数组，$dp[0] = 0$。
3. 枚举平方数 $j^2$（$j$ 从 $1$ 到 $\lfloor \sqrt{n} \rfloor$）：
   - 内层枚举 $i$ 从 $j^2$ 到 $n$：$dp[i] = \min(dp[i], dp[i - j^2] + 1)$。
4. 输出 $dp[n]$。

### 复杂度分析

- **时间复杂度**：$O(n \sqrt{n}) \approx 3 \times 10^7$。
- **空间复杂度**：$O(n)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    const int INF = 1e9;
    vector<int> dp(n + 1, INF);
    dp[0] = 0;
    
    for (int j = 1; j * j <= n; j++) {
        int sq = j * j;
        for (int i = sq; i <= n; i++) {
            dp[i] = min(dp[i], dp[i - sq] + 1);
        }
    }
    
    cout << dp[n] << "\n";
    return 0;
}
```
