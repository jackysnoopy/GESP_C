# [GESP202403 六级] 好斗的牛

## 题目描述

有 $10^9$ 个牛棚一字排开，需要把 $n$ 头牛安置进去。第 $i$ 头牛的攻击范围是 $(a_i,b_i)$，即如果它的左边 $a_i$ 个牛棚或右边 $b_i$ 个牛棚内有其他牛，它就会挑事。

留下**一段连续**的牛棚（卖掉其余牛棚），问最少需要留下多少个牛棚，才能保证存在一种方案把全部 $n$ 头牛都安置进去且没有牛会挑事。

## 输入格式

第一行一个正整数 $n$。

第二行 $n$ 个正整数 $a_1,a_2,\dots,a_n$。

第三行 $n$ 个正整数 $b_1,b_2,\dots,b_n$。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例 1

```
输入：
2
1 2
1 2

输出：
4
```

**解释**：留下第 $1,2,3,4$ 个牛棚，在第 $1$ 和第 $4$ 个位置分别放两头牛。

### 样例 2

```
输入：
3
1 2 3
3 2 1

输出：
7
```

## 提示

- 对 $20\%$ 的数据，$n=2$。
- 另有 $20\%$ 的数据，$n=3$。
- 对 $80\%$ 的数据，$n\le 8$。
- 对于所有数据，$1\le n\le 9$，$1\le a_i,b_i\le 10^3$。

## 解题思路

### 问题分析

$n\le 9$，非常小，可以直接**枚举所有排列**（$9! = 362880$ 种），对每种排列计算所需的最少牛棚数，取最小值。

### 核心思路

对一种排列 $p_1,p_2,\dots,p_n$（表示牛从左到右的顺序），将牛 $p_1$ 放在最左边，则：
- 第 $1$ 头牛占据位置 $1$。
- 第 $i$（$i\ge 2$）头牛必须满足和第 $i-1$ 头牛之间不相干。第 $i-1$ 头牛要求右边 $b_{p_{i-1}}$ 个牛棚内无牛，第 $i$ 头牛要求左边 $a_{p_i}$ 个牛棚内无牛，因此两牛之间的间距至少为：
  \[
  \text{gap} = \max(b_{p_{i-1}},\ a_{p_i}) + 1
  \]
  即第 $i$ 头牛的位置 = 第 $i-1$ 头牛的位置 + $\max(b_{p_{i-1}},a_{p_i}) + 1$。

最终需要的牛棚总数 = 最后一头牛的位置 - 第一头牛的位置 + 1。

对所有排列取最小值即可。

### 算法流程

1. 读入 $n$，数组 $a$ 和 $b$。
2. 初始排列为 $0,1,\dots,n-1$。
3. 使用 `next_permutation` 枚举所有排列。
4. 对每种排列计算所需牛棚数，更新最小值。
5. 输出最小值。

### 复杂度分析

- **时间复杂度**：$O(n!\times n)$，$n\le 9$ 时约 $3.3\times 10^6$ 次操作。
- **空间复杂度**：$O(n)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> b[i];
    
    vector<int> perm(n);
    iota(perm.begin(), perm.end(), 0);
    
    int ans = INT_MAX;
    do {
        int pos = 1;
        for (int i = 1; i < n; i++) {
            int gap = max(b[perm[i-1]], a[perm[i]]) + 1;
            pos += gap;
        }
        ans = min(ans, pos);
    } while (next_permutation(perm.begin(), perm.end()));
    
    cout << ans << '\n';
    
    return 0;
}
```
