# [GESP202406 六级] 计算得分

## 题目描述

小杨想要计算由 $m$ 个小写字母组成的字符串的得分。

小杨设置了一个包含 $n$ 个正整数的计分序列 $A=[a_1,a_2,\dots,a_n]$。如果字符串的一个子串由 $k$（$1\le k\le n$）个 $\texttt{abc}$ 首尾相接组成，那么可以得到分数 $a_k$，且字符串包含的字符不能够重复计算得分。整个字符串的得分是计分子串的总和。

求对于给定的字符串，最大总得分是多少。

## 输入格式

第一行一个正整数 $n$。

第二行 $n$ 个正整数 $A_1,A_2,\dots,A_n$。

第三行一个正整数 $m$。

第四行一个由 $m$ 个小写字母组成的字符串。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例 1

```
输入：
3
3 1 2
13
dabcabcabcabz

输出：
9
```

**解释**：最优方式为 $\texttt{d+abc+abc+abc+abz}$，总得分 $3+3+3=9$。

## 提示

| 子任务 | 占比 | $n$ | $m$ | $a_i$ | 特殊性质 |
| :-: | :-: | :-: | :-: | :-: | :-: |
| $1$ | $20\%$ | $\le 20$ | $\le 10^5$ | $\le 1000$ | $a_i\ge a_{i+1}$ |
| $2$ | $40\%$ | $\le 3$ | $\le 10^5$ | $\le 1000$ | |
| $3$ | $40\%$ | $\le 20$ | $\le 10^5$ | $\le 1000$ | |

对于全部数据，$1\le n\le 20$，$1\le m\le 10^5$，$1\le a_i\le 1000$。

## 解题思路

### 问题分析

字符串中只有连续的 $\texttt{abc}$ 片段才能得分。我们需要在字符串中找到所有连续的 $\texttt{abc}$ 重复段，对每一段独立计算最大得分，然后求和。

### 核心思路

**识别连续 $\texttt{abc}$ 段**：
- 从左到右扫描字符串，检查每个位置 $i$ 能否匹配 $\texttt{abc}$（即 $s[i]=\texttt{'a'}, s[i+1]=\texttt{'b'}, s[i+2]=\texttt{'c'}$）。
- 如果找到 $\texttt{abc}$，则记录其起始位置，并跳到 $i+3$ 继续查找下一个。
- 若下一个 $\texttt{abc}$ 恰好在上一个 $\texttt{abc}$ 结尾处无缝衔接（起始位置相差 $3$），则它们属于同一个连续段。

**对每个连续段的 DP**：
设一段包含 $L$ 个连续的 $\texttt{abc}$。将其拆分为若干组，每组 $k$ 个 $\texttt{abc}$ 得分 $a_k$（$1\le k\le n$）。这是一个完全背包式的 DP：

- 定义 $f[i]$ 为前 $i$ 个 $\texttt{abc}$ 的最大得分。
- $f[0]=0$，$f[i]=\max_{1\le k\le \min(n,i)} (f[i-k] + a_k)$。
- 该段最大得分为 $f[L]$。

**最终答案**：对所有段得分求和。

### 算法流程

1. 读入 $n$、$A$ 数组、$m$ 和字符串 $s$。
2. 扫描字符串，找出所有 $\texttt{abc}$ 起始位置，按连续关系分组。
3. 对每个连续段长度 $L$，运行 DP 计算最大得分。
4. 输出总分。

### 复杂度分析

- **时间复杂度**：$O(m + L_{\max}\times n)$，$L_{\max}\le m/3$，$n\le 20$。
- **空间复杂度**：$O(m)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];
    
    int m;
    string s;
    cin >> m >> s;
    
    // Find all "abc" start positions
    vector<int> pos;
    for (int i = 0; i + 2 < m; i++) {
        if (s[i] == 'a' && s[i + 1] == 'b' && s[i + 2] == 'c') {
            pos.push_back(i);
            i += 2; // skip "bc"
        }
    }
    
    long long ans = 0;
    int sz = (int)pos.size();
    for (int i = 0; i < sz; ) {
        int j = i;
        while (j + 1 < sz && pos[j + 1] == pos[j] + 3) j++;
        int len = j - i + 1; // number of consecutive "abc"s
        
        // DP for this segment
        vector<long long> f(len + 1, 0);
        for (int k = 1; k <= len; k++) {
            for (int t = 1; t <= min(n, k); t++) {
                f[k] = max(f[k], f[k - t] + a[t]);
            }
        }
        ans += f[len];
        i = j + 1;
    }
    
    cout << ans << '\n';
    
    return 0;
}
```
