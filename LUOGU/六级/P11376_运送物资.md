# [GESP202412 六级] 运送物资

## 题目描述

小杨管理着 $m$ 辆货车，每辆货车每天需要向 A 市和 B 市运送若干次物资。小杨同时拥有 $n$ 个运输站点，这些站点位于 A 市和 B 市之间。

每次运送物资时，货车从初始运输站点出发，前往 A 市或 B 市，之后返回初始运输站点。A 市、B 市和运输站点的位置可以视作数轴上的三个点，其中 A 市的坐标为 $0$，B 市的坐标为 $x$，运输站点的坐标为 $p$ 且有 $0 \lt p \lt x$。货车每次去 A 市运送物资的总行驶路程为 $2p$，去 B 市运送物资的总行驶路程为 $2(x - p)$。

对于第 $i$ 个运输站点，其位置为 $p_i$ 且至多作为 $c_i$ 辆车的初始运输站点。小杨想知道，在最优分配每辆货车的初始运输站点的情况下，所有货车每天的最短总行驶路程是多少。

## 输入格式

第一行包含三个正整数 $n,m,x$，代表运输站点数量、货车数量和两市距离。

之后 $n$ 行，每行包含两个正整数 $p_i$ 和 $c_i$，代表第 $i$ 个运输站点的位置和最多容纳车辆数。

之后 $m$ 行，每行包含两个正整数 $a_i$ 和 $b_i$，代表第 $i$ 辆货车每天需要向 A 市运送 $a_i$ 次物资，向 B 市运送 $b_i$ 次物资。

## 输出格式

输出一个正整数，代表所有货车每天的最短总行驶路程。

## 样例

### 样例输入
```
3 4 10
1 1
2 1
8 3
5 3
7 2
9 0
1 10000
```

### 样例输出
```
40186
```

## 提示

第 $1$ 辆车的初始运输站点为站点 $3$，第 $2$ 辆车的初始运输站点为站点 $2$。第 $3$ 辆车的初始运输站点为站点 $1$，第 $4$ 辆车的初始运输站点为站点 $3$。此时总驶路程最短，为 $40186$。

| 子任务编号 | 数据点占比 | $n$ | $x$ | $c_i$ |
| :--------: | :--------: | :-: | :-: | :---: |
| $1$ | $20\%$ | $2$ | $2$ | $1$ |
| $2$ | $20\%$ | $\leq 10^5$ | $\leq 10^5$ | $1$ |
| $3$ | $60\%$ | $\leq 10^5$ | $\leq 10^5$ | $\leq 10^5$ |

对于全部数据，保证 $1\leq n,m\leq 10^5$，$2\leq x\leq 10^8$，$0\lt p_i\lt x$，$1\leq c_i\leq 10^5$，$0\leq a_i,b_i\leq 10^5$。数据保证 $\sum c_i\geq m$。

## 解题思路

### 问题分析

每辆货车 $i$ 若分配到站点 $j$（坐标 $p_j$），其日行驶路程为 $2a_i p_j + 2b_i(x - p_j)$。

关键观察：货车去 A 市的次数 $a_i$ 和去 B 市的次数 $b_i$ 决定了它应该分配到哪个位置的站点：
- 若 $a_i > b_i$（去 A 市更多），应分配到靠近 A 市的小 $p$ 站点，减少 $2p$ 的路程。
- 若 $a_i < b_i$（去 B 市更多），应分配到靠近 B 市的大 $p$ 站点，减少 $2(x-p)$ 的路程。
- 若 $a_i = b_i$，分配到任意站点路程相同，均为 $2a_i x$。

### 核心思路

将货车按差值 $d_i = a_i - b_i$ 分为三组：正差值（去 A 市多）、负差值（去 B 市多）、零差值。采用贪心策略：

1. **正差值货车**：按差值从大到小排序，优先分配到最小 $p$ 的站点（靠近 A 市）。
2. **负差值货车**：按差值从小到大排序（即 $|d_i|$ 从大到小），优先分配到最大 $p$ 的站点（靠近 B 市）。
3. **零差值货车**：路程固定为 $2a_i x$，无需特殊处理。

站点按 $p$ 升序排列，使用双指针 $l$（最小 $p$）和 $r$（最大 $p$）分别处理正、负差值货车，同时跟踪每个站点的剩余容量。

### 算法流程

1. 读入 $n, m, x$。
2. 读入所有站点 $(p_i, c_i)$，按 $p_i$ 升序排序。
3. 读入所有货车 $(a_i, b_i)$，按差值 $d_i = a_i - b_i$ 分为三组：
   - `pos`：$d_i > 0$，按 $d_i$ 降序排序。
   - `neg`：$d_i < 0$，按 $d_i$ 升序排序（最负的优先）。
   - `zero`：$d_i = 0$。
4. 初始化双指针 $l = 0$（指向最小 $p$），$r = n-1$（指向最大 $p$）。
5. 分配正差值货车：遍历 `pos`，从站点 $l$ 开始分配，每分配一辆 $c_l--$，若 $c_l = 0$ 则 $l++$。
6. 分配负差值货车：遍历 `neg`，从站点 $r$ 开始分配，每分配一辆 $c_r--$，若 $c_r = 0$ 则 $r--$。
7. 零差值货车直接累加 $2a_i x$ 到答案。
8. 输出总路程。

### 复杂度分析

- **时间复杂度**：$O((n+m) \log (n+m))$，排序为主。
- **空间复杂度**：$O(n+m)$，存储站点和货车信息。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

struct Station {
    ll p, c;
};

struct Truck {
    ll a, b;
    ll diff;
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n, m;
    ll x;
    cin >> n >> m >> x;

    vector<Station> stations(n);
    for (int i = 0; i < n; ++i) {
        cin >> stations[i].p >> stations[i].c;
    }

    vector<Truck> pos, neg, zero;
    for (int i = 0; i < m; ++i) {
        ll a, b;
        cin >> a >> b;
        if (a > b) pos.push_back({a, b, a - b});
        else if (a < b) neg.push_back({a, b, a - b});
        else zero.push_back({a, b, 0});
    }

    sort(pos.begin(), pos.end(), [](const Truck& t1, const Truck& t2) {
        return t1.diff > t2.diff;
    });
    
    sort(neg.begin(), neg.end(), [](const Truck& t1, const Truck& t2) {
        return t1.diff < t2.diff;
    });

    sort(stations.begin(), stations.end(), [](const Station& s1, const Station& s2) {
        return s1.p < s2.p;
    });

    ll ans = 0;
    int l = 0, r = n - 1;
    
    int pi = 0;
    while (pi < pos.size()) {
        while (l <= r && stations[l].c == 0) l++;
        if (l > r) break;
        ll cost = 2 * pos[pi].a * stations[l].p + 
                  2 * pos[pi].b * (x - stations[l].p);
        ans += cost;
        stations[l].c--;
        pi++;
    }

    int ni = 0;
    while (ni < neg.size()) {
        while (l <= r && stations[r].c == 0) r--;
        if (l > r) break;
        ll cost = 2 * neg[ni].a * stations[r].p + 
                  2 * neg[ni].b * (x - stations[r].p);
        ans += cost;
        stations[r].c--;
        ni++;
    }

    for (auto& t : zero) {
        ans += 2 * t.a * x;
    }

    cout << ans << "\n";
    return 0;
}
```
