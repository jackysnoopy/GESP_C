# [GESP样题 六级] 亲朋数

## 题目描述

给定一串长度为 $L$、由数字 $0\sim 9$ 组成的数字串 $S$。容易知道，它的连续子串共有 $\frac{L(L + 1)}2$ 个。如果某个子串对应的数（允许有前导零）是 $p$ 的倍数，则称该子串为数字串 $S$ 对于 $p$ 的亲朋数。

例如，数字串 $S$ 为"$12342$"、$p$ 为 $2$，则在 $15$ 个连续子串中，亲朋数有"$12$"、"$1234$"、"$12342$"、"$2$"、"$234$"、"$2342$"、"$34$"、"$342$"、"$4$"、"$42$"、"$2$"共 $11$ 个。注意其中"$2$"出现了 $2$ 次，但由于其在 $S$ 中的位置不同，记为不同的亲朋数。

现在，告诉你数字串 $S$ 和正整数 $p$，你能计算出有多少个亲朋数吗？

## 输入格式

输入的第一行，包含一个正整数 $p$。约定 $2 \leq p \leq 128$。

输入的第二行，包含一个长为 $L$ 的数字串 $S$。约定 $1 \leq L \leq 10^6$。

## 输出格式

输出一行一个整数表示答案。

## 样例

### 样例输入 1
```
2
102
```

### 样例输出 1
```
5
```

### 样例输入 2
```
2
12342
```

### 样例输出 2
```
11
```

## 提示

**样例 1 解释**

$5$ 个亲朋数，分别 $10$、$102$、$0$、$02$、$2$。

## 解题思路

### 问题分析

统计数字串 $S$ 的所有子串中，数值（允许前导 $0$）能被 $p$ 整除的个数。$L$ 可达 $10^6$，不能枚举所有子串。

### 核心思路

动态规划优化。从左到右扫描字符串，维护以当前位置为结尾的所有子串的模 $p$ 余数计数。

设 $cnt[r]$ 表示以当前位置为结尾的子串中，数值 $\bmod p = r$ 的个数。当扫描到新字符 $d$（数字值）时：
- 之前所有以 $i-1$ 结尾的子串 $t$ 变成 $10 \times t + d$，余数从 $r$ 变为 $(10r + d) \bmod p$。
- 新子串仅包含 $d$ 本身，余数为 $d \bmod p$。

因此转移为：
$$new\_cnt[(10r + d) \bmod p] += cnt[r] \quad (\forall r)$$
$$new\_cnt[d \bmod p] += 1$$

每次扫描到新字符时，$new\_cnt[0]$ 的累加即为答案。

由于 $p \le 128$，每步的 $p$ 轮循环总复杂度 $O(Lp) \approx 1.28 \times 10^8$，C++ 可过。

### 算法流程

1. 读入 $p$ 和字符串 $S$。
2. 初始化长度为 $p$ 的计数数组 $cnt$ 全为 $0$。
3. 遍历每个字符 $d$：
   - 创建新数组 $new\_cnt$，初始为 $0$。
   - $new\_cnt[d \bmod p] += 1$。
   - 对每个余数 $r$，$new\_cnt[(10r + d) \bmod p] += cnt[r]$。
   - 累加 $new\_cnt[0]$ 到答案。
   - 交换 $cnt$ 和 $new\_cnt$。
4. 输出答案。

### 复杂度分析

- **时间复杂度**：$O(L \cdot p)$。
- **空间复杂度**：$O(p)$。

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int p;
    string s;
    cin >> p >> s;

    vector<long long> cnt(p, 0);
    long long ans = 0;

    for (char ch : s) {
        int d = ch - '0';
        vector<long long> new_cnt(p, 0);
        new_cnt[d % p]++;

        for (int r = 0; r < p; r++) {
            if (cnt[r] == 0) continue;
            int nr = (r * 10 + d) % p;
            new_cnt[nr] += cnt[r];
        }

        ans += new_cnt[0];
        cnt.swap(new_cnt);
    }

    cout << ans << "\n";
    return 0;
}
```
