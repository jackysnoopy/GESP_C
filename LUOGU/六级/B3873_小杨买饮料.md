# [GESP202309 六级] 小杨买饮料

## 题目描述

小杨来到了一家商店，打算购买一些饮料。这家商店总共出售 $N$ 种饮料，编号从 $0$ 至 $N-1$，其中编号为 $i$ 的饮料售价 $c_i$ 元，容量 $l_i$ 毫升。

小杨的需求如下：

1. 每种饮料至多购买 $1$ 瓶；
2. 购买总容量不低于 $L$；
3. 在满足 1 和 2 的前提下，花费尽可能少。

如果无法满足，输出 `no solution`。

## 输入格式

第一行两个整数 $N,L$。

接下来 $N$ 行，每行两个整数 $c_i,l_i$。

## 输出格式

一行一个整数表示最少花费，或 `no solution`。

## 样例

### 样例输入 1
```
5 100
100 2000
2 50
4 40
5 30
3 20
```

### 样例输出 1
```
9
```

### 样例输入 2
```
5 141
100 2000
2 50
4 40
5 30
3 20
```

### 样例输出 2
```
100
```

### 样例输入 3
```
4 141
2 50
4 40
5 30
3 20
```

### 样例输出 3
```
no solution
```

## 提示

对于 $40\%$ 的测试点：$N \le 20$，$1 \le L \le 100$，$l_i \le 100$。

对于 $70\%$ 的测试点：$l_i \le 100$。

对于 $100\%$ 的测试点：$1 \le N \le 500$，$1 \le L \le 2000$，$1 \le c_i,l_i \le 10^6$。

## 解题思路

### 问题分析

$N$ 种饮料，每种最多一瓶，是典型的 **0/1 背包** 问题。目标是**容量 ≥ L** 的最小花费。$L \le 2000$ 是瓶颈，$l_i$ 可以很大（$\le 10^6$），但我们可以将超出 $L$ 的容量视为 $L$。

### 核心思路

定义 `dp[v]` 为达到**至少** $v$ 毫升所需的最小花费，其中 $v$ 上限到 $L$（超过 $L$ 的视作 $L$）。

状态转移：对于每种饮料 $(c, l)$，倒序枚举当前容量 $v$（$0 \le v < L$），则

$$
\text{new\_v} = \min(L,\; v + l)
$$
$$
dp[\text{new\_v}] = \min(dp[\text{new\_v}],\; dp[v] + c)
$$

初始化 $dp[0] = 0$，其余为无穷大。

最终若 $dp[L]$ 为无穷大则输出 `no solution`，否则输出 $dp[L]$。

### 算法流程

1. 读入 $N, L$ 及每种饮料的 $c_i, l_i$。
2. 初始化 $dp$ 数组大小为 $L+1$，$dp[0] = 0$。
3. 对每种饮料，倒序 $v$ 从 $L-1$ 到 $0$ 更新 $dp$。
4. 输出结果。

### 复杂度分析

- **时间复杂度**：$O(N \times L) = 500 \times 2000 = 10^6$，轻松通过。
- **空间复杂度**：$O(L)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, L;
    cin >> n >> L;
    
    const long long INF = 1e18;
    vector<long long> dp(L + 1, INF);
    dp[0] = 0;
    
    for (int i = 0; i < n; i++) {
        int c, l;
        cin >> c >> l;
        for (int v = L - 1; v >= 0; v--) {
            if (dp[v] == INF) continue;
            int nv = min(L, v + l);
            dp[nv] = min(dp[nv], dp[v] + c);
        }
    }
    
    if (dp[L] == INF) {
        cout << "no solution\n";
    } else {
        cout << dp[L] << "\n";
    }
    
    return 0;
}
```
