# [GESP202406 六级] 二叉树

## 题目描述

小杨有一棵包含 $n$ 个节点的二叉树，根节点编号为 $1$。每个节点要么是白色（`0`），要么是黑色（`1`）。

之后小杨会进行 $q$ 次操作，每次选择一个节点 $a_i$，将以该节点为根的**子树内所有节点**的颜色反转（`0` 变 `1`，`1` 变 `0`）。

小杨想知道 $q$ 次操作全部完成后每个节点的颜色。

## 输入格式

第一行一个正整数 $n$。

第二行 $(n-1)$ 个正整数，第 $i$（$1\le i\le n-1$）个数表示编号为 $(i+1)$ 的节点的父亲节点编号，数据保证是一棵二叉树。

第三行一个长度为 $n$ 的 $\texttt{01}$ 串，表示初始颜色。

第四行一个正整数 $q$。

接下来 $q$ 行每行一个正整数 $a_i$，表示操作选择的节点编号。

## 输出格式

输出一行一个长度为 $n$ 的 $\texttt{01}$ 串，表示最终每个节点的颜色。

## 样例

### 样例输入
```
6
3 1 1 3 4
100101
3
1
3
2
```

### 样例输出
```
010000
```

## 提示

对于 $20\%$ 的数据：$n, q \le 10^5$，且树为链（节点 $i$ 的父亲为 $i-1$）。

对于 $40\%$ 的数据：$n, q \le 1000$。

对于 $100\%$ 的数据：$n, q \le 10^5$。

## 解题思路

### 问题分析

对树上的多操作进行子树颜色反转。$n, q \le 10^5$，不能每次操作暴力遍历子树。

### 核心思路

利用 **DFS 序** 将子树区间映射到数组连续区间上，再用**差分数组**统计每个节点被反转的次数。

**DFS 序**：从根开始深度优先遍历，每个节点 $u$ 记录进入时间戳 `in[u]` 和退出时间戳 `out[u]`。子树 $u$ 中的所有节点正好对应区间 `[in[u], out[u]]`。

**差分数组**：对于每次操作 $a_i$，在 `diff[in[a_i]]` 处 +1，`diff[out[a_i]+1]` 处 -1。最后求前缀和得到每个时间戳对应的反转次数。若反转次数为奇数，颜色翻转；否则不变。

用一个 `pos` 数组建立时间戳到节点编号的映射，方便最后输出。

### 算法流程

1. 读入 $n$，根据父节点信息构建邻接表。
2. DFS 计算 `in[u]`、`out[u]` 和 `pos[t]`（时间戳 $t$ 对应的节点）。
3. 读入初始颜色字符串。
4. 读入 $q$ 及操作序列，更新差分数组。
5. 计算差分前缀和，结合初始颜色输出最终颜色。

### 复杂度分析

- **时间复杂度**：$O(n + q)$，DFS 与差分均线性。
- **空间复杂度**：$O(n)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<vector<int>> children(n + 1);
    for (int v = 2; v <= n; v++) {
        int p;
        cin >> p;
        children[p].push_back(v);
    }
    
    string color_str;
    cin >> color_str;
    vector<int> color(n + 1);
    for (int i = 1; i <= n; i++) {
        color[i] = color_str[i - 1] - '0';
    }
    
    // DFS 序
    vector<int> in(n + 1), out(n + 1), pos(n + 1);
    int timer = 0;
    auto dfs = [&](auto&& self, int u) -> void {
        in[u] = ++timer;
        pos[timer] = u;
        for (int v : children[u]) {
            self(self, v);
        }
        out[u] = timer;
    };
    dfs(dfs, 1);
    
    // 差分数组
    vector<int> diff(n + 5, 0);
    int q;
    cin >> q;
    while (q--) {
        int a;
        cin >> a;
        diff[in[a]]++;
        diff[out[a] + 1]--;
    }
    
    // 前缀和 + 输出
    int cur = 0;
    string ans(n, '0');
    for (int t = 1; t <= n; t++) {
        cur += diff[t];
        int u = pos[t];
        int fc = color[u] ^ (cur & 1);
        ans[u - 1] = '0' + fc;
    }
    
    cout << ans << "\n";
    return 0;
}
```
