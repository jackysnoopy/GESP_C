# [GESP202512 六级] 路径覆盖

## 题目描述

给定一棵有 $n$ 个结点的有根树 $T$，根结点编号为 $1$。初始时所有结点均为白色。你需要将若干结点染为黑色，使得所有叶子到根的路径上至少有一个黑色结点。将结点 $i$ 染为黑色需要代价 $c_i$，求最小化染色代价之和。

叶子是指没有子结点的结点。

## 输入格式

第一行一个正整数 $n$。

第二行 $n-1$ 个正整数 $f_2,f_3,\dots,f_n$，其中 $f_i$ 表示结点 $i$ 的父结点，保证 $f_i<i$。

第三行 $n$ 个正整数 $c_1,c_2,\dots,c_n$。

## 输出格式

一行一个整数，表示最小染色代价之和。

## 样例

### 样例 1

```
输入：
4
1 2 3
5 6 2 3

输出：
2
```

### 样例 2

```
输入：
7
1 1 2 2 3 3
64 16 15 4 3 2 1

输出：
10
```

## 提示

- 对于 $40\%$ 的测试点，$2\le n\le 16$。
- 对于另外 $20\%$ 的测试点，$f_i=i-1$。
- 对于所有测试点，$2\le n\le 10^5$，$1\le c_i\le 10^9$。

## 解题思路

### 问题分析

"所有叶子到根的路径上至少有一个黑色结点"等价于：对于每个叶子，从它往上走直到根，至少要遇到一个黑色结点。这是一个经典的树形 DP 问题，本质上是在树上选择一组结点"覆盖"所有叶子到根的路径。

### 核心思路

定义两个 DP 状态：

- $f(u)$：考虑 $u$ 的子树，且 **$u$ 的祖先中没有黑色结点**（即覆盖必须在 $u$ 的子树内完成）时的最小代价。
- $g(u)$：考虑 $u$ 的子树，且 **$u$ 已被某个祖先的黑色结点覆盖**（无需在 $u$ 的子树内再选黑色结点）时的最小代价。此时子树内可以不选任何结点，代价为 $0$。

转移：

- **叶子结点**：
  - $f(u)=c_u$（必须染自己）。
  - $g(u)=0$（已被覆盖，什么都不做）。
- **非叶子结点**：
  - $g(u)=0$（已被覆盖，子树内无需再染）。
  - $f(u)=\min\big(c_u,\ \sum_{v\in \text{son}(u)} f(v)\big)$：
    - 选择一：染 $u$，代价 $c_u$，此时子树全部被覆盖，后代结点不需要再付出代价（$g$ 值全 $0$）。
    - 选择二：不染 $u$，则需要在每个子树的内部完成覆盖，即 $\sum f(v)$。

最终答案为 $f(1)$（根没有祖先，覆盖必须从根或子树内开始）。

### 算法流程

1. 根据 $f_i$ 建树（邻接表）。
2. DFS 后序遍历，先处理子结点，再计算 $f(u)$。
3. 输出 $f(1)$。

### 复杂度分析

- **时间复杂度**：$O(n)$，每个结点遍历一次。
- **空间复杂度**：$O(n)$，邻接表存储树。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<vector<int>> g(n + 1);
    for (int i = 2; i <= n; i++) {
        int f;
        cin >> f;
        g[f].push_back(i);
    }
    
    vector<int> c(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> c[i];
    }
    
    vector<ll> f(n + 1, 0);
    
    function<void(int)> dfs = [&](int u) {
        if (g[u].empty()) {
            f[u] = c[u];
            return;
        }
        ll sum = 0;
        for (int v : g[u]) {
            dfs(v);
            sum += f[v];
        }
        f[u] = min((ll)c[u], sum);
    };
    
    dfs(1);
    cout << f[1] << '\n';
    
    return 0;
}
```
