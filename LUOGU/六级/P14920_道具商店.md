# [GESP202512 六级] 道具商店

## 题目描述

道具商店里有 $n$ 件道具可供挑选。第 $i$ 件道具可为玩家提升 $a_i$ 点攻击力，需要 $c_i$ 枚金币才能购买，每件道具只能购买一次。现在有 $k$ 枚金币，问最多可以提升多少点攻击力。

## 输入格式

第一行两个正整数 $n,k$。

接下来 $n$ 行，每行两个正整数 $a_i,c_i$。

## 输出格式

输出一行一个整数，表示最多可以提升的攻击力点数。

## 样例

### 样例 1

```
输入：
3 5
99 1
33 2
11 3

输出：
132
```

### 样例 2

```
输入：
4 100
10 1
20 11
40 33
100 99

输出：
110
```

## 提示

- 对于 $60\%$ 的测试点，$1\le k\le 500$，$1\le c_i\le 500$。
- 对于所有测试点，$1\le n\le 500$，$1\le k\le 10^9$，$1\le a_i\le 500$，$1\le c_i\le 10^9$。

## 解题思路

### 问题分析

$n$ 件物品，每件只能选一次（0/1 背包），在金币上限 $k$ 内最大化总攻击力。

$k$ 最大可达 $10^9$，无法用容量做 DP。但 $a_i\le 500$，$n\le 500$，**攻击力总和**上界为 $500\times 500=250000$，可以用攻击力做 DP 维度。

### 核心思路

**反向 DP**：定义 $dp[j]$ 为获得**恰好** $j$ 点攻击力所需的最小金币数。转移：

$$
dp[j] = \min(dp[j],\ dp[j-a_i] + c_i)
$$

初始 $dp[0]=0$，其余为 $\infty$（用一个大数如 `1e18` 表示不可达）。

最终遍历 $j$ 从大到小，找到第一个 $dp[j]\le k$ 的 $j$ 即为答案。

### 算法流程

1. 读入 $n,k$ 及所有 $(a_i,c_i)$。
2. 计算攻击力上界 $M = \sum a_i$（若超过 $250000$ 可截断为 $250000$）。
3. 初始化 $dp$ 数组，$dp[0]=0$。
4. 对每件道具，倒序更新 $dp$ 数组。
5. 从 $M$ 向下扫描，输出第一个 $dp[j]\le k$ 的 $j$。

### 复杂度分析

- **时间复杂度**：$O(n\times M)$，$M\le 250000$，约 $1.25\times 10^8$ 次操作，C++ 可以接受。
- **空间复杂度**：$O(M)$。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    ll k;
    cin >> n >> k;
    
    vector<int> a(n);
    vector<ll> c(n);
    int sum_a = 0;
    for (int i = 0; i < n; i++) {
        cin >> a[i] >> c[i];
        sum_a += a[i];
    }
    
    const ll INF = 1e18;
    vector<ll> dp(sum_a + 1, INF);
    dp[0] = 0;
    
    for (int i = 0; i < n; i++) {
        for (int j = sum_a; j >= a[i]; j--) {
            if (dp[j - a[i]] != INF) {
                dp[j] = min(dp[j], dp[j - a[i]] + c[i]);
            }
        }
    }
    
    for (int j = sum_a; j >= 0; j--) {
        if (dp[j] <= k) {
            cout << j << '\n';
            break;
        }
    }
    
    return 0;
}
```
