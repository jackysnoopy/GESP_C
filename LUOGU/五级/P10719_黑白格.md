# [GESP202406 五级] 黑白格

## 题目描述

小杨有一个 $n$ 行 $m$ 列的网格图，其中每个格子要么是白色，要么是黑色。

小杨想知道至少包含 $k$ 个黑色格子的最小子矩形包含了多少个格子。

## 输入格式

第一行包含三个正整数 $n,m,k$，含义如题面所示。

之后 $n$ 行，每行⼀个长度为 $m$ 的 $\texttt{01}$ 串，代表网格图第 $i$ 行格子的颜色，如果为 $\texttt{0}$，则对应格子为白色，否则为黑色。

## 输出格式

输出一个整数，代表至少包含 $k$ 个黑色格子的最小子矩形包含格子的数量，如果不存在则输出 $0$。

## 样例

### 样例 #1

**输入**
```
4 5 5
00000
01111
00011
00011
```

**输出**
```
6
```

## 提示

#### 样例解释

对于样例 $1$，假设 $(i,j)$ 代表第 $i$ 行第 $j$ 列，至少包含 $5$ 个黑色格子的最小子矩形的四个顶点为 $(2,4)$，$(2,5)$，$(4,4)$，$(4,5)$，共包含 $6$ 个格子。

#### 数据范围

对于全部数据，保证有 $1\le n,m\le 100$，$1\le k\le n\times m$。

| 子任务编号 | 得分 | $n,m$ |
| :--: | :--: | :--: |
| $1$ | $20$ | $\le 10$  |
| $2$ | $40$ | $n=1$，$1\le m\le 100$ |
| $3$ | $40$ | $\le 100$|

## 解题思路

### 问题分析

在一个 $n \times m$ 的网格中，求至少包含 $k$ 个 $\texttt{1}$ 的最小连续子矩形的面积。如果不存在则输出 $0$。

### 核心思路

**二维前缀和**可以 $O(1)$ 查询任意子矩形内 $\texttt{1}$ 的个数。然后**枚举所有可能子矩形**，检查是否 $\ge k$，取最小面积。

### 算法流程

1. 构建二维前缀和数组 $pref$。
2. 定义查询函数快速获取子矩形内 $\texttt{1}$ 的个数。
3. 枚举所有左上角和右下角，检查条件并更新最小面积。
4. 输出结果。

### 复杂度分析

- **时间复杂度：** $O(n^2 m^2)$。
- **空间复杂度：** $O(nm)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <climits>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m, k;
    cin >> n >> m >> k;
    vector<string> grid(n);
    for (int i = 0; i < n; i++) cin >> grid[i];
    vector<vector<int>> pref(n + 1, vector<int>(m + 1, 0));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            pref[i][j] = pref[i-1][j] + pref[i][j-1] - pref[i-1][j-1] + (grid[i-1][j-1] == '1');
        }
    }
    auto query = [&](int r1, int c1, int r2, int c2) {
        return pref[r2][c2] - pref[r1-1][c2] - pref[r2][c1-1] + pref[r1-1][c1-1];
    };
    int ans = INT_MAX;
    for (int r1 = 1; r1 <= n; r1++) {
        for (int r2 = r1; r2 <= n; r2++) {
            for (int c1 = 1; c1 <= m; c1++) {
                for (int c2 = c1; c2 <= m; c2++) {
                    if (query(r1, c1, r2, c2) >= k) {
                        ans = min(ans, (r2 - r1 + 1) * (c2 - c1 + 1));
                    }
                }
            }
        }
    }
    cout << (ans == INT_MAX ? 0 : ans) << "\n";
    return 0;
}
```
