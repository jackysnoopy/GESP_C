# [GESP202603 五级] 有限不循环小数

## 题目描述
若 $\frac{1}{a}$ 可化为一个有限的，不循环的小数，则称 $a$ 为**终止数**。

请你求出在 $L$ 到 $R$ 中终止数的数量。

## 输入格式
输入一行，包含两个整数 $L,R$。

## 输出格式
输出一行，包含一个整数，表示 $L$ 到 $R$ 中终止数的数量。

## 背景
对应的选择、判断题：<https://ti.luogu.com.cn/problemset/1209>

## 样例
### 样例 1
```
2 11
```
```
5
```

## 提示
### 样例解释

在 $[2,11]$ 终止数有 $2$、$4$、$5$、$8$、$10$。

### 数据范围


保证 $1 \le L \le R \le 10^6$。

## 解题思路

### 问题分析
1/a 可化为有限不循环小数 ↔ a 的质因数只包含 2 和 5。统计 [L,R] 中形如 2ⁱ×5ʲ 的数的个数。

### 核心思路
枚举所有 2ⁱ×5ʲ ≤ R 的数并判断是否在范围内。

### 算法流程
1. 读入 L, R
2. 枚举 2ⁱ×5ʲ ≤ R
3. 统计落在 [L,R] 内的个数
4. 输出结果

### 复杂度分析
- 时间复杂度：O(logR × logR)
- 空间复杂度：O(1)

## 参考代码
```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long L, R;
    cin >> L >> R;
    
    int ans = 0;
    for (long long a = 1; a <= R; a *= 2) {
        for (long long b = a; b <= R; b *= 5) {
            if (b >= L) ans++;
        }
    }
    
    cout << ans << '\n';
    return 0;
}
```
