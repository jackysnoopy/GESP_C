# [GESP202312 五级] 小杨的幸运数

## 题目描述

小杨认为，所有大于等于 $a$ 的完全平方数都是他的超级幸运数。

小杨还认为，所有超级幸运数的倍数都是他的幸运数。自然地，小杨的所有超级幸运数也都是幸运数。

对于一个非幸运数，小杨规定，可以将它一直 $+1$，直到它变成一个幸运数。我们把这个过程叫做幸运化。例如，如果 $a=4$，那么 $4$ 是最小的幸运数，而 $1$ 不是，但我们可以连续对 $1$ 做 $3$ 次 $+1$ 操作，使其变为 $4$，所以我们可以说， $1$ 幸运化后的结果是 $4$。

现在，小杨给出 $N$ 个数，请你首先判断它们是不是幸运数；接着，对于非幸运数，请你将它们幸运化。

## 输入格式

第一行 $2$ 个正整数 $a, N$。

接下来 $N$ 行，每行一个正整数 $x$ ，表示需要判断（幸运化）的数。

## 输出格式

输出 $N$ 行，对于每个给定的 $x$ ，如果它是幸运数，请输出 `lucky`，否则请输出将其幸运化后的结果。

## 提示

**样例解释 1**

$1$ 虽然是完全平方数，但它小于 $a$，因此它并不是超级幸运数，也不是幸运数。将其进行 $3$ 次 $+1$ 操作后，最终得到幸运数 $4$。

$4$ 是幸运数，因此直接输出 `lucky`。

$5$ 不是幸运数，将其进行 $3$ 次 $+1$ 操作后，最终得到幸运数 $8$。

$9$ 是幸运数，因此直接输出 `lucky`。

**数据规模**

对于 $30\%$ 的测试点，保证 $a,x \le 100,N \le 100$。

对于 $60\%$ 的测试点，保证 $a,x \le 10^6$。

对于所有测试点，保证 $a \le 1,000,000$；保证 $N \le 2 \times 10^5$；保证 $1 \le x \le 1,000,001$。

## 解题思路

### 问题分析

超级幸运数是所有大于等于a的完全平方数。幸运数包括所有超级幸运数及其倍数。对于每个查询，需要判断是否是幸运数，如果不是则找到比它大的最小幸运数。

### 核心思路

使用筛法预处理标记所有幸运数，并预处理每个数的下一个幸运数。

### 算法流程

1. 预处理阶段：
   - 遍历所有可能的完全平方数i*i (i从1开始)
   - 对于每个i*i ≥ a的数，标记它为超级幸运数
   - 同时标记它的所有倍数为幸运数
2. 预处理每个数的下一个幸运数：
   - 从MAX到1反向遍历
   - 对于每个数i，如果它是幸运数则next_luck[i]=i
   - 否则next_luck[i]=next_luck[i+1]
3. 查询处理阶段：
   - 对于每个查询x
   - 如果x是幸运数则输出"lucky"
   - 否则输出next_luck[x]

### 复杂度分析

- **时间复杂度：** 
  - 预处理阶段：O(MAX log log MAX) (类似埃拉托斯特尼筛法)
  - 查询阶段：O(1) 每个查询
- **空间复杂度：** O(MAX) (存储luck和next_luck数组)

### 算法优化

1. 使用反向预处理next_luck数组可以O(1)回答每个查询
2. 筛法实现时只处理i*i ≥ a的情况，减少不必要的计算
3. 使用数组存储比vector更高效

## 参考代码
## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cmath>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int a, N;
    cin >> a >> N;
    int MAX = 2000000;
    vector<bool> lucky(MAX + 1, false);
    int start = (int)ceil(sqrt(a));
    for (int s = start; (long long)s * s <= MAX; s++) {
        int sq = s * s;
        for (int m = sq; m <= MAX; m += sq) lucky[m] = true;
    }
    while (N--) {
        int x;
        cin >> x;
        if (lucky[x]) { cout << "lucky\n"; continue; }
        int cur = x;
        while (++cur <= MAX)
            if (lucky[cur]) { cout << cur << "\n"; break; }
    }
    return 0;
}
```
