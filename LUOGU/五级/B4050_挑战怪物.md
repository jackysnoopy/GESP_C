# [GESP202409 五级] 挑战怪物

## 题目描述

小杨正在和一个怪物战斗，怪物的血量为 $h$，只有当怪物的血量**恰好**为 $0$ 时小杨才能够成功击败怪物。

小杨有两种攻击怪物的方式：
- 物理攻击。假设当前为小杨第 $i$ 次使用物理攻击，则会对怪物造成 $2^{i - 1}$ 点伤害。
- 魔法攻击。小杨选择任意一个质数 $x$（不能超过怪物当前血量），对怪物造成 $x$ 点伤害。由于小杨并不擅长魔法，他只能使用**至多一次**魔法攻击。

小杨想知道自己能否击败怪物，如果能，小杨想知道自己最少需要多少次攻击。

## 输入格式

**本题单个测试点内有多组测试数据**。第一行包含一个正整数 $t$，代表测试用例组数。

接下来是 $t$ 组测试用例。对于每组测试用例，只有一行一个整数 $h$，代表怪物血量。

## 输出格式

对于每组测试用例，如果小杨能够击败怪物，输出一个整数，代表小杨需要的最少攻击次数，如果不能击败怪物，输出 $-1$。

## 样例

### 样例 1
```
输入：
3
6
188
9999

输出：
2
4
-1
```

### 样例解释

对于第一组测试用例，一种可能的最优方案为：小杨先对怪物使用魔法攻击，选择质数 $5$ 造成 $5$ 点伤害，之后对怪物使用第 $1$ 次物理攻击，造成 $2^{1 - 1} = 1$ 点伤害，怪物血量恰好为 $0$，小杨成功击败怪物。

## 提示

### 数据规模与约定

$1 \le t \le 10$，$1 \le h \le 10^5$。

## 解题思路

### 问题分析

怪物血量 $h$，两种攻击方式：
- 物理攻击：第 $i$ 次攻击造成 $2^{i-1}$ 伤害
- 魔法攻击：选择质数 $x$（$\le$ 当前血量），造成 $x$ 伤害，至多使用一次

求最小攻击次数使血量恰好为 $0$。

### 核心思路

**贪心策略**：优先使用物理攻击，因为物理攻击的伤害是累加的（$1, 2, 4, 8, \ldots$）。

**关键观察**：
1. 如果 $h$ 本身是质数，只需1次魔法攻击
2. 否则，依次使用物理攻击（第1次1点，第2次2点，第3次4点...）
3. 每次使用物理攻击后，检查剩余血量：
   - 若剩余血量为0，完成
   - 若剩余血量为质数，再使用1次魔法攻击
   - 若剩余血量小于0，无法完成

### 算法流程

1. 判断 $h$ 是否为质数，若是则直接输出1
2. 循环 $i$ 从1开始：
   - $h$ 减去 $2^{i-1}$
   - 若 $h < 0$，输出 $-1$（无法完成）
   - 若 $h = 0$，输出 $i$（纯物理攻击）
   - 若 $h > 0$ 且 $h$ 为质数，输出 $i+1$（物理+魔法）

### 复杂度分析

- **时间复杂度**：$O(\sqrt{h} \cdot \log h)$，循环最多 $\log h$ 次，每次判断质数 $O(\sqrt{h})$
- **空间复杂度**：$O(1)$

## 参考代码

```cpp
#include<bits/stdc++.h>  
using namespace std;

bool zs(int a){
    if(a==1) return 0;
    for(int i=2; i<=sqrt(a); i++){
        if(a%i==0) return 0;
    }
    return 1;
}

int main(){
    int t;
    cin >> t;
    int h;
    while(t){
        t--;
        cin >> h;
        if(zs(h)){cout << 1 << endl; continue;}
        for(int i=1; i>=1; i++){
            h -= pow(2, i-1);
            if(h < 0){cout << -1 << endl; break;}
            if(h == 0){cout << i << endl; break;}
            if(h > 0 && zs(h)){cout << i+1 << endl; break;}
        }
    }
    return 0;
}
```
