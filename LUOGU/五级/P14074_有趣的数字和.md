# [GESP202509 五级] 有趣的数字和

## 题目描述

如果一个正整数的二进制表示包含奇数个 $1$，那么小 A 就会认为这个正整数是有趣的。

例如，$7$ 的二进制表示为 $(111)_2$，包含 $1$ 的个数为 $3$ 个，所以 $7$ 是有趣的。但是 $9=(1001)_2$ 包含 $2$ 个 $1$，所以 $9$ 不是有趣的。

给定正整数 $l,r$，请你统计满足 $l\le n\le r$ 的有趣的整数 $n$ 之和。

## 输入格式

一行，两个正整数 $l,r$，表示给定的正整数。

## 输出格式

一行，一个正整数，表示 $l,r$ 之间有趣的整数之和。

## 样例

### 样例 #1

**输入**
```
3 8
```

**输出**
```
19
```

### 样例 #2

**输入**
```
65 36248
```

**输出**
```
328505490
```

## 提示

**【数据范围】**

对于 $40\%$ 的测试点，保证 $1\le l\le r\le 10^4$。

对于另外 $30\%$ 的测试点，保证 $l=1$ 并且 $r=2^k-1$，其中 $k$ 是大于 $1$ 的正整数。

对于所有测试点，保证 $1 \le l\le r\le 10^9$。

**【提示】**

由于本题的数据范围较大，整数类型请使用 long long。

## 解题思路

### 问题分析

求区间 $[l, r]$ 中二进制表示有奇数个 $1$ 的整数之和。暴力遍历 $O(r-l)$ 在 $r \le 10^9$ 时超时，需要更高效的方法。

利用前缀和思想：$[l, r]$ 的答案 = $[0, r]$ 的答案 $-$ $[0, l-1]$ 的答案。问题转化为高效计算 $f(n)$：$[0, n]$ 中二进制 $1$ 个数为奇数的数之和。

### 核心思路

**数位 DP 递归**。将 $[0, n]$ 按最高位拆分为两部分：

设 $x$ 为不超过 $n$ 的最大 $2$ 的幂（如 $n=38$ 时 $x=32$）。

1. **$[0, x-1]$**：这些数最高位为 $0$，不改变 $1$ 的奇偶性。直接用公式计算奇偶性为 $p$ 的数量和总和。
2. **$[x, n]$**：这些数最高位为 $1$，使奇偶性翻转。递归求解 $[0, n-x]$ 中奇偶性为 $1-p$ 的结果，再加上高位 $x$ 的贡献（每个数都要加 $x$）。

**合并**：数量相加，总和 = 左半总和 + 右半子问题总和 + $x \times$ 右半数量。

### 边界情况

- $n = 0$：只有 $0$（偶数个 $1$），$p=0$ 时数量 $1$，$p=1$ 时数量 $0$。
- $n = 1$：$0$（偶）、$1$（奇），$p=1$ 时数量 $1$ 总和 $1$，$p=0$ 时数量 $1$ 总和 $0$。
- $n = 2^k - 1$：完整的 $k$ 位区间，数量 $= (n+1)/2$，总和 $= n(n+1)/4$。

### 算法流程

1. 读入 $l, r$。
2. 定义递归函数 `fun(n, p)`：返回 $[0, n]$ 中二进制 $1$ 个数奇偶性为 $p$ 的 $\{数量, 总和\}$。
   - 基础情况：$n \le 1$ 时用 `baseCase` 处理，$n = 2^k-1$ 时用公式。
   - 递归：拆分为 $[0, x-1]$ 和 $[x, n]$，合并结果。
3. 答案 $= fun(r, 1).second - fun(l-1, 1).second$。

### 复杂度分析

- **时间复杂度**：$O(\log n)$，每次递归将 $n$ 减半。
- **空间复杂度**：$O(\log n)$，递归栈深度。

## 参考代码

```cpp
#include <iostream>
#include <utility>

using namespace std;

using ll = long long;
using info = pair<ll, ll>;

ll offset(ll n)
{
	ll res = 1;
	while ((n >> 1) >= res)
	{
		res <<= 1;
	}
	return res;
}

info baseCase(ll n, ll p)
{
	if (n == 0)
	{
		return make_pair(1 - p, 0);
	}
	if (n == 1)
	{
		return make_pair(1, p);
	}
	return make_pair((n + 1) / 2, n * (n + 1) / 4);
}

info fun(ll n, ll p)
{
	if (n <= 1)
	{
		return baseCase(n, p);
	}
	ll x = offset(n);
	
	info a = baseCase(x - 1, p);
	info b = fun(n - x, 1 - p);
	
	return make_pair(a.first + b.first, a.second + b.second + x * b.first);
}

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(nullptr);

	ll l = 0;
	ll r = 0;
	cin >> l >> r;

	cout << fun(r, 1).second - fun(l - 1, 1).second << '\n';

	return 0;
}
```
