# [GESP202509 五级] 有趣的数字和

## 题目描述

如果一个正整数的二进制表示包含奇数个 $1$，那么小 A 就会认为这个正整数是有趣的。

例如，$7$ 的二进制表示为 $(111)_2$，包含 $1$ 的个数为 $3$ 个，所以 $7$ 是有趣的。但是 $9=(1001)_2$ 包含 $2$ 个 $1$，所以 $9$ 不是有趣的。

给定正整数 $l,r$，请你统计满足 $l\le n\le r$ 的有趣的整数 $n$ 之和。

## 输入格式

一行，两个正整数 $l,r$，表示给定的正整数。

## 输出格式

一行，一个正整数，表示 $l,r$ 之间有趣的整数之和。

## 样例

### 样例 #1

**输入**
```
3 8
```

**输出**
```
19
```

### 样例 #2

**输入**
```
65 36248
```

**输出**
```
328505490
```

## 提示

**【数据范围】**

对于 $40\%$ 的测试点，保证 $1\le l\le r\le 10^4$。

对于另外 $30\%$ 的测试点，保证 $l=1$ 并且 $r=2^k-1$，其中 $k$ 是大于 $1$ 的正整数。

对于所有测试点，保证 $1 \le l\le r\le 10^9$。

**【提示】**

由于本题的数据范围较大，整数类型请使用 long long。

## 解题思路

### 问题分析

求区间 $[l, r]$ 中二进制表示有奇数个 $1$ 的整数之和。

### 核心思路

直接遍历，使用 `__builtin_popcountll` 判断每个数。

### 算法流程

1. 遍历 $i$ 从 $l$ 到 $r$。
2. 如果 $\text{popcount}(i)$ 为奇数，累加 $i$。

### 复杂度分析

- **时间复杂度：** $O(r-l)$。
- **空间复杂度：** $O(1)$。

## 参考代码
## 参考代码

```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long l, r;
    cin >> l >> r;
    long long sum = 0;
    for (long long i = l; i <= r; i++) {
        if (__builtin_popcountll(i) & 1) sum += i;
    }
    cout << sum << "\n";
    return 0;
}
```
