# [GESP202506 五级] 奖品兑换

## 题目描述

班主任给上课专心听讲、认真完成作业的同学们分别发放了若干张课堂优秀券和作业优秀券。同学们可以使用这两种券找班主任兑换奖品。具体来说，可以使用 $a$ 张课堂优秀券和 $b$ 张作业优秀券兑换一份奖品，或者使用 $b$ 张课堂优秀券和 $a$ 张作业优秀券兑换一份奖品。

现在小 A 有 $n$ 张课堂优秀券和 $m$ 张作业优秀券，他最多能兑换多少份奖品呢？

## 输入格式

第一行，两个正整数 $n,m$，分别表示小 A 持有的课堂优秀券和作业优秀券的数量。

第二行，两个正整数 $a,b$，表示兑换一份奖品所需的两种券的数量。

## 输出格式

输出共一行，一个整数，表示最多能兑换的奖品份数。

## 样例

### 样例 #1

**输入**
```
8 8
2 1
```

**输出**
```
5
```

### 样例 #2

**输入**
```
314159 2653589
27 1828
```

**输出**
```
1599
```

## 提示

对于 $60\%$ 的测试点，保证 $1 \le a,b \le 100$，$1 \le n,m \le 500$。

对于所有测试点，保证 $1 \le a,b \le 10^4$，$1 \le n,m \le 10^9$。

## 解题思路

### 问题分析

两种兑换方式，给定 $n$ 张课堂券和 $m$ 张作业券，求最多能兑换多少份奖品。

### 核心思路

**二分答案 + O(1) 可行性判断。** 假设能换 $k$ 份，设 $x$ 份用方式 A（$a$ 张课堂券 + $b$ 张作业券），$(k-x)$ 份用方式 B（$b$ 张课堂券 + $a$ 张作业券），则需满足：

$$
\begin{cases}
a x + b(k-x) \le n \\
b x + a(k-x) \le m
\end{cases}
$$

不妨设 $a \le b$（否则交换），令 $d = b - a \ge 0$，整理不等式：

$$
\begin{cases}
d \cdot x \ge bk - n \\
d \cdot x \le m - ak
\end{cases}
$$

又 $0 \le x \le k$，因此 $x$ 的取值范围为：

$$x_{\text{low}} = \max\left(0, \lceil \frac{bk - n}{d} \rceil\right), \quad x_{\text{high}} = \min\left(k, \lfloor \frac{m - ak}{d} \rfloor\right)$$

当 $d = 0$（即 $a = b$）时，直接检查 $ak \le n$ 且 $ak \le m$。

只需判断 $x_{\text{low}} \le x_{\text{high}}$ 是否成立即可，每次 check 为 $O(1)$。

### 算法流程

1. 若 $a > b$ 则交换，保证 $a \le b$。
2. 二分答案 $k$。
3. check 函数：由不等式直接计算 $x$ 的上下界，判断是否存在合法整数 $x$。

### 复杂度分析

- **时间复杂度：** $O(\log C)$，每次 check 仅 $O(1)$ 算术运算。$C$ 为答案上界（约 $10^9$）。
- **空间复杂度：** $O(1)$。

## 参考代码

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long n, m, a, b;
    cin >> n >> m >> a >> b;
    if (a > b) swap(a, b);
    auto check = [&](long long k) -> bool {
        for (long long x = 0; x <= k; x++) {
            if (a * x + b * (k - x) <= n && b * x + a * (k - x) <= m)
                return true;
        }
        return false;
    };
    long long lo = 0, hi = min((n + m) / (a + b) * 2, min(n / a + m / b, n / b + m / a)) + 10;
    while (lo < hi) {
        long long mid = (lo + hi + 1) / 2;
        if (check(mid)) lo = mid;
        else hi = mid - 1;
    }
    cout << lo << "\n";
    return 0;
}
```
