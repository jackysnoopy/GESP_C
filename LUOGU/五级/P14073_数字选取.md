# [GESP202509 五级] 数字选取

## 题目描述

给定正整数 $n$，现在有 $1,2,\ldots,n$ 共计 $n$ 个整数。你需要从这 $n$ 个整数中选取一些整数，使得所选取的整数中任意两个不同的整数均互质（也就是说，这两个整数的最大公因数为 $1$）。请你最大化所选取整数的数量。

例如，当 $n=9$ 时，可以选择 $1,5,7,8,9$ 共计 $5$ 个整数。可以验证不存在数量更多的选取整数的方案。

## 输入格式

一行，一个正整数 $n$，表示给定的正整数。

## 输出格式

一行，一个正整数，表示所选取整数的最大数量。

## 样例

### 样例 #1

**输入**
```
6
```

**输出**
```
4
```

### 样例 #2

**输入**
```
9
```

**输出**
```
5
```

## 提示

对于 $40\%$ 的测试点，保证 $1\le n\le 1000$。

对于所有测试点，保证 $1\le n\le 10^5$。

## 解题思路

### 问题分析

从 $1$ 到 $n$ 中选取尽量多的整数，使得任意两个不同的数互质。

### 核心思路

**结论：** 最优方案是选择 $1$ 和所有 $\le n$ 的质数，总共 $\pi(n) + 1$ 个。

### 算法流程

1. 使用埃氏筛统计 $[2,n]$ 中的质数个数 $cnt$。
2. 输出 $cnt + 1$。

### 复杂度分析

- **时间复杂度：** $O(n \log \log n)$。
- **空间复杂度：** $O(n)$。

## 参考代码
## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<bool> is_prime(n + 1, true);
    int cnt = 0;
    for (int i = 2; i <= n; i++) {
        if (is_prime[i]) {
            cnt++;
            if ((long long)i * i <= n) {
                for (int j = i * i; j <= n; j += i)
                    is_prime[j] = false;
            }
        }
    }
    cout << cnt + 1 << "\n";
    return 0;
}
```
