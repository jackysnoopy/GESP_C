# P13009 【MX-X13-T4】「KDOI-12」好胜是人的本能，功利是社会的本性。

## 题目描述

给定一个长度为 $n$ 的正整数序列 $a_1, \ldots, a_n$ 以及一个正整数 $m$ 满足 $m \ge \max a_i$。

你可以对序列进行任意次操作（也可以不操作）。每次操作你可以选择一个区间 $[l,r]$，然后对于所有 $l\leq i\leq r$，令 $a_i\gets\bigl\lfloor\frac{m}{a_i}\bigr\rfloor$。

求可以得到的 $\sum_{j=1}^na_j$ 的最大值以及对应的最少操作次数。


## 输入格式

**本题有多组测试数据。**

第一行，一个正整数 $T$，表示测试数据组数。对于每组测试数据：

* 第一行，两个正整数 $n,m$。
* 第二行，$n$ 个正整数 $a_1, \ldots, a_n$。


## 输出格式

对于每组测试数据，一行，两个非负整数，分别表示 $\sum_{j=1}^na_j$ 的最大值以及对应的最少操作次数。

## 解题思路

### 问题分析

给定正整数序列 a 和 m（m ≥ max(a_i)）。操作：选区间 [l,r]，对所有 a_i 执行 a_i ← ⌊m/a_i⌋。求可得到的最大和及达到最大和的最少操作次数。

### 核心思路

- 对每个元素 a_i，考虑三种状态：原始值 v0 = a_i，操作一次 v1 = ⌊m/a_i⌋，操作两次 v2 = ⌊m/v1⌋。
- 最大值 vmax = max(v0, v1, v2)，最大和即为各元素 vmax 之和。
- 需要操作的次数：vmax == v0 需 0 次，vmax == v1 需 1 次，vmax == v2 需 2 次。
- 最少操作次数 = 覆盖所有需操作元素的最少区间数。需 1 次的元素需恰好被覆盖 1 次，需 2 次的元素需被覆盖 2 次。

### 算法流程

1. 对每个元素计算 v0、v1、v2 和 vmax，累加最大和。
2. 标记每个元素需要 0、1 或 2 次操作。
3. 第一轮：统计 need ≥ 1 的连续段数（最少区间覆盖一次）。
4. 第二轮：统计 need == 2 的连续段数（额外覆盖一次）。
5. 输出最大和与总操作次数。

### 复杂度分析

- 时间复杂度：O(n)，每个元素只处理常数次。
- 空间复杂度：O(n)，存储各状态值。
