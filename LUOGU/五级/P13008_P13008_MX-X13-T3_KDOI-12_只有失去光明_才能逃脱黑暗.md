# P13008 【MX-X13-T3】「KDOI-12」只有失去光明，才能逃脱黑暗。

## 题目描述

给定一个非负整数 $x$，你要经过若干次以下操作将其变成 $y$，求最小代价：

* 选择一个 $0\leq i\leq k$，花费 $a_i$ 代价将 $x$ 加或减 $2^i$。

**注意：你在操作时不需要保证 $x$ 为非负整数。**


## 输入格式

**本题有多组测试数据。**

第一行，一个正整数 $T$，表示测试数据组数。对于每组测试数据：

* 第一行，三个非负整数 $x,y,k$。
* 第二行，$k+1$ 个正整数 $a_0, \ldots, a_k$。


## 输出格式

对于每组测试数据，一行，一个非负整数，表示最小代价。

对于样例的第二组测试数据：经过以下两次操作即可让 $x$ 变为 $y$，且代价最小：

## 解题思路

### 问题分析

给定整数 x、y 和 k 种操作（第 i 种可加减 2^i，代价 a_i）。求将 x 变为 y 的最小代价。操作过程中 x 可以为负数。

### 核心思路

- 问题等价于：用 2 进制表示差值 diff = |y-x|，在每一位上决定使用多少次 2^i 操作。
- 逐位 DP：从低位到高位处理，状态为进位 carry ∈ {-2, -1, 0, 1, 2}。
- 对每一位 b（diff 的第 i 位），枚举使用次数 ci ∈ {-3..3}，使得 (b + carry + ci) 为偶数。
- 新的进位 next_carry = (b + carry + ci) / 2，代价增加 |ci| × a_i。

### 算法流程

1. 计算 diff = |y-x|，若为 0 则代价为 0。
2. 初始化 dp[carry=0] = 0，其余为 INF。
3. 从第 0 位到第 31 位逐位处理：
   - 取 diff 的当前位 b。
   - 枚举当前进位 carry 和使用次数 ci，满足整除条件。
   - 更新下一位的 dp 值。
4. 最终答案为 dp[carry=0]。

### 复杂度分析

- 时间复杂度：O(32 × 5 × 7) ≈ O(1)，每位常数次转移。
- 空间复杂度：O(1)，仅用常数大小的 dp 数组。
