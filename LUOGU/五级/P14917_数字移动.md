# [GESP202512 五级] 数字移动

## 题目描述

小 A 有一个包含 $N$ 个正整数的序列 $A=\{A_1,A_2,\cdots,A_N\}$，序列 $A$ 恰好包含 $\frac{N}{2}$ 对不同的正整数。形式化地，对于任意 $1 \le i \le N$，存在唯一一个 $j$ 满足 $1\le j \le N, i\neq j, A_i=A_j$。

小 A 希望每对相同的数字在序列中相邻，为了实现这一目的，小 A 每次操作会选择任意 $i(1\le i\le N)$，将当前序列的第 $i$ 个数字移动到任意位置，并花费对应数字的体力。

例如，假设序列 $A=\{1,2,1,3,2,3\}$，小 A 可以选择 $i=2$，将 $A_2=2$ 移动到 $A_3=1$ 的后面，此时序列变为 $\{1,1,2,3,2,3\}$，耗费 $2$ 点体力。小 A 也可以选择 $i=3$，将 $A_3=1$ 移动到 $A_2=2$ 的前面，此时序列变为 $\{1,1,2,3,2,3\}$，花费 $1$ 点体力。

小 A 可以执行任意次操作，但他希望自己每次花费的体力尽可能小。小 A 希望你能帮他计算出一个最小的 $x$，使得他能够在每次花费的体力均不超过 $x$ 的情况下令每对相同的数字在序列中相邻。

## 输入格式

第一行一个正整数 $N$，代表序列长度，保证 $N$ 为偶数。

第二行包含 $N$ 个正整数 $A_1,A_2,\ldots,A_N$，代表序列 $A$。且对于任意 $1\le i\le N$，存在唯一一个 $j$ 满足 $1\le j \le N,i\neq j,A_i=A_j$。

数据保证小 A 至少需要执行一次操作。

## 输出格式

输出一行，代表满足要求的 $x$ 的最小值。

## 样例

### 样例 #1

**输入**
```
6
1 2 1 3 2 3
```

**输出**
```
2
```

## 提示

对于 $40\%$ 的测试点，保证 $1\le N,A_i\le 100$。

对于所有测试点，保证 $1\le N,A_i\le 10^5$。

## 解题思路

### 问题分析

序列包含 $\frac{N}{2}$ 对相同的数。每次移动某个数花费其数值大小的体力。目标是找到最小的 $x$，使得只移动值 $\le x$ 的数就能让所有对子相邻。

### 核心思路

**二分答案 $x$**。关键观察：值 $> x$ 的数不能被移动（花费会超过 $x$），因此它们必须**已经处于合法位置**——即在原序列中提取出所有值 $> x$ 的元素后，它们必须能形成连续的相等对子。

例如序列 $\{1, 2, 1, 3, 2, 3\}$，若 $x = 1$：
- 值 $> 1$ 的元素为 $\{2, 3, 2, 3\}$
- 检查：$(2, 3)$ 不相等，$(2, 3)$ 不相等 → 不合法

若 $x = 2$：
- 值 $> 2$ 的元素为 $\{3, 3\}$
- 检查：$(3, 3)$ 相等 → 合法

### 算法流程

1. 读入 $N$ 和序列 $A$。
2. 二分答案 $x$，范围 $[1, \max(A)]$：
   - 提取所有值 $> x$ 的元素到数组 $B$。
   - 检查 $B$ 中元素是否能两两配对：$B[0]=B[1], B[2]=B[3], \ldots$
   - 若能配对，说明 $x$ 可行，尝试更小的 $x$；否则需要更大的 $x$。
3. 输出最小的可行 $x$。

### 复杂度分析

- **时间复杂度**：$O(N \log C)$，二分 $\log C$ 次，每次遍历 $N$ 个元素。
- **空间复杂度**：$O(N)$，存储序列和临时数组。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

bool can(int x, const vector<int>& A) {
    vector<int> B;
    for (int v : A) {
        if (v > x) B.push_back(v);
    }
    
    for (int i = 0; i < (int)B.size(); i += 2) {
        if (B[i] != B[i + 1]) return false;
    }
    return true;
}

int main() {
    
    int N;
    cin >> N;
    vector<int> A(N);
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }

    int maxA = *max_element(A.begin(), A.end());


    int left = 1, right = maxA, ans = maxA;
    while (left <= right) {
        int mid = (left + right) / 2;
        if (can(mid,A)) {
            ans = mid;
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }

    cout << ans <<endl;
    return 0;
}
```
