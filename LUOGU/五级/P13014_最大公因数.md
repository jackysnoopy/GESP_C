# [GESP202506 五级] 最大公因数

## 题目描述

对于两个正整数 $a,b$，他们的最大公因数记为 $\gcd(a,b)$。对于 $k > 3$ 个正整数 $c_1,c_2,\dots,c_k$，他们的最大公因数为：

$$\gcd(c_1,c_2,\dots,c_k)=\gcd(\gcd(c_1,c_2,\dots,c_{k-1}),c_k)$$

给定 $n$ 个正整数 $a_1,a_2,\dots,a_n$ 以及 $q$ 组询问。对于第 $i(1 \le i \le q)$ 组询问，请求出 $a_1+i,a_2+i,\dots,a_n+i$ 的最大公因数，也即 $\gcd(a_1+i,a_2+i,\dots,a_n+i)$。

## 输入格式

第一行，两个正整数 $n,q$，分别表示给定正整数的数量，以及询问组数。

第二行，$n$ 个正整数 $a_1,a_2,\dots,a_n$。

## 输出格式

输出共 $q$ 行，第 $i$ 行包含一个正整数，表示 $a_1+i,a_2+i,\dots,a_n+i$ 的最大公因数。

## 样例

### 样例 #1

**输入**
```
5 3
6 9 12 18 30
```

**输出**
```
1
1
3
```

### 样例 #2

**输入**
```
3 5
31 47 59
```

**输出**
```
4
1
2
1
4
```

## 提示

对于 $60\%$ 的测试点，保证 $1 \le n \le 10^3$，$1 \le q \le 10$。

对于所有测试点，保证 $1 \le n \le 10^5$，$1 \le q \le 10^5$，$1 \le a_i \le 1000$。

## 解题思路

### 问题分析

### 核心数学原理推导

我们要计算的是：
$$S_i = \gcd(a_1+i, a_2+i, \dots, a_n+i)$$

最大公因数有一个关键性质：如果 $d$ 是这组数的公约数，那么 $d$ 必须能整除任意两个数之差。
因此，$S_i$ 必然是所有两点间差的公约数。

对于任意 $j, k$，我们有：
$$a_j+i - (a_k+i) = a_j - a_k$$
这意味着 $S_i$ 必须是 $\{|a_j - a_k|\}$ 的最大公因数 $\text{GCD}_{pairs}$ 的约数。

我们只需要让所有差的 GCD 最小化到足够小的集合即可。我们可以固定一个基准点，比如使用 $a_1$ 作为基准：
$$S_i = \gcd(a_1+i, (a_2+i) - (a_1+i), (a_3+i) - (a_1+i), \dots, (a_n+i) - (a_1+i))$$

利用 $\gcd(X, Y, Z) = \gcd(X, \gcd(Y, Z))$，我们可以将问题简化为：
$$S_i = \gcd(a_1+i, |a_2-a_1|, |a_3-a_1|, \dots, |a_n-a_1|)$$

**步骤 1: 计算 $G$ (预处理)**
我们首先计算出所有差值的最大公因数 $G$:
$$G = \gcd(|a_2-a_1|, |a_3-a_1|, \dots, |a_n-a_1|)$$

**步骤 2: 求解 $S_i$ (查询)**
一旦我们得到了 $G$，那么对于每个查询 $i$，答案就简化为：
$$S_i = \gcd(a_1+i, G)$$

### 特殊情况处理

1.  **当 $N=1$ 时:** 只有一个数 $a_1$。则 $\gcd(a_1+i) = a_1+i$. (此时 $G$ 的计算公式不适用，需要单独判断)。
2.  **当所有 $a_i$ 都相同时:** $a_1=a_2=\dots=a_n$。那么所有的差值都是 0。我们应该直接使用原始的 $\gcd(a_1+i, a_2+i, \dots) = a_1+i$. (此时 $G$ 的计算会因为 $\gcd(\dots, 0)$ 而出现问题，应避免依赖 $G$)

**最终鲁棒性流程：**
1.  如果 $N=1$，答案直接是 $a_1+i$。
2.  如果 $N>1$，则 $G = \gcd(|a_2-a_1|, |a_3-a_1|, \dots, |a_n-a_1|)$。答案为 $\gcd(a_1+i, G)$。

### 复杂度分析

*   **预处理 (计算 $G$):** 需要进行 $N-1$ 次 GCD 计算。由于 $A_i \le 1000$，GCD 的时间复杂度是极快的，可以忽略不计。总时间复杂度：$O(N \cdot \log A)$。
*   **查询:** 对每个 $Q$ 个查询，计算一次 $\gcd(a_1+i, G)$。总时间复杂度：$O(Q \cdot \log (\text{Max Value}))$.

由于 $N, Q \le 10^5$，这个方法的时间复杂度是线性的级别，可以轻松通过所有测试点。

### 核心思路

利用 gcd 性质：$\gcd(x_1,\dots,x_n) = \gcd(x_1, x_2-x_1, \dots, x_n-x_1)$。

### 算法流程

1. 计算 $g = \gcd(|a_2-a_1|, \dots, |a_n-a_1|)$。
2. 每个询问输出 $\gcd(a_1+i, g)$。

### 复杂度分析

- **时间复杂度：** $O(n + q \log C)$。
- **空间复杂度：** $O(n)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cstdlib>
using namespace std;

int gcd(int a, int b) {
    if (a < 0) a = -a;
    if (b < 0) b = -b;
    while (b != 0) {
        int t = a % b;
        a = b;
        b = t;
    }
    return a;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, q;
    cin >> n >> q;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    // 计算 D = gcd(a[1]-a[0], a[2]-a[0], ..., a[n-1]-a[0])
    int D = 0;
    for (int i = 1; i < n; i++) {
        D = gcd(D, a[i] - a[0]);
    }
    
    // 处理每个询问
    int i=1;
    while (q--) {

        cout << gcd(a[0] + i, D) << endl;
        i++;
    }
    
    return 0;
}
```
