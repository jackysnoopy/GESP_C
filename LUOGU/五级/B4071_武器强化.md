# [GESP202412 五级] 武器强化

## 题目描述

小杨有 $n$ 种武器和 $m$ 种强化材料。第 $i$ 种强化材料会适配第 $p_i$ 种武器，小杨可以花费 $c_i$ 金币将该材料对应的适配武器修改为任意武器。

小杨最喜欢第 $1$ 种武器，因此他希望适配该武器的强化材料种类数**严格大于**其他的武器，请你帮小杨计算为了满足该条件最少需要花费多少金币。

## 输入格式

第一行包含两个正整数 $n,m$，含义如题面所示。

之后 $m$ 行，每行包含两个正整数 $p_i,c_i$，代表第 $i$ 种强化材料的适配武器和修改花费。

## 输出格式

输出一个整数，代表能够使适配第 $1$ 种武器的强化材料种类数**严格大于**其他的武器最少需要花费的金币。

## 样例

### 样例 #1

**输入**
```
4 4
1 1
2 1
3 1
3 2
```

**输出**
```
1
```

## 提示

#### 样例解释

花费 $1$，将第三种强化材料的适配武器由 $3$ 改为 $1$。此时，武器 $1$ 有 $2$ 种强化材料适配，武器 $2$ 和武器 $3$ 都各有 $1$ 种强化材料适配，满足适配第 $1$ 种武器的强化材料种类数**严格大于**其他的武器。

#### 数据范围

对于 $100\%$ 的数据，保证 $1\le n,m\le 1\, 000$，$1\le p_i\le n$，$1\le c_i\le 10^9$。

| 子任务编号 | 得分占比 | $n$ | $m$ |
| :--: | :--: | :--: | :--: |
| $1$ | $20\%$ | $\le 2$ | $\le 1\, 000$ |
| $2$ | $20\%$ | $\le 1\,000$ | $\le 2$ |
| $3$ | $60\%$ | $\le 1\, 000$ | $\le 1\, 000$ |

## 解题思路

### 问题分析

有 $n$ 种武器和 $m$ 种强化材料。初始时每种材料适配一种武器。目标是使适配第 $1$ 种武器的材料数严格大于其他任何武器。

关键约束：修改一个材料的适配武器需要花费 $c_i$ 金币，我们可以将任意材料改为适配任意武器。

### 核心思路

**枚举最终武器 $1$ 的材料数 $target$**，对每个 $target$ 计算最小花费：

1. **削减其他武器**：对于每个其他武器 $i$（$i \ge 2$），如果其材料数 $\ge target$，则必须移除多余的材料（至少保留 $target-1$ 个），移除时优先选择花费最小的材料（贪心）。
2. **补充武器 $1$**：移除的材料和未被使用的材料放入"候选池"，如果武器 $1$ 的材料数仍不足 $target$，从候选池中选取花费最小的材料补充。

### 算法流程

1. 将每种武器的材料花费分别存入 `costs[i]` 并排序。
2. 枚举 `target` 从 `baseCnt`（武器 $1$ 的初始材料数）到 $m$（总材料数）。
3. 对每个 `target`：
   - 遍历武器 $2$ 到 $n$，计算每个武器需要移除的材料数 `over = size - target + 1`。
   - 移除花费最小的 `over` 个材料，花费累加，这些材料转为适配武器 $1$。
   - 未被移除的材料放入候选池 `needModify`。
   - 如果武器 $1$ 的材料数已 $\ge target$，直接更新答案。
   - 否则从 `needModify` 中取花费最小的材料补充，直到达到 `target`。
4. 取所有 `target` 中的最小花费作为答案。

### 复杂度分析

- **时间复杂度：** $O(m \log m + n \cdot m)$，枚举 $target$ 最多 $m$ 次，每次遍历 $n$ 个武器并排序。
- **空间复杂度：** $O(m)$，存储材料花费和候选池。

## 参考代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    int n, m;
    cin >> n >> m;
    
    vector<vector<int>> costs(n + 1);
    for (int i = 0; i < m; i++) {
        int p, c;
        cin >> p >> c;
        costs[p].push_back(c);
    }
    
    for (int i = 1; i <= n; i++) {
        sort(costs[i].begin(), costs[i].end());
    }
    
    long long ans = LLONG_MAX;
    int baseCnt = costs[1].size();
    
    for (int target = baseCnt; target <= m; target++) {
        long long curCost = 0;
        int curCnt1 = baseCnt;
        vector<int> needModify;
        
        for (int i = 2; i <= n; i++) {
            int over = costs[i].size() - target + 1;
            for (int j = 0; j < over && j < (int)costs[i].size(); j++) {
                curCost += costs[i][j];
                curCnt1++;
            }
            int start = max(0, over);
            for (int j = start; j < (int)costs[i].size(); j++) {
                needModify.push_back(costs[i][j]);
            }
        }
        
        if (curCnt1 >= target) {
            ans = min(ans, curCost);
            continue;
        }
        
        sort(needModify.begin(), needModify.end());
        int need = target - curCnt1;
        if (need <= (int)needModify.size()) {
            for (int i = 0; i < need; i++) {
                curCost += needModify[i];
            }
            ans = min(ans, curCost);
        }
    }
    
    cout << ans << "\n";
    return 0;
}
```
