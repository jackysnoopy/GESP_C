# [GESP202603 五级] 找数

## 题目描述
给定一个包含 $n$ 个互不相同的正整数的数组 $A$ 与一个包含 $m$ 个互不相同的正整数的数组 $B$，请你帮忙计算有多少个数在数组 $A$ 与数组 $B$ 中均出现。

## 输入格式
第一行包含两个整数 $n,m$。

第二行包含 $n$ 个正整数 $a_1,a_2,\cdots,a_n$ 表示数组 $A$。

第三行包含 $m$ 个正整数 $b_1,b_2,\cdots,b_m$ 表示数组 $B$。

## 输出格式
输出一个整数，表示在数组 $A$ 与数组 $B$ 中均出现的数的个数。

## 背景
对应的选择、判断题：<https://ti.luogu.com.cn/problemset/1209>

## 样例
### 样例 1
```
3 5
4 2 3
3 1 5 4 6
```
```
2
```

## 提示
### 样例解释


样例 1 中，$4$、$3$ 在数组 $A$ 与 $B$ 中均出现。


### 数据范围

对于 $40\%$ 的数据，保证 $1 \leq n,m \leq 1000$。

对于 $100\%$ 的数据，保证 $1 \leq n,m \leq 10^5$，$1 \leq a_i,b_i \leq 10^9$。

## 解题思路

### 问题分析
求两个无重复元素数组的交集元素个数。

### 核心思路
将 A 的元素存入哈希集合，遍历 B 统计。

### 算法流程
1. 读入 n, m 和数组 A, B
2. 将 A 的元素加入 unordered_set
3. 遍历 B 统计在 set 中的元素个数
4. 输出结果

### 复杂度分析
- 时间复杂度：O(n + m)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    unordered_set<int> st;
    for (int i = 0; i < n; i++) {
        int x;
        cin >> x;
        st.insert(x);
    }
    
    int ans = 0;
    for (int i = 0; i < m; i++) {
        int x;
        cin >> x;
        if (st.count(x)) ans++;
    }
    
    cout << ans << '\n';
    return 0;
}
```
