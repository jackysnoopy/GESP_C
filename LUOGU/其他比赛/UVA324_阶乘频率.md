# UVA324 - Factorial Frequencies（阶乘频率）

## 题意

对于每个输入的 n（1 ≤ n ≤ 100），计算 n! 并统计每个数字（0-9）在结果中出现的次数。输入以 0 结束。

## 解题思路

1. **高精度乘法**：n! 的结果可能非常大（100! 约有 158 位），需要用数组模拟高精度乘法
2. **逐位存储**：将数字按位存储在 vector 中，低位在前
3. **统计频率**：计算完成后统计每个数字的出现次数
4. **格式化输出**：按指定格式输出结果

## 算法复杂度

- 时间复杂度：O(n × L)，L 为结果的位数
- 空间复杂度：O(L)，存储高精度结果

## 注意事项

- 100! 的位数约 158 位，需要用高精度运算
- 输出格式要求每行 5 个数字，共两行
- 每组输出之间有空行
- 输入以 0 结束
