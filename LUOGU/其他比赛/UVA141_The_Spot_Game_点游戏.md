# UVA141 - The Spot Game (点游戏)

## 题目描述

两人在 5×5 的网格上玩游戏。每回合玩家可以执行以下操作之一：
- `+ x y`：在位置 (x,y) 放置一个点
- `- x y`：移除位置 (x,y) 的点
- `@ x y`：在 (x,y) 和 (N-1-y, N-1-x) 都放置一个点

游戏结束条件：棋盘出现**旋转对称**（即旋转180度后图案不变）。

## 输入格式

多组测试数据，每组第一行是操作数 n，接下来 n 行是操作。

## 输出格式

- 如果 Player 1 获胜：`Player 1 wins after move X`
- 如果 Player 2 获胜：`Player 2 wins after move X`
- 平局：`Draw`

## 算法分析

1. **模拟操作**：逐个执行操作更新棋盘
2. **对称性检查**：每次操作后检查棋盘是否旋转对称
3. **胜负判断**：导致对称的操作者输，对方获胜

## 复杂度分析

- 时间复杂度：O(n × N²)，其中 N=5
- 空间复杂度：O(N²)

## 示例输入

```
4
1 1 +
3 1 +
3 3 +
5 5 +
5
1 1 +
3 1 +
3 3 +
2 3 -
5 5 +
0
```

## 示例输出

```
Player 2 wins after move 4
Draw
```

## 代码说明

1. 使用 `isRotationalSymmetry()` 检查旋转对称性
2. 旋转对称条件：`board[i][j] == board[N-1-j][N-1-i]`
3. `@` 操作同时影响两个对称位置
4. 注意操作者编号：操作 i 由 Player (i%2+1) 执行
