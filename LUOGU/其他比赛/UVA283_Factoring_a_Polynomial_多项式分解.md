# UVA283 - Factoring a Polynomial (多项式分解)

## 题目描述

给定一个多项式，找出所有的整数根。

## 输入格式

- 多组测试数据
- 每组第一行是多项式的次数 n
- 接下来 n+1 个整数，从高次到低次的系数
- n=0 时结束

## 输出格式

- 输出所有整数根，按升序排列
- 如果没有整数根，输出"No integer roots"

## 算法思路

1. 根据有理根定理，整数根一定是常数项的因子
2. 枚举常数项的所有因子，代入多项式验证
3. 使用秦九韶算法高效计算多项式值

## 复杂度

- 时间：O(d × σ(d))，其中 d 是常数项
- 空间：O(n)
