# UVA125 - Numbering Roads (道路编号)

## 题目描述

给定一个城市的道路网络，包含 n 个交叉口和 r 条道路。每条道路连接两个交叉口 i 和 j（1 ≤ i < j ≤ n）。

城市的道路编号规则如下：
- 首先编号从交叉口 1 出发的所有道路（按到达的交叉口编号递增）
- 然后编号从交叉口 2 出发的所有道路
- 以此类推

对于每条输入的道路，输出其编号。

## 输入格式

- 每个测试用例包含两行
- 第一行：两个整数 r（道路数）和 n（交叉口数）
- 接下来 r 行，每行两个整数 u 和 v，表示连接交叉口 u 和 v 的道路
- 当 r = 0 且 n = 0 时输入结束
- 1 ≤ n ≤ 20，0 ≤ r ≤ n*(n-1)/2

## 输出格式

- 每个测试用例输出以 "Case X:" 开头
- 每条道路输出格式为 "u-v road_number"
- 测试用例之间空一行

## 样例

输入：
```
4 4
1 4
2 3
2 4
1 2
3 3
1 2
1 3
2 3
0 0
```

输出：
```
Case 1:
1-4 5
2-3 3
2-4 4
1-2 1

Case 2:
1-2 1
1-3 2
2-3 3
```

## 解题思路

### 问题分析

这是一个简单的模拟题。关键是理解道路编号的规则：
- 从交叉口 1 开始，按顺序编号所有从该点出发的道路
- 编号顺序是先按起点编号递增，再按终点编号递增

### 核心思路

1. 遍历所有可能的道路组合 (i, j)，其中 1 ≤ i < j ≤ n
2. 维护一个计数器记录当前道路编号
3. 对于每条输入的道路，在所有组合中查找匹配项，输出其编号

### 算法流程

1. 读取 r 和 n
2. 初始化道路编号计数器
3. 对每条输入的道路 (u, v)：
   - 遍历所有 (i, j) 组合，其中 i < j
   - 找到匹配的道路，输出其编号
4. 重复直到 r = 0 且 n = 0

### 复杂度分析

- 时间复杂度：O(r * n²)，其中 r 是道路数，n 是交叉口数
- 空间复杂度：O(1)

## 参考代码

```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int r, n;
    int caseNum = 0;
    
    while (cin >> r >> n && (r != 0 || n != 0)) {
        if (caseNum > 0) cout << "\n";
        cout << "Case " << ++caseNum << ":\n";
        
        for (int i = 0; i < r; i++) {
            int u, v;
            cin >> u >> v;
            
            int roadNum = 0;
            for (int a = 1; a < n; a++) {
                for (int b = a + 1; b <= n; b++) {
                    roadNum++;
                    if ((a == u && b == v) || (a == v && b == u)) {
                        cout << u << "-" << v << " " << roadNum << "\n";
                        goto next;
                    }
                }
            }
            next:;
        }
    }
    
    return 0;
}
```
