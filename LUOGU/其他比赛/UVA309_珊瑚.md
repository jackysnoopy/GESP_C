# UVA309 - Coral (珊瑚)

## 题目描述

珊瑚在水中生长，给定若干珊瑚的初始大小和生长率。在给定时刻，求编号最大的珊瑚的编号（即如果多个珊瑚大小相同，输出编号最大的那个）。

**输入格式**：
- 多组测试数据
- 每组数据以整数 $n$（珊瑚数量）开始
- 接下来 $n$ 行，每行两个整数：初始大小和生长率
- 以 $n=0$ 结束

**输出格式**：
- 对每组数据，输出最大珊瑚的编号（从 1 开始）

## 样例输入

```
3
1 2
3 1
2 2
2
1 1
2 1
0
```

## 样例输出

```
3
2
```

## 解题思路

1. **直接模拟**：对每个珊瑚，计算其在给定时刻的大小 = 初始大小 + 生长率。
2. **比较得出最大值**：遍历所有珊瑚，记录最大值及其编号。
3. **编号规则**：如果多个珊瑚大小相同，输出编号最大的那个（题目要求）。

## 时间复杂度

$O(n)$，其中 $n$ 为珊瑚数量。

## 空间复杂度

$O(1)$，只需常数额外空间。
