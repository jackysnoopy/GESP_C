# UVA275 - Exploring Pyramids (探索金字塔)

## 题目描述

给定一个金字塔结构，计算从底部到顶部的不同路径数量。

## 输入格式

- 多组测试用例
- 每组第一行包含一个整数n（n=0表示结束）

## 输出格式

对于每组测试用例，输出路径数量对1000000007取模的结果。

## 样例

### 输入
```
1
2
3
0
```

### 输出
```
1
3
13
```

## 提示

- 1 ≤ n ≤ 300
- 结果需要对1000000007取模

## 解题思路

### 问题分析

这是一个组合数学问题，需要计算金字塔结构中的路径数量。

### 核心思路

使用动态规划，定义dp[i][j]表示从第i层第j个位置到顶部的路径数。

状态转移方程：
- dp[i][j] = dp[i-1][j-1] + (i-1) * dp[i-1][j]

### 算法流程

1. 初始化dp[1][1] = 1
2. 从第2层开始递推
3. 最后累加所有dp[n][j]作为答案

### 复杂度分析

- 时间复杂度：O(n²)
- 空间复杂度：O(n²)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cstring>
using namespace std;

const int MAXN = 305;
const int MOD = 1000000007;

long long dp[MAXN][MAXN];

int main() {
    int n;
    while (cin >> n && n) {
        memset(dp, 0, sizeof(dp));
        dp[1][1] = 1;
        
        for (int i = 2; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                dp[i][j] = (dp[i-1][j-1] + (i-1) * dp[i-1][j]) % MOD;
            }
        }
        
        long long ans = 0;
        for (int j = 1; j <= n; j++) {
            ans = (ans + dp[n][j]) % MOD;
        }
        cout << ans << "\n";
    }
    return 0;
}
```
