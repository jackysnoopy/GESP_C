# UVA280 - Vertex (顶点)

## 题目描述

给定一个有向图，对于每个查询的起始顶点，找出所有不能从该顶点到达的顶点。

## 输入格式

- 多组测试数据
- 每组第一行是顶点数 n (1 ≤ n ≤ 100)
- 接下来若干行描述边，每行格式：起点 后续可达顶点列表 0
- 最后一行是0表示边描述结束
- 接下来是查询数 q，以及 q 个起始顶点
- n=0 时结束

## 输出格式

- 对每个查询，输出不可达顶点数及顶点编号

## 算法思路

1. 使用邻接表存储有向图
2. 对每个查询顶点进行DFS/BFS
3. 统计未访问的顶点数量

## 复杂度

- 时间：O(Q × (V + E))
- 空间：O(V + E)
