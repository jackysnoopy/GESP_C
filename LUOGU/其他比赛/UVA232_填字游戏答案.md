# UVA232 - Crossword Answers (填字游戏答案)

## 题目描述

给定一个填字游戏的网格（`*` 表示黑色格子，其他为字母），按编号规则提取所有横向和纵向的单词。

编号规则：如果一个格子是横向或纵向单词的起始位置，则分配一个编号。

## 输入格式

- 多组测试用例
- 每组第一行：行数 r 和列数 c
- 接下来 r 行：每行 c 个字符
- r=0, c=0 结束

## 输出格式

- 先输出 Across（横向）单词
- 再输出 Down（纵向）单词
- 每个单词格式：编号.单词

## 解题思路

1. 给网格编号
2. 提取横向单词（从左到右）
3. 提取纵向单词（从上到下）

## 注意事项

- 横向单词在最左边或左边是黑色格子时开始
- 纵向单词在最上边或上边是黑色格子时开始
- 编号从1开始递增
