# UVA329 - 铅笔和橡皮 (Pencils and Erasers)

## 题意

一位老师有预算 P，需要购买铅笔（价格 A）和橡皮（价格 B）。找出所有满足 x*A + y*B = P 的非负整数对 (x, y)。

## 思路

这是一个线性丢番图方程的非负整数解问题。由于 A 和 B 都是正整数，我们可以枚举 x 的值（从 0 到 P/A），然后检查对应的 y 是否为非负整数。

## 算法

1. 读入 P, A, B
2. 枚举 x 从 0 到 P/A
3. 对于每个 x，计算 rem = P - x*A
4. 如果 rem 能被 B 整除，则 y = rem/B 是一个有效解
5. 输出所有有效解

## 复杂度

- 时间复杂度：O(P/A)
- 空间复杂度：O(1)
