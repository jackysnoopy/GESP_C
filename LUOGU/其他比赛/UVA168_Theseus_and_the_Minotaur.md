# UVA168 - Theseus and the Minotaur（修斯与牛头人）

## 题目描述

修斯（Theseus）需要在迷宫中找到出口（灯），同时被牛头怪（Minotaur）追击。迷宫由走廊（`.`）和墙壁（`#`）组成。牛头怪每轮先移动一步到距离修斯最近的走廊位置，然后修斯移动一步。如果牛头怪追上修斯，输出修斯被抓住的位置；如果修斯到达灯的位置，输出牛头怪的位置。

## 输入格式

多组测试用例。每组以行列数开始，然后是迷宫地图，最后是修斯、牛头怪和灯的坐标。

## 输出格式

修斯被抓住的位置坐标，或牛头怪的位置坐标

## 算法思路

1. 从灯的位置BFS预计算到每个走廊格子的最短距离
2. 模拟追击过程：牛头怪选择距离修斯最近的相邻走廊格子移动
3. 如果牛头怪和修斯在同一格，修斯被抓住
4. 如果修斯到达灯的位置，输出牛头怪位置

**时间复杂度**: O(R × C) per test case
