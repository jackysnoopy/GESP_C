# UVA259 - Software Allocation (软件分配)

## 题意
有26个程序需要分配到10台计算机上，每个程序需要一定数量的计算资源（最多26个时段），每台计算机最多分配10个时段。每个程序只能在一台计算机上运行。判断是否存在可行分配方案。

## 解析
建图后使用最大流求解。源点连接每个程序（容量=所需时段），程序连接可运行的计算机（容量=1），计算机连接汇点（容量=10）。

## 代码要点
- 建立最大流模型
- 源点 -> 程序节点（容量=需求量）
- 程序节点 -> 可用计算机节点（容量=1）
- 计算机节点 -> 汇点（容量=10）
- 判断最大流是否等于总需求
