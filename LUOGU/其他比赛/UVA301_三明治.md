# UVA301 - Sandwiches（三明治）

## 题目大意

$n$ 个人排队买三明治，有 $m$ 种三明治，每种有特定价格。每个人对某种三明治有偏好。某些人如果得不到想要的类型，可以选择不要。求最大收益。

## 算法思路

本题使用 **DFS 搜索 + 剪枝** 的方法：

1. 按顺序考虑每个人
2. 对每个人有两种选择：
   - 给他想要的三明治（如果还有库存），收益增加该三明治价格
   - 不给他任何三明治
3. 递归搜索所有可能的选择，取最大收益

由于 $n$ 和 $m$ 的范围较小，DFS 可以在合理时间内完成。

## 时间复杂度

最坏 $O(2^n)$，但由于剪枝和数据范围限制，实际运行很快。

## 输入输出示例

```
输入：
3 3
10 20 30
1 1 2
0 0 0

输出：
50
```
