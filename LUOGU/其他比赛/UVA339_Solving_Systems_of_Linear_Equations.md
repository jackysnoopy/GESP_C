# UVA339 - Solving Systems of Linear Equations

## 题目描述

给定 $n$ 个线性方程组成的方程组（$n$ 个未知数，$n$ 个方程），使用高斯消元法求解。需要处理三种情况：

1. **唯一解**：输出每个未知数的值
2. **无解**：输出 "No solution."
3. **无穷多解**：输出 "Infinitely many solutions."

## 输入格式

多组测试数据。每组第一行一个整数 $n$（方程数），接下来 $n$ 行每行 $n + 1$ 个浮点数（前 $n$ 个为系数，最后一个为常数项）。$n = 0$ 时结束。

## 输出格式

```
Test Case X
x1=v1,x2=v2,...,xn=vn
```

或

```
No solution.
```

或

```
Infinitely many solutions.
```

## 样例输入

```
2
2 1 5
1 3 8
3
1 1 1 6
2 3 1 15
1 2 3 14
0
```

## 样例输出

```
Test Case 1
x1=1.400000,x2=2.200000

Test Case 2
x1=1.000000,x2=2.000000,x3=3.000000

```

## 解题思路

1. **高斯消元法**：使用部分主元选择（partial pivoting）提高数值稳定性
2. 逐列处理，对每列找最大绝对值元素作为主元
3. 交换行后，消去其他行的该列元素
4. 回代求解或判断无解/无穷多解
5. 当某行所有系数为 0 但常数项不为 0 时，无解
6. 当系数矩阵的秩小于 $n$ 时，无穷多解

## 时间复杂度

$O(n^3)$

## 空间复杂度

$O(n^2)$
