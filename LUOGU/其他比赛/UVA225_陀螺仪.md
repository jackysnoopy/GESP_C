# UVA225 - Gyroscopes (陀螺仪)

## 题目描述

一个机器人从原点出发，朝北方。给定一系列命令：
- `F`：向前移动1步
- `L`：向左转90度
- `R`：向右转90度

按顺序执行所有命令后，输出机器人走过的网格地图。其中 `*` 表示起点，`0`-`9` 表示该格子被访问的次数。

## 输入格式

- 第一行：测试用例数 T
- 每个测试用例：一行命令字符串

## 输出格式

- 每个测试用例输出网格地图
- 相邻测试用例之间空一行

## 解题思路

1. 模拟机器人在网格上的移动
2. 记录每个格子的访问次数
3. 找出边界，输出对应网格

## 代码要点

- 使用方向数组 `dx[]`, `dy[]` 模拟四个方向
- 使用 `map<pair<int,int>, int>` 记录访问次数
- 最后根据边界输出网格

## 时间复杂度

O(n)，其中 n 是命令字符串长度
