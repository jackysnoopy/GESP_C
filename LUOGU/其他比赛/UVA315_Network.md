# UVA315 - Network (网络)

## 题目描述
给定一个连通的无向图，求其中所有割点（关节点）的数量。割点是指删除该点后图不再连通的点。

## 输入格式
多组测试数据。每组第一行为节点数 n（n=0 结束）。
接下来若干行，每行描述一个节点的邻接关系：节点编号后跟邻接节点列表，以0结尾。

## 输出格式
每行输出割点的数量。

## 样例
```
输入：
5
1 2 3 4 0
2 1 3 0
3 1 2 0
4 1 5 0
5 4 0
0

输出：
1
```

## 提示
- 使用 Tarjan 算法求割点
- 根节点是割点当且仅当它有两棵以上子树
- 非根节点 u 是割点当且仅当存在子节点 v 满足 low[v] >= dfn[u]

## 解题思路

### 问题分析
经典的图论问题，求无向图的割点。

### 核心思路
- Tarjan DFS 算法
- 记录每个节点的 dfn（发现时间）和 low（可回溯到的最早时间）
- 根据 low 和 dfn 的关系判断割点

### 复杂度分析
- 时间：O(V + E)
- 空间：O(V + E)
