# UVA218 - Mouse Holes (老鼠洞)

## 题目描述

一只老鼠从起点出发，需要经过所有其他点后返回起点，求最短路径长度（四舍五入取整）。

## 输入格式

多组测试数据。每组第一行 n（含起点），接下来 n 行每行两个实数 x, y。n=0 结束。

## 输出格式

输出网络编号和所需的最短电线长度（四舍五入到整数）。

## 解题思路

### 问题分析
经典的旅行商问题（TSP），n ≤ 14。

### 核心思路
- 使用状压 DP 求解 TSP
- `dp[cur][mask]` 表示当前在点 cur，已访问 mask 中的点，到终点的最短距离
- 从起点出发，经过所有点后回到起点

### 算法流程
1. 计算所有点对距离
2. 记忆化搜索：`dp(cur, mask) = min(dp(i, mask|1<<i) + dist[cur][i])`
3. 终止条件：mask 全为1时返回到起点的距离

### 复杂度分析
- 时间复杂度：O(N² × 2^N)
- 空间复杂度：O(N × 2^N)
