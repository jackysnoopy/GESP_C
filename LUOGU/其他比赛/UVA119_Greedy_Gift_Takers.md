# UVA119 - Greedy Gift Takers

## 题目描述

一群朋友互相送礼。每个人有一份名单，列出要送礼的人。每人准备一定金额的礼物，平分给名单上的人。如果无法整除，多余的钱留给自己。

输入包括：
- 朋友数量
- 朋友名单
- 每人的送礼信息（送礼者、总金额、接收者数量和名单）

输出每人的净收支（收到的礼物减去送出的礼物）。

## 输入格式

- 第一行：朋友数量 n
- 第二行：n 个朋友的名字
- 后续若干行：每人送礼的信息

## 输出格式

- 每行一个名字和对应的净收支

## 样例输入

```
5
Dave Steven Dawn
Emily Rose
Dave 200 3 Steven Dawn Rose
Steven 500 2 Dave Dawn
```

## 样例输出

```
Dave 300
Steven 450
Dawn -200
Emily 0
Rose -500
```

## 解题思路

### 核心思想

**模拟**送礼过程，使用 map 记录每人的净收支。

### 算法步骤

1. 读入朋友名单，建立名字到编号的映射
2. 初始化每人净收支为 0
3. 对每个送礼者：
   - 读取总金额和接收者数量
   - 计算每人的份额 = 总金额 / 接收者数量
   - 送礼者减少 total 金额
   - 每个接收者增加 份额金额
4. 按原始顺序输出每人的净收支

### 关键点

- 整数除法，多余的钱留给送礼者
- 使用 map 维护名字到索引的映射
- 输出顺序与输入名单顺序一致

## 复杂度分析

- **时间复杂度**：O(n × k)，k 为送礼者数量
- **空间复杂度**：O(n)
