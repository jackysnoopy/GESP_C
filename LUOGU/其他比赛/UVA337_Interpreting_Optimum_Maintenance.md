# UVA337 - Inter-preting Optimum Maintenance

## 题目描述

给定 $n$ 个事件和 $m$ 个偏序约束（$A < B$ 表示事件 $A$ 必须在事件 $B$ 之前发生），求满足所有约束的合法排列数。

## 输入格式

多组测试数据。每组第一行两个整数 $n$（事件数）和 $m$（约束数），接下来 $n$ 行事件名，$m$ 行每行两个事件名（表示先后关系）。$n = 0$ 且 $m = 0$ 时结束。

## 输出格式

```
Case X: Y valid orderings
```

## 样例输入

```
3 2
A
B
C
A B
B C
0 0
```

## 样例输出

```
Case 1: 1 valid orderings
```

## 解题思路

1. 将事件名映射为编号
2. 约束转化为有向图中的边
3. 枚举所有排列，检查是否满足约束
4. 也可以用拓扑排序 + 计数的方法优化

## 时间复杂度

- 枚举排列：$O(n! \cdot m)$
- 使用拓扑排序可优化至 $O(n \cdot m + n!)$

## 空间复杂度

$O(n + m)$
