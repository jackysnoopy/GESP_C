# UVA319 - Predict the Winner (预测赢家)

## 题目描述
两个玩家从一排 n 个数的两端轮流取数。每个玩家每次只能从当前序列的左端或右端取一个数。两个玩家都采取最优策略。判断第一个玩家是否能赢（或平局）。

## 输入格式
第一行为 T（测试数据组数）。
每组第一行为 n（数字个数），接下来 n 个整数。

## 输出格式
每行输出 "First"、"Second" 或 "Tie"。

## 样例
```
输入：
2
3
1 5 3
3
1 5 100

输出：
First
Second
```

## 提示
- 区间DP / 博弈论
- dp[l][r] 表示在区间 [l,r] 上当前玩家能获得的最大净胜分
- 状态转移：dp[l][r] = max(a[l] - dp[l+1][r], a[r] - dp[l][r-1])

## 解题思路

### 问题分析
经典的两人零和博弈问题，双方都采取最优策略。

### 核心思路
- 定义 dp[l][r] 为当前玩家在区间 [l,r] 上的最优净胜分
- 转移方程：dp[l][r] = max(a[l] - dp[l+1][r], a[r] - dp[l][r-1])
- 基础情况：dp[i][i] = a[i]
- 最终 dp[0][n-1] > 0 则先手赢，< 0 则后手赢，= 0 则平局

### 复杂度分析
- 时间：O(n²)
- 空间：O(n²)
