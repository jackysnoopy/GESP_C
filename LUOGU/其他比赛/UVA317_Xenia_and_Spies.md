# UVA317 - Xenia and Spies (谢尼亚与间谍)

## 题目描述
有 n 个间谍排成一行，一些间谍向左传递消息，一些向右传递。每个间谍收到消息后会在限定距离内传给下一个间谍。求消息从第一个人传到最后一个人的最短时间。

## 输入格式
第一行为 T（测试数据组数）。
每组第一行为 n, m, L。
接下来 n 行，每行两个整数 x[i] 和 dir[i]，表示第 i 个间谍的位置和方向（0=左, 1=右）。

## 输出格式
每行输出最短传递时间。

## 样例
```
输入：
1
5 3 10
0 1
5 0
10 1
15 0
20 1

输出：
10
```

## 提示
- 贪心匹配：每个向左的间谍找最近的向右间谍
- 使用排序和二分优化

## 解题思路

### 问题分析
消息传递的最优化问题。

### 核心思路
- 按位置排序间谍
- 贪心匹配向左和向右的间谍
- 计算传递距离

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
