# UVA320 - Border（边界）

## 题意

给定一个50×50的网格，从起始坐标 (x, y) 出发，按照一系列方向（N/S/E/W）移动。需要输出网格，其中路径经过的格子标记为 `X`，空格子标记为 `.`。

## 解题思路

1. **模拟移动**：从起始点开始，依次按照方向字符串中的每个方向移动一步
2. **标记路径**：每移动一步，将起点和终点都标记为路径格子 `X`
3. **输出网格**：从 y=49 到 y=0 逐行输出，x 从 0 到 49

## 算法复杂度

- 时间复杂度：O(L)，L 为方向字符串长度
- 空间复杂度：O(1)（固定 50×50 网格）

## 注意事项

- 起始坐标是浮点数，需要取整作为网格坐标
- 输出格式要求每行之间无多余空格
