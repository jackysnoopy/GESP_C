# UVA240 - String Popping (字符串弹出)

## 题目描述

给定一个由大写字母组成的字符串。你可以反复执行以下操作：选择字符串中两个相邻且相同的字符，将它们从字符串中删除。目标是找到最少的字符数可以保留在字符串中（即无法继续消除时的字符数）。

注意：消除操作可能产生新的相邻相同字符对，需要反复尝试直到无法消除。

## 输入输出格式

**输入：** 第一行T表示测试用例数。每组测试数据给出一个字符串。

**输出：** 每组测试数据输出一行，表示最终剩余字符的最小数量。

## 思路分析

1. **贪心/模拟：** 由于消除顺序不影响最终结果（只要持续消除直到无法消除），可以直接模拟消除过程
2. **栈模拟：** 使用栈来模拟消除过程，遇到相同字符就弹出
3. **递归消除：** 消除后需要检查新产生的相邻字符对

## 复杂度分析

- 时间复杂度：O(n)，栈模拟一遍字符串
- 空间复杂度：O(n)，栈空间
