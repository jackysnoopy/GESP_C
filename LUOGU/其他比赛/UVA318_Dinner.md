# UVA318 - Dinner (晚餐)

## 题目描述
有 n 个男士和 m 位女士参加晚餐。每位男士有一个偏好值 a[i]，每位女士有一个偏好值 b[j]。如果 a[i] + b[j] <= 1000，则两人可以配对。求最多能配对多少对。

## 输入格式
第一行为 T（测试数据组数）。
每组第一行为 n, m。
接下来 n 个整数表示男士偏好值。
再 m 个整数表示女士偏好值。

## 输出格式
每行输出最大配对数。

## 样例
```
输入：
1
3 3
500 600 700
400 500 600

输出：
3
```

## 提示
- 二分图最大匹配
- 使用匈牙利算法或Dinic算法

## 解题思路

### 问题分析
经典的二分图最大匹配问题。

### 核心思路
- 建立二分图：男士为左部，女士为右部
- 如果 a[i] + b[j] <= 1000，则连边
- 使用匈牙利算法求最大匹配

### 复杂度分析
- 时间：O(n × m × n)
- 空间：O(n × m)
