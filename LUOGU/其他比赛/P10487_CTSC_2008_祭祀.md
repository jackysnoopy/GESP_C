# [CTSC 2008] 祭祀

## 题目描述
偏序集最长反链。

## 输入格式
偏序关系。

## 输出格式
最大集合大小。

## 样例
### 样例 1
```
3 1
1 2
1 3
```
```
2
```

## 解题思路
### 问题分析
本题要求在偏序集中找到最大的反链，即一个集合，其中任意两个元素之间都不可比较。根据 Dilworth 定理，偏序集的最长反链长度等于其最小链划分的大小。也可以转化为：最长反链长度 = 最长递增子序列中取反后的最长不增序列，即偏序集的宽度。

### 核心思路
1. 将偏序关系转化为 DAG 上的最长路径问题。对于给定的偏序关系 $a \le b$，建立从 $a$ 到 $b$ 的有向边。
2. 根据 Dilworth 定理，**最长反链长度 = 最小链覆盖数 = 最大匹配数的补**。
3. 构建二分图：将每个点拆成左右两部分，若存在偏序关系 $u \le v$ 则连边，求最大匹配。**答案 = $n - $ 最大匹配数**。
4. 对于 DAG 上的最小链覆盖，也可以通过传递闭包 + 二分图最大匹配求解。

### 算法流程
1. 读入 $n$ 个元素和偏序关系，建立 DAG
2. 求传递闭包（Floyd-Warshall 或 bitset 优化），确定所有可达关系
3. 建立二分图，对每条偏序关系 $(u, v)$ 连接左 $u$ 右 $v$
4. 求最大二分图匹配（匈牙利算法）
5. 输出 $n - $ 最大匹配数

### 复杂度分析
- 时间复杂度：$O(n^3)$（传递闭包）+ $O(n^2 \times m)$（匈牙利算法），其中 $n$ 为元素数，$m$ 为偏序关系数
- 空间复杂度：$O(n^2)$
