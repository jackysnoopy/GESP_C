# UVA304 - The Fixed Point Fixed Theorem（不动点定理）

## 题目描述

给定函数 f: {1,2,...,n} → {1,2,...,n}，求 f 反复应用后的不动点。即求最小的 t 使得 f^t(x) = f^{t+1}(x)。

## 关键思路

**核心观察：反复应用函数 f 会进入循环。**

对于有限集合上的函数，从任意点出发反复应用 f，最终都会进入一个循环。不动点就是循环的起点。

## 算法：Floyd判圈法

1. **第一阶段**：快指针走两步，慢指针走一步，直到相遇
2. **第二阶段**：一个指针回到起点，两个都走一步，再次相遇就是环起点（不动点）

## 代码说明

- `slow = f[slow]` 慢指针走一步
- `fast = f[f[fast]]` 快指针走两步
- 两次相遇间的距离就是环的起点

## 复杂度

- 时间：O(n)
- 空间：O(n)
