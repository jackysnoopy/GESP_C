# UVA142 - Mouse Clicks (鼠标点击)

## 题目描述

在 500×500 的屏幕上，有多个可点击的矩形区域。每个区域由左上角和右下角坐标定义，并带有标签。

对于每个鼠标点击位置，确定：
1. 点击了哪个区域（如果点击了多个区域，选择最后定义的）
2. 如果没有点击任何区域，输出 `Miss`

## 输入格式

区域定义：
```
R x1 y1 x2 y2 label
```
- `R` 表示矩形区域
- `(x1,y1)` 和 `(x2,y2)` 是对角坐标
- `label` 是区域标签（最多9个字符）

鼠标点击：
```
mx my
```
- 坐标为 -1 -1 时结束输入

## 输出格式

每个点击输出对应的区域标签或 `Miss`。

## 算法分析

1. **区域存储**：将所有区域存储在向量中
2. **点击检测**：从后往前检查每个区域
3. **边界处理**：点击在边界上也算点击成功

## 复杂度分析

- 时间复杂度：O(n)，n 为区域数量
- 空间复杂度：O(n)

## 示例输入

```
R 10 10 20 20 A
R 30 30 40 40 B
15 15
25 25
35 35
-1 -1
E
```

## 示例输出

```
A
Miss
B
```

## 代码说明

1. 使用 `Region` 结构体存储区域信息
2. 坐标交换确保 `x1 <= x2` 和 `y1 <= y2`
3. 从后往前遍历实现"后定义优先"
4. 点击坐标在区域内（含边界）即判定为命中
