# UVA221 - Urban Elevations (城市海拔)

## 题目描述

给定平面上若干矩形建筑物的左下角坐标、宽度（东西方向）、深度（南北方向）和高度。从南向北看（y 轴正方向为北），找出所有可见的建筑物编号。

## 输入格式

多组测试数据。每组第一行 n，接下来 n 行每行 5 个实数 x, y, w, d, h。n=0 结束。

## 输出格式

输出可见建筑物的编号（从1开始），按编号排序。

## 解题思路

### 问题分析
从南方观察，一个建筑物是否可见取决于其正面是否有被更高的建筑物遮挡。

### 核心思路
- 枚举每个建筑物的 x 范围内所有可能的投影区间
- 对每个 x 区间中点，检查是否有更高建筑物在前方遮挡
- 只要有一个子区间可见，该建筑物就可见

### 算法流程
1. 收集所有建筑物的 x 坐标和 x+w 坐标，排序去重
2. 对每个建筑物，检查每个 x 子区间的中点
3. 在该中点处，检查 y 方向（前方）是否有更高的建筑物遮挡

### 复杂度分析
- 时间复杂度：O(N³)
- 空间复杂度：O(N)
