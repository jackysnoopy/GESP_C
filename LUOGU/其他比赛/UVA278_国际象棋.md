# UVA278 - Chess (国际象棋)

## 题目描述

给定棋子类型和棋盘大小，计算最多可以放置多少个该棋子而不相互攻击。

## 输入格式

- 第一行包含一个整数T，表示测试用例数量
- 每个测试用例包含：
  - 一个字符c，表示棋子类型（r=车, Q=后, k=马, K=王）
  - 两个整数r c，表示棋盘大小

## 输出格式

对于每个测试用例，输出最多可以放置的棋子数量。

## 样例

### 输入
```
4
r 3 3
Q 3 3
k 3 3
K 3 3
```

### 输出
```
3
3
5
4
```

## 提示

- 棋子类型：
  - r：车（Rook），可以攻击同一行或同一列
  - Q：后（Queen），可以攻击同一行、同一列或对角线
  - k：马（Knight），攻击"L"形
  - K：王（King），攻击相邻的8个格子
- 1 ≤ r, c ≤ 1000

## 解题思路

### 问题分析

不同棋子的放置策略不同：

1. **车（r）**：每行每列最多放一个，答案为 min(r, c)
2. **后（Q）**：每行每列最多放一个，答案为 min(r, c)
3. **马（k）**：可以放在同色格子上，答案为 ⌈r×c/2⌉
4. **王（K）**：每2×2区域最多放一个，答案为 ⌈r/2⌉ × ⌈c/2⌉

### 核心思路

根据棋子类型使用不同的计算公式。

### 算法流程

1. 读取棋子类型和棋盘大小
2. 根据棋子类型计算答案
3. 输出结果

### 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        char piece;
        int r, c;
        cin >> piece >> r >> c;
        
        int ans = 0;
        switch (piece) {
            case 'r':
                ans = min(r, c);
                break;
            case 'Q':
                ans = min(r, c);
                break;
            case 'k':
                ans = (r * c + 1) / 2;
                break;
            case 'K':
                ans = ((r + 1) / 2) * ((c + 1) / 2);
                break;
        }
        cout << ans << "\n";
    }
    return 0;
}
```
