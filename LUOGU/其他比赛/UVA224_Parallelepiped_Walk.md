# UVA224 - Parallelepiped Walk (平行六面体行走)

## 题目描述

在一个三维空间中，一只蚂蚁在第一层平面上行走。如果它走到已经访问过的位置，就上升一层（z坐标+1），并回到原点重新开始。

## 输入格式

多组测试数据。每组第一行 n，接下来 n 行每行一个方向字符和步数。n=0 结束。

## 输出格式

输出最终位置的 x, y, z 坐标。

## 解题思路

### 问题分析
模拟蚂蚁在网格上的行走过程，遇到重复位置时升层。

### 核心思路
- 用 set 记录当前层访问过的位置
- 每次移动后检查是否重复
- 重复则 z++ 并清空当前位置回到原点

### 算法流程
1. 读入方向和步数，逐步移动
2. 每步检查当前位置是否已在 set 中
3. 若在，z++，清空 set，回到 (0,0,z)
4. 若不在，加入 set，继续移动

### 复杂度分析
- 时间复杂度：O(N × max_dist)
- 空间复杂度：O(N × max_dist)
