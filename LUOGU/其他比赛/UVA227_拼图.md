# UVA227 - Puzzle (拼图)

## 题目描述

5x5的拼图板上有25个格子，其中24个放着字母，1个为空格。通过移动相邻格子的字母到空格中来拼图。

命令：
- `A`：空格上方的格子下移到空格
- `B`：空格下方的格子上移到空格
- `L`：空格右方的格子左移到空格
- `R`：空格左方的格子右移到空格

输入多组拼图配置和移动序列，输出最终拼图。

## 输入格式

- 多组测试用例
- 每组5行：5x5的拼图（空格用空格表示）
- 之后一行：移动命令序列
- 以 `Z` 开头的行结束输入

## 输出格式

- 每组输出5行，表示最终拼图
- 每行字符间用空格分隔
- 如果移动非法，输出错误信息

## 解题思路

1. 找到空格位置
2. 按命令模拟移动
3. 检查移动是否合法（不越界）

## 注意事项

- 命令序列中可能包含换行符
- 需要处理输入中的空格
