# UVA326 - 斯特恩-布罗科特数系 (The Stern-Brocot Number System)

## 题意

给定一个正分数 p/q，求它在 Stern-Brocot 树中的表示。Stern-Brocot 树从 0/1 和 1/0 开始，不断取中值（mediant）来构建。输出用 L 和 R 表示的路径。

## 思路

Stern-Brocot 树的构建过程：
1. 初始左边界 a/b = 0/1，右边界 c/d = 1/0
2. 中值为 (a+c)/(b+d)
3. 如果目标分数小于中值，走左（L），更新右边界
4. 如果目标分数大于中值，走右（R），更新左边界
5. 直到找到目标分数

## 算法

1. 初始化左右边界
2. 重复比较目标分数与中值
3. 根据比较结果更新边界并记录路径
4. 输出结果

## 复杂度

- 时间复杂度：O(log(p+q))
- 空间复杂度：O(1)
