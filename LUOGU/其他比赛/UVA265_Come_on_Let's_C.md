# UVA265 Come on, Let's C

## 题目描述

本题要求在二维网格中统计特定字符的出现次数。

## 输入格式

第一行包含测试用例数量 T。
每个测试用例包含：
- 第一行：网格的行数 n 和列数 m
- 接下来 n 行：每行包含 m 个字符

## 输出格式

对于每个测试用例，输出字符 'C' 在网格中出现的总次数。

## 样例

输入：
```
2
3 4
ABCA
CDEF
GCHI
2 3
ABC
CDE
```

输出：
```
字符C的个数: 3
字符C的个数: 1
```

## 解题思路

1. 读取测试用例数量
2. 对每个测试用例，读取网格尺寸和内容
3. 遍历整个网格，统计字符 'C' 的出现次数
4. 输出结果

## 时间复杂度

O(n × m)，其中 n 和 m 分别为网格的行数和列数。
