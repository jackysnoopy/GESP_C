# UVA302 - John's Trip（约翰的旅行）

## 题目大意

John 在城市间旅行，每条路有编号。给定道路连接关系，求欧拉回路，按字典序输出道路编号。

## 算法思路

使用 **Hierholzer 算法** 求欧拉回路：

1. 检查是否存在欧拉回路（所有点度数为偶数）
2. 从任意点开始 DFS，每次走一条未用的边
3. 当无法继续前进时，将当前边编号加入路径
4. 最终逆序输出得到欧拉回路

**关键细节**：
- 需要按字典序遍历邻接表（边已按编号排序）
- Hierholzer 算法得到的是逆序路径，需要反转输出
- 处理多组输入，组间用空行分隔

## 时间复杂度

$O(E)$，其中 $E$ 为边数。

## 输入输出示例

```
输入：
1 2 1
2 3 2
3 1 6
1 2 5
2 3 3
3 1 4
0 0

输出：
1 2 3 6 5 4
```
