# UVA335 - Louvre

## 题目描述

卢浮宫的展览厅之间有走廊相连。给定 $n$ 个展厅和 $m$ 条走廊（每条走廊有长度），求任意两个展厅之间的最短距离。

## 输入格式

多组测试数据。每组第一行两个整数 $n$（展厅数）和 $m$（走廊数），接下来 $m$ 行每行三个整数 $u$、$v$、$w$（走廊连接的两个展厅和长度），然后一个整数 $q$（查询数），$q$ 行每行两个整数（起点和终点）。$n = m = 0$ 时结束。

## 输出格式

```
Network X
Minimum cost: Y
```

## 样例输入

```
3 2
1 2 5
2 3 7
2
1 3
2 1
0 0
```

## 样例输出

```
Network 1
Minimum cost: 12
Minimum cost: 5
```

## 解题思路

1. 使用 Floyd-Warshall 算法求所有点对最短路径
2. 初始化：$d[i][j] = 0$（$i = j$），$d[i][j] = \infty$（$i \ne j$）
3. 读入边后执行三重循环更新最短路径
4. 对每个查询输出结果

## 时间复杂度

$O(n^3)$（Floyd-Warshall）

## 空间复杂度

$O(n^2)$
