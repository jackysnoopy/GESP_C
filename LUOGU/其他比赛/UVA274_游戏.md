# UVA274 - The Game (游戏)

## 题目描述

判断一个由'U'和'D'组成的序列是否可能是一个有效的括号序列，其中'U'代表上括号，'D'代表下括号。

## 输入格式

- 多行，每行一个字符串，由'U'和'D'组成

## 输出格式

- 对于每行，输出"Possible"或"Impossible"

## 样例

### 输入
```
UD
UUDD
UDUD
DU
```

### 输出
```
Possible
Possible
Possible
Impossible
```

## 提示

- 字符串长度不超过1000
- 'U'和'D'的数量必须相等
- 任何时候'D'的数量不能超过'U'的数量

## 解题思路

### 问题分析

这是一个经典的括号匹配问题，类似于判断括号序列是否有效。

### 核心思路

1. 使用栈的思想，遇到'U'入栈，遇到'D'出栈
2. 任何时候栈不能为空（即'D'不能多于'U'）
3. 最后栈必须为空（'U'和'D'数量相等）

### 算法流程

1. 遍历字符串
2. 维护计数器，遇到'U'加1，遇到'D'减1
3. 如果计数器变为负数，则无效
4. 最后计数器必须为0

### 复杂度分析

- 时间复杂度：O(n)，n是字符串长度
- 空间复杂度：O(1)

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string line;
    while (getline(cin, line)) {
        int n = line.length();
        int up = 0, down = 0;
        bool possible = true;
        
        for (char c : line) {
            if (c == 'U') {
                up++;
            } else {
                down++;
            }
            if (down > up) {
                possible = false;
                break;
            }
        }
        
        if (up != down) possible = false;
        
        if (possible) {
            cout << "Possible\n";
        } else {
            cout << "Impossible\n";
        }
    }
    return 0;
}
```
