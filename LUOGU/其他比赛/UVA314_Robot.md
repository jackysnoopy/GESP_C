# UVA314 - Robot (机器人)

## 题目描述
一个机器人在一个 R×C 的网格中，可以执行三种操作：向左转、向右转、前进1-3步（不能穿过障碍物）。给定起点和终点，求从起点到终点的最少操作次数。

## 输入格式
多组测试数据，每组第一行为 R C，R=C=0 表示结束。
接下来 R 行为网格地图，'0' 为空地，'1' 为障碍物。
最后一行为起点坐标、方向（1=北, 2=东, 3=南, 4=西）和终点坐标。

## 输出格式
每行输出最少操作次数，无法到达输出 -1。

## 样例
```
输入：
5 5
00000
00000
00000
00000
00000
1 1 1 3 5
0 0

输出：
8
```

## 提示
- BFS 状态扩展：(x, y, direction)
- 三种操作：左转(+1代价)、右转(+1代价)、前进1-3步(+1代价)
- 注意前进不能穿过障碍物

## 解题思路

### 问题分析
机器人在网格中移动，需要同时考虑位置和朝向。

### 核心思路
- 状态：(x, y, dir) 表示位置和朝向
- BFS扩展三种操作：左转、右转、前进
- 前进最多3步，遇到障碍物停止

### 复杂度分析
- 时间：O(R × C × 4)
- 空间：O(R × C × 4)
