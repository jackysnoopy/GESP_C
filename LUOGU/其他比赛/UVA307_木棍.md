# UVA307 - Sticks (木棍)

## 题目大意

乔治有一些等长的木棍，每根被切成若干段。给定所有小段的长度，求原始木棍的**最短可能长度**和对应的**最少数量**。

例如：木棍长度为 10，被切成 [2, 4, 5, 7] 四段，则原始长度为 10，数量为 1（假设这些段来自同一根）。

## 算法思路

**回溯 + 剪枝**：

1. **枚举原始长度**：从最长的段开始枚举，原始长度必须能整除总长度。
2. **回溯尝试还原**：
   - 按从大到小的顺序尝试用小段组成原始长度的木棍
   - 使用 `used[]` 标记已使用的小段
3. **关键剪枝**：
   - **排序剪枝**：将小段从大到小排序，优先用大的段
   - **去重剪枝**：相同长度的段跳过
   - **首段剪枝**：如果当前木棍的第一段就失败，直接返回
   - **末段剪枝**：如果加上当前段恰好凑满但后续失败，直接返回（因为换一种组合也无意义）
   - **空木棍剪枝**：如果从空木棍开始就失败，说明后续也无法成功

## 复杂度

- 最坏情况指数级，但剪枝效果极好

## 关键点

- 从大到小排序 + 去重是核心优化
- 多种剪枝策略配合使用，大幅减少搜索空间
