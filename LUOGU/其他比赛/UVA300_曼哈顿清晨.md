# UVA300 - Manhattan Mornings（曼哈顿清晨）

## 题目大意

小红在曼哈顿送信。给定 $n$ 封信的起始坐标和目标坐标，她只能向东（$x$ 增大）或向南（$y$ 减小）走。求她最少需要走多少步才能把所有信送完。

## 算法思路

由于小红只能向东或向南走，所以一封信能被送达的条件是：起点 $(x_1, y_1)$ 在终点 $(x_2, y_2)$ 的左上方，即 $x_1 \le x_2$ 且 $y_1 \ge y_2$。

我们将信按起点的 $x$ 坐标排序（$x$ 相同时按 $y$ 降序），然后对终点的 $y$ 坐标求 **最长递增子序列（LIS）**。

**关键结论**：最少步数 = $n - \text{LIS长度}$

## 时间复杂度

排序 $O(n \log n)$，LIS $O(n \log n)$，总复杂度 $O(n \log n)$。

## 输入输出示例

```
输入：
5
0 4 4 0
2 3 3 1
3 2 5 0
3 3 4 2
1 5 5 3

输出：
2
```
