# UVA140 - Bandwidth (带宽)

## 题目描述

给定一个无向图，顶点标记为 A-Z（最多8个顶点）。找出一个顶点排列，使得**带宽**最小。

**带宽定义**：排列中任意相邻顶点在原图中的最大距离。

## 输入格式

多行输入，每行描述一个图：
- 格式：`顶点A -> 邻接顶点;顶点B -> 邻接顶点;...`
- 输入以 `#` 结尾

## 输出格式

输出最小带宽对应的排列和带宽值。

## 算法分析

1. **回溯法**：枚举所有可能的顶点排列
2. **剪枝优化**：当当前带宽超过已知最优解时剪枝
3. **位运算优化**：使用位掩码加速邻接关系查询

## 复杂度分析

- 时间复杂度：O(n!)，最坏情况需要枚举所有排列
- 空间复杂度：O(n)

## 示例输入

```
A: FB;B:GC;C:GD;D:FE;E:G;F:G;G:ABDEF;
#
```

## 示例输出

```
E G D F B C A -> 2
```

## 代码说明

1. 使用 `adj[26][26]` 存储邻接矩阵
2. `pos[26]` 记录每个顶点在排列中的位置
3. `solve()` 函数递归构建排列并计算带宽
4. 剪枝条件：当新增带宽超过最优解时停止搜索
