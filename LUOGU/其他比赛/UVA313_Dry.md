# UVA313 - Dry (干燥)

## 题目描述
给定长方体的长、宽、高，计算其体积与表面积之和，结果取整输出。

## 输入格式
多组测试数据，每组四个浮点数 L W D H（长、宽、深、高），全0表示结束。

## 输出格式
每行输出体积与表面积之和（四舍五入取整）。

## 样例
```
输入：
3.0 5.0 4.0 2.0
0 0 0 0

输出：
116
```

## 提示
- 体积 = L × W × D
- 表面积 = 2(LW + LD + WD)
- 结果四舍五入取整

## 解题思路

### 问题分析
简单的几何计算题。

### 核心思路
- 计算长方体体积和表面积
- 相加后四舍五入

### 复杂度分析
- 时间：O(1)
- 空间：O(1)
