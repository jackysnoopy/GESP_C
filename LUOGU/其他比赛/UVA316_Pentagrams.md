# UVA316 - Pentagrams (五角星)

## 题目描述
给定 n 个点，求能组成的五角星（五元组）的数量。即从 n 个点中选择 5 个点的方案数。

## 输入格式
多组测试数据，每行一个整数 n（点数），n=0 结束。

## 输出格式
每行输出组合数 C(n,5)。

## 样例
```
输入：
5
6
0

输出：
1
6
```

## 提示
- 组合数 C(n,5) = n! / (5! × (n-5)!)
- 注意数据范围，使用 long long

## 解题思路

### 问题分析
简单的组合数学问题。

### 核心思路
- 计算 C(n,5) = n*(n-1)*(n-2)*(n-3)*(n-4) / 120
- 使用 long long 防止溢出

### 复杂度分析
- 时间：O(1)
- 空间：O(1)
