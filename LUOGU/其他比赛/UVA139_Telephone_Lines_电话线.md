# UVA139 - Telephone Lines (电话线)

## 题目描述

给定电话号码列表，需要格式化这些号码。

电话号码有两种格式：
1. **本地号码**：7位数字
2. **长途号码**：带区号前缀，格式为 `+区号-本地号码` 或 `(区号)本地号码`

## 格式化规则

1. 如果号码长度为7位且没有区号前缀，它是本地号码，格式为 `+本地号码`
2. 如果号码有区号前缀（`+`或`()`），格式化为 `+区号-本地号码`

## 输入格式

多行输入，每行包含：
- 用户姓名（字符串）
- 电话号码

输入以 `#` 结尾。

## 输出格式

按用户姓名字典序排序，输出格式化后的号码。

## 算法分析

1. **字符串解析**：解析输入的姓名和号码
2. **号码格式化**：识别区号前缀，提取本地号码和区号
3. **排序**：按姓名字典序排序

## 复杂度分析

- 时间复杂度：O(n log n)，其中 n 是记录数
- 空间复杂度：O(n)

## 示例输入

```
Smith +1-716-1234567
Johnson (212) 555-1234
Williams 5551234
#
```

## 示例输出

```
Johnson,+2125551234
Smith,+17161234567
Williams,+5551234
```

## 代码说明

1. 使用 `formatNumber()` 函数解析和格式化号码
2. 检测区号前缀（`+`或`()`）
3. 提取纯数字部分验证长度
4. 按姓名排序后输出
