# UVA330 - Inventory Maintenance

## 题目描述

实现一个库存管理系统，支持以下三种操作：

- **BUY** `item` `price`：以给定价格购入商品，加入库存
- **SELL** `item` `price`：以给定价格卖出商品（商品必须在库存中），记录利润（卖出价-买入价）
- **LIST**：按字母顺序列出当前库存中的所有商品
- **#**：结束输入

输入结束后，先输出所有已卖出商品及其利润，再输出总利润。

## 输入格式

多行命令，每行一个命令，以 `#` 结尾。

- `BUY item price`：购入商品
- `SELL item price`：卖出商品
- `LIST`：列出库存

## 输出格式

1. 对于每个 `LIST` 命令，按字典序输出库存中的商品名（每行一个）
2. 输入结束后输出：

```
--- PROFIT ---
商品名 利润
...
--- TOTAL ---
总利润
```

## 样例输入

```
BUY A 100
BUY B 200
LIST
SELL B 350
LIST
SELL A 80
#
```

## 样例输出

```
A
B
--- PROFIT ---
B 150
--- TOTAL ---
150
A -20
--- TOTAL ---
130
```

## 解题思路

1. 使用 `map<string, int>` 存储当前库存（商品名→买入价格）
2. 使用 `vector` 记录所有卖出操作的记录
3. BUY：将商品加入库存
4. SELL：查找商品，计算利润，从库存移除
5. LIST：遍历 map 按字典序输出
6. 最后汇总输出所有卖出记录和总利润

## 时间复杂度

- 每次操作 $O(\log n)$，其中 $n$ 为库存大小
- 总时间复杂度 $O(m \log n)$，$m$ 为操作次数

## 空间复杂度

$O(n)$，用于存储库存和卖出记录。
