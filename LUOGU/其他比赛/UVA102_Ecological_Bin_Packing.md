# UVA102 - Ecological Bin Packing

## 题目描述

有三个箱子，每个箱子装有三种颜色的瓶子：棕色（B）、透明（C）和绿色（G）。你需要决定每个箱子最终应该装哪种颜色的瓶子，使得移动瓶子的总数最少。

对于每种颜色，只能有一个箱子装该颜色的瓶子。求最小移动次数，并输出对应的颜色排列（字典序最小）。

## 输入格式

每行包含9个整数，表示三个箱子中各颜色瓶子的数量：

```
b1 c1 g1 b2 c2 g2 b3 c3 g3
```

其中 `bi`、`ci`、`gi` 分别表示第 i 个箱子中棕色、透明、绿色瓶子的数量。

输入以 EOF 结束。

## 输出格式

对于每组输入，输出一行：

```
<排列> <最小移动次数>
```

其中排列由三个字母组成，表示第1、2、3个箱子分别装什么颜色的瓶子。若有多个最优排列，输出字典序最小的。

## 样例输入

```
1 2 3 4 5 6 7 8 9
5 10 5 20 100 20 1 1 100
```

## 样例输出

```
BCG 30
BCG 57
```

## 解题思路

1. **枚举排列**：3个箱子放3种颜色，共有 3! = 6 种排列方式
2. **计算移动数**：对于每种排列，移动数 = 总瓶数 - 已在正确位置的瓶子数
3. **选择最优**：选择移动次数最少的排列，若有平局选字典序最小的

### 关键观察

不需要真的移动瓶子，只需要计算"不需要移动"的瓶子数。对于排列 `perm`，如果第 i 个箱子放颜色 `perm[i]`，那么第 i 个箱子中颜色为 `perm[i]` 的瓶子不需要移动。

## 复杂度分析

- **时间复杂度**：O(1)，固定6种排列
- **空间复杂度**：O(1)
