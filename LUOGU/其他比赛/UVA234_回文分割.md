# UVA234 - Partitioning by Palindromes (回文分割)

## 题目描述

将一个字符串分割成最少的子串，使得每个子串都是回文串。

## 输入格式

- 多组测试用例
- 每组第一行：单词数 N
- 接下来 N 行：每个单词
- N=0 结束

## 输出格式

- 输出最少分割次数

## 解题思路

1. 将所有单词拼接成一个字符串
2. 使用动态规划：`dp[i]` 表示前 i 个字符最少需要分割几次
3. 对每个位置 i，尝试所有 j < i，如果 s[j+1..i] 是回文，则 `dp[i] = min(dp[i], dp[j] + 1)`

## 复杂度

O(n²) 检查回文 + O(n²) 动态规划

## 注意事项

- 预处理回文信息可以优化到 O(n)
- 单个字符本身也是回文
