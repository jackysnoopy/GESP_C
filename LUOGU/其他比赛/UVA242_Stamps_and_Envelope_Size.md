# UVA242 - Stamps and Envelope Size (邮票与信封)

## 题目描述

给定最多可以使用的邮票数量K和需要覆盖的数值范围1..N。邮票有面值，每次贴邮票的总面值等于邮票张数乘以面值之和。目标是选择一组邮票面值，使得用不超过K张邮票可以覆盖1到N之间的所有整数值，且覆盖范围尽可能大。

## 输入输出格式

**输入：** 多组测试数据。每组先给出K（最多邮票数），再给出N（需要覆盖的上限）。

**输出：** 每组输出最大覆盖范围和对应的邮票面值（从大到小排列）。

## 思路分析

1. **贪心枚举：** 枚举第一张邮票的面值，然后贪心地选择后续邮票面值
2. **DP验证：** 对每组邮票面值，用DP计算能覆盖的最大范围
3. **优先级：** 当覆盖范围相同时，选择邮票张数更少的方案

## 复杂度分析

- 时间复杂度：O(N * K^2 * N) 但N较小可接受
- 空间复杂度：O(N)
