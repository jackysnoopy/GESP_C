# UVA338 - Long Multiplication

## 题目描述

给定两个正整数，模拟竖式长乘法的过程，展示每一步的部分积和最终结果。

## 输入格式

多组测试数据，每行两个正整数。以 EOF 结束。

## 输出格式

右对齐输出，第一行为第一个部分积，第二行为被乘数右对齐，然后画分隔线，最后输出结果。两组数据之间空一行。

## 样例输入

```
123 45
999 9
```

## 样例输出

```
  123
   45
-----
  615
 492
-----
5535

  999
    9
-----
 8991
-----
8991
```

## 解题思路

1. 对乘数的每一位（从最低位开始），与被乘数相乘，得到部分积
2. 每个部分积后面补相应数量的 0（表示位移）
3. 将所有部分积相加得到最终结果
4. 使用字符串模拟大数加法和乘法

## 时间复杂度

$O(n \cdot m)$，其中 $n$、$m$ 分别为两个数的位数

## 空间复杂度

$O(n + m)$
