# UVA109 - Scramble Sort

## 题目描述

给定一个由数字和字符串交错组成的列表，按以下规则排序：
- 数字按**递增顺序**排列
- 字符串按**字典序**排列
- **位置模式保持不变**：原来哪个位置是数字的，排序后仍然是数字；原来哪个位置是字符串的，排序后仍然是字符串

## 输入格式

多组测试数据，每组一行，包含若干个用逗号分隔的值（数字或字符串），值之间可能有空格。

输入以 EOF 结束。

## 输出格式

对每组数据输出一行，按上述规则排序后的结果，用逗号分隔。

## 样例输入

```
1, 2, 3, B, A, 4
100, -1000, A, 33, -3, 9876, Z
```

## 样例输出

```
1, 2, 3, A, B, 4
-1000, -3, A, 33, 100, 9876, Z
```

## 解题思路

### 核心思想

**分离 → 排序 → 合并**

1. **解析输入**：按逗号分割字符串，得到每个元素
2. **判断类型**：判断每个元素是数字还是字符串
3. **分离存储**：将数字和字符串分别存入两个数组
4. **独立排序**：数字升序排序，字符串按字典序排序
5. **按模式合并**：遍历原数组的类型标记，依次从对应排序数组中取出元素

### 关键点

- 需要正确解析逗号分隔的输入，注意空格处理
- 判断数字时需考虑负数和小数点的情况
- 合并时使用双指针分别跟踪数字和字符串的当前取值位置

## 复杂度分析

- **时间复杂度**：O(n log n)，主要由排序决定
- **空间复杂度**：O(n)，存储分离后的数字和字符串

## 代码思路

```cpp
// 1. 解析输入，按逗号分割
// 2. 遍历每个元素，判断是数字还是字符串
// 3. 分别排序
// 4. 按原位置模式合并输出
```
