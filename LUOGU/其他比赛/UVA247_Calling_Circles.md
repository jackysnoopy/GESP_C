# UVA247 - Calling Circles (通话圈)

## 题目描述

给定一组人之间的通话记录（有向图中的边），找出所有的"通话圈"。一个通话圈是指一组人A,B,C,...,使得A打给B，B打给C，...，最后某人打给A。也就是说，圈中的每对人都互相可以到达对方。

## 输入输出格式

**输入：** 多组测试数据。每组先给出n（人名数）和m（通话记录数），接下来m行每行两个名字，表示一次通话。n=0且m=0时结束。

**输出：** 每组输出"Calling circles for data set X:"，然后每行输出一个圈的所有成员名（按字母排序），用逗号分隔。

## 思路分析

1. **强连通分量：** 通话圈等价于有向图的强连通分量
2. **Floyd-Warshall：** 由于人数较少（≤25），可以用Floyd-Warshall计算传递闭包
3. **找SCC：** 如果两个人i和j满足 i→j 且 j→i，则它们在同一个SCC中
4. **排序输出：** 每个SCC内的名字按字母排序

## 复杂度分析

- 时间复杂度：O(n³)，Floyd-Warshall
- 空间复杂度：O(n²)
