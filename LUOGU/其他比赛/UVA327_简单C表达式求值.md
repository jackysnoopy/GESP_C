# UVA327 - 简单C表达式求值 (Evaluating Simple C Expressions)

## 题意

计算一个简单的 C 语言表达式，支持变量（a-z，按位置赋值 1-26）、运算符（+、-、*、/）、括号，以及前置/后置 ++/--。

## 思路

使用递归下降解析器处理表达式：
1. 解析优先级最低的加减运算（`+`、`-`）
2. 解析乘除运算（`*`、`/`）
3. 解析基本元素（变量、数字、括号）
4. 处理前置/后置 ++/--

## 算法

1. 读入表达式
2. 初始化变量值
3. 使用递归下降解析器计算表达式
4. 输出结果

## 复杂度

- 时间复杂度：O(n)，n 为表达式长度
- 空间复杂度：O(1)
