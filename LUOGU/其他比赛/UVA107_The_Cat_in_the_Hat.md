# UVA107 - The Cat in the Hat

## 题目描述

戴帽子的猫（主猫）下面有若干个工人猫，每个工人猫又有相同数量的工人猫，如此递归。最底层的猫没有工人（不戴帽子）。

给定：
- N：不含主猫的工人猫数量（即有下属的猫的数量，不含主猫本身）
- H：工作树的高度（步数）

求：
1. 没有工人的猫的数量（底层猫）
2. 总工作量（第 i 层的猫贡献 i 单位工作量）

## 输入格式

每行两个整数 N 和 H，以 `0 0` 结束。

## 输出格式

每行两个整数：
- 第一个数：没有工人的猫的数量
- 第二个数：总工作量

## 样例输入

```
21 2
6 3
2 2
1 1
0 0
```

## 样例输出

```
0 0
8 34
4 10
1 1
```

## 解题思路

### 核心思想

本题是等比数列求和问题。猫的层级结构形成一棵树：

- 第 0 层：1 只猫（主猫）
- 第 1 层：W 只猫
- 第 2 层：W² 只猫
- ...
- 第 H-1 层：W^(H-1) 只猫（有工人的最底层）
- 第 H 层：W^H 只猫（没有工人的底层猫）

### 数学关系

**等比数列求和公式**：
$$N = W + W^2 + \cdots + W^{H-1} = \frac{W(W^{H-1} - 1)}{W - 1} \quad (W > 1)$$

**底层猫数**：
$$\text{nonWorkingCats} = W^H$$

**总工作量**：
$$\text{totalWork} = \sum_{i=1}^{H} i \cdot W^i$$

### 算法步骤

1. **二分搜索求 W**：在 [1, N] 范围内二分搜索 W，使得等比数列之和等于 N
2. **计算底层猫数**：W^H
3. **计算总工作量**：逐层累加 i × W^i

### 关键点

- 当 H=1 时，N=0（没有工人猫），直接输出对应结果
- 二分搜索时注意溢出处理（使用 long long）
- 边界条件：当 N=0 且 H=0 时终止

## 复杂度分析

- **时间复杂度**：O(H × log N)，二分搜索 O(log N)，每次验证需要 O(H) 计算等比数列
- **空间复杂度**：O(1)
