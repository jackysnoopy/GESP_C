# 『MdOI R1』Decrease

## 题目描述
给定一个 $n \times n$ 的矩阵，你可以进行若干次操作。

每次操作，你可以将一个 $k \times k$ 的 **连续** 子矩阵里的所有数全都加上 $1$ 或者全都减去 $1$。

初始时，矩阵中有 $m$ 个位置上的数不为 $0$，其它位置上的数均为 $0$。

请你求出至少需要多少次操作，可以将矩形中所有数都变为 $0$。

## 输入格式
第一行三个整数 $n,m,k$，分别表示矩阵大小，非 $0$ 格数和每次修改的连续子矩阵大小。

接下来 $m$ 行，每行三个整数 $x,y,z$，表示初始时矩阵的第 $x$ 行第 $y$ 列上的数为 $z$。

## 输出格式
一行一个整数，表示最少操作次数。

特别地，如果无法使矩阵中所有数都变为 $0$，输出 `-1`。

## 样例
**样例 1**
- 输入：
```
4 14 3
1 1 1
1 2 1
1 3 1
2 1 1
2 2 3
2 3 3
2 4 2
3 1 1
3 2 3
3 3 3
3 4 2
4 2 2
4 3 2
4 4 2
```
- 输出：
```
3
```

**样例 2**
- 输入：
```
3 1 2
1 1 1
```
- 输出：
```
-1
```

**样例 3**
- 输入：
```
4 5 1
1 1 5
2 2 -3
2 3 -4
3 3 1
4 4 2
```
- 输出：
```
15
```

## 提示
**【样例 1 解释】**:  

给出的矩阵为：  

```
1 1 1 0
1 3 3 2
1 3 3 2
0 2 2 2
```

具体步骤：  

先将以第一行第一列为左上角的连续子矩阵执行 **减 1 操作** 一次；

再将以第二行第二列为左上角的连续子矩阵执行 **减 1 操作** 两次。

总共三次。

**【样例 2 解释】**：  

给出的矩阵为：  

```
1 0 0
0 0 0
0 0 0
```

只通过 $2\times 2$ 的连续子矩阵操作不可能使得所有格子上的数都变为 $0$。

**【数据范围】**   

**本题采用捆绑测试。**

| 子任务编号 |    $n\leq$     | $k\leq$ | 分值 |
| :--------: | :------------: | :-----: | :--: |
|     1      |     $10^3$     |   $1$   |  11  |
|     2      |      $20$      |  $20$   |  14  |

|     4      |     $10^3$     | $10^3$  |  34  |

数据保证如果有解，答案不超过 $2^{63}-1$。

本题读入量较大，建议使用较快的读入方式。

## 解题思路

### 问题分析
需要通过 $k \times k$ 子矩阵的加减操作，将整个矩阵变为全 0。这是一个二维差分问题。

### 核心思路
从左上角到右下角贪心处理。对于每个非零位置 $(i, j)$，如果其值不为 0，则需要在 $(i, j)$ 位置进行操作。操作的影响范围是 $(i, j)$ 到 $(i+k-1, j+k-1)$。使用二维差分数组来高效处理区间操作。

### 算法流程
1. 读入矩阵，构建二维数组
2. 计算二维差分数组
3. 从左上角到右下角遍历每个位置：
   - 如果当前位置的差分值不为 0，说明需要进行操作
   - 记录操作次数（绝对值）
   - 在差分数组上进行相应的更新
4. 检查是否有解（某些位置可能无法被 $k \times k$ 子矩阵覆盖）
5. 输出答案

### 复杂度分析
- 时间复杂度：$O(n^2 + m)$
- 空间复杂度：$O(n^2)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m, k;
    cin >> n >> m >> k;
    vector<vector<long long>> a(n + 3, vector<long long>(n + 3, 0));
    while (m--) {
        int x, y, v;
        cin >> x >> y >> v;
        a[x][y] += v;
    }
    vector<vector<long long>> d(n + 3, vector<long long>(n + 3, 0));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            d[i][j] = a[i][j] - a[i-1][j] - a[i][j-1] + a[i-1][j-1];
        }
    }
    long long ans = 0;
    for (int i = 1; i + k - 1 <= n; i++) {
        for (int j = 1; j + k - 1 <= n; j++) {
            d[i][j] += d[i-1][j] + d[i][j-1] - d[i-1][j-1];
            if (d[i][j] != 0) {
                long long val = d[i][j];
                ans += abs(val);
                d[i][j] -= val;
                d[i][j+k] += val;
                d[i+k][j] += val;
                d[i+k][j+k] -= val;
            }
        }
    }
    cout << ans << "\n";
    return 0;
}
```
