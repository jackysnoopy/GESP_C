# UVA118 - Mutant Flatworld Explorers

## 题目描述

在一个矩形网格世界中，机器人从指定位置出发，按照给定的指令移动。网格有边界，如果机器人试图移出边界：

1. 如果该位置之前有机器人掉落并留下了"气味"，则忽略这条指令
2. 否则，机器人掉落出边界，该位置留下气味，机器人丢失

方向使用 N（北）、S（南）、E（东）、W（西）表示。R 表示右转 90°，L 表示左转 90°，F 表示向前移动一格。

## 输入格式

- 第一行：网格的宽和高
- 后续若干组：机器人的起始位置、朝向和指令序列

## 输出格式

- 输出机器人的最终位置和朝向
- 如果机器人掉落出界，额外输出 "LOST"

## 样例输入

```
5 3
1 1 E
RFRFRFRF
3 2 N
FRRFLLFFRRFLL
0 3 W
LLFFFLFLFL
```

## 样例输出

```
1 1 E
3 3 N LOST
2 3 S
```

## 解题思路

### 核心思想

**网格模拟**：按指令移动机器人，处理边界和气味标记。

### 算法步骤

1. 初始化方向数组：E→S→W→N（顺时针）
2. 对每条指令：
   - R：方向右转
   - L：方向左转
   - F：尝试向前移动
3. 如果向前移动会越界：
   - 检查当前格是否有气味标记
   - 有：忽略该指令
   - 无：标记该格有气味，机器人丢失

### 关键点

- 方向转换使用模运算
- 气味标记使用 set 存储
- 一旦丢失，后续指令全部忽略

## 复杂度分析

- **时间复杂度**：O(n)，n 为指令长度
- **空间复杂度**：O(n)，存储气味标记
