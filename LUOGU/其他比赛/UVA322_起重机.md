# UVA322 - Crane（起重机）

## 题意

给定 n 个堆栈，每个堆栈中放有编号为 1 到 n 的箱子。起重机每次只能从某个堆栈的顶部取箱子。目标是按顺序取出编号 1, 2, 3, ..., n 的箱子。判断是否可能并输出取出顺序。

## 解题思路

1. **贪心模拟**：按目标顺序 1 到 n 依次处理
2. **查找箱子**：对于每个目标编号，遍历所有堆栈，找到顶部为目标编号的堆栈
3. **取出箱子**：找到后从该堆栈弹出顶部
4. **判断失败**：如果找不到目标编号的箱子，输出 "No"

## 算法复杂度

- 时间复杂度：O(n²)，最坏情况下需要遍历所有堆栈
- 空间复杂度：O(n²)，存储堆栈数据

## 注意事项

- 堆栈是后进先出（LIFO）结构
- 箱子编号为 1 到 n
- 每个堆栈的输入第一行是该堆栈中箱子的数量
