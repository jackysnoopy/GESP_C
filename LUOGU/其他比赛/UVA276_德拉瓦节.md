# UVA276 - Drava Festival (德拉瓦节)

## 题目描述

给定n个城市和m条道路，找出任意两个城市之间的最短距离的最大值。

## 输入格式

- 多组测试用例
- 每组第一行包含两个整数n m，分别表示城市数量和道路数量
- 接下来m行，每行三个整数u v w，表示城市u和v之间有一条长度为w的道路
- n=0表示结束

## 输出格式

对于每组测试用例，输出任意两个城市之间的最短距离的最大值。

## 样例

### 输入
```
3 2
1 2 10
2 3 20
4 5
1 2 5
2 3 10
3 4 5
4 5 10
0
```

### 输出
```
30
20
```

## 提示

- 1 ≤ n ≤ 100
- 1 ≤ m ≤ 5000
- 道路是双向的

## 解题思路

### 问题分析

这是一个经典的全源最短路径问题，需要求所有点对之间的最短路径，然后取最大值。

### 核心思路

使用Floyd-Warshall算法求所有点对之间的最短路径，然后取最大值。

### 算法流程

1. 初始化距离矩阵
2. 使用Floyd-Warshall算法求所有点对最短路径
3. 遍历所有点对，取最大距离

### 复杂度分析

- 时间复杂度：O(n³)
- 空间复杂度：O(n²)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cstring>
#include <algorithm>
using namespace std;

const int MAXN = 105;
const int INF = 0x3f3f3f3f;

int n, m;
int dist[MAXN][MAXN];

void floyd() {
    for (int k = 1; k <= n; k++) {
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= n; j++) {
                if (dist[i][k] + dist[k][j] < dist[i][j]) {
                    dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }
    }
}

int main() {
    while (cin >> n >> m && n) {
        memset(dist, 0x3f, sizeof(dist));
        for (int i = 1; i <= n; i++) dist[i][i] = 0;
        
        for (int i = 0; i < m; i++) {
            int u, v, w;
            cin >> u >> v >> w;
            dist[u][v] = dist[v][u] = w;
        }
        
        floyd();
        
        int ans = 0;
        for (int i = 1; i <= n; i++) {
            for (int j = i + 1; j <= n; j++) {
                ans = max(ans, dist[i][j]);
            }
        }
        cout << ans << "\n";
    }
    return 0;
}
```
