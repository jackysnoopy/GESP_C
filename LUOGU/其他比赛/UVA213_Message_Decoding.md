# UVA213 - Message Decoding (消息解码)

## 题目描述

一种压缩编码方案：首先给出一个字符集头部（包含所有可能出现的字符），然后给出编码后的二进制消息。消息由若干段组成，每段以3位长度前缀开头，然后是若干等长的二进制编码。编码0表示新段开始（长度可能变化），全1编码保留不用。

## 输入格式

第一行：字符集头部字符串
之后交替出现：消息长度（字符串形式）和二进制消息

## 输出格式

解码后的字符串

## 解题思路

### 核心思路
1. 读入头部字符表
2. 每段开始读3位得到编码位数 L
3. 读 L 位得到编码值 val
4. val=0 时进入下一段
5. val 全为1时该段结束
6. 将 val 映射到头部字符表的索引输出

### 复杂度分析
- 时间复杂度：O(N)，N为消息总长度
- 空间复杂度：O(1)
