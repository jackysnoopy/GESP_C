# UVA303 - Ants（蚂蚁）

## 题目描述

n只蚂蚁在长度为L的木棍上爬行，速度为1单位/秒。当两只蚂蚁相遇时，它们会同时掉头继续爬行。求每只蚂蚁的最终位置。

## 关键思路

**核心观察：相遇掉头等价于互相穿过。**

当两只蚂蚁相遇并掉头时，由于所有蚂蚁速度相同，我们可以将每只蚂蚁的标签互换。这样从宏观上看，蚂蚁就像是互相穿过一样继续前进。

## 算法步骤

1. 记录每只蚂蚁的初始位置和方向
2. 将向右走和向左走的蚂蚁分别按位置排序
3. 从左穿过的蚂蚁位置就是排序后对应位置
4. 从右穿过的蚂蚁位置需要映射到左边

## 代码说明

- 用 `dir` 字段记录方向（1右，-1左）
- 分别排序左右方向的蚂蚁
- 合并结果输出

## 复杂度

- 时间：O(n log n)
- 空间：O(n)
