# UVA161 - Traffic Lights (交通灯)

## 题目描述

给定起始时间 h:m:s，计算从该时刻起，在5小时内所有交通灯同时为绿灯的次数。交通灯每2秒切换一次颜色（绿→黄→红→绿...），周期为14秒。

## 输入格式

每行三个整数 h m s，以 `0 0 0` 结尾。

## 输出格式

输出同时为绿灯的次数。

## 样例

略。

## 解题思路

### 问题分析

交通灯周期为14秒（绿2秒+黄2秒+红10秒），需要找到从起始时间开始5小时内所有同时为绿灯的时刻。

### 核心思路

1. 将起始时间转换为秒
2. 遍历5小时内的每一秒
3. 检查该秒是否满足条件

### 复杂度分析

- 时间复杂度：O(5 * 3600) = O(1)
- 空间复杂度：O(1)

## 参考代码

```cpp
#include <iostream>
#include <cstdio>
using namespace std;

int main() {
    int h, m, s;
    while (cin >> h >> m >> s && (h || m || s)) {
        int total = h * 3600 + m * 60 + s;
        int cnt = 0;
        for (int i = 0; i < 18000; i++) {
            int t = total + i;
            if (t % 2 == 0 && t % 3 == 0 && t % 5 == 0 && t % 7 == 0 && t % 11 == 0) {
                cnt++;
            }
        }
        printf("%02d:%02d:%02d\n", cnt / 3600, (cnt % 3600) / 60, cnt % 60);
    }
    return 0;
}
```
