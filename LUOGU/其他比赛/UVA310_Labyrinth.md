# UVA310 - Labyrinth (迷宫)

## 题目描述
在一个迷宫中，给定起点和终点，机器人可以在四个方向上移动。每次改变方向需要消耗代价1，同方向移动代价为0。求从起点到终点的最小代价。

## 输入格式
多组测试数据，每组第一行为 R C（行数和列数），R=C=0 表示结束。
接下来 R 行为迷宫地图，'#' 表示墙壁，其他为可通行区域。
最后一行为起点和终点坐标（1-indexed）。

## 输出格式
每行输出一个整数，表示最小移动代价。

## 样例
```
输入：
3 3
..#
#..
...
1 1 3 3
0 0

输出：
4
```

## 提示
- 使用BFS+状态扩展（记录方向状态）
- 改变方向代价为1，同方向移动代价为0
- 使用0-1BFS或Dijkstra算法

## 解题思路

### 问题分析
机器人在网格中移动，方向改变有代价，需要找最小代价路径。

### 核心思路
- 状态：(x, y, dir) 表示位置和当前方向
- 同方向移动代价0，转向代价1
- 使用0-1BFS（双端队列）或Dijkstra求最短路

### 复杂度分析
- 时间：O(R × C × 4)
- 空间：O(R × C × 4)
