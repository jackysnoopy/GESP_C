# UVA286 - Palindromic Paths (回文路径)

## 题目描述

在一个 n×n 的字符网格中，从左上角到右下角，每次只能向右或向下走。统计路径上字符组成回文串的路径数。

## 输入格式

- 多组测试数据
- 每组第一行是网格大小 n
- 接下来 n 行，每行 n 个字符
- n=0 时结束

## 输出格式

- 输出回文路径的数量

## 算法思路

1. 使用DFS枚举所有从(0,0)到(n-1,n-1)的路径
2. 检查每条路径上的字符是否构成回文串
3. 可用回文性质优化：只比较路径前半部分和后半部分

## 复杂度

- 时间：O(C(2n-2, n-1))
- 空间：O(n)
