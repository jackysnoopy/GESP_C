# UVA285 - Pipelining (管道铺设)

## 题目描述

在区间 [L, R] 上铺设管道，给定若干管道段（起点、终点、费用），求从 L 到 R 的最小费用。

## 输入格式

- 多组测试数据
- 每组第一行是管道段数 n
- 接下来 n 行，每行是起点 终点 费用
- 然后是 L 和 R
- n=0 时结束

## 输出格式

- 输出最小费用，无法到达则输出-1

## 算法思路

1. 按终点排序管道段
2. 使用动态规划：dp[i] 表示到达位置 i 的最小费用
3. 对每条管道段，尝试更新其终点的最小费用

## 复杂度

- 时间：O(N × (R-L))
- 空间：O(R)
