# UVA332 - Radar Extraction

## 题目描述

雷达系统采集到一系列信号读数。需要从这些读数中提取"峰值"——即一个读数不小于其左右邻居（边界处只有一个邻居）。提取所有峰值并按原始顺序输出。

## 输入格式

多组测试数据。每组第一行一个整数 $n$（$1 \le n \le 1000$），第二行 $n$ 个整数（读数）。$n = 0$ 时结束。

## 输出格式

对每组数据，第一行输出峰值数量，接下来每行最多输出 10 个值（空格分隔），两组数据之间输出一个空行。

## 样例输入

```
5
1 3 2 4 1
3
1 2 1
0
```

## 样例输出

```
3
1 3 4

1
2

```

## 解题思路

1. 遍历所有读数，判断每个位置是否为峰值
2. 峰值条件：对于位置 $i$，$a[i] \ge a[i-1]$ 且 $a[i] \ge a[i+1]$（边界特殊处理）
3. 收集所有峰值，按格式输出

## 时间复杂度

$O(n)$

## 空间复杂度

$O(n)$
