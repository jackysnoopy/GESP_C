# UVA230 - Book Thieves (书贼)

## 题目描述

图书管理员需要将书架上的书按字母顺序排列。给定书架上书的当前排列，找出最少需要移动多少本书才能使所有书按字母顺序排列。

每次只能将一本书从当前位置取出，然后插入到其他位置。

## 输入格式

- 多组测试用例
- 每组以标题开始
- 接下来是书名列表（每行一本）
- 以 "END" 结束
- 最后一组以 "END" 结束输入

## 输出格式

- 输出每组的标题
- 输出移动操作

## 解题思路

1. 读入书籍列表
2. 对副本排序得到目标排列
3. 使用贪心策略，从前往后确定每本书的正确位置
4. 模拟移动过程

## 注意事项

- 需要跟踪书的当前和目标位置
- 移动时要更新所有位置信息
