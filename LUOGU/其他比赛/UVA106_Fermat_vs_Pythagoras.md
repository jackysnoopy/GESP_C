# UVA106 - Fermat vs Pythagoras

## 题目描述

给定正整数 N（1 ≤ N < 1000000），找出满足以下条件的**本原毕达哥拉斯三元组** (a, b, c)：

- a² + b² = c²
- 1 ≤ a < b < c ≤ N
- gcd(a, b, c) = 1（即 a, b, c 互质）

同时统计从 1 到 N 中有多少个整数**不属于任何**毕达哥拉斯三元组（包括非本原的）。

对于每个输入的 N，输出两个整数：
1. 本原毕达哥拉斯三元组的个数
2. 1 到 N 中不能作为任何毕达哥拉斯三元组一部分的数字个数

## 输入输出格式

### 输入
多组测试数据，每行一个整数 N，读到文件结束 (EOF)。

### 输出
对于每组输入，输出一行，包含两个整数，用空格分隔。

## 样例输入

```
10
25
100
```

## 样例输出

```
1 4
4 9
16 27
```

## 解题思路

### 欧几里得公式（Euclid's Formula）

本原毕达哥拉斯三元组可以通过以下公式生成：

对于互质的正整数 m > n，且 m - n 为奇数：
- a = m² - n²
- b = 2mn
- c = m² + n²

满足 gcd(a, b, c) = 1 当且仅当 gcd(m, n) = 1 且 m - n 为奇数。

### 算法步骤

1. **枚举生成本原三元组**：枚举所有满足条件的 (m, n) 对，其中 m > n ≥ 1，m - n 为奇数，gcd(m, n) = 1。
2. **标记倍数**：对于每个本原三元组 (a, b, c)，将所有 k·a, k·b, k·c（k ≥ 1 且 ≤ N/c）标记为"已使用"。
3. **统计结果**：本原三元组计数器加一，最后统计未被标记的数字个数。

### 为什么需要标记倍数？

题目要求统计不能作为**任何**毕达哥拉斯三元组一部分的数字，包括非本原三元组（如 (6, 8, 10) 是 (3, 4, 5) 的两倍）。因此需要将本原三元组的所有倍数都标记出来。

### 枚举范围

由于 c = m² + n² ≤ N，所以 m² ≤ N，即 m ≤ √N。因此外层循环 m 的范围是 [2, √N]。

## 复杂度分析

- **时间复杂度**：O(N · log N)。枚举 (m, n) 对的数量为 O(N)，每个本原三元组标记倍数的操作为 O(N/c)，总计约 O(N · log N)。
- **空间复杂度**：O(N)，用于布尔数组标记已使用的数字。
