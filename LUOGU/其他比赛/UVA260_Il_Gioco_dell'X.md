# UVA260 - Il Gioco dell'X（X游戏）

## 题目描述

两人在一个 N×N 的六角网格棋盘上下棋。黑方（`b`）从上往下连通，白方（`w`）从左往右连通。给定最终棋盘状态，判断谁赢了。

黑方的目标：找到一条由 `b` 组成的路径，从第 0 行连通到第 N-1 行。
白方的目标：找到一条由 `w` 组成的路径，从第 0 列连通到第 N-1 列。

保证恰好有一方获胜。

## 输入格式

- 多组测试数据，每组以整数 N 开头（棋盘大小，N ≤ 200）
- 接下来 N 行，每行 N 个字符（`b`、`w` 或 `.`）
- N = 0 时结束输入

## 输出格式

每组输出一行：`X b wins` 或 `X w wins`，其中 X 是测试数据编号（从 1 开始）

## 算法思路

由于保证只有一方获胜，只需分别对黑方和白方进行 BFS：

1. **黑方 BFS**：从所有第 0 行的 `b` 出发，检查能否到达第 N-1 行
2. **白方 BFS**：从所有第 0 列的 `w` 出发，检查能否到达第 N-1 列

**六角网格邻接关系**：每个格子有 6 个邻居，具体取决于行号的奇偶性：
- 偶数行：左上、上、左、右、左下、下
- 奇数行：上、右上、左、右、下、右下

**时间复杂度**：O(N²)，每个格子最多被访问一次
