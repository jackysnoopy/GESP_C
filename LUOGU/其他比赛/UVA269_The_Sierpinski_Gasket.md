# UVA269 - The Sierpinski Gasket (谢尔宾斯基三角)

## 题目描述

谢尔宾斯基三角（Sierpinski Gasket）是一个著名的分形图案。给定一个整数 $n$（$0 \le n \le 8$），你需要画出 $2^n \times 2^n$ 大小的谢尔宾斯基三角。

谢尔宾斯基三角的规律是：对于一个 $2^n \times 2^n$ 的网格，位置 $(i, j)$ 处是星号 `*` 当且仅当 $i \ \& \ j = 0$（按位与运算）。

## 输入格式

一行一个整数 $n$。

## 输出格式

输出 $2^n$ 行，每行 $2^n$ 个字符，由 `*` 和空格组成。

## 样例输入

```
3
```

## 样例输出

```
********
* ** **
**  **
*   *
** **
* *
*
```

## 解题思路

### 规律分析

谢尔宾斯基三角的核心规律：在 $2^n \times 2^n$ 的网格中，位置 $(i, j)$ 处是星号当且仅当 $i \ \& \ j = 0$。

- 当 $n = 0$ 时，输出一个 `*`
- 当 $n = 1$ 时，输出 $2 \times 2$ 的图案：
  ```
  **
  * 
  ```
- 当 $n = 2$ 时，输出 $4 \times 4$ 的图案，以此类推

### 算法步骤

1. 读入整数 $n$
2. 计算网格大小 $size = 2^n$
3. 对于每个位置 $(i, j)$，判断 $(i \ \& \ j) == 0$ 是否成立
4. 成立则输出 `*`，否则输出空格

### 时间复杂度

$O(2^{2n})$，即 $O(size^2)$。

### 空间复杂度

$O(1)$，仅使用常数个变量。

### 位运算原理

按位与运算 $i \ \& \ j = 0$ 意味着 $i$ 和 $j$ 的二进制表示中没有相同位置都为 1 的位。这恰好对应了谢尔宾斯基三角的自相似结构。
