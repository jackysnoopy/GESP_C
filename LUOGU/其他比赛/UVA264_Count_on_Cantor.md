# UVA264 - Count on Cantor (数康托)

## 题目描述

康托（Cantor）是德国数学家，他最重要的发现之一就是正有理数的枚举。他用如下方式把正有理数排列成一个序列：

$$1/1, 1/2, 2/1, 3/1, 2/2, 1/3, 1/4, 2/3, 3/2, 4/1, 1/5, 2/4, 3/3, 4/2, 5/1, \ldots$$

这些有理数按照"对角线"方式排列：将分子和分母之和相等的有理数归为一组（对角线），每条对角线内的有理数按分子递增或递减排列。

你的任务是：给定一个正整数 $N$，找出序列中第 $N$ 个有理数。

## 输入格式

多组测试数据。每组一行，包含一个正整数 $N$（$1 \le N \le 10^7$）。

## 输出格式

对每组输入，输出一行：`The Nth term is a/b.`

## 样例输入

```
8
20
```

## 样例输出

```
The 8th term is 2/3.
The 20th term is 5/2.
```

## 解题思路

### 规律分析

康托序列按"对角线"分组：

```
对角线1: 1/1
对角线2: 1/2  2/1
对角线3: 3/1  2/2  1/3
对角线4: 1/4  2/3  3/2  4/1
对角线5: 5/1  4/2  3/3  2/4  1/5
```

- 第 $k$ 条对角线包含 $k$ 个有理数，分子和分母之和为 $k+1$
- **奇数**条对角线：$k/1, (k{-}1)/2, \ldots, 1/k$（分子递减）
- **偶数**条对角线：$1/k, 2/(k{-}1), \ldots, k/1$（分子递增）

前 $k$ 条对角线共有 $\frac{k(k+1)}{2}$ 个有理数。

### 算法步骤

1. 找到第 $N$ 项所在的对角线 $k$，使得 $\frac{(k-1)k}{2} < N \le \frac{k(k+1)}{2}$
2. 计算在该对角线内的偏移：$pos = N - \frac{(k-1)k}{2}$
3. 根据 $k$ 的奇偶性确定分子和分母：
   - $k$ 为奇数：$\text{num} = k - pos + 1,\ \text{den} = pos$
   - $k$ 为偶数：$\text{num} = pos,\ \text{den} = k - pos + 1$

### 时间复杂度

$O(T \cdot \sqrt{N})$，其中 $T$ 为测试组数，$\sqrt{N}$ 为查找对角线编号。

### 空间复杂度

$O(1)$，仅使用常数个变量。
