# UVA331 - Misha and Chess

## 题目描述

米沙正在学习国际象棋中的排序问题。给定一个排列，要求找出其中可以选出的最大子序列的逆序对数。即从 $n$ 个数中选出一个子序列（保持原顺序），使得这个子序列内部的逆序对数最多。

注意：选出的子序列中的元素值不能有重复。

## 输入格式

多组测试数据。每组第一行一个整数 $n$（$1 \le n \le 20$），第二行 $n$ 个整数。$n = 0$ 时结束。

## 输出格式

对每组数据输出一行：

```
There are X swap(s).
```

其中 $X$ 为最大逆序对数。

## 样例输入

```
3
1 2 3
3
3 2 1
0
```

## 样例输出

```
There are 0 swap(s).
There are 3 swap(s).
```

## 解题思路

1. 枚举所有子集（位掩码枚举 $2^n$ 种选择方式）
2. 对每种子集，提取选出的元素，检查是否有重复值
3. 计算选出子序列的逆序对数
4. 取所有合法子集中的最大逆序对数

## 时间复杂度

- $O(2^n \cdot n^2)$，适合 $n \le 20$

## 空间复杂度

$O(n)$
