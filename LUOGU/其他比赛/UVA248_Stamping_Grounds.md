# UVA248 - Stamping Grounds (印第安营地)

## 题目描述

在一个R×C的网格中，每个格子可以是空地、障碍物或探险者的起始位置（用N/S/E/W表示方向）。每个探险者会沿着指定方向一直走到碰到障碍物或边界为止。求所有探险者最终位置的最小包围矩形，该矩形区域内所有格子用'*'标记，其余用'.'标记。

## 输入输出格式

**输入：** 多组测试数据。每组先给出R和C，然后R行C列的网格。R=0且C=0时结束。

**输出：** 每组输出"Simulation X:"，然后输出结果网格。

## 思路分析

1. **模拟移动：** 每个探险者沿指定方向移动直到受阻
2. **记录终点：** 记录每个探险者的最终位置
3. **计算最小包围矩形：** 找到包含所有终点的最小矩形
4. **标记输出：** 矩形内标记'*'，其余标记'.'

## 复杂度分析

- 时间复杂度：O(R*C) per test case
- 空间复杂度：O(R*C)
