# UVA245 - Uncompress (解压)

## 题目描述

给定一个压缩后的文本文件，对其进行解压缩。压缩格式如下：一个数字紧跟一个单词表示该单词重复出现的次数（但实际只需要写一次），数字表示该单词的长度。例如，"3hello"表示"hello"这个5字母的单词出现了3次，输出为"hellohellohello"。

## 输入输出格式

**输入：** 一个压缩后的文本文件，可以包含多行。

**输出：** 解压后的原始文本。

## 思路分析

1. **栈模拟：** 使用栈来处理解压缩
2. **解析规则：** 遇到数字，读取后续单词，将该单词按数字重复次数压栈
3. **处理非压缩部分：** 非数字字符直接压栈
4. **输出：** 栈中内容即为解压结果

## 复杂度分析

- 时间复杂度：O(N)，N为输出文本长度
- 空间复杂度：O(N)
