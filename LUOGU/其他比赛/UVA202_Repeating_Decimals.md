# UVA202 - Repeating Decimals (循环小数)

## 题目描述

给定两个正整数 a 和 b，计算 a/b 的小数表示。如果结果是循环小数，用括号标注循环节。

## 输入格式

多组测试数据，每组一行两个整数 a, b。

## 输出格式

输出 a / b 的小数表示，循环节用括号括起来。如果循环节超过50位，只输出前50位并提示"Too many decimal digits."。

## 样例

输入：
```
76 25
5 43
1 3
```

输出：
```
76 / 25 = 3.04
5 / 43 = 0.(116279069767441860465)
1 / 3 = 0.(3)
```

## 解题思路

### 问题分析
将分数转换为小数表示，识别循环部分。

### 核心思路
- 使用长除法，每一步记录余数
- 如果某个余数重复出现，说明进入了循环
- 循环节从该余数第一次出现的位置开始

### 算法流程
1. 每次余数乘10，除以 b 得到一位数字
2. 用 map 记录每个余数出现的位置
3. 如果余数变为0，说明是有限小数
4. 如果余数重复出现，从记录的位置开始到末尾即为循环节

### 复杂度分析
- 时间复杂度：O(b)，最多 b 个不同的余数
- 空间复杂度：O(b)
