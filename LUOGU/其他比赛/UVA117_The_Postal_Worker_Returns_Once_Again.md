# UVA117 - The Postal Worker Returns Once Again

## 题目描述

邮递员需要走遍城市中所有街道恰好一次，然后返回起点。每条街道有一个权重（长度）。如果图不是欧拉图（存在奇数度的顶点），需要添加一些重复边使图变为欧拉图，同时最小化总权重。

这是**中国邮递员问题（Chinese Postman Problem）**的变种。

## 输入格式

- 多组测试数据，每组由若干行街道描述组成
- 每条街道格式：两个两位字母的城市名 + 一个数字权重
- 空行表示一组数据结束

## 输出格式

- 输出最短的邮递员路线总权重

## 解题思路

### 核心思想

**中国邮递员问题**：在图中找到最小权重的欧拉回路。

### 算法步骤

1. **判断欧拉图**：所有顶点度数为偶数则是欧拉图
2. **找奇数度顶点**：如果不是欧拉图，找到所有奇数度顶点
3. **最小权完美匹配**：将奇数度顶点两两配对，使配对间的最短路径总和最小
4. **计算结果**：原始总权重 + 额外添加的最小权重

### 关键点

- 使用 Dijkstra 算法求奇数度顶点间的最短路径
- 使用状态压缩 DP 进行最小权完美匹配
- 字符串到整数的映射

## 复杂度分析

- **时间复杂度**：O(V³ + E log E)，V 为顶点数，E 为边数
- **空间复杂度**：O(V²)
