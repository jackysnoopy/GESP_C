# UVA220 - Circle Battle (圆形战斗)

## 题目描述

n 个人均匀站成一个圆圈，半径为 r。求相邻两人之间的距离。

## 输入格式

多组测试数据，每组一行两个整数 n 和实数 r，n=0 且 r=0 时结束。

## 输出格式

输出相邻两人之间的距离，保留2位小数。

## 解题思路

### 问题分析
n 个点均匀分布在半径为 r 的圆上，求相邻两点距离。

### 核心思路
- 相邻两点对应的圆心角为 2π/n
- 根据弦长公式：d = 2r × sin(π/n)

### 算法流程
1. 计算圆心角 angle = 2π/n
2. 弦长 d = 2r × sin(angle/2)

### 复杂度分析
- 时间复杂度：O(1)
- 空间复杂度：O(1)
