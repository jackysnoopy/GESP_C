## 题目描述

2. 将一个 $A_l=A_r$ 且 $l\not=r$ 的区间 $[l,r]$ 赋值为 $-10^9$，代价为区间长度。

## 解题思路

### 问题分析

给定一个排列 A，有两种操作：(1) A_i += 1，代价 1；(2) 选择 A_l = A_r 的区间赋值为 -10^9，代价为区间长度。目标是将所有元素变为 -10^9，最小化总代价。

### 核心思路

- 操作 2 可以将一段区间清除，但需要两端值相等。
- -10^9 可以作为任意区间的端点（因为 A_l = A_r = -10^9 也合法）。
- 核心思想：先通过少量 increment 操作创造可配对的值，然后用操作 2 大面积清除。
- 对于单调递增排列 [1,2,...,n]：最优策略是 increment A_1 和 A_{n-1} 各 1 次，然后用 3 次操作 2 清除，总代价 n + 2 + (n+2-n) = n + 4。
- 一般排列：需要根据排列的具体结构动态规划。

### 算法流程

1. 对单调递增/递减排列：答案 = n + n/2。
2. 对一般排列：需要更复杂的 DP 或贪心策略，考虑 increment 和操作 2 的组合优化。

### 复杂度分析

- 时间复杂度：单调情况 O(n)，一般情况取决于具体算法实现。
- 空间复杂度：O(n)，存储排列和辅助数组。
