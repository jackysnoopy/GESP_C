# [蓝桥杯 2023 省 Java B] 最大开支

## 题目描述

小蓝所在学校周边新开业了一家游乐园，小蓝作为班长，打算组织大家去游乐园玩。已知一共有 $N$ 个人参加这次活动，游乐园有 $M$ 个娱乐项目，每个项目都需要买门票后才可进去游玩。门票的价格并不是固定的，团购的人越多单价越便宜，当团购的人数大于某个阈值时，这些团购的人便可以免费进入项目进行游玩。这 $M$ 个娱乐项目是独立的，所以只有选择了同一个项目的人才可以参与这个项目的团购。第 $i$ 个项目的门票价格 $H_i(X)$ 与团购的人数 $X$ 的关系可以看作是一个函数：

$$H_i(X) = \max(K_i \times X + B_i, 0)$$

其中 $\max$ 表示取二者之中的最大值。当 $H_i = 0$ 时说明团购人数达到了此项目的免单阈值。

这 $N$ 个人可以根据自己的喜好选择 $M$ 个娱乐项目中的一种，或者有些人对这些娱乐项目都没有兴趣，也可以选择不去任何一个项目。每个人最多只会选择一个娱乐项目，如果多个人选择了同一个娱乐项目，那么他们都将享受对应的团购价格。小蓝想知道他至少需要准备多少钱，使得无论大家如何选择，他都有能力支付得起所有 $N$ 个人购买娱乐项目的门票钱。

## 输入格式

第一行两个整数 $N$、$M$，分别表示参加活动的人数和娱乐项目的个数。接下来 $M$ 行，每行两个整数，其中第 $i$ 行为 $K_i$、$B_i$，表示第 $i$ 个游乐地点的门票函数中的参数。

## 输出格式

一个整数，表示小蓝至少需要准备多少钱，使得大家无论如何选择项目，自己都支付得起。

## 样例

### 样例 1

**输入：**
```
4 2
-4 10
-2 7
```

**输出：**
```
12
```

### 样例说明

当 $a = 1, b = 2$ 时花费最大，为 $12$。此时 $1$ 个人去第一个项目（单价 $10-4=6$，花费 $6$），$2$ 个人去第二个项目（单价 $7-4=3$，花费 $6$），$1$ 个人没去任何项目。总花费 $6+6=12$。

## 提示

- 对于 $30\%$ 的评测用例，$1 \leq N, M \leq 10$。
- 对于 $50\%$ 的评测用例，$1 \leq N, M \leq 1000$。
- 对于 $100\%$ 的评测用例，$1 \leq N, M, B_i \leq 10^5$，$-10^5 \leq K_i < 0$。

## 解题思路

### 问题分析

这是一道**极小极大（Minimax）**问题。小蓝需要准备足够的钱来应对最坏情况——即"对手"（选择项目的人群）选择使总花费最大化的分配方案。

关键观察：对于项目 $i$，当 $X$ 个人选择时，总花费为 $\text{Cost}_i(X) = \max(K_i \cdot X + B_i, 0) \cdot X$。由于 $K_i < 0$，这是一个**凹函数**（先增后减的抛物线），在某个最优人数处达到最大值。

### 核心思路：贪心 + 优先队列

**边际成本分析**：第 $X$ 个人选择项目 $i$ 带来的额外花费（边际成本）为：

$$\text{MC}_i(X) = \text{Cost}_i(X) - \text{Cost}_i(X-1) = K_i(2X-1) + B_i$$

由于 $K_i < 0$，边际成本随人数递减。当 $\text{MC}_i(X) \leq 0$ 时，再分配人到该项目不会增加总花费。

**贪心策略**：对手每次将一个人分配到边际成本最高的项目。这等价于将所有正边际成本按降序排列，取前 $N$ 个（或直到边际成本非正）。

### 算法流程

1. 对每个项目，计算第 1 个人的边际成本 $\text{MC}_i(1) = K_i + B_i$
2. 将正边际成本放入最大堆
3. 每次弹出最大值，累加到总花费，然后计算该项目的下一个边际成本并压入堆
4. 重复直到分配了 $N$ 个人或没有正边际成本

### 复杂度分析

- **时间复杂度**：$O(N \log M)$，每个人最多进/出堆一次
- **空间复杂度**：$O(M)$，存储项目参数和堆

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<long long> k(m), b(m);
    vector<long long> cnt(m, 0);
    priority_queue<pair<long long, int>> pq;
    
    for (int i = 0; i < m; i++) {
        cin >> k[i] >> b[i];
        // 第 1 个人的边际成本：k*1 + b
        long long mc = k[i] + b[i];
        if (mc > 0) {
            pq.push({mc, i});
        }
    }
    
    long long total = 0;
    int people = 0;
    
    // 贪心：每次选择边际成本最高的项目分配一个人
    while (people < n && !pq.empty()) {
        auto top = pq.top();
        pq.pop();
        long long mc = top.first;
        int i = top.second;
        
        if (mc <= 0) break;
        
        total += mc;
        cnt[i]++;
        people++;
        
        // 计算下一个边际成本：k*(2*x+1) + b
        long long next_mc = k[i] * (2 * cnt[i] + 1) + b[i];
        if (next_mc > 0) {
            pq.push({next_mc, i});
        }
    }
    
    cout << total << endl;
    return 0;
}
```
