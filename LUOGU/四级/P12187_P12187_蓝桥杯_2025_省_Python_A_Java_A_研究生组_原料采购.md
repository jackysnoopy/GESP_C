# P12187 [蓝桥杯 2025 省 Python A/Java A/研究生组] 原料采购

## 题目描述

小蓝负责一家工厂的原料采购。

工厂有一辆运货卡车，其容量为 $m$。

工厂附近的采购点都在同一条路的同一方向上，一共有 $n$ 个，每个采购点和工厂的距离各不相同。其中，第 $i$ 个采购点的价格为 $a_i$, 库存为 $b_i$, 距离为 $c_i$。

卡车每行驶一单位长度的路径就需要额外花费 $o$。（返程没有花费，你也可以认为 $o$ 实际是行驶两单位长度的花费）

请计算将卡车装满最少需要花费多少钱，如果没有任何方案可以装满请输出 $-1$。


## 输入格式

输入的第一行包含三个正整数 $n, m, o$，相邻整数之间使用一个空格分隔。

接下来 $n$ 行，每行包含三个正整数 $a_i, b_i, c_i$ 表示一个采购点，相邻整数之间使用一个空格分隔。


## 输出格式

输出一行包含一个整数表示答案，即装满卡车所需的最小花费。

## 解题思路

### 问题分析

有 n 个采购点，每个点有价格、库存、距离。卡车容量为 m，行驶每单位距离花费 o。从工厂出发（距离 0）到最远采购点，需要装满卡车，求最小花费。

关键观察：对于固定的最大行驶距离，应优先选择价格最低的货物。枚举最远到达的采购点，用数据结构快速查询前 m 单位货物的最低价格。

### 核心思路

- 按距离排序采购点，依次加入。
- 使用 BIT（树状数组）维护货物数量和总价。
- 对于每个采购点 i（作为最远点），查询购买 m 单位货物的最低价格 + 行驶到 i 的油费 c_i × o。
- 使用 BIT 上的"第 k 小"查询，找到第 m 个单位货物所在的价格区间。

### 算法流程

1. 读入采购点，按价格离散化。
2. 依次加入每个采购点到 BIT，更新数量和总价。
3. 当总库存 ≥ m 时，用 BIT 的 findKth 找到第 m 个单位所在价格位。
4. 计算该价格下的最低花费 + 行驶费用，更新答案。
5. 输出最小花费，若无解输出 -1。

### 复杂度分析

- 时间复杂度：O(n log n)，每个采购点的 BIT 操作为 O(log n)。
- 空间复杂度：O(n)，BIT 数组和采购点存储。
