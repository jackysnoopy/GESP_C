# [GESP202312 四级] 田忌赛马

## 题目描述

你要和田忌赛马。你们各自有 $N$ 匹马，并且要进行 $N$ 轮比赛，每轮比赛，你们都要各派出一匹马决出胜负。

你的马匹的速度分别为 $u_1,u_2,\cdots，u_n$，田忌的马匹的速度分别为 $v_1,v_2,\cdots,v_n$。田忌会按顺序派出他的马匹，请问你要如何排兵布阵，才能赢得最多轮次的比赛？巧合的是，你和田忌的所有马匹的速度两两不同，因此不可能出现平局。

## 输入格式

第一行一个整数 $N$。保证 $1\le N \le 5\times 10^4$

接下来一行 $N$ 个用空格隔开的整数，依次为 $u_1,u_2,\cdots,u_n$，表示你的马匹们的速度。保证 $1\le u_i\le 2N$。

接下来一行 $N$ 个用空格隔开的整数，依次为 $v_1,v_2,\cdots,v_n$，表示田忌的马匹们的速度。保证 $1\le v_i\le 2N$。

## 输出格式

输出一行，表示你最多能获胜几轮。

## 样例

### 样例 #1

**输入**
```
3
1 3 5
2 4 6
```

**输出**
```
2
```

**解释**

第 1 轮，田忌派出速度为 2 的马匹，你可以派出速度为 3 的马匹迎战，本轮你获胜。

第 2 轮，田忌派出速度为 4 的马匹，你可以派出速度为 5 的马匹迎战，本轮你获胜。

第 3 轮，田忌派出速度为 6 的马匹，你可以派出速度为 1 的马匹迎战，本轮田忌获胜。

如此，你可以赢得 2 轮比赛。

### 样例 #2

**输入**
```
5
10 3 5 8 7
4 6 1 2 9
```

**输出**
```
5
```

## 提示

## 解题思路

### 问题分析

田忌按顺序派出马匹（顺序固定为 $v_1$ 到 $v_n$），我们需要安排自己的马匹迎战顺序，最大化赢的轮数。所有速度两两不同，无平局。

### 核心思路

经典田忌赛马贪心策略：

1. 将自己的马按速度从小到大排序。
2. 对于田忌的每一匹马（按输入顺序），用自己的最慢马去比：
   - 如果最慢马能赢过田忌的这匹马，就用最慢马去赢，增加胜场。
   - 如果最慢马赢不了，就用最快的马去"牺牲"。
3. 具体实现可以用 `multiset` 维护自己的马，用 `lower_bound` 找刚好大于田忌马的速度的马。

更简单的实现方式：
1. 将双方马匹排序。
2. 用双指针法：用自己最慢的马 vs 田忌最慢的马，或者自己最快的马 vs 田忌最快的马。

但实际上田忌是按顺序出马的，所以正确的做法是：
- 将自己的马排序存入 `multiset`。
- 遍历田忌的每匹马 $v_i$：
  - 在 `multiset` 中查找 $> v_i$ 的最小的马（即刚好能赢的）。
  - 如果找到，则胜场+1，并删除这匹马。
  - 如果找不到，则用自己最慢的马去输（删除最小元素）。

### 算法流程

1. 读入 $N$。
2. 读入自己的 $N$ 匹马，存入 `multiset<int>`。
3. 读入田忌的 $N$ 匹马，存入数组 $v$。
4. 遍历田忌的每匹马 $v_i$：
   - 用 `lower_bound` 找 `*it > v_i` 的马。
   - 如果找到，胜场+1，删除 `it`。
   - 如果找不到，删除最小的马（送给对方赢）。
5. 输出胜场。

### 复杂度分析

- **时间复杂度：** $O(N \log N)$，来自排序和 `multiset` 操作。
- **空间复杂度：** $O(N)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int N;
    cin >> N;
    multiset<int> myHorses;
    for (int i = 0; i < N; i++) {
        int x;
        cin >> x;
        myHorses.insert(x);
    }
    vector<int> tian(N);
    for (int i = 0; i < N; i++) cin >> tian[i];
    
    int win = 0;
    for (int v : tian) {
        auto it = myHorses.upper_bound(v);
        if (it != myHorses.end()) {
            win++;
            myHorses.erase(it);
        } else {
            myHorses.erase(myHorses.begin());
        }
    }
    cout << win << "\n";
    return 0;
}
```
