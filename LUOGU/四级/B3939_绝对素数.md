# [GESP样题 四级] 绝对素数

## 题目描述

如果一个两位数是素数，且它的数字位置经过对换后仍为素数，则称为绝对素数，例如 $13$。给定两个正整数 $A, B$，请求出大于等于 $A$、小于等于 $B$ 的所有绝对素数。

## 输入格式

输入 $1$ 行，包含两个正整数 $A$ 和 $B$。保证 $10<A<B<100$。

## 输出格式

若干行，每行一个绝对素数，从小到大输出。

## 样例

### 样例 #1

**输入**
```
11 20
```

**输出**
```
11
13
17
```

## 提示

## 解题思路

### 问题分析

给定范围 $[A, B]$（两位数），找出所有满足两个条件且不重复的数：
1. 本身是素数。
2. 十位和个位交换后得到的数也是素数。

### 核心思路

1. 写一个判断素数的函数 `isPrime(x)`。
2. 从 $A$ 遍历到 $B$，对每个数判断是否绝对素数。
3. 交换数字位置：$reversed = (x \% 10) \times 10 + (x / 10)$。

### 算法流程

1. 读入 $A, B$。
2. 从 $i = A$ 到 $B$：
   - 判断 $i$ 是否为素数。
   - 计算 $rev = (i \% 10) \times 10 + (i / 10)$。
   - 判断 $rev$ 是否为素数。
   - 如果两者都是素数，输出 $i$。

### 复杂度分析

- **时间复杂度：** $O((B-A) \times \sqrt{B})$，$B-A < 90$。
- **空间复杂度：** $O(1)$。

## 参考代码

```cpp
#include <iostream>
#include <cmath>
using namespace std;

bool isPrime(int x) {
    if (x < 2) return false;
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) return false;
    }
    return true;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int A, B;
    cin >> A >> B;
    for (int i = A; i <= B; i++) {
        int rev = (i % 10) * 10 + (i / 10);
        if (isPrime(i) && isPrime(rev)) {
            cout << i << "\n";
        }
    }
    return 0;
}
```
