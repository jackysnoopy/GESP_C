# [GESP202409 四级] 区间排序

## 题目描述

小杨有一个包含 $n$ 个正整数的序列 $a$。

小杨计划对序列进行多次升序排序，每次升序排序小杨会选择一个区间 $[l,r]$（$l \leq r$）并对区间内所有数字进行升序排序。每次升序排序会在上一次升序排序的结果上进行。

小杨想请你计算出多次升序排序后的序列。

## 输入格式

第一行包含一个正整数 $n$，含义如题面所示。

第二行包含 $n$ 个正整数 $a_1, a_2, \dots a_n$，代表序列 $a$。

第三行包含一个正整数 $q$，代表排序次数。

之后 $q$ 行，每行包含两个正整数 $l, r$，代表将区间 $[l_i, r_i]$ 内所有数字进行升序排序。

## 输出格式

输出一行包含 $n$ 个正整数，代表多次升序排序后的序列。

## 样例

### 样例 #1

**输入**
```
5
3 4 5 2 1
3
4 5
3 4
1 3
```
**输出**
```
1 3 4 5 2
```

## 解题思路

### 问题分析

给定初始序列和 $q$ 个区间，按顺序对每个区间内的元素进行升序排序（使用 `sort` 函数），输出最终序列。

数据范围 $n, q \le 100$，可以直接模拟。

### 核心思路

直接使用 C++ 的 `sort` 函数对每个区间进行排序。注意输入的 $l,r$ 是 1-indexed，在代码中需转换为 0-indexed。

### 算法流程

1. 读取 $n$ 和数组 $a$。
2. 读取 $q$。
3. 对每个区间 $[l,r]$：
   - 将 $l$ 和 $r$ 转换为 0-indexed：`l--`，`r--`。
   - 对 $a[l]$ 到 $a[r]$ 使用 `sort(a.begin() + l, a.begin() + r + 1)` 进行升序排序。
4. 输出最终序列。

### 复杂度分析

- **时间复杂度：** $O(q \times (r-l+1) \log (r-l+1))$，最坏 $O(q \times n \log n)$。
- **空间复杂度：** $O(n)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    int q;
    cin >> q;
    while (q--) {
        int l, r;
        cin >> l >> r;
        l--, r--;
        sort(a.begin() + l, a.begin() + r + 1);
    }
    for (int i = 0; i < n; i++) cout << a[i] << " \n"[i == n - 1];
    return 0;
}
```
