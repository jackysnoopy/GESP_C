# P12386 [蓝桥杯 2023 省 Python B] 混乱的数组

## 题目描述

给定一个正整数 $x$，请找出一个尽可能短的仅含正整数的数组 $A$ 使得 $A$ 中恰好有 $x$ 对 $i, j$ 满足 $i < j$ 且 $A_i > A_j$。

如果存在多个这样的数组，请输出字典序最小的那个。


## 输入格式

输入一行包含一个整数表示 $x$。


## 输出格式

输出两行。

第一行包含一个整数 $n$，表示所求出的数组长度。

第二行包含 $n$ 个整数 $A_i$，相邻整数之间使用一个空格分隔，依次表示数组中的每个数。


## 样例

**输入样例 1**
```
3
```
**输出样例 1**
```
3
3 2 1
```

## 解题思路

### 问题分析

给定 x，找一个最短的正整数数组，使得恰好有 x 对逆序对 (i<j, A_i > A_j)，且字典序最小。

### 核心思路

- 首先确定数组长度 n：找到最小的 n 使得 C(n,2) = n*(n-1)/2 ≥ x。
- 然后贪心构造字典序最小的排列：从左到右确定每个位置的值，选择尽可能小的数，使得剩余位置仍能容纳剩余的逆序对数。

### 算法流程

1. 若 x=0，输出 [1]。
2. 找最小 n 使得 n*(n-1)/2 ≥ x。
3. 初始化候选数组 s = [1, 2, ..., n]。
4. 对于每个位置 i（从 0 到 n-1），剩余 rem = n-i 个元素：
   - 枚举选择 s 中第 j 个元素，该元素会产生 j 个逆序对（比它小的元素排在后面）。
   - 检查 remaining - j 是否在 [0, C(rem-1, 2)] 范围内。
   - 选择满足条件的最小 j，将对应元素加入结果，从候选中移除。

### 复杂度分析

- 时间复杂度：O(n²)，每个位置枚举候选元素。
- 空间复杂度：O(n)，存储候选和结果数组。
