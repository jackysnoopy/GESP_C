# P12234 [蓝桥杯 2023 国 Java A] 最大算式

## 题目描述

给定 $n$ 个非负整数 $A_i$，你可以在不改变这些数顺序的前提下任意在他们之间插入 $\tt{+},\tt{*},\tt{(},\tt{)}$ 四种符号。

请问在得到的算式合法的前提下，算式的结果最大可以是多少？

由于结果很大，你只需要输出答案对 $10^9 + 7$ 取模的结果即可。


## 输入格式

输入的第一行包含一个整数 $n$。

第二行包含 $n$ 个整数，分别表示 $A_1, A_2, \cdots, A_n$，相邻两个整数之间使用一个空格分隔。


## 输出格式

输出一行包含一个整数表示答案。


## 样例

**输入样例 1**
```
7
1 2 1 1 1 2 1
```
**输出样例 1**
```
27
```

## 解题思路

### 问题分析

给定 n 个数，可在相邻数之间插入 +、*、(、) ，求算式最大值。由于 + 和 * 的结合性，问题等价于：将序列分成若干组，每组内求和，组间求积，使总乘积最大。

### 核心思路

- 贪心性质：每组最多 3 个元素（因为将 4 个以上的数分组可以进一步拆分来增大乘积）。
- 使用 DP：dp[i] = 前 i 个元素的最大乘积（取对数比较大小）。
- 对每个位置，尝试最后一组大小为 1、2、3，取最优。

### 算法流程

1. 计算前缀和 prefix[i]，用于 O(1) 求组内和。
2. dp[0] = 1（空序列乘积为 1），dp_log[0] = 0。
3. 对 i = 1 到 n：
   - 尝试大小 1：dp[i] = max(dp[i], dp[i-1] × a[i-1])
   - 尝试大小 2：dp[i] = max(dp[i], dp[i-2] × (a[i-2]+a[i-1]))
   - 尝试大小 3：dp[i] = max(dp[i], dp[i-3] × (a[i-3]+a[i-2]+a[i-1]))
4. 使用对数进行大小比较，避免溢出；实际值对 10^9+7 取模。

### 复杂度分析

- 时间复杂度：O(n)，每个位置只尝试 3 种分组。
- 空间复杂度：O(n)，存储 dp 值和对数值。
