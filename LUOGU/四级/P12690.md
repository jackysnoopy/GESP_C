# P12690 [KOI 2022 Round 1] ABBC

## 题目描述

有一个只由 A、B、C 构成的字符串 $S$，长度为 $|S|$。你可以对这个字符串执行以下操作：

- 删除一个 A 及其后的某个 B；
- 删除一个 B 及其后的某个 C。

每个字符最多只能被删除一次。

例如，考虑字符串 ABCBA。将字符从左到右编号为 1、2、3……，可以如下操作：

- 删除第 1 个字符 A 和第 4 个字符 B，然后删除第 2 个字符 B 和第 3 个字符 C。此时操作次数为 2，剩余字符串为 A。

请你求出最多可以进行多少次这样的操作。

## 输入格式

第一行输入字符串 $S$。

## 输出格式

输出一行，表示最多可以进行的操作次数。

## 样例

**输入样例 1**
```
ABCBA
```
**输出样例 1**
```
2
```

**输入样例 2**
```
ABCBBACBABB
```
**输出样例 2**
```
5
```

## 提示

**约束条件**

- $1 \leq |S| \leq 300\,000$
- $S$ 中的所有字符均为 A、B 或 C

## 解题思路

### 问题分析

本题要求在只含 A、B、C 的字符串上，通过两种操作（删除A+B 或 删除B+C）最大化操作次数。每个字符只能被删除一次。

核心难点在于：**B 是两种操作共用的资源**。一个 B 要么参与 AB 操作（作为被删的 B），要么参与 BC 操作（作为被删的 B），要么不参与任何操作。因此需要合理分配 B 的使用。

### 核心思路

**两遍贪心匹配**：

1. **第一遍：从右到左贪心匹配 BC**
   - 维护一个栈存储遇到的 C
   - 从右向左扫描，遇到 B 时如果有待匹配的 C，就配对一次 BC 操作
   - 这样保证 B 在 C 左边（满足操作要求），且贪心地用最右的 B 匹配最左的 C

2. **第二遍：从左到右贪心匹配 AB**
   - 标记第一遍中被 BC 操作使用的 B
   - 从左向右扫描，用栈维护 A，遇到未被使用的 B 时配对 AB 操作

**为什么先 BC 后 AB？**

BC 操作的约束更严格：B 必须在 C 左边，且 C 的数量通常较少。先贪心匹配 BC 可以最大化利用有限的 C。然后用剩余的 B 贪心匹配 AB，因为 A 的数量通常较多，AB 匹配更容易成功。

### 算法流程

1. 从右向左扫描字符串：
   - 遇到 C 入栈
   - 遇到 B 且栈非空 → 配对 BC，标记该 B 已使用
2. 从左向右扫描字符串：
   - 遇到 A → a_count++
   - 遇到未使用的 B 且 a_count > 0 → 配对 AB，ab_match++，a_count--
3. 输出 ab_match + bc_match

### 正确性证明

设最优解中做了 $x$ 次 AB 操作和 $y$ 次 BC 操作。贪心的 BC 匹配不会减少 $y$（因为贪心匹配了尽可能多的 BC），而剩余的 B 贪心匹配 AB 也不会减少 $x$（因为每个 A 都贪心地找最近的可用 B）。因此贪心解等于最优解。

### 复杂度分析

- **时间复杂度**：$O(|S|)$，两遍扫描各 $O(|S|)$
- **空间复杂度**：$O(|S|)$，用于存储标记数组

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;
    int n = (int)s.size();

    // 标记每个B是否被BC匹配使用
    vector<bool> b_used(n, false);

    // 第一遍：从右到左贪心匹配BC
    // 遇到C入栈，遇到B且栈非空则配对
    vector<int> c_stack;
    int bc_match = 0;
    for (int i = n - 1; i >= 0; i--) {
        if (s[i] == 'C') {
            c_stack.push_back(i);
        } else if (s[i] == 'B') {
            if (!c_stack.empty()) {
                // B(i) 在 C(c_stack.top()) 左边，满足操作要求
                b_used[i] = true;
                c_stack.pop_back();
                bc_match++;
            }
        }
    }

    // 第二遍：从左到右贪心匹配AB，只用未被BC匹配的B
    int ab_match = 0;
    int a_count = 0;
    for (int i = 0; i < n; i++) {
        if (s[i] == 'A') {
            a_count++;
        } else if (s[i] == 'B' && !b_used[i]) {
            if (a_count > 0) {
                ab_match++;
                a_count--;
            }
        }
    }

    cout << ab_match + bc_match << "\n";
    return 0;
}
```
