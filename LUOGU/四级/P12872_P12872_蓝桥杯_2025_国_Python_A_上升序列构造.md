# P12872 [蓝桥杯 2025 国 Python A] 上升序列构造

## 题目描述

小蓝手里有一个长度为 $n$ 的十进制正整数序列 $(a_1, a_2, \cdots, a_n)$，他希望修改这个序列中的一些数使其变为一个上升序列，即满足对于所有的 $i$，$2 \leq i \leq n$，都有 $a_{i-1} < a_i$。他只能通过在这些正整数的十进制表示中增加数字 0 来构造这个上升序列，他想知道最少增加多少个 0 可以满足条件。

## 输入格式

输入的第一行包含一个正整数 $n$。

第二行包含 $n$ 个正整数 $a_1, a_2, \cdots, a_n$，相邻整数之间使用一个空格分隔。

## 输出格式

输出一行包含一个整数表示答案。

## 样例

**输入样例 1**
```
6
527559 483873 913413 181072 822487 853172
```
**输出样例 1**
```
8
```

## 提示

**【样例说明】**

其中一种方案，更改后的序列为 $(527559, 4083873, 9013413, 10081072, 80022487, 85003172)$，共增加 8 个 $0$。

**【评测用例规模与约定】**

对于 20% 的评测用例，$1 \leq n \leq 10$；

对于所有评测用例，$1 \leq n \leq 5000$，$1 \leq a_i \leq 10^6$。增加 0 之后允许超过 $10^6$。

## 解题思路

### 问题分析

本题要求通过在数字的十进制表示中插入数字0，使序列变为严格递增，求最少插入0的个数。

关键观察：
- 插入0可以在数字的任意位置（首位、中间、末尾），但**前导0不改变数值**
- 插入k个0后，数值要么不变（0都在前导位置），要么位数增加（数值增大）
- 插入0的位置不同，得到的数值也不同

### 核心思路：动态规划

定义 $dp[i][k]$ 为：前 $i$ 个数构成递增序列，且第 $i$ 个数插入 $k$ 个0后的最优状态（最小总插入数 + 最小可能值）。

**状态转移**：对于第 $i$ 个数插入 $k$ 个0，遍历第 $i-1$ 个数的所有可能插入数 $k'$，在第 $i$ 个数插入 $k$ 个0后能得到的值中，找到大于第 $i-1$ 个数对应值的最小值，更新状态。

**关键预处理**：对于每个数 $a_i$ 和插入数 $k$，预处理出所有可能的值并排序，便于二分查找。

### 值的生成

一个 $d$ 位数插入 $k$ 个0后，相当于在 $d+1$ 个间隙（包括首尾）中选择位置放入0，保持原数各位的相对顺序。每种选择对应一个不同的值。用DFS枚举所有 $\binom{d+k}{k}$ 种情况。

### 算法流程

1. 提取每个数的各位数字
2. 对每个 $(a_i, k)$ 预处理所有可能值并排序
3. DP转移：$dp[i][k] = \min_{k'} \{ dp[i-1][k'].total + k,\ \text{大于}dp[i-1][k'].value\text{的最小值} \}$
4. 答案为 $\min_k dp[n-1][k].total$

### 复杂度分析

- **预处理**：$O(n \times K \times \binom{d+K}{K})$，其中 $K=7$ 为插入0上界
- **DP转移**：$O(n \times K^2 \times \log(\binom{d+K}{K}))$
- **空间复杂度**：$O(n \times K \times \binom{d+K}{K})$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <functional>
using namespace std;

typedef long long ll;
const ll INF = 1e18;

int n;
ll a[5005];
int digs[5005][8];  // 每个数的各位数字
int dlen[5005];      // 每个数的位数
vector<ll> vals[5005][8]; // vals[i][k] = a[i]插入k个0后的所有值（排序）

// 生成a[idx]插入k个0后的所有可能值
void gen_vals(int idx, int k) {
    int d = dlen[idx];
    vector<ll>& res = vals[idx][k];
    res.clear();

    function<void(int, int, ll)> dfs = [&](int pos, int rem, ll v) {
        if (pos == d) {
            if (rem == 0) res.push_back(v);
            return;
        }
        // 在当前位置之前插入z个0
        for (int z = 0; z <= rem; z++) {
            ll nv = v;
            for (int i = 0; i < z; i++) nv *= 10;
            nv = nv * 10 + digs[idx][pos];
            dfs(pos + 1, rem - z, nv);
        }
    };

    dfs(0, k, 0);
    sort(res.begin(), res.end());
}

// dp[k] = {min_inserts, min_value}
pair<ll, ll> prev_dp[8];
pair<ll, ll> curr_dp[8];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        ll x = a[i];
        vector<int> d;
        while (x > 0) {
            d.push_back(x % 10);
            x /= 10;
        }
        reverse(d.begin(), d.end());
        dlen[i] = (int)d.size();
        for (int j = 0; j < dlen[i]; j++) {
            digs[i][j] = d[j];
        }
    }

    // 预处理所有可能值
    for (int i = 0; i < n; i++) {
        for (int k = 0; k <= 7; k++) {
            gen_vals(i, k);
        }
    }

    // 初始化第0行
    for (int k = 0; k <= 7; k++) {
        if (!vals[0][k].empty()) {
            prev_dp[k] = {k, vals[0][k][0]};
        } else {
            prev_dp[k] = {INF, INF};
        }
    }

    // DP转移
    for (int i = 1; i < n; i++) {
        for (int k = 0; k <= 7; k++) {
            curr_dp[k] = {INF, INF};
            for (int k2 = 0; k2 <= 7; k2++) {
                if (prev_dp[k2].first == INF) continue;
                ll pv = prev_dp[k2].second;
                // 二分查找大于pv的最小值
                auto it = upper_bound(vals[i][k].begin(), vals[i][k].end(), pv);
                if (it != vals[i][k].end()) {
                    ll nt = prev_dp[k2].first + k;
                    ll nv = *it;
                    if (nt < curr_dp[k].first ||
                        (nt == curr_dp[k].first && nv < curr_dp[k].second)) {
                        curr_dp[k] = {nt, nv};
                    }
                }
            }
        }
        for (int k = 0; k <= 7; k++) {
            prev_dp[k] = curr_dp[k];
        }
    }

    ll ans = INF;
    for (int k = 0; k <= 7; k++) {
        ans = min(ans, prev_dp[k].first);
    }
    cout << ans << "\n";

    return 0;
}
```
