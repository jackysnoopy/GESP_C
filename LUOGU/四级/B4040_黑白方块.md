# [GESP202409 四级] 黑白方块

## 题目描述

小杨有一个 $n$ 行 $m$ 列的网格图，其中每个格子要么是白色，要么是黑色。

小杨想知道网格图中是否存在一个满足如下条件的子矩形：
- 子矩形由 $4$ 行 $4$ 列组成；
- 子矩形的第 $1$ 行和第 $4$ 行只包含白色格子；
- 对于子矩形的第 $2$ 行和第 $3$ 行，只有第 $1$ 个和第 $4$ 个格子是白色的，其余格子都是黑色的。

请你编写程序帮助小杨判断。

## 输入格式

第一行包含一个正整数 $t$，代表测试用例组数。

接下来是 $t$ 组测试用例。对于每组测试用例，一共 $n+1$ 行。

第一行包含两个正整数 $n,m$，含义如题面所示。

之后 $n$ 行，每行一个长度为 $m$ 的 $01$ 串，代表网格图第 $i$ 行格子的颜色，如果为 $0$，则对应格子为白色，否则为黑色。

## 输出格式

对于每组测试用例，如果存在，输出 Yes，否则输出 No。

## 样例

### 样例 #1

**输入**
```
3
1 4
0110
5 5
00000
01100
01100
00001
01100
5 5
00000
01100
01110
00001
01100
```
**输出**
```
No
Yes

```

## 解题思路

### 问题分析

在网格中查找是否存在一个 $4 \times 4$ 的子矩形，满足特定的黑白模式：

```
0000
0110
0110
0000
```

其中 $0$ 表示白色，$1$ 表示黑色。

### 核心思路

由于网格大小 $n,m \le 100$，直接枚举所有可能的 $4 \times 4$ 子矩形的左上角位置，检查是否匹配即可。

子矩形的行范围是 $[i, i+3]$，列范围是 $[j, j+3]$。对于每个候选子矩形，逐行逐列检查是否与目标模式一致。

### 算法流程

对每组测试数据：

1. 读取 $n,m$ 和网格。
2. 枚举左上角 $(i,j)$，其中 $0 \le i \le n-4$，$0 \le j \le m-4$。
3. 对每个候选，检查 $4$ 行 $4$ 列是否全部匹配模式：
   - 第 $1$ 行（$i$）：全为 `'0'`。
   - 第 $2$ 行（$i+1$）：第 $j$ 列和 $j+3$ 列为 `'0'`，$j+1$ 和 $j+2$ 列为 `'1'`。
   - 第 $3$ 行（$i+2$）：与第 $2$ 行相同。
   - 第 $4$ 行（$i+3$）：全为 `'0'`。
4. 如果找到匹配，输出 `Yes`，否则输出 `No`。

### 复杂度分析

- **时间复杂度：** $O(t \times n \times m \times 16) = O(t \times n \times m)$，每组数据最多枚举 $(n-3)(m-3)$ 个位置。
- **空间复杂度：** $O(n \times m)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t;
    cin >> t;
    while (t--) {
        int n, m;
        cin >> n >> m;
        vector<string> g(n);
        for (int i = 0; i < n; i++) cin >> g[i];
        bool found = false;
        for (int i = 0; i + 3 < n && !found; i++) {
            for (int j = 0; j + 3 < m && !found; j++) {
                bool ok = true;
                for (int r = 0; r < 4 && ok; r++) {
                    for (int c = 0; c < 4 && ok; c++) {
                        if (r == 0 || r == 3) {
                            if (g[i + r][j + c] != '0') ok = false;
                        } else {
                            if (c == 0 || c == 3) {
                                if (g[i + r][j + c] != '0') ok = false;
                            } else {
                                if (g[i + r][j + c] != '1') ok = false;
                            }
                        }
                    }
                }
                if (ok) found = true;
            }
        }
        cout << (found ? "Yes\n" : "No\n");
    }
    return 0;
}
```
