# P11969 「ALFR Round 7」T2 Game

## 题目描述

给定 $t,n$，以及长度为 $n$ 的排列 $a_{1\sim n}$，两个人轮流进行以下操作：

- 选择 $1\le i,j\le n$ 并且交换 $a_i,a_j$（$i, j$ 可以相同）。

两个人一共进行 $t$ 次操作（一个人算一次操作）。先手希望最终的排列字典序越小越好，后手希望字典序越大越好。若两人足够聪明，问最终得到的排列。

## 输入格式

第一行两个正整数 $t,n$。

第二行 $n$ 个 $1\sim n$ 的正整数，表示排列 $a$。

## 输出格式

一行一个排列，表示最终答案。

## 样例

**输入样例 1**
```
1 6
1 2 4 3 6 5
```
**输出样例 1**
```
1 2 3 4 6 5
```

## 解题思路

### 问题分析

本题是一个排列上的二人零和博弈问题。先手希望字典序最小（接近 $1,2,...,n$），后手希望字典序最大。每次操作可以交换任意两个位置。

### 核心思路

**操作次数分配**：先手操作 $\lceil t/2 \rceil$ 次，后手操作 $\lfloor t/2 \rfloor$ 次。

**关键观察**：

1. **当 $t \geq 2(n-1)$ 时**：排序一个排列最多需要 $n-1$ 次交换。先手有 $\lceil t/2 \rceil \geq n-1$ 次操作，可以完全排序。后手虽然会破坏，但先手每次修复后仍有富余操作继续排序。

2. **当 $t < 2(n-1)$ 时**：先手只能排好有限个位置。博弈的关键在于位置1的争夺：
   - 后手每次操作都会把位置1的值换成最大值（对字典序影响最大）
   - 先手如果去修复位置1，就没有多余操作排其他位置
   - 如果先手不修复位置1，位置1会被后手控制

**博弈平衡分析**：

- **$t$ 为奇数**（先手最后操作）：先手可以在最后一步把最小值放回位置1。第1次操作排好第一个错位对，后续操作与后手在位置1上拉锯，最终位置1 = 1。
  
- **$t$ 为偶数**（后手最后操作）：后手最终把最大值 $n$ 放到位置1。先手的策略是排一个不涉及位置1和最大值位置的错位对，这样后手的交换不会影响先手排好的位。

### 算法流程

1. 如果 $t \geq 2(n-1)$，直接输出 $1,2,...,n$。
2. 如果 $t$ 为奇数：找到第一个错位位置 $i$（$a[i] \neq i$），交换 $a[i]$ 和 $a[\text{pos}[i]]$（使该位置正确）。
3. 如果 $t$ 为偶数：找到一个不涉及位置1和最大值位置的错位对进行交换，然后把位置1换成最大值 $n$。

### 复杂度分析

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long t;
    int n;
    cin >> t >> n;
    
    vector<int> a(n);
    vector<int> pos(n + 1);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        pos[a[i]] = i;
    }
    
    // 操作次数足够多时先手可以完全排序
    if (t >= 2LL * (n - 1)) {
        for (int i = 1; i <= n; i++) {
            cout << i << (i < n ? " " : "\n");
        }
        return 0;
    }
    
    if (t % 2 == 1) {
        // t是奇数：先手排第一个错位对，最终位1=1
        for (int i = 0; i < n; i++) {
            if (a[i] != i + 1) {
                swap(a[i], a[pos[i + 1]]);
                break;
            }
        }
    } else {
        // t是偶数：先手排一个安全的错位对，后手控制位1
        int max_pos = pos[n];
        int swap_pos = -1;
        for (int i = 1; i < n; i++) {
            if (a[i] != i + 1 && i != max_pos && pos[i + 1] != 0) {
                swap_pos = i;
                break;
            }
        }
        if (swap_pos != -1) {
            swap(a[swap_pos], a[pos[swap_pos + 1]]);
        }
        if (max_pos != 0) {
            swap(a[0], a[max_pos]);
        }
    }
    
    for (int i = 0; i < n; i++) {
        cout << a[i] << (i < n - 1 ? " " : "\n");
    }
    
    return 0;
}
```
