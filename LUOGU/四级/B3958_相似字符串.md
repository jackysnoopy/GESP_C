# [GESP202403 四级] 相似字符串

## 题目描述

对于两个字符串 $A$ 和 $B$，如果 $A$ 可以通过删除一个字符，**或**插入一个字符，**或**修改一个字符变成 $B$，那么我们说 $A$ 和 $B$ 是相似的。

比如 $\texttt{apple}$ 可以通过插入一个字符变成 $\texttt{applee}$，可以通过删除一个字符变成 $\texttt{appe}$，也可以通过修改一个字符变成 $\texttt{bpple}$。因此 $\texttt{apple}$ 和 $\texttt{applee}$、$\texttt{appe}$、$\texttt{bpple}$ 都是相似的。但 $\texttt{applee}$ 并不能 通过任意一个操作变成 $\texttt{bpple}$，因此它们并不相似。

特别地，两个完全相同的字符串也是相似的。

给定 $T$ 组 $A,B$，请你分别判断它们是否相似。

## 输入格式

第一行一个正整数 $T$。  
接下来 $T$ 行，每行两个用空格隔开的字符串 $A$ 和 $B$。  

## 输出格式

对组 $A,B$，如果他们相似，输出 ``similar``，否则输出 ``not similar``。

## 样例

### 样例 #1

**输入**
```
5
apple applee
apple appe
apple bpple
applee bpple
apple apple
```

**输出**
```
similar
similar
similar
not similar
similar
```

## 提示

对全部的测试数据，保证 $1 \leq T \leq 100$，$A$ 和 $B$ 的长度不超过 $50$，仅含小写字母。

## 解题思路

### 问题分析

判断两个字符串是否可以通过一次编辑操作（增/删/改）变得相同。等价于计算编辑距离是否为 0 或 1。

### 核心思路

根据两个字符串的长度差分类讨论：

1. **长度相等**：检查不同字符的个数。如果不同字符数 $\le 1$，则相似（0个不同则完全相同，1个不同则修改一次）。
2. **长度差 1**：检查是否可以通过一次插入或删除操作匹配。用双指针同时遍历两个字符串，遇到不同字符时，较长字符串的指针+1，同时记录操作次数。
   - 如果操作次数 $\le 1$，则相似。
3. **长度差 $\ge 2$**：一次操作不可能完成，直接 `not similar`。

### 算法流程

1. 读入 $T$。
2. 对每组 $A, B$：
   - 如果 $|A| = |B|$：统计不同位置数量 $\le 1$ 则相似。
   - 如果 $|A| - |B| = 1$：$A$ 比 $B$ 多一个字符，用双指针检查是否 $A$ 删除一个字符可变成 $B$。
   - 如果 $|B| - |A| = 1$：同理，$B$ 删除一个字符可变成 $A$。
   - 否则，不相似。
3. 输出结果。

### 复杂度分析

- **时间复杂度：** $O(|A| + |B|)$，每个测试用例线性扫描一次。
- **空间复杂度：** $O(1)$。

## 参考代码

```cpp
#include <iostream>
#include <string>
using namespace std;

bool similar(const string &a, const string &b) {
    int la = a.length(), lb = b.length();
    if (la == lb) {
        int diff = 0;
        for (int i = 0; i < la; i++) {
            if (a[i] != b[i]) diff++;
        }
        return diff <= 1;
    }
    if (la - lb == 1) {
        int i = 0, j = 0, op = 0;
        while (i < la && j < lb) {
            if (a[i] != b[j]) {
                i++;
                op++;
            } else {
                i++; j++;
            }
        }
        return op <= 1;
    }
    if (lb - la == 1) {
        int i = 0, j = 0, op = 0;
        while (i < la && j < lb) {
            if (a[i] != b[j]) {
                j++;
                op++;
            } else {
                i++; j++;
            }
        }
        return op <= 1;
    }
    return false;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        string a, b;
        cin >> a >> b;
        cout << (similar(a, b) ? "similar\n" : "not similar\n");
    }
    return 0;
}
```
