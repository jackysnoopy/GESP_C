# P12007 【MX-X10-T3】[LSOT-4] 全国联赛？

## 题目描述

北宇治的吹奏部一共有 $n$ 个学生，编号为 $1$ 到 $n$。已建立 $m$ 对配合关系（$0 \le m \le n-1$），每对配合关系 $u,v,w$ 表示在 $u$ 或 $v$ 演奏后另一人能在 $w$ 单位的时间后立刻演奏完成配合。若有 $n-m-1$ 种特训方案，第 $i$ 种方案可以让任意两个成员建立配合关系，训练效果为 $a_i$。

定义不协调度为所有 $1 \le x < y \le n$ 的 $x,y$ 之间误差时间的最小值之和。要求选择新配合关系的端点，使不协调度最小。输出对 $10^9+7$ 取模。

## 输入格式

第一行两个整数 $n, m$。接下来 $m$ 行每行 $u,v,w$。最后一行 $n-m-1$ 个整数 $a_1, \ldots, a_{n-m-1}$。

## 输出格式

一个整数，最小不协调度对 $10^9+7$ 取模。

## 样例

**输入样例 1**
```
7 3
1 2 1
3 5 3
3 7 2
1 2 3
```
**输出样例 1**
```
76
```

## 解题思路

### 问题分析

本题本质是在已有 $m$ 条边（森林）基础上，添加 $n-m-1$ 条权值给定的新边（端点可选），构成一棵生成树，使所有点对最短路径之和最小。

### 核心思路

在树上，所有点对最短路径之和等于每条边的贡献之和：边权 $w$ 乘以其两侧子树大小之积 $s_1 \times s_2$。

1. **已有边**：端点固定，先按权值排序做 Kruskal，加入不形成环的边。
2. **新边**：权值给定、端点可选，按权值从小到大处理。每次选择两个最大的连通块合并（让小权值边跨越大连通块，大权值边跨越小连通块，最小化加权贡献）。
3. **计算贡献**：在最终生成树上做 DFS，对每条边计算 $w \times s \times (n-s)$，其中 $s$ 是子树大小。

### 算法流程

1. 初始化并查集，按权值排序加入已有边。
2. 收集所有连通块，放入优先队列。
3. 新边权值从小到大，每次弹出最大两个连通块合并，记录边信息。
4. 在生成树上 DFS 计算每条边的贡献并求和。

### 复杂度分析

- 时间复杂度：$O(n \log n)$（排序 + 优先队列 + DFS）
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
#include <functional>
using namespace std;

const int MAXN = 1000006;
const long long MOD = 1e9 + 7;

int fa[MAXN], sz[MAXN];

int find(int x) {
    while (fa[x] != x) { fa[x] = fa[fa[x]]; x = fa[x]; }
    return x;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    for (int i = 1; i <= n; i++) { fa[i] = i; sz[i] = 1; }
    
    struct Edge { int u, v; long long w; };
    vector<Edge> edges;
    for (int i = 0; i < m; i++) {
        int u, v; long long w;
        cin >> u >> v >> w;
        edges.push_back({u, v, w});
    }
    
    int need = n - m - 1;
    vector<long long> new_edges;
    if (need > 0) {
        new_edges.resize(need);
        for (int i = 0; i < need; i++) cin >> new_edges[i];
    }
    
    sort(edges.begin(), edges.end(),
         [](const Edge& a, const Edge& b) { return a.w < b.w; });
    sort(new_edges.begin(), new_edges.end());
    
    // 已有边建树
    vector<vector<pair<int,long long>>> tree(n + 1);
    for (auto& e : edges) {
        int fu = find(e.u), fv = find(e.v);
        if (fu != fv) {
            tree[e.u].push_back({e.v, e.w});
            tree[e.v].push_back({e.u, e.w});
            if (sz[fu] < sz[fv]) swap(fu, fv);
            fa[fv] = fu; sz[fu] += sz[fv];
        }
    }
    
    // 新边连接最大两个连通块
    priority_queue<pair<long long, int>> pq;
    for (int i = 1; i <= n; i++) {
        if (fa[i] == i) pq.push({sz[i], i});
    }
    
    for (long long w : new_edges) {
        if (pq.size() < 2) break;
        auto p1 = pq.top(); pq.pop();
        auto p2 = pq.top(); pq.pop();
        tree[p1.second].push_back({p2.second, w});
        tree[p2.second].push_back({p1.second, w});
        int r = p1.second;
        long long nsz = p1.first + p2.first;
        pq.push({nsz, r});
    }
    
    long long ans = 0;
    vector<int> subtree(n + 1, 0);
    vector<bool> vis(n + 1, false);
    
    function<void(int, int)> dfs = [&](int u, int p) {
        vis[u] = true;
        subtree[u] = 1;
        for (auto& e : tree[u]) {
            if (e.first == p) continue;
            dfs(e.first, u);
            subtree[u] += subtree[e.first];
            long long s1 = subtree[e.first];
            long long s2 = n - s1;
            ans = (ans + (e.second % MOD) * (s1 % MOD) % MOD * (s2 % MOD)) % MOD;
        }
    };
    
    dfs(1, -1);
    cout << ans << endl;
    return 0;
}
```
