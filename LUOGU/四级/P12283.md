# [蓝桥杯 2024 国 Python A] 排练

## 题目描述

小蓝的班上有 $n$ 名同学正在为校庆活动排练节目，他们在舞台上站成一排。若把他们看做在数轴上，从左到右第 $i$ 名同学的位置即为 $a_i$。为了美观，小蓝想在队伍中插入一些新同学，使得在新队伍中，对于任意三名相邻的同学的位置 $a_{i-1}, a_i, a_{i+1}$ 满足 $a_{i+1} - a_i \leq 2(a_i - a_{i-1})$。小蓝想知道最少增加多少名同学可以满足条件。

## 输入格式

输入的第一行包含一个正整数 $n$。

第二行包含 $n$ 个正整数 $a_1, a_2, \cdots, a_n$，相邻整数之间使用一个空格分隔。

## 输出格式

输出一行包含一个整数表示答案。

## 样例

### 样例 1

**输入：**
```
4
1 3 16 17
```

**输出：**
```
2
```

### 样例说明

其中一种方案：在 $6, 10$ 处插入两名同学，新队伍为 $1, 3, 6, 10, 16, 17$，满足条件。

验证：
- $(3-1)=2$，$(6-3)=3$，$(10-6)=4$：$3 \le 2\times 2=4$ ✓，$4 \le 2\times 3=6$ ✓
- $(10-6)=4$，$(16-10)=6$，$(17-16)=1$：$6 \le 2\times 4=8$ ✓，$1 \le 2\times 6=12$ ✓

## 提示

- 对于 $20\%$ 的评测用例，$1 \leq n \leq 10^3$；
- 对于所有评测用例，$1 \leq n \leq 10^5$，$1 \leq a_i \leq 10^8$，$a_{i+1} > a_i$。

## 解题思路

### 问题分析

我们需要在递增序列的相邻元素之间插入新元素，使得对于任意三个相邻元素，后一个差值不超过前一个差值的两倍。目标是让插入数量最少。

关键观察：**插入只发生在相邻原始位置之间，不同间隔之间相互独立。** 因为条件只涉及连续三个位置，对于相邻原始位置 $a_{i-1}$ 和 $a_i$ 之间的插入不影响 $a_{i-2}$ 和 $a_{i-1}$ 之间的约束。

### 核心思路

对于每对相邻原始位置 $(a_{i-1}, a_i)$，设它们之间的间距为 `gap = a[i] - a[i-1]`，进入这个间隔时的"前置差值"为 `prev_diff = a[i-1] - a[i-2]`（第一个间隔的前置差值就是 `gap` 本身，因为我们假设前面没有额外插入）。

在间隔内，我们从 $a_{i-1}$ 出发，每一步的差值最多是前一步差值的 2 倍。为了最小化插入数量，我们应该**尽可能增大每步的差值**。

贪心策略：每步差值取 `min(gap - 已走距离, 2 × 前一步差值)`，直到剩余距离可以在一步内走完。

### 算法流程

1. 对每对相邻位置 $(a_{i-1}, a_i)$：
   - 计算 `gap = a[i] - a[i-1]`，`prev_diff` 为进入间隔的前置差值
   - 从 `cur_max = 2 * prev_diff` 开始，每步走过 `cur_max` 距离（差值最大化）
   - 如果 `已走距离 + cur_max < gap`，说明还需要更多插入：累加 `cur_max` 到已走距离，`inserts++`，`cur_max` 翻倍
   - 当 `已走距离 + cur_max >= gap` 时停止，最后一步直接到达 $a_i$
2. 将所有间隔的 `inserts` 求和

### 正确性证明

贪心的正确性：如果在某步选择一个更小的差值 $d' < d_{max}$，那么下一步的最大差值也会变小（$2d' < 2d_{max}$）。这意味着后续能走的总距离只会更短，需要更多的步数。因此最大化每步差值是最优策略。

### 复杂度分析

- **时间复杂度**：$O(n \log V)$，其中 $V = 10^8$ 是位置的最大值。每对相邻位置的处理中，差值每步翻倍，所以最多 $\log_2(V) \approx 27$ 步。
- **空间复杂度**：$O(n)$，存储位置数组。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    long long ans = 0;
    
    for (int i = 1; i < n; i++) {
        long long gap = a[i] - a[i - 1];
        // 前置差值：上一对相邻位置的间距
        long long prev_diff = (i >= 2) ? a[i - 1] - a[i - 2] : gap;
        
        // 贪心：每步差值最大化（2倍增长），直到剩余距离可一步到达
        long long used = 0;           // 已走距离
        long long cur_max = 2 * prev_diff;  // 当前步最大差值
        int inserts = 0;
        
        while (used + cur_max < gap) {
            used += cur_max;
            inserts++;
            cur_max = 2 * cur_max;
            if (cur_max > gap) cur_max = gap;  // 防溢出
        }
        // 剩余 gap - used <= cur_max，一步到达 a[i]
        
        ans += inserts;
    }
    
    cout << ans << "\n";
    return 0;
}
```
