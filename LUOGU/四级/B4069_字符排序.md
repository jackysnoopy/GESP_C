# [GESP202412 四级] 字符排序

## 题目描述

小杨有 $n$ 个仅包含小写字母的字符串 $s_1,s_2,\ldots,s_n$，小杨想将这些字符串按一定顺序排列后拼接到一起构成字符串 $t$。小杨希望最后构成的字符串 $t$ 满足：

- 假设 $t_i$ 为字符串 $t$ 的第 $i$ 个字符，对于所有的 $j\lt i$ 均有 $t_j\le t_i$。两个字符的大小关系与其在字母表中的顺序一致，例如 $\texttt{e}\lt \texttt{g}\lt \texttt{p} \lt \texttt{s}$。

小杨想知道是否存在满足条件的字符串排列顺序。

## 输入格式

第一行包含一个正整数 $T$，代表测试数据组数。

对于每组测试数据，第一行包含一个正整数 $n$，含义如题面所示。

之后 $n$ 行，每行包含一个字符串 $s_i$。

## 输出格式

对于每组测试数据，如果存在满足条件的排列顺序，输出（一行一个）$\texttt{1}$，否则输出（一行一个） $\texttt{0}$。

## 样例

### 样例 #1

**输入**
```
3
3
aa
ac
de
2
aac
bc
1
gesp
```
**输出**
```
1
0
0
```

## 解题思路

### 问题分析

需要判断能否将 $n$ 个字符串排列成一个顺序，使得拼接后的字符串 $t$ 是非递减的（即每个字符 $\le$ 后面的字符）。

### 核心思路

1. **每个字符串内部必须有序**：由于拼接后的 $t$ 是非递减的，每个字符串内部的字符也必须是非递减的。如果有任何一个字符串内部不是非递减的，则答案为 $0$。

2. **排序后验证**：将字符串按（第一个字符，最后一个字符）排序。因为要让整个 $t$ 有序，前面的字符串的最后一个字符必须 $\le$ 后面的字符串的第一个字符。

3. 排序后拼接所有字符串，检查拼接结果是否非递减。

### 算法流程

对每组测试数据：

1. 读取 $n$ 和 $n$ 个字符串。
2. 检查每个字符串是否内部有序（非递减）。若存在无序字符串，输出 $0$ 并继续下一组。
3. 将字符串按（第一个字符，最后一个字符）排序，若第一个字符相同则按最后一个字符排序，若仍相同则按字符串本身排序。
4. 将所有字符串拼接起来。
5. 检查拼接结果是否非递减，若是则输出 $1$，否则输出 $0$。

### 复杂度分析

- **时间复杂度：** $O(n \log n \times L)$，其中 $L$ 为字符串平均长度。
- **空间复杂度：** $O(n \times L)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        vector<string> s(n);
        bool ok = true;
        for (int i = 0; i < n; i++) {
            cin >> s[i];
            for (int j = 1; j < (int)s[i].size(); j++) {
                if (s[i][j] < s[i][j - 1]) ok = false;
            }
        }
        if (!ok) {
            cout << "0\n";
            continue;
        }
        sort(s.begin(), s.end(), [](const string &a, const string &b) {
            if (a[0] != b[0]) return a[0] < b[0];
            if (a.back() != b.back()) return a.back() < b.back();
            return a < b;
        });
        string t;
        for (const string &str : s) t += str;
        ok = true;
        for (int i = 1; i < (int)t.size(); i++) {
            if (t[i] < t[i - 1]) {
                ok = false;
                break;
            }
        }
        cout << (ok ? "1\n" : "0\n");
    }
    return 0;
}
```
