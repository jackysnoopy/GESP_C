# P12568 使元素互异

## 题目描述

给定一个长度为 n 的数组 a。每次操作可以选择一个子数组 [l, r] 和一个值 x，将子数组中所有元素加上 x。

要求使得数组中所有元素互不相同，求最少需要多少次操作。

## 输入格式

第一行一个整数 n，表示数组长度。
第二行 n 个整数 a[i]，表示数组元素。

## 输出格式

一个整数，表示最少操作次数。

## 样例

**样例输入：**
```
5
1 2 2 3 3
```

**样例输出：**
```
2
```

## 提示

- 1 ≤ n ≤ 100000
- -10^9 ≤ a[i] ≤ 10^9

## 解题思路

### 问题分析

本题要求通过子数组加法操作使所有元素互异。关键在于统计重复元素的数量。

### 核心思路

核心观察：每次操作最多可以修复一个重复值。具体来说：
1. 统计数组中重复元素的总数
2. 每个值如果出现 k 次，则需要 k-1 次操作来使其互异
3. 总操作次数 = 所有重复元素的数量

更精确地，答案等于：n - 不同值的数量。

### 算法流程

1. 对数组排序
2. 统计相邻元素中相等的对数（即重复数）
3. 答案就是重复数

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    vector<long long> sorted_a = a;
    sort(sorted_a.begin(), sorted_a.end());
    int duplicates = 0;
    for (int i = 1; i < n; i++) {
        if (sorted_a[i] == sorted_a[i-1]) duplicates++;
    }
    cout << duplicates << endl;
    return 0;
}
```
