# [NOISG 2019] Feast

## 题目描述

$N$ 盘食物排成一列，第 $i$ 盘满足感为 $A_i$（可能为负）。$K$ 个人各分配一段连续食物，区间不重叠。求最大满足感总和。

## 输入格式

第一行两个整数 $N, K$。
第二行 $N$ 个整数 $A_1, A_2, \ldots, A_N$。

## 输出格式

输出一个整数表示最大满足感总和。

## 样例

### 样例 1
输入：
```
6 1
1 -2 3 -1 5 -6
```
输出：
```
7
```

## 提示

- $1 \le K \le N \le 3 \times 10^5$

## 解题思路

### 问题分析

选择 $K$ 个不重叠连续子段，最大化总和。

### 核心思路

DP：$dp[i][j]$ 表示前 $i$ 个元素选 $j$ 个子段的最大和。

### 复杂度分析

- 时间复杂度：$O(N \times K)$
- 空间复杂度：$O(N \times K)$
