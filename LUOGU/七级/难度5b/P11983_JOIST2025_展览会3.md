# [JOIST 2025] 展览会 3

## 题目描述
N 幅画作美观值 A_i，M 家杂志报道连续段 [L_j, R_j]。排列画作使杂志报道吸引力序列字典序最大。

## 输入格式
第一行 N M，第二行 A_i，接下来 M 行每行 L_j R_j。

## 输出格式
M 行，每行杂志报道的吸引力。

## 样例
输入：
```
4 4
1 2 1 2
1 1
2 3
4 4
3 4
```
输出：
```
2
2
1
2
```

## 提示
- N, M ≤ 10^5, A_i ≤ N

## 解题思路

### 问题分析
需要排列画作使得影响力大的杂志报道吸引力最大。

### 核心思路
贪心按影响力从大到小处理杂志，优先分配最大美观值的画作。

### 算法流程
1. 按杂志影响力排序
2. 对每家杂志，在其覆盖范围内选择最大可用美观值

### 复杂度分析
- 时间：O(NM)
- 空间：O(N + M)
