# [GCJ 2020 #1C] Oversized Pancake Choppers

## 题目描述

有 $N$ 个圆形煎饼切片，每个内角为 $A_i$ 纳度。需要 $D$ 个食客，每人要一块相同大小的切片。求最少切割次数。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $N, D$，第二行 $N$ 个整数 $A_1, \ldots, A_N$。

## 输出格式

每个测试用例输出 `Case #x: y`。

## 样例

### 样例 1
输入：
```
4
1 3
1
5 2
10 5 359999999999 123456789 10
...
```
输出：
```
Case #1: 2
Case #2: 0
...
```

## 提示

- $1 \le T \le 100$, $1 \le A_i < 360 \times 10^9$

## 解题思路

### 问题分析

贪心问题，对于每个可能的目标大小，计算最少切割次数。

### 核心思路

枚举可能的目标大小，贪心地从大到小切割。

### 复杂度分析

- 时间复杂度：$O(T \times N \times D)$
- 空间复杂度：$O(N)$
