# [KOI 2022 Round 2] 食事计划

## 题目描述

有 $N$ 个餐厅，每个餐厅售卖一种食物。找到字典序最小的排列 $P$，使得相邻餐厅的食物类型不同。

## 输入格式

第一行整数 $N$。
第二行 $N$ 个整数 $A_1, A_2, \ldots, A_N$。

## 输出格式

若无合法排列输出 $-1$，否则输出字典序最小的合法排列。

## 样例

### 样例 1
输入：
```
9
1 1 1 2 2 3 3 4 3
```
输出：
```
1 4 2 5 6 3 7 8 9
```

## 提示

- $1 \le N \le 300000$

## 解题思路

### 问题分析

贪心构造问题，需要找到字典序最小的合法排列。

### 核心思路

贪心地每次选择最小的可用食物类型，确保不超过总数的一半。

### 复杂度分析

- 时间复杂度：$O(N \log N)$
- 空间复杂度：$O(N)$
