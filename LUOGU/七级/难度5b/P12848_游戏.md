# [蓝桥杯 2025 国 A] 游戏

## 题目描述

$n$ 个槽位，$n-1$ 个石块，初始第 $n$ 个槽位为空。目标是将编号为 $i$ 的石块移动到编号为 $i$ 的槽位。可以花费 1 的代价使任意两个槽位相连。求最小代价。

## 输入格式

第一行整数 $n$。
第二行 $n-1$ 个正整数 $a_1, a_2, \ldots, a_{n-1}$。

## 输出格式

输出一个整数表示最小代价。

## 样例

### 样例 1
输入：
```
5
4 1 2 3
```
输出：
```
1
```

## 提示

- $1 \le n \le 3 \times 10^5$

## 解题思路

### 问题分析

置换环问题，计算置换的环数，答案为环数减1。

### 核心思路

将石块位置看作置换，计算环数，最小代价为环数减1。

### 复杂度分析

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
