# [CSP-S 2023] 种树

## 题目描述

有 n 片地块组成一棵树，每片地块需要种一棵树并使其高度不低于 a_i。第 1 天在 1 号地块种树，之后每天可以选择一个与已种树地块邻接的未种树地块种树。第 x 天，i 号地块的树长高 max(b_i + x × c_i, 1) 米。求使所有树都长到目标高度的最少天数。

## 输入格式

第一行一个整数 n。  
第二行 n 个整数 a_i。  
第三行 n 个整数 b_i。  
第四行 n 个整数 c_i。  
接下来 n-1 行，每行两个整数 u, v，表示树上的一条边。

## 输出格式

一个整数，表示最少天数。

## 样例

### 样例 1

**输入：**
```
3
3 5 2
1 1 1
1 0 0
1 2
1 3
```

**输出：**
```
5
```

## 提示

- 1 ≤ n ≤ 10^5
- 0 ≤ a_i ≤ 10^18
- |b_i| ≤ 10^9
- |c_i| ≤ 10^9

## 解题思路

### 问题分析

我们需要在树上安排种植顺序，使得每棵树在种植后到第 T 天的累计生长量不低于 a_i。生长量取决于种植日期和参数 b_i, c_i。由于种植必须从 1 号节点开始并向邻接节点扩展，这形成了一个树上的 BFS 序约束。

### 核心思路

**二分答案 + 树形贪心**：

1. **二分天数 T**：答案具有单调性，若 T 天可行，则 T+1 天也可行。
2. **计算最晚种植日 L_i**：对于每个节点 i，计算它最晚可以在第几天种下，仍能长到 a_i。这需要计算从种植日到第 T 天的累计生长量。
3. **树形 DP 检查可行性**：从叶子向上贪心，每个节点尽量晚种，为父节点留出更多时间。按子节点的截止时间升序排列，依次分配时间。

### 算法流程

1. 二分 T ∈ [n, 2×10^18]。
2. 对每个 T，计算每个节点的最晚种植日 L_i（二分或公式计算）。
3. 树形 DP：对每个节点 u，按子节点 D[v] 升序排列，D[u] = min(L_u, min(D[v_i] - 1 - sum_before)。
4. 若 D[1] ≥ 1，则 T 可行。

### 复杂度分析

- 时间：O(n × log(max_T) × log(n))，二分 T 约 60 次，每次 O(n)。
- 空间：O(n)。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 100005;
vector<int> adj[MAXN];
long long a[MAXN], b[MAXN], c[MAXN];
int n;
long long T;

long long cappedSum(long long bi, long long ci, long long x, long long cap) {
    if (x > T) return 0;
    __int128 sum = 0;
    __int128 cap128 = cap;

    if (ci == 0) {
        long long g = max(bi, 1LL);
        __int128 num = T - x + 1;
        sum = num * g;
        if (sum >= cap128) return cap;
        return (long long)sum;
    }

    if (ci > 0) {
        long long j0;
        if (bi + ci >= 1) j0 = 1;
        else {
            long long num = 1 - bi;
            j0 = (num + ci - 1) / ci;
            if (j0 < 1) j0 = 1;
        }
        long long low = max(j0, x);
        if (low > x) {
            sum += low - x;
            if (sum >= cap128) return cap;
        }
        if (low <= T) {
            __int128 num = T - low + 1;
            sum += num * bi + (__int128)ci * (low + T) * num / 2;
            if (sum >= cap128) return cap;
        }
    } else {
        long long j0 = (bi < 1) ? 0 : (bi - 1) / (-ci);
        long long p1_end = min(j0, T);
        if (p1_end >= x) {
            __int128 num = p1_end - x + 1;
            sum += num * bi + (__int128)ci * (x + p1_end) * num / 2;
            if (sum >= cap128) return cap;
        }
        long long p2_start = max(j0 + 1, x);
        if (p2_start <= T) {
            sum += T - p2_start + 1;
            if (sum >= cap128) return cap;
        }
    }
    if (sum >= cap128) return cap;
    return (long long)sum;
}

long long computeL(int i) {
    if (a[i] == 0) return T;
    long long lo = 1, hi = T;
    while (lo < hi) {
        long long mid = (lo + hi + 1) / 2;
        if (cappedSum(b[i], c[i], mid, a[i]) >= a[i]) lo = mid;
        else hi = mid - 1;
    }
    return lo;
}

long long D[MAXN];
int sz[MAXN];

void dfs(int u, int parent) {
    sz[u] = 1;
    vector<pair<long long, int>> children;
    for (int v : adj[u]) {
        if (v == parent) continue;
        dfs(v, u);
        children.push_back({D[v], sz[v]});
        sz[u] += sz[v];
    }
    sort(children.begin(), children.end());
    D[u] = computeL(u);
    long long sum = 0;
    for (size_t i = 0; i < children.size(); i++) {
        D[u] = min(D[u], children[i].first - 1 - sum);
        sum += children[i].second;
    }
}

bool check(long long T_val) {
    T = T_val;
    for (int i = 1; i <= n; i++) { D[i] = 0; sz[i] = 0; }
    dfs(1, 0);
    return D[1] >= 1;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= n; i++) cin >> b[i];
    for (int i = 1; i <= n; i++) cin >> c[i];
    for (int i = 0; i < n - 1; i++) {
        int u, v; cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    long long lo = n, hi = 2000000000000000000LL;
    while (lo < hi) {
        long long mid = (lo + hi) / 2;
        if (check(mid)) hi = mid;
        else lo = mid + 1;
    }
    cout << lo << endl;
    return 0;
}
```
