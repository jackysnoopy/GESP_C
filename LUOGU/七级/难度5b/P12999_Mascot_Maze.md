# [GCJ 2022 #3] Mascot Maze

## 题目描述

$N$ 个房间组成的迷宫，每个房间有左右两个出口。用 13 个吉祥物字母给房间着色，使得任意三个连续房间的吉祥物都不同。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $N$，第二行 $L_1, \ldots, L_N$，第三行 $R_1, \ldots, R_N$。

## 输出格式

每个测试用例输出 `Case #x: y`，$y$ 为着色方案或 `IMPOSSIBLE`。

## 样例

### 样例 1
输入：
```
4
3
2 1 1
3 3 2
6
3 1 4 1 2 3
5 3 5 2 4 5
...
```
输出：
```
Case #1: IMPOSSIBLE
Case #2: TSHIRT
...
```

## 提示

- $3 \le N \le 10^5$

## 解题思路

### 问题分析

图着色问题，约束为任意三个连续节点颜色不同。

### 核心思路

贪心着色，对每个节点选择与其距离 1 和 2 的节点都不同的颜色。

### 复杂度分析

- 时间复杂度：$O(N)$
- 空间复杂度：$O(N)$
