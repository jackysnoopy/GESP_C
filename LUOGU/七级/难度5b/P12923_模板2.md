# [POI 2021/2022 R3] 模板 2

## 题目描述

给定一个字符串，找到所有可能的模板长度 $L$，使得存在一个长度为 $L$ 的模板及其反转能拼出原字符串。

## 输入格式

一行字符串。

## 输出格式

一行所有可能的模板长度，升序排列，空格分隔。

## 样例

### 样例 1
输入：
```
abcabcabacbabcab
```
输出：
```
5 16
```

## 提示

- $1 \le n \le 10^6$

## 解题思路

### 问题分析

字符串匹配问题，需要找到所有可能的模板长度。

### 核心思路

对于每个长度 $L$，检查是否存在模板使得原字符串能被覆盖。

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$
