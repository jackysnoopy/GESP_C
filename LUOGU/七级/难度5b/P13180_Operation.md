# [GCJ 2017 Finals] Operation

## 题目描述

卡片游戏，每张卡片有运算符和操作数。给定起始值 $S$，选择卡片顺序使得最终结果最大。输出为不可约分数。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $S, C$，接下来 $C$ 行每行 $O_i, V_i$。

## 输出格式

每个测试用例输出 `Case #x: y z`，表示 $y/z$。

## 样例

### 样例 1
输入：
```
5
1 2
- 3
* 2
...
```
输出：
```
Case #1: -1 1
Case #2: -3 2
...
```

## 提示

- $1 \le C \le 1000$

## 解题思路

### 问题分析

排列优化问题，需要找到最优卡片顺序。

### 核心思路

分类讨论：加法、减法、乘法、除法的最优排列策略。

### 复杂度分析

- 时间复杂度：$O(C \log C)$
- 空间复杂度：$O(C)$
