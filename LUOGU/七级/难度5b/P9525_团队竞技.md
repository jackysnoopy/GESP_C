# [JOIST 2022] 团队竞技 / Team Contest

## 题目描述

JOI 大学有 N 只海狸，每只有三项能力值（思考值 X_i，行动值 Y_i，运气值 Z_i）。选 3 只组成队伍，要求每个成员都有一项能力值严格大于其他两人的对应能力值。最大化三人最大 X + 最大 Y + 最大 Z 之和。

## 输入格式

第一行一个整数 N。
接下来 N 行，每行三个整数 X_i, Y_i, Z_i。

## 输出格式

一行一个整数，不存在则输出 -1。

## 样例

### 样例 1
输入：
```
5
3 1 4
2 3 1
1 5 5
4 4 2
5 2 3
```
输出：
```
13
```

### 样例 2
输入：
```
8
1 1 1
1 1 5
1 5 1
5 1 1
1 5 5
5 1 5
5 5 1
5 5 5
```
输出：
```
15
```

### 样例 3
输入：
```
4
1 2 3
1 2 3
1 2 3
1 2 3
```
输出：
```
-1
```

## 提示

- 3 ≤ N ≤ 150000
- 1 ≤ X_i, Y_i, Z_i ≤ 10^8

## 解题思路

### 问题分析

需要选 3 只海狸组队，每人恰好有一项能力严格最大。最大化总能力值。

### 核心思路

枚举 + 排序 + 前缀最大值。

由于每人必须有一项严格最大的能力，三人分别负责 X、Y、Z。对每种能力排序后，取 top 候选进行检查。

### 算法流程

1. 对 X、Y、Z 分别按降序排序
2. 枚举 6 种分配方式（谁负责哪种能力）
3. 对每种分配，从排序后的列表中取 top-5 候选进行三重枚举
4. 检查是否满足"严格大于"条件，更新答案

### 复杂度分析

- 时间复杂度：O(N log N)
- 空间复杂度：O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

struct Beaver {
    int x, y, z;
};

int n;
Beaver b[150005];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n;
    for (int i = 0; i < n; i++) {
        cin >> b[i].x >> b[i].y >> b[i].z;
    }
    
    if (n < 3) {
        cout << -1 << "\n";
        return 0;
    }
    
    vector<int> idx_x(n), idx_y(n), idx_z(n);
    for (int i = 0; i < n; i++) idx_x[i] = idx_y[i] = idx_z[i] = i;
    
    sort(idx_x.begin(), idx_x.end(), [&](int a, int b) { return b.x < a.x; });
    sort(idx_y.begin(), idx_y.end(), [&](int a, int b) { return b.y < a.y; });
    sort(idx_z.begin(), idx_z.end(), [&](int a, int b) { return b.z < a.z; });
    
    long long ans = -1;
    
    int perms[6][3] = {{0,1,2},{0,2,1},{1,0,2},{1,2,0},{2,0,1},{2,1,0}};
    
    for (int p = 0; p < 6; p++) {
        for (int i = 0; i < min(5, n); i++) {
            for (int j = 0; j < min(5, n); j++) {
                for (int k = 0; k < min(5, n); k++) {
                    int bi = idx_x[i];
                    int bj = idx_y[j];
                    int bk = idx_z[k];
                    
                    if (bi == bj || bi == bk || bj == bk) continue;
                    
                    if (b[bi].x <= b[bj].x || b[bi].x <= b[bk].x) continue;
                    if (b[bj].y <= b[bi].y || b[bj].y <= b[bk].y) continue;
                    if (b[bk].z <= b[bi].z || b[bk].z <= b[bj].z) continue;
                    
                    long long total = (long long)max({b[bi].x, b[bj].x, b[bk].x}) +
                                      (long long)max({b[bi].y, b[bj].y, b[bk].y}) +
                                      (long long)max({b[bi].z, b[bj].z, b[bk].z});
                    ans = max(ans, total);
                }
            }
        }
    }
    
    cout << ans << "\n";
    return 0;
}
```
