# 反回文串

## 题目描述

给定一个长度为 $n$ 的字符串 $s$，你需要把 $s$ 分成若干个非空子序列，使得每一个子序列都**不是**回文的，并最大化划分成的子序列数。

## 输入格式

第一行包含一个正整数 $c$，表示测试点编号。$c=0$ 表示该测试点为样例。

第二行包含一个正整数 $q$，表示测试数据组数。

对于每组测试数据：
- 第一行包含一个正整数 $n$
- 第二行包含一个长度为 $n$ 的字符串 $s$

## 输出格式

对于每组测试数据：
- 如果不存在合法方案，输出 `Shuiniao`
- 否则输出 `Huoyu`，第二行输出 $k$，接下来 $k$ 行描述每个子序列

## 样例

**样例输入：**
```
0
4
4
kdoi
7
ccccccc
7
sszcdjr
7
abacaca
```

**样例输出：**
```
Huoyu
2
2 1 2
2 3 4
Shuiniao
Huoyu
3
3 1 2 3
2 4 5
2 6 7
Huoyu
3
2 1 4
3 2 3 5
2 6 7
```

## 提示

- 测试点 $1\sim 5$：$n \le 18$
- 测试点 $6\sim 11$：$n \le 1000$
- 测试点 $12\sim 20$：$n \le 10^5$

特殊性质 A：$n$ 是偶数，且每个字符出现次数不超过 $n/2$。
特殊性质 B：字符串中仅有 `a` 和 `b`。

## 解题思路

### 问题分析

将字符串分成尽可能多的非空子序列，使得每个子序列都不是回文串。这是构造题。

### 核心思路

1. 如果所有字符相同，无法构造（输出 Shuiniao）
2. 否则，贪心地将字符分组，每组尽量短（2个字符），确保不是回文
3. 对于两个字符的组，选择不同的字符配对
4. 剩余字符加入已有组或单独成组（需验证非回文）

### 算法流程

1. 统计字符频率，若只有一个字符出现则无解
2. 贪心配对不同字符形成长度为 2 的非回文子序列
3. 处理剩余字符，尝试加入已有组

### 复杂度分析

- 时间复杂度：$O(n)$ 每组数据
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int c, q;
    cin >> c >> q;

    while (q--) {
        int n;
        cin >> n;
        string s;
        cin >> s;

        vector<int> cnt(26, 0);
        for (char ch : s) cnt[ch - 'a']++;
        int maxcnt = *max_element(cnt.begin(), cnt.end());

        if (maxcnt == n) {
            cout << "Shuiniao\n";
            continue;
        }

        vector<vector<int>> groups;
        vector<bool> used(n, false);

        for (int i = 0; i < n; i++) {
            if (!used[i]) {
                vector<int> cur;
                cur.push_back(i);
                used[i] = true;
                for (int j = i + 1; j < n; j++) {
                    if (!used[j] && s[j] != s[i]) {
                        cur.push_back(j);
                        used[j] = true;
                        break;
                    }
                }
                if (cur.size() == 1) {
                    for (int j = 0; j < n; j++) {
                        if (!used[j]) {
                            cur.push_back(j);
                            used[j] = true;
                            break;
                        }
                    }
                }
                if (cur.size() >= 2) groups.push_back(cur);
            }
        }

        cout << "Huoyu\n" << groups.size() << "\n";
        for (auto& g : groups) {
            cout << g.size();
            for (int idx : g) cout << " " << idx + 1;
            cout << "\n";
        }
    }

    return 0;
}
```
