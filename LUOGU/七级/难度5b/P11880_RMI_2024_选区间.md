# [RMI 2024] 选区间

## 题目描述
无限长数列 A 初始全 0。给定 n 个区间 [l_i, r_i]，对每个区间选择操作1（区间+1）或操作2（区间外+1），使最终最大值最小。

## 输入格式
第一行 n，接下来 n 行每行 l_i r_i。

## 输出格式
第一行最小最大值，第二行 01 串表示选择。

## 样例
输入：
```
5
10 10
6 6
1 7
2 5
2 7
```
输出：
```
2
11110
```

## 提示
- n ≤ 2×10^5, l_i, r_i ≤ 2n

## 解题思路

### 问题分析
需要选择操作使得数列中最大值最小。

### 核心思路
使用差分数组计算区间覆盖次数，贪心选择操作。

### 算法流程
1. 计算每个位置被区间覆盖的次数
2. 根据覆盖情况选择操作

### 复杂度分析
- 时间：O(n)
- 空间：O(n)
