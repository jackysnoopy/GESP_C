# P13031 Digit Blocks

## 题目描述

本题是交互题（Google Code Jam 2021 Round 1B Digit Blocks）。

给定 N 个塔（每个塔有 B 个位置，共 N×B 个位置），你需要将 N×B 个数字（0-9）分配到这些塔中。每个塔从底部开始填充，每 B 个数字组成一个 B 位数。

目标：使得所有 N 个塔组成的 B 位数之和最大。

## 输入格式

第一行四个整数 T, N, B, P：
- T：测试用例数量
- N：塔的数量
- B：每个塔的位数
- P：一个参数

每个测试用例中，交互过程会逐个给出 N×B 个数字 D。

## 输出格式

对于每个数字 D，输出一个整数 (1 到 N)，表示将 D 放置到哪个塔的顶部。

## 提示

- 1 ≤ T ≤ 100
- 1 ≤ N ≤ 100, 1 ≤ B ≤ 20

## 解题思路

### 问题分析

这是一个交互式贪心问题，需要逐个决定数字放置到哪个塔中，目标是最大化最终的数值和。

### 核心思路

贪心策略：优先将大数字放到高位（底部位置），小数字放到低位（顶部位置）。具体策略需要根据当前各塔的填充状态动态调整。

### 复杂度分析

- 时间复杂度：O(T × N × B)
- 空间复杂度：O(N × B)
