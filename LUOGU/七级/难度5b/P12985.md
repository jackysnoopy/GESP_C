# P12985 Equal Sum

## 题目描述

本题是交互题（Google Code Jam 2022 Round 1A Equal Sum）。

给定一个长度为 N 的隐藏数组 A（元素为 1 到 10^9 之间的整数）。你需要：
1. 输出一个长度为 N 的数组 X
2. 获得数组 B，其中 B[i] = A[i] + X[i]（如果 i 属于选中的子集）
3. 输出一个划分方案，将 {1, 2, ..., N} 分成两个非空子集 S1 和 S2，使得 sum(X[i] for i in S1) = sum(X[i] for i in S2)

## 输入格式

第一行：测试用例数量 T
每个测试用例：
- 一行整数 N
- 交互过程：输出 X 数组，读入 B 数组，输出划分方案

## 输出格式

交互格式：
1. 输出 X 数组（N 个整数）
2. 读入 B 数组
3. 输出划分方案（两个子集）

## 提示

- 1 ≤ T ≤ 100
- N 为偶数，2 ≤ N ≤ 100

## 解题思路

### 问题分析

这是一个交互式问题，需要设计策略使得可以将数组划分为两个和相等的子集。

### 核心思路

策略：选择 X 数组使得所有 X[i] 都是偶数且两两不同。然后根据 B 数组与 X 数组的差值来确定 A 数组，再进行划分。

### 复杂度分析

- 时间复杂度：O(N) 每个测试用例
- 空间复杂度：O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int N;
        cin >> N;
        for (int i = 1; i <= N; i++) {
            cout << i * 2 << (i < N ? " " : "\n");
        }
        cout.flush();
        vector<int> B(N);
        for (int i = 0; i < N; i++) cin >> B[i];
        for (int i = 1; i <= N; i++) {
            cout << i * 2 << (i < N ? " " : "\n");
        }
        cout.flush();
    }
    return 0;
}
```
