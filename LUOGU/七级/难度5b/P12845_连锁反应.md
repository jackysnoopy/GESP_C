# [蓝桥杯 2025 国 A] 连锁反应

## 题目描述

数轴上有 $n$ 枚炸弹，第 $i$ 枚位置为 $p_i$，爆炸范围 $[p_i - l_i, p_i + r_i]$。求最少主动引爆多少枚炸弹能使所有炸弹爆炸。

## 输入格式

第一行整数 $n$。
接下来 $n$ 行，每行三个整数 $p_i, l_i, r_i$。

## 输出格式

输出一个整数表示答案。

## 样例

### 样例 1
输入：
```
5
3 1 1
1 0 2
6 2 1
9 4 2
10 1 1
```
输出：
```
2
```

## 提示

- $1 \le n \le 200000$

## 解题思路

### 问题分析

贪心问题，按位置排序后，选择能覆盖最多未覆盖炸弹的炸弹引爆。

### 核心思路

按位置排序，贪心地选择覆盖范围最远的炸弹引爆。

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$
