# 闪耀之塔

## 题目描述
闪耀之塔是一棵节点编号从 $1\sim 2^n -1$ 编号，以 $1$ 为根，共有 $n$ 层的满二叉树。非叶子节点 $i$ 的左右儿子编号分别为 $2i$ 和 $2i+1$。需要给所有节点附上互不相同的权值（范围 $[1,2^n-1]$），使得 $\sum f(i)$ 最大，其中 $f(u) = val_u + \sum_{v\in S(u)} f(v)$。求在此条件下编号为 $p$ 的节点的 $f(p)$ 的最大值（对 $10^9+7$ 取模）。

## 输入格式
第一行包含两个正整数 $n,q$。接下来 $q$ 组询问，每组第一行一个整数 $k$，第二行一个长度为 $k$ 的 01 串表示 $p$ 的二进制表示。

## 输出格式
每组询问输出一行答案。

## 样例
### 样例 1
输入：
```
2 1
2
11
```
输出：
```
3
```

### 样例 2
输入：
```
10 3
4
1001
8
10110110
3
111
```
输出：
```
84582
5362
163710
```

## 提示
$1 \le k\le n \le 10^{12}$，$1 \le q\le 1000$，$1 \le k\le 10^4$。

## 解题思路

### 问题分析
总和 $\sum f(i)$ 等价于 $\sum_{x} val_x \cdot (depth(x)+1)$，因为每个节点的权值在总和中被计算了（深度+1）次。要最大化总和，应将最大权值分配给最深层的节点。

### 核心思路
在最优分配下，深度 $d$（从0开始）的节点获得值域 $[2^d, 2^{d+1}-1]$。对于深度为 $d$ 的节点 $p$，其 $f(p)$ 的最大值等于其子树中所有节点取各自深度上最大可能值的和。利用等比数列求和可得闭式公式：

$$g(d) = \frac{(4^{n-d}-1)(2^{d+2}-1)}{6} - \frac{2^{n-d}-1}{2}$$

节点 $p$ 的深度为 $k-1$（$k$ 为其二进制表示长度）。

### 算法流程
1. 对每组询问，解析二进制串得到 $k$，令 $d=k-1$
2. 用快速幂计算 $4^{n-d}$, $2^{d+2}$, $2^{n-d}$ 模 $10^9+7$
3. 代入公式计算答案

### 复杂度分析
- 时间：$O(q \log n)$，每次快速幂 $O(\log n)$
- 空间：$O(1)$

## 参考代码
```cpp
#include <iostream>
#include <string>
using namespace std;

const long long MOD = 1e9 + 7;
const long long INV6 = 166666668;
const long long INV2 = 500000004;

long long power(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp & 1) result = result * base % mod;
        base = base * base % mod;
        exp >>= 1;
    }
    return result;
}

long long solve(long long n, long long d) {
    long long nd = n - d;
    long long t1 = (power(4, nd, MOD) - 1 + MOD) % MOD;
    long long t2 = (power(2, d + 2, MOD) - 1 + MOD) % MOD;
    long long t3 = (power(2, nd, MOD) - 1 + MOD) % MOD;
    long long ans = t1 * t2 % MOD * INV6 % MOD;
    ans = (ans - t3 * INV2 % MOD + MOD) % MOD;
    return ans;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long n;
    int q;
    cin >> n >> q;
    while (q--) {
        int k;
        string s;
        cin >> k >> s;
        long long d = k - 1;
        cout << solve(n, d) << "\n";
    }
    return 0;
}
```
