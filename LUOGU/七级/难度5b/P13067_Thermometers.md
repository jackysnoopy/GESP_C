# [GCJ 2020 #3] Thermometers

## 题目描述

岛屿海岸线建模为周长 $K$ 的圆。温度计部署在特定点，无温度计的点使用最近温度计的温度。给定温度数据，求最少温度计数量。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $K, N$，第二行 $N$ 个整数 $X_1, \ldots, X_N$，第三行 $N$ 个整数 $T_1, \ldots, T_N$。

## 输出格式

每个测试用例输出 `Case #x: y`。

## 样例

### 样例 1
输入：
```
3
2 2
0 1
184 330
...
```
输出：
```
Case #1: 2
Case #2: 3
...
```

## 提示

- $2 \le K \le 10^9$, $2 \le N \le 100$

## 解题思路

### 问题分析

贪心问题，分析温度变化点确定最少温度计数量。

### 核心思路

每次温度变化至少需要一个温度计，统计温度变化次数。

### 复杂度分析

- 时间复杂度：$O(T \times N)$
- 空间复杂度：$O(N)$
