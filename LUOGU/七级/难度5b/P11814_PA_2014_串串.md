# [PA 2014] 串串

## 题目描述
给定 k 个长度为 n 的整数序列 A_1,...,A_k。构造序列 B 使得 max dist(A_i, B) 最小，其中 dist(a,b) = sum |a_i - b_i|。

## 输入格式
第一行 n k，接下来 k 行每行 n 个整数。

## 输出格式
n 个整数表示构造的 B 序列。

## 样例
输入：
```
5 3
1 -1 2 -1 2
1 2 2 1 2
2 2 -1 1 1
```
输出：
```
1 2 2 1 2
```

## 提示
- 2 ≤ n ≤ 10^5, 2 ≤ k ≤ 5
- |A_{i,j}| ≤ 10^9

## 解题思路

### 问题分析
需要最小化 max_i dist(A_i, B)，其中 dist 是 L1 距离。

### 核心思路
对于每个位置 j，独立地选择 b_j 使得 max_i |a_{i,j} - b_j| 最小。取中位数可使 L1 距离最小化。

### 算法流程
1. 对每个位置 j，收集所有 a_{i,j}
2. 取中位数作为 b_j

### 复杂度分析
- 时间：O(nk log k)
- 空间：O(nk)
