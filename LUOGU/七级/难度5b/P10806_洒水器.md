# [CEOI 2024] 洒水器

## 题目描述
瓦茨拉夫有一个美丽的花园，花园里种了一排 $M$ 朵花。在这一排上，瓦茨拉夫还放置了 $N$ 个洒水器来浇花。

洒水器的位置由数字 $s_1, s_2, \ldots , s_N$ 给出。花的位置由数字 $f_1, f_2, \ldots , f_M$ 给出。两者都是非递减顺序。

瓦茨拉夫可以单独将每个洒水器向左或向右旋转，并设置它们的喷水强度——所有洒水器共享同一个水管，因此喷射距离相同。

如果喷水强度为 $K$，并且第 $i$ 个洒水器向左旋转，它将浇灌位置在 $s_i - K$ 到 $s_i$ 之间的所有花。类似地，如果第 $j$ 个洒水器向右旋转，它将浇灌位置在 $s_j$ 到 $s_j+K$ 之间的所有花。

你的任务是决定是否可以浇灌所有的花。如果是，你应该找到最小的足够喷水强度，以及相应的洒水器配置。

## 输入格式
输入的第一行包含两个用空格隔开的整数 $N$ 和 $M$。第二行包含 $N$ 个空格分隔的整数 $s_1, s_2, \ldots , s_N$，表示洒水器的位置。第三行包含 $M$ 个空格分隔的整数 $f_1, f_2, \ldots , f_M$，表示花的位置。

## 输出格式
如果无法浇灌所有的花，则输出数字 $-1$。

如果可以，输出应该包含两行。第一行输出数字 $K$，表示浇灌所有花所需的最小的喷水强度。在第二行，输出长度为 $N$ 的字符串 $c$，其中 $c_i$ 为 `L`，如果第 $i$ 个洒水器应该向左旋转，否则为 `R`。

## 样例
**样例输入 1：**
```
3 3
10 10 10
5 11 16
```
**样例输出 1：**
```
6
LLR
```

**样例输入 2：**
```
1 2
1000
1 2000
```
**样例输出 2：**
```
-1
```

## 提示
对于所有输入数据，满足：
- $1 \leq N,M \leq 10^5$
- $0 \leq s_{i} \leq 10^9$
- $0 \leq f_{i} \leq 10^9$

## 解题思路

### 问题分析
给定 N 个洒水器和 M 朵花的位置，每个洒水器可以选择向左或向右旋转，所有洒水器共享相同的喷水强度 K。需要找到最小的 K 使得所有花都能被浇灌。

### 核心思路
1. 二分搜索喷水强度 K
2. 对于每个 K，判断是否能用 N 个洒水器覆盖所有 M 朵花
3. 判断方法：贪心匹配，从左到右处理每朵花，为其分配最左边可用的洒水器

### 算法流程
1. 二分搜索 K 的范围 [0, 2×10^9]
2. 对于每个 K，使用贪心算法判断可行性：
   - 对每朵花，找到能覆盖它的最左边洒水器
   - 如果找不到则不可行
3. 找到最小可行 K 后，重新构造洒水器方向

### 复杂度分析
- 时间复杂度：$O((N+M) \log V)$，其中 $V$ 是坐标范围
- 空间复杂度：$O(N+M)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

int n, m;
vector<long long> s, f;

bool can(long long K) {
    vector<bool> used(n, false);
    int j = 0;

    for (int i = 0; i < m; i++) {
        while (j < n && s[j] + K < f[i]) j++;
        bool covered = false;
        for (int k = j; k < n && s[k] - K <= f[i]; k++) {
            if (!used[k] && abs(s[k] - f[i]) <= K) {
                used[k] = true;
                covered = true;
                break;
            }
        }
        if (!covered) return false;
    }
    return true;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n >> m;
    s.resize(n);
    f.resize(m);
    for (int i = 0; i < n; i++) cin >> s[i];
    for (int i = 0; i < m; i++) cin >> f[i];

    long long lo = 0, hi = 2e9, ans = -1;
    while (lo <= hi) {
        long long mid = (lo + hi) / 2;
        if (can(mid)) {
            ans = mid;
            hi = mid - 1;
        } else {
            lo = mid + 1;
        }
    }

    if (ans == -1) {
        cout << -1 << "\n";
        return 0;
    }

    vector<int> assign(n, -1);
    vector<bool> used(n, false);
    int j = 0;
    for (int i = 0; i < m; i++) {
        while (j < n && s[j] + ans < f[i]) j++;
        for (int k = j; k < n && s[k] - ans <= f[i]; k++) {
            if (!used[k] && abs(s[k] - f[i]) <= ans) {
                used[k] = true;
                assign[k] = (s[k] <= f[i]) ? 1 : 0;
                break;
            }
        }
    }

    for (int i = 0; i < n; i++) {
        if (assign[i] == -1) assign[i] = 0;
    }

    cout << ans << "\n";
    string result;
    for (int i = 0; i < n; i++) {
        result += (assign[i] == 0 ? 'L' : 'R');
    }
    cout << result << "\n";

    return 0;
}
```
