# [GCJ 2017 Qualification] Fashion Show

## 题目描述

$N \times N$ 网格放置模特，有 `+`, `x`, `o` 三种类型。满足同行/列至少一个 `+`，同对角线至少一个 `x`。最大化风格点数。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $N, M$，接下来 $M$ 行描述已放置的模特。

## 输出格式

每个测试用例输出 `Case #x: y z`，然后 $z$ 行描述添加/替换的模特。

## 样例

### 样例 1
输入：
```
3
2 0
1 1
o 1 1
3 4
+ 2 3
+ 2 1
x 3 1
+ 2 2
```
输出：
```
Case #1: 4 3
o 2 2
+ 2 1
x 1 1
...
```

## 提示

- $1 \le N \le 100$

## 解题思路

### 问题分析

贪心构造问题。

### 核心思路

贪心地在空位放置 `o`（如果可能），否则放置 `+` 或 `x`。

### 复杂度分析

- 时间复杂度：$O(N^2)$
- 空间复杂度：$O(N^2)$
