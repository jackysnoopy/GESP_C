# [KOI 2024 Round 2] 最大异或

## 题目描述

给定一个由 `0` 和 `1` 组成的字符串 $S$。定义 $f(s)$ 为子串 $s$ 对应的二进制数值。求 $g(s_1, s_2) = f(s_1) \oplus f(s_2)$ 的最大值。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $N$，第二行字符串 $S$。

## 输出格式

每个测试用例输出一行，表示最大异或值的二进制表示。

## 样例

### 样例 1
输入：
```
4
3
010
5
10101
5
00100
5
11111
```
输出：
```
11
11111
110
11110
```

## 提示

- $1 \le T \le 100$, $2 \le N \le 10^7$

## 解题思路

### 问题分析

需要找到两个子串，使得它们对应的二进制数异或值最大。

### 核心思路

使用Trie树存储所有子串的值，然后对每个子串查询最大异或值。

### 复杂度分析

- 时间复杂度：$O(N^2 \times 30)$（暴力），需要优化
- 空间复杂度：$O(N^2)$
