# [PA 2016] 卡牌

## 题目描述
Alice 和 Bob 各有 $n$ 张编号 $1\sim n$ 的卡牌。玩 $n-1$ 局：每局 Alice 先弃掉 Bob 的一张牌，然后 Bob 弃掉 Alice 的一张牌。最终各剩一张。有 $m$ 对胜负关系，未给出的为平局。双方最优策略下判断 Alice 的结果。

## 输入格式
第一行 $T$。每组：第一行 $n,m$，接下来 $m$ 行 $x\ w\ y$（$w$ 为 $>$ 或 $<$）。

## 输出格式
每组一行：WYGRANA / REMIS / PRZEGRANA。

## 样例
输入：
```
3
5 5
5 > 5
1 > 5
3 > 5
4 > 5
2 > 5
2 2
1 > 1
1 > 2
1 1
1 < 1
```
输出：
```
WYGRANA
REMIS
PRZEGRANA
```

## 提示
$1\le T\le 20$，$1\le n\le 10^5$，$0\le m\le 2\times 10^5$。

## 解题思路

### 问题分析
经过 $n-1$ 轮后，Alice 实际上决定了 Bob 最终保留哪张牌 $y$，Bob 决定了 Alice 保留哪张牌 $x$。结果由 $(x,y)$ 的胜负关系决定。

### 核心思路
Alice 选 $y$，Bob 选 $x$。Alice 想最大化结果，Bob 想最小化。答案为 $\max_y \min_x \text{outcome}(x,y)$。对每个 $y$，若存在 $x$ 使 Alice 输则 $\min=0$；若存在未知关系（默认平局）则 $\min\le 1$；否则 $\min=2$。

### 算法流程
1. 对每个 $y$ 记录已知关系和数量
2. 对每个 $y$ 计算 Bob 能迫使的最差结果
3. 取所有 $y$ 中最好的结果

### 复杂度分析
- 时间：$O(T(n + m))$
- 空间：$O(n + m)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <string>
#include <map>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n, m;
        cin >> n >> m;
        vector<map<int, int>> rel(n + 1);
        vector<int> cntKnown(n + 1, 0);
        for (int i = 0; i < m; i++) {
            int x, y;
            char w;
            cin >> x >> w >> y;
            int val = (w == '>') ? 2 : 0;
            rel[y][x] = val;
            cntKnown[y]++;
        }
        int best = 0;
        for (int y = 1; y <= n; y++) {
            int worst = 1;
            if (cntKnown[y] < n) {
                worst = 1;
            }
            for (auto& p : rel[y]) {
                if (p.second == 0) {
                    worst = 0;
                    break;
                }
            }
            if (cntKnown[y] == n) {
                worst = 2;
                for (auto& p : rel[y]) {
                    if (p.second < worst) {
                        worst = p.second;
                    }
                }
            }
            if (worst > best) best = worst;
        }
        if (best == 2) cout << "WYGRANA\n";
        else if (best == 1) cout << "REMIS\n";
        else cout << "PRZEGRANA\n";
    }
    return 0;
}
```
