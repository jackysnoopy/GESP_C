# [THUPC 2023 初赛] 大富翁

## 题目描述

小 W 和小 H 在玩大富翁。地图是一棵 n 个节点的有根树，1 号节点为根。每个节点有价格 w_x。

对于两个不同节点 x, y，若 x 是 y 的祖先，则称 x **支配** y。

两人轮流购买未被购买的节点。购买节点 x 时：
1. 向系统支付 w_x
2. 若 x 支配 n1 个对手节点，被 n2 个对手节点支配：
   - n1 > n2 时，对手向买方支付 n1 - n2
   - n1 < n2 时，买方向对手支付 n2 - n1

两人都采用最优策略，求先手（小 W）最终能赚多少游戏币。

## 输入格式

第一行一个正整数 n。

第二行 n 个非负整数，第 i 个整数为 i 号节点的价格 w_i。

第三行 n-1 个正整数，第 i 个正整数表示第 i+1 号节点的父亲编号。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例 1

输入：
```
7
0 0 1 0 0 0 0
1 1 2 2 3 3
```

输出：
```
2
```

## 提示

$1 \le n \le 2 \times 10^5$，$0 \le w_x \le 2 \times 10^5$。

来自 THUPC2023 初赛。

## 解题思路

### 问题分析

两人在树上交替购买节点，购买时不仅支付价格，还会因为祖先-后代关系产生额外的金钱流动。关键在于理解每对祖先-后代关系对收益的贡献。

### 核心思路

对于每对祖先-后代关系 (u, v)（u 是 v 的祖先）：
- 若 W 买 u、H 买 v：H 向 W 支付 1（+1 收益）
- 若 H 买 u、W 买 v：W 向 H 支付 1（-1 收益）

定义 val(x) = (子树大小 - 1) - 深度，表示节点 x 在所有祖先-后代关系中的"净贡献"。

可以证明：W 的总收益 = Σ_{W的节点} (val(x) - w_x)。

因此问题转化为：两人交替从 n 个数中选数，W 想最大化选中数的和。贪心策略：按 (val(x) - w_x) 降序排列，W 取奇数位（第 1, 3, 5, ... 个）。

### 算法流程

1. 读入树结构，计算每个节点的子树大小和深度
2. 计算 val(x) = (sz[x] - 1) - dep[x]
3. 计算 diff(x) = val(x) - w[x]
4. 按 diff 降序排列
5. W 取第 0, 2, 4, ... 个（0-indexed），累加 diff 即为答案

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<long long> w(n + 1);
    for (int i = 1; i <= n; i++) cin >> w[i];

    vector<int> parent(n + 1, 0);
    vector<vector<int>> children(n + 1);
    for (int i = 2; i <= n; i++) {
        int p;
        cin >> p;
        parent[i] = p;
        children[p].push_back(i);
    }

    vector<int> sz(n + 1, 1), dep(n + 1, 0);
    for (int i = 2; i <= n; i++) dep[i] = dep[parent[i]] + 1;

    vector<int> order;
    vector<int> stk = {1};
    while (!stk.empty()) {
        int u = stk.back(); stk.pop_back();
        order.push_back(u);
        for (int v : children[u]) stk.push_back(v);
    }
    for (int i = (int)order.size() - 1; i >= 0; i--) {
        int u = order[i];
        for (int v : children[u]) sz[u] += sz[v];
    }

    vector<long long> diff(n);
    for (int i = 1; i <= n; i++) {
        long long val = (long long)(sz[i] - 1) - dep[i];
        diff[i - 1] = val - w[i];
    }

    sort(diff.begin(), diff.end(), greater<long long>());

    long long ans = 0;
    for (int i = 0; i < n; i += 2) ans += diff[i];

    cout << ans << "\n";
    return 0;
}
```
