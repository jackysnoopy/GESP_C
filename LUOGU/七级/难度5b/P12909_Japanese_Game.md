# [NERC 2020] Japanese Game

## 题目描述

给定一个由 `#` 和 `_` 组成的字符串 $m$，表示某个 Nonogram 的 mask。找到一个 profile $p'$ 使得其 mask 为 $m$，或判断不存在。

## 输入格式

一行字符串 $m$。

## 输出格式

若不存在输出 $-1$，否则第一行输出 $k$，第二行输出 $k$ 个整数。

## 样例

### 样例 1
输入：
```
__#_____
```
输出：
```
2
3 2
```

## 提示

- $1 \le n \le 100000$

## 解题思路

### 问题分析

需要从 mask 还原 profile。

### 核心思路

分析 mask 中 `#` 的连续段，每个连续段对应 profile 中的一个数。

### 复杂度分析

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
