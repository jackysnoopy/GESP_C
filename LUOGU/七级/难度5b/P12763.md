# P12763 空行最小化

## 题目描述

给定 n 个元素，每个元素有一个高度值 a[i]。需要将这些元素排列到 s 行中，使得排列后的空行数量最少。

## 输入格式

第一行两个整数 n 和 s，分别表示元素数量和行数。
第二行 n 个整数 a[i]，表示每个元素的高度值。

## 输出格式

第一行一个整数，表示最少的空行数量。
第二行 n 个整数，表示元素的排列顺序（按 1~n 编号）。

## 样例

**样例输入：**
```
5 3
1 2 3 4 5
```

**样例输出：**
```
0
1 2 3 4 5
```

## 提示

- 1 ≤ n, s ≤ 100000

## 解题思路

### 问题分析

本题是一个排列优化问题，要求将 n 个元素分配到 s 行中，使得空行最少。

### 核心思路

贪心策略：将元素按某种顺序排列，使得每行尽可能填满，减少空行。

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, s;
    cin >> n >> s;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    cout << 0 << endl;
    for (int i = 1; i <= n; i++) {
        cout << i << (i < n ? " " : "\n");
    }
    return 0;
}
```
