# [省选联考 2025] 推箱子

## 题目描述
无穷数轴上有 n 个箱子，初始在 a_i，需要在时刻 t_i 及之后在 b_i。每次可移动一个箱子一格。判断是否可行。

## 输入格式
多组测试数据。每组第一行 n，接下来 n 行每行 a_i b_i t_i。

## 输出格式
每组一行 Yes 或 No。

## 样例
输入：
```
0 2
2
4 5 1
6 7 1
3
4 5 3
7 6 1
10 8 4
```
输出：
```
No
Yes
```

## 提示
- T ≤ 6, n ≤ 2×10^5
- a_i, b_i ≤ 10^9, t_i ≤ 10^16

## 解题思路

### 问题分析
需要判断箱子移动的时间约束是否满足。

### 核心思路
每个箱子需要至少 |b_i - a_i| 时间来移动。

### 算法流程
1. 检查每个箱子的时间约束
2. 考虑箱子间的相互影响

### 复杂度分析
- 时间：O(n)
- 空间：O(n)
