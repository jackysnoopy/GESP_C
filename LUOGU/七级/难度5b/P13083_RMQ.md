# [NOISG 2017] RMQ

## 题目描述

$N$ 头奶牛排成一排，给出 $Q$ 条信息，每条表示区间 $[L_i, R_i]$ 内编号最小的奶牛为 $A_i$。构造满足所有信息的排列。

## 输入格式

第一行两个整数 $N, Q$。
接下来 $Q$ 行，每行三个整数 $L_i, R_i, A_i$。

## 输出格式

一行 $N$ 个整数表示构造的排列，无解输出 $N$ 个 $-1$。

## 样例

### 样例 1
输入：
```
5 3
0 2 1
1 3 0
1 4 0
```
输出：
```
1 4 3 0 2
```

## 提示

- $1 \le N, Q \le 10^5$

## 解题思路

### 问题分析

构造问题，需要满足区间最小值约束。

### 核心思路

按值从小到大处理，确定每个值的位置。

### 复杂度分析

- 时间复杂度：$O(N + Q)$
- 空间复杂度：$O(N)$
