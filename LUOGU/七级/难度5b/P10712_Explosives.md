# Explosives

## 题目描述
有 $n$ 座工厂和 $n$ 座矿井排列在直线上。卡车从位置 0 出发，需要从每座工厂领取一个炸弹送到某座矿井。操作 `pickup(x)` 从当前位置开到工厂 $x$ 领取炸弹（车上最多 $c-1$ 个）；`offload(x)` 从当前位置开到矿井 $x$ 送出炸弹（车上至少 $1$ 个）。车上装有炸弹时行驶需支付 $|x-y|$ 元。求最小花费的操作序列。

## 输入格式
第一行 $n, c$。第二行 $n$ 个工厂坐标。第三行 $n$ 个矿井坐标。

## 输出格式
第一行最小花费。第二行 $2n$ 个坐标表示操作序列。

## 样例
输入：
```
3 2
12 14 4
9 5 8
```
输出：
```
7
4 5 14 12 9 8
```

## 提示
$1 \le n, c \le 1000$，$1 \le a_i, b_i \le 10000$。

## 解题思路
### 问题分析
卡车携带炸弹行驶时产生费用，空车免费。核心是合理安排 pickup/offload 顺序，尽量减少满载行驶距离。

### 核心思路
贪心策略：空车时尽量前往远处工厂批量拾取，满载时尽快送至最近矿井。利用空车免费的特性，最大化批量操作。

### 算法流程
1. 空车时选择最远未访问工厂
2. 满载或有矿井更近时选择最近矿井卸载
3. 循环直到完成所有操作

### 复杂度分析
时间 $O(n^2)$，空间 $O(n)$。
