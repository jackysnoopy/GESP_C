# 「DROI」Round 1 下坠

## 题目描述

$f$ 是定义在 $\mathbb{N^+}$ 上的函数。

我们令 $a_i$ 表示 $x$ 从低到高第 $i$ 位，那么 $f(x)= \prod_{i=1}^{len} (a_i+1)$（$len$ 表示 $x$ 的位数）。

如果对于一个数 $x$，存在 $y$ 使得 $f(y)=x$，那我们称 $x$ 是下坠数。

现在有 $Q$ 次询问，每次询问会给出一个正整数 $k$。

令 $x$ 表示所有下坠数中第 $k$ 小的下坠数，那么请你找到一个**最小的** $y$，使得 $f(y)=x$。若不存在一个 $y \in [1,10^{18}]$ 满足条件，则输出 $-1$。

## 输入格式

第一行输入一个整数 $Q$，表示询问次数。

接下来输入一行 $Q$ 个数，第 $i$ 个数表示第 $i$ 次询问的 $k$。

## 输出格式

输出一行 $Q$ 个数，第 $i$ 个数表示第 $i$ 次询问你找到的答案。

## 样例

### 样例1
**输入：**
```
3
1 2 3
```
**输出：**
```
1 2 3
```

### 样例2
**输入：**
```
3
9 14 46666666
```
**输出：**
```
9 18 -1
```

## 提示

**【样例解释】**

注意到 $f$ 的定义域是 $\mathbb{N^+}$，所以 $1$ 不是下坠数。则前三个下坠数分别为 $2,3,4$，对应的 $y$ 值则为 $1,2,3$。

**【数据范围】**

对于 $100\%$ 的数据满足：$Q \leq 10^5$，$k \leq 5 \times 10^7$。

## 解题思路

### 问题分析

我们需要找出所有下坠数，并对每个下坠数找到最小的 $y$ 使得 $f(y) = x$。

### 核心思路

**DFS 生成所有可能的 $f(y)$ 值**

1. $f(y)$ 的值取决于 $y$ 的每一位数字
2. 每一位数字 $d$ 贡献因子 $(d+1)$
3. 用 DFS 枚举所有可能的数字组合，记录每个 $f(y)$ 对应的最小 $y$

### 算法流程

1. 用 DFS 枚举所有可能的 $y$（按数字非递减顺序，避免重复）
2. 对每个 $y$，计算 $f(y)$ 并记录最小的 $y$
3. 将所有下坠数排序
4. 对每个询问 $k$，输出第 $k$ 个下坠数对应的最小 $y$

### 复杂度分析

- 时间复杂度：$O(D)$，其中 $D$ 是所有可能的 $f(y)$ 值的数量
- 空间复杂度：$O(D)$

## 参考代码

```cpp
#include <cstdio>
#include <algorithm>
#include <vector>
#include <map>
using namespace std;
const long long MAXY = 1000000000000000000LL;
map<long long, long long> mp;
vector<pair<long long, long long>> fall;
void dfs(long long prod, long long y, int last_digit) {
    if (prod > 1 && (mp.find(prod) == mp.end() || y < mp[prod])) {
        mp[prod] = y;
    }
    if (y > MAXY / 10) return;
    for (int d = last_digit; d <= 9; d++) {
        long long new_prod = prod * (d + 1);
        if (new_prod > 50000000LL) break;
        long long new_y = y * 10 + d;
        if (new_y > MAXY) break;
        dfs(new_prod, new_y, d);
    }
}
int main() {
    mp.clear();
    for (int d = 1; d <= 9; d++) {
        dfs(d + 1, d, d);
    }
    for (auto& p : mp) {
        fall.push_back({p.first, p.second});
    }
    sort(fall.begin(), fall.end());
    int Q;
    scanf("%d", &Q);
    while (Q--) {
        long long k;
        scanf("%lld", &k);
        if (k > (long long)fall.size()) {
            printf("-1");
        } else {
            printf("%lld", fall[k - 1].second);
        }
        if (Q > 0) printf(" ");
    }
    printf("\n");
    return 0;
}
```
