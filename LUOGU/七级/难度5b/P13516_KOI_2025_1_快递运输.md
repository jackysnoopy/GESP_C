# [KOI 2025 #1] 快递运输

## 题目描述

树形快递网络，机器人在节点上，有通信范围 X。动态添加/删除机器人，判断是否能将快递从节点 1 运到节点 N。

## 输入格式

第一行 N, Q，接下来 N-1 条边，Q 个操作（1 A B 添加 / 2 C 删除）。

## 输出格式

Q 行 YES/NO。

## 样例

```
11 10
1 3 3
...
```

## 解题思路

### 问题分析
机器人通信范围 X 使其能覆盖节点 p 周围距离 ≤X 的所有地点。需要判断机器人覆盖区间能否从 1 连接到 N。

### 核心思路
- 计算每个节点到节点 1 的距离 dist1
- 每个机器人覆盖路径上的区间 [dist1[pos]-X, dist1[pos]+X]
- 检查区间是否能覆盖 [0, totalDist]

### 算法流程
1. BFS 计算 dist1 和 distN
2. 维护机器人覆盖区间
3. 贪心检查区间覆盖

### 复杂度分析
- 时间：O(N + Q log Q)
- 空间：O(N + Q)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int N, Q; cin >> N >> Q;
    vector<vector<pair<int,long long>>> adj(N+1);
    for (int i=0;i<N-1;i++) {
        int u,v; long long w; cin>>u>>v>>w;
        adj[u].push_back({v,w}); adj[v].push_back({u,w});
    }
    vector<long long> dist1(N+1,-1), distN(N+1,-1);
    auto bfs = [&](int s, vector<long long>& d) {
        d[s]=0; vector<int> q={s}; int h=0;
        while(h<(int)q.size()){int u=q[h++];for(auto&[v,w]:adj[u])if(d[v]==-1){d[v]=d[u]+w;q.push_back(v);}}
    };
    bfs(1,dist1); bfs(N,distN);
    long long total = dist1[N];
    struct Robot{int pos;long long range;bool active;};
    vector<Robot> robots; vector<int> rScene(Q+1,-1);
    for (int j=1;j<=Q;j++) {
        int type; cin>>type;
        if(type==1){int A;long long B;cin>>A>>B;robots.push_back({A,B,true});rScene[j]=robots.size()-1;}
        else{int C;cin>>C;robots[rScene[C]].active=false;}
        vector<pair<long long,long long>> intervals;
        for(auto&r:robots){if(!r.active)continue;
            long long s=max(0LL,dist1[r.pos]-r.range),e=min(total,dist1[r.pos]+r.range);
            if(s<=e)intervals.push_back({s,e});}
        sort(intervals.begin(),intervals.end());
        long long reach=0; int idx=0;
        while(reach<total&&idx<(int)intervals.size()){
            if(intervals[idx].first>reach)break;
            long long mr=reach;
            while(idx<(int)intervals.size()&&intervals[idx].first<=reach){mr=max(mr,intervals[idx].second);idx++;}
            reach=mr;
        }
        cout<<(reach>=total?"YES":"NO")<<"\n";
    }
    return 0;
}
```
