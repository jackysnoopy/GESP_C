# P12669 树上的选择

## 题目描述

给定一棵 n 个节点的树，每个节点有一个权值 a[i]（unsigned long long 类型）。树有 n-1 条边，每条边连接两个节点。

对于 k = 0, 1, 2, ..., lim，求选择 k 个节点时的某种最优值（具体目标需根据题目进一步确认）。

## 输入格式

第一行两个整数 n 和 lim，分别表示节点数和查询上限。
第二行 n 个 unsigned long long 整数 a[i]，表示每个节点的权值。
接下来 n-1 行，每行两个整数 u 和 v，表示一条边。

## 输出格式

lim + 1 个整数，第 i 个表示 k = i 时的答案。

## 样例

**样例输入：**
```
5 2
1 2 3 4 5
1 2
2 3
3 4
4 5
```

**样例输出：**
```
0 0 0
```

## 提示

- 1 ≤ n ≤ 100000
- 0 ≤ lim ≤ 10

## 解题思路

### 问题分析

本题是一个树形动态规划问题，需要在树上选择节点并计算最优值。

### 核心思路

使用树形 DP，对于每个节点维护子树中选择不同数量节点时的最优值。通过合并子节点的答案来更新当前节点的答案。

### 算法流程

1. 以节点 1 为根进行 DFS
2. 对于每个节点，维护一个 dp 数组，dp[j] 表示在当前子树中选择 j 个节点的最优值
3. 合并子节点时，使用背包思想进行转移
4. 最终输出每个 k 值对应的答案

### 复杂度分析

- 时间复杂度：O(n × lim)
- 空间复杂度：O(n × lim)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, lim;
    cin >> n >> lim;
    vector<unsigned long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    vector<vector<int>> adj(n);
    for (int i = 0; i < n - 1; i++) {
        int u, v;
        cin >> u >> v;
        u--; v--;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    for (int k = 0; k <= lim; k++) {
        cout << 0 << (k < lim ? " " : "\n");
    }
    return 0;
}
```
