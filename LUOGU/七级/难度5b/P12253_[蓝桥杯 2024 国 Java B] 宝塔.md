# [蓝桥杯 2024 国 Java B] 宝塔

## 题目描述

小蓝最近迷上了宝塔游戏，这是一款迷宫填数游戏，玩家需要按照规则将高度为 $1$ 至 $N$ 的宝塔放置在一个 $N \times N$ 大小的水平放置的棋盘当中，每个宝塔占据棋盘的一个方格。具体规则如下：

1. 宝塔的高度只有 $1, 2, \ldots, N$。
2. 每一行、每一列中不可以出现相同高度的宝塔。
3. 棋盘周围的箭头中的数字表示从这个位置看向棋盘时可以看到的宝塔个数。

你需要输出 $25$ 个数字表示答案，按照棋盘布局从左至右、从上至下的顺序输出即可。

## 输入格式

本题为结果填空题，无输入。

## 输出格式

输出一个整数，即 $25$ 个数字拼接后的结果。

## 样例

无（结果填空题）。

## 提示

这是一道结果填空的题，你只需要算出结果后提交即可。本题的结果为一个整数，在提交答案时只需要编写一个程序输出这个整数，输出多余的内容将无法得分。

## 解题思路

### 问题分析

本题本质上是一个 **$5 \times 5$ 数独变体**，附加了"可见宝塔数"的约束。棋盘周围有四个方向的箭头数字，分别表示从上、下、左、右四个方向观察时能看到的宝塔数量。

### 核心思路

使用 **回溯搜索（DFS）** 逐格填充宝塔高度：

1. **约束条件**：每行每列宝塔高度 $1 \sim 5$ 各出现一次（数独规则）。
2. **可见性约束**：当填满一行时，检查该行从左/右方向的可见宝塔数；当填满一列时，检查该列从上/下方向的可见宝塔数。
3. **早期剪枝**：在填完一行或一列后立即验证可见性约束，不满足则回溯。

### 算法流程

1. 用 `used_row[i][v]` 和 `used_col[j][v]` 记录行/列是否已使用某个值。
2. 按行优先顺序逐格枚举 $1 \sim 5$，跳过已使用的值。
3. 当填完一行（`col == 4`）时，验证 `left_` 和 `right_` 约束。
4. 当填完最后一行（`row == 4`）时，验证 `top` 和 `bottom` 约束。
5. 找到唯一解后输出拼接结果。

### 复杂度分析

- 时间复杂度：$O(N!^N)$ 理论最坏，但由于约束严格，实际搜索空间很小。
- 空间复杂度：$O(N^2)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int n = 5;
int grid[5][5];
int top[5] = {2, 2, 1, 3, 3};
int bottom[5] = {1, 4, 2, 2, 3};
int left_[5] = {2, 2, 3, 2, 1};
int right_[5] = {3, 3, 1, 2, 4};
bool used_row[5][6];
bool used_col[5][6];

bool checkVisible(int arr[], int constraint) {
    int count = 0, maxHeight = 0;
    for (int i = 0; i < n; i++) {
        if (arr[i] > maxHeight) {
            maxHeight = arr[i];
            count++;
        }
    }
    return count == constraint;
}

bool solve(int row, int col) {
    if (row == n) return true;
    int nextRow = (col == n - 1) ? row + 1 : row;
    int nextCol = (col == n - 1) ? 0 : col + 1;
    for (int val = 1; val <= n; val++) {
        if (used_row[row][val] || used_col[col][val]) continue;
        grid[row][col] = val;
        used_row[row][val] = true;
        used_col[col][val] = true;
        bool ok = true;
        if (col == n - 1) {
            int arr[5];
            for (int i = 0; i < n; i++) arr[i] = grid[row][i];
            if (!checkVisible(arr, left_[row])) ok = false;
            if (ok) {
                for (int i = 0; i < n; i++) arr[i] = grid[row][n - 1 - i];
                if (!checkVisible(arr, right_[row])) ok = false;
            }
        }
        if (ok && row == n - 1) {
            int arr[5];
            for (int i = 0; i < n; i++) arr[i] = grid[i][col];
            if (!checkVisible(arr, top[col])) ok = false;
            if (ok) {
                for (int i = 0; i < n; i++) arr[i] = grid[n - 1 - i][col];
                if (!checkVisible(arr, bottom[col])) ok = false;
            }
        }
        if (ok) {
            if (solve(nextRow, nextCol)) return true;
        }
        grid[row][col] = 0;
        used_row[row][val] = false;
        used_col[col][val] = false;
    }
    return false;
}

int main() {
    solve(0, 0);
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            cout << grid[i][j];
    cout << endl;
    return 0;
}
```
