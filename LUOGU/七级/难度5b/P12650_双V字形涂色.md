# [KOI 2024 Round 2] 双 V 字形涂色

## 题目描述

给定 $N \times M$ 网格图，每个格子为白色或黑色。定义"V字形涂色"操作：选择一个白色格子作为起点，沿左上对角线和右上对角线涂色。执行两次操作，求最多能涂多少蓝色格子。

## 输入格式

第一行两个整数 $N, M$。
接下来 $N$ 行，每行一个长度为 $M$ 的字符串，`1` 表示白色，`0` 表示黑色。

## 输出格式

输出一个整数，表示最多蓝色格子数。

## 样例

### 样例 1
输入：
```
5 11
10001000000
01000100000
00100110001
00010101010
00001000100
```
输出：
```
13
```

## 提示

- $1 \le N, M \le 3000$

## 解题思路

### 问题分析

需要执行两次V字形涂色操作，最大化覆盖的白色格子数。

### 核心思路

预处理每个格子作为起点时V字形能覆盖的格子数，然后枚举两次操作的起点，计算并集大小。

### 复杂度分析

- 时间复杂度：$O(N \times M \times (N + M))$
- 空间复杂度：$O(N \times M)$
