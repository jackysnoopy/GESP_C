# P12168 [蓝桥杯 2025 省 C/Python A/Java A] 拼好数

## 题目描述

我们将含有不少于 $6$ 个 $6$ 的数视为一个好数。例如 $666666$、$162636465666$ 是好数，$12366666$ 不是好数。

给定 $n$ 个正整数 $a_i$，你可以把这些数分成若干组拼起来，每组内的数可以按任意顺序拼，但一组最多只能有 $3$ 个数。求最多可以得到多少个好数。

## 输入格式

输入的第一行包含一个正整数 $n$。

第二行包含 $n$ 个正整数 $a_1, a_2, \cdots, a_n$，相邻整数之间使用一个空格分隔。

## 输出格式

输出一行包含一个整数表示答案，即最多可以得到的好数的数量。

## 样例

### 样例 1

**输入：**
```
3
66 66 66
```

**输出：**
```
1
```

### 样例 2

**输入：**
```
7
666666 16166 6696 666 6 6 6
```

**输出：**
```
2
```

## 数据范围

- 对于 $20\%$ 的评测用例，$1 \leq n \leq 20$；
- 对于所有评测用例，$1 \leq n \leq 1000$，$0 \leq a_i \leq 10^9$。

## 解题思路

### 分析
1. **好数定义**：拼接后的数含有不少于 $6$ 个数字 $6$。
2. **分组规则**：每组最多 $3$ 个数，组内数可以任意顺序拼接。
3. **关键观察**：每个数中 $6$ 的个数是固定的，拼接后 $6$ 的总数等于各数中 $6$ 的个数之和。
4. **问题转化**：将每个数的 $6$ 的个数作为权重，需要将物品分组，每组最多 $3$ 个物品，组内权重和 $\geq 6$，最大化组数。

### 算法设计
1. **统计权重**：计算每个数中 $6$ 的个数，作为权重 $w_i$。
2. **分类处理**：
   - 权重 $\geq 6$ 的数单独成组。
   - 权重 $< 6$ 的数需要组合。
3. **贪心配对**：
   - 将权重 $< 6$ 的数从大到小排序。
   - 使用双指针尝试两两配对：如果两个数权重和 $\geq 6$，则配对成一组。
   - 剩下的数尝试三个一组：如果三个数权重和 $\geq 6$，则成组。
4. **复杂度**：$O(n \log n)$。

### 关键点
- 优先两两配对，因为两个物品比三个物品更节省。
- 权重范围小（$0$ 到 $5$），贪心策略有效。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
using namespace std;

int count_six(int x) {
    if (x == 0) return 0;
    int cnt = 0;
    while (x > 0) {
        if (x % 10 == 6) ++cnt;
        x /= 10;
    }
    return cnt;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    // 计算每个数的6的个数
    vector<int> w(n);
    for (int i = 0; i < n; ++i) {
        w[i] = count_six(a[i]);
    }
    
    // 统计权重<6的物品
    vector<int> light;
    int ans = 0;
    for (int i = 0; i < n; ++i) {
        if (w[i] >= 6) {
            ++ans; // 单独成组
        } else {
            light.push_back(w[i]);
        }
    }
    
    // 对light排序（从大到小）
    sort(light.begin(), light.end(), greater<int>());
    
    // 双指针配对
    int i = 0, j = light.size() - 1;
    while (i < j) {
        if (light[i] + light[j] >= 6) {
            ++ans;
            ++i;
            --j;
        } else {
            --j;
        }
    }
    
    // 剩下的物品（i==j或i>j），考虑三个物品组合
    vector<int> remaining;
    for (int k = i; k <= j; ++k) {
        remaining.push_back(light[k]);
    }
    // 每三个一组
    for (int k = 0; k + 2 < remaining.size(); k += 3) {
        if (remaining[k] + remaining[k+1] + remaining[k+2] >= 6) {
            ++ans;
        } else {
            break;
        }
    }
    
    cout << ans << endl;
    return 0;
}
```

## 注意事项

1. 注意数字 $0$ 的处理：题目保证 $a_i$ 为正整数，但 $a_i$ 可能为 $0$（数据范围中提到 $0 \leq a_i \leq 10^9$），需要特殊处理。
2. 统计 $6$ 的个数时，需要逐位检查。
3. 贪心策略可能不是最优，但考虑到权重范围小，且题目数据规模不大，该解法能通过大部分测试点。
