# [GCJ 2017 #3] Mountain Tour

## 题目描述

$C$ 个营地，$2C$ 条单向徒步路线。每条路线从一个营地出发到另一个营地，有出发时间和持续时间。求完成所有路线并回到营地 1 的最短时间。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $C$，接下来 $2C$ 行每行 $E_i, L_i, D_i$。

## 输出格式

每个测试用例输出 `Case #x: y`。

## 样例

### 样例 1
输入：
```
2
2
2 1 5
2 0 3
1 4 4
1 6 3
...
```
输出：
```
Case #1: 32
Case #2: 192
```

## 提示

- $2 \le C \le 1000$

## 解题思路

### 问题分析

哈密顿路径问题，需要找到最短时间完成所有路线。

### 核心思路

状态压缩DP或贪心策略。

### 复杂度分析

- 时间复杂度：$O(C \times 2^C)$（小数据），需要优化
- 空间复杂度：$O(C \times 2^C)$
