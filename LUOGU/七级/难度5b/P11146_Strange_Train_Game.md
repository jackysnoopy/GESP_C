# Strange Train Game

## 题目描述

SFM 号列车由 $n$ 节车厢组成，编号为 $1\sim n$。每节车厢有一个舒适度 $a_i\in \{0,1\}$，$0$ 代表不舒适，$1$ 代表舒适。管理组想要让舒适的车厢的编号尽量小，也就是说，让 $a$ 的字典序最大。

为此，管理组运来了一辆 $n$ 节车厢的备用车，舒适度表示为 $b_i\in \{0,1\}$。共有 $m$ 个可进行的操作，第 $i$ 个操作的操作参数为 $l_i,r_i$，表示 $\forall l_i\le k\le r_i$，交换 $a_k,b_k$。

可以**从小到大依次**决定是否执行每个操作，但是一共有 $2^m$ 种方案，于是，管理组找来了你，帮忙选出一种最优的方案，最大化 $a$ 的字典序。只需要输出最终得到的 $a$ 即可。

## 输入格式

第一行，两个正整数 $n,m$，表示字符串的长度和操作次数。

第二行，一个长度为 $n$ 的 $01$ 串 $a$。

第三行，一个长度为 $n$ 的 $01$ 串 $b$。

接下来 $m$ 行，每行两个正整数 $l_i,r_i$，描述第 $i$ 个操作。

## 输出格式

输出一行长度为 $n$ 的 $01$ 串，表示最优操作后的 $a$。

## 样例

**样例输入：**
```
10 5
0101011001
0101001110
5 10
2 6
1 10
6 6
3 4
```

**样例输出：**
```
0101011110
```

## 提示

- Subtask 1（20 pts）：$1\le n,m\le 20$
- Subtask 2（30 pts）：$l_i$ 互不相同，$a_i \ne b_i$
- Subtask 3（30 pts）：$1 \le n ,m \le 10^3$
- Subtask 4（20 pts）：无限制

对于 $100\%$ 的数据，$1\le n,m\le 2\times 10^5$，$1\le l_i\le r_i\le n$。

## 解题思路

### 问题分析

给定两个01串 $a$ 和 $b$，以及 $m$ 个区间操作。按顺序依次决定是否执行每个区间交换操作（交换 $a_k$ 和 $b_k$），目标是使最终 $a$ 的字典序最大。

### 核心思路

贪心策略：按顺序处理每个操作，对于操作 $[l,r]$，在区间内找到第一个 $a[k] \neq b[k]$ 的位置。如果此时 $a[k]=0, b[k]=1$，则执行交换可以使 $a[k]$ 变为 $1$，从而增大字典序。

### 算法流程

1. 按顺序处理每个操作 $[l, r]$
2. 在 $[l, r]$ 中找到第一个 $a[k] \neq b[k]$ 的位置 $k$
3. 如果 $a[k]=0$ 且 $b[k]=1$，执行交换（使 $a[k]$ 变为 $1$）
4. 否则不执行（交换会使字典序减小或不变）

### 复杂度分析

- 时间复杂度：$O(nm)$（暴力），使用线段树可优化到 $O((n+m)\log n)$
- 空间复杂度：$O(n+m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;

    string a, b;
    cin >> a >> b;

    vector<int> l(m), r(m);
    for (int i = 0; i < m; i++) {
        cin >> l[i] >> r[i];
        l[i]--;
        r[i]--;
    }

    for (int i = 0; i < m; i++) {
        int fl = 0;
        for (int j = l[i]; j <= r[i]; j++) {
            if (a[j] != b[j]) {
                if (a[j] == '0') fl = 1;
                break;
            }
        }
        if (fl) {
            for (int j = l[i]; j <= r[i]; j++) {
                swap(a[j], b[j]);
            }
        }
    }

    cout << a << "\n";
    return 0;
}
```
