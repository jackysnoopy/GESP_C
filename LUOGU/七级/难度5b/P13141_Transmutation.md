# [GCJ 2018 #1B] Transmutation

## 题目描述

$M$ 种金属，每种金属可通过公式由两种原料金属合成。求最多能合成多少克铅（金属 1）。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $M$，接下来 $M$ 行每行 $R_{i1}, R_{i2}$，最后一行 $G_1, \ldots, G_M$。

## 输出格式

每个测试用例输出 `Case #x: y`。

## 样例

### 样例 1
输入：
```
3
3
2 3
1 3
1 2
5 2 3
...
```
输出：
```
Case #1: 7
Case #2: 4
Case #3: 0
```

## 提示

- $1 \le T \le 100$, $2 \le M \le 100$

## 解题思路

### 问题分析

流网络问题，需要最大化铅的产量。

### 核心思路

建图后使用最大流算法。

### 复杂度分析

- 时间复杂度：$O(M^3)$
- 空间复杂度：$O(M^2)$
