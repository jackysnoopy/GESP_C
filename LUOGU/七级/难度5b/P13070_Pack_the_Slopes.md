# [GCJ 2020 Finals] Pack the Slopes

## 题目描述

山上有 $N$ 个休息点，通过 $N-1$ 条单向雪道连接。每条雪道有容量和价格。求最大滑雪人数和最小费用。

## 输入格式

第一行测试用例数 $T$。
每个测试用例：第一行 $N$，接下来 $N-1$ 行每行 $U_i, V_i, S_i, C_i$。

## 输出格式

每个测试用例输出 `Case #x: y z`。

## 样例

### 样例 1
输入：
```
2
4
1 2 2 5
1 3 2 5
3 4 1 -2
...
```
输出：
```
Case #1: 4 18
Case #2: 7 15
```

## 提示

- $2 \le N \le 10^5$

## 解题思路

### 问题分析

树上的最大流最小费用问题。

### 核心思路

DFS遍历树，贪心地分配滑雪者，计算最大流量和最小费用。

### 复杂度分析

- 时间复杂度：$O(N)$
- 空间复杂度：$O(N)$
