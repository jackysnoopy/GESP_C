# 动态车道优化

## 题目描述

大道有两个方向，每个方向有一条基础车道和 $n$ 条动态车道。动态车道可以调转方向但需要时间 $C$。求最大负载量的最小值。

## 输入格式

多组测试数据。每组：第一行 $n, m, C$，第二行 $m$ 个正整数表示方向 1 的车流量，第三行 $m$ 个正整数表示方向 2 的车流量。

## 输出格式

每组输出一个小数，表示最大负载量的最小值。

## 样例

### 样例 1
输入：
```
4
1 3 1
1 1 3
2 1 1
...
```
输出：
```
1.5000000000
2.0000000000
...
```

## 提示

- $1 \le n \le 10^5$, $\sum m \le 5 \times 10^5$

## 解题思路

### 问题分析

二分答案 + 贪心验证。

### 核心思路

二分最大负载量，检查是否能通过合理分配车道满足约束。

### 复杂度分析

- 时间复杂度：$O(m \log(\text{max\_load}))$
- 空间复杂度：$O(m)$
