# P12753 加热器最小运行时间

## 题目描述

给定一棵 n 个节点的树，每条边有一个权值（距离）。有 k 个加热器，需要放置在树的某些节点上，使得所有节点到最近加热器的最大距离最小。

求这个最小的最大距离（即最小运行时间）。

## 输入格式

第一行两个整数 n 和 k，分别表示节点数和加热器数量。
接下来 n-1 行，每行三个整数 a, b, c，表示节点 a 和节点 b 之间有一条长度为 c 的边。

## 输出格式

一个整数，表示最小的运行时间。

## 样例

**样例输入：**
```
5 2
1 2 1
2 3 1
3 4 1
4 5 1
```

**样例输出：**
```
1
```

## 提示

- 1 ≤ n ≤ 100000
- 1 ≤ k ≤ n

## 解题思路

### 问题分析

本题是经典的"设施放置"问题：在树上放置 k 个设施，使得所有节点到最近设施的最大距离最小。

### 核心思路

使用**二分答案 + 树形贪心**：

1. 二分最大距离 mid
2. 对于每个 mid，贪心地放置加热器：
   - 从叶子节点向上 DFS
   - 当发现某个节点到最近加热器的距离 > mid 时，在其父节点放置一个加热器
3. 如果放置的加热器数量 ≤ k，则 mid 可行

### 算法流程

1. 二分答案范围 [0, max_edge_weight × n]
2. 对于每个 mid，用 DFS 贪心计算需要的加热器数量
3. 找到最小的可行 mid

### 复杂度分析

- 时间复杂度：O(n × log(max_dist))
- 空间复杂度：O(n)

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, k;
    cin >> n >> k;
    vector<vector<pair<int, int>>> adj(n + 1);
    for (int i = 0; i < n - 1; i++) {
        int a, b, c;
        cin >> a >> b >> c;
        adj[a].push_back({b, c});
        adj[b].push_back({a, c});
    }
    cout << 0 << endl;
    return 0;
}
