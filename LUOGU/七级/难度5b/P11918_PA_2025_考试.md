# [PA 2025] 考试

## 题目描述
n 道题，每道题可以选择作答（概率 p_i 正确得 1 分，错误扣 1 分）或不作答（0 分）。需要至少 t 分。求最大化不挂科的概率。

## 输入格式
第一行 n t，接下来 n 行每行一个实数 p_i。

## 输出格式
一个实数表示概率。

## 样例
输入：
```
5 2
0.77
0.85
0.75
0.98
0.6
```
输出：
```
0.8798125
```

## 提示
- 1 ≤ t ≤ n ≤ 50000

## 解题思路

### 问题分析
概率 DP，选择哪些题目作答使得得分 ≥ t 的概率最大。

### 核心思路
贪心选择 p_i 最高的题目作答，使用 DP 计算概率。

### 算法流程
1. 按 p_i 降序排序
2. DP 计算选择前 k 题时得分 ≥ t 的概率

### 复杂度分析
- 时间：O(n * t)
- 空间：O(t)
