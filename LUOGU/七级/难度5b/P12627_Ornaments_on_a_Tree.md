# [ICPC 2025 NAC] Ornaments on a Tree

## 题目描述

给定一棵 $n$ 个节点的树，每个节点可以放一个非负整数权重的装饰品。部分节点权重已固定，其余可自由选择。约束：每个节点与其所有子节点的权重之和不超过 $K$。求最大总权重。

## 输入格式

第一行两个整数 $N, K$。
第二行 $N$ 个整数，$-1$ 表示可自由选择，否则为固定权重。
接下来 $N-1$ 行，每行两个整数表示边。

## 输出格式

输出最大总权重，若无解输出 $-1$。

## 样例

### 样例 1
输入：
```
5 10
-1 2 3 -1 -1
1 2
1 3
2 4
2 2 5
```
输出：
```
18
```

## 提示

- $1 \le N \le 2 \times 10^5$
- $0 \le K \le 10^9$

## 解题思路

### 问题分析

树形DP问题，每个节点的约束是自身权重加上所有子节点权重之和不超过 $K$。

### 核心思路

自底向上贪心处理：对于自由节点，设置为 $K$ 减去子节点权重之和（但不小于 $0$）；对于固定节点，检查约束是否满足。

### 算法流程

1. DFS遍历树
2. 对于每个节点，计算子节点权重之和
3. 自由节点：$w[u] = \max(0, K - \text{sum\_children})$
4. 固定节点：检查 $w[u] + \text{sum\_children} \le K$

### 复杂度分析

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$
