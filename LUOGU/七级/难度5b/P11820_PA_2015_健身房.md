# [PA 2015] 健身房

## 题目描述
n 个人预约健身，每人给定时间段 [l_i, r_i] 和器材 p_i。同一时间不能两人用同一器材。希望没人的时刻尽量多。

## 输入格式
第一行 n k，接下来 n 行每行 l_i r_i p_i。

## 输出格式
无解输出 NIE，否则输出最少有人的时刻数，以及每人分配的时间。

## 样例
输入：
```
4 2
1 3 1
1 1 1
1 3 2
3 3 2
```
输出：
```
2
3
1
1
3
```

## 提示
- n ≤ 10^6, k ≤ 10^9, l_i, r_i ≤ 10^9

## 解题思路

### 问题分析
区间调度问题，需要为每个人分配一个时间点，使得同一器材不冲突。

### 核心思路
贪心按右端点排序，依次分配最早的可用时间。

### 算法流程
1. 按 r_i 排序
2. 对每个人，分配器材 p_i 上最早可用且在 [l_i, r_i] 内的时间

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n + k)
