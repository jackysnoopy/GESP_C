# [GCJ 2013 #1C] Pogo

## 题目描述

Pogo stick 第 i 次跳 i 个单位。从 (0,0) 出发，到达 (X,Y)，求最短路径的方向序列。

## 输入格式

第一行 T，每组包含 X、Y。

## 输出格式

Case #x: 方向字符串（NESW）。

## 样例

```
2
3 4
-3 4
```

输出：
```
Case #1: ENWSEN
Case #2: ENSWN
```

## 解题思路

### 问题分析
需要找到最小的 n 使得 S=n(n+1)/2 ≥ |X|+|Y| 且 S 与 |X|+|Y| 同奇偶。

### 核心思路
1. 找到满足条件的最小 n
2. excess = (S - |X| - |Y|) / 2，需要从 {1..n} 中选若干数使其和为 excess
3. 将选中的跳跃方向反转（朝反方向跳），未选中的朝目标方向跳

### 算法流程
1. 递增 n 直到 S 满足条件
2. 贪心选择子集使和为 excess
3. 构造方向字符串

### 复杂度分析
- 时间：O(√(|X|+|Y|))
- 空间：O(1)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T; cin >> T;
    for (int t = 1; t <= T; t++) {
        int X, Y; cin >> X >> Y;
        int ax = abs(X), ay = abs(Y);
        int n = 0; long long S = 0;
        while (S < ax+ay || (S-ax-ay)%2 != 0) { n++; S += n; }
        int excess = (int)((S-ax-ay)/2);
        vector<bool> flip(n+1, false);
        for (int i = n; i >= 1 && excess > 0; i--)
            if (i <= excess) { flip[i] = true; excess -= i; }
        string result = "";
        for (int i = 1; i <= n; i++) {
            if (i % 2 == 1) result += flip[i] ? ((X>=0)?"W":"E") : ((X>=0)?"E":"W");
            else result += flip[i] ? ((Y>=0)?"S":"N") : ((Y>=0)?"N":"S");
        }
        cout << "Case #" << t << ": " << result << "\n";
    }
    return 0;
}
```
