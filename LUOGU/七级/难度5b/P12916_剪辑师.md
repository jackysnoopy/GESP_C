# [POI 2021/2022 R1] 剪辑师

## 题目描述

$n$ 部视频，剪辑第 $i$ 部需要 $t_i$ 天，必须在第 $d_i$ 天之前发布。求最多能按时发布多少部视频，并给出工作计划。

## 输入格式

第一行整数 $n$。
接下来 $n$ 行，每行两个整数 $t_i, d_i$。

## 输出格式

第一行输出 $m$。
接下来 $m$ 行，每行两个整数 $f_i, k_i$。

## 样例

### 样例 1
输入：
```
5
4 5
2 4
5 3
1 9
3 10
```
输出：
```
3
2 3
4 7
5 8
```

## 提示

- $1 \le n \le 5 \times 10^5$

## 解题思路

### 问题分析

经典调度问题，按截止时间排序后贪心选择。

### 核心思路

按截止时间排序，使用优先队列维护已选择的视频，贪心替换。

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$
