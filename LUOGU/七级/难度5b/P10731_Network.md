# Network

## 题目描述
给定 $n$ 台服务器和 $n-1$ 条双向数据线构成的树，以及 $m$ 个程序，第 $j$ 个程序运行在 $a_j$ 和 $b_j$ 上。程序可以运行当且仅当两台服务器都在运行且通过运行中的服务器连通。求最少停运多少台服务器使所有程序停止。

## 输入格式
第一行 $n, m$。接下来 $n-1$ 行边。再 $m$ 行每行 $a_j, b_j$。

## 输出格式
第一行最少停运数，第二行服务器编号。

## 样例
见题目描述。

## 提示
$1 \le n, m \le 200000$。

## 解题思路
### 问题分析
每个程序对应树上一条路径。停运一台服务器会断开经过该点的所有路径。需要找到最小顶点集覆盖所有程序路径。

### 核心思路
贪心算法：每次选择覆盖最多未覆盖程序路径的服务器停运，直到所有路径被覆盖。

### 算法流程
1. 预处理 LCA（倍增法）
2. 统计每个节点被多少程序路径经过
3. 贪心选择覆盖最多的节点，更新覆盖状态
4. 重复直到所有程序被覆盖

### 复杂度分析
时间 $O(n \log n + m \cdot d)$，其中 $d$ 为树深度。
