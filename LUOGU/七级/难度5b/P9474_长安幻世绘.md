# [yLOI2022] 长安幻世绘

## 题目描述

n 个彩灯排成一排，亮度互不相同。选出 m 个互不相邻的彩灯，使它们的极差（最大亮度 - 最小亮度）最小。

## 输入格式

第一行两个整数 n, m。
第二行 n 个整数 a_i，表示彩灯亮度。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例 1
输入：
```
5 3
1 2 3 4 5
```
输出：
```
4
```

### 样例 2
输入：
```
6 3
1 7 8 3 4 6
```
输出：
```
4
```

## 提示

- 1 ≤ n ≤ 10^5
- 1 ≤ m ≤ ⌈n/2⌉
- 1 ≤ a_i ≤ 10^9，a_i 互不相同

## 解题思路

### 问题分析

需要在元素互不相同的数组中选出 m 个互不相邻的子列，使极差最小。

### 核心思路

二分答案 + 贪心检查。

1. 将彩灯按亮度排序
2. 二分极差 d
3. 对于每个 d，检查是否存在方案：枚举最小值 a[i]，在排序后的数组中从 i 开始，贪心选取在 [a[i], a[i]+d] 范围内且原位置不相邻的 m 个元素

### 算法流程

1. 排序所有彩灯的亮度
2. 二分极差 d ∈ [0, max_a - min_a]
3. 对于每个 mid = (lo+hi)/2：
   - 枚举每个可能的最小值 a[i]
   - 从 a[i] 开始贪心选取，确保相邻选取的元素在原位置上差 ≥ 2
   - 若能在范围内选够 m 个，则 hi = mid，否则 lo = mid+1

### 复杂度分析

- 时间复杂度：O(n² log(max_a))（可优化到 O(n log n log(max_a))）
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    vector<pair<int, int>> vp(n);
    for (int i = 0; i < n; i++) {
        cin >> vp[i].first;
        vp[i].second = i;
    }
    
    sort(vp.begin(), vp.end());
    vector<int> a(n), pos(n);
    for (int i = 0; i < n; i++) {
        a[i] = vp[i].first;
        pos[i] = vp[i].second;
    }
    
    int lo = 0, hi = a[n-1] - a[0];
    while (lo < hi) {
        int mid = (lo + hi) / 2;
        bool ok = false;
        for (int i = 0; i < n && !ok; i++) {
            int cnt = 0;
            int last_pos = -2;
            for (int j = i; j < n && a[j] - a[i] <= mid; j++) {
                if (cnt == 0 || pos[j] - last_pos >= 2) {
                    cnt++;
                    last_pos = pos[j];
                    if (cnt >= m) { ok = true; break; }
                }
            }
        }
        if (ok) hi = mid;
        else lo = mid + 1;
    }
    
    cout << lo << "\n";
    return 0;
}
```
