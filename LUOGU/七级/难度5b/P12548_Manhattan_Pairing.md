# [UOI 2025] Manhattan Pairing

## 题目描述

给定 $2n$ 个点，所有点的 $y$ 坐标为 $0$ 或 $1$。将它们分成 $n$ 对，使得每对点的曼哈顿距离的最大值最小。

## 输入格式

第一行包含整数 $n$。
接下来 $2n$ 行，每行两个整数 $x_i, y_i$。

## 输出格式

输出一个整数，表示最优分组下每对点曼哈顿距离的最大值。

## 样例

### 样例 1
输入：
```
1
3 1
1 0
```
输出：
```
3
```

### 样例 2
输入：
```
3
18 0
3 0
1 0
10 0
8 0
14 0
```
输出：
```
4
```

## 提示

- $1 \le n \le 10^5$
- $0 \le x_i \le 10^9$, $0 \le y_i \le 1$

## 解题思路

### 问题分析

需要将 $2n$ 个点配对，使得每对的曼哈顿距离最大值最小。由于 $y$ 只有 $0$ 或 $1$，曼哈顿距离为 $|x_1 - x_2| + |y_1 - y_2|$。

### 核心思路

二分答案。对于给定的最大距离 $d$，检查是否能将所有点配对使得每对距离不超过 $d$。

### 算法流程

1. 二分答案 $d$
2. 对于每个 $d$，贪心地尝试配对
3. 同类型点（$y$ 相同）配对距离为 $|x_1 - x_2|$
4. 不同类型点配对距离为 $|x_1 - x_2| + 1$

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$
