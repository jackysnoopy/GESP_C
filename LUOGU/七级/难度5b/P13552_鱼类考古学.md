# 鱼类考古学

## 题目描述

给定 n 个非负整数 a_i，进行 n-1 次操作（恰好 n-k 次 AND 和 k-1 次加法），求最终最大值。

## 输入格式

第一行 T，每组包含 n, k 和数组 a。

## 输出格式

一个整数。

## 样例

```
3
3 2
1 2 3
4 3
2 5 6 4
5 1
1 3 7 9 11
```

输出：
```
3
12
1
```

## 解题思路

### 问题分析
n-k 次 AND 将 n 个数合并为 k 组，k-1 次加法将 k 组求和。需要最大化最终结果。

### 核心思路
- k=n：全加法，答案为 sum(a)
- k=1：全 AND，答案为 AND(a)
- 一般情况：枚举分组方案，AND 每组后求和

### 算法流程
1. 特判 k=1 和 k=n
2. 小数据集：枚举所有分组
3. 大数据集：贪心分组

### 复杂度分析
- 时间：O(k^n) 暴力，O(n) 贪心
- 空间：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T; cin >> T;
    while (T--) {
        int n, k; cin >> n >> k;
        vector<int> a(n);
        for (int i=0;i<n;i++) cin>>a[i];
        long long result = 0;
        if (k == n) {
            for (int x : a) result += x;
        } else if (k == 1) {
            result = a[0];
            for (int i=1;i<n;i++) result &= a[i];
        } else {
            // Greedy: try different partitions
            result = 0;
            if (n <= 20) {
                vector<int> assign(n, 0);
                function<void(int,int)> solve = [&](int idx, int mg) {
                    if (idx == n) {
                        long long total = 0;
                        for (int g=0;g<=mg;g++) {
                            long long gv = -1; bool found = false;
                            for (int i=0;i<n;i++) if (assign[i]==g) {
                                if (!found) { gv=a[i]; found=true; } else gv&=a[i];
                            }
                            if (found) total += gv;
                        }
                        result = max(result, total);
                        return;
                    }
                    for (int g=0;g<=min(mg+1,k-1);g++) { assign[idx]=g; solve(idx+1,max(g,mg)); }
                };
                solve(0, -1);
            } else {
                vector<int> sa = a;
                sort(sa.begin(), sa.end(), greater<int>());
                long long s = 0;
                for (int i=0;i<k-1&&i<n;i++) s += sa[i];
                long long ga = (k<=n) ? sa[k-1] : 0;
                for (int i=k;i<n;i++) ga &= sa[i];
                result = s + ga;
            }
        }
        cout << result << "\n";
    }
    return 0;
}
```
