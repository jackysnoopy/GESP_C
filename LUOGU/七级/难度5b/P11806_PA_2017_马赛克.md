# [PA 2017] 马赛克

## 题目描述
给定二维平面上的 n 个整点 (x_i, y_i)。构造 n 个正方形，第 i 个正方形左下角是 (x_i, y_i)，使得：
- n 个正方形的并是一个矩形
- 任意两个正方形的交面积为 0

输出每个正方形的边长，或输出 NIE 表示无解。

## 输入格式
多组测试数据。每组第一行 n，接下来 n 行每行 x_i y_i。

## 输出格式
TAK l_1 l_2 ... l_n 或 NIE。

## 样例
输入：
```
3
2
0 0
0 1
2
3 2
2 3
4
1 1
2 1
3 1
1 2
```
输出：
```
TAK 1 1
NIE
TAK 1 1 3 2
```

## 提示
- T ≤ 50, n ≤ 2000, sum n ≤ 5000
- 0 ≤ x_i, y_i ≤ 10^9

## 解题思路

### 问题分析
需要构造正方形使得它们拼成一个矩形且互不重叠。

### 核心思路
按 x 坐标排序，从左到右贪心构造。对于每个点，计算其正方形边长使得能与右侧的点对齐。

### 算法流程
1. 按 x 坐标排序所有点
2. 从左到右扫描，确定每个正方形的边长
3. 验证构造是否合法

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
