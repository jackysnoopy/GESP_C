# [THUPC 2024 初赛] 转化

## 题目描述

n 种颜色的球，第 i 种有 a_i 个。两类工具：
- 第一类 b_i 个，可以把一个指定颜色球变为任意颜色球
- 第二类 c_i 个，可以把一个指定颜色球变为两个同色球

每个工具最多用一次。求每种颜色最多能有多少个球，以及总球数最多多少。

## 输入格式

第一行一个整数 n。
接下来 n 行，每行三个整数 a_i, b_i, c_i。

## 输出格式

第一行 n 个整数，第 i 个表示颜色 i 最多能有多少个球。
第二行一个整数，表示总球数最多多少。

## 样例

```
输入：
2
1 0 1
0 1 1

输出：
2 0
3
```

## 提示

- 1 ≤ n ≤ 351493
- 0 ≤ a_i, b_i, c_i ≤ 10^9

## 解题思路

### 问题分析

需要分别求每种颜色的最大球数和总球数最大值。

### 核心思路

**最大化颜色 i 的球数：**
1. 先用第二类工具翻倍：a_i + c_i（需要至少一个球）
2. 再用其他颜色的第一类工具转化：对于 j ≠ i，可转化 min(b_j, a_j + c_j) 个球

**最大化总球数：**
- 总球数 = 初始球数 + 使用的第二类工具数
- 第二类工具每个净增一个球，一类工具不改变总数
- 最大总数 = sum(a_i) + sum(c_i)

### 算法流程

1. 预计算 total_S = sum_{a_j≥1} min(b_j, a_j + c_j)
2. 对每个颜色 i：S = total_S - (a_i ≥ 1 ? min(b_i, a_i + c_i) : 0)
3. max_balls_i = (a_i + S ≥ 1) ? a_i + c_i + S : 0

### 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<long long> a(n), b(n), c(n);
    long long total_a = 0, total_c = 0, total_S = 0;

    for (int i = 0; i < n; i++) {
        cin >> a[i] >> b[i] >> c[i];
        total_a += a[i];
        total_c += c[i];
        if (a[i] >= 1) {
            total_S += min(b[i], a[i] + c[i]);
        }
    }

    long long max_total = total_a + total_c;

    for (int i = 0; i < n; i++) {
        long long S = total_S;
        if (a[i] >= 1) {
            S -= min(b[i], a[i] + c[i]);
        }

        long long max_balls;
        if (a[i] + S >= 1) {
            max_balls = a[i] + c[i] + S;
        } else {
            max_balls = 0;
        }

        cout << max_balls << (i + 1 < n ? " " : "\n");
    }
    cout << max_total << "\n";

    return 0;
}
```
