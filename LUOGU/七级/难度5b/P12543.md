# P12543 能量效率

## 题目描述

本题是交互题（APIO 2025）。

给定一个长度为 n 的环形数组 v（值域为 0 到 49999）。你需要实现一个 rotate 操作来调整数组，并计算能量效率。

能量效率定义为所有无序对 (i, j) 的 min(|v[i]-v[j]|, 50000-|v[i]-v[j]|) 之和。

## 输入格式

第一行一个整数 n，表示数组长度。
第二行 n 个整数 v[i]，表示数组初始值。

## 输出格式

一个整数，表示能量效率。

## 提示

- 2 ≤ n ≤ 100000
- 0 ≤ v[i] < 50000

## 解题思路

### 问题分析

本题涉及环形数组上的距离计算。能量效率是所有点对之间的"环形距离"之和。

### 核心思路

环形距离的定义：对于两个值 a 和 b，环形距离为 min(|a-b|, 50000-|a-b|)。

计算方法：
1. 将所有元素排序
2. 对于每个元素，统计在不同距离范围内的元素数量
3. 利用前缀和或双指针计算总能量效率

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void energy(int n, vector<int> v) {
    long long total = 0;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            int diff = abs(v[i] - v[j]);
            total += min(diff, 50000 - diff);
        }
    }
    cout << total << endl;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> v(n);
    for (int i = 0; i < n; i++) cin >> v[i];
    energy(n, v);
    return 0;
}
```

> 注：当前为暴力计算，需要优化为 O(n log n) 的解法。
