# [THUPC 2023 决赛] 物理实验

## 题目描述

物理学家小 I 需要完成 n 种物理实验，第 i 种实验的重要度是 2^{-i}。有 m 轮宇宙射线，第 j 轮在完成 a_j 种实验后干扰最脆弱的未完成且未被干扰的实验。需要安排实验顺序使完成的 n-m 种实验重要度总和最大。

## 输入格式

第一行两个正整数 n, m。
第二行 m 个整数 a_1, a_2, ..., a_m。
接下来 m 行，每行 n 个整数，描述每轮宇宙射线的脆弱程度排序。

## 输出格式

一行 n-m 个整数，表示实验顺序。

## 样例

### 样例 1
输入：
```
3 1
1
1 2 3
```
输出：
```
1 3
```

### 样例 2
输入：
```
3 1
1
2 3 1
```
输出：
```
2 1
```

### 样例 3
输入：
```
6 2
1 3
3 2 4 5 6 1
5 4 1 3 6 2
```
输出：
```
1 4 5 2
```

## 提示

- 3 ≤ n ≤ 600
- 1 ≤ m ≤ ⌊(n-1)/2⌋
- 1 ≤ a_1 < a_2 < ... < a_m < n-m

## 解题思路

### 问题分析

需要在 m 轮宇宙射线干扰下，安排 n-m 个实验的顺序，使得完成的实验重要度总和最大。重要度 2^{-i} 意味着编号越小越重要。

### 核心思路

贪心策略：尽量先做编号大的实验（不重要的），保留编号小的实验（重要的）不被干扰。

每轮射线会干扰当前最脆弱（编号最小）的未完成且未被干扰的实验。因此需要在射线到来之前，确保重要的实验已经被完成或被保护。

### 算法流程

1. 按步骤模拟，每步选择一个实验完成
2. 每次选择当前可用的编号最大的实验（保留小编号）
3. 完成实验后检查是否有射线触发，若有则干扰最脆弱的实验

### 复杂度分析

- 时间复杂度：O(n²)
- 空间复杂度：O(n²)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstring>
using namespace std;

int n, m;
int a[605];
int p[605][605];
bool done[605];
bool disabled[605];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n >> m;
    for (int i = 0; i < m; i++) cin >> a[i];
    for (int i = 0; i < m; i++)
        for (int j = 1; j <= n; j++)
            cin >> p[i][j];
    
    memset(done, 0, sizeof(done));
    memset(disabled, 0, sizeof(disabled));
    
    vector<int> order;
    int experiment_count = 0;
    int ray_idx = 0;
    
    for (int step = 0; step < n - m; step++) {
        int chosen = -1;
        for (int i = n; i >= 1; i--) {
            if (done[i] || disabled[i]) continue;
            chosen = i;
            break;
        }
        if (chosen == -1) break;
        
        order.push_back(chosen);
        done[chosen] = true;
        experiment_count++;
        
        while (ray_idx < m && a[ray_idx] == experiment_count) {
            for (int j = 1; j <= n; j++) {
                int exp = p[ray_idx][j];
                if (!done[exp] && !disabled[exp]) {
                    disabled[exp] = true;
                    break;
                }
            }
            ray_idx++;
        }
    }
    
    for (int i = 0; i < (int)order.size(); i++) {
        if (i > 0) cout << " ";
        cout << order[i];
    }
    cout << "\n";
    
    return 0;
}
```
