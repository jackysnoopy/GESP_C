# [EGOI 2021] Luna likes Love / 卢娜爱磕 cp

## 题目描述

2n 个人排成一排，每人拿 1~n 的整数（每个出现两次），相同数字为情侣。

两种操作：
1. 让任意两个紧挨着的人交换位置。
2. 如果一对情侣互相紧挨着，让他们去约会（离开队伍，后面的人补位）。

求所有人约会的最少操作次数。

## 输入格式

第一行一个整数 n。

第二行 2n 个整数 a_i，依次表示队伍中朋友拿到的数字。

## 输出格式

一行，一个整数，表示最少操作次数。

## 样例

### 样例 1

输入：
```
3
3 1 2 1 2 3
```

输出：
```
4
```

### 样例 2

输入：
```
5
5 1 2 3 2 3 1 4 5 4
```

输出：
```
7
```

## 提示

$1 \le n \le 5 \times 10^5$。

来自 EGOI2021 Day 1 Problem B。

## 解题思路

### 问题分析

每次约会需要一次操作，共 n 次约会。关键是计算最少需要多少次交换操作。

将每对情侣看作一个区间 [left, right]。两个区间"交叉"（overlap 但互不包含）时，需要额外的交换来解决冲突。

### 核心思路

最小交换次数 = 区间交叉对数。

证明：每次交换最多消除一个交叉，而每个交叉至少需要一次交换。

区间交叉的定义：区间 [L1, R1] 和 [L2, R2] 交叉 ⟺ L1 < L2 < R1 < R2（或对称情况）。

### 算法流程

1. 记录每对情侣的位置区间 [left, right]
2. 按 left 升序排序
3. 用树状数组统计交叉对数：对于当前区间 [L, R]，查询已处理区间中 right 落在 (L, R) 内的个数
4. 答案 = 交叉对数 + n

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

struct Fenwick {
    vector<int> tree;
    int n;
    Fenwick(int n) : tree(n + 1, 0), n(n) {}
    void update(int i, int delta) {
        for (; i <= n; i += i & (-i)) tree[i] += delta;
    }
    int query(int i) {
        int sum = 0;
        for (; i > 0; i -= i & (-i)) sum += tree[i];
        return sum;
    }
    int query(int l, int r) {
        if (l > r) return 0;
        return query(r) - query(l - 1);
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> a(2 * n);
    for (int i = 0; i < 2 * n; i++) cin >> a[i];

    vector<int> L(n + 1, -1), R(n + 1, -1);
    for (int i = 0; i < 2 * n; i++) {
        if (L[a[i]] == -1) L[a[i]] = i + 1;
        else R[a[i]] = i + 1;
    }

    vector<pair<int, int>> inv(n);
    for (int i = 1; i <= n; i++) {
        inv[i - 1] = {L[i], R[i]};
    }

    sort(inv.begin(), inv.end());

    Fenwick fenw(2 * n);
    long long crossings = 0;

    for (auto& p : inv) {
        int l = p.first, r = p.second;
        crossings += fenw.query(l + 1, r - 1);
        fenw.update(r, 1);
    }

    cout << crossings + n << "\n";
    return 0;
}
```
