# [省选联考 2025] 幸运数字

## 题目描述
n 个二元组 (a_i, b_i)，a_i ∈ [l_{i,1}, r_{i,1}]，b_i ∈ [l_{i,2}, r_{i,2}]。加入 a_i 个 b_i 到集合 S，求 S 的中位数可能有多少种。

## 输入格式
多组测试数据。每组第一行 n，接下来 n 行每行四个整数。

## 输出格式
每组一行，可能的中位数个数。

## 样例
输入：
```
0 4
2
1 2 1 1
1 1 2 2
...
```
输出：
```
1
2
4
3
```

## 提示
- T ≤ 400, n ≤ 2×10^5, sum n ≤ 6×10^5

## 解题思路

### 问题分析
需要确定中位数的可能取值范围。

### 核心思路
通过分析 b_i 的取值范围和 a_i 的计数，确定中位数的上下界。

### 算法流程
1. 收集所有可能的 b_i 值范围
2. 计算中位数的最小和最大可能值

### 复杂度分析
- 时间：O(n)
- 空间：O(n)
