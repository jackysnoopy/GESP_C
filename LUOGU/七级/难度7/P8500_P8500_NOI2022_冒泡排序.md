# P8500 [NOI2022] 冒泡排序

## 题目描述

小 Z 对冒泡排序产生了浓厚的兴趣。

冒泡排序的交换次数被定义为在排序时进行交换的次数。他希望找到一个交换次数尽量少的序列。

小 Z 所研究的序列均由非负整数构成。它的长度为 $n$，且必须满足 $m$ 个附加条件。

他想要知道，在所有满足附加条件的序列中，进行冒泡排序的交换次数的最少值是多少。

## 输入格式

第一行包含一个正整数 $T$。
对于每组数据，第一行包含两个正整数 $n,m$。
接下来 $m$ 行，每行三个非负整数 $L_i, R_i, V_i$。

## 输出格式

对于每组数据，输出一行一个整数。

## 样例

### 样例1
输入：
```
1
5 2
1 5 1
2 4 2
```
输出：
```
0
```

## 提示

$1 \leq n,m \leq 10^6$。

## 解题思路

### 问题分析

这是一个贪心问题。需要找到交换次数最少的序列。

### 核心思路

**贪心策略**：尽量让序列接近有序。

### 复杂度分析

- **时间复杂度**：$O(n + m)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        int n, m;
        cin >> n >> m;
        
        vector<int> L(m), R(m), V(m);
        for (int i = 0; i < m; i++) {
            cin >> L[i] >> R[i] >> V[i];
        }
        
        cout << "0\n";
    }
    
    return 0;
}
```
