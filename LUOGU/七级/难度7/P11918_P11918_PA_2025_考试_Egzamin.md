# P11918 [PA 2025] 考试 / Egzamin

## 题目描述

期末考试有 $n$ 道题。对于每道题，你都可以选择是否作答。

如果选择作答第 $i$ 题，有 $p_i$ 的概率作答正确，$(1-p_i)$ 的概率作答错误。

每道题的评分规则如下：
- 作答正确：得 $1$ 分；
- 不作答：得 $0$ 分；
- 作答错误：倒扣 $1$ 分。

为了不挂科，你需要获得至少 $t$ 分。

在自由选择是否作答每道题的前提下，最大化不挂科的概率。

## 输入格式

第一行，两个正整数 $n,t$。
接下来 $n$ 行，第 $i$ 行一个实数 $p_i$。

## 输出格式

输出一行一个实数，表示所求的概率。

## 样例

### 样例1
输入：
```
4 3
0.5 0.5 0.5 0.5
```
输出：
```
0.5
```

## 提示

- $ 1 \leq t \leq n \leq 50\, 000 $；
- $p_i\in [0,1]$，且至多有 $9$ 位小数。

## 解题思路

### 问题分析

这是一个概率DP问题。需要选择作答的题目，使得获得至少 $t$ 分的概率最大。

### 核心思路

**动态规划**：
1. 按正确概率排序
2. 使用DP计算不挂科的概率

### 算法流程

```
按正确概率排序
使用DP计算不挂科的概率
输出答案
```

### 复杂度分析

- **时间复杂度**：$O(n \cdot t)$
- **空间复杂度**：$O(t)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <iomanip>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, t;
    cin >> n >> t;
    
    vector<double> p(n);
    for (int i = 0; i < n; i++) {
        cin >> p[i];
    }
    
    vector<int> idx(n);
    for (int i = 0; i < n; i++) idx[i] = i;
    sort(idx.begin(), idx.end(), [&](int i, int j) {
        return p[i] > p[j];
    });
    
    vector<double> dp(t + 1, 0);
    dp[0] = 1;
    
    for (int i = 0; i < n; i++) {
        int id = idx[i];
        vector<double> new_dp(t + 1, 0);
        
        for (int j = 0; j <= t; j++) {
            if (dp[j] == 0) continue;
            
            new_dp[j] = max(new_dp[j], dp[j]);
            
            if (j + 1 <= t) {
                new_dp[j + 1] = max(new_dp[j + 1], dp[j] * p[id]);
            }
            
            if (j - 1 >= 0) {
                new_dp[j - 1] = max(new_dp[j - 1], dp[j] * (1 - p[id]));
            }
        }
        
        dp = new_dp;
    }
    
    double ans = 0;
    for (int j = t; j <= t; j++) {
        ans = max(ans, dp[j]);
    }
    
    cout << fixed << setprecision(20) << ans << "\n";
    
    return 0;
}
```
