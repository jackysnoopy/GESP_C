# P9495 「SFCOI-3」进行一个魔的除 I

## 题目描述

房间中共有 $n$ 盏灯，初始状态可以用 $a_1\dots a_n$ 表示，其中 $\tt 0$ 表示这盏灯初始是关闭的，$\tt 1$ 表示这盏灯初始是打开的。

从第一天早晨开始，魔王与勇士轮流行动：
- 每天早晨，魔王可以选择连续的两盏灯，将它们的状态全部设定为 $\tt 0$；
- 每天晚上，勇士可以选择任意的至多三盏灯，将它们的状态全部设定为 $\tt 1$。

假设双方均采用最优策略，勇士想知道，最少需要多少天才能将所有灯状态设定为 $\tt 1$。

## 输入格式

第一行一个整数 $T$，表示数据组数。
对于每组数据：
- 第一行一个整数 $n$。
- 第二行共 $n$ 个数，依次表示 $a_1\dots a_n$。

## 输出格式

对于每组数据，一行一个整数，代表勇士抓到魔王所需要的最少天数。

## 样例

### 样例1
输入：
```
3
5
00000
5
11111
5
01010
```
输出：
```
2
0
1
```

## 提示

$1 \le T \le 10^4$，$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个博弈论问题。需要计算最小天数。

### 核心思路

**贪心策略**：每次尽量打开更多的灯。

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        int n;
        cin >> n;
        
        string a;
        cin >> a;
        
        int ones = 0;
        for (char c : a) {
            if (c == '1') ones++;
        }
        
        cout << (n - ones + 2) / 3 << "\n";
    }
    
    return 0;
}
```
