# P8121 「RdOI R3.5 附加」ACP-II

## 题目描述

你需要控制机器人玩躲避子弹的游戏。游戏的画面是一个在第一象限，坐标范围从 $(0,0)$ 至 $(n,m)$ 的矩形。机器人初始生成在坐标 $(0,0)$。

你可以通过以下几种指令来控制这个机器人：
- 0: 机器人静止不动
- 1: 机器人向左移动
- 2: 机器人向下移动
- 3: 机器人向上移动
- 4: 机器人向右移动

游戏中会生成 $b$ 个子弹，游戏共进行 $d$ 秒。

## 输入格式

第一行包含 $n, m, k, d, b$。
接下来 $b$ 行，每行包含子弹的信息。

## 输出格式

输出 $d$ 行，每行一个指令。

## 样例

### 样例1
输入：
```
10 10 1 5 2
1 5 0 0 1 0
3 5 0 0 0 1
```
输出：
```
4
4
4
4
4
```

## 提示

这是交互题。

## 解题思路

### 问题分析

这是一个交互问题。需要控制机器人躲避子弹。

### 核心思路

**贪心策略**：尽量向右移动。

### 复杂度分析

- **时间复杂度**：$O(d)$
- **空间复杂度**：$O(1)$

## 参考代码

```cpp
#include <iostream>
#include <vector>

    }

}
```
