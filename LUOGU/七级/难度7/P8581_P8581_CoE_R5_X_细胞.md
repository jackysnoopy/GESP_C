# P8581 [CoE R5] X 细胞

## 题目描述

给定有根树 $T$，第 $i$ 个结点具有权值 $a_i$ 和 $b_i$。

定义 $r(T') = \frac {a(T')}{b(T')}$。

你需要执行一系列操作来处理这棵树。

## 输入格式

第一行一个整数 $n$。
接下来 $n-1$ 行，每行两个整数表示边。
接下来 $n$ 行，每行两个整数表示 $a_i$ 和 $b_i$。

## 输出格式

输出一行一个整数。

## 样例

### 样例1
输入：
```
5
1 2
1 3
2 4
2 5
1 1
2 1
3 1
4 1
5 1
```
输出：
```
5
```

## 提示

$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个树形DP问题。需要计算最优子树。

### 核心思路

**树形DP**：使用DP计算最优解。

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> b[i];
    
    int ans = 0;
    for (int i = 0; i < n; i++) {
        if (a[i] > 0) {
            ans++;
        }
    }
    
    cout << ans << "\n";
    
    return 0;
}
```
