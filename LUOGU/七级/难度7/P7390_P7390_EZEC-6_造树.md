# P7390 「EZEC-6」造树

## 题目描述

你要帮 djy 造一棵树，满足以下条件：
- 由 $n$ 个点组成。
- $i$ 号点的度数为 $a_i$。

定义一条边 $(i,j)$ 的价值为 $b_i\times b_j$，你要在满足上述两个条件下，使所有边的价值和最大。

## 输入格式

第一行一个整数 $type$，表示数据生成方式。
第二行一个整数 $n$。
第三行 $n$ 个整数，表示 $a_i$。
第四行 $n$ 个整数，表示 $b_i$。

## 输出格式

一行一个整数，表示最大价值和。

## 样例

### 样例1
输入：
```
0
4
1 2 2 1
1 2 3 4
```
输出：
```
14
```

## 提示

$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个贪心问题。需要构造树使得边的价值和最大。

### 核心思路

**贪心策略**：优先连接价值高的点。

### 复杂度分析

- **时间复杂度**：$O(n \log n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int type;
    cin >> type;
    
    int n;
    cin >> n;
    
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> b[i];
    
    ll ans = 0;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            if (a[i] > 0 && a[j] > 0) {
                ans += (ll)b[i] * b[j];
                a[i]--;
                a[j]--;
            }
        }
    }
    
    cout << ans << "\n";
    
    return 0;
}
```
