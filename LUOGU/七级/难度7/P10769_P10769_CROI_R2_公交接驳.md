# P10769 「CROI · R2」公交接驳

## 题目描述

H 市有一条共有 $n$ 个换乘站的街道。你需要在公交线路上安排 $k$ 班公交车。

对于每一班车，你需要安排其发车的时刻和发车的换乘站，且需要保证从任意一个换乘站下车的乘客均能坐上公交车。

定义换乘站 $i$ 的不满意度为在该站点处，乘客等待首辆到达该站的公交车所花费的时间与乘客登上的公交车的始发站的重视程度的乘积。

你需要最小化所有 $n$ 个换乘站的不满意度之和。

## 输入格式

第一行包含一个正整数 $n$。
第二行包含 $n-1$ 个整数 $s_1,s_2,...,s_{n-1}$。
第三行包含 $n$ 个整数 $v_i$。
第四行包含一个正整数 $p$。
接下来共有 $p$ 组数据。

## 输出格式

输出 $p$ 行，每行 $q_i$ 个整数，表示在当前询问的 $k$ 的情况下，所有换乘站不满意度之和的最小值。

## 样例

### 样例1
输入：
```
3
3 4
6 2 1
2
1 3 7
2
1 2
2 3 5
3
1 2 4
```
输出：
```
12 0
36 4 0
```

## 提示

对于所有测试点，保证 $1\leq n\leq 1000$，$1\leq p\leq 10$，$s_i\geq t_{i+1}-t_i\geq 0$。

## 解题思路

### 问题分析

这是一个调度问题，需要安排 $k$ 班公交车的发车时间和站点，使得不满意度最小。

### 核心思路

**贪心策略**：
1. 当 $k$ 足够大时，可以在每个站点发车，不满意度为0
2. 当 $k$ 较小时，需要合理安排发车时间和站点

### 算法流程

```
对于每个询问k:
    如果k >= n，输出0
    否则模拟k班公交车的运行
    计算不满意度
```

### 复杂度分析

- **时间复杂度**：$O(p \cdot q \cdot n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<ll> s(n - 1);
    for (int i = 0; i < n - 1; i++) cin >> s[i];
    
    vector<ll> v(n);
    for (int i = 0; i < n; i++) cin >> v[i];
    
    int p;
    cin >> p;
    
    while (p--) {
        vector<ll> t(n);
        for (int i = 0; i < n; i++) cin >> t[i];
        
        int qi;
        cin >> qi;
        
        vector<ll> k(qi);
        for (int i = 0; i < qi; i++) cin >> k[i];
        
        for (int qi_idx = 0; qi_idx < qi; qi_idx++) {
            ll ki = k[qi_idx];
            
            if (ki >= n) {
                cout << "0";
                if (qi_idx < qi - 1) cout << " ";
                continue;
            }
            
            ll total_unsat = 0;
            vector<ll> bus_time(n, 0);
            
            for (int bus = 0; bus < ki; bus++) {
                int start = bus % n;
                ll departure = t[start];
                
                ll current_time = departure;
                for (int station = start; station < n; station++) {
                    if (station > start) {
                        current_time += s[station - 1];
                    }
                    bus_time[station] = max(bus_time[station], current_time);
                }
            }
            
            for (int i = 0; i < n; i++) {
                ll wait = max(0LL, bus_time[i] - t[i]);
                total_unsat += wait * v[i];
            }
            
            cout << total_unsat;
            if (qi_idx < qi - 1) cout << " ";
        }
        cout << "\n";
    }
    
    return 0;
}
```
