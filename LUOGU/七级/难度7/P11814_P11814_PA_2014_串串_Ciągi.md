# P11814 [PA 2014] 串串 / Ciągi

## 题目描述

给定 $k$ 个长度为 $n$ 的整数序列 $A_1,A_2,\cdots,A_k$。

定义两个长度为 $n$ 的序列 $a,b$ 的**距离**为：
$$\operatorname{dist}(a,b)=\sum_{1\le i\le n} |a_i-b_i|$$

你需要构造一个长度为 $n$ 的整数序列 $B$，最小化 $\displaystyle \max_{1\le i\le k} \operatorname{dist}(A_i,B)$ 的值。

## 输入格式

第一行，两个正整数 $n,k$。
接下来 $k$ 行，第 $i$ 行 $n$ 个整数 $A_{i,1},A_{i,2},\cdots,A_{i,n}$。

## 输出格式

输出 $n$ 个整数，表示你构造的 $B$ 序列。

## 样例

### 样例1
输入：
```
5 3
1 -1 2 -1 2
1 2 2 1 2
2 2 -1 1 1
```
输出：
```
1 2 2 1 2
```

## 提示

- $2\le n\le 10^5$；
- $2\le k\le 5$；
- $|A_{i,j}|\le 10^9$。

## 解题思路

### 问题分析

这是一个优化问题。需要构造序列 $B$，使得 $B$ 与所有 $A_i$ 的最大距离最小。

### 核心思路

**中位数策略**：对于每个位置 $j$，选择 $B[j]$ 为所有 $A_i[j]$ 的中位数。

### 算法流程

```
for each position j:
    收集所有A_i[j]
    选择中位数作为B[j]
输出B
```

### 复杂度分析

- **时间复杂度**：$O(k \cdot n \log k)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, k;
    cin >> n >> k;
    
    vector<vector<int>> A(k, vector<int>(n));
    for (int i = 0; i < k; i++) {
        for (int j = 0; j < n; j++) {
            cin >> A[i][j];
        }
    }
    
    vector<int> B(n);
    for (int j = 0; j < n; j++) {
        vector<int> vals(k);
        for (int i = 0; i < k; i++) {
            vals[i] = A[i][j];
        }
        sort(vals.begin(), vals.end());
        B[j] = vals[k / 2];
    }
    
    for (int j = 0; j < n; j++) {
        if (j > 0) cout << " ";
        cout << B[j];
    }
    cout << "\n";
    
    return 0;
}
```
