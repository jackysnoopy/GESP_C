# P11239 [KTSC 2024 R2] 跳跃游戏

## 题目描述

KOI 公司推出的跳跃游戏是一款通过多次跳跃穿越 $N$ 个踏板的游戏。

对于每个 $i$ $(0 \leq i \leq N-1)$，主角可以选择走一步或跳跃。如果选择走一步，主角将移动到 $i+1$ 号踏板，得分不变。如果选择跳跃，主角将移动到 $i+K$ 号踏板，得分增加 $A[i]$。

游戏在主角到达 $N-1$ 号踏板的右侧时成功结束。游戏的目标是通过合理控制主角的行动，使得分最大化并成功结束游戏。

## 输入格式

示例评测程序按以下格式读取输入：
- 第 $1$ 行：$N\,Q\,K$
- 第 $2+i$ 行：$L[i]\,R[i]$

## 输出格式

第 $1$ 行：`play_game` 返回的值

## 样例

### 样例1
输入：
```
5 3 2
0 2
1 3
2 4
```
输出：
```
3
```

## 提示

对于所有输入数据，满足 $1 \leq N \leq 10^{12}$，$1 \leq Q \leq 2.5\cdot 10^5$，$1 \leq K \leq N$。

## 解题思路

### 问题分析

这是一个动态规划问题。需要在 $N$ 个踏板上选择走一步或跳跃，使得总得分最大。

### 核心思路

**动态规划**：
1. 使用差分数组处理区间加操作
2. 计算每个踏板的 $A[i]$
3. 使用DP计算最大得分

### 算法流程

```
使用差分数组处理Q次操作
计算每个踏板的A[i]
使用DP计算最大得分
返回答案
```

### 复杂度分析

- **时间复杂度**：$O(N + Q)$
- **空间复杂度**：$O(N)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

long long play_game(long long N, int Q, long long K, vector<long long> L, vector<long long> R) {
    vector<ll> diff(N + 2, 0);
    for (int i = 0; i < Q; i++) {
        diff[L[i]] += 1;
        diff[R[i] + 1] -= 1;
    }
    
    vector<ll> A(N, 0);
    ll cur = 0;
    for (int i = 0; i < N; i++) {
        cur += diff[i];
        A[i] = cur;
    }
    
    vector<ll> dp(N + 1, -1);
    dp[0] = 0;
    
    for (int i = 0; i < N; i++) {
        if (dp[i] < 0) continue;
        
        if (i + 1 <= N) {
            dp[i + 1] = max(dp[i + 1], dp[i]);
        }
        
        if (i + K <= N) {
            dp[i + K] = max(dp[i + K], dp[i] + A[i]);
        }
    }
    
    ll ans = 0;
    for (int i = N; i <= N; i++) {
        ans = max(ans, dp[i]);
    }
    
    return ans;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    ll N, K;
    int Q;
    cin >> N >> Q >> K;
    
    vector<ll> L(Q), R(Q);
    for (int i = 0; i < Q; i++) {
        cin >> L[i] >> R[i];
    }
    
    cout << play_game(N, Q, K, L, R) << "\n";
    
    return 0;
}
```
