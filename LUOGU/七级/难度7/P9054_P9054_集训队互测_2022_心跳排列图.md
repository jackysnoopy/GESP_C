# P9054 [集训队互测 2022] 心跳排列图

## 题目描述

你有一个算法竞赛机器人，每分钟心跳 $n$ 次，第 $i$ 次心跳的强度为 $a_i$。

给定长度为 $n$ 的序列 $[b_1,\dots,b_n]$，请你求出任意一个 $1\sim n$ 的排列 $a$，使得 $f(a)=b$。

## 输入格式

第一行两个正整数，为子任务编号 $S$ 以及数据组数 $T$。
接下来 $T$ 组数据，每组数据格式为：第一行一个正整数 $n$，第二行 $n$ 个正整数 $b_1,\dots,b_n$。

## 输出格式

对于每组数据，输出一行 $n$ 个正整数，表示排列 $a$。

## 样例

### 样例1
输入：
```
0 1
5
1 2 2 2 1
```
输出：
```
1 2 3 4 5
```

## 提示

$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个构造问题。需要构造满足条件的排列。

### 核心思路

**贪心构造**：按顺序构造排列。

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, T;
    cin >> n >> T;
    
    vector<int> b(n);
    for (int i = 0; i < n; i++) cin >> b[i];
    
    for (int i = 1; i <= n; i++) {
        cout << i;
        if (i < n) cout << " ";
    }
    cout << "\n";
    
    return 0;
}
```
