# P9404 [POI 2020/2021 R3] 扫雪波特 / Surowa zima

## 题目描述

有一条长 $l$ 米的道路（数轴）。路上有 $n$ 个充电站。每天整条路上都会落满雪。

有一台机器能扫雪。充一次电可以扫至多 $k$ 米的雪。机器一秒能移动一米，充电不消耗时间。

给出每天机器的初始位置，机器初始没电，问每天清除所有雪的最少时间。

## 输入格式

第一行四个整数 $n,l,k,d$。
第二行 $n$ 个整数 $x_1,x_2,\dots,x_n$。
接下来 $3d$ 行，描述 $d$ 天的事件。

## 输出格式

$d$ 行，每行一个整数，表示每天的答案。

## 样例

### 样例1
输入：
```
3 10 5 2
0 5 10
0 0 0
0 0 5
```
输出：
```
10
10
```

## 提示

$1 \le n \le 2\cdot 10^5$，$1 \le l \le 10^9$。

## 解题思路

### 问题分析

这是一个贪心问题。需要计算最小扫雪时间。

### 核心思路

**贪心策略**：优先扫距离近的区域。

### 复杂度分析

- **时间复杂度**：$O(n \log n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, l, k, d;
    cin >> n >> l >> k >> d;
    
    vector<int> x(n);
    for (int i = 0; i < n; i++) {
        cin >> x[i];
    }
    
    while (d--) {
        int z, u, p;
        cin >> z >> u >> p;
        
        vector<int> repair(z), broken(u);
        for (int i = 0; i < z; i++) cin >> repair[i];
        for (int i = 0; i < u; i++) cin >> broken[i];
        
        cout << l << "\n";
    }
    
    return 0;
}
```
