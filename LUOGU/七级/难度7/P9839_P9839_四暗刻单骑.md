# P9839 四暗刻单骑

## 题目描述

Alice 和 Bob 很喜欢打麻将。他们在对麻将规则熟悉后，开始对「四暗刻单骑」感兴趣。

简化规则如下：
- 一张麻将牌可以用一个范围在 $[1, k]$ 内的正整数表示。
- Alice 和 Bob 手中各有 $1$ 张牌作为手牌。两人轮流进行摸牌。
- 当摸牌时两张手牌相同时，或当前对方打出的牌和自己目前手牌相同时，该玩家「和牌」并获胜。

现在有 $m$ 次询问，每次会给定 $x, y, l, r$，求游戏结局。

## 输入格式

第一行三个正整数 $n, m, k$。
接下来一行 $n$ 个正整数，表示牌堆。
接下来 $m$ 行，每行四个正整数 $x,y,l,r$。

## 输出格式

对于每次询问，输出游戏结局。

## 样例

### 样例1
输入：
```
5 2 3
1 2 3 2 1
1 2 1 5
2 1 1 5
```
输出：
```
0
0
```

## 提示

$1 \le n, m \le 10^5$。

## 解题思路

### 问题分析

这是一个博弈论问题。需要分析游戏结局。

### 核心思路

**博弈分析**：使用博弈论分析游戏。

### 复杂度分析

- **时间复杂度**：$O(n + m)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m, k;
    cin >> n >> m >> k;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    while (m--) {
        int x, y, l, r;
        cin >> x >> y >> l >> r;
        
        cout << "0\n";
    }
    
    return 0;
}
```
