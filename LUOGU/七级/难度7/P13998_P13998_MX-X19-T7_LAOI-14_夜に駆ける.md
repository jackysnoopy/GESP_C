# P13998 【MX-X19-T7】「LAOI-14」夜に駆ける

## 题目描述

给定正整数 $n, m$，保证 $m$ 为偶数。

现有一个机器人在一个 $n$ 行 $m$ 列的非负整数矩阵 $A$ 中。机器人会循环进行如下操作：
1. 移动到 $y-1$ 或 $y+1$ 或不移动，不得移动超出矩阵。
2. 向下移动一步，若超出矩阵大小则停止循环。

现在请你构造一个 $n\times m$ 的矩阵 $A$，其中矩阵的每一行中都恰好有 $\frac{m}{2}$ 个 $0$ 和 $\frac{m}{2}$ 个 $1$，使得机器人所经过的所有 $A_{i,j}$ 之和尽可能小。

## 输入格式

仅一行，两个正整数 $n, m$，保证 $m$ 为偶数。

## 输出格式

输出 $n$ 行，每行 $m$ 个非负整数（$0$ 或 $1$），表示你构造的矩阵。

## 样例

### 样例1
输入：
```
2 4
```
输出：
```
0101
1010
```

## 提示

本题使用自定义校验器。

## 解题思路

### 问题分析

这是一个构造问题。需要构造矩阵使得机器人路径和最小。

### 核心思路

**构造策略**：交替放置0和1。

### 算法流程

```
对于每一行:
    交替放置0和1
输出矩阵
```

### 复杂度分析

- **时间复杂度**：$O(n \cdot m)$
- **空间复杂度**：$O(n \cdot m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    for (int i = 0; i < n; i++) {
        string row;
        for (int j = 0; j < m; j++) {
            if (i % 2 == 0) {
                row += (j % 2 == 0) ? '0' : '1';
            } else {
                row += (j % 2 == 0) ? '1' : '0';
            }
        }
        cout << row << "\n";
    }
    
    return 0;
}
```
