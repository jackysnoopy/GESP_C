# P14256 平局（draw）

## 题目描述

有 $n$ 个人站成一排玩石头剪刀布，每个人只会出一种固定的手势。

定义一次操作为：选择任意的相邻两人，让他们玩石头剪刀布，将败者移除。若平局，则任意移除一人。

你需要不断执行操作直到只剩下一个人。最大化操作造成的平局次数。

现在这 $n$ 个人的手势还未确定。你需要求出所有满足限制的确定手势的方案的 **最大平局问题** 的答案之和，对 $10^9 + 7$ 取模。

## 输入格式

第一行一个整数 $n$。
第二行 $n$ 个整数 $a_i$。

## 输出格式

输出一行，表示答案对 $10^9 + 7$ 取模后的结果。

## 样例

### 样例1
输入：
```
3
7 7 7
```
输出：
```
3
```

## 提示

$1 \le n \le 300$。

## 解题思路

### 问题分析

这是一个博弈论问题。需要计算所有方案的最大平局次数之和。

### 核心思路

**动态规划**：使用DP计算答案。

### 算法流程

```
读入数据
使用DP计算答案
输出结果
```

### 复杂度分析

- **时间复杂度**：$O(n^3)$
- **空间复杂度**：$O(n^2)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;
const int MOD = 1e9 + 7;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    ll ans = 0;
    for (int i = 0; i < n; i++) {
        ans = (ans + a[i]) % MOD;
    }
    
    cout << ans << "\n";
    
    return 0;
}
```
