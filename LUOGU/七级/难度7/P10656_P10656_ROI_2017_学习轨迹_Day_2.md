# P10656 [ROI 2017] 学习轨迹 (Day 2)

## 题目描述

THU 和 PKU 同时开设了一批课程。你可以选择听 THU 的 $l_1 \sim r_1$ 节课，同时在 PKU 听 $l_2 \sim r_2$ 节课。同一类别的课你不能听两次。

你需要求出可能的听课方案中乐趣度最大的是多少以及具体的安排。

## 输入格式

第一行两个整数 $n,m$。
接下来四行，分别表示 $a,x,b,y$。

## 输出格式

第一行一个整数表示最大乐趣度。
第二行两个整数 $l_1,r_1$（如果不需要在THU听课，输出 `0 0`）。
第三行两个整数 $l_2,r_2$（如果不需要在PKU听课，输出 `0 0`）。

## 样例

### 样例1
输入：
```
7 5
3 1 4 8 6 9 2
2 7 4 10 1 5 3
9 2 11 3 8
3 5 3 4 12
```
输出：
```
39
2 6
2 4
```

## 解题思路

### 问题分析

这是一个区间选择问题，需要在两个序列上选择区间，使得总乐趣度最大，且没有类别冲突。

### 核心思路

**暴力枚举**（适用于小数据）：
1. 枚举THU的区间 $[l_1, r_1]$
2. 枚举PKU的区间 $[l_2, r_2]$
3. 检查类别冲突
4. 记录最优解

### 算法流程

```
枚举所有可能的区间组合
检查类别冲突
更新最优解
输出答案
```

### 复杂度分析

- **时间复杂度**：$O(n^2 m^2)$（暴力）
- **空间复杂度**：$O(n + m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<int> a(n), x(n), b(m), y(m);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < n; i++) cin >> x[i];
    for (int i = 0; i < m; i++) cin >> b[i];
    for (int i = 0; i < m; i++) cin >> y[i];
    
    ll ans = 0;
    int best_l1 = 0, best_r1 = 0, best_l2 = 0, best_r2 = 0;
    
    for (int l1 = 0; l1 <= n; l1++) {
        for (int r1 = l1 - 1; r1 < n; r1++) {
            for (int l2 = 0; l2 <= m; l2++) {
                for (int r2 = l2 - 1; r2 < m; r2++) {
                    vector<bool> used(n + m + 1, false);
                    bool ok = true;
                    ll total = 0;
                    
                    for (int i = l1; i <= r1; i++) {
                        if (used[a[i]]) {
                            ok = false;
                            break;
                        }
                        used[a[i]] = true;
                        total += x[i];
                    }
                    if (!ok) continue;
                    
                    for (int i = l2; i <= r2; i++) {
                        if (used[b[i]]) {
                            ok = false;
                            break;
                        }
                        used[b[i]] = true;
                        total += y[i];
                    }
                    if (!ok) continue;
                    
                    if (total > ans) {
                        ans = total;
                        best_l1 = l1 + 1;
                        best_r1 = r1 + 1;
                        best_l2 = l2 + 1;
                        best_r2 = r2 + 1;
                    }
                }
            }
        }
    }
    
    cout << ans << "\n";
    if (best_l1 == 0) {
        cout << "0 0\n";
    } else {
        cout << best_l1 << " " << best_r1 << "\n";
    }
    if (best_l2 == 0) {
        cout << "0 0\n";
    } else {
        cout << best_l2 << " " << best_r2 << "\n";
    }
    
    return 0;
}
```
