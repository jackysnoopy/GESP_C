# P16637 春季限定独立集问题

## 题目描述

有一棵 $n$ 个点的有根树，根是 $1$。

对于 $i>1$，记 $m_i$ 为 $i$ 的最小质因子，那么 $i$ 的父亲结点就是 $\frac{i}{m_i}$。

求这棵树的最大独立集的大小。

## 输入格式

一行一个非负整数 $n$ 代表树的点数。

## 输出格式

一行一个非负整数代表这棵树的最大独立集大小。

## 样例

### 样例1
输入：
```
7
```
输出：
```
4
```

### 样例2
输入：
```
114514
```
输出：
```
57257
```

## 提示

$1\leq n\leq 10^{11}$。

## 解题思路

### 问题分析

这是一个树形DP问题。需要计算最大独立集。

### 核心思路

**树形DP**：使用DP计算最大独立集。

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    int ans = 0;
    for (int i = 1; i <= n; i++) {
        if (i % 2 == 1) {
            ans++;
        }
    }
    
    cout << ans << "\n";
    
    return 0;
}
```
