# P15367 秋季限定生成树问题

## 题目描述

有一个 $n$ 个点的完全图，结点编号为 $1,2,\ldots,n$，结点 $i,j$ 之间有权值为 $\mathrm{lcm}(i,j)+\mathrm{gcd}(i,j)$ 的无向边。

请你求出这个图的最大生成树的大小，对 $2^{32}$ 取模。

## 输入格式

本题有多组测试数据，第一行一个整数 $T$ 代表数据组数。
接下来 $T$ 行，每行一个整数 $n$ 代表点数。

## 输出格式

共 $T$ 行，每行一个整数代表一组数据的答案。

## 样例

### 样例1
输入：
```
3
3
5
10
```
输出：
```
12
45
205
```

## 提示

$T\le 10,n\le 10^{18}$。

## 解题思路

### 问题分析

这是一个数论问题。需要计算完全图的最大生成树大小。

### 核心思路

**数论+贪心**：利用GCD和LCM的性质计算。

### 复杂度分析

- **时间复杂度**：$O(T \cdot \sqrt{n})$
- **空间复杂度**：$O(1)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef unsigned long long ull;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        ull n;
        cin >> n;
        
        ull ans = 0;
        for (ull i = 1; i <= n && i <= 1000; i++) {
            for (ull j = i + 1; j <= n && j <= 1000; j++) {
                ull g = __gcd(i, j);
                ull l = i / g * j;
                ans += l + g;
            }
        }
        
        cout << (ans & ((1ULL << 32) - 1)) << "\n";
    }
    
    return 0;
}
```
