# P9250 [集训队作业 2019] 蜀道难

## 题目描述

「蜀道之难，难于上青天……」

这是一道有趣的题目，涉及图论和组合数学。

## 输入格式

第一行一个整数 $n$。
接下来若干行输入。

## 输出格式

输出一行一个整数。

## 样例

### 样例1
输入：
```
5
```
输出：
```
0
```

## 提示

$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个图论问题。

### 核心思路

**贪心策略**：使用贪心算法求解。

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n;

}
```
