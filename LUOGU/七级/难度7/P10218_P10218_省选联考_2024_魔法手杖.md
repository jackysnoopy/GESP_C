# P10218 [省选联考 2024] 魔法手杖

## 题目描述

小 $\omega$ 的魔法手杖镶嵌着 $n$ 颗魔法水晶，魔力值分别为 $a_1,a_2,\dots,a_n$。

他可以进行一次秘术：选择 $x$，将所有魔法水晶的魔力值由 $a_i$ 变为 $(a_i \oplus x)$。

他还可以花费 $b_i$ 的体力值对第 $i$ 个魔法水晶进行定向加强，将原本应变为 $(a_i \oplus x)$ 的魔力值变为 $(a_i+x)$。

小 $\omega$ 想知道他在加强魔法手杖后，魔法手杖的魔力值最大能为多少。

## 输入格式

第一行包含两个整数 $c,T$，表示测试点编号与测试数据组数。
对于每组数据：
- 第一行三个整数 $n,m,k$；
- 第二行 $n$ 个整数 $a_1,a_2,\dots,a_n$；
- 第三行 $n$ 个整数 $b_1,b_2,\dots,b_n$。

## 输出格式

对于每组测试数据输出一行一个整数表示小 $\omega$ 能获得魔法手杖魔力值的最大值。

## 样例

### 样例1
输入：
```
1 2
5 2 3
1 1 2 3 7
1 1 0 3 2
1 1 1
1
0
```
输出：
```
5
2
```

## 提示

对于所有测试数据，保证 $1 \leq n \leq 10^5$，$0 \leq m \leq 10^9$，$0 \leq k \leq 120$。

## 解题思路

### 问题分析

需要选择子集 $S$ 和值 $x$，使得 $\min(\min_{i \in S}(a_i+x),\min_{i \notin S}(a_i \oplus x))$ 最大，且 $S$ 中元素的 $b_i$ 之和不超过 $m$。

### 核心思路

**贪心+枚举**：
1. 枚举 $x$ 的值（当 $k$ 较小时）
2. 对于每个 $x$，计算每个元素的值 $a_i \oplus x$
3. 按值排序，贪心选择 $S$ 使得最小值最大

### 算法流程

```
for each x:
    计算所有 a[i] ^ x
    按值排序
    贪心选择S使得cost <= m
    更新答案
```

### 复杂度分析

- **时间复杂度**：$O(2^k \cdot n \log n)$（当 $k$ 较小时）
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int c, T;
    cin >> c >> T;
    
    while (T--) {
        int n, k;
        ll m;
        cin >> n >> m >> k;
        
        vector<ll> a(n), b(n);
        for (int i = 0; i < n; i++) cin >> a[i];
        for (int i = 0; i < n; i++) cin >> b[i];
        
        ll ans = 0;
        
        if (k <= 20) {
            for (ll x = 0; x < (1LL << k); x++) {
                vector<ll> vals(n);
                for (int i = 0; i < n; i++) {
                    vals[i] = a[i] ^ x;
                }
                
                vector<int> idx(n);
                for (int i = 0; i < n; i++) idx[i] = i;
                sort(idx.begin(), idx.end(), [&](int i, int j) {
                    return vals[i] < vals[j];
                });
                
                ll cost = 0;
                int chosen = 0;
                for (int i = 0; i < n; i++) {
                    if (cost + b[idx[i]] <= m) {
                        cost += b[idx[i]];
                        chosen++;
                    } else {
                        break;
                    }
                }
                
                if (chosen > 0) {
                    ll min_val = vals[idx[chosen - 1]];
                    ans = max(ans, min_val);
                }
            }
        }
        
        cout << ans << "\n";
    }
    
    return 0;
}
```
