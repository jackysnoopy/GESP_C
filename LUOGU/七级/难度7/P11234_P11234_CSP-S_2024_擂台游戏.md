# P11234 [CSP-S 2024] 擂台游戏

## 题目描述

小 S 想要举办一场擂台游戏。如果共有 $2^k$ 名选手参加，那么游戏分为 $k$ 轮进行。

每位选手都有一个能力值。对于每场比赛，会先抽签决定一个数 $0/1$，表示谁是擂主。擂主获胜当且仅当他的能力值 $a\geq R$。

小 S 先后陆续收到了 $n$ 位选手的报名信息。小 S 关心的是，补充尽量少的选手使总人数为 $2$ 的整次幂，且所有可能成为总冠军的选手的编号之和是多少。

## 输入格式

第一行包含两个正整数 $n, m$。
第二行包含 $n$ 个非负整数 $a'_1,a'_2,\dots,a'_n$。
第三行包含 $m$ 个正整数 $c_1, c_2, \dots , c_m$。
接下来的 $K$ 行当中，第 $R$ 行包含 $2^{K-R}$ 个数。
接下来一行包含一个正整数 $T$。
接下来共 $T$ 行，每行描述一组数据。

## 输出格式

共输出 $T$ 行，对于每组数据，输出一行包含一个整数。

## 样例

### 样例1
输入：
```
5 5
0 0 0 0 0
5 4 1 2 3
1001
10
1
4
2 1 0 0
1 2 1 0
0 2 3 1
2 2 0 1
```
输出：
```
5
19
7
1
```

## 提示

对于所有测试数据，保证：$2 \leq n, m \leq 10^5$，$0 \leq a_i, X_j < 2^{31}$，$1 \leq c_i \leq n$，$1 \leq T \leq 256$。

## 解题思路

### 问题分析

这是一个模拟问题，需要模拟擂台游戏的过程。

### 核心思路

**模拟比赛过程**：
1. 对于每个询问，计算前 $c_i$ 个人的可能冠军
2. 模拟比赛过程
3. 计算可能成为总冠军的选手编号之和

### 算法流程

```
对于每组数据:
    计算真实能力值
    对于每个询问:
        模拟比赛过程
        计算可能的冠军
        累加答案
```

### 复杂度分析

- **时间复杂度**：$O(T \cdot m \cdot n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<ll> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    vector<int> c(m);
    for (int i = 0; i < m; i++) cin >> c[i];
    
    int K = 0;
    while ((1 << K) < n) K++;
    
    vector<vector<int>> d(K + 1);
    for (int R = 1; R <= K; R++) {
        int cnt = (1 << (K - R));
        string s;
        cin >> s;
        d[R].resize(cnt);
        for (int i = 0; i < cnt; i++) {
            d[R][i] = s[i] - '0';
        }
    }
    
    int T;
    cin >> T;
    
    while (T--) {
        ll X0, X1, X2, X3;
        cin >> X0 >> X1 >> X2 >> X3;
        
        vector<ll> real_a(n);
        for (int i = 0; i < n; i++) {
            ll X;
            if (i % 4 == 0) X = X0;
            else if (i % 4 == 1) X = X1;
            else if (i % 4 == 2) X = X2;
            else X = X3;
            real_a[i] = a[i] ^ X;
        }
        
        ll result = 0;
        
        for (int qi = 0; qi < m; qi++) {
            int ci = c[qi];
            
            int k = 0;
            while ((1 << k) < ci) k++;
            
            vector<ll> powers(1 << k, 0);
            for (int i = 0; i < ci; i++) {
                powers[i] = real_a[i];
            }
            
            for (int R = 1; R <= k; R++) {
                vector<ll> new_powers;
                int cnt = (1 << (k - R));
                for (int G = 0; G < cnt; G++) {
                    ll p1 = powers[2 * G];
                    ll p2 = powers[2 * G + 1];
                    int host = d[R][G];
                    
                    ll host_power = (host == 0) ? p1 : p2;
                    ll other_power = (host == 0) ? p2 : p1;
                    
                    if (host_power >= R) {
                        new_powers.push_back(host_power);
                    } else {
                        new_powers.push_back(other_power);
                    }
                }
                powers = new_powers;
            }
            
            ll sum = 0;
            for (int i = 0; i < ci; i++) {
                sum += (i + 1);
            }
            
            result ^= ((qi + 1) * sum);
        }
        
        cout << result << "\n";
    }
    
    return 0;
}
```
