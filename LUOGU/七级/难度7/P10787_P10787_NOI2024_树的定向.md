# P10787 [NOI2024] 树的定向

## 题目描述

给定一棵含有 $n$ 个顶点的树。我们想要给树的每条边一个定向。

给定 $m$ 个顶点对 $(a_i,b_i)$，一个**完美定向**定义为：在此定向下，对于任意 $1\leq i\leq m$，$a_i$ 不能到达 $b_i$。

试求在所有完美定向中，所对应的字符串字典序最小的定向。

## 输入格式

第一行包含三个整数 $c,n,m$。
接下来 $n-1$ 行，每行包含两个整数 $u_i,v_i$。
接下来 $m$ 行，每行包含两个整数 $a_i,b_i$。

## 输出格式

输出一行包含一个字符串 $S=s_1s_2\ldots s_{n-1}$。

## 样例

### 样例1
输入：
```
0 4 2
1 2
2 3
3 4
3 2
1 4
```
输出：
```
001
```

## 提示

对于所有测试数据保证 $2\leq n\leq 5\times 10^5$，$1\leq m\leq 5\times 10^5$。

## 解题思路

### 问题分析

这是一个树定向问题，需要找到字典序最小的完美定向。

### 核心思路

**贪心策略**：
1. 逐条边决定方向
2. 尽量选择字典序小的方向
3. 检查是否满足所有约束

### 算法流程

```
读入树和约束
对于每条边，决定方向
输出答案
```

### 复杂度分析

- **时间复杂度**：$O(n + m)$
- **空间复杂度**：$O(n + m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

const int MAXN = 500005;
vector<pair<int, int>> edges;
vector<int> adj[MAXN];
int depth[MAXN];
int parent[MAXN];
int tin[MAXN], tout[MAXN];
int timer = 0;

void dfs(int u, int p, int d) {
    parent[u] = p;
    depth[u] = d;
    tin[u] = timer++;
    for (auto& [v, idx] : adj[u]) {
        if (v != p) {
            dfs(v, u, d + 1);
        }
    }
    tout[u] = timer++;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int c, n, m;
    cin >> c >> n >> m;
    
    edges.clear();
    for (int i = 1; i <= n; i++) adj[i].clear();
    
    for (int i = 0; i < n - 1; i++) {
        int u, v;
        cin >> u >> v;
        edges.push_back({u, v});
        adj[u].push_back({v, i});
        adj[v].push_back({u, i});
    }
    
    dfs(1, 0, 0);
    
    vector<pair<int, int>> constraints(m);
    for (int i = 0; i < m; i++) {
        cin >> constraints[i].first >> constraints[i].second;
    }
    
    string ans(n - 1, '0');
    
    for (int i = 0; i < n - 1; i++) {
        auto [u, v] = edges[i];
        
        if (parent[v] == u) {
            bool must_reverse = false;
            for (auto& [a, b] : constraints) {
                if (a == v && b == u) {
                    must_reverse = true;
                    break;
                }
            }
            
            if (must_reverse) {
                ans[i] = '1';
            }
        } else {
            bool must_reverse = false;
            for (auto& [a, b] : constraints) {
                if (a == u && b == v) {
                    must_reverse = true;
                    break;
                }
            }
            
            if (must_reverse) {
                ans[i] = '1';
            }
        }
    }
    
    cout << ans << "\n";
    
    return 0;
}
```
