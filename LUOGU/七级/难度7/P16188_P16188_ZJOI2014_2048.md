# P16188 [ZJOI2014] 2048

## 题目描述

$2048$ 是最近十分风靡的一款小游戏。

现在，我们想让 AI 尽量获得更高的分数。游戏中玩家不知道随机生成瓦片的位置和大小，在本题中，我们给定随机算法，并且每次只生成标着 $2$ 的瓦片。

## 输入格式

第一行包含一个整数 $seed$，表示随机种子。

## 输出格式

输出若干行，每行一个字符表示操作方向（L/R/U/D）。

## 样例

### 样例1
输入：
```
12345
```
输出：
```
L
D
L
D
...
```

## 提示

本题为交互题。

## 解题思路

### 问题分析

这是一个博弈问题。需要设计AI策略获得高分。

### 核心思路

**贪心策略**：交替向左和向下移动。

### 复杂度分析

- **时间复杂度**：$O(1)$
- **空间复杂度**：$O(1)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int seed;
    cin >> seed;
    
    for (int i = 0; i < 1000; i++) {
        if (i % 2 == 0) {
            cout << "L\n";
        } else {
            cout << "D\n";
        }
    }
    
    return 0;
}
```
