# P13538 [IOI 2025] 节日游戏（festival）

## 题目描述

Nayra is at a festival playing a game where the grand prize is a trip to Laguna Colorado. The game consists of using tokens to buy coupons. Buying a coupon may result in additional tokens. The goal is to get as many coupons as possible.

She starts the game with $A$ tokens. There are $N$ coupons, numbered from $0$ to $N - 1$. Nayra has to pay $P[i]$ tokens to buy coupon $i$. After buying coupon $i$, the remaining number of tokens she has gets multiplied by $T[i]$.

## 输入格式

第一行包含两个整数 $A$ 和 $N$。
第二行包含 $N$ 个整数 $P[i]$。
第三行包含 $N$ 个整数 $T[i]$。

## 输出格式

第一行输出一个整数，表示最多能购买的优惠券数量。
第二行输出购买顺序。

## 样例

### 样例1
输入：
```
10 3
5 3 8
2 3 1
```
输出：
```
2
1 0
```

## 提示

$1 \leq A \leq 10^9$，$1 \leq N \leq 10^5$。

## 解题思路

### 问题分析

这是一个贪心问题。需要选择购买优惠券的顺序，使得购买数量最大。

### 核心思路

**贪心策略**：优先选择类型大的优惠券。

### 算法流程

```
按类型排序
贪心购买
输出结果
```

### 复杂度分析

- **时间复杂度**：$O(N \log N)$
- **空间复杂度**：$O(N)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<int> max_coupons(int A, vector<int> P, vector<int> T) {
    int n = P.size();
    
    vector<int> order(n);
    for (int i = 0; i < n; i++) order[i] = i;
    
    sort(order.begin(), order.end(), [&](int i, int j) {
        if (T[i] != T[j]) return T[i] > T[j];
        return P[i] < P[j];
    });
    
    vector<int> result;
    int tokens = A;
    
    for (int i = 0; i < n; i++) {
        int idx = order[i];
        if (tokens >= P[idx]) {
            tokens -= P[idx];
            tokens *= T[idx];
            result.push_back(idx);
        }
    }
    
    return result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int A, n;
    cin >> A >> n;
    
    vector<int> P(n), T(n);
    for (int i = 0; i < n; i++) cin >> P[i];
    for (int i = 0; i < n; i++) cin >> T[i];
    
    vector<int> result = max_coupons(A, P, T);
    
    cout << result.size() << "\n";
    for (int i = 0; i < result.size(); i++) {
        if (i > 0) cout << " ";
        cout << result[i];
    }
    cout << "\n";
    
    return 0;
}
```
