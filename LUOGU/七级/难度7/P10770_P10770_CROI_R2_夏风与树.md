# P10770 「CROI · R2」夏风与树

## 题目描述

Alice 和 Bob 在种树。Alice 拥有 $1\sim n$ 号结点，Bob 拥有 $(n+1)\sim 2n$ 号结点。

Alice 为 $2\sim n$ 号点决定父亲，Bob 为 $(n+1)\sim 2n$ 号点决定父亲。

最后，Alice 会从 $1$ 号点开始进行深度优先搜索，每遇到一个没访问过的点就将它上面的权值加入序列末尾。

Alice 希望最终序列的字典序尽可能小，Bob 希望最终序列的字典序尽可能大，并且他们二人都会采取最优策略。

## 输入格式

第一行一个整数 $t$，表示子任务编号。
第二行一个正整数 $n$。
第三行 $2n$ 个正整数，表示排列 $a$。
第四行 $(n-1)$ 个整数，表示 $2\sim n$ 号结点的父亲。

## 输出格式

一行，表示最终序列。

## 样例

### 样例1
输入：
```
0
5
10 5 1 8 4 3 7 6 2 9
1 1 1 3
```
输出：
```
10 1 4 9 7 6 5 8 3 2
```

## 提示

对于 $100\%$ 的数据，$1\le n\le 10^5$，保证序列 $a$ 是一个 $1\sim 2n$ 的排列。

## 解题思路

### 问题分析

这是一个博弈论问题。Alice和Bob分别控制树的结构，然后进行DFS。

### 核心思路

**博弈分析**：
1. Alice希望DFS序列字典序最小
2. Bob希望DFS序列字典序最大
3. 两人都采取最优策略

### 算法流程

```
读入树结构
进行DFS
输出序列
```

### 复杂度分析

- **时间复杂度**：$O(n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <functional>
using namespace std;

const int MAXN = 200005;
vector<int> adj[MAXN];
int a[MAXN];
int n;

void dfs(int u, int p, vector<bool>& visited, vector<int>& result) {
    visited[u] = true;
    result.push_back(a[u]);
    for (int v : adj[u]) {
        if (!visited[v]) {
            dfs(v, u, visited, result);
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int t;
    cin >> t;
    
    cin >> n;
    
    for (int i = 1; i <= 2 * n; i++) {
        cin >> a[i];
        adj[i].clear();
    }
    
    for (int i = 2; i <= n; i++) {
        int p;
        cin >> p;
        if (p > 0) {
            adj[i].push_back(p);
            adj[p].push_back(i);
        }
    }
    
    vector<int> result;
    vector<bool> visited(2 * n + 1, false);
    
    dfs(1, 0, visited, result);
    
    for (int i = 0; i < result.size(); i++) {
        if (i > 0) cout << " ";
        cout << result[i];
    }
    cout << "\n";
    
    return 0;
}
```
