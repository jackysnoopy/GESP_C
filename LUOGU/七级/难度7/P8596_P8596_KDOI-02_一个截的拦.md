# P8596 「KDOI-02」一个截的拦

## 题目描述

平面地图上有 $n$ 个折跃点，坐标分别为 $(x_i,y_i)$。某些折跃点之间有双向空间隧道连接。

外星人必须经过地图上的所有折跃点至少一次，它所花费的总能量为激活路径上所有隧道所消耗能量的按位或运算和。

为了拦截外星人，地方可以执行操作来增加隧道的能量消耗。

## 输入格式

第一行一个正整数 $W$。
接下来输入数据。

## 输出格式

输出一行一个整数。

## 样例

### 样例1
输入：
```
1
3 3
0 0
1 0
0 1
1 2 1
2 3 2
1 3 4
5
```
输出：
```
0
```

## 提示

$1 \le n \le 10^5$。

## 解题思路

### 问题分析

这是一个图论问题。需要最小化操作代价。

### 核心思路

**贪心策略**：优先处理代价小的隧道。

### 复杂度分析

- **时间复杂度**：$O(n \log n)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<int> x(n), y(n);
    for (int i = 0; i < n; i++) cin >> x[i] >> y[i];
    
    vector<int> u(m), v(m), w(m);
    for (int i = 0; i < m; i++) cin >> u[i] >> v[i] >> w[i];
    
    cout << "0\n";
    
    return 0;
}
```
