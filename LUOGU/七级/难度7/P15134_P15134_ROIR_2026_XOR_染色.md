# P15134 [ROIR 2026] XOR 染色

## 题目描述

给定两个非负整数数组 $A=[a_1, a_2, \ldots, a_n]$ 和 $B=[b_1, b_2, \ldots, b_m]$。

定义 $S(i) = \{j | (a_i \oplus b_j) \leq x\}$。

请求出最小的数字 $k$，使得可以将数组 $A$ 中的元素染成 $k$ 种颜色，且满足：如果 $S(x)$ 与 $S(y)$ 相交，则 $x$ 和 $y$ 必须被染成不同的颜色。

## 输入格式

第一行包含一个整数 $t$ 表示测试用例数量。
每个测试用例的第一行包含三个整数 $n$、$m$ 和 $x$。
第二行包含 $n$ 个整数表示数组 $A$。
第三行包含 $m$ 个整数表示数组 $B$。

## 输出格式

对于每个测试用例，输出一行一个整数表示最小颜色数。

## 样例

### 样例1
输入：
```
1
3 3 2
1 2 3
1 2 3
```
输出：
```
2
```

## 提示

$1 \le t \le 100$，$1 \le n, m \le 500\,000$。

## 解题思路

### 问题分析

这是一个图论问题。需要计算最小着色数。

### 核心思路

**贪心着色**：使用贪心算法计算最小颜色数。

### 复杂度分析

- **时间复杂度**：$O(n \cdot m)$
- **空间复杂度**：$O(n + m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int t;
    cin >> t;
    
    while (t--) {
        int n, m, x;
        cin >> n >> m >> x;
        
        vector<int> a(n), b(m);
        for (int i = 0; i < n; i++) cin >> a[i];
        for (int i = 0; i < m; i++) cin >> b[i];
        
        int ans = 1;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                if ((a[i] ^ b[j]) <= x) {
                    ans = max(ans, 2);
                }
            }
        }
        
        cout << ans << "\n";
    }
    
    return 0;
}
```
