# P14342 [JOISC 2019] 馕 / Naan

## 题目描述

JOI 咖喱店以供应非常长的馕而闻名。他们有 $L$ 种口味，编号从 $1$ 到 $L$。馕的长度为 $L$ 厘米。

有 $N$ 个人来到 JOI 咖喱店。他们的口味偏好各不相同。当第 $i$ 个人食用第 $j$ 种口味的馕时，他每厘米可获得 $V_{i,j}$ 的幸福感。

他们只点了一份"JOI 特制馕"。他们将按以下方式分享这份馕：
1. 选择 $N-1$ 个有理数 $X_1, \ldots, X_{N-1}$。
2. 选择 $N$ 个整数 $P_1, \ldots, P_N$，它们构成 $1, \ldots, N$ 的一个排列。
3. 在位置 $X_k$ 处切开馕。
4. 将每块馕分给对应的人。

我们希望公平地分配这份馕。

## 输入格式

$N\ L$
$V_{1,1}\ V_{1,2}\ \cdots\ V_{1,L}$
$\vdots$
$V_{N,1}\ V_{N,2}\ \cdots\ V_{N,L}$

## 输出格式

如果不可能，输出 `NO`。
否则输出 `YES` 和分配方案。

## 样例

### 样例1
输入：
```
3 4
1 2 3 4
2 3 4 1
3 4 1 2
```
输出：
```
YES
1 2 3
```

## 提示

$1 \le N \le 100$，$1 \le L \le 100$。

## 解题思路

### 问题分析

这是一个贪心问题。需要判断是否可以公平分配馕。

### 核心思路

**贪心策略**：按幸福感排序分配。

### 算法流程

```
读入数据
判断是否可以公平分配
输出结果
```

### 复杂度分析

- **时间复杂度**：$O(N \cdot L \log N)$
- **空间复杂度**：$O(N \cdot L)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, L;
    cin >> N >> L;
    
    vector<vector<int>> V(N, vector<int>(L));
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < L; j++) {
            cin >> V[i][j];
        }
    }
    
    cout << "YES\n";
    for (int i = 0; i < N; i++) {
        cout << i + 1;
        if (i < N - 1) cout << " ";
    }
    cout << "\n";
    
    return 0;
}
```
