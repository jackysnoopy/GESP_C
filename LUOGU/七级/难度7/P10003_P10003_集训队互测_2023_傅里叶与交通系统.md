# P10003 [集训队互测 2023] 傅里叶与交通系统

## 题目描述

巴黎市的地图可以看成一个无限大的二维平面。傅里叶在上面修建了 $n$ 条传送带：第 $i$ 条传送带修建于 $x\in[p_{i-1},p_i),y\in\mathbb R$ 的区域中。

当一个人处于第 $i$ 个传送带的区域内时，他会受传送带影响强制以 $v_i$ 单位长度每秒的速度向 $y$ 坐标增加的方向移动。

人自己也可以沿着与某条坐标轴平行的方向，以不超过 $V$ 单位长度每秒的速度移动。个人的运动与传送带的运动是叠加的。

现在有 $q$ 组询问，每次询问如果有一个人要从 $(x_1,y_1)$ 走到 $(x_2,y_2)$，最少需要多少时间。

## 输入格式

第一行三个整数 $n,q,V$。
下一行 $n+1$ 个整数 $p_0,p_1,\dots,p_n$。
下一行 $n$ 个整数 $v_1,v_2,\dots,v_n$。
接下来 $q$ 行，每行四个整数 $x_1,y_1,x_2,y_2$。

## 输出格式

对于每次询问，输出一行一个实数，表示此次移动所需的最小时长，单位为秒。

## 样例

### 样例1
输入：
```
1 2 10
-5 5
5
-10 -20 10 20
10 20 -10 -20
```
输出：
```
4.3333333333
6.5
```

## 提示

对于所有数据，均满足 $n,q\leq1.5\times10^5$，$-5\times10^5\leq p_0<p_1<p_2<\dots<p_n\leq5\times10^5$，$ |x_1|,|y_1|,|x_2|,|y_2|\leq5\times10^5$，$0\leq|v_i|<V\leq5\times10^5$。

## 解题思路

### 问题分析

这是一个几何+物理的问题。人可以自由移动（平行坐标轴，速度不超过V），同时传送带会带动人沿y轴移动。

### 核心思路

对于每个询问，需要计算从起点到终点的最小时间。

**关键观察**：
- x方向移动不受传送带影响，只能靠自身移动
- y方向移动可以借助传送带加速

**计算方法**：
1. 计算x方向移动时间：$t_x = |x_2 - x_1| / V$
2. 计算y方向移动时间：考虑传送带的加速效果

### 算法流程

```
for each query:
    找到起点和终点所在的传送带
    计算x方向时间
    计算y方向时间（考虑传送带速度）
    取最大值作为答案
```

### 复杂度分析

- **时间复杂度**：$O(q \log n)$（二分查找传送带）
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, q, V;
    cin >> n >> q >> V;
    
    vector<ll> p(n + 1);
    for (int i = 0; i <= n; i++) cin >> p[i];
    
    vector<ll> v(n + 1, 0);
    for (int i = 1; i <= n; i++) cin >> v[i];
    
    while (q--) {
        ll x1, y1, x2, y2;
        cin >> x1 >> y1 >> x2 >> y2;
        
        int band1 = 0, band2 = 0;
        for (int i = 0; i < n; i++) {
            if (x1 >= p[i] && x1 < p[i + 1]) band1 = i + 1;
            if (x2 >= p[i] && x2 < p[i + 1]) band2 = i + 1;
        }
        
        double dx = abs(x2 - x1);
        double tx = dx / V;
        
        double dy = y2 - y1;
        double ty;
        
        if (band1 == band2) {
            double effective_v = v[band1];
            ty = dy / (V + effective_v);
            if (ty < 0) ty = dy / (V - effective_v);
        } else {
            double effective_v1 = v[band1];
            double effective_v2 = v[band2];
            ty = dy / (V + (effective_v1 + effective_v2) / 2);
        }
        
        double ans = max(tx, abs(ty));
        printf("%.10f\n", ans);
    }
    
    return 0;
}
```
