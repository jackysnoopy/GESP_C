# P11979 [KTSC 2021] 射击游戏 / gun

## 题目描述

有一款由两名玩家共同参与的在线射击游戏。游戏中，$N$ 座建筑物从左到右排列在水平地面上。

在时间 $i$（$i \geq 1$）时，两名玩家同时发射一发子弹，子弹会摧毁满足 $A_i \geq H$ 且未被摧毁的最左侧建筑物。

问题的目标是：给定 $N$ 座建筑物的高度，找到摧毁所有建筑物的最短时间 $S$，以及每个时间点两名玩家的子弹发射高度。

## 输入格式

第 $1$ 行：$N$
第 $2$ 行：$A[0] \ A[1] \ \cdots \ A[N - 1]$

## 输出格式

第 $i$ 行：函数 `min_shooting_buildings` 返回的数组的第 $i$ 个元素。

## 样例

### 样例1
输入：
```
5
3 1 4 1 5
```
输出：
```
5 5
4 4
3 3
2 2
1 1
```

## 提示

- $1 \leq N \leq 10^5$；
- $1 \leq A_i \leq N$；
- $A_i$ 互不相同。

## 解题思路

### 问题分析

这是一个模拟问题。需要模拟射击过程，找到摧毁所有建筑物的最短时间。

### 核心思路

**模拟过程**：
1. 每次选择两个高度发射子弹
2. 摧毁满足条件的建筑物
3. 重复直到所有建筑物被摧毁

### 算法流程

```
while 有建筑物未被摧毁:
    选择两个高度
    摧毁满足条件的建筑物
    记录结果
输出结果
```

### 复杂度分析

- **时间复杂度**：$O(n^2)$
- **空间复杂度**：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<pair<int, int>> min_shooting_buildings(vector<int> A) {
    int n = A.size();
    vector<pair<int, int>> result;
    
    vector<bool> destroyed(n, false);
    int remaining = n;
    
    while (remaining > 0) {
        int h1 = 1, h2 = 1;
        
        for (int i = 0; i < n; i++) {
            if (!destroyed[i]) {
                h1 = A[i];
                break;
            }
        }
        
        bool found = false;
        for (int i = 0; i < n; i++) {
            if (!destroyed[i] && A[i] != h1) {
                h2 = A[i];
                found = true;
                break;
            }
        }
        
        if (!found) {
            h2 = h1;
        }
        
        for (int i = 0; i < n; i++) {
            if (!destroyed[i] && A[i] >= min(h1, h2)) {
                destroyed[i] = true;
                remaining--;
            }
        }
        
        result.push_back({h1, h2});
    }
    
    return result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> A(n);
    for (int i = 0; i < n; i++) {
        cin >> A[i];
    }
    
    vector<pair<int, int>> result = min_shooting_buildings(A);
    
    for (auto& [h1, h2] : result) {
        cout << h1 << " " << h2 << "\n";
    }
    
    return 0;
}
```
