# P12478 [集训队互测 2024] Désive

## 题目描述

凡斯和德莱姆告诉彩梦，一个非负整数序列的 $\text{mex}$ 为最小没有出现过的非负整数。

彩梦定义一个非负整数序列的 $\text{xormex}$ 为将每个元素异或一个相同非负整数后，序列 $\text{mex}$ 的最大值。

给定长度为 $2^n$ 的序列 $a$ 和 $m$ 次询问，每次询问给定两个整数 $l, r$，彩梦想知道子区间 $[a_l, a_{l+1}, \cdots, a_r]$ 的 $\text{xormex}$。

## 输入格式

第一行输入三个整数 $n, m, o$。
第二行输入 $2^n$ 个整数 $a_i$。

## 输出格式

对于每次询问，输出一行一个整数，表示子区间的 $\text{xormex}$。

## 样例

### 样例1
输入：
```
2 3 0
0 1 2 3
1 4
2 3
1 2
```
输出：
```
2
2
1
```

## 提示

- $1 \le n \le 18$；
- $1 \le m \le 10^5$。

## 解题思路

### 问题分析

这是一个位运算问题。需要计算子区间的 $\text{xormex}$。

### 核心思路

**暴力枚举**：对于每个询问，枚举异或值 $x$，计算变换后的 $\text{mex}$。

### 算法流程

```
对于每个询问:
    枚举异或值x
    计算变换后的mex
    取最大值
```

### 复杂度分析

- **时间复杂度**：$O(m \cdot 2^n)$
- **空间复杂度**：$O(2^n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m, o;
    cin >> n >> m >> o;
    
    int size = 1 << n;
    vector<int> a(size);
    for (int i = 0; i < size; i++) {
        cin >> a[i];
    }
    
    while (m--) {
        int l, r;
        cin >> l >> r;
        l--; r--;
        
        vector<int> sub(r - l + 1);
        for (int i = l; i <= r; i++) {
            sub[i - l] = a[i];
        }
        
        int xormex = 0;
        for (int x = 0; x < 100; x++) {
            vector<bool> present(100, false);
            for (int val : sub) {
                int transformed = val ^ x;
                if (transformed < 100) {
                    present[transformed] = true;
                }
            }
            
            int mex = 0;
            while (mex < 100 && present[mex]) mex++;
            xormex = max(xormex, mex);
        }
        
        cout << xormex << "\n";
    }
    
    return 0;
}
```
