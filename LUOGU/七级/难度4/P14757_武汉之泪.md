# 武汉之泪

## 题目描述

给定两个长度为 $n$ 的单调递增的序列 $a,b$，你可以对序列 $a$ 执行如下操作任意次：

选择下标 $i\ (1\le i<n)$，满足 $a_i+1=a_{i+1}$，然后将 $a_i,a_{i+1}$ 变为 $a_i+1,a_{i+1}+1$ 或 $a_i-1,a_{i+1}-1$。

在任意时刻，你需要保证序列 $a$ 单调递增（即 $a_1<a_2<\dots<a_n$）。

问是否可以使序列 $a$ 变为序列 $b$。

## 输入格式

本题有多组数据。第一行一个正整数 $T\ (1\le T\le10^4)$，表示数据组数。

对于每组数据：

第一个正整数 $n\ (1\le n\le2\times10^5)$。表示序列 $a,b$ 的长度。

第二行 $n$ 个正整数，第 $i$ 个正整数为 $a_i\ (1\le a_1<a_2<\dots<a_n\le10^9)$。

第三行 $n$ 个正整数，第 $i$ 个正整数为 $b_i\ (1\le b_1<b_2<\dots<b_n\le10^9)$。

保证 $T$ 组数据中 $n$ 的和不超过 $2\times10^5$。

## 输出格式

对于每组数据：

输出一行一个字符串，若可以使序列 $a$ 变为序列 $b$，输出 `YES`，否则输出 `NO`（不区分大小写）。

## 样例

### 样例输入 1
```
4
3
1 2 3
1 2 5
4
1 2 5 9
3 7 8 9
3
1 2 3
1 5 6
3
2 3 4
1 2 3
```

### 样例输出 1
```
NO
YES
YES
NO
```

## 提示

难度：普及+/提高

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;

    while (T--) {
        int n;
        cin >> n;

        vector<long long> a(n), b(n);
        for (int i = 0; i < n; i++) cin >> a[i];
        for (int i = 0; i < n; i++) cin >> b[i];

        bool possible = true;
        for (int i = 0; i < n; i++) {
            if (a[i] > b[i]) {
                possible = false;
                break;
            }
            if (i > 0 && a[i] - a[i - 1] != b[i] - b[i - 1]) {
                if (a[i] - a[i - 1] < b[i] - b[i - 1]) {
                    possible = false;
                    break;
                }
            }
        }

        cout << (possible ? "YES" : "NO") << "\n";
    }

    return 0;
}
```

## 解题思路

### 问题分析
给定两个单调递增序列 $a$ 和 $b$，可对 $a$ 执行操作：选择满足 $a_i + 1 = a_{i+1}$ 的相邻对，将两者同时 $+1$ 或 $-1$。问能否将 $a$ 变为 $b$。

### 核心思路
关键观察：
- 操作只能在差值恰好为 1 的相邻元素间进行，且同时 $+1$ 或 $-1$
- 操作后差值变为 2 或 0（但 0 会破坏单调递增，故实际只能 $+1$ 使差值变大）
- 因此相邻差值只能增大或保持不变，不能减小

判断条件：
1. $a_i \leq b_i$ 对所有 $i$ 成立（只能通过 $+1$ 增大元素值）
2. 相邻差值不增大：$a_i - a_{i-1} \geq b_i - b_{i-1}$（差值只能增大，不能减小）

### 算法流程
1. 对每个位置 $i$ 检查 $a_i \leq b_i$
2. 对每对相邻位置检查 $a_i - a_{i-1} \geq b_i - b_{i-1}$
3. 两个条件同时满足则输出 YES，否则输出 NO

### 复杂度分析
- **时间复杂度**：$O(n)$，单次遍历
- **空间复杂度**：$O(n)$，存储序列
