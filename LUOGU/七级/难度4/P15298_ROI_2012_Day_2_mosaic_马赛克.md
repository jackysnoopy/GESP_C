# [ROI 2012 Day 2] mosaic 马赛克

## 题目描述

翻译来源：[loj #5459. 「ROI 2012 Day 2」马赛克](https://loj.ac/p/5459)。

ABBYY 公司的所有磁性马赛克元素均为矩形。只有当两个元素至少有一个尺寸（长度、宽度或两者）相同时，才能将它们连接在一起。磁性元素不可旋转或翻转。如果两个马赛克元素无法连接，则称它们为**不和谐对**。例如，$1 \times 2$ 和 $2 \times 3$ 是一对不和谐对，而 $2 \times 3$ 和 $1 \times 3$ 或 $2 \times 3$ 和 $2 \times 3$ 则是和谐对。
 
ABBYY 的设计师将所有马赛克元素排成一行，但未将它们连接在一起。我们将一行中连续的若干元素称为一个**集合**。他们选择了一些集合用于创建艺术装置，并需要确定每个集合中是否存在不和谐对的元素。
 
你需要编写一个程序，对于不同连续元素集合，确定构成不和谐对的元素编号，或者报告该集合中不存在这样的对。

## 输入格式

输入文件的第一行包含一个整数 $N$ $(2 \leq N \leq 100000)$，表示马赛克元素的数量。

接下来的 $N$ 行，每行包含两个整数 $A_i$ 和 $B_i$ $(1 \leq A_i, B_i \leq 10^{9}, 1 \leq i \leq N)$，分别表示第 $i$ 个马赛克元素的长度和宽度。

第 $(N + 2)$ 行包含一个整数 $K$ $(1 \leq K \leq 100000)$，表示需要检查不和谐对的集合数量。

接下来的 $K$ 行，每行包含两个整数 $N_1$ 和 $N_2$ $(1 \leq N_1 < N_2 \leq N)$，分别表示每个集合的第一个和最后一个元素的编号，需要在该集合中查找不和谐对。

## 输出格式

输出文件应包含 $K$ 行，每行包含两个以空格分隔的整数，表示对应集合中构成不和谐对的马赛克元素编号。如果存在多个解，可以输出任意一个。如果集合中不存在不和谐对，则在对应行输出 `0 0`。

## 样例

### 样例输入 1
```
4
2 2
1 2
1 3
2 3
2
2 3
2 4
```

### 样例输出 1
```
0 0
4 2
```

## 解题思路

### 问题分析
给定 N 个矩形元素，每个有长和宽。两个元素"不和谐"当且仅当它们没有任何一个尺寸相同。对 K 个区间查询，找出区间内任意一对不和谐元素。

### 核心思路
暴力枚举：对每个查询区间 [N1, N2]，双层循环枚举所有元素对 (i, j)，检查是否不和谐。不和谐条件：A[i]≠A[j] 且 A[i]≠B[j] 且 B[i]≠A[j] 且 B[i]≠B[j]。

### 算法流程
1. 读取 N 个元素的尺寸
2. 对每个查询 (N1, N2)：
   - 双重循环枚举区间内所有元素对
   - 检查四个尺寸值是否均不相同
   - 找到则输出并停止；未找到输出 "0 0"

### 复杂度分析
- 时间复杂度：O(K × L²)，其中 L 为查询区间长度，最坏情况 O(K × N²)
- 空间复杂度：O(N)，存储元素尺寸

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    vector<int> A(N), B(N);
    for (int i = 0; i < N; i++) cin >> A[i] >> B[i];

    int K;
    cin >> K;

    for (int q = 0; q < K; q++) {
        int n1, n2;
        cin >> n1 >> n2;
        n1--; n2--;

        bool found = false;
        for (int i = n1; i < n2 && !found; i++) {
            for (int j = i + 1; j <= n2 && !found; j++) {
                if (A[i] != A[j] && A[i] != B[j] && B[i] != A[j] && B[i] != B[j]) {
                    cout << i + 1 << " " << j + 1 << "\n";
                    found = true;
                }
            }
        }

        if (!found) cout << "0 0\n";
    }

    return 0;
}
```

## 解题思路

### 问题分析
给定n个矩形马赛克元素，每个有长和宽。两个元素"和谐"当且仅当至少有一个尺寸相同。需要在查询的区间内找到一对"不和谐"的元素。

### 核心思路
暴力枚举：对于每个查询区间，遍历所有元素对，检查是否四个尺寸都不同（不和谐）。找到即输出，否则输出"0 0"。

### 算法流程
1. 读入n个元素的长和宽
2. 对于每个查询[n1, n2]：
   - 枚举区间内所有元素对(i, j)
   - 若A[i]≠A[j]且A[i]≠B[j]且B[i]≠A[j]且B[i]≠B[j]，则输出这对
   - 若未找到则输出"0 0"

### 复杂度分析
- 时间复杂度：O(K × n²)，暴力枚举
- 空间复杂度：O(n)
