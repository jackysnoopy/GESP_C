# 【MX-X11-T2】「蓬莱人形 Round 1」所以我放弃了音乐

## 题目描述

原题链接：。

---

$$\text{考えたってわからないし}$$

$$\text{青空の下、君を待った}$$

$$\text{風が吹いた正午、昼下がりを抜け出す想像}$$

$$\text{ねぇ、これからどうなるんだろうね}$$

$$\text{進め方教わらないんだよ}$$

在一个无限大的棋盘上有 $n$ 个**位置互不相同**的棋子 $(x_i,y_i)$，你需要通过进行若干次以下操作删除全部的棋子：

1. 选择一个格子 $(x,y)$。

2. 若 $(x,y)$ 上有棋子，则把这个棋子删掉，否则结束当前操作。

3. **依次**检查坐标为 $(x+1,y+1)$，$(x+1,y)$，$(x+1,y-1)$ 的格子上是否有棋子。当检查到第一个有棋子的格子时，停止检查，并将当前的 $(x,y)$ 更新为该格子的坐标后返回第二步。如果这三个格子都没有棋子，结束当前操作。

你要回答，最少操作多少次能把所有棋子删光。

## 输入格式

第一行一个正整数 $n$ 表示棋盘上棋子的个数。

接下来 $n$ 行，每行两个正整数 $x_i,y_i$ 表示一个棋子的位置，**保证没有两个位置相同的棋子**。

## 输出格式

一行一个正整数，表示最少操作多少次能把所有棋子删光。


## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <map>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<pair<int, int>> pieces(n);
    for (int i = 0; i < n; i++) cin >> pieces[i].first >> pieces[i].second;

    map<int, int> diag1, diag2;
    for (auto& [x, y] : pieces) {
        diag1[x + y]++;
        diag2[x - y]++;
    }

    int ans = 0;
    for (auto& [k, v] : diag1) ans = max(ans, v);
    for (auto& [k, v] : diag2) ans = max(ans, v);

    cout << ans << "\n";
    return 0;
}
```

## 解题思路

### 问题分析

在一个无限大的棋盘上有 n 个位置互不相同的棋子。每次操作可以选择一个起始位置，若该位置有棋子则删除，然后依次检查右上(x+1,y+1)、右(x+1,y)、右下(x+1,y-1)三个位置，找到第一个有棋子的继续删除，直到找不到为止。求最少操作次数。

### 核心思路

观察操作的移动方向：每次删除棋子后，下一步只会向 x 增大的方向移动（右上、右、右下），且移动的 x 坐标恰好增加1。因此，一次操作中删除的所有棋子必须满足：
- x 坐标依次递增（相差恰好为1）
- 要么 y 坐标递增（对应 x+y 相同的对角线方向）
- 要么 y 坐标递减（对应 x-y 相同的对角线方向）

所以一次操作最多能删除同一条"主对角线"（x+y=k）或"反对角线"（x-y=k）上的所有棋子。答案就是所有对角线上棋子数量的最大值。

### 算法流程

1. 读入所有棋子的坐标
2. 对每个棋子 (x, y)，分别统计 x+y 和 x-y 的出现次数，存入两个 map
3. 遍历两个 map，取所有计数的最大值作为答案
4. 输出答案

### 复杂度分析

- 时间复杂度：O(n log n)，map 的插入和查询均为 O(log n)
- 空间复杂度：O(n)，存储所有棋子及两个 map
