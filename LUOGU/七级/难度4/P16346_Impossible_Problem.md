# Impossible Problem

## 题目描述

本题出处：https://www.luogu.com.cn/contest/314899

有一个长度为 $n$ 的序列 $a$，每次可以选择一个区间满足这个区间的数字和为奇数，然后将这个区间中的所有数加上 $1$。

求是否可以将序列中的所有数字变为相同数字。

若可以则输出 `Yes`，否则输出 `No`。

## 输入格式

本题多测，第一行输入一行一个正整数 $T$。

对于每组测试数据：

- 第一行输入一行一个正整数 $n$。
- 第二行输入一行 $n$ 个非负整数表示序列 $a$。

## 输出格式

对于每组测试数据：

- 一行一个字符串表示你的答案。

## 样例

### 样例输入 1
```
2
2
1 2
2
2 4
```

### 样例输出 1
```
Yes
No
```

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;

    while (T--) {
        int n;
        cin >> n;

        vector<long long> a(n);
        for (int i = 0; i < n; i++) cin >> a[i];

        long long sum = 0;
        for (int i = 0; i < n; i++) sum += a[i];

        if (sum % n != 0) {
            cout << "No\n";
            continue;
        }

        long long target = sum / n;
        long long diff = 0;
        bool possible = true;

        for (int i = 0; i < n; i++) {
            diff += a[i] - target;
            if (diff < 0) {
                possible = false;
                break;
            }
        }

        cout << (possible ? "Yes" : "No") << "\n";
    }

    return 0;
}
```

## 解题思路

### 问题分析
给定序列a，每次可以选择和为奇数的区间将所有数+1。判断是否能将所有数变为相同值。

### 核心思路
参考代码采用前缀和差分的思想：首先检查总和是否能被n整除（否则无法均分）。然后计算每个位置与目标值的差值累积，若累积差值出现负数则无解。

### 算法流程
1. 计算序列总和
2. 若总和%n≠0则输出"No"
3. 计算目标值target=sum/n
4. 计算前缀差分，若出现负数则无解
5. 输出"Yes"或"No"

### 复杂度分析
- 时间复杂度：O(T × n)
- 空间复杂度：O(n)

## 解题思路

### 问题分析

给定一个序列，每次操作可以选择一个和为奇数的区间，将区间内所有数加 1。判断是否能将所有数变为相同数字。

### 核心思路

关键观察：每次操作将一个和为奇数的区间所有数加 1，相当于改变了奇偶性。通过分析前缀和的性质：

- 如果所有数之和不能被 $n$ 整除，显然不可能
- 否则目标值 $target = \text{sum} / n$
- 对于每个位置，计算当前值与目标值的差的累积
- 如果累积差出现负值，说明无法通过合法操作达到目标

### 算法流程

1. 计算序列总和，如果不能被 $n$ 整除，输出 No
2. 计算目标值 $target = \text{sum} / n$
3. 从左到右遍历，维护累积差值
4. 如果任何时刻累积差值为负，说明无法完成，输出 No
5. 否则输出 Yes

### 复杂度分析

- 时间复杂度：$O(n)$，每个测试点只需一次遍历
- 空间复杂度：$O(n)$ 存储序列
