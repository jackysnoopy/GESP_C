# [ECUSTPC 2026 Spring] 烈焰风暴 2

## 题目描述

:::epigraph
你需要我的帮助？
:::

奶龙正在游戏内杀怪，这是一个回合制游戏，他遭遇了 $n$ 只怪物，它们的生命值分别为 $h_1, h_2, \dots, h_n$.

每个回合，奶龙可以选择释放下面其中一个技能一次：

- 火球术，选定一只怪物，对其造成 $x$ 点伤害，并消耗奶龙 $a$ 点法力值。
- 烈焰风暴，对所有怪物造成 $y$ 点伤害，并消耗奶龙 $b$ 点法力值。

当怪物受到伤害时会扣减等量的生命值，若其生命值小于等于 $0$, 怪物将会被击杀，当所有 $n$ 只怪物被击杀时则奶龙赢得战斗胜利。

请帮助奶龙规划技能使用，使其消耗的法力值最小并赢得战斗，并求出需要消耗的最小法力值。

## 输入格式

本题每个测试点只有一组测试数据。

第一行输入 $5$ 个整数 $n, x, y, a, b\ (1 \le n, x, y, a, b \le 2 \times 10^6)$，分别表示怪物的数量，火球术和烈焰风暴的伤害，以及这两个技能对应的法力值消耗。

随后一行输入 $n$ 个整数 $h_1, h_2, \dots, h_n\ (1 \le h_i \le 10^7)$，表示怪物的血量。

## 输出格式

输出一行一个整数，表示奶龙赢得战斗所需消耗的最小法力值。

## 样例

### 样例输入 1
```
4 10 8 5 7
9 17 23 40
```

### 样例输出 1
```
31
```

### 样例输入 2
```
5 3 1 2 5
1 2 3 4 10
```

### 样例输出 2
```
17
```

## 提示

### 样例 1 解释

此时奶龙一种可行的技能使用方案是使用 $3$ 次烈焰风暴，随后再对最后一只怪物使用 $2$ 次火球术。

这样其他怪物受到 $8 \times 3 = 24$ 点伤害，最后一只怪物受到 $24 + 10 \times 2 = 44$ 点伤害，都被击杀。

此时法力值消耗为 $3 \times 7 + 2 \times 5 = 31$ 点，可以证明这是最优的。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;

    while (T--) {
        long long x;
        int a1, a2, a3, a4;
        cin >> x >> a1 >> a2 >> a3 >> a4;

        long long best = x;

        for (int mask = 0; mask < (1 << 4); mask++) {
            int cnt[4] = {0};
            for (int i = 0; i < 4; i++) {
                if (mask & (1 << i)) cnt[i]++;
            }

            long long val = x;
            for (int i = 0; i < 4; i++) {
                for (int j = 0; j < cnt[i]; j++) {
                    if (i == 0) val = (long long)(10 * sqrt(val));
                    else if (i == 1) val = (long long)(0.7 * val + 30);
                    else if (i == 2) val = (long long)(1.2 * val);
                    else val = val + 5;
                }
            }

            best = max(best, val);
        }

        cout << best << "\n";
    }

    return 0;
}
```

## 解题思路

### 问题分析
回合制游戏，有n只怪物。火球术对单体造成x伤害消耗a法力，烈焰风暴对全体造成y伤害消耗b法力。求最小法力消耗击败所有怪物。

### 核心思路
参考代码展示了另一种问题的解法（枚举所有技能组合）。对于本题，需要枚举使用烈焰风暴的次数，计算剩余怪物需要的火球术次数，取最小总消耗。

### 算法流程
1. 枚举使用烈焰风暴的次数t
2. 计算每只怪物剩余血量：max(0, h_i - t*y)
3. 计算需要的火球术次数：ceil(剩余血量/x)
4. 总消耗=t*b + 火球次数*a
5. 取最小值

### 复杂度分析
- 时间复杂度：O(max(h_i)/y × n)
- 空间复杂度：O(n)

## 解题思路

### 问题分析

给定初始值 $x$ 和 4 种操作（每种操作可使用 0 或 1 次），每种操作对值有不同的变换效果。求所有可能操作组合下的最大最终值。

### 核心思路

代码采用暴力枚举策略：枚举 4 种操作的所有 $2^4 = 16$ 种使用组合（每种操作用或不用），对每种组合按顺序应用操作，记录最大结果。

**操作定义**：
- 操作 0：$val = 10 \times \sqrt{val}$
- 操作 1：$val = 0.7 \times val + 30$
- 操作 2：$val = 1.2 \times val$
- 操作 3：$val = val + 5$

### 算法流程

1. 读入测试数据组数 $T$。
2. 对每组数据，读入 $x$ 和 4 个操作参数。
3. 枚举所有 $2^4$ 种操作组合（用位掩码表示）。
4. 对每种组合，按顺序应用选中的操作，计算最终值。
5. 取所有组合的最大值输出。

### 复杂度分析

- 时间复杂度：$O(T \times 2^4 \times 4) = O(T)$，常数级别。
- 空间复杂度：$O(1)$，仅使用常数额外空间。
