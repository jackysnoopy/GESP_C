# [蓝桥杯 2024 国 Python A] 排练

## 题目描述

本题测试数据不保证正确。

小蓝的班上有 $n$ 名同学正在为校庆活动排练节目，他们在舞台上站成一排。若把他们看做在数轴上，从左到右第 $i$ 名同学的位置即为 $a_i$。为了美观，小蓝想在队伍中插入一些新同学，使得在新队伍中，对于任意三名相邻的同学的位置 $a_{i-1}, a_i, a_{i+1}$ 满足 $a_{i+1} - a_i \leq 2(a_i - a_{i-1})$。小蓝想知道最少增加多少名同学可以满足条件。

## 输入格式

输入的第一行包含一个正整数 $n$。

第二行包含 $n$ 个正整数 $a_1, a_2, \cdots, a_n$ ，相邻整数之间使用一个空格分隔。

## 输出格式

输出一行包含一个整数表示答案。

## 样例

### 样例输入 1
```
4
1 3 16 17
```

### 样例输出 1
```
2
```

## 提示

### 样例说明

其中一种方案：在 $6, 10$ 处插入两名同学，新队伍为 $1, 3, 6, 10, 16, 17$，满足条件。

### 评测用例规模与约定

- 对于 $20\%$ 的评测用例，$1 \leq n \leq 10^3$；
- 对于所有评测用例，$1 \leq n \leq 10^5$，$1 \leq a_i \leq 10^8$，$a_{i+1} > a_i$。

## 解题思路

### 问题分析

给定 $n$ 个递增排列的同学位置，要求在相邻三名同学中满足 $a_{i+1} - a_i \leq 2(a_i - a_{i-1})$，即后一个间距不超过前一个间距的 2 倍。求最少插入多少名同学。

### 核心思路

贪心策略：对于每对相邻位置 $(a_{i-1}, a_i)$，若间距过大不满足条件，则在中间插入新同学来缩小间距。每次插入时，新同学的位置应尽可能大（即尽可能靠近 $a_i$），使得新形成的间距恰好满足 $a_{i+1} - a_i \leq 2(a_i - a_{i-1})$ 的约束。

具体地，设当前间距为 $d = a_i - a_{i-1}$，允许的最大间距为 $2d$。若 $a_i - a_{i-1} > 2d$，则需要在中间插入点，每次插入将间距减半（或按比例缩小），直到满足条件。

### 算法流程

1. 读入 $n$ 和数组 $a$。
2. 遍历每对相邻位置 $(a_{i-1}, a_i)$：
   - 计算间距 $d = a_i - a_{i-1}$。
   - 若间距超过 $2d$（即 $a_i - a_{i-1} > 2d$），则循环插入新同学，每次将间距缩小，直到满足条件。
3. 输出总插入次数。

### 复杂度分析

- 时间复杂度：$O(n \log C)$，其中 $C$ 为最大位置值，每次插入使间距至少减半。
- 空间复杂度：$O(n)$，存储数组。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    int ans = 0;
    for (int i = 1; i < n; i++) {
        long long diff = a[i] - a[i - 1];
        long long maxAllowed = 2 * diff;
        while (a[i] - a[i - 1] > maxAllowed) {
            ans++;
        }
    }

    cout << ans << "\n";
    return 0;
}
```
