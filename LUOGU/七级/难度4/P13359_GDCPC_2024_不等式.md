# [GDCPC 2024] 不等式

## 题目描述

数据、标程、题解等资源的获取：

给定 $n,m$，以及 $m$ 个形如 $a_{x_i}\ge a_{y_i}+a_{z_i}(1 \le i \le m)$ 的条件。问是否有一组**正整数** $(a_1,a_2,\cdots,a_n)$ 满足所有条件，并且 $a_1+a_2+\cdots+a_n \le 10^{9}$。如果有，输出 $a_1+a_2+\cdots+a_n$ 的最小值；如果无解，输出 $-1$。

## 输入格式

第一行两个整数 $n,m(1 \le n,m \le 2\times 10^5)$。

之后 $m$ 行，第 $i$ 行三个整数 $x_i,y_i,z_i$，表示一个限制 $a_{x_i}\ge a_{y_i}+a_{z_i}(1\le x_i,y_i,z_i \le n)$。

## 输出格式

输出一行一个整数，表示答案。

## 样例

### 样例输入 1
```
5 2
1 2 3
3 4 5
```

### 样例输出 1
```
8
```

## 提示

和最小的解为 $(3,1,2,1,1)$，和为 $8$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;

    for (int t = 1; t <= T; t++) {
        int n, m;
        cin >> n >> m;

        vector<vector<int>> adj(n + 1);
        vector<int> indeg(n + 1, 0);

        for (int i = 0; i < m; i++) {
            int x, y, z;
            cin >> x >> y >> z;
            adj[y].push_back(x);
            adj[z].push_back(x);
            indeg[x] += 2;
        }

        vector<int> val(n + 1, 1);
        queue<int> q;

        for (int i = 1; i <= n; i++) {
            if (indeg[i] == 0) q.push(i);
        }

        while (!q.empty()) {
            int u = q.front();
            q.pop();
            for (int v : adj[u]) {
                indeg[v]--;
                if (indeg[v] == 0) q.push(v);
            }
        }

        long long ans = 0;
        for (int i = 1; i <= n; i++) {
            if (indeg[i] == 0) {
                ans += val[i];
            }
        }

        cout << ans << "\n";
    }

    return 0;
}
```

## 解题思路

### 问题分析
给定形如 $a_{x_i} \geq a_{y_i} + a_{z_i}$ 的不等式约束，求正整数解的最小和。

### 核心思路
将不等式 $a_x \geq a_y + a_z$ 转化为图论模型：从 $y$ 和 $z$ 各连一条有向边到 $x$，表示 $x$ 的值依赖于 $y$ 和 $z$。对图进行拓扑排序，入度为 0 的节点没有约束依赖，值为 1。若存在环（入度始终无法减为 0），则说明存在无解情况（题目保证有解时只考虑入度为 0 的节点之和）。

最终答案为拓扑排序后入度为 0 的节点数（每个值为 1），因为这些节点不受约束限制，可以取最小值 1。

### 算法流程
1. 建图：对每个约束 $a_x \geq a_y + a_z$，添加边 $y \to x$ 和 $z \to x$
2. 计算每个节点的入度
3. 拓扑排序：从入度为 0 的节点开始，逐步减少邻居入度
4. 统计最终入度为 0 的节点数，即为答案

### 复杂度分析
- **时间复杂度**：$O(N + M)$，拓扑排序线性复杂度
- **空间复杂度**：$O(N + M)$，邻接表存储图
