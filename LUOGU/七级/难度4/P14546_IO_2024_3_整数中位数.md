# [IO 2024 #3] 整数中位数

## 题目描述

莫图努伊岛上的居民信奉女神特菲提，她特别喜爱中位数和整数。

为了纪念她，人们发明了一个游戏：
- 选择 $n$ 个互不相同的整数，第 $i$ 个数为 $a_i$。
- 创建一个空的数字集合 $m$。
- 按照任意顺序逐个将数字添加到 $m$ 中，使得在每一步添加完当前数字后，集合 $m$ 元素的中位数始终保持为整数。

在此问题中，中位数定义如下：将集合 $m$ 的元素按升序排列。如果集合大小为奇数，则中位数等于所得序列的中间元素；如果为偶数，则中位数等于两个中间元素的平均值。

莫阿娜在出发冒险前给希米亚留下了 $n$ 个互不相同的整数。请帮助希米亚在莫阿娜回来之前完整地填满集合 $m$。

## 输入格式

第一行包含一个整数 $n$——数字的数量（$1 \le n \le 10^5$）。

第二行列出 $n$ 个不同的整数 $a_i$——为游戏选择的数字（$1 \le a_i \le 10^9$）。

## 输出格式

按照将数字添加到集合中的顺序输出数组 $a$ 中的数字，使得集合的中位数始终保持为整数。

如果无法将所有数字添加到 $m$ 中，则输出单个数字 $-1$。

## 样例

### 样例输入 1
```
5
1 2 3 4 5
```

### 样例输出 1
```
4 2 5 1 3
```

### 样例输入 2
```
2
2 9
```

### 样例输出 2
```
-1
```

## 提示

难度：普及+/提高

## 解题思路

### 问题分析

给定 $n$ 个互不相同的整数，需要找到一种添加顺序，使得每次向集合中添加一个数后，集合的中位数始终为整数。中位数的定义：奇数个元素时取中间值，偶数个元素时取两个中间值的平均值。要求输出满足条件的添加顺序。

### 核心思路

将数组排序后，采用双指针交替从两端取数的策略：当结果数组长度为偶数时取当前最大值（右指针），长度为奇数时取当前最小值（左指针）。这样构造保证了每一步添加后中位数为整数：

- 集合大小为偶数 $2k$ 时，中位数是第 $k$ 和第 $k+1$ 小元素的平均值。由于我们交替取大小，这两个元素的奇偶性之和为偶数，平均值为整数。
- 集合大小为奇数 $2k+1$ 时，中位数就是中间那个元素，天然为整数。

### 算法流程

1. 读入 $n$ 个整数 $a_i$，将其升序排序。
2. 初始化左指针 $left = 0$（指向最小值），右指针 $right = n - 1$（指向最大值）。
3. 当 $left \le right$ 时循环：
   - 若当前结果数组大小为偶数，将 $a[right]$ 加入结果，$right$ 左移。
   - 若当前结果数组大小为奇数，将 $a[left]$ 加入结果，$left$ 右移。
4. 输出结果数组。

### 复杂度分析

- **时间复杂度**：$O(n \log n)$，主要由排序决定。构造序列为 $O(n)$。
- **空间复杂度**：$O(n)$，存储排序后的数组和结果数组。

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    sort(a.begin(), a.end());

    vector<int> result;
    int left = 0, right = n - 1;

    while (left <= right) {
        if (result.size() % 2 == 0) {
            result.push_back(a[right--]);
        } else {
            result.push_back(a[left++]);
        }
    }

    for (int i = 0; i < n; i++) {
        if (i) cout << " ";
        cout << result[i];
    }
    cout << "\n";

    return 0;
}
```
