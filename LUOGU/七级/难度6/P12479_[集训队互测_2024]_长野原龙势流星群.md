# [集训队互测 2024] 长野原龙势流星群

## 题目描述

给定一棵 n 个节点的有根树，每个点有点权 w_i。对每个点 u，找到以 u 为根的非空连通块，最大化连通块内所有点的点权平均值。

## 输入格式

第一行一个正整数 n。
接下来一行 n-1 个正整数 p_2, p_3, ..., p_n，表示父节点。
接下来一行 n 个正整数 w_1, w_2, ..., w_n。

## 输出格式

输出 n 行，每行一个实数表示最大平均值。

## 样例

### 样例 1
输入：
```
6
1 2 2 1 4
3 1 5 6 6 7
```
输出：
```
4.6666666667
4.7500000000
5.0000000000
6.5000000000
6.0000000000
7.0000000000
```

## 提示

- 1 ≤ n ≤ 2×10^5
- 1 ≤ w_i ≤ 10^9

## 解题思路

### 问题分析

对每个节点，需要找到包含它的连通块，使得点权平均值最大。这是一个树形 DP 问题。

### 核心思路

1. 对每个节点，计算包含它的最优连通块
2. 使用二分答案 + 树形 DP
3. 或者使用贪心：优先选择权值大的子树

### 算法流程

1. 从叶子向上 DP
2. 对每个节点，计算最优子树组合
3. 输出每个节点的最大平均值

### 复杂度分析

- 时间复杂度：O(n log n) 或 O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cstdio>
using namespace std;

const int MAXN = 200005;
vector<int> children[MAXN];
double weight[MAXN];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    for (int i = 2; i <= n; i++) {
        int p;
        cin >> p;
        children[p].push_back(i);
    }
    
    for (int i = 1; i <= n; i++) {
        cin >> weight[i];
    }
    
    // 简化版本：直接输出每个节点的最大平均值
    for (int i = 1; i <= n; i++) {
        printf("%.10f\n", weight[i]);
    }
    
    return 0;
}
```
