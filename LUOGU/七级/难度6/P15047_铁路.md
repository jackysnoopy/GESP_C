# P15047 [UOI 2022 II Stage] 铁路

## 题目描述

8字形铁路，n盏路灯。一盏在交叉点。相邻路灯不同色。上下支路各至少2盏。求放置方案。

## 输入格式

第一行 n k g。
第二行 a数组。

## 输出格式

若无解输出-1，否则输出放置方案。

## 解题思路

### 核心思路
- 构造题
- 交叉点颜色确定后，分支形成环
- 环上相邻不同色是2-染色问题

### 复杂度
- 时间: O(n)
- 空间: O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, k, g;
    cin >> n >> k >> g;
    vector<int> colors(n);
    vector<int> count(k + 1, 0);
    for (int i = 0; i < n; i++) { cin >> colors[i]; count[colors[i]]++; }
    int maxCount = *max_element(count.begin() + 1, count.end());
    if (maxCount > n / 2) { cout << "-1\n"; return 0; }
    int x = (n - 1) / 2;
    int y = n - 1 - x;
    if (x < 2 || y < 2) { x = 2; y = n - 3; if (y < 2) { cout << "-1\n"; return 0; } }
    cout << x << " " << y << "\n";
    for (int i = 0; i < x; i++) cout << (i % 2 == 0 ? 1 : 2) << (i + 1 < x ? " " : "\n");
    cout << "3\n";
    for (int i = 0; i < y; i++) cout << (i % 2 == 0 ? 1 : 2) << (i + 1 < y ? " " : "\n");
    return 0;
}
```
