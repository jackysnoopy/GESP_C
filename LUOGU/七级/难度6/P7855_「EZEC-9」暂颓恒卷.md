# 「EZEC-9」暂颓恒卷

## 题目描述

有 n 个人坐成一横排，每人有一种状态（恒卷、恒颓、暂卷、暂颓）。通过交换暂人，使得稳定状态下的卷数最大。

## 输入格式

第一行两个数 n 与 k。
第二行 n 个数，表示第 i 个人为第 ai 种人。

## 输出格式

输出一行一个数，为交换小于等于 k 对暂人后优秀程度的最大值。

## 样例

### 样例 1
输入：
```
8 3
1 3 3 2 4 3 4 1
```
输出：
```
7
```

## 提示

对于 100% 的数据，ai∈{1,2,3,4}，a1,an∈{1,2}，4≤n≤2×10^6，0≤k≤2×10^6。

## 解题思路

### 问题分析
需要通过交换暂人来最大化稳定状态下的卷数。

### 核心思路
使用贪心算法，优先交换能增加卷数的暂人对。

### 算法流程
1. 识别恒卷、恒颓、暂卷、暂颓人
2. 贪心交换暂卷人和暂颓人
3. 模拟稳定状态变化直到收敛
4. 计算最终卷数

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 2000005;
int n, k;
int a[MAXN];

int main() {
    scanf("%d %d", &n, &k);
    for (int i = 1; i <= n; i++) scanf("%d", &a[i]);
    
    vector<int> state(n + 1);
    for (int i = 1; i <= n; i++) {
        if (a[i] == 1) state[i] = 1;
        else if (a[i] == 2) state[i] = 0;
        else state[i] = -1;
    }
    
    auto countRoll = [](vector<int>& s) {
        int cnt = 0;
        for (int i = 1; i < (int)s.size(); i++) {
            if (s[i] == 1) cnt++;
        }
        return cnt;
    };
    
    vector<int> temp_a = a;
    for (int i = 1; i <= n && k > 0; i++) {
        if (temp_a[i] == 3) {
            for (int j = i + 1; j <= n && k > 0; j++) {
                if (temp_a[j] == 4) {
                    swap(temp_a[i], temp_a[j]);
                    k--;
                    break;
                }
            }
        }
    }
    
    vector<int> final_state(n + 1);
    for (int i = 1; i <= n; i++) {
        if (temp_a[i] == 1) final_state[i] = 1;
        else if (temp_a[i] == 2) final_state[i] = 0;
        else final_state[i] = -1;
    }
    
    for (int iter = 0; iter < 100; iter++) {
        bool changed = false;
        vector<int> next_state = final_state;
        for (int i = 2; i < n; i++) {
            if (final_state[i] == -1) {
                int left = final_state[i-1];
                int right = final_state[i+1];
                if (temp_a[i] == 3) {
                    if (left == 1 || right == 1) next_state[i] = 1;
                    else next_state[i] = 0;
                } else {
                    if (left == 0 || right == 0) next_state[i] = 0;
                    else next_state[i] = 1;
                }
                if (next_state[i] != final_state[i]) changed = true;
            }
        }
        final_state = next_state;
        if (!changed) break;
    }
    
    int ans = countRoll(final_state);
    printf("%d\n", ans);
    return 0;
}
```
