# [CSP-S2019] 划分

## 题目描述
n组数据规模a_i。划分成连续段，使段和递增。最小化各段和的平方和。

## 输入格式
第一行n type。type=0直接给a_i，type=1特殊生成。

## 输出格式
最小平方和。

## 样例
输入：
```
5 0
5 1 7 9 9
```
输出：
```
247
```

## 解题思路

### 问题分析
- 贪心+斜率优化DP
- 关键观察：最优划分中，每段的和递增

### 核心思路
- 从后往前贪心，每次尽量取最长的合法段
- 使用前缀和+二分查找

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, type;
    cin >> n >> type;
    
    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    vector<long long> pre(n + 1, 0);
    for (int i = 0; i < n; i++) pre[i + 1] = pre[i] + a[i];
    
    long long ans = 0;
    long long last = 0;
    
    for (int i = n - 1; i >= 0; ) {
        long long sum = 0;
        int j = i;
        while (j >= 0 && (j == i || pre[i + 1] - pre[j + 1] <= last)) {
            sum = pre[i + 1] - pre[j];
            j--;
        }
        ans += sum * sum;
        last = sum;
        i = j;
    }
    
    cout << ans << "\n";
    return 0;
}
```
