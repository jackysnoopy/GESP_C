# P6821 [PA 2012] Tanie linie

## 题目描述

给定含 n 个数的序列，求至多 k 个不相交子段的和的最大值。

## 输入格式

第一行两个整数 n, k。第二行 n 个整数。

## 输出格式

一个整数，最大和。

## 数据范围

- 1 ≤ k ≤ n ≤ 10^6
- -10^9 ≤ a_i ≤ 10^9

## 样例

**输入：**
```
5 2
7 -3 4 -9 5
```

**输出：**
```
13
```

## 题解

### 核心思路

经典 DP 问题。定义 `dp[i][j]` 表示前 i 个数中选 j 个不相交子段的最大和。

每个数有三种选择：
1. **不选**：`dp[i][j] = dp[i-1][j]`
2. **作为新子段的开始**：`dp[i][j] = max(dp[i-1][0..j-1]) + a[i]`
3. **接在上一个子段末尾**：`dp[i][j] = dp[i-1][j] + a[i]`

### 滚动数组优化

观察到第 i 行只依赖第 i-1 行，可以用滚动数组降维。预处理 `pre[j] = max(dp[i-1][0..j-1])` 来处理转移2。

### 复杂度

- 时间：O(nk)
- 空间：O(k)
