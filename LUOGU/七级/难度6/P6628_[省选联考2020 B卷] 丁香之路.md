# [省选联考2020 B卷] 丁香之路

## 题目描述
n个顶点无向图，边权|i-j|。m条丁香花道路必须经过。从s出发，第i个朋友以i为终点。求每人最短时间。

## 输入格式
第一行n m s，接下来m行每行u v。

## 输出格式
n个答案。

## 样例
输入：
```
4 3 1
1 2
4 2
3 1
```
输出：
```
6 7 8 7
```

## 解题思路

### 问题分析
- 必须经过m条边，中国邮路问题变种
- 可以转化为最短路径问题

### 核心思路
- 预处理所有点对最短距离
- 对每个终点，计算最短路径

### 复杂度分析
- 时间复杂度：O(n^2)
- 空间复杂度：O(n^2)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 2505;
vector<int> adj[MAXN];
int dist[MAXN][MAXN];

void bfs(int s, int n) {
    vector<int> q;
    q.push_back(s);
    dist[s][s] = 0;
    int head = 0;
    while (head < (int)q.size()) {
        int u = q[head++];
        for (int v : adj[u]) {
            if (dist[s][v] == -1) {
                dist[s][v] = dist[s][u] + 1;
                q.push_back(v);
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m, s;
    cin >> n >> m >> s;
    
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            dist[i][j] = -1;
    
    for (int i = 0; i < m; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    
    for (int i = 1; i <= n; i++) bfs(i, n);
    
    for (int i = 1; i <= n; i++) {
        if (i == s) cout << "0";
        else cout << " " << dist[s][i];
    }
    cout << "\n";
    
    return 0;
}
```
