# [蓝桥杯 2025 国 A] 连锁反应

## 题目描述

数轴上放置了 n 枚炸弹，第 i 枚位置为 p_i，爆炸范围为 [p_i-l_i, p_i+r_i]。当一枚爆炸时，范围内的其他未爆炸炸弹会被引爆。求至少需要主动引爆多少枚。

## 输入格式

第一行 n。
接下来 n 行，每行三个整数 p_i, l_i, r_i。

## 输出格式

输出一个整数表示答案。

## 样例

### 样例 1
输入：
```
5
3 1 1
1 0 2
6 2 1
9 4 2
10 1 1
```
输出：
```
2
```

## 提示

- 1 ≤ n ≤ 200000
- 0 ≤ p_i, l_i, r_i ≤ 10^5

## 解题思路

### 问题分析

这是一个区间覆盖问题。每次引爆可以覆盖一个区间，需要最少次数覆盖所有点。

### 核心思路

1. 按位置排序
2. 贪心：每次选择能引爆最多炸弹的
3. 使用 BFS/DFS 模拟引爆过程

### 算法流程

1. 按位置排序
2. 从未引爆的炸弹开始，BFS 引爆所有能引爆的
3. 计数需要主动引爆的次数

### 复杂度分析

- 时间复杂度：O(n^2) 最坏，O(n) 平均
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> p(n), l(n), r(n);
    for (int i = 0; i < n; i++) {
        cin >> p[i] >> l[i] >> r[i];
    }
    
    vector<int> idx(n);
    for (int i = 0; i < n; i++) idx[i] = i;
    sort(idx.begin(), idx.end(), [&](int a, int b) {
        return p[a] < p[b];
    });
    
    int ans = 0;
    vector<bool> exploded(n, false);
    
    for (int i = 0; i < n; i++) {
        if (exploded[idx[i]]) continue;
        
        ans++;
        vector<int> queue = {idx[i]};
        exploded[idx[i]] = true;
        
        int head = 0;
        while (head < queue.size()) {
            int u = queue[head++];
            int left = p[u] - l[u];
            int right = p[u] + r[u];
            
            for (int v = 0; v < n; v++) {
                if (!exploded[idx[v]] && p[idx[v]] >= left && p[idx[v]] <= right) {
                    exploded[idx[v]] = true;
                    queue.push_back(idx[v]);
                }
            }
        }
    }
    
    cout << ans << endl;
    return 0;
}
```
