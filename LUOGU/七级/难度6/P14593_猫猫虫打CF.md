# P14593 [LNCPC 2025] 猫猫虫打 CF

## 题目描述

猫猫虫喜欢打 CF，能力值初始为 x=0。有 n 场比赛，第 i 场难度为 a_i，隐藏分为 b_i。当且仅当 x <= a_i 时参加比赛，然后 x = max(b_i, x)。可任意安排比赛顺序，求最多参加多少场。

## 输入格式

第一行一个整数 n。
接下来 n 行，每行两个整数 a_i, b_i。

## 输出格式

一个整数，最多参加比赛数量。

## 样例

### 样例 1
输入：
```
10
19 18
6 15
5 8
4 20
18 3
16 9
0 7
5 17
2 13
15 17
```
输出：
```
5
```

## 解题思路

### 问题分析
每次参加比赛后能力值变为 max(b_i, x)，所以能力值只会增加或不变。要最大化参加场数，需要贪心选择。

### 核心思路
1. 按难度 a_i 从小到大排序
2. 用优先队列维护所有当前能参加的比赛(按 b_i 排序)
3. 每次选 b_i 最大的比赛参加，更新能力值
4. 重复直到无法参加更多比赛

### 算法流程
1. 排序 contests 按 a_i
2. 维护当前能力值 ability = 0
3. 将 a_i <= ability 的比赛加入优先队列
4. 取队首(最大 b_i)参加，更新 ability = max(ability, b_i)
5. 重复步骤 3-4

### 复杂度分析
- 时间复杂度: O(n log n)
- 空间复杂度: O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<pair<long long, long long>> contests(n);
    for (int i = 0; i < n; i++) {
        cin >> contests[i].first >> contests[i].second;
    }
    
    sort(contests.begin(), contests.end());
    
    priority_queue<long long> pq;
    long long ability = 0;
    int idx = 0;
    int count = 0;
    
    while (idx < n || !pq.empty()) {
        while (idx < n && contests[idx].first <= ability) {
            pq.push(contests[idx].second);
            idx++;
        }
        
        if (pq.empty()) {
            if (idx < n) {
                ability = contests[idx].first;
                continue;
            }
            break;
        }
        
        long long best = pq.top();
        pq.pop();
        
        if (best >= ability) {
            ability = max(ability, best);
            count++;
        }
    }
    
    cout << count << "\n";
    return 0;
}
```
