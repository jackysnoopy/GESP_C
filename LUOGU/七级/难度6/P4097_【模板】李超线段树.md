# P4097 【模板】李超线段树

## 题目描述

维护一个平面直角坐标系，支持以下操作：

1. 插入一条线段，两端点横坐标分别为 $l, r$，纵坐标分别为 $k \times l + b$ 和 $k \times r + b$
2. 查询某个横坐标 $x$ 处所有线段的最大纵坐标

## 输入格式

第一行一个整数 $n$，表示操作次数。

接下来 $n$ 行，每行一个操作：

- `1 l r k b`：插入一条线段
- `2 x`：查询横坐标 $x$ 处的最大纵坐标

## 输出格式

对于每个查询操作，输出一行一个整数，表示最大纵坐标（对 $39989$ 取模）。

## 样例

输入：
```
3
1 1 3 1 2
1 2 4 -1 6
2 3
```
输出：
```
5
```

## 解题思路

### 问题分析

这是一个李超线段树模板题。需要维护线段集合，支持插入线段和查询最大值。

### 核心思路

1. 使用线段树维护区间内最优线段
2. 插入线段时，比较当前线段和区间内最优线段
3. 查询时，遍历所有覆盖查询点的区间，取最大值

### 算法流程

1. 构建线段树
2. 插入线段时，从根节点开始，比较当前线段和区间内最优线段
3. 查询时，遍历所有覆盖查询点的区间，取最大值

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

struct Line {
    int k, b;
    int id;
    Line() : k(0), b(0), id(0) {}
    Line(int k, int b, int id) : k(k), b(b), id(id) {}
    int eval(int x) const {
        return k * x + b;
    }
};

const int MOD = 39989;
const int MAXN = 40000;

Line tree[MAXN * 4];
int lastans = 0;

void update(int node, int l, int r, Line newLine) {
    int mid = (l + r) / 2;
    bool leftBetter = newLine.eval(l) > tree[node].eval(l);
    bool midBetter = newLine.eval(mid) > tree[node].eval(mid);
    
    if (midBetter) {
        swap(tree[node], newLine);
    }
    
    if (l == r) return;
    
    if (leftBetter != midBetter) {
        update(node * 2, l, mid, newLine);
    } else {
        update(node * 2 + 1, mid + 1, r, newLine);
    }
}

int query(int node, int l, int r, int x) {
    int result = tree[node].eval(x);
    if (l == r) return result;
    
    int mid = (l + r) / 2;
    if (x <= mid) {
        result = max(result, query(node * 2, l, mid, x));
    } else {
        result = max(result, query(node * 2 + 1, mid + 1, r, x));
    }
    
    return result;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    for (int i = 0; i < MAXN * 4; i++) {
        tree[i] = Line(0, -2e9, 0);
    }
    
    while (n--) {
        int op;
        cin >> op;
        
        if (op == 1) {
            int l, r, k, b;
            cin >> l >> r >> k >> b;
            l = (l + lastans - 1) % MOD + 1;
            r = (r + lastans - 1) % MOD + 1;
            if (l > r) swap(l, r);
            update(1, 1, MOD, Line(k, b, 0));
        } else {
            int x;
            cin >> x;
            x = (x + lastans - 1) % MOD + 1;
            lastans = query(1, 1, MOD, x);
            cout << lastans << "\n";
        }
    }
    
    return 0;
}
```
