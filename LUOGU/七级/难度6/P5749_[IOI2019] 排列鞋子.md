# P5749 [IOI2019] 排列鞋子

## 题目描述
有 \(2n\) 只鞋排成一行。目标：位置 \(2i\) 放左脚鞋，\(2i+1\) 放右脚鞋，且这两只鞋大小相同。每次操作可以交换相邻两只鞋。求最少交换次数。

输入：第一行 \(n\)，第二行 \(2n\) 个整数，负数表示左脚，绝对值为大小。

输出：最少交换次数。

约束：\(n \le 10^5\)。

## 输入格式
第一行一个整数 \(n\)。
第二行 \(2n\) 个整数，第 \(i\) 个整数 \(a_i\) 表示第 \(i\) 只鞋：若 \(a_i < 0\) 表示左脚，大小为 \(-a_i\)；若 \(a_i > 0\) 表示右脚，大小为 \(a_i\)。

## 输出格式
一个整数，表示最少交换次数。

## 样例
样例1：
输入：
```
2
2 1 -1 -2
```
输出：
```
4
```

样例2：
输入：
```
3
-2 2 2 -2 -2 2
```
输出：
```
1
```

## 提示
- \(1 \le n \le 10^5\)
- 所有鞋的大小在 \(1\) 到 \(n\) 之间，每种大小的左脚鞋和右脚鞋各恰好两只。

## 解题思路

### 问题分析
我们需要将当前排列通过交换相邻元素变成目标排列，目标排列中每连续两个位置是一双鞋（左脚在前，右脚在后）。由于可以自由选择如何配对和排列这些鞋，我们需要找到一种目标排列，使得从当前排列到目标排列的最小交换次数最少。

### 核心思路
最小交换次数等于将当前排列重新排列成目标排列的逆序对数。因此，我们需要构造一个目标排列，使得逆序对数最小。

构造目标排列的贪心策略：
1. 使用栈从左到右扫描当前排列，将可配对的鞋（大小相同，一左一右）立即配对，并按照配对顺序放入目标排列（左脚在前，右脚在后）。
2. 配对完成后，目标排列中的每双鞋连续且左脚在前。
3. 为当前排列中的每只鞋分配目标位置：按照当前排列的顺序，依次匹配目标排列中最早出现的未匹配的相同鞋。
4. 计算分配的目标位置序列的逆序对数，即为答案。

### 算法流程
1. 读入数据。
2. 使用栈配对，构建目标排列序列 `target_seq`。
3. 对于每种鞋，维护一个队列记录其在目标排列中的位置。
4. 按照当前排列的顺序，为每只鞋分配目标位置，得到 `target_pos` 数组。
5. 使用树状数组计算 `target_pos` 的逆序对数。
6. 输出逆序对数。

### 复杂度分析
- 时间复杂度：\(O(n \log n)\)，主要消耗在树状数组计算逆序对。
- 空间复杂度：\(O(n)\)。

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <stack>
#include <map>
#include <queue>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> a(2*n);
    for (int i = 0; i < 2*n; i++) {
        cin >> a[i];
    }
    vector<int> target_seq;
    stack<int> st;
    for (int i = 0; i < 2*n; i++) {
        int x = a[i];
        if (x < 0) {
            if (!st.empty() && st.top() > 0 && -x == st.top()) {
                int right = st.top();
                st.pop();
                target_seq.push_back(x);
                target_seq.push_back(right);
            } else {
                st.push(x);
            }
        } else {
            if (!st.empty() && st.top() < 0 && x == -st.top()) {
                int left = st.top();
                st.pop();
                target_seq.push_back(left);
                target_seq.push_back(x);
            } else {
                st.push(x);
            }
        }
    }
    map<int, queue<int>> pos_queue;
    for (int i = 0; i < 2*n; i++) {
        pos_queue[target_seq[i]].push(i);
    }
    vector<int> target_pos(2*n);
    for (int i = 0; i < 2*n; i++) {
        int x = a[i];
        target_pos[i] = pos_queue[x].front();
        pos_queue[x].pop();
    }
    vector<int> bit(2*n+1, 0);
    auto update = [&](int idx, int delta) {
        idx++;
        while (idx < (int)bit.size()) {
            bit[idx] += delta;
            idx += idx & -idx;
        }
    };
    auto query = [&](int idx) {
        idx++;
        int sum = 0;
        while (idx > 0) {
            sum += bit[idx];
            idx -= idx & -idx;
        }
        return sum;
    };
    long long inversions = 0;
    for (int i = 0; i < 2*n; i++) {
        int t = target_pos[i];
        inversions += i - query(t);
        update(t, 1);
    }
    cout << inversions << "\n";
    return 0;
}
```
