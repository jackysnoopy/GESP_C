# P14394 [JOISC 2016] 俄罗斯套娃 / Matryoshka

## 题目描述

N个套娃，每个有直径R_i和高度H_i。套娃可以收纳直径和高度都更小的套娃。给定Q组查询(A,B)，求直径>=A且高度<=B的套娃中，最少有多少个没被套。

## 输入格式

第一行 N Q。
接下来 N 行：R_i H_i。
接下来 Q 行：A_j B_j。

## 输出格式

Q 行，每行一个整数。

## 样例

输入：
```
7 3
9 5
3 7
10 6
5 10
2 6
10 10
4 1
10 5
3 5
3 9
```
输出：
```
0
1
2
```

## 解题思路

### 核心思路
- 按直径从大到小排序套娃和查询
- 用树状数组维护高度维度的LIS
- 每次加入满足直径条件的套娃，查询高度<=B的最长递增子序列

### 复杂度
- 时间: O((N+Q) log N)
- 空间: O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstring>
using namespace std;

const int MAXN = 200005;
int bit[MAXN];

void update(int x, int val) {
    for (; x < MAXN; x += x & (-x))
        bit[x] = max(bit[x], val);
}

int query(int x) {
    int res = 0;
    for (; x > 0; x -= x & (-x))
        res = max(res, bit[x]);
    return res;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, q;
    cin >> n >> q;
    
    vector<pair<int, int>> dolls(n);
    for (int i = 0; i < n; i++) cin >> dolls[i].first >> dolls[i].second;
    
    vector<pair<int, int>> queries(q);
    for (int i = 0; i < q; i++) { cin >> queries[i].first >> queries[i].second; queries[i].second = i; }
    
    vector<int> dollOrder(n), queryOrder(q);
    for (int i = 0; i < n; i++) dollOrder[i] = i;
    for (int i = 0; i < q; i++) queryOrder[i] = i;
    
    sort(dollOrder.begin(), dollOrder.end(), [&](int i, int j) {
        return dolls[i].first != dolls[j].first ? dolls[i].first > dolls[j].first : dolls[i].second < dolls[j].second;
    });
    sort(queryOrder.begin(), queryOrder.end(), [&](int i, int j) { return queries[i].first > queries[j].first; });
    
    memset(bit, 0, sizeof(bit));
    vector<int> ans(q);
    int ptr = 0;
    
    for (int qi = 0; qi < q; qi++) {
        int idx = queryOrder[qi];
        int A = queries[idx].first, B = queries[idx].second;
        while (ptr < n && dolls[dollOrder[ptr]].first >= A) {
            int d = dollOrder[ptr];
            if (dolls[d].second <= B) {
                int lis = query(dolls[d].second - 1) + 1;
                update(dolls[d].second, lis);
            }
            ptr++;
        }
        ans[idx] = query(B);
    }
    
    for (int i = 0; i < q; i++) cout << ans[i] << "\n";
    return 0;
}
```
