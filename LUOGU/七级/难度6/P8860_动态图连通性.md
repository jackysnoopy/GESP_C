# 动态图连通性

## 题目描述
给定一张 $n$ 点 $m$ 边的有向图，初始时存在一条从 $1$ 到 $n$ 的路径。

你需要处理 $q$ 组询问，每组询问给定一个 $[1,m]$ 中的正整数 $x$，如果原图中的第 $x$ 条边仍存在且当前的图中删去原图中的第 $x$ 条边后仍有一条从 $1$ 到 $n$ 的路径，则删除原图中的第 $x$ 条边。

你需要报告每组询问中是否删去了第 $x$ 条边。

## 输入格式
输入第一行三个正整数 $n,m,q$，分别表示有向图的点数，边数以及询问个数。

接下来 $m$ 行，第 $i$ 行两个正整数 $u_i,v_i$，表示第 $i$ 条边由 $u_i$ 连向 $v_i$。

接下来 $q$ 行，每行一个正整数 $x$，具体含义同题目描述。

## 输出格式
输出共 $q$ 行，每行一个正整数 $0$ 或 $1$。

## 样例
### 样例1
输入:
```
5 6 5
1 2
2 3
3 5
2 4
4 5
5 1
1
2
3
4
5
```
输出:
```
0
1
1
0
0
```

## 提示
- 时间限制：1000ms
- 内存限制：262144KB

## 解题思路

### 问题分析
需要动态维护图的连通性，支持边的删除操作。

### 核心思路
1. 使用并查集维护连通性
2. 对于每个询问，检查删除边后是否仍连通

### 算法流程
1. 初始化边的存在状态
2. 对于每个询问：
   - 检查边是否存在
   - 临时删除边，检查连通性
   - 根据结果决定是否永久删除

### 复杂度分析
- 时间复杂度：$O(q \cdot n)$
- 空间复杂度：$O(n + m)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 200005;
vector<int> adj[MAXN];
bool edge_exists[MAXN];
int n, m, q;

bool has_path(int u, int target) {
    if (u == target) return true;
    
    for (int v : adj[u]) {
        if (edge_exists[v] && has_path(v, target)) {
            return true;
        }
    }
    return false;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n >> m >> q;
    
    vector<pair<int, int>> edges(m + 1);
    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        edges[i] = {u, v};
        adj[u].push_back(i);
        edge_exists[i] = true;
    }
    
    for (int i = 0; i < q; i++) {
        int x;
        cin >> x;
        
        if (!edge_exists[x]) {
            cout << 0 << "\n";
            continue;
        }
        
        edge_exists[x] = false;
        
        if (has_path(1, n)) {
            cout << 1 << "\n";
        } else {
            edge_exists[x] = true;
            cout << 0 << "\n";
        }
    }
    
    return 0;
}
```
