# P15245 [WC2026] 二进制

## 题目描述

给定x,y，操作: x*=2, y*=2, x+=1, y+=1。求使x=y的最少操作次数。

## 输入格式

多组数据。每组：x y。

## 输出格式

每组一行：最少操作次数。

## 解题思路

### 核心思路
- 从高位到低位处理
- 贪心：每次除以2或减1

### 复杂度
- 时间: O(log(max(x,y)))
- 空间: O(1)

## 参考代码

```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    long long x, y;
    cin >> x >> y;
    long long ans = 0;
    while (x != y) {
        if (x > y) {
            if (x % 2 == 0) { x /= 2; ans++; }
            else { x--; ans++; }
        } else {
            if (y % 2 == 0) { y /= 2; ans++; }
            else { y--; ans++; }
        }
    }
    cout << ans << "\n";
    return 0;
}
```
