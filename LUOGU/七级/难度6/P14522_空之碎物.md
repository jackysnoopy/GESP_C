# P14522 【MX-S11-T3】空之碎物

## 题目描述

定义x⊙y为不退位减法（即异或）。多重集S，每次合并两个数为x⊙y或y⊙x，求最终剩一个数的最大值。给定序列a，求所有区间[l,r]的f(l,r)之和。

## 输入格式

第一行 n。
第二行 a数组。

## 输出格式

一个整数，答案对998244353取模。

## 解题思路

### 核心思路
- x⊙y = x XOR y
- 问题转化为求区间内数的线性基最大异或值
- 暴力O(n^2 * 25)

### 复杂度
- 时间: O(n^2 * 25)
- 空间: O(25)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<long long> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    long long MOD = 998244353;
    long long result = 0;
    for (int l = 0; l < n; l++) {
        vector<long long> basis(25, 0);
        for (int r = l; r < n; r++) {
            long long x = a[r];
            for (int b = 24; b >= 0; b--) {
                if ((x >> b) & 1) {
                    if (basis[b] == 0) { basis[b] = x; break; }
                    x ^= basis[b];
                }
            }
            long long maxXor = 0;
            for (int b = 24; b >= 0; b--) maxXor = max(maxXor, maxXor ^ basis[b]);
            result = (result + maxXor) % MOD;
        }
    }
    cout << result << "\n";
    return 0;
}
```
