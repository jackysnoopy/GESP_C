# [HNOI/AHOI2018] 排列

## 题目描述
给定a_1..a_n和w_1..w_n。合法排列满足：若p_k = a_{p_j}则k < j。权值 = Σ i×w_{p_i}。求最大权值或-1。

## 输入格式
第一行n，第二行a_1..a_n，第三行w_1..w_n。

## 输出格式
一个整数。

## 样例
输入：
```
3 
0 1 1 
5 7 3 
```
输出：
```
32
```

## 提示
n ≤ 5×10^5, 0 ≤ a_i ≤ n, 1 ≤ w_i ≤ 10^9。

## 解题思路

### 问题分析
构建依赖图：i依赖a[i]（若a[i]>0）。先拓扑排序检查合法性，再贪心分配位置。

### 核心思路
1. 拓扑排序确定节点先后顺序
2. 按权重降序排列，将高权重节点分配到最后可用位置
3. 使用BIT追踪可用位置

### 算法流程
1. 建图并拓扑排序
2. 计算每个节点的拓扑排名
3. 按权重降序，为每个节点分配最靠后的合法位置
4. 累加位置×权重

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
