# 序列

## 题目描述
N个点的有向图，按顺序加入E条不同的边。每次加入后记录SCC个数得到序列B。两个边序列本质相同当且仅当B相同。求本质不同的边序列数。

## 输入格式
一行整数N。

## 输出格式
一行N×(N-1)个整数，第i个数表示E=i时的答案。

## 样例
输入：
```
3
```
输出：
```
1 2 4 7 7 7
```

## 提示
N ≤ 400。

## 解题思路

### 问题分析
本质不同的边序列数取决于SCC个数序列B的不同取值。

### 核心思路
组合计数：逐步加边，跟踪SCC结构变化。每次加边可能合并SCC或不改变。

### 算法流程
1. 枚举所有可能的SCC结构
2. 计数产生不同B序列的边序列数
3. 使用DP/组合公式

### 复杂度分析
- 时间：O(N³) 或更高
- 空间：O(N²)
