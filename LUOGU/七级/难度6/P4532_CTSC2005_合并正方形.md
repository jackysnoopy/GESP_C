# [CTSC2005] 合并正方形

## 题目描述
二维平面上有A类点（X轴上）和B类点（任意位置，有权值）。初始在相邻A类点间连45度正方形。每次合并相邻正方形，合并费用取决于B类点在各区域的权值和。求最大K使前K个A类点的最小合并费用≤L。

## 输入格式
第一行M和L，接下来M行每行一个点的命令。

## 输出格式
一个整数K_max。

## 样例
输入：
```
8 30.0
A -2
A 0
B 7 8 5.0
B 4 -3 2.0
B -3 4 1.0
A 2
B -4 5 1.0
A 4
```
输出：
```
3
```

## 提示
A类点≤30000, M≤100000。

## 解题思路

### 问题分析
合并正方形的费用与B类点的位置和权值相关。需要找到最优合并顺序使总费用最小。

### 核心思路
DP预处理合并费用，使用二分答案+DP验证。费用计算涉及B类点在合并后正方形的5个区域中的权值和。

### 算法流程
1. 预处理每对正方形合并的费用
2. 使用区间DP求最小合并费用
3. 二分K验证f(K)≤L

### 复杂度分析
- 时间：O(K^3) 或优化后更低
- 空间：O(K^2)
