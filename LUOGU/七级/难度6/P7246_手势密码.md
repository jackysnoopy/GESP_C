# 手势密码

## 题目描述

有一棵带点权的树。定义一次操作为选择树上的一条简单路径，并将这条简单路径上的所有点点权减去 1。问至少需要多少次操作，使树上所有点的点权恰好变为 0。

## 输入格式

第一行一个整数 op，表示数据输入方式类型。

如果 op=1：
- 第二行一个正整数 n，表示结点个数
- 第三行 n 个非负整数 a1,a2,...,an，表示每个结点的点权
- 接下来 n-1 行，每行两个正整数 u,v，表示一条连接结点 u 和结点 v 的树边

如果 op=2：
- 第二行两个整数 seed 和 n，表示生成器的种子与结点个数

## 输出格式

一行一个整数，表示最少的操作次数。

## 样例

### 样例 1
输入：
```
1
4
1 2 1 2
1 2
2 3
2 4
```
输出：
```
2
```

### 样例 2
输入：
```
1
8
1 4 2 8 5 7 8 1
1 2
2 3
2 4
2 5
1 6
6 7
6 8
```
输出：
```
19
```

## 提示

对于 100% 的数据，op∈{1,2},1≤seed≤10^9,1≤n≤3×10^6,0≤ai≤10^9,1≤u,v≤n，保证 u≠v。

## 解题思路

### 问题分析
题目要求用最少的操作次数将树上所有点的点权变为0，每次操作可以选择一条简单路径并将路径上所有点权减1。

### 核心思路
从叶子节点向上贪心处理。对于每个节点，如果它的点权非零，那么必须通过经过该节点的路径来消除这个点权。最优策略是将这个点权传递给父节点，由父节点继续处理。

### 算法流程
1. 按节点编号从大到小排序（保证先处理叶子节点）
2. 对于每个非根节点，如果其点权非零，将操作次数加上点权的绝对值
3. 将点权传递给父节点
4. 最后处理根节点的剩余点权

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 3000005;
int a[MAXN], fa[MAXN];
long long ans = 0;
int op, n, seed;

int Rand(int x) {
    seed = (1ll * seed * 0x66CCF + 19260817ll) % x + 1;
    seed = (1ll * seed * 0x77CCF + 20060428ll) % x + 1;
    seed = (1ll * seed * 0x88CCF + 12345678ll) % x + 1;
    seed = (1ll * seed * 0x33CCCCFF + 10086001ll) % x + 1;
    return seed;
}

void solve() {
    if (op == 1) {
        scanf("%d", &n);
        for (int i = 1; i <= n; i++) scanf("%d", &a[i]);
        for (int i = 1; i < n; i++) {
            int u, v;
            scanf("%d %d", &u, &v);
            fa[v] = u;
        }
    } else {
        scanf("%d %d", &seed, &n);
        for (int i = 1; i <= n; i++) a[i] = Rand(1000000000);
        for (int i = 2; i <= n; i++) fa[i] = Rand(i - 1);
    }
    
    vector<int> order(n);
    for (int i = 0; i < n; i++) order[i] = i + 1;
    sort(order.begin(), order.end(), [](int x, int y) { return x > y; });
    
    for (int u : order) {
        if (u == 1) continue;
        if (a[u] > 0) {
            ans += a[u];
            a[fa[u]] += a[u];
        } else if (a[u] < 0) {
            ans -= a[u];
            a[fa[u]] += a[u];
        }
    }
    ans += abs(a[1]);
    printf("%lld\n", ans);
}

int main() {
    scanf("%d", &op);
    solve();
    return 0;
}
```
