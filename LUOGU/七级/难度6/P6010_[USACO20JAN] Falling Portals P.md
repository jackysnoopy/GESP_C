# P6010 [USACO20JAN] Falling Portals P

## 题目描述

有 N 个世界，世界 i 初始位于 (i, A_i)，并以速度 i 沿 y 轴负方向移动（即 y 坐标随时间减小）。当两个世界的 y 坐标相同时，传送门同步，奶牛可以瞬间从一个世界传送到另一个世界。

奶牛 i 想从世界 i 前往世界 Q_i。求每头奶牛所需的最短时间（以分数 a/b 形式输出，若无法到达则输出 -1）。

## 输入格式

第一行一个整数 N。
第二行 N 个整数 A_1, A_2, ..., A_N。
第三行 N 个整数 Q_1, Q_2, ..., Q_N。

## 输出格式

N 行，每行一个分数 a/b 或 -1。

## 样例

### 样例输入
```
4
3 5 10 2
3 3 2 1
```

### 样例输出
```
7/2
7/2
5/1
-1
```

## 提示

- 2 ≤ N ≤ 2×10^5
- 1 ≤ A_i ≤ 10^9
- 1 ≤ Q_i ≤ N

## 解题思路

### 问题分析

世界 i 和世界 j 的同步时间为 t_{ij} = (A_i - A_j) / (i - j)。若 t_{ij} ≥ 0，则两者在非负时间相遇，奶牛可在此时传送。

奶牛需要从 i 经过一系列传送到达 Q_i。传送必须按时间非递减顺序进行（即每次传送时间不小于前一次）。

### 核心思路

将每个世界视为图中的节点，若 t_{ij} ≥ 0 则在 i 和 j 之间添加一条边，边权为 t_{ij}。问题转化为：对于每个 i，求从 i 到 Q_i 的最短路径，其中路径的"时间"定义为路径上最大边权（因为必须按顺序传送，最后传送的时间决定总时间）。

使用 Dijkstra 算法求解：对于每个奶牛 i，以世界 i 为起点（初始时间 0），通过优先队列扩展，每次从当前世界 u 跳到世界 v 时，要求同步时间 t_{uv} ≥ 当前时间（保证时间非递减），并以 t_{uv} 作为到达 v 的时间。当到达 Q_i 时即得到答案。

### 算法流程

1. 对于每头奶牛 i：
   - 若 i == Q_i，直接输出 0/1。
   - 否则以 i 为起点运行 Dijkstra，使用优先队列按时间排序。
   - 对于当前世界 u（时间为 t），枚举所有其他世界 v，计算同步时间 t_{uv}。
   - 若 t_{uv} ≥ t 且 t_{uv} 小于已知到达 v 的时间，则更新并入队。
   - 当首次弹出 Q_i 时即为最短时间。

### 复杂度分析

- 每头奶牛的 Dijkstra 需要枚举所有其他世界，单次 O(N log N)。
- 总复杂度 O(N^2 log N)，在 N ≤ 2×10^5 时不可行，仅适用于小规模数据。
