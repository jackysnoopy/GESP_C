# [eJOI 2021] Waterfront

## 题目描述
N丛灌木，每丛每天生长dailyGrowth_i。每天剪枝k次，每次减x。求M天后最高灌木高度的最小值。

## 输入格式
第一行N M k x，接下来N行每行height_i dailyGrowth_i。

## 输出格式
一个非负整数。

## 样例
### 样例1
输入：
```
4 3 4 3
2 5
3 2
0 4
2 8
```
输出：
```
8
```

## 提示
1 ≤ N,M ≤ 10^4

## 解题思路

### 问题分析
二分答案，检查M天内能否将所有灌木剪到限制以下。

### 核心思路
二分最终高度limit，计算每丛需要剪多少次，检查总次数≤M*k。

### 算法流程
1. 二分答案
2. 对每丛计算需要剪的次数
3. 检查总次数是否足够

### 复杂度分析
- 时间复杂度：O(N log H)
- 空间复杂度：O(N)
