# P14404 [JOISC 2016] 最差的记者 2 / Worst Reporter 2

## 题目描述

N名选手，2小时后排名(A_i,B_i)，结束时排名(C_i,D_i)。修改尽可能少的A_i,C_i使得存在排列x满足A_i=C_{x_i}且B_i<=D_{x_i}。

## 输入格式

第一行 N。
接下来 N 行：A_i B_i。
接下来 N 行：C_i D_i。

## 输出格式

一个整数，最少修改数。

## 解题思路

### 核心思路
- 按分数从大到小排序
- 贪心匹配同国家选手
- 用multiset维护可匹配的选手

### 复杂度
- 时间: O(N log N)
- 空间: O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <set>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<pair<int, int>> a(n), c(n);
    for (int i = 0; i < n; i++) cin >> a[i].first >> a[i].second;
    for (int i = 0; i < n; i++) cin >> c[i].first >> c[i].second;
    sort(a.begin(), a.end(), [](const pair<int,int>& x, const pair<int,int>& y) { return x.second > y.second; });
    sort(c.begin(), c.end(), [](const pair<int,int>& x, const pair<int,int>& y) { return x.second > y.second; });
    vector<multiset<int>> cScores(n + 1);
    for (int i = 0; i < n; i++) cScores[c[i].first].insert(c[i].second);
    vector<int> order(n);
    for (int i = 0; i < n; i++) order[i] = i;
    sort(order.begin(), order.end(), [&](int i, int j) { return a[i].second > a[j].second; });
    int ans = 0;
    for (int i : order) {
        int country = a[i].first;
        int score = a[i].second;
        auto it = cScores[country].lower_bound(score);
        if (it != cScores[country].end()) {
            ans++;
            cScores[country].erase(it);
        }
    }
    cout << n - ans << "\n";
    return 0;
}
```
