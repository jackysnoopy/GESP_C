# [Wdoi-3] 夜雀 treating

## 题目描述
2n+1种食材排成一排，每次选中间食材加入最终食材并删除一个。求最终n+1种食材中最大连续评分长度。

## 输入格式
第一行n，第二行2n+1个评分。

## 输出格式
一个整数，最大满意度。

## 样例
### 样例1
输入：
```
3
4 7 3 6 1 2 5
```
输出：
```
3
```

## 提示
1 ≤ n ≤ 2×10^5

## 解题思路

### 问题分析
每次选中间删一个，选n+1个食材，最大化最长连续评分。

### 核心思路
二分答案+贪心检查：对于给定长度L，检查是否存在连续L个评分可以被选中。

### 算法流程
1. 二分最长连续长度L
2. 对每个可能的连续评分区间，检查是否可以选中L个
3. 检查条件：选中位置在原序列中"密集"

### 复杂度分析
- 时间复杂度：O(n^2 log n)
- 空间复杂度：O(n)
