# [NOI2003] 逃学的小孩

## 题目描述
树上有N个居住点，Chris在C点，父母先去A和B中距C较近的一个，找不到再去另一个。求最坏情况下找到Chris的时间。

## 输入格式
第一行N和M，接下来M行每行U, V, T。

## 输出格式
一个整数T。

## 样例
输入：
```
4 3
1 2 1
2 3 1
3 4 1
```
输出：
```
4
```

## 提示
N ≤ 2×10^5, T_i ≤ 10^9。

## 解题思路

### 问题分析
最坏情况是A和B选为树的直径两端，C选为使min(d(C,A),d(C,B))最大的节点。

### 核心思路
1. 求树的直径D
2. 答案 = D + max_C min(d(C,A), d(C,B))，其中A,B为直径端点
3. max_C min(d(C,A), d(C,B))即为直径的"半径"

### 算法流程
1. 两次DFS求直径端点u, v和直径长度D
2. 分别以u, v为根求各节点距离
3. 答案 = D + max_i min(du[i], dv[i])

### 复杂度分析
- 时间：O(N)
- 空间：O(N)
