# 【模板】最小树形图

## 题目描述
给定n个节点m条有向边的图，求以r为根的最小树形图（最小权值和），不存在则输出-1。

## 输入格式
第一行n,m,r，接下来m行每行u,v,w。

## 输出格式
一个整数。

## 样例
输入：
```
4 6 1
1 2 3
1 3 1
4 1 2
4 2 2
3 2 1
3 4 1
```
输出：
```
3
```

## 提示
n ≤ 100, m ≤ 10^4, w ≤ 10^6。

## 解题思路

### 问题分析
最小树形图即有向图的最小生成树（以指定节点为根）。

### 核心思路
Edmonds'算法（朱刘算法）：迭代找最小入边→找环→缩环→调整边权，直到无环。

### 算法流程
1. 对每个非根节点选最小入边
2. 检查是否有环
3. 若有环，缩环并调整边权
4. 重复直到无环，累加权值

### 复杂度分析
- 时间：O(V × E)
- 空间：O(V + E)
