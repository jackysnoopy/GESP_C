# [NOIP 2018 提高组] 旅行

## 题目描述
n个城市m条路（m=n或m=n-1）。从起点出发，每次走向未访问城市或沿原路后退。求字典序最小的访问序列。

## 输入格式
第一行n,m，接下来m行每行u,v。

## 输出格式
一行n个整数。

## 样例
输入：
```
6 5 
1 3 
2 3 
2 5 
3 4 
4 6
```
输出：
```
1 3 2 5 4 6
```

## 提示
n ≤ 5000, m = n-1 或 m = n。

## 解题思路

### 问题分析
树（m=n-1）直接DFS贪心。基环树（m=n）需要枚举环上断哪条边。

### 核心思路
1. 树：DFS从1开始，每次选最小未访问邻居
2. 基环树：找到环，枚举断边，取字典序最小结果

### 算法流程
1. 排序邻接表
2. DFS贪心访问
3. 若m=n，找环并枚举断边

### 复杂度分析
- 时间：O(n²)（基环树枚举断边）
- 空间：O(n+m)
