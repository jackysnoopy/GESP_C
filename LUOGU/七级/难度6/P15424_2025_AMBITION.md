# P15424 2025 AMBITION

## 题目描述

Y形路口，左右两条车道。每辆车左转或右转。不能同时占用同一单元。求最小总时间。

## 输入格式

多组数据。每组：n m，左车道字符串，右车道字符串。

## 输出格式

每组一行：最小时间。

## 解题思路

### 核心思路
- 贪心调度
- 冲突检测：左车道R和右车道L冲突

### 复杂度
- 时间: O(n + m)
- 空间: O(n + m)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n, m;
        cin >> n >> m;
        string left, right;
        cin >> left >> right;
        int time = 0;
        int li = 0, ri = 0;
        while (li < n || ri < m) {
            time++;
            bool leftCanGo = (li < n);
            bool rightCanGo = (ri < m);
            if (leftCanGo && rightCanGo) {
                if (left[li] == 'R' && right[ri] == 'L') { li++; }
                else { li++; ri++; }
            } else if (leftCanGo) li++;
            else ri++;
        }
        time++;
        cout << time << "\n";
    }
    return 0;
}
```
