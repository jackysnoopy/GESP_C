# 「KDOI-10」水杯降温

## 题目描述

小 S 有一棵包含 n 个节点的有根树，节点 i 上放置了一个初始水温为 ai 的水杯。可以进行两种操作：使用加热装置使子树内所有水杯水温增加1，或从叶子节点向根方向吹风使路径上所有水杯水温减少1。判断能否将所有节点的水杯水温都变为0。

## 输入格式

第一行包含一个正整数 c，表示测试点编号。
第二行包含一个正整数 t，表示测试数据组数。
对于每组测试数据：
- 第一行包含一个正整数 n。
- 第二行 n-1 个正整数 f2,...,fn。
- 第三行 n 个整数 ai。

## 输出格式

对于每组测试数据，输出 Huoyu 或 Shuiniao。

## 提示

对于 100% 的数据，1≤t≤1000，2≤n≤10^5，Σn≤10^6。

## 解题思路

### 问题分析
需要判断能否通过操作将所有水温变为0。

### 核心思路
计算每个子树的水温总和，如果所有子树水温总和都非负，则可行。

### 算法流程
1. 构建树结构
2. DFS计算每个子树的水温总和
3. 检查是否所有子树水温总和都非负

### 复杂度分析
- 时间复杂度：O(Σn)
- 空间复杂度：O(Σn)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 100005;
int n;
int father[MAXN];
long long a[MAXN];
vector<int> children[MAXN];
long long subtree_sum[MAXN];

void dfs(int u) {
    subtree_sum[u] = a[u];
    for (int v : children[u]) {
        dfs(v);
        subtree_sum[u] += subtree_sum[v];
    }
}

int main() {
    int c, t;
    scanf("%d %d", &c, &t);
    
    while (t--) {
        scanf("%d", &n);
        
        for (int i = 1; i <= n; i++) {
            children[i].clear();
        }
        
        for (int i = 2; i <= n; i++) {
            scanf("%d", &father[i]);
            children[father[i]].push_back(i);
        }
        
        for (int i = 1; i <= n; i++) {
            scanf("%lld", &a[i]);
        }
        
        dfs(1);
        
        bool possible = true;
        for (int i = 1; i <= n; i++) {
            if (subtree_sum[i] < 0) {
                possible = false;
                break;
            }
        }
        
        if (possible) {
            printf("Huoyu\n");
        } else {
            printf("Shuiniao\n");
        }
    }
    
    return 0;
}
```
