# [JXOI2018] 排序问题

## 题目描述
序列x长度n，添加m个[l,r]内的整数。最大化Gobo sort期望执行轮数。输出对998244353取模。

## 输入格式
第一行T，每组两行：n,m,l,r和x_1..x_n。

## 输出格式
T个整数。

## 样例
输入：
```
2
3 3 1 2
1 3 4
3 3 5 7
1 3 4
```
输出：
```
180
720
```

## 提示
T≤10^5, n≤2×10^5, m≤10^7, ∑n≤2×10^6。

## 解题思路

### 问题分析
期望轮数 = (n+m)! / ∏(count_v!)。要最大化，需最小化∏(count_v!)。

### 核心思路
将m个新值均匀分配到[l,r]中的各值上，使各值出现次数尽可能相等。

### 算法流程
1. 预处理阶乘和逆元
2. 统计[l,r]内已有元素的频率
3. 将m个新值均匀分配：base = total/distinct, rem = total%distinct
4. 计算(n+m)! / ∏(count_v!) mod 998244353

### 复杂度分析
- 时间：O(MAXN + T×n)
- 空间：O(MAXN)
