# P5569 [SDOI2008] 石子合并

## 题目描述

一排 N 堆石子，每次合并相邻两堆，得分为新堆石子数。求最小总得分。

## 输入格式

第一行 N，接下来 N 行每行一个 a_i。

## 输出格式

最小得分。

## 样例

**输入：**

```
4
1
1
1
1
```

**输出：**

```
8
```

## 解题思路

这是经典的区间 DP 问题——石子合并（最小代价）。使用 **Knuth 优化**（四边形不等式优化）将 O(n^3) 朴素 DP 优化到 O(n^2)。

### 状态定义

`dp[i][j]` 表示将第 i 堆到第 j 堆合并成一堆的最小代价。

### 转移方程

```
dp[i][j] = min(dp[i][k] + dp[k+1][j]) + sum(i, j)    (i ≤ k < j)
```

其中 `sum(i, j)` 是第 i 堆到第 j 堆的石子总数（前缀和 O(1) 求解）。

### Knuth 优化

当 cost 函数满足**四边形不等式**时，决策点具有单调性：

```
opt[i][j-1] ≤ opt[i][j] ≤ opt[i+1][j]
```

利用此性质，内层枚举 k 的范围被限制在 `[opt[i][j-1], opt[i+1][j]]` 内，均摊 O(1)，总复杂度 O(n^2)。

### 初始化

- `dp[i][i] = 0`（单堆无需合并）
- `opt[i][i] = i`

### 复杂度

- 时间：O(n^2)
- 空间：O(n^2)
