# [Cnoi2021] 自我主义的平衡者

## 题目描述
n位居民评分，平均分>a_i打0分，否则打m分。初始平均分0。求不同排列下最终平均分的最大值和最小值。

## 输入格式
第一行n m，第二行a_i。

## 输出格式
两个实数，最大值和最小值。

## 样例
### 样例1
输入：
```
5 5
1 2 3 4 5
```
输出：
```
4.00 2.00
```

## 提示
1 < n,m ≤ 10^5

## 解题思路

### 问题分析
每个人根据当前平均分和自己的预期决定打0分或m分。

### 核心思路
贪心：最大化时按a_i降序，最小化时按a_i升序。

### 算法流程
1. 按策略排序
2. 模拟每人评分
3. 计算最终平均分

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
