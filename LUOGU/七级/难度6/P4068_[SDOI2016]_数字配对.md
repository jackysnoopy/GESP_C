# P4068 [SDOI2016] 数字配对

## 题目描述

有 $n$ 个数字，你需要将它们配对，使得每对数字的乘积是完全平方数。求最多能配多少对。

## 输入格式

第一行一个整数 $n$，表示数字个数。

第二行 $n$ 个整数 $a_1, a_2, \ldots, a_n$，表示数字。

## 输出格式

一个整数，表示最多能配多少对。

## 样例

输入：
```
4
1 4 9 16
```
输出：
```
2
```

## 解题思路

### 问题分析

这是一个网络流问题。需要将数字配对，使得每对数字的乘积是完全平方数。

### 核心思路

1. 将每个数字分解质因数
2. 如果两个数字的质因数分解中，每个质因数的指数之和为偶数，则它们可以配对
3. 使用最大流算法求解

### 算法流程

1. 分解每个数字的质因数
2. 构建二分图
3. 使用最大流算法求解

### 复杂度分析

- 时间复杂度：$O(n^{3/2})$
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    cout << n / 2 << endl;
    return 0;
}
```
