# [NOIP 2018 提高组] 赛道修建

## 题目描述
树上有n个节点，n-1条边。修建m条赛道（每条边至多被一条赛道经过）。使最短赛道长度最大。

## 输入格式
第一行n,m，接下来n-1行每行a_i,b_i,l_i。

## 输出格式
一个整数。

## 样例
输入：
```
7 1 
1 2 10 
1 3 5 
2 4 9 
2 5 8 
3 6 6 
3 7 7
```
输出：
```
31
```

## 提示
n ≤ 5×10^4, m ≤ n-1, l_i ≤ 10^4。

## 解题思路

### 问题分析
树上选择m条不相交路径，使最短路径长度最大。

### 核心思路
二分答案+树形DP贪心验证。DFS返回子树中最长未匹配路径，贪心配对。

### 算法流程
1. 二分最短赛道长度mid
2. DFS：每个子树返回最长未匹配路径
3. 贪心配对：两条路径之和≥mid则配对
4. 统计配对数是否≥m

### 复杂度分析
- 时间：O(n log W)
- 空间：O(n)
