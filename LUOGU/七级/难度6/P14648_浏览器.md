# P14648 [POI 2025/2026 #1] 浏览器 / Przeglądarka internetowa

## 题目描述

浏览器支持输入字母、BACKSPACE、ENTER、TAB(自动补全最近访问的前缀)。给定n个网站地址，求最少按键次数。

## 输入格式

第一行 n。
接下来 n 行：网站地址。

## 输出格式

第一行：最少按键次数。
第二行：按键序列。

## 解题思路

### 核心思路
- 构建Trie树
- 选择访问顺序最大化TAB使用
- 贪心按字典序排列

### 复杂度
- 时间: O(S * 26)
- 空间: O(S)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <map>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<string> urls(n);
    for (int i = 0; i < n; i++) cin >> urls[i];
    vector<int> order(n);
    for (int i = 0; i < n; i++) order[i] = i;
    sort(order.begin(), order.end(), [&](int i, int j) { return urls[i] < urls[j]; });
    int total = 0;
    string current = "";
    for (int idx : order) {
        const string& url = urls[idx];
        int common = 0;
        while (common < (int)current.size() && common < (int)url.size() && current[common] == url[common]) common++;
        total += url.size() - common + 1;
        current = url;
    }
    cout << total << "\n";
    current = "";
    for (int idx : order) {
        const string& url = urls[idx];
        int common = 0;
        while (common < (int)current.size() && common < (int)url.size() && current[common] == url[common]) common++;
        for (int i = common; i < (int)url.size(); i++) cout << url[i];
        cout << "E";
        current = url;
    }
    cout << "\n";
    return 0;
}
```
