# [集训队互测 2021] 数列重排

## 题目描述
定义一个数列区间的 $\textrm{mex}$ 为区间中最小的没有出现过的自然数，定义一个数列的价值为其中 $\textrm{mex}\geq k$ 的区间数量。

给定 $n$ 个小于 $m$ 的自然数和一个区间 $[l,r]$，令 $f(k)$ 表示 $n$ 个数构成的数列所有重排列中数列价值的最大值，对于每一个 $k\in [l,r]$，求出 $f(k)$。

## 输入格式
第一行四个整数 $m,l,r,X$。

第二行一个长度为 $m$ 的 $01$ 串，若其中第 $i$ 个位置为 $1$ 则数字 $i-1$ 的出现次数为 $X+1$，否则出现次数为 $X$。

## 输出格式
令 $ans=\displaystyle{\bigoplus_{i=l}^r}  (233^if(i)\bmod 998244353)$，输出一行一个整数 $ans$。

## 样例
### 样例1
输入:
```
2 0 1 2
10
```
输出:
```
3034
```

## 提示
- 时间限制：1000ms
- 内存限制：262144KB

## 解题思路

### 问题分析
需要计算所有 $k$ 对应的 $f(k)$，然后按位异或。

### 核心思路
1. 分析 $f(k)$ 的计算公式
2. 优化计算过程

### 算法流程
1. 统计每个数字的出现次数
2. 对于每个 $k$，计算 $f(k)$
3. 累加异或结果

### 复杂度分析
- 时间复杂度：$O(m)$
- 空间复杂度：$O(m)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MOD = 998244353;
const int BASE = 233;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int m, l, r, X;
    cin >> m >> l >> r >> X;
    
    string s;
    cin >> s;
    
    int extra = 0;
    for (char c : s) {
        if (c == '1') extra++;
    }
    
    long long ans = 0;
    
    for (int k = l; k <= r; k++) {
        long long fk = 0;
        
        if (k == 0) {
            fk = (long long)(m * X + extra) * (m * X + extra + 1) / 2;
        } else if (k <= m) {
            fk = (long long)(m * X + extra - k) * (m * X + extra - k + 1) / 2;
        }
        
        long long power = 1;
        for (int i = 0; i < k; i++) {
            power = (power * BASE) % MOD;
        }
        
        ans ^= (power * fk) % MOD;
    }
    
    cout << ans << endl;
    
    return 0;
}
```
