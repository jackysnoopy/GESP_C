# 要有光

## 题目描述

万物有灵，法术亦是如此。任何法术都等价为一段仅包括大小写字母的字符串 S=s₁s₂…sₙ。

初始法术 S=S₀（S₀ 是一个回文串），有 5 种变换：

- **光归**（代价 A）：保留最大回文后缀
- **光辉**（代价 B）：在 S₀ 中寻找更大的回文扩展当前串
- **光隐**（代价 C）：删除等长的前缀和后缀（长度 ≤ k，结果非空）
- **光腾**（代价 D）：左右加一对逆法术对（结果须为 S₀ 的子串）
- **光弋**（代价 E）：前端加一个字符（之后不能再用其他变换）

对于每个查询 [l, r]，求将 S 变为 T=S₀[l,r] 的最小代价。

## 输入格式

第一行：字符串 S₀
第二行：正整数 k
第三行：五个正整数 A B C D E
第四行：正整数 q
接下来 q 行：每行两个正整数 l r

## 输出格式

q 行，每行一个整数，表示第 i 个询问的答案。

## 样例

**样例 1：**
```
ababa
2
3 2 4 2 1
3

```
0

```

**样例 2：**

```

## 提示

- 1 ≤ q, |S| ≤ 10^5
- 1 ≤ A, B, C, D, E ≤ 10^9
- 1 ≤ l ≤ r ≤ |S|
- 1 ≤ k ≤ 5

## 解题思路

### 问题分析

本题要求对每个查询 [l, r]，找到将初始回文串 S₀ 变为子串 T=S₀[l,r] 的最小代价。关键在于理解各变换操作的性质和适用条件。

### 核心思路

分析各操作的特点：

1. **光归**（代价 A）：可以从当前串提取最长回文后缀。由于 S₀ 是回文串，光归可以提取 S₀ 的任何回文子串。
2. **光隐**（代价 C）：要求当前串是回文串，删除等长的前后缀。可以从 S₀ 直接提取某些子串。
3. **光弋**（代价 E）：在前端添加字符，使用后不能用其他操作。但光隐可以在光弋之后使用。

**关键观察：**
- 如果 T 是回文串：可以直接用光归提取（代价 A），或用光隐删除前后缀得到（代价 C），取最小值
- 如果 T 不是回文串：需要先用光隐减少字符串，再用光弋添加字符（代价 C + E）

### 算法流程

1. 用 **Manacher 算法**预处理 S₀ 的所有回文子串
2. 对每个查询 [l, r]：
   - 检查 S₀[l, r] 是否为回文串
   - 如果是回文：答案 = min(A, C)
   - 如果不是回文：答案 = C + E

### 复杂度分析

- 时间复杂度：O(|S| + q)
- 空间复杂度：O(|S|)

## 参考代码

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

const int MAXN = 200005;

int p[MAXN];

void manacher(const string& s, int* p) {
    int n = s.size();
    for (int i = 0; i < n; i++) p[i] = 0;
    int c = 0, r = 0;
    for (int i = 0; i < n; i++) {
        if (i < r) {
            p[i] = min(p[2 * c - i], r - i);
        }
        while (i - p[i] - 1 >= 0 && i + p[i] + 1 < n &&
               s[i - p[i] - 1] == s[i + p[i] + 1]) {
            p[i]++;
        }
        if (i + p[i] > r) {
            c = i;
            r = i + p[i];
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s0;
    cin >> s0;
    int n = s0.size();
    
    int k;
    cin >> k;
    
    long long A, B, C, D, E;
    cin >> A >> B >> C >> D >> E;
    
    int q;
    cin >> q;
    
    // Build transformed string for Manacher
    string t(2 * n + 1, '#');
    for (int i = 0; i < n; i++) {
        t[2 * i + 1] = s0[i];
    }
    
    manacher(t, p);
    
    // Process queries
    while (q--) {
        int l, r;
        cin >> l >> r;
        l--; r--;
        
        int len = r - l + 1;
        int center = l + r + 1;
        bool isPal = (p[center] >= len);
        
        if (isPal) {
            cout << min(A, C) << "\n";
        } else {
            cout << C + E << "\n";
        }
    }
    
    return 0;
}
```
