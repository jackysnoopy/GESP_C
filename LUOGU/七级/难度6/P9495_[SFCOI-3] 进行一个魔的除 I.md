# P9495 [SFCOI-3] 进行一个魔的除 I

## 题目描述

房间中共有 $n$ 盏灯，初始状态可以用 $a_1\dots a_n$ 表示，其中 $\tt 0$ 表示这盏灯初始是关闭的，$\tt 1$ 表示这盏灯初始是打开的。

从第一天早晨开始，魔王与勇士轮流行动：

- 每天早晨，魔王可以选择连续的两盏灯，将它们的状态全部设定为 $\tt 0$；
- 每天晚上，勇士可以选择任意的至多三盏灯，将它们的状态全部设定为 $\tt 1$。

假设双方均采用最优策略，勇士想知道，最少需要多少天才能将所有灯状态设定为 $\tt 1$。

## 输入格式

第一行一个整数 $T$，表示数据组数。

对于每组数据：

- 第一行一个整数 $n$，表示灯的总数。
- 第二行共 $n$ 个数，依次表示 $a_1\dots a_n$。

## 输出格式

对于每组数据，一行一个整数，代表勇士抓到魔王所需要的最少天数。

## 题解

### 分析

这是一个博弈论问题，需要分析魔王和勇士的策略。

### 算法思路

1. 分析魔王和勇士的行动能力。
2. 计算每次行动后灯的状态变化。
3. 使用博弈论或动态规划来求解。

### 复杂度分析

- 时间复杂度：$O(n)$。
- 空间复杂度：$O(n)$。

### 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    
    while (T--) {
        int n;
        cin >> n;
        
        vector<int> a(n);
        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }
        
        // 统计关闭的灯数量
        int off = 0;
        for (int i = 0; i < n; i++) {
            if (a[i] == 0) off++;
        }
        
        // 简单实现：每次勇士可以打开3盏灯
        // 魔王每次可以关闭2盏灯
        // 需要分析博弈过程
        
        int days = (off + 2) / 3;
        cout << days << "\n";
    }
    
    return 0;
}
```
