# [POI 2021/2022 R3] 模板 2 / Szablon 2

## 题目描述

Bajtazar 想在墙上写字。订购一个镂空模板会免费附赠一个反向模板。找出所有可能的模板长度。

## 输入格式

一行字符串，由小写字母组成。

## 输出格式

输出所有可能的模板长度，升序排列，空格分隔。

## 样例

### 样例 1
输入：
```
abcabcabacbabcab
```
输出：
```
5 16
```

## 提示

- 1 ≤ n ≤ 1000000

## 解题思路

### 问题分析

模板和其反向可以拼出原串。需要找到所有可能的模板长度。

### 核心思路

1. 使用 Z-algorithm 或 KMP
2. 检查每个长度是否可行
3. 模板和反向模板的拼接

### 算法流程

1. 计算 Z 数组
2. 枚举模板长度
3. 验证是否能拼出原串

### 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s;
    cin >> s;
    int n = s.size();
    
    vector<int> z(n, 0);
    for (int i = 1, l = 0, r = 0; i < n; i++) {
        if (i <= r) z[i] = min(r - i + 1, z[i - l]);
        while (i + z[i] < n && s[z[i]] == s[i + z[i]]) z[i]++;
        if (i + z[i] - 1 > r) l = i, r = i + z[i] - 1;
    }
    
    vector<int> ans;
    for (int len = 1; len <= n; len++) {
        bool ok = true;
        for (int i = 0; i + len <= n; i += len) {
            int remaining = min(len, n - i);
            if (z[i] < remaining && i + len < n) {
                ok = false;
                break;
            }
        }
        if (ok) ans.push_back(len);
    }
    
    for (int i = 0; i < ans.size(); i++) {
        if (i > 0) cout << " ";
        cout << ans[i];
    }
    cout << endl;
    
    return 0;
}
```
