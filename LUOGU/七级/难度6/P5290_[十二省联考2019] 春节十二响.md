# [十二省联考2019] 春节十二响

## 题目描述
n个子程序构成一棵树。内存分段机制：非祖先-后代关系的子程序可放同一段。段大小=段内最大Mi。求最小总内存。

## 输入格式
第一行n，第二行M1..Mn，第三行f2..fn。

## 输出格式
最小内存。

## 样例
输入：
```
5
10 20 20 30 30
1 1 2 2
```
输出：
```
60
```

## 解题思路

### 问题分析
- 树上的独立集问题变种
- 关键观察：对于链的情况，答案=最大Mi+次大Mi

### 核心思路
- 使用贪心+优先队列合并
- 对每个节点，将其子节点的优先队列合并，保留最大值
- 最终答案为根节点队列中所有元素之和

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

const int MAXN = 200005;
vector<int> adj[MAXN];
int M[MAXN];
priority_queue<int> pq[MAXN];
int ans = 0;

void dfs(int u) {
    for (int v : adj[u]) {
        dfs(v);
        if (pq[v].size() > pq[u].size()) swap(pq[u], pq[v]);
        while (!pq[v].empty()) {
            pq[u].push(pq[v].top());
            pq[v].pop();
        }
    }
    pq[u].push(M[u]);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> M[i];
    for (int i = 2; i <= n; i++) {
        int f;
        cin >> f;
        adj[f].push_back(i);
    }
    
    dfs(1);
    
    while (pq[1].size() > 1) {
        int a = pq[1].top(); pq[1].pop();
        int b = pq[1].top(); pq[1].pop();
        ans += max(a, b);
        pq[1].push(max(a, b));
    }
    
    cout << ans + pq[1].top() << "\n";
    return 0;
}
```
