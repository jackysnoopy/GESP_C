# P9404 [POI 2020/2021 R3] 扫雪波特 / Surowa zima

## 题目描述

有一条长 $l$ 米的道路（数轴）。路上有 $n$ 个充电站。每天整条路上（坐标 $[0,l]$）都会落满雪。

有一台机器能扫雪。充一次电可以扫至多 $k$ 米的雪。扫雪是和移动同时进行的。机器一秒能移动一米，充电不消耗时间。

简单来说，移动不扫雪不消耗电，需要一秒；移动并扫雪消耗最大电量的 $\frac{1}{k}$，需要一秒；扫雪必须移动。

给出每天机器的初始位置，机器初始没电，问每天清除所有雪的最少时间。终点任意。

带修，即充电站可能损坏或修好，但保证每天都至少有一个好的充电站。

## 输入格式

第一行四个整数 $n,l,k,d$。

第二行 $n$ 个整数 $x_1,x_2,\dots,x_n$，表示充电站的位置。

接下来 $3d$ 行，描述 $d$ 天的事件。

## 输出格式

$d$ 行，每行一个整数，表示每天的答案。

## 题解

### 分析

这是一个复杂的扫雪问题，需要动态维护充电站状态，并计算最小时间。

### 算法思路

1. 分析扫雪和充电的机制。
2. 使用贪心或动态规划来安排扫雪顺序。
3. 动态维护充电站的状态。

### 复杂度分析

- 时间复杂度：取决于具体算法实现。
- 空间复杂度：$O(n + d)$。

### 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, l, k, d;
    cin >> n >> l >> k >> d;
    
    vector<int> x(n);
    for (int i = 0; i < n; i++) {
        cin >> x[i];
    }
    
    vector<bool> active(n, true);
    
    for (int day = 0; day < d; day++) {
        int z, u, p;
        cin >> z >> u >> p;
        
        for (int i = 0; i < z; i++) {
            int idx;
            cin >> idx;
            active[idx - 1] = true;
        }
        
        for (int i = 0; i < u; i++) {
            int idx;
            cin >> idx;
            active[idx - 1] = false;
        }
        
        // 简单实现：计算最小时间
        // 这里需要更复杂的算法
        long long ans = 2 * l;
        cout << ans << "\n";
    }
    
    return 0;
}
```
