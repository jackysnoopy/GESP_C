# P14718 [RMI 2025] 松鼠 / Squirrel

## 题目描述

三角形矩阵，数1..N(N+1)/2各出现一次。松鼠沿对角线走，在(i,i)处选矩形区域内房间吃坚果。求最大坚果数。

## 输入格式

第一行 N。
接下来 N 行：三角形矩阵。

## 输出格式

第一行：最大坚果数。
第二行：每个对角线位置吃的坚果数。

## 解题思路

### 核心思路
- 贪心：按值从大到小处理
- 在每个对角线位置选择矩形区域内最大值

### 复杂度
- 时间: O(N^3)
- 空间: O(N^2)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int N;
    cin >> N;
    vector<vector<int>> A(N, vector<int>(N));
    for (int i = 0; i < N; i++) for (int j = 0; j <= i; j++) cin >> A[i][j];
    long long answer = 0;
    vector<int> solution(N);
    for (int i = 0; i < N; i++) {
        int maxVal = 0;
        for (int r = i; r < N; r++) for (int c = 0; c <= i; c++) maxVal = max(maxVal, A[r][c]);
        answer += maxVal;
        solution[i] = maxVal;
        for (int r = i; r < N; r++) for (int c = 0; c <= i; c++) if (A[r][c] == maxVal) { A[r][c] = 0; goto done; }
        done:;
    }
    cout << answer << "\n";
    for (int i = 0; i < N; i++) cout << solution[i] << (i + 1 < N ? " " : "\n");
    return 0;
}
```
