# 「EZEC-6」分组

## 题目描述
给n个数分组，每组内数按位或后累加，要求累加结果最小的情况下最多分成多少组。

## 输入格式
第一行n，第二行n个非负整数。

## 输出格式
一个正整数，最优解中最多分成的组数。

## 样例
### 样例1
输入：
```
6
1 1 4 5 1 4
```
输出：
```
1
```

### 样例2
输入：
```
7
1 9 1 9 8 1 0
```
输出：
```
2
```

## 提示
0 ≤ ai ≤ 10^18，1 ≤ n ≤ 10^7

## 解题思路

### 问题分析
分组后每组按位或再累加。要最小化总和并最大化组数。

### 核心思路
最优方案：每个非零数单独一组（不改变按位或结果），0可以附加到任意组。答案就是非零数个数。

### 算法流程
1. 统计非零数个数
2. 答案为max(1, 非零数个数)

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(1)
