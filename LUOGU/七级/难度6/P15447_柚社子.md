# P15447 「IXOI R1」柚社子

## 题目描述

有根树森林，每棵树有(l_a,r_a)约束。选若干点组成序列，满足祖先约束。求字典序最小序列。

## 输入格式

第一行 n。
接下来 n 行：x_i [l_i r_i]。

## 输出格式

第一行：序列长度。
第二行：序列。

## 解题思路

### 核心思路
- 贪心选择
- DFS计算子树大小
- 满足[l,r]约束

### 复杂度
- 时间: O(n)
- 空间: O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> parent(n + 1);
    vector<int> L(n + 1), R(n + 1);
    vector<vector<int>> children(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> parent[i];
        if (parent[i] == 0) { cin >> L[i] >> R[i]; }
        else children[parent[i]].push_back(i);
    }
    vector<int> sz(n + 1, 1);
    for (int i = n; i >= 1; i--) for (int c : children[i]) sz[i] += sz[c];
    vector<int> result;
    for (int i = 1; i <= n; i++) {
        if (parent[i] == 0) {
            vector<int> dfsOrder;
            vector<int> q; q.push_back(i);
            int head = 0;
            while (head < (int)q.size()) {
                int u = q[head++];
                if (u != i) dfsOrder.push_back(u);
                for (int c : children[u]) q.push_back(c);
            }
            for (int j = 0; j < L[i] && j < (int)dfsOrder.size(); j++) result.push_back(dfsOrder[j]);
        }
    }
    cout << result.size() << "\n";
    for (int i = 0; i < (int)result.size(); i++) cout << result[i] << (i + 1 < (int)result.size() ? " " : "\n");
    return 0;
}
```
