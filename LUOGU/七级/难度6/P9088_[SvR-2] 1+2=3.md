# P9088 [SvR-2] 1+2=3

## 题目描述

你有一些木棒，每个木棒左边有一个数，右边有一个数，数只有 $0,1,2$，你要将所有木棒拼起来，使相邻的数和为 $3$ 的对数最大。

例如，$1\text{ - }2$ 和 $1\text{ - }0$ 两个木棒，如果按 $1\text{ - }0,1\text{ - }2$ 这样拼，相邻的数和为 $3$ 的对数是 $0$；而按 $1\text{ - }\textcolor{red}{\underline{\textbf 2}},\textcolor{red}{\underline{\textbf 1}}\text{ - }0$ 这样拼相邻的数和为 $3$ 的对数是 $1$，因为 $2+1=3$。

## 输入格式

**本题有多组数据。**

输入的第一行一个正整数表示数据组数 $T$。

对于每组数据，一行 $9$ 个非负整数，分别表示 $0\text{ - }0,0\text{ - }1,0\text{ - }2,1\text{ - }0,1\text{ - }1,1\text{ - }2,2\text{ - }0,2\text{ - }1,2\text{ - }2$ 型木棒的个数。

## 输出格式

$T$ 行，每行一个整数表示答案。

## 题解

### 分析

这是一个组合优化问题，需要将木棒拼接起来，使得相邻数和为3的对数最大。

木棒有9种类型，每种类型由两个数字组成。相邻数和为3的情况只有：
- 前一个木棒右边是1，后一个木棒左边是2（即1-2和2-1的拼接）
- 前一个木棒右边是2，后一个木棒左边是1（即2-1和1-2的拼接）

### 算法思路

1. 首先直接配对1-2和2-1木棒，因为这是最直接的和为3的情况。
2. 考虑剩余木棒的配对：
   - 1-2木棒后面可以接2-0, 2-1, 2-2木棒（左边是2）
   - 2-1木棒后面可以接1-0, 1-1, 1-2木棒（左边是1）
3. 使用贪心策略，尽可能多地配对。

### 时间复杂度

$O(T)$，其中 $T$ 是数据组数。

### 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        vector<long long> cnt(9);
        for (int i = 0; i < 9; i++) cin >> cnt[i];
        
        long long ans = 0;
        
        // 直接配对1-2和2-1木棒
        long long pairs = min(cnt[5], cnt[7]);
        ans += pairs;
        cnt[5] -= pairs;
        cnt[7] -= pairs;
        
        // 考虑其他配对方式
        // 1-2木棒后面可以接2-0, 2-1, 2-2木棒（左边是2）
        // 2-1木棒后面可以接1-0, 1-1, 1-2木棒（左边是1）
        
        // 尝试更复杂的配对
        // 由于题目限制较大，可能需要数学推导
        
        // 简单情况：如果还有1-2木棒，可以与2-0,2-2配对
        if (cnt[5] > 0) {
            long long can_pair = min(cnt[5], cnt[6] + cnt[8]);
            ans += can_pair;
            cnt[5] -= can_pair;
        }
        
        // 如果还有2-1木棒，可以与1-0,1-1配对
        if (cnt[7] > 0) {
            long long can_pair = min(cnt[7], cnt[3] + cnt[4]);
            ans += can_pair;
            cnt[7] -= can_pair;
        }
        
        cout << ans << "\n";
    }
    return 0;
}
```

## 注意事项

1. 本题有多组数据，注意初始化。
2. 数据范围较大，使用long long类型。
3. 需要考虑所有可能的配对方式，贪心策略可能不是最优解。
