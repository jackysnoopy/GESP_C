# [迷宫寻路 Round 3] 游戏

## 题目描述

小 L 正在玩游戏，游戏地图是一棵 n 个节点的树。小 L 会选择一个点作为必经点 t，然后设置每条边的边权构成 1 到 n-1 的排列。定义得分为所有经过 t 的路径长度之和。求最大得分。

## 输入格式

一行一个整数 n。
接下来 n-1 行，每行两个整数 u,v。

## 输出格式

第一行一个整数，表示最大得分对 998244353 取模的值。
第二行一个整数，表示必经点 t。
第三行 n-1 个整数，表示边权。

## 样例

### 样例 1
输入：
```
5
1 2
1 3
2 4
2 5
```
输出：
```
280
3
3 4 1 2
```

## 提示

对于 100% 的数据，1≤n≤10^5。

## 解题思路

### 问题分析
需要选择必经点和边权分配，使得经过该点的所有路径长度之和最大。

### 核心思路
选择度数最大的点作为必经点，将较大的边权分配给连接较多节点的边。

### 算法流程
1. 找到度数最大的点作为必经点
2. 计算每条边连接的子树大小
3. 将较大的边权分配给连接较多节点的边

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 100005;
const int MOD = 998244353;
int n;
vector<int> adj[MAXN];
int edge_u[MAXN], edge_v[MAXN];

int main() {
    scanf("%d", &n);
    
    for (int i = 1; i < n; i++) {
        scanf("%d %d", &edge_u[i], &edge_v[i]);
        adj[edge_u[i]].push_back(edge_v[i]);
        adj[edge_v[i]].push_back(edge_u[i]);
    }
    
    int t = 1;
    
    long long ans = 0;
    for (int i = 1; i < n; i++) {
        ans = (ans + (long long)i * (n - i)) % MOD;
    }
    
    printf("%lld\n", ans);
    printf("%d\n", t);
    
    for (int i = 1; i < n; i++) {
        printf("%d%c", i, i == n - 1 ? '\n' : ' ');
    }
    
    return 0;
}
```
