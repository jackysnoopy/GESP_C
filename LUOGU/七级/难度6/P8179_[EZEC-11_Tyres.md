# [EZEC-11] Tyres

## 题目描述
n套轮胎跑m圈，第i套第j圈需a_i+b_i*(j-1)^2秒。换胎需t秒，第一套不需换。求最小总时间。

## 输入格式
第一行n m t，接下来n行每行a_i b_i。

## 输出格式
一个整数，最小总时间。

## 样例
### 样例1
输入：
```
2 4 50
10 100
100 1
```
输出：
```
365
```

## 提示
1 ≤ n,b_i ≤ 500，1 ≤ m ≤ 2×10^5

## 解题思路

### 问题分析
选择轮胎使用顺序和每套轮胎跑的圈数，最小化总时间。

### 核心思路
DP：dp[i][j] = 使用前i套轮胎跑j圈的最小时间。

### 算法流程
1. 预计算每套轮胎跑l圈的成本
2. DP转移
3. 输出dp[n][m]

### 复杂度分析
- 时间复杂度：O(n×m)
- 空间复杂度：O(m)
