# 序列 sequence

## 题目描述
给定序列，每次操作将某个数+1或-1。要求将序列变成非降数列，求最少操作次数。

## 输入格式
第一行n，第二行n个整数。

## 输出格式
一个整数。

## 样例
输入：
```
5
3 2 -1 2 11
```
输出：
```
4
```

## 提示
n ≤ 5×10^5, |a_i| ≤ 10^9。

## 解题思路

### 问题分析
经典CF13C加强版。将序列变为非递减，最小化总变化量（L1距离）。

### 核心思路
斜率技巧/优先队列。维护一个分段线性凸函数，对每个元素插入|x-a_i|，然后取前缀最小值使其非递减。

### 算法流程
1. 使用最大堆维护函数的"拐点"
2. 对每个a[i]，插入两个a[i]到堆中
3. 弹出堆顶（最大值），累加差值到答案
4. 最终结果即为最小操作次数

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
