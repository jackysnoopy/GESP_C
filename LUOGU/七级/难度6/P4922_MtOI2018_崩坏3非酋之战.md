# [MtOI2018] 崩坏3？非酋之战！

## 题目描述
女武神在位置1，boss在位置0，资源在位置n。boss每秒移动1格。女武神在boss到达其位置时攻击，攻击后后退1格。8种攻击方式各有不同效果。求能否在boss到达资源前击杀boss。

## 输入格式
第一行n，第二行hp和atk。

## 输出格式
两行：位置/伤害 和 结果文字。

## 样例
输入：
```
10
198 100
```
输出：
```
2
Tech Otakus Save The World!
```

## 提示
n ≤ 10000, atk ≤ 10^6。

## 解题思路

### 问题分析
模拟游戏过程，选择最优攻击策略最大化伤害。

### 核心思路
贪心/模拟：轮流使用各种攻击，利用冻结和燃烧效果最大化伤害输出。

### 算法流程
1. 模拟boss移动和女武神攻击
2. 根据当前状态选择最优攻击
3. 累计伤害，判断boss是否被击杀

### 复杂度分析
- 时间：O(n)
- 空间：O(1)
