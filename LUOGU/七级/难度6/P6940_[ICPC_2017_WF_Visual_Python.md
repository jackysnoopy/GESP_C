# [ICPC 2017 WF] Visual Python++

## 题目描述
Visual Python++语言中，语句块用矩形表示。给定n对左上角p和右下角y，需要匹配成矩形，使得矩形要么嵌套要么不相交，且边框不重叠。

## 输入格式
第一行n，接下来n行每行r c（左上角），再n行每行r c（右下角）。

## 输出格式
n行，每行一个整数表示匹配。若无解输出"syntax error"。

## 样例
### 样例1
输入：
```
2
4 7
9 8
14 17
19 18
```
输出：
```
2
1
```

## 提示
1 ≤ n ≤ 10^5，坐标 ≤ 10^9

## 解题思路

### 问题分析
需要将n个左上角和n个右下角配对，使得形成的矩形要么嵌套要么不相交。

### 核心思路
按行号排序左上角和右下角，使用栈匹配：对于每个左上角，找到最小的包含它的右下角。检查嵌套性质。

### 算法流程
1. 对左上角按(r,c)升序排序
2. 对右下角按(r,c)升序排序
3. 用栈维护候选右下角
4. 检查匹配后的嵌套合法性

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
