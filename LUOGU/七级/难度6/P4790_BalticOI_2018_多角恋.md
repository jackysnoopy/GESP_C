# [BalticOI 2018] 多角恋

## 题目描述
N个点的有向图，每个点出度为1。每次修改一条边的终点花费1。求最少修改次数使所有点形成若干个不相交的二元环。

## 输入格式
第一行N，接下来N行每行两个字符串s,t。

## 输出格式
一个整数或-1。

## 样例
输入：
```
8
leonard emmy
ada emmy
isaac leonard
emmy pierre
pierre bernhard
bernhard emmy
sofia karl
karl sofia
```
输出：
```
3
```

## 提示
N ≤ 100000。

## 解题思路

### 问题分析
函数图（每个点出度1）。已有的二元环不需要修改，其余点需要修改边。

### 核心思路
找到所有已有的二元环。不在二元环中的每个点需要修改其出边。答案 = 不在二元环中的点数。

### 算法流程
1. 建图，找所有二元环
2. 答案 = N - 2×(二元环数量)

### 复杂度分析
- 时间：O(N)
- 空间：O(N)
