# P14990 马赛克

## 题目描述

n*m黑白矩阵。不美观度=棋盘格2x2子矩阵数。选若干行和列改变颜色，最小化不美观度。输出sum w(i)^k。

## 输入格式

多组数据。每组：n m k，矩阵。

## 输出格式

每组一行答案。

## 解题思路

### 核心思路
- 状压DP枚举行和列的选择
- n,m<=12，2^n * 2^m枚举

### 复杂度
- 时间: O(2^n * 2^m * n * m)
- 空间: O(n * m)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 12;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n, m, k;
        cin >> n >> m >> k;
        int grid[MAXN][MAXN];
        for (int i = 0; i < n; i++) for (int j = 0; j < m; j++) cin >> grid[i][j];
        long long result = 0;
        for (int rowMask = 0; rowMask < (1 << n); rowMask++) {
            for (int colMask = 0; colMask < (1 << m); colMask++) {
                int temp[MAXN][MAXN];
                for (int i = 0; i < n; i++) for (int j = 0; j < m; j++) {
                    temp[i][j] = grid[i][j];
                    if (rowMask & (1 << i)) temp[i][j] ^= 1;
                    if (colMask & (1 << j)) temp[i][j] ^= 1;
                }
                int ugliness = 0;
                for (int i = 0; i < n - 1; i++) for (int j = 0; j < m - 1; j++)
                    if (temp[i][j] == temp[i+1][j+1] && temp[i][j+1] == temp[i+1][j] && temp[i][j] != temp[i][j+1]) ugliness++;
                result += (long long)ugliness * ugliness;
            }
        }
        cout << result << "\n";
    }
    return 0;
}
```
