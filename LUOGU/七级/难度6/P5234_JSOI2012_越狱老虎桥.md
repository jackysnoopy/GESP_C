# [JSOI2012] 越狱老虎桥

## 题目描述
N层电网，M条高压线连接。敌人新增一条秘密高压线。小分队需破坏一条高压线使至少一层不通电。求最少特工数。

## 输入格式
第一行N,M，接下来M行每行a_i,b_i,T_i。

## 输出格式
一个整数或-1。

## 样例
输入：
```
4 3
1 2 1
1 3 2
1 4 3
```
输出：
```
3
```

## 提示
N ≤ 500000, M ≤ 1000000, T_i ≤ 100000。

## 解题思路

### 问题分析
图的边连通性问题。敌人加边后，我们需要破坏一条边使图不连通。

### 核心思路
求原图的桥（割边）。答案为最小权桥的权值。若无桥则输出-1。

### 算法流程
1. Tarjan算法求桥
2. 输出最小权桥的权值
3. 若无桥输出-1

### 复杂度分析
- 时间：O(N + M)
- 空间：O(N + M)
