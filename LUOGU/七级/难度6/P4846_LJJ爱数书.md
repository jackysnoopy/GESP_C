# LJJ爱数书

## 题目描述
序列A，每次操作使连续区间+1或-1并mod k。f(A,k)为使所有元素变0的最少操作次数。m次询问求f(A[l..r], k)。

## 输入格式
第一行n,m，第二行A，接下来m行每行l,r,k。

## 输出格式
m行每行一个整数。

## 样例
输入：
```
7 2
8 8 8 0 8 8 8
1 7 9
3 5 17
```
输出：
```
2
16
```

## 提示
n ≤ 200000, m ≤ 100000, k > max(A[i])。

## 解题思路

### 问题分析
差分数组上的模k问题。每次操作改变差分数组中两个位置。

### 核心思路
f(A,k) = Σ min(d[i], k-d[i]) / 2，其中d[i]是差分数组的绝对值mod k。

### 算法流程
1. 对每个查询计算差分数组
2. 累加min(d[i], k-d[i])
3. 除以2得到答案

### 复杂度分析
- 时间：O(n × m)（暴力），可用前缀和优化
- 空间：O(n)
