# [ICPC 2016 WF] Swap Space

## 题目描述
管理一个大型集群的计算机硬盘，需要将所有硬盘从旧文件系统统一为新文件系统。重新格式化会改变硬盘容量。需要购买额外硬盘空间来暂存数据，求最小购买量。

## 输入格式
第一行一个整数n，接下来n行每行两个整数a（旧容量）和b（新容量）。

## 输出格式
一个整数，表示最小额外购买的存储空间。

## 样例
### 样例1
输入：
```
4
6 6
1 7
3 5
3 5
```
输出：
```
1
```

### 样例2
输入：
```
4
2 2
3 3
5 1
5 10
```
输出：
```
5
```

## 提示
1 ≤ n ≤ 10^6，1 ≤ a,b ≤ 10^9

## 解题思路

### 问题分析
每块硬盘格式化后容量变化，需要在格式化前将数据移走。要求最小化购买的额外空间。

### 核心思路
将硬盘分为两类：
- **b ≥ a**：格式化后容量增大或不变，优先处理。按a升序排序，先处理容量小的。
- **b < a**：格式化后容量减小，后处理。按b降序排序，先处理剩余空间大的。

### 算法流程
1. 分类排序
2. 维护当前可用空间freeSpace
3. 如果freeSpace不够放当前硬盘数据，需购买额外空间
4. 格式化后更新freeSpace

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
