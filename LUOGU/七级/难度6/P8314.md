# P8314 树的中心覆盖

## 题目描述

给定一棵 n 个节点的树，每条边有权值。需要选择 k 个节点作为"中心"，使得所有节点到最近中心的最大距离最小。

求这个最小的最大距离，并输出选择的 k 个中心节点。

## 输入格式

第一行两个整数 n 和 k，分别表示节点数和中心数量。
接下来 n-1 行，每行三个整数 a, b, w，表示节点 a 和节点 b 之间有一条长度为 w 的边。

## 输出格式

第一行一个整数，表示最小的最大距离。
第二行 k 个整数，表示选择的中心节点编号。

## 样例

**样例输入：**
```
5 2
1 2 1
2 3 1
3 4 1
4 5 1
```

**样例输出：**
```
1
2 4
```

## 提示

- 1 ≤ n ≤ 200000
- 1 ≤ k ≤ n
- 1 ≤ w ≤ 10^9

## 解题思路

### 问题分析

本题是经典的"k-中心"问题（k-center problem）在树上的应用。需要放置 k 个中心，使得所有节点到最近中心的最大距离最小。

### 核心思路

使用**二分答案 + BFS 贪心**：

1. **二分答案**：二分最大距离 mid
2. **验证**：对于每个 mid，贪心地选择中心：
   - 从未覆盖的节点开始 BFS
   - 每次选择一个未覆盖的节点作为中心
   - 从该中心出发，覆盖距离 ≤ mid 的所有节点
   - 统计需要的中心数量
3. 如果中心数量 ≤ k，则 mid 可行

### 算法流程

1. 二分答案范围 [0, max_dist]
2. 对于每个 mid，用 BFS 贪心计算需要的中心数量
3. 找到最小的可行 mid
4. 在最终的 mid 下，重新运行贪心算法，记录选择的中心节点

### 复杂度分析

- 时间复杂度：O(n × log(max_dist))
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;

const int MAXN = 200005;
vector<pair<int, int>> adj[MAXN];
int n, k;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n >> k;
    for (int i = 0; i < n - 1; i++) {
        int a, b, w;
        cin >> a >> b >> w;
        adj[a].push_back(make_pair(b, w));
        adj[b].push_back(make_pair(a, w));
    }
    int low = 0, high = 1e9;
    while (low < high) {
        int mid = (low + high) / 2;
        int centers = 0;
        vector<bool> covered(n + 1, false);
        for (int i = 1; i <= n; i++) {
            if (!covered[i]) {
                centers++;
                queue<pair<int, int>> q;
                q.push(make_pair(i, 0));
                covered[i] = true;
                while (!q.empty()) {
                    int u = q.front().first;
                    int dist = q.front().second;
                    q.pop();
                    if (dist >= mid) continue;
                    for (size_t j = 0; j < adj[u].size(); j++) {
                        int v = adj[u][j].first;
                        int w = adj[u][j].second;
                        if (!covered[v]) {
                            covered[v] = true;
                            q.push(make_pair(v, dist + w));
                        }
                    }
                }
            }
        }
        if (centers <= k) high = mid;
        else low = mid + 1;
    }
    cout << low << endl;
    vector<int> centers;
    vector<bool> covered(n + 1, false);
    for (int i = 1; i <= n && (int)centers.size() < k; i++) {
        if (!covered[i]) {
            centers.push_back(i);
            queue<pair<int, int>> q;
            q.push(make_pair(i, 0));
            covered[i] = true;
            while (!q.empty()) {
                int u = q.front().first;
                int dist = q.front().second;
                q.pop();
                if (dist >= low) continue;
                for (size_t j = 0; j < adj[u].size(); j++) {
                    int v = adj[u][j].first;
                    int w = adj[u][j].second;
                    if (!covered[v]) {
                        covered[v] = true;
                        q.push(make_pair(v, dist + w));
                    }
                }
            }
        }
    }
    for (int i = 0; i < (int)centers.size(); i++) {
        cout << centers[i] << (i + 1 == (int)centers.size() ? "\n" : " ");
    }
    return 0;
}
```
