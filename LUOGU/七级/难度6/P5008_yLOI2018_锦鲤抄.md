# [yLOI2018] 锦鲤抄

## 题目描述
有向图，每个点有权值。任意时刻可选择一个有入度的点获得其权值并删除。最多选k个点，求最大总权值。

## 输入格式
第一行n,m,k，第二行权值w_i，接下来m行边。

## 输出格式
一个整数。

## 样例
输入：
```
7 7 3
10 2 8 4 9 5 7
1 2
1 3
1 4
2 5
3 6
3 7
4 7
```
输出：
```
24
```

## 提示
n ≤ 5×10^5, m ≤ 2×10^6, k ≤ n。

## 解题思路

### 问题分析
每次选择有入度的点（非源点），贪心选择权值最大的。

### 核心思路
使用优先队列，每次选权值最大的有入度节点，删除后更新邻居入度。

### 算法流程
1. 统计入度，将入度>0的节点加入优先队列
2. 每次弹出权值最大的节点，累加权值
3. 删除节点及其出边，更新邻居入度
4. 重复k次

### 复杂度分析
- 时间：O((n+m) log n)
- 空间：O(n+m)
