# [九省联考 2018] IIIDX

## 题目描述
给定n首曲目难度d和实数k。第i首曲目在Pass第⌊i/k⌋首后解锁。安排曲目顺序使d_i ≥ d_{⌊i/k⌋}。若多解，使d_1最大，d_2最大，以此类推。

## 输入格式
第一行n和k，第二行n个整数d。

## 输出格式
一行n个整数。

## 样例
输入：
```
4 2.0
114 514 1919 810
```
输出：
```
114 810 514 1919
```

## 提示
n ≤ 500000, 1 < k ≤ 10^9, 1 ≤ d ≤ 10^9。

## 解题思路

### 问题分析
需要安排曲目使得每个位置的难度不小于其依赖位置的难度。第i个位置依赖⌊i/k⌋个位置。

### 核心思路
贪心+并查集。从大到小处理难度值，对于每个难度值，将其分配到最靠右的可用位置，确保字典序最大。

### 算法流程
1. 将难度值从大到小排序
2. 维护并查集追踪可用位置
3. 对于每个难度值组，分配到满足约束的最右位置
4. 确保d_i ≥ d_{⌊i/k⌋}

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
