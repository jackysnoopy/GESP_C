# 「SvR-2」G64

## 题目描述

定义对于两棵有根二叉树 T1,T2，merge(T1,T2) 的结果是一棵二叉树，满足其根节点的左子树为 T1，右子树为 T2。定义函数 Gx(T)，给一棵 n 个节点的以 1 为根的有根二叉树，q 次询问，每次询问给定 x,i，求 Gx(Ti) 的最大独立集。

## 输入格式

第一行两个整数 n,q。
之后 n 行，每行两个整数 lsi, rsi 表示其左儿子和右儿子。
之后 q 行，每行两个整数 x, i，表示一次询问。

## 输出格式

q 行，每行一个数，表示这次询问的 Gx(Ti) 的最大独立集大小对 998244353 取模的结果。

## 样例

### 样例 1
输入：
```
5 3
2 3
0 4
5 0
0 0 
0 0
2 5 
2 1
1 1
```
输出：
```
5
24
6
```

### 样例 2
输入：
```
5 1
2 3
0 4
5 0
0 0 
0 0
64 1
```
输出：
```
592424678
```

## 提示

对于 100% 的数据，1≤x≤10^9，1≤n≤5×10^5，1≤q≤5×10^5。

## 解题思路

### 问题分析
需要计算二叉树经过 Gx 变换后的最大独立集。

### 核心思路
使用动态规划计算最大独立集，考虑二叉树的结构。

### 算法流程
1. 定义 dp[u][0] 和 dp[u][1] 分别表示不选和选节点 u 的最大独立集
2. 递归计算左右子树
3. 合并结果

### 复杂度分析
- 时间复杂度：O(n + q)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 500005;
const int MOD = 998244353;
int n, q;
int ls[MAXN], rs[MAXN];

struct NodeInfo {
    long long f0, f1;
};

NodeInfo dfs(int u) {
    if (u == 0) return {0, 0};
    
    NodeInfo left = dfs(ls[u]);
    NodeInfo right = dfs(rs[u]);
    
    NodeInfo res;
    res.f0 = max(left.f0, left.f1) + max(right.f0, right.f1);
    res.f1 = left.f0 + right.f0 + 1;
    
    return res;
}

int main() {
    scanf("%d %d", &n, &q);
    
    for (int i = 1; i <= n; i++) {
        scanf("%d %d", &ls[i], &rs[i]);
    }
    
    for (int i = 0; i < q; i++) {
        int x, idx;
        scanf("%d %d", &x, &idx);
        
        NodeInfo res = dfs(idx);
        long long ans = max(res.f0, res.f1) % MOD;
        printf("%lld\n", ans);
    }
    
    return 0;
}
```
