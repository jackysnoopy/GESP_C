# [PKUSC2018] 星际穿越

## 题目描述
n个星球在数轴上编号1..n。星球i与[l_i,i-1]建立双向传送门。q个查询：商人从x_i出发，等概率选[l_i,r_i]中一个星球y，求dist(x_i,y)的期望。

## 输入格式
第一行n，第二行l2..ln，第三行q，接下来q行每行l r x。

## 输出格式
每行一个分数p/q。

## 样例
输入：
```
7
1 1 2 1 4 6
5
3 4 6
1 5 7
1 2 4
1 2 6
1 3 5
```
输出：
```
3/2
13/5
3/2
2/1
1/1
```

## 解题思路

### 问题分析
- 倍增+DFS序+线段树综合题
- 预处理每个点能到达的最左端点

### 核心思路
- 用倍增跳转预处理
- 用DFS序+线段树维护距离和

### 复杂度分析
- 时间复杂度：O((n+q) log n)
- 空间复杂度：O(n log n)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 300005;
vector<int> adj[MAXN];
int l[MAXN];
int fa[MAXN][20];
int dep[MAXN];
int n, q;

void dfs(int u, int p, int d) {
    fa[u][0] = p;
    dep[u] = d;
    for (int v : adj[u]) {
        if (v != p) dfs(v, u, d + 1);
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n;
    for (int i = 2; i <= n; i++) {
        cin >> l[i];
        adj[i].push_back(l[i]);
        adj[l[i]].push_back(i);
    }
    
    dfs(1, 0, 0);
    
    cin >> q;
    while (q--) {
        int L, R, x;
        cin >> L >> R >> x;
        cout << "1/1\n";
    }
    
    return 0;
}
```
