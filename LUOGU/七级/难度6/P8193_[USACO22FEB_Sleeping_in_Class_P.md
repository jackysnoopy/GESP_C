# [USACO22FEB] Sleeping in Class P

## 题目描述
N堂课记录a_i次睡觉，可合并相邻或分割。求使所有数变为q_i的最小操作次数。

## 输入格式
第一行N，第二行a_i，第三行Q，接下来Q行每行q_i。

## 输出格式
每行一个答案，不可能输出-1。

## 样例
### 样例1
输入：
```
6
1 2 3 1 1 4
7
1
2
3
4
5
6
12
```
输出：
```
6
6
4
5
-1
4
5
```

## 提示
1 ≤ N,Q ≤ 10^5

## 解题思路

### 问题分析
合并/分割操作使所有数变为q，求最小操作次数。

### 核心思路
检查total%q==0，贪心分割成和为q的段，操作次数=n-段数。

### 算法流程
1. 检查total是否能被q整除
2. 贪心分割
3. 计算操作次数

### 复杂度分析
- 时间复杂度：O(N×Q)
- 空间复杂度：O(N)
