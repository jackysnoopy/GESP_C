# [CCO 2021] Travelling Merchant

## 题目描述
n个城市m条有向边，每条边需要资产≥ri才能走，走过增加pi资产。求从每个城市出发的最小初始资产。

## 输入格式
第一行n m，接下来m行每行ai bi ri pi。

## 输出格式
n个整数，无解输出-1。

## 样例
### 样例1
输入：
```
5 5
3 1 4 0
2 1 3 0
1 3 1 1
3 2 3 1
4 2 0 2
```
输出：
```
2 3 3 1 -1
```

## 提示
2 ≤ n,m ≤ 2×10^5

## 解题思路

### 问题分析
找从每个城市出发的最小初始资产，使得可以永远旅行。

### 核心思路
SPFA/BFS检测正权环，有正权环则初始资产为0。

### 算法流程
1. 对每个城市做SPFA
2. 检测是否有正权环回到起点
3. 有正权环则答案为0，否则-1

### 复杂度分析
- 时间复杂度：O(n×m)
- 空间复杂度：O(n+m)
