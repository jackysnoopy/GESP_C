# P9250 [集训队作业 2019] 蜀道难

## 题目描述

给定一棵 $n$ 个点的树，需要为每个点分配一个 $1$ 到 $n$ 的标号，使得：

1. 边权为两端点标号差的绝对值。
2. 整棵树的权值为所有边权和。
3. 任意两点，要么它们之间（包括端点）的路径的标号是单调的，要么存在第三点分别与这两点满足此条件。

求在满足条件的情况下，整棵树最小的权值。

并且有若干次操作：在树中加入一个新的叶子，每次操作后也要求最小的权值。

## 输入格式

第一行一个正整数 $n$ 表示初始的节点数。

接下来 $n-1$ 行，每行两个正整数 $u_i,v_i$，表示初始的边。

接下来一行一个非负整数 $q$ 表示修改次数。

接下来 $q$ 行，每行一个正整数 $f_i$，表示添加一个点和一条新边。

## 输出格式

共 $q+1$ 行，每行一个非负整数，依次表示初始和每次修改之后的最小权值。

## 题解

### 分析

这是一个复杂的树上问题，需要动态维护树的标号分配。

### 算法思路

1. 首先需要理解题目的约束条件。
2. 考虑如何分配标号使得权值最小。
3. 动态维护树的结构变化。

### 复杂度分析

- 时间复杂度：取决于具体算法实现。
- 空间复杂度：$O(n)$。

### 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 1e5 + 5;

vector<int> adj[MAXN];
int n, q;
long long ans = 0;

void dfs(int u, int parent, int depth) {
    for (int v : adj[u]) {
        if (v != parent) {
            ans += abs(depth - (depth + 1));
            dfs(v, u, depth + 1);
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> n;
    
    for (int i = 1; i < n; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    
    // 简单实现：按深度分配标号
    dfs(1, -1, 0);
    cout << ans << "\n";
    
    cin >> q;
    while (q--) {
        int f;
        cin >> f;
        n++;
        adj[f].push_back(n);
        adj[n].push_back(f);
        
        // 重新计算
        ans = 0;
        dfs(1, -1, 0);
        cout << ans << "\n";
    }
    
    return 0;
}
```
