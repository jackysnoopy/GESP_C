# P6831 [IOI 2020] 嘉年华奖券

## 题目描述

交互题。有 n 种颜色的奖券，每种 m 张。进行 k 轮游戏，每轮你需要为每种颜色选一张奖券。游戏负责人会选择一个整数 b，使得所有选中的奖券的 |a[i] - b| 之和最小。

你需要实现两个函数：
- `find_maximum(k, x)`：返回 k 轮游戏的最大总奖励。
- `allocate_tickets(s)`：给出每轮每种颜色的奖券编号分配方案。

## 数据范围

- 2 ≤ n ≤ 1500
- 1 ≤ k ≤ m ≤ 1500

## 样例

**样例1：**
```
k=2, x=[[0,2,5],[1,1,3]]
输出: 7
```

**样例2：**
```
k=1, x=[[5,9],[1,4],[3,6],[2,7]]
输出: 12
```

## 题解

### 核心思路

对于每轮游戏，负责人选择 b 使 sum|a[i]-b| 最小，此时 b 是各颜色选中奖券的中位数。

**关键观察**：将每种颜色的奖券从小到大排序，第 r 轮选择第 r 小的奖券（循环分配），这样可以最大化每轮的总奖励。

### 分配策略

1. 对每种颜色的奖券值排序
2. 第 i 种颜色第 j 轮使用 `x[i][j % m]`（循环分配）
3. 计算每轮的中位数和总奖励

### 复杂度

- 时间：O(nm log m)
- 空间：O(nm)
