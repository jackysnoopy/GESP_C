# [APIO2025] 转杆

## 题目描述

有 n 根杆，每根杆可以旋转。能量效率定义为所有杆对之间锐角之和。可以通过选择多根杆同时旋转来调整。求最大能量效率。

## 输入格式

第一行 n。
第二行 n 个整数 v[i]。

## 输出格式

输出最大能量效率。

## 样例

### 样例 1
输入：
```
2
20000 10000
```
输出：
```
25000
```

## 提示

- 2 ≤ n ≤ 100000
- 0 ≤ v[i] ≤ 49999

## 解题思路

### 问题分析

这是一个交互式问题，需要通过旋转操作最大化杆之间的锐角之和。

### 核心思路

1. 锐角定义为 min(|v[i]-v[j]|, 50000-|v[i]-v[j]|)
2. 最优策略是让杆均匀分布
3. 可以通过多次旋转操作实现

### 算法流程

1. 计算初始能量效率
2. 贪心旋转使分布更均匀
3. 输出最大值

### 复杂度分析

- 时间复杂度：O(n^2)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> v(n);
    for (int i = 0; i < n; i++) cin >> v[i];
    
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            int diff = abs(v[i] - v[j]);
            ans += min(diff, 50000 - diff);
        }
    }
    
    cout << ans << endl;
    return 0;
}
```
