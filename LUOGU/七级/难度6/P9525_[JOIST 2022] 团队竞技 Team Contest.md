# P9525 [JOIST 2022] 团队竞技 / Team Contest

## 题目描述

JOI 大学有 $N$ 只海狸，每只海狸有三项能力值：思考值，行动值和运气值。

一支队伍由三名队员组成，需要满足以下条件：每个成员都有一项能力值严格大于其他两人的对应能力值。

在所有符合条件的组队中，想要选一个总能力最强的队伍，总能力定义为：三人最大思考值，三人最大行动值和三人最大运气值之和。

请你求出，是否存在一个符合条件的组队，如果是，计算队伍总能力可能的最大值。

## 输入格式

第一行一个整数 $N$ 表示海狸数。

接下来 $N$ 行每行三个整数 $X_i,Y_i,Z_i$ 表示海狸的各项能力值。

## 输出格式

一行一个整数，如果不存在符合条件的组队，输出 `-1`，否则输出队伍总能力的最大值。

## 题解

### 分析

这是一个组合优化问题，需要选择三只海狸组成队伍。

### 算法思路

1. 枚举所有可能的三只海狸组合。
2. 检查是否符合条件。
3. 计算总能力并取最大值。

### 复杂度分析

- 时间复杂度：$O(N^3)$。
- 空间复杂度：$O(N)$。

### 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int N;
    cin >> N;
    
    vector<tuple<int, int, int>> beavers(N);
    for (int i = 0; i < N; i++) {
        int x, y, z;
        cin >> x >> y >> z;
        beavers[i] = {x, y, z};
    }
    
    // 简单实现：枚举所有可能的组合
    int ans = -1;
    
    for (int i = 0; i < N; i++) {
        for (int j = i + 1; j < N; j++) {
            for (int k = j + 1; k < N; k++) {
                auto [x1, y1, z1] = beavers[i];
                auto [x2, y2, z2] = beavers[j];
                auto [x3, y3, z3] = beavers[k];
                
                // 检查是否符合条件
                bool ok = true;
                
                // 检查每个成员是否有一项能力严格大于其他两人
                // 这里需要更复杂的检查逻辑
                
                if (ok) {
                    int total = max({x1, x2, x3}) + max({y1, y2, y3}) + max({z1, z2, z3});
                    ans = max(ans, total);
                }
            }
        }
    }
    
    cout << ans << "\n";
    return 0;
}
```

## 注意事项

1. 需要理解"每个成员都有一项能力值严格大于其他两人的对应能力值"的条件。
2. 枚举所有组合可能效率较低，需要优化。
3. 注意处理不存在符合条件组队的情况。
