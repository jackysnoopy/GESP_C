# 『SpOI - R1』我看到了，谢谢你们

## 题目描述

有一个长度为 n 的字符串 S。有 n+1 位候选人站在这 n+1 个前缀上。每个人有一个票数 ai 和花费 wi。得票数量严格超过总票数一半的人可以当选总统。你可以干涉一个候选人，让他进行指定的选择。求干涉的最小代价。

## 输入格式

第一行一个整数 T，表示数据组数。
对于每组数据：
一行三个整数 n,q,type。
接下来一行一个长度为 n 的字符串 S。
接下来 n+1 行，每行两个整数 ai, wi。
接下来 q 行，每行三个整数 o',p',x'。

## 输出格式

共 q+1 行，输出干涉的最小代价。

## 提示

对于 100% 的数据，1≤T≤2000，1≤n≤10^5，0≤q≤10^5。

## 解题思路

### 问题分析
需要找到干涉一个候选人的最小代价，使其能成为总统。

### 核心思路
计算每个候选人成为总统所需的最小代价。

### 算法流程
1. 计算总票数
2. 对每个候选人计算成为总统的最小代价
3. 取最小值

### 复杂度分析
- 时间复杂度：O(T * n * q)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 100005;
int n, q;
string S;
long long a[MAXN], w[MAXN];

int main() {
    scanf("%d", &n, &q);
    cin >> S;
    
    for (int i = 0; i <= n; i++) {
        scanf("%lld %lld", &a[i], &w[i]);
    }
    
    vector<long long> prefix_sum(n + 2, 0);
    for (int i = 0; i <= n; i++) {
        prefix_sum[i + 1] = prefix_sum[i] + a[i];
    }
    
    long long total = prefix_sum[n + 1];
    
    for (int i = 0; i <= q; i++) {
        long long ans = 1e18;
        
        for (int j = 0; j <= n; j++) {
            long long need = total / 2 + 1;
            long long have = a[j];
            
            if (have >= need) {
                ans = min(ans, 0LL);
                continue;
            }
            
            long long cost = w[j];
            ans = min(ans, cost);
        }
        
        printf("%lld\n", ans);
        
        if (i < q) {
            int o, p;
            long long x;
            scanf("%d %d %lld", &o, &p, &x);
            
            if (o == 1) {
                total = total - a[p] + x;
                a[p] = x;
            } else {
                w[p] = x;
            }
        }
    }
    
    return 0;
}
```
