# P4226 [清华集训 2017] 避难所

## 题目描述

在一场灾难中，$n$ 个人需要逃到避难所。每个人有一个编号 $1 \sim n$，第 $i$ 个人需要 $t_i$ 秒才能到达避难所。避难所的门会在 $T$ 秒后关闭。每个人可以在任意时刻出发，但必须在 $T$ 秒内到达避难所。求最多能有多少人到达避难所。

## 输入格式

第一行两个整数 $n, T$，表示人数和时间限制。

第二行 $n$ 个整数 $t_1, t_2, \ldots, t_n$，表示每个人需要的时间。

## 输出格式

一个整数，表示最多能到达避难所的人数。

## 样例

输入：
```
3 5
1 2 3
```
输出：
```
3
```

## 解题思路

### 问题分析

这是一个数学问题。需要安排每个人出发的时刻，使得尽可能多的人在 $T$ 秒内到达避难所。

### 核心思路

1. 将每个人需要的时间排序
2. 贪心地安排出发时刻
3. 优先安排需要时间少的人

### 算法流程

1. 读取每个人需要的时间
2. 排序
3. 贪心地安排出发时刻
4. 统计能到达的人数

### 复杂度分析

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, T;
    cin >> n >> T;
    
    vector<int> t(n);
    for (int i = 0; i < n; i++) cin >> t[i];
    sort(t.begin(), t.end());
    
    int ans = 0;
    long long time = 0;
    for (int i = 0; i < n; i++) {
        time += t[i];
        if (time <= T) {
            ans++;
        } else {
            break;
        }
    }
    
    cout << ans << endl;
    return 0;
}
```
