# 「语文」凑字数

## 题目描述
N句话长度a_i，作文纸每行L字，首行缩进2字。需至少M行。K个评分部分各S分。拆分句子有得分惩罚。空行扣C分。求最大得分。

## 输入格式
第一行N,M,L,K,S,C，第二行a_i，接下来N-1行每行K个整数。

## 输出格式
一个非负整数。

## 样例
输入：
```
4 4 12 2 10 5
5 5 10 4
2 -1
0 0
1 1
```
输出：
```
18
```

## 提示
N,M,a_i ≤ 200, L ≤ 200, K ≤ 5。

## 解题思路

### 问题分析
多维度优化问题：需要平衡句子分组（影响评分部分）和行数（影响字数分）。

### 核心思路
DP：dp[i][j] = 前i句话用j行的最大得分。对于每个评分部分独立计算后求和。

### 算法流程
1. 对每个评分部分，计算拆分惩罚
2. DP：dp[i][j] = max(dp[k][j-1] - 分割惩罚)
3. 考虑行数不足M时的字数扣分

### 复杂度分析
- 时间：O(N^2 × M × K)
- 空间：O(N × M)
