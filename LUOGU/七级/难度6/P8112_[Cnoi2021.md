# [Cnoi2021] 符文破译

## 题目描述
T的所有非空前缀是合法魔法词缀。将S分成最少的魔法词缀，无法划分输出Fake。

## 输入格式
第一行|T| |S|，第二行T，第三行S。

## 输出格式
最少段数或Fake。

## 样例
### 样例1
输入：
```
3 5
aba
abaab
```
输出：
```
2
```

## 提示
1 ≤ |S|,|T| ≤ 10^7

## 解题思路

### 问题分析
将S划分成T的前缀，最小化段数。

### 核心思路
DP：dp[i] = S[0..i-1]的最少划分段数。预处理每个位置的最长匹配。

### 算法流程
1. 预处理S每个位置与T的最长公共前缀
2. DP转移：dp[i+len] = min(dp[i+len], dp[i]+1)

### 复杂度分析
- 时间复杂度：O(|S|×|T|)
- 空间复杂度：O(|S|)
