# P6760 [THUPC 2019] 大碗宽面

## 题目描述

有 $m$ 碗宽面，第 $i$ 碗有 $n_i$ 根面条，宽度分别为 $a_{i,1}, a_{i,2}, \ldots, a_{i,n_i}$。

定义 $f(u,v)$ 为：将第 $u$ 碗和第 $v$ 碗的所有面条混合后，第 $\left\lfloor \frac{n_u + n_v + 1}{2} \right\rfloor$ 小的宽度。

求 $R(u) = \bigoplus_{v=1}^{m} (f(u,v) + u + v)$，其中 $\bigoplus$ 表示按位异或。

## 输入格式

第一行一个正整数 $m$，表示碗的数量。

接下来 $m$ 行，每行第一个数 $n_i$ 表示第 $i$ 碗的面条数，后面 $n_i$ 个数表示面条宽度。

## 输出格式

$m$ 行，第 $i$ 行输出 $R(i)$。

## 样例

### 样例输入

```
3
3 1 2 3
3 4 5 6
2 4 2
```

### 样例输出

```
4
7
7
```

## 提示

- $1 \le m \le 10^4$
- $1 \le n_i \le 500$
- 面条宽度为正整数

## 解题思路

### 问题分析

对每一对碗 $(u,v)$，需要求合并后第 $k$ 小的元素，其中 $k = \lfloor (n_u+n_v+1)/2 \rfloor$。然后计算所有对的异或值。

### 核心思路

由于 $n_i \le 500$，合并后的总长度最多为 $1000$，可以直接用归并的方式求第 $k$ 小元素。先对每碗的面条排序，然后双指针归并求中位数。

### 算法流程

1. 读入数据，对每碗面条排序
2. 枚举所有对 $(u,v)$，其中 $u \ne v$
3. 对每对，用双指针归并求第 $k$ 小元素
4. 将结果累加到 $R(u)$ 的异或值中

### 复杂度分析

- 时间：$O(m^2 \cdot n)$，其中 $n$ 为最大碗大小，约 $5 \times 10^9$ 操作在边界情况可能较慢，但实际 $n_i \le 500$ 且大多数碗较小，可接受
- 空间：$O(\sum n_i)$
