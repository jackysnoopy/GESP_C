# P3634 [APIO2012] 守卫

## 题目描述

在一个 $n \times n$ 的网格中，有 $m$ 个守卫。每个守卫可以照亮其所在的行和列。你需要选择最少的守卫，使得整个网格都被照亮。

## 输入格式

第一行两个整数 $n, m$，表示网格大小和守卫数量。

接下来 $m$ 行，每行两个整数 $x, y$，表示守卫的位置。

## 输出格式

一个整数，表示最少需要选择的守卫数量。

## 样例

输入：
```
3 3
1 1
2 2
3 3
```
输出：
```
2
```

## 解题思路

### 问题分析

这是一个二分图匹配问题。需要选择最少的守卫，使得每行每列都被覆盖。

### 核心思路

1. 将问题转化为二分图匹配
2. 左部节点表示行，右部节点表示列
3. 每个守卫连接其所在的行和列
4. 求最小顶点覆盖

### 算法流程

1. 构建二分图
2. 使用匈牙利算法求最大匹配
3. 最小顶点覆盖 = 最大匹配

### 复杂度分析

- 时间复杂度：$O(n \times m)$
- 空间复杂度：$O(n + m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<int> row(n + 1, 0), col(n + 1, 0);
    for (int i = 0; i < m; i++) {
        int x, y;
        cin >> x >> y;
        row[x] = 1;
        col[y] = 1;
    }
    
    int ans = 0;
    for (int i = 1; i <= n; i++) {
        if (row[i]) ans++;
        if (col[i]) ans++;
    }
    
    cout << ans << endl;
    return 0;
}
```
