# P5592 美德的讲坛 (难度7)

## 题意

- 有 $n$ 个少年，每人有 60 种衣着特点，特征值为 $2^{i-1}$（即第 $i$ 种特点对应第 $i$ 位为 1）
- 两个少年的厌恶度 = 只在一方出现的特点的特征值之和 = 两人的特征值异或（XOR）
- 将少年分组，要求每组内任意两人的厌恶度 $< x$
- 求最大组大小
- 有 $q$ 次修改操作（每次修改一个少年的特征值）

## 分析

### 核心观察

两个少年的厌恶度就是 $a_i \oplus a_j$。我们需要找一个最大子集，使得其中任意两个元素的异或值 $< x$。

### Trie 树贪心

从最高位到最低位逐位考虑：

1. **若 $x$ 的某位为 0**：该位上，组内所有元素必须相同（都为 0 或都为 1），否则异或后该位为 1，导致结果 $\geq x$。此时只能从两个子树中选一个。

2. **若 $x$ 的某位为 1**：该位上，组内元素可以不同（异或后该位为 1，不会超过 $x$）。此时可以尝试合并两个子树，但需确保低位的异或 $< x$ 的低位部分。

### 合并条件

当 $x$ 的某位为 1 时，合并两个子树的条件是：两个子树中任意元素的异或（仅考虑低位）$< x$ 的低位部分。

这可以通过计算两个子树间的最大异或值来判断：若最大异或值 $< x$ 的低位部分，则可以合并。

### 算法流程

1. 建立 60 位二进制 Trie 树
2. 对于每次查询，从根节点开始递归：
   - 若 $x$ 的当前位为 0：取两个子树中较大的答案
   - 若 $x$ 的当前位为 1：
     - 计算两个子树间的最大异或值
     - 若最大异或值 $< x$ 的低位部分：合并两个子树
     - 否则：取两个子树中较大的答案

## 复杂度

- 时间：$O((n + q) \times 60)$，每次查询/修改操作 $O(60)$
- 空间：$O(n \times 60)$，Trie 树空间

## 代码

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

typedef long long ll;

const int MAX_BITS = 60;

struct TrieNode {
    int child[2];
    int cnt;
    TrieNode() {
        child[0] = child[1] = -1;
        cnt = 0;
    }
};

TrieNode trie[6000005];
int tot = 0;

int new_node() {
    trie[++tot] = TrieNode();
    return tot;
}

void insert(ll x) {
    int u = 1;
    for (int i = MAX_BITS - 1; i >= 0; i--) {
        int bit = (x >> i) & 1;
        if (trie[u].child[bit] == -1) {
            trie[u].child[bit] = new_node();
        }
        u = trie[u].child[bit];
        trie[u].cnt++;
    }
}

void erase(ll x) {
    int u = 1;
    for (int i = MAX_BITS - 1; i >= 0; i--) {
        int bit = (x >> i) & 1;
        u = trie[u].child[bit];
        trie[u].cnt--;
    }
}

ll max_xor(int u1, int u2, int bit) {
    if (bit < 0 || u1 == -1 || u2 == -1) return 0;
    ll res = 0;
    if (trie[u1].child[0] != -1 && trie[u2].child[1] != -1) {
        res = max(res, (1LL << bit) + max_xor(trie[u1].child[0], trie[u2].child[1], bit - 1));
    }
    if (trie[u1].child[1] != -1 && trie[u2].child[0] != -1) {
        res = max(res, (1LL << bit) + max_xor(trie[u1].child[1], trie[u2].child[0], bit - 1));
    }
    if (res == 0) {
        if (trie[u1].child[0] != -1 && trie[u2].child[0] != -1) {
            res = max(res, max_xor(trie[u1].child[0], trie[u2].child[0], bit - 1));
        }
        if (trie[u1].child[1] != -1 && trie[u2].child[1] != -1) {
            res = max(res, max_xor(trie[u1].child[1], trie[u2].child[1], bit - 1));
        }
    }
    return res;
}

int query(int u, ll x, int bit) {
    if (u == -1 || bit < 0) return trie[u].cnt;
    int b = (x >> bit) & 1;
    if (b == 0) {
        int lc = trie[u].child[0], rc = trie[u].child[1];
        if (lc == -1) return query(rc, x, bit - 1);
        if (rc == -1) return query(lc, x, bit - 1);
        return max(query(lc, x, bit - 1), query(rc, x, bit - 1));
    } else {
        int lc = trie[u].child[0], rc = trie[u].child[1];
        if (lc == -1 || rc == -1) {
            int c = (lc != -1) ? lc : rc;
            return query(c, x, bit - 1);
        }
        ll mx = max_xor(lc, rc, bit - 1);
        ll lim = x & ((1LL << bit) - 1);
        if (mx < lim) {
            return trie[lc].cnt + trie[rc].cnt;
        } else {
            return max(query(lc, x, bit - 1), query(rc, x, bit - 1));
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n, q;
    ll x;
    cin >> n >> q >> x;

    new_node();

    for (int i = 0; i < n; i++) {
        ll a;
        cin >> a;
        insert(a);
    }

    cout << query(1, x, MAX_BITS - 1) << "\n";

    for (int i = 0; i < q; i++) {
        int pos;
        ll val;
        cin >> pos >> val;
        pos--;

        int u = 1;
        for (int j = MAX_BITS - 1; j >= 0; j--) {
            int bit = (val >> j) & 1;
            u = trie[u].child[bit];
        }
        ll old_val = 0;
        int uu = 1;
        for (int j = MAX_BITS - 1; j >= 0; j--) {
            int bit = (uu >> j) & 1;
            if (trie[uu].child[bit] != -1 && trie[trie[uu].child[bit]].cnt > 0) {
                old_val |= (1LL << j);
                uu = trie[uu].child[bit];
            } else {
                uu = trie[uu].child[1 - bit];
            }
        }

        erase(old_val);
        insert(val);
        cout << query(1, x, MAX_BITS - 1) << "\n";
    }

    return 0;
}
```

## 说明

1. **Trie 树构建**：将每个少年的特征值插入 60 位二进制 Trie 树，从最高位到最低位
2. **删除操作**：每次修改时，先找到旧值的 Trie 路径并删除，再插入新值
3. **最大异或计算**：使用贪心策略，每次尝试选择相反的分支以最大化异或值
4. **查询函数**：递归遍历 Trie 树，根据 $x$ 的位决定是否可以合并子树
