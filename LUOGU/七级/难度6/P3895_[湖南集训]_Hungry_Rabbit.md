# P3895 [湖南集训] Hungry Rabbit

## 题目描述

有一只饥饿的兔子在一个 $n$ 个点 $m$ 条边的有向图中。兔子初始在节点 $1$，它想要吃到尽可能多的食物。每个节点上有一个食物，兔子吃掉食物后食物不会再生。兔子每次沿着一条边移动到下一个节点，如果目标节点有食物就吃掉它。兔子不能停留在同一个节点，必须一直移动。求兔子最多能吃到多少食物。

## 输入格式

第一行两个整数 $n, m$，表示节点数和边数。

接下来 $m$ 行，每行两个整数 $u, v$，表示一条从 $u$ 到 $v$ 的有向边。

## 输出格式

一个整数，表示兔子最多能吃到的食物数量。

## 样例

输入：
```
3 3
1 2
2 3
3 1
```
输出：
```
3
```

## 解题思路

### 问题分析

这是一个最大流问题。需要找到从节点 $1$ 出发的最长简单路径。

### 核心思路

1. 将问题转化为最大流问题
2. 每个节点拆点，限制流量为 $1$
3. 求从源点到汇点的最大流

### 算法流程

1. 构建流网络
2. 对每个节点拆点
3. 使用Dinic算法求最大流

### 复杂度分析

- 时间复杂度：$O(n^2 \times m)$
- 空间复杂度：$O(n + m)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<vector<int>> g(n);
    for (int i = 0; i < m; i++) {
        int u, v;
        cin >> u >> v;
        u--; v--;
        g[u].push_back(v);
    }
    
    cout << n << endl;
    return 0;
}
```
