# P6047 丝之割

## 题目信息

- **题目来源**: 洛谷 P6047
- **难度**: 七级（难度6）
- **标签**: 网络流, 最小割, Dinic

## 题目描述

多弦琴有 $m$ 条弦 $(u_i, v_i)$。切割操作选 $(i, j)$，破坏所有 $u > i$ 且 $v < j$ 的弦，代价为 $a_i \times b_j$。求破坏所有弦的最小代价。

**输入格式**:
- 第一行：$n$ $m$
- 第二行：$a_1, a_2, \ldots, a_n$
- 第三行：$b_1, b_2, \ldots, b_n$
- 接下来 $m$ 行：每行两个整数 $u$ $v$，表示一条弦

**输出格式**: 一个整数，表示最小代价。

**约束条件**:
- $n, m \leq 3 \times 10^5$
- $2 \leq u \leq n$
- $1 \leq v \leq n-1$

## 样例

### 样例1
```
输入:
5 2
3 9 1 9 9
9 9 1 9 3
2 1
5 4

输出:
6
```

### 样例2
```
输入:
5 1
3 9 1 9 9
9 9 1 9 3
5 1

输出:
81
```

## 解题思路

### 问题分析

将弦看作二维平面上的点 $(u, v)$，切割操作 $(i, j)$ 可以破坏所有满足 $u > i$ 且 $v < j$ 的弦。我们的目标是找到一组切割操作，使得所有弦都被破坏，且总代价最小。

### 关键观察

每条弦 $(u, v)$ 可以被切割 $(i, j)$ 破坏，当且仅当 $i < u$ 且 $j > v$。

这意味着对于每条弦，我们需要：
- 选择一个小于 $u$ 的左边界 $i$
- 选择一个大于 $v$ 的右边界 $j$

### 建模为最小割问题

构造一个流网络：
- 源点 $S$，汇点 $T$
- 对于每个 $i \in \{1, 2, \ldots, n-1\}$，创建左节点 $L_i$，从 $S$ 到 $L_i$ 连边，容量为 $a_i$
- 对于每个 $j \in \{2, 3, \ldots, n\}$，创建右节点 $R_j$，从 $R_j$ 到 $T$ 连边，容量为 $b_j$
- 对于每条弦 $(u, v)$，从 $L_{u-1}$ 到 $R_{v+1}$ 连一条容量为 $\infty$ 的边

### 最小割的含义

在这个流网络中，最小割的值等于最小代价：
- 如果 $L_i$ 在 $T$ 侧（被割），表示我们选择了左边界 $i$，代价为 $a_i$
- 如果 $R_j$ 在 $S$ 侧（被割），表示我们选择了右边界 $j$，代价为 $b_j$
- 无穷边保证了对于每条弦，必须选择相应的左边界或右边界

### 算法选择

使用 **Dinic 算法** 求最大流（等价于最小割），时间复杂度为 $O(E\sqrt{V})$，对于本题的数据范围是可行的。

## 代码实现

```cpp
#include <cstdio>
#include <cstring>
#include <algorithm>
#include <queue>
using namespace std;

typedef long long ll;
const int MAXN = 600005;
const int MAXM = 300005;
const ll INF = 1e18;

int n, m;
ll a[MAXN], b[MAXN];

struct Edge {
    int to, next;
    ll cap;
} edges[MAXM * 6];

int head[MAXN], edgeCnt;
int level[MAXN], cur[MAXN];

void addEdge(int u, int v, ll cap) {
    edges[edgeCnt] = {v, head[u], cap};
    head[u] = edgeCnt++;
    edges[edgeCnt] = {u, head[v], 0};
    head[v] = edgeCnt++;
}

bool bfs(int s, int t) {
    memset(level, -1, sizeof(level));
    level[s] = 0;
    queue<int> q;
    q.push(s);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int i = head[u]; i != -1; i = edges[i].next) {
            int v = edges[i].to;
            if (level[v] == -1 && edges[i].cap > 0) {
                level[v] = level[u] + 1;
                q.push(v);
            }
        }
    }
    return level[t] != -1;
}

ll dfs(int u, int t, ll flow) {
    if (u == t) return flow;
    ll ret = 0;
    for (int& i = cur[u]; i != -1; i = edges[i].next) {
        int v = edges[i].to;
        if (level[v] == level[u] + 1 && edges[i].cap > 0) {
            ll f = dfs(v, t, min(flow - ret, edges[i].cap));
            edges[i].cap -= f;
            edges[i ^ 1].cap += f;
            ret += f;
            if (ret == flow) break;
        }
    }
    return ret;
}

ll maxFlow(int s, int t) {
    ll flow = 0;
    while (bfs(s, t)) {
        memcpy(cur, head, sizeof(head));
        flow += dfs(s, t, INF);
    }
    return flow;
}

int main() {
    scanf("%d %d", &n, &m);
    
    memset(head, -1, sizeof(head));
    edgeCnt = 0;
    
    int S = 0, T = 2 * n + 1;
    
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &a[i]);
    }
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &b[i]);
    }
    
    for (int i = 1; i < n; i++) {
        addEdge(S, i, a[i]);
    }
    for (int j = 2; j <= n; j++) {
        addEdge(j + n, T, b[j]);
    }
    
    for (int i = 0; i < m; i++) {
        int u, v;
        scanf("%d %d", &u, &v);
        addEdge(u - 1, v + 1 + n, INF);
    }
    
    printf("%lld\n", maxFlow(S, T));
    
    return 0;
}
```

## 复杂度分析

- **时间复杂度**: $O((n + m)\sqrt{n})$，Dinic 算法在二分图上的复杂度
- **空间复杂度**: $O(n + m)$

## 关键点

1. **问题建模**: 将切割问题转化为最小割问题
2. **流网络构造**: 左节点对应左边界选择，右节点对应右边界选择
3. **无穷边的作用**: 保证每条弦必须被至少一个切割覆盖
4. **Dinic 算法**: 高效求解最大流/最小割
