# P5258 [JSOI2013] 旅行时的困惑

## 题目描述

Waldives有N个小岛，N-1条有向快艇专线形成一棵树。政府希望新建最少的公交线路，使得任意两个小岛互相到达。每条公交线路对应树上一条路径，覆盖的树边方向与原方向相反。求最少公交线路数。

## 输入格式

第一行一个整数N，表示小岛数量。接下来N-1行，每行两个整数x y，表示从x到y的有向边。

## 输出格式

输出一个整数，表示最少需要的公交线路数。

## 样例

### 样例1
**输入：**
```
4
0 1
1 2
1 3
```
**输出：**
```
2
```

## 数据范围

- $1 \leq N \leq 10^5$

## 题解

### 问题分析

给定一棵有向树，需要添加最少的公交线路使得整个图强连通。每条公交线路对应树上的一条路径，覆盖的树边方向与原方向相反。

### 关键观察

对于一棵有向树（边从父节点指向子节点）：
- **叶子节点**（出度为0的节点）无法到达任何其他节点
- 要使所有节点互相到达，每个叶子节点都需要一条从它出发的路径
- 每条公交线路可以覆盖一条从叶子到根的路径，同时反转路径上的所有边

### 结论

**答案 = 树中叶子节点的数量（出度为0的节点数）**

### 证明

1. **必要性**：每个叶子节点出度为0，必须作为某条公交线路的起点才能到达其他节点。因此至少需要叶子数量条线路。

2. **充分性**：从每个叶子节点出发，沿树向上走到根节点，形成一条公交线路。这条线路覆盖的边方向被反转，使得：
   - 叶子节点现在有出边可以到达其他节点
   - 根节点现在有入边可以被其他节点到达
   - 所有中间节点都获得了新的连接

3. 这样，每个叶子都有一条向上的路径，所有节点都变得互相可达。

### 算法实现

1. 读入N和N-1条边
2. 统计每个节点的出度
3. 出度为0的节点数即为答案

### 复杂度分析

- **时间复杂度**：O(N)
- **空间复杂度**：O(N)

## 代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> inDegree(n, 0);
    vector<int> outDegree(n, 0);
    
    for (int i = 0; i < n - 1; i++) {
        int x, y;
        cin >> x >> y;
        outDegree[x]++;
        inDegree[y]++;
    }
    
    int leaves = 0;
    for (int i = 0; i < n; i++) {
        if (outDegree[i] == 0) {
            leaves++;
        }
    }
    
    cout << leaves << endl;
    
    return 0;
}
```
