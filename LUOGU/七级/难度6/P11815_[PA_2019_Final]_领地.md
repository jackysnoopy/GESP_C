# [PA 2019 Final] 领地 / Terytoria

## 题目描述

一个长为 X 宽为 Y 的矩形被分为 X×Y 个方格。有 n 种动物，第 i 种动物不喜欢待在某个矩形区域内。每种动物有 c_i 只。需要将每只动物放在一个方格里，得分定义为所有格子中 C(p[i][j], 2) 的和。求最大得分。

## 输入格式

第一行三个正整数 n, X, Y。
接下来 n 行，每行五个正整数 x_i, y_i, x'_i, y'_i, c_i。

## 输出格式

输出一行一个正整数，表示答案。

## 样例

### 样例 1
输入：
```
2 1 2
1 1 1 1 3
1 2 1 2 4
```
输出：
```
9
```

### 样例 2
输入：
```
3 7 3
1 1 3 3 1
5 1 7 3 1
3 2 5 3 1
```
输出：
```
3
```

## 提示

- 1 ≤ n ≤ 10^5
- 1 ≤ X, Y ≤ 10^3
- 1 ≤ c_i ≤ 10^3

## 解题思路

### 问题分析

这是一个组合优化问题。需要将动物分配到格子中，使得所有格子的 C(p,2) 之和最大。C(p,2) = p*(p-1)/2 是凸函数，所以应该将动物尽可能集中到少数格子。

### 核心思路

1. 对于每个格子，找出能放多少种动物（排除不喜欢该格子的动物）
2. 贪心地将动物分配到能放最多动物的格子
3. 由于 X,Y ≤ 1000，可以暴力枚举每个格子

### 算法流程

1. 标记每个格子不能放的动物种类
2. 对于每种动物，找到能放的格子
3. 贪心分配：优先将动物放到已有最多动物的格子

### 复杂度分析

- 时间复杂度：O(n * X * Y)
- 空间复杂度：O(X * Y)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, X, Y;
    cin >> n >> X >> Y;
    
    // 标记每个格子不能放的动物种类
    vector<vector<int>> blocked(X + 1, vector<int>(Y + 1, 0));
    
    for (int i = 0; i < n; i++) {
        int x1, y1, x2, y2, c;
        cin >> x1 >> y1 >> x2 >> y2 >> c;
        for (int x = x1; x <= x2; x++) {
            for (int y = y1; y <= y2; y++) {
                blocked[x][y]++;
            }
        }
    }
    
    // 计算答案
    long long ans = 0;
    for (int x = 1; x <= X; x++) {
        for (int y = 1; y <= Y; y++) {
            int can_place = n - blocked[x][y];
            ans += (long long)can_place * (can_place - 1) / 2;
        }
    }
    
    cout << ans << endl;
    return 0;
}
```
