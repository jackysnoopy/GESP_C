# P14402 [JOISC 2016] 危险的滑冰 / Dangerous Skating

## 题目描述

R*C网格，'#'是冰块，'.'可通行。JOI君从(r1,c1)出发，每次朝一个方向滑到冰块前停下，起点产生冰块。求到(r2,c2)的最少移动次数。

## 输入格式

第一行 R C。
接下来 R 行：网格。
接下来两行：起点和终点坐标。

## 输出格式

一个整数，最少移动次数或-1。

## 样例

输入：
```
5 5
#####
#...#
#...#
#...#
#####
2 2
3 3
```
输出：
```
4
```

## 解题思路

### 核心思路
- BFS搜索，状态为当前位置
- 每次向4个方向滑行直到遇到冰块
- 滑行过的路径上只有起点变冰块

### 复杂度
- 时间: O(R*C*(R+C))
- 空间: O(R*C)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstring>
using namespace std;

const int MAXN = 1005;
const int dx[] = {0, 0, 1, -1};
const int dy[] = {1, -1, 0, 0};

int R, C;
char grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int r1, c1, r2, c2;

int bfs() {
    if (r1 == r2 && c1 == c2) return 0;
    memset(dist, -1, sizeof(dist));
    vector<pair<int, int>> q;
    dist[r1][c1] = 0;
    q.push_back({r1, c1});
    int head = 0;
    while (head < (int)q.size()) {
        int x = q[head].first;
        int y = q[head].second;
        head++;
        int d = dist[x][y];
        for (int dir = 0; dir < 4; dir++) {
            int nx = x + dx[dir];
            int ny = y + dy[dir];
            if (nx < 1 || nx > R || ny < 1 || ny > C || grid[nx][ny] == '#') continue;
            while (nx + dx[dir] >= 1 && nx + dx[dir] <= R && ny + dy[dir] >= 1 && ny + dy[dir] <= C && grid[nx + dx[dir]][ny + dy[dir]] != '#') {
                nx += dx[dir];
                ny += dy[dir];
            }
            if (dist[nx][ny] == -1) {
                dist[nx][ny] = d + 1;
                if (nx == r2 && ny == c2) return d + 1;
                q.push_back({nx, ny});
            }
        }
    }
    return -1;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cin >> R >> C;
    for (int i = 1; i <= R; i++) for (int j = 1; j <= C; j++) cin >> grid[i][j];
    cin >> r1 >> c1 >> r2 >> c2;
    cout << bfs() << "\n";
    return 0;
}
```
