# 【MX-S6-T3】「KDOI-11」简单的字符串问题 2

## 题目描述

给定 n 个字符串 S1, ..., Sn 以及一个字符串 T。定义一个好的字符串是某个 Si 的前缀或为空。定义一个好的三元组 (l,r,k) 满足存在 k 个好的字符串拼接起来等于 T[l,r]。求好的三元组数量。

## 输入格式

第一行，三个非负整数 id,n,K。
接下来 n 行，每行一个字符串 Si。
接下来一行，一个字符串 T。

## 输出格式

第一行，一个非负整数，表示好的三元组的数量。
第二行，|T| 个非负整数，第 i 个表示满足 l≤i≤r 的好的三元组 (l,r,k) 的数量。

## 样例

### 样例 1
输入：
```
0 1 2
ab
abaab
```
输出：
```
13
5 3 5 6 3
```

### 样例 2
输入：
```
0 3 2
abc
ac
b
bacabcab
```
输出：
```
27
4 9 6 11 10 5 6 5
```

## 提示

对于 100% 的数据，1≤n≤10，1≤|Si|≤5×10^4，1≤|T|,K≤5×10^5。

## 解题思路

### 问题分析
需要统计好的三元组数量，并计算每个位置被覆盖的次数。

### 核心思路
使用Trie树存储所有好的字符串，然后枚举子串检查是否能被分解。

### 算法流程
1. 将所有Si插入Trie树
2. 枚举T的所有子串
3. 检查子串是否能被分解为k个好的字符串
4. 统计答案

### 复杂度分析
- 时间复杂度：O(|T|^2 * K)
- 空间复杂度：O(Σ|Si|)

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 500005;
int id, n, K;
string S[MAXN];
string T;

int main() {
    scanf("%d %d %d", &id, &n, &K);
    
    for (int i = 0; i < n; i++) {
        cin >> S[i];
    }
    cin >> T;
    
    int len = T.length();
    
    struct TrieNode {
        int children[26];
        int count;
        TrieNode() {
            for (int i = 0; i < 26; i++) children[i] = -1;
            count = 0;
        }
    };
    
    vector<TrieNode> trie(1);
    
    for (int i = 0; i < n; i++) {
        int node = 0;
        for (char c : S[i]) {
            int idx = c - 'a';
            if (trie[node].children[idx] == -1) {
                trie[node].children[idx] = trie.size();
                trie.emplace_back();
            }
            node = trie[node].children[idx];
            trie[node].count++;
        }
    }
    
    long long total = 0;
    vector<int> ans(len, 0);
    
    for (int i = 0; i < len; i++) {
        int cnt = 0;
        for (int j = i; j < len && cnt < K; j++) {
            int node = 0;
            bool isPrefix = true;
            for (int k = i; k <= j; k++) {
                int idx = T[k] - 'a';
                if (trie[node].children[idx] == -1) {
                    isPrefix = false;
                    break;
                }
                node = trie[node].children[idx];
            }
            
            if (isPrefix && trie[node].count > 0) {
                cnt++;
                total++;
                for (int k = i; k <= j; k++) {
                    ans[k]++;
                }
            }
        }
    }
    
    printf("%lld\n", total);
    for (int i = 0; i < len; i++) {
        printf("%d%c", ans[i], i == len - 1 ? '\n' : ' ');
    }
    
    return 0;
}
```
