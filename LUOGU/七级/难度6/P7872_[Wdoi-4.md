# [Wdoi-4] 觉姐姐和恋妹妹

## 题目描述
n×m网格，觉从(1,1)走到(xs,ys)可移动物品，恋从(1,1)走到(xk,yk)看到所有物品。求恋最大愉悦度。

## 输入格式
第一行n m，接下来n行每行m个整数，最后一行xs ys xk yk。

## 输出格式
一个整数，最大愉悦度。

## 样例
### 样例1
输入：
```
4 4
0 4 5 3
3 2 -1 2
-1 3 -3 -1
0 4 2 4
3 3 4 4
```
输出：
```
22
```

## 提示
1 ≤ n,m ≤ 300

## 解题思路

### 问题分析
觉可以移动物品，恋选择最优路径。

### 核心思路
恋选择从(0,0)到(xk,yk)的最优路径，最大化路径上w[i][j]之和。

### 算法流程
1. DP计算恋的最优路径
2. 输出路径价值和

### 复杂度分析
- 时间复杂度：O(n×m)
- 空间复杂度：O(n×m)
