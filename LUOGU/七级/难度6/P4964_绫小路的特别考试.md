# 绫小路的特别考试

## 题目描述
n名学生坐成一排，每名学生有能力值w_i和对讲机范围d_i。询问：难度为x时有多少学生会做？修改：改变绫小路的能力值。

## 输入格式
使用伪随机数生成器输入。第一行n,m,c，第二行seed等参数，接下来k行调整d值。

## 输出格式
一个整数T_q。

## 样例
输入：
```
3 3 2
19720918 0 1 2 0
```
输出：
```
700
```

## 提示
n, m ≤ 2×10^6。

## 解题思路

### 问题分析
图上的可达性问题：能力值≥x的学生是源点，通过广播范围传播。

### 核心思路
使用并查集或BFS处理每次查询。由于数据量大，需要高效的离线处理或数据结构。

### 算法流程
1. 按能力值排序学生
2. 使用DSU维护连通分量
3. 对每个查询，统计包含源点的分量大小

### 复杂度分析
- 时间：O((n+m) α(n))
- 空间：O(n)
