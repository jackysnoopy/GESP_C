# [USACO09OPEN] Tower of Hay G

## 题目描述
n包干草按顺序送来，每包宽度Wi。堆塔时每层干草必须连续，上层宽度≤下层宽度。求最高塔高。

## 输入格式
第一行n，接下来n行每行Wi。

## 输出格式
一个整数。

## 样例
输入：
```
3
1
2
3
```
输出：
```
2
```

## 提示
n ≤ 100000, Wi ≤ 10000。

## 解题思路

### 问题分析
将序列分成连续的h段，每段宽度非递增（从上到下）。

### 核心思路
二分答案h，从右到左贪心验证：每层取尽可能少的干草使宽度≥下层。

### 算法流程
1. 二分层数h
2. 从右到左，每层贪心取最少干草
3. 检查是否所有干草恰好用完

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
