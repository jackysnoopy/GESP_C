# 【MX-S1-T4】先见之明

## 题目描述

给定 n 个非负整数 a1, a2, ..., an。有 q 次询问，每次询问给定一个非负整数 k，需要从 2^a1, ..., 2^an 中取出一部分使得它们的和 ≥ k，且和最小。

## 输入格式

第一行两个正整数 n,q。
第二行 n 个非负整数 a1,a2,...,an。
接下来 q 行，每行第一个非负整数为 m，接下来 m 个非负整数为 p1,p2,...,pm。

## 输出格式

共 q 行，每行一个整数，表示答案对 998244353 取模后的结果，或输出 -1 表示无解。

## 样例

### 样例 1
输入：
```
3 3
0 0 1
0
2 1 0
1 3
```
输出：
```
0
3
-1
```

## 提示

对于 100% 的数据，1≤n,q≤10^6，0≤m≤10^6，Σm≤5×10^6，0≤ai,pi≤10^6。

## 解题思路

### 问题分析
需要从给定的2的幂次中选择子集，使其和≥k且最小。

### 核心思路
贪心算法，从高位向低位考虑，优先使用已有的幂次。

### 算法流程
1. 统计每个幂次的出现次数
2. 对每个询问，从高位向低位贪心选择
3. 如果需要借位，从更高位分解

### 复杂度分析
- 时间复杂度：O(n + q * max_bit)
- 空间复杂度：O(max_bit)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 1000005;
const int MOD = 998244353;
int n, q;
int a[MAXN];

int main() {
    scanf("%d %d", &n, &q);
    
    vector<int> cnt(MAXN, 0);
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
        if (a[i] < MAXN) cnt[a[i]]++;
    }
    
    for (int i = 0; i < q; i++) {
        int m;
        scanf("%d", &m);
        
        vector<int> p(m);
        for (int j = 0; j < m; j++) {
            scanf("%d", &p[j]);
        }
        
        if (m == 0) {
            printf("0\n");
            continue;
        }
        
        long long ans = 0;
        bool possible = true;
        vector<int> have = cnt;
        
        for (int j = 0; j < m && possible; j++) {
            int bit = p[j];
            
            if (have[bit] > 0) {
                have[bit]--;
                ans = (ans + (1LL << bit)) % MOD;
            } else {
                bool found = false;
                for (int k = bit + 1; k < MAXN; k++) {
                    if (have[k] > 0) {
                        have[k]--;
                        for (int l = k - 1; l > bit; l--) {
                            have[l] += 2;
                        }
                        have[bit] += 2;
                        have[bit]--;
                        ans = (ans + (1LL << bit)) % MOD;
                        found = true;
                        break;
                    }
                }
                if (!found) {
                    possible = false;
                }
            }
        }
        
        if (!possible) {
            printf("-1\n");
        } else {
            printf("%lld\n", ans);
        }
    }
    
    return 0;
}
```
