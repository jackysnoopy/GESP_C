# P6163 [Cnoi2020] 领域极限

## 题目描述

给定 n 个整数 a_i，每个数有一个区间限制 [l_i, r_i]。求最小化 sum_{i=1}^n sum_{j=1}^n |a_i - a_j|，其中 a_i ∈ [l_i, r_i]。

## 输入格式

第一行一个整数 n。
接下来 n 行，每行两个整数 l_i, r_i，表示第 i 个数的区间限制。

## 输出格式

一个整数，表示最小值。

## 样例

### 样例 1

输入：
```
3
1 2
2 3
3 4
```
输出：
```
4
```

### 样例 2

输入：
```
8
...
```
输出：
```
910
```

## 约束

- 1 ≤ n ≤ 10^5
- 0 ≤ l_i ≤ r_i ≤ 10^9

## 题解

### 问题分析

目标函数 S = sum_{i=1}^n sum_{j=1}^n |a_i - a_j| = 2 * sum_{1 ≤ i < j ≤ n} |a_i - a_j|。  
对于固定的值集合，S 与值的分布有关。当所有 a_i 尽可能接近时，S 最小。

### 核心思想

设选择的值的中位数为 m。对于每个 i，最优选择是将 a_i 设为区间 [l_i, r_i] 中最接近 m 的值，即 a_i = clamp(m, l_i, r_i) = max(l_i, min(r_i, m))。  
此时，S 是关于 m 的凸函数，因此可以用三分搜索在 [0, 10^9] 上求最小值。

### 算法步骤

1. 读取所有区间，分别存储 l_i 和 r_i 并排序。
2. 对于给定的 m，计算每个 a_i = clamp(m, l_i, r_i)。  
   - 将区间分为三组：  
     * L 组：l_i > m，此时 a_i = l_i  
     * M 组：l_i ≤ m ≤ r_i，此时 a_i = m  
     * R 组：r_i < m，此时 a_i = r_i  
   - 通过二分查找快速确定每组大小。
3. 合并三组的值并排序，利用前缀和思想计算 S。  
   - 具体地，设排序后的值为 s_1 ≤ s_2 ≤ ... ≤ s_n，则  
     sum_{i<j} |a_i - a_j| = sum_{k=1}^n s_k * (2k - n - 1)  
     S = 2 * sum_{k=1}^n s_k * (2k - n - 1)
4. 在 [0, 10^9] 上三分搜索 m，找到使 S 最小的 m。
5. 输出最小 S。

### 复杂度分析

- 排序：O(n log n)
- 三分搜索：O(log 10^9) ≈ 30 次迭代，每次迭代 O(n) 计算 S
- 总时间复杂度：O(n log n + n log 10^9) ≈ O(n log n)，满足 n ≤ 10^5
- 空间复杂度：O(n)

### 代码说明

C++ 代码使用标准库，避免使用 bits/stdc++.h。通过全局变量存储排序后的 l 和 r，计算函数 compute_cost(m) 返回 S 的值。三分搜索采用整数版本，确保 a_i 为整数。
