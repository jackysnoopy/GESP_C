# [COCI 2016/2017 #4] Rekonstruiraj

## 题目描述
给定闭区间[A,B]和空集合，每次操作选定实数x，将0,x,2x,...中在[A,B]内的加入集合。给定最终集合，求最少操作次数的方案。

## 输入格式
第一行K，第二行A B，接下来K行每行一个实数。

## 输出格式
第一行N（操作次数），接下来N行每行一个实数。

## 样例
### 样例1
输入：
```
4
1 2
1
1.4
1.5
2
```
输出：
```
2
0.5
0.7
```

## 提示
1 ≤ K ≤ 50，1 ≤ A ≤ B ≤ 10^6

## 解题思路

### 问题分析
每次操作生成等差数列0,x,2x,...在[A,B]内的部分。需要覆盖给定集合的所有数。

### 核心思路
贪心：从最大数开始，找能生成它的最大x（即该数本身），然后移除能被该x生成的所有数。

### 算法流程
1. 按降序处理每个数
2. 对未处理的数val，选择x=val
3. 移除所有能被val生成的数（即val的倍数且在[A,B]内）

### 复杂度分析
- 时间复杂度：O(K^2)
- 空间复杂度：O(K)
