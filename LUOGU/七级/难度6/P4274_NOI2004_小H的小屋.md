# [NOI2004] 小H的小屋

## 题目描述
小H的小屋东西长100 Hil，北墙斜率k1，南墙斜率k2。北墙角有m块草坪，南墙角有n块草坪（m≤n）。分点是1到99的整数，北墙分点集合X1是南墙分点集合X2的子集。求草坪最小占地总面积。

## 输入格式
一行四个数 k1, k2, m, n。

## 输出格式
一个实数，精确到小数点后一位。

## 样例
输入：
```
0.5 0.2 2 4
```
输出：
```
3000.0
```

## 提示
2 ≤ m ≤ n ≤ 100，南北墙距离很远不会重叠。

## 解题思路

### 问题分析
将区间[1,99]分成n段（南墙），北墙的分点必须是南墙分点的子集，即北墙的每一段是若干连续南墙段的合并。每段长度为L分成p个北墙子段的代价为 L²×(k2 + k1/p)。

### 核心思路
三维DP：dp[i][j][p]表示覆盖区间[1,i]，使用j个南墙段和p个北墙段的最小面积。

### 算法流程
1. 枚举最后一个南墙段的起始位置k
2. 该段长度为i-k，分配np个北墙段
3. 代价为 (i-k)²×(k2 + k1/np)
4. 取所有方案的最小值

### 复杂度分析
- 时间：O(99 × n × m × 99) ≈ O(10^6)
- 空间：O(99 × n × m)
