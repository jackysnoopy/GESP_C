# [BalticOI 2016] 交换 (Day2)

## 题目描述
给定1..n的排列。进行n-1轮操作，第k轮可交换x[k]和x[⌊k/2⌋]。求字典序最小的序列。

## 输入格式
第一行n，第二行排列。

## 输出格式
一行n个整数。

## 样例
输入：
```
5
3 4 2 5 1
```
输出：
```
2 1 3 4 5
```

## 提示
n ≤ 2×10^5。

## 解题思路

### 问题分析
操作构成二叉树结构（类似堆）。位置i的子树包含所有在操作中可达的位置。

### 核心思路
贪心：从位置1到n，每次选择子树中最小的可用值。使用线段树维护区间最小值。

### 算法流程
1. 构建线段树维护x值
2. 对每个位置i，枚举子树区间查询最小值
3. 选择最小值，标记已使用

### 复杂度分析
- 时间：O(n log²n)
- 空间：O(n)
