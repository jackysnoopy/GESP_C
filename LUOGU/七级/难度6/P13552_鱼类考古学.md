# 鱼类考古学

## 题目描述

给定 n 个非负整数，可以进行两种操作：
1. 选择两个数 x,y 合并为 x AND y
2. 选择两个数 x,y 合并为 x+y

需要使用操作 1 恰好 n-k 次，求最后剩下的数的最大值。

## 输入格式

第一行 T。
每个用例第一行 n, k。
第二行 n 个整数 a_i。

## 输出格式

输出一个整数表示答案。

## 样例

### 样例 1
输入：
```
3
3 2
1 2 3
4 3
2 5 6 4
5 1
1 3 7 9 11
```
输出：
```
3
12
1
```

## 提示

- 1 ≤ n ≤ 10^6
- 0 ≤ a_i < 2^30

## 解题思路

### 问题分析

需要在 AND 和加法操作之间选择，使得最终结果最大。

### 核心思路

1. AND 操作会减少值，加法会增加
2. 需要权衡两种操作的使用次数
3. 贪心选择最优组合

### 算法流程

1. 分析数字的二进制位
2. 贪心选择操作顺序
3. 计算最大值

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        int n, k;
        cin >> n >> k;
        
        vector<int> a(n);
        for (int i = 0; i < n; i++) cin >> a[i];
        
        long long ans = 0;
        
        if (k == 1) {
            ans = a[0];
            for (int i = 1; i < n; i++) ans &= a[i];
        } else {
            sort(a.begin(), a.end(), greater<int>());
            ans = a[0];
            for (int i = 1; i < min(k, n); i++) ans += a[i];
            for (int i = k; i < n; i++) ans &= a[i];
        }
        
        cout << ans << "\n";
    }
    return 0;
}
```
