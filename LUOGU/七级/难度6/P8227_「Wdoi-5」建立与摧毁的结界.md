# 「Wdoi-5」建立与摧毁的结界

## 题目描述

境界可以被抽象为由圆括号组成的括号序列。给定两个长度为 n 的合法括号序列 A 和 B，可以用以下操作对序列 A 进行变换：
- 选择任意非负整数 k，选择括号序列的一个子串满足其为一个 k 阶嵌套括号序列，然后将其变为 k 阶平铺括号序列。
- 选择任意非负整数 k，选择括号序列的一个子串满足其为一个 k 阶平铺括号序列，然后将其变为 k 阶嵌套括号序列。

求将序列 A 变换为序列 B 所需的最少操作数。

## 输入格式

第一行共有一个整数 n，表示 A 与 B 括号序列的长度。
接下来一行，共有 n 个字符，描述括号序列 A。
接下来一行，共有 n 个字符，描述括号序列 B。

## 输出格式

共一行，一个整数，表示将 A 转换为 B 需要的最少步数。

## 样例

### 样例 1
输入：
```
14
((()())(()()))
()()()()()()()
```
输出：
```
6
```

### 样例 2
输入：
```
14
((()())(()()))
(()())(()())()
```
输出：
```
10
```

## 提示

对于 100% 的数据，n≤10^6。

## 解题思路

### 问题分析
需要将一个括号序列变换为另一个，每次操作可以将嵌套括号序列变为平铺括号序列，反之亦然。

### 核心思路
计算每个位置的深度差，最小操作数等于深度差绝对值之和的一半。

### 算法流程
1. 计算两个括号序列每个位置的深度
2. 计算深度差的绝对值之和
3. 除以2得到最小操作数

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    int n;
    scanf("%d", &n);
    
    string A, B;
    cin >> A >> B;
    
    auto getDepth = [](const string& s) {
        vector<int> depth(s.size());
        int cur = 0;
        for (int i = 0; i < (int)s.size(); i++) {
            if (s[i] == '(') {
                cur++;
                depth[i] = cur;
            } else {
                depth[i] = cur;
                cur--;
            }
        }
        return depth;
    };
    
    vector<int> depthA = getDepth(A);
    vector<int> depthB = getDepth(B);
    
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        ans += abs(depthA[i] - depthB[i]);
    }
    
    printf("%lld\n", ans / 2);
    return 0;
}
```
