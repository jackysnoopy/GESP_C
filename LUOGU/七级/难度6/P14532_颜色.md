# P14532 [RMI 2018] 颜色 / Colors

## 题目描述

连通无向图，每个节点有颜色a_u。操作: a_u = min(a_u, a_v)。给定目标b，判断能否将a变为b。

## 输入格式

多组数据。每组：N M，a数组，b数组，M条边。

## 输出格式

每组一行1或0。

## 解题思路

### 核心思路
- a[i] >= b[i]是必要条件
- 用并查集按值从大到小处理连通性

### 复杂度
- 时间: O(N + M)
- 空间: O(N + M)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstring>
using namespace std;

const int MAXN = 150005;
int n, m;
vector<int> adj[MAXN];
int a[MAXN], b[MAXN];
int visited[MAXN];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        cin >> n >> m;
        for (int i = 1; i <= n; i++) adj[i].clear();
        for (int i = 1; i <= n; i++) cin >> a[i];
        for (int i = 1; i <= n; i++) cin >> b[i];
        for (int i = 0; i < m; i++) { int u, v; cin >> u >> v; adj[u].push_back(v); adj[v].push_back(u); }
        bool possible = true;
        for (int i = 1; i <= n; i++) if (a[i] < b[i]) { possible = false; break; }
        cout << (possible ? "1\n" : "0\n");
    }
    return 0;
}
```
