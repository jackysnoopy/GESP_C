# [KSN2021] Delivering Balls

## 题目描述
给定序列H，从(S,H[S])到(T,H[T])，移动消耗：行-1为1体力，行不变为2体力，行+1为4体力。求最小体力。

## 输入格式
第一行N，第二行H，第三行Q，接下来Q行每行Si Ti。

## 输出格式
每行一个最小体力。

## 样例
### 样例1
输入：
```
4
9 1 8 2
2
1 3
4 2
```
输出：
```
6
31
```

## 提示
2 ≤ N ≤ 2×10^5，Q ≤ 2×10^5

## 解题思路

### 问题分析
在约束条件下找最小体力路径。

### 核心思路
贪心：尽量走水平（2体力），必要时爬升（4体力），尽量下坡（1体力）。

### 算法流程
1. 对每个查询，沿路径计算体力
2. 水平移动2体力，爬升4×高度差，下坡1×高度差

### 复杂度分析
- 时间复杂度：O(Q×N)
- 空间复杂度：O(N)
