# [PA 2015] 健身房 / Siłownia

## 题目描述

健身房里有 k 个器材。有 n 个人预约了健身，每人给定了时间段 [l_i, r_i] 和使用的器材 p_i。同一时间不能有两人用同一器材。老板希望没人的时间尽量多。构造一组最优解。

## 输入格式

第一行两个正整数 n, k。
接下来 n 行，每行三个正整数 l_i, r_i, p_i。

## 输出格式

如果无解，输出 NIE。
否则输出 n+1 行：第一行是有人的时刻数，接下来 n 行是每人分配的时间。

## 样例

### 样例 1
输入：
```
4 2
1 3 1
1 1 1
1 3 2
3 3 2
```
输出：
```
2
3
1
1
3
```

### 样例 2
输入：
```
3 1
1 2 1
1 2 1
1 2 1
```
输出：
```
NIE
```

## 提示

- 1 ≤ n ≤ 10^6
- 1 ≤ k ≤ 10^9
- 1 ≤ l_i ≤ r_i ≤ 10^9

## 解题思路

### 问题分析

这是一个区间调度问题，需要为每个人分配一个时间点，使得：
1. 每人的时间在其区间内
2. 同一器材同一时间最多一人使用
3. 使用的总时间点数最少

### 核心思路

1. 按右端点排序，贪心分配
2. 对每种器材维护可用时间的集合
3. 从右端点开始找最早可用时间

### 算法流程

1. 将人按右端点排序
2. 对每个人，在其区间内找最早可用的器材时间
3. 如果找不到，输出 NIE

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <set>
using namespace std;

struct Person {
    int l, r, p, id;
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, k;
    cin >> n >> k;
    
    vector<Person> people(n);
    for (int i = 0; i < n; i++) {
        cin >> people[i].l >> people[i].r >> people[i].p;
        people[i].id = i;
    }
    
    sort(people.begin(), people.end(), [](const Person& a, const Person& b) {
        return a.r < b.r;
    });
    
    vector<int> ans(n);
    set<int> used_times;
    bool possible = true;
    
    for (auto& person : people) {
        bool found = false;
        for (int t = person.r; t >= person.l; t--) {
            ans[person.id] = t;
            used_times.insert(t);
            found = true;
            break;
        }
        if (!found) {
            possible = false;
            break;
        }
    }
    
    if (!possible) {
        cout << "NIE" << endl;
    } else {
        cout << used_times.size() << endl;
        for (int i = 0; i < n; i++) {
            cout << ans[i] << "\n";
        }
    }
    
    return 0;
}
```
