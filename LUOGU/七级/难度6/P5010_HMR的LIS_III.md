# HMR的LIS Ⅲ

## 题目描述
求序列的IBvl序列（连续元素差值在(L,R)之间的最长子序列）长度，以及字典序第k小的位置序列。

## 输入格式
第一行n,k,L,R，第二行n个整数。

## 输出格式
第一行长度，第二行位置序列。

## 样例
输入：
```
5 3 2 4
6 8 0 2 7
```
输出：
```
1
3
```

## 提示
n ≤ 5×10^5, k ≤ 10^13。

## 解题思路

### 问题分析
带约束的LIS变体：相邻元素差值在(L,R)之间。

### 核心思路
DP + 线段树优化。dp[i] = max(dp[j]+1) for j满足L < a[i]-a[j] < R。字典序第k小需要回溯。

### 算法流程
1. 坐标压缩a值
2. 线段树维护dp值
3. 对每个i，查询区间(a[i]-R, a[i]-L)的最大dp
4. 回溯找到第k小序列

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
