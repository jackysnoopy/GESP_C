# [POI 2018 R2] 诗集 Book of poetry

## 题目描述

n 首诗要印刷成诗集，每页 s 行。每首诗有标题（1行）和正文（a_i 行）。标题不能在最后一行。求最少空行数和对应排列。

## 输入格式

第一行 n, s。
第二行 n 个整数 a_1, a_2, ..., a_n。

## 输出格式

第一行一个整数 k（最少空行数）。
第二行 n 个整数（排列）。

## 样例

### 样例 1
输入：
```
3 5
2 5 1
```
输出：
```
0
2 3 1
```

## 提示

- 1 ≤ n ≤ 500000
- 2 ≤ s ≤ 1000000
- 1 ≤ a_i ≤ 1000000

## 解题思路

### 问题分析

需要将诗作排列，使得每页的空行数最少。标题不能在最后一行，所以如果前一首诗结束于倒数第二行，最后一行需留空。

### 核心思路

1. 计算每首诗占用的行数：1（标题）+ a_i（正文）
2. 贪心排列，尽量填满每页
3. 考虑标题位置约束

### 算法流程

1. 按行数排序
2. 贪心分配到各页
3. 计算空行数

### 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
