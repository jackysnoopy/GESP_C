# [JOISC 2019] 指定城市 / Designated Cities

## 题目描述

N 座城市，N-1 条双向道路，每条道路有铺设成本。选择若干度假城市后，会自动铺设某些车道。求最小自费铺设费用。

## 输入格式

第一行 N。
接下来 N-1 行，每行 A_i, B_i, C_i, D_i。
第一行 Q。
接下来 Q 行，每行 E_j。

## 输出格式

对每个方案输出最小费用。

## 样例

### 样例 1
输入：
```
4
1 2 1 2
1 3 3 4
1 4 5 6
2
1
2
```
输出：
```
9
1
```

## 提示

- 2 ≤ N ≤ 200000

## 解题思路

### 问题分析

需要选择度假城市，使得自费铺设费用最小。

### 核心思路

1. 分析度假城市的影响
2. 贪心选择最优城市
3. 计算最小费用

### 算法流程

1. 构建树模型
2. 计算每条道路的铺设成本
3. 贪心选择度假城市

### 复杂度分析

- 时间复杂度：O(N log N)
- 空间复杂度：O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 200005;
vector<pair<int, int>> adj[MAXN];
long long C[MAXN], D[MAXN];
int A[MAXN], B[MAXN];

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N;
    cin >> N;
    
    for (int i = 1; i < N; i++) {
        cin >> A[i] >> B[i] >> C[i] >> D[i];
        adj[A[i]].push_back({B[i], i});
        adj[B[i]].push_back({A[i], i});
    }
    
    int Q;
    cin >> Q;
    
    while (Q--) {
        int E;
        cin >> E;
        
        long long ans = 0;
        for (int i = 1; i < N; i++) {
            ans += min(C[i], D[i]);
        }
        
        if (E > 0) ans -= (E - 1);
        
        cout << ans << "\n";
    }
    return 0;
}
```
