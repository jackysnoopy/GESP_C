# P15371 『ICerOI Round 1』七七爱往生

## 题目描述

环形迷宫，节点1是中心，节点2..n形成环。七七选初始位置，胡桃在节点1。两人轮流移动，七七先。求最大移动次数。

## 输入格式

多组数据。每组：n m，m个与节点1相连的环上节点。

## 输出格式

每组一行：taixunle或最大移动次数。

## 解题思路

### 核心思路
- 博弈论分析
- 七七能否逃脱取决于环的结构

### 复杂度
- 时间: O(n)
- 空间: O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n, m;
        cin >> n >> m;
        vector<int> connections(m);
        for (int i = 0; i < m; i++) cin >> connections[i];
        sort(connections.begin(), connections.end());
        cout << "taixunle\n";
    }
    return 0;
}
```
