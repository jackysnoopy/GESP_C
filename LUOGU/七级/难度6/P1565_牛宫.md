# P1565 牛宫

## 题目描述
AP 神牛准备给自己盖一座很华丽的矩形宫殿。他看中了一块 N×M 的矩形空地，每个格子有海拔高度。宫殿的平均海拔必须在海平面之上（平均数 > 0），求最大宫殿面积。

## 输入格式
第一行为 N 和 M，之后 N 行每行 M 个数描述海拔。

## 输出格式
输出宫殿最大面积。

## 样例
输入：
```
3 2
4 0
-10 8
-2 -2
```
输出：
```
4
```

## 解题思路

### 问题分析
在 N×M 的矩阵中找一个子矩阵，使得其元素平均值 > 0（即总和 > 0），且面积最大。

### 核心思路
固定上下两行 i 和 j，计算每列在 [i, j] 区间内的列和。问题转化为：在列和数组中找最长的连续子数组，使得子数组和 > 0。

### 算法流程
1. 预处理列前缀和
2. 枚举上下边界 i, j（O(N²)）
3. 对于每对 (i, j)，计算列和数组
4. 使用前缀和 + 单调栈找最长正和子数组（O(M)）
5. 更新答案

### 复杂度分析
- 时间复杂度：O(N² × M)
- 空间复杂度：O(N × M)

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<vector<long long>> a(n + 1, vector<long long>(m + 1));
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= m; j++)
            cin >> a[i][j];
    
    vector<vector<long long>> colSum(n + 1, vector<long long>(m + 1, 0));
    for (int j = 1; j <= m; j++)
        for (int i = 1; i <= n; i++)
            colSum[i][j] = colSum[i - 1][j] + a[i][j];
    
    long long ans = 0;
    
    for (int i = 1; i <= n; i++) {
        for (int j = i; j <= n; j++) {
            vector<long long> prefix(m + 1, 0);
            for (int k = 1; k <= m; k++)
                prefix[k] = prefix[k - 1] + (colSum[j][k] - colSum[i - 1][k]);
            
            vector<int> stk;
            for (int k = 0; k <= m; k++) {
                if (stk.empty() || prefix[k] < prefix[stk.back()])
                    stk.push_back(k);
            }
            
            for (int k = m; k >= 1; k--) {
                while (stk.size() > 1 && prefix[stk[stk.size() - 2]] < prefix[k])
                    stk.pop_back();
                if (!stk.empty() && prefix[k] > prefix[stk.back()]) {
                    long long area = (long long)(j - i + 1) * (k - stk.back());
                    ans = max(ans, area);
                }
            }
        }
    }
    
    cout << ans << endl;
    return 0;
}
```
