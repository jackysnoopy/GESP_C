# [RC-05] 排列

## 题目描述
给定1~n的排列p，恰好K次交换相邻元素，求字典序最小的排列。

## 输入格式
第一行n K，第二行排列p。

## 输出格式
交换后的排列。

## 样例
### 样例1
输入：
```
5 2
2 1 4 3 5
```
输出：
```
1 2 3 4 5
```

## 提示
1 ≤ n ≤ 5×10^5，0 ≤ K ≤ 10^12

## 解题思路

### 问题分析
每次交换相邻元素，K次交换后字典序最小。

### 核心思路
贪心：对于每个位置，尝试将最小的可用元素移到该位置，代价为移动距离。

### 算法流程
1. 从前往后处理每个位置
2. 找最小的可用元素，计算移动代价
3. 代价不超过剩余次数则移动

### 复杂度分析
- 时间复杂度：O(n^2)
- 空间复杂度：O(n)
