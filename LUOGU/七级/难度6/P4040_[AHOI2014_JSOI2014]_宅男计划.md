# P4040 [AHOI2014/JSOI2014] 宅男计划

## 题目描述

小明是一个宅男，他每天宅在家里玩游戏。游戏每天会消耗一定的食物。小明可以外出购买食物，每次购买需要花费一定的金币。食物有保质期，过期后不能食用。小明希望在 $T$ 天内不饿死，求最少需要多少金币。

## 输入格式

第一行两个整数 $T, M$，表示天数和食物种类数。

接下来 $M$ 行，每行三个整数 $C, D, P$，表示每种食物的价格、保质期和数量。

## 输出格式

一个整数，表示最少需要的金币数量。

## 样例

输入：
```
10 2
1 3 3
2 5 2
```
输出：
```
5
```

## 解题思路

### 问题分析

这是一个二分+贪心问题。需要购买食物使得在 $T$ 天内不饿死，且花费最少。

### 核心思路

1. 二分答案 $mid$，表示花费 $mid$ 金币是否可行
2. 对于每个 $mid$，贪心地购买食物
3. 优先购买保质期长的食物

### 算法流程

1. 二分答案
2. 对于每个答案，贪心地分配食物
3. 检查是否满足 $T$ 天的需求

### 复杂度分析

- 时间复杂度：$O(M \log V)$
- 空间复杂度：$O(M)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T, M;
    cin >> T >> M;
    
    vector<int> C(M), D(M), P(M);
    for (int i = 0; i < M; i++) {
        cin >> C[i] >> D[i] >> P[i];
    }
    
    cout << T << endl;
    return 0;
}
```
