# [ONTAK2015] Tasowanie

## 题目描述
给定两个数字串A和B，二路归并得到字典序最小的新串T。

## 输入格式
第一行n，第二行A，第三行m，第四行B。

## 输出格式
n+m个整数，字典序最小的T。

## 样例
### 样例1
输入：
```
6
1 2 3 1 2 4
7
1 2 2 1 3 4 3
```
输出：
```
1 1 2 2 1 2 3 1 2 3 4 3 4
```

## 提示
1 ≤ n,m ≤ 2×10^5

## 解题思路

### 问题分析
将两个串归并，保持相对顺序，字典序最小。

### 核心思路
贪心：每次选剩余串中字典序较小的第一个元素。相等时比较剩余部分。

### 算法流程
1. 比较两个串当前首元素
2. 选较小的；相等时比较剩余部分
3. 处理剩余元素

### 复杂度分析
- 时间复杂度：O(n+m)
- 空间复杂度：O(n+m)
