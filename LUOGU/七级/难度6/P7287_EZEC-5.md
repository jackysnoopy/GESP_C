# 「EZEC-5」魔法

## 题目描述
小明有两种魔法：花费a将区间+1，花费b将区间×2。求使存在子区间和≥s的最小魔法值。

## 输入格式
第一行n,a,b,s，第二行n个整数。

## 输出格式
一个整数，最小花费。

## 样例
### 样例1
输入：
```
5 2 3 102
-3 -1 1 -2 0
```
输出：
```
17
```

## 提示
1 ≤ n ≤ 10^5，1 ≤ a,b ≤ 10^9

## 解题思路

### 问题分析
需要找到使最大子段和≥s的最小代价。加法每次+1，乘法每次×2。

### 核心思路
枚举使用k次乘法（0到31次），对于每次乘法计算需要多少次加法使子段和满足条件。

### 算法流程
1. 计算初始最大子段和
2. 枚举k次乘法，代价k×b
3. 计算需要的加法次数：ceil(s/2^k)-maxSub
4. 取最小总代价

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
