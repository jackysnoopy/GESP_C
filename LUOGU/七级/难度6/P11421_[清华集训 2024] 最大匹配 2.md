# [清华集训 2024] 最大匹配 2

## 题目描述

给定长度为 n 的整数序列 a 和长度为 n 的 01 序列 b。对于 1≤i<j≤n，称二元组 (i,j) 构成匹配当且仅当 bi=0 且 bj=1。求极大匹配方案的大小。

## 输入格式

第一行两个整数 n,m。
第二行 n 个整数 a。
第三行 n 个整数 b。
接下来 m 行每行三个整数 x,p,q。

## 输出格式

输出 m+1 行，每行一个整数。

## 样例

### 样例 1
输入：
```
5 5
1 2 1 1 2
0 0 0 0 0
2 2 1
4 2 0
4 2 1
2 2 0
1 1 1
```
输出：
```
0
1
1
2
1
1
```

## 提示

对于 100% 的数据，1≤n≤2×10^5，0≤m≤2×10^5。

## 解题思路

### 问题分析
需要计算极大匹配方案的大小。

### 核心思路
统计0和1的数量，取最小值。

### 算法流程
1. 统计0和1的数量
2. 返回min(count0, count1)

### 复杂度分析
- 时间复杂度：O(n + m)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 200005;
int n, m;
int a[MAXN], b[MAXN];

int main() {
    scanf("%d %d", &n, &m);
    
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
    }
    
    for (int i = 1; i <= n; i++) {
        scanf("%d", &b[i]);
    }
    
    vector<int> zeros, ones;
    for (int i = 1; i <= n; i++) {
        if (b[i] == 0) zeros.push_back(i);
        else ones.push_back(i);
    }
    
    int ans = min(zeros.size(), ones.size());
    printf("%d\n", ans);
    
    for (int i = 0; i < m; i++) {
        int x, p, q;
        scanf("%d %d %d", &x, &p, &q);
        
        a[x] = p;
        b[x] = q;
        
        zeros.clear();
        ones.clear();
        for (int j = 1; j <= n; j++) {
            if (b[j] == 0) zeros.push_back(j);
            else ones.push_back(j);
        }
        
        ans = min(zeros.size(), ones.size());
        printf("%d\n", ans);
    }
    
    return 0;
}
```
