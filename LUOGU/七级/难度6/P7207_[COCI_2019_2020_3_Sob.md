# [COCI 2019/2020 #3] Sob

## 题目描述
给定两个正整数N,M，从A={0,1,...,N-1}和B={M,...,M+N-1}中各选N个数配对(x,y)，使得x&y=x，所有x互不相同，所有y互不相同。

## 输入格式
两个整数N,M。

## 输出格式
N行，每行两个整数xi yi。

## 样例
### 样例1
输入：
```
1 3
```
输出：
```
0 3
```

### 样例2
输入：
```
3 5
```
输出：
```
0 7
1 5
2 6
```

## 提示
1 ≤ N ≤ M，N+M ≤ 10^6

## 解题思路

### 问题分析
需要找到N对(x,y)满足x&y=x，即y的二进制中x为1的位y也必须为1。

### 核心思路
贪心：按x的popcount降序处理，优先匹配约束强的x。对于每个x，从可用的y中找到第一个满足条件的。

### 算法流程
1. 按popcount降序排序x
2. 对每个x，在可用y中找到第一个满足(x&y)==x的
3. 标记y已使用

### 复杂度分析
- 时间复杂度：O(N^2)
- 空间复杂度：O(N)
