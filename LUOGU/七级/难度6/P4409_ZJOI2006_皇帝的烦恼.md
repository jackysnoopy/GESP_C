# [ZJOI2006] 皇帝的烦恼

## 题目描述
n个将军围成一圈，第i个将军要求a_i种不同颜色勋章。相邻将军不能有相同颜色勋章。求最少需要多少种勋章。

## 输入格式
第一行整数n，接下来n行每行一个整数a_i。

## 输出格式
一个整数。

## 样例
输入：
```
4
2
2
1
1
```
输出：
```
4
```

## 提示
n ≤ 2×10^4, a_i ≤ 10^5。

## 解题思路

### 问题分析
环形排列，相邻不能共享颜色。每种颜色最多可以分配给⌊n/2⌋个不相邻的将军。

### 核心思路
二分答案C。下界为max(max(a[i]), ⌈Σa[i]/⌊n/2⌋⌉)。用贪心/DP验证。

### 算法流程
1. 二分答案C
2. 对于给定C，验证是否可行
3. 考虑相邻将军的重叠约束

### 复杂度分析
- 时间：O(n log Σa[i])
- 空间：O(n)
