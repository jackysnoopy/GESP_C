# [LNOI2022] 吃

## 题目描述
小 A 很喜欢吃东西。

小 A 面前有 $n$ 份食物，第 $i$ 份有参数 $a_i$ 和 $b_i$。小 A 可以按照任意顺序吃掉这 $n$ 份食物。当她吃掉编号为 $i$ 的食物时，她可以选择将自己的体重乘以 $a_i$ 或者将自己的体重加上 $b_i$。每份食物只能吃恰好一次。

小 A 的初始体重为 $1$，请求出她吃完 $n$ 份食物后能达到的最大体重。答案可能很大，你只需要输出其对 $(10^9 + 7)$ 取模后的结果。

## 输入格式
第一行输入一个整数 $n$ 表示食物的数量。第二行 $n$ 个整数 $a_1, a_2, \ldots, a_n$，第三行 $n$ 个整数 $b_1, b_2, \ldots, b_n$，表示每份食物的参数。

## 输出格式
输出一个整数，表示小 A 可以得到的最大体重对 $(10^9 + 7)$ 取模后的结果。

## 样例
### 样例1
输入:
```
5
1 2 3 4 5
100 200 300 400 500
```
输出:
```
18060
```

## 提示
- 时间限制：1000ms
- 内存限制：262144KB

## 解题思路

### 问题分析
需要决定每份食物是乘还是加，使得最终体重最大。

### 核心思路
1. 贪心策略：优先乘大的数
2. 考虑加法和乘法的权衡

### 算法流程
1. 按 $a_i$ 降序排序
2. 对于每份食物，选择乘或加
3. 取模输出

### 复杂度分析
- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(n)$

## 参考代码
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MOD = 1e9 + 7;
const int MAXN = 500005;
int a[MAXN], b[MAXN];
int n;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n;
    
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    for (int i = 0; i < n; i++) {
        cin >> b[i];
    }
    
    vector<int> idx(n);
    for (int i = 0; i < n; i++) idx[i] = i;
    sort(idx.begin(), idx.end(), [&](int x, int y) {
        return a[x] > a[y];
    });
    
    long long ans = 1;
    long long sum = 0;
    
    for (int i = 0; i < n; i++) {
        int id = idx[i];
        if (a[id] > 1 && sum * (a[id] - 1) > b[id]) {
            ans = ans * a[id] % MOD;
        } else {
            sum += b[id];
            ans = (ans + b[id]) % MOD;
        }
    }
    
    cout << ans << endl;
    
    return 0;
}
```
