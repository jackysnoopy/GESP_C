# P14462 【MX-S10-T3】寻宝游戏

## 题目描述

n+10^6个桶，前n个有a_i个泡面，后10^6个空。每次选3个不同桶i,j,k，从i,j各拿1个放入k。求最少操作次数使所有泡面到一个桶。

## 输入格式

多组数据。每组：n q，a数组，q个查询[l,r]。

## 输出格式

每组每个查询一行答案。

## 解题思路

### 核心思路
- 总泡面数S，非空桶数c
- 答案 = S - max(1, ceil(c/2))

### 复杂度
- 时间: O(n + q)
- 空间: O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T;
    cin >> T;
    while (T--) {
        int n, q;
        cin >> n >> q;
        vector<long long> a(n);
        for (int i = 0; i < n; i++) cin >> a[i];
        vector<long long> prefixSum(n + 1, 0);
        vector<int> prefixNonZero(n + 1, 0);
        for (int i = 0; i < n; i++) {
            prefixSum[i + 1] = prefixSum[i] + a[i];
            prefixNonZero[i + 1] = prefixNonZero[i] + (a[i] > 0 ? 1 : 0);
        }
        while (q--) {
            int l, r;
            cin >> l >> r;
            l--; r--;
            long long S = prefixSum[r + 1] - prefixSum[l];
            int c = prefixNonZero[r + 1] - prefixNonZero[l];
            if (S == 0) cout << "0\n";
            else if (c == 1) cout << "0\n";
            else if (S == 1) cout << "-1\n";
            else cout << S - max(1LL, (long long)(c + 1) / 2) << "\n";
        }
    }
    return 0;
}
```
