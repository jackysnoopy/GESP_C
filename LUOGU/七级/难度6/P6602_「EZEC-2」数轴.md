# 「EZEC-2」数轴

## 题目描述
n次操作，每次在x_i增加a_i个标记，然后选区间[l,r]使标记数<=k，求最大r-l。

## 输入格式
第一行n m k，接下来n行每行x_i a_i。

## 输出格式
n行答案。

## 样例
输入：
```
5 4 0
2 1
3 1
0 1
1 1
4 1
```
输出：
```
1
1
0
0
-1
```

## 解题思路

### 问题分析
- k<=100且a_i<=100，标记总数有限
- 用map维护标记分布

### 核心思路
- 维护前缀和，找到满足条件的最大区间

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码
```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m, k;
    cin >> n >> m >> k;
    
    map<int, int> marks;
    long long total = 0;
    
    for (int i = 0; i < n; i++) {
        int x, a;
        cin >> x >> a;
        marks[x] += a;
        total += a;
        
        if (total <= k) {
            cout << m << "\n";
        } else {
            int left = 0;
            long long cur = total;
            
            for (auto it = marks.begin(); it != marks.end(); ++it) {
                if (cur - it->second > k) {
                    cur -= it->second;
                    left = it->first + 1;
                } else {
                    break;
                }
            }
            
            cout << m - left << "\n";
        }
    }
    
    return 0;
}
```
