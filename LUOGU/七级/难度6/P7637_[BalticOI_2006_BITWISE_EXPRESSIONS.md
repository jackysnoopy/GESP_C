# [BalticOI 2006] BITWISE EXPRESSIONS

## 题目描述
给定表达式由P个子表达式按位AND连接，每个子表达式内变量按位OR。每个变量有取值范围[Aj,Bj]，求表达式最大值。

## 输入格式
第一行N P，第二行P个子表达式变量数，接下来N行每行Aj Bj。

## 输出格式
一个整数，表达式最大值。

## 样例
### 样例1
输入：
```
8 4
3 1 2 2
2 4
1 4
0 0
1 7
1 4
1 2
3 4
2 3
```
输出：
```
6
```

## 提示
1 ≤ N ≤ 100，0 ≤ Aj ≤ Bj ≤ 2×10^9

## 解题思路

### 问题分析
表达式 = (OR of group1) & (OR of group2) & ... & (OR of groupP)

### 核心思路
从高位到低位贪心，尝试将每一位设为1。检查每个子表达式是否能在该位上为1。

### 算法流程
1. 从高位到低位枚举
2. 对每位，检查每个子表达式是否有变量能在该位为1
3. 所有子表达式都满足则该位可以设为1

### 复杂度分析
- 时间复杂度：O(N × 31)
- 空间复杂度：O(N)
