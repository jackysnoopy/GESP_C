# P3474 [POI 2008] KUP-Plot purchase

## 题目描述

Byteasar打算购买一块工业用地。

他的财产估计为 $k$ 币盾，这正是他想在地块上花费的金额。

然而，找到一块正好价值 $k$ 币盾的地块并非易事。

因此，Byteasar准备购买更贵的地块。

他考虑贷款。Byteotian信贷银行将授予他最多 $k$ 币盾的贷款。因此，Byteasar在地块上最多可以花费 $2k$ 币盾，而且他希望花费不少于 $k$ 币盾。

Byteasar想购买的地块区域是一个边长为 $n$ 米的正方形。当前的地主们为每平方米设定了不同的价格。Byteasar已经与他们每个人交谈过，然后准备了该区域的价格地图。地图描绘了每个1米×1米正方形的价格。显然，有 $n^2$ 个这样的正方形。

现在是找到梦想地块的时候了。它必须是矩形的，由完整的单位正方形组成。

任务：编写一个程序，读取 $k$ 和 $n$ 以及区域的价格地图，确定一个价格在区间 $[k, 2k]$ 内的地块，或者说明不存在这样的地块。

## 输入格式

第一行包含两个整数 $k$ 和 $n$，用空格分隔，$1 \le k \le 10^9$，$1 \le n \le 2000$。

接下来的 $n$ 行，每行包含 $n$ 个非负整数，用空格分隔。

第 $j+1$ 行的第 $i$ 个数表示坐标 $(i, j)$ 处单位正方形的价格。

每平方米的价格不超过 $2\,000\,000\,000$ 币盾。

## 输出格式

如果不存在价格在区间 $[k, 2k]$ 内的地块，程序应输出一行 `NIE`（波兰语"否"）。

否则，应输出一行四个正整数 $x_1, y_1, x_2, y_2$，用空格分隔，表示矩形的坐标。

$(x_1, y_1)$ 表示矩形左上角，$(x_2, y_2)$ 表示矩形右下角。

矩形由满足 $x_1 \le x \le x_2, y_1 \le y \le y_2$ 的正方形 $\{x, y\}$ 组成。

矩形中所有正方形的价格之和 $c$ 应满足 $k \le c \le 2k$。

如果多个矩形地块满足此条件，任选一个即可。

## 样例

输入：
```
8 4
1 2 1 3
25 1 2 1
4 20 3 3
3 30 12 2
```
输出：
```
2 1 4 2
```

## 解题思路

### 问题分析

这是一个二维前缀和问题，需要在 $n \times n$ 的矩阵中找到一个子矩阵，使其元素和在 $[k, 2k]$ 范围内。

### 核心思路

1. 先检查单个格子是否在 $[k, 2k]$ 范围内
2. 如果没有，使用二维前缀和 + 双指针寻找满足条件的矩形
3. 对于每个可能的矩形行范围，使用双指针在列方向上滑动

### 算法流程

1. 计算二维前缀和
2. 枚举矩形的上下边界 $(i, j)$
3. 对于固定的行范围，使用双指针在列方向上寻找满足条件的矩形
4. 如果找到，输出坐标；否则输出 "NIE"

### 复杂度分析

- 时间复杂度：$O(n^3)$
- 空间复杂度：$O(n^2)$

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;
int main() {
    ios::sync_with_stdio(false); cin.tie(nullptr);
    ll k;
    int n;
    cin >> k >> n;
    vector<vector<ll>> a(n + 1, vector<ll>(n + 1));
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            cin >> a[i][j];
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            a[i][j] += a[i-1][j] + a[i][j-1] - a[i-1][j-1];
    for (int i = 1; i <= n; i++) {
        for (int j = i; j <= n; j++) {
            int l = 0;
            for (int r = 1; r <= n; r++) {
                while (l < r) {
                    ll sum = a[j][r] - a[j][l] - a[i-1][r] + a[i-1][l];
                    if (sum > 2 * k) l++;
                    else break;
                }
                ll sum = a[j][r] - a[j][l] - a[i-1][r] + a[i-1][l];
                if (sum >= k && sum <= 2 * k) {
                    cout << i << " " << l + 1 << " " << j << " " << r << endl;
                    return 0;
                }
            }
        }
    }
    cout << "NIE" << endl;
    return 0;
}
```
