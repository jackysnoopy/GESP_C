# P3002 瓶颈路

## 思路

核心算法：图论

## 代码

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    string source = "";
    string target = "";
    
    // Read source string
    string line;
    getline(cin, line); // consume newline
    while ((int)source.size() < n) {
        getline(cin, line);
        source += line;
    }
    
    // Read target string
    while ((int)target.size() < m) {
        getline(cin, line);
        target += line;
    }
    
    // Greedy: find longest prefix of target that appears in source
    int cuts = 0;
    int target_pos = 0;
    
    while (target_pos < m) {
        // Find the longest substring starting at target_pos that appears in source
        int max_len = 0;
        int best_start = -1;
        
        for (int i = 0; i < n; i++) {
            int len = 0;
            while (i + len < n && target_pos + len < m && source[i + len] == target[target_pos + len]) {
                len++;
            }
            if (len > max_len) {
                max_len = len;
                best_start = i;
            }
        }
        
        if (max_len == 0) {
            // Need to cut a single character
            cuts++;
            target_pos++;
        } else {
            // Use the longest match
            cuts++;
            target_pos += max_len;
        }
    }
    
    cout << cuts << "\n";
    return 0;
}
```

## 解题思路

### 问题分析

给定源字符串 source（长度 n）和目标字符串 target（长度 m），每次操作可以从 source 中切出一个子串拼接到目标串末尾。求最少操作次数使目标串完全由 source 的子串拼接而成。

### 核心思路

贪心算法。每次从 target 当前位置开始，在 source 中寻找能匹配的最长子串，用该子串覆盖尽可能多的 target 字符，直到 target 被完全覆盖。

### 算法流程

1. 初始化 target_pos = 0，cuts = 0
2. 当 target_pos < m 时循环：
   - 枚举 source 的每个起始位置 i
   - 从 i 和 target_pos 开始同时向后匹配，记录最大匹配长度 max_len
   - 若 max_len > 0：cuts++，target_pos += max_len
   - 若 max_len = 0：cuts++，target_pos++（切单个字符）
3. 输出 cuts

### 复杂度分析

- 时间复杂度：O(n×m)，最坏情况每个 target 字符都要匹配整个 source
- 空间复杂度：O(n+m)
