# 「ROI 2025 Day1」索契游乐园

## 题目描述

沿着一条直线摆放了 n 个目标，坐标为 x_i。从 x_0 进入，小球补给点在 x_0 + kd 处。击中目标消耗 (x - x_i)^2 能量。有 m 位选手，第 j 位移动距离 d 消耗 t_j 能量。求每位选手的最少能量。

## 输入格式

第一行 n。
第二行 n 个整数 x_i。
第三行 x_0 和 d。
第四行 m。
接下来 m 行，每行一个整数 t_j。

## 输出格式

对每位选手输出最少能量。

## 样例

### 样例 1
输入：
```
3
4 0 7
2 3
7
0
1
2
3
4
23
25
```
输出：
```
3
7
10
12
13
32
33
```

## 提示

- 1 ≤ n ≤ 3×10^5
- 1 ≤ m ≤ 6×10^5
- 0 ≤ t_j ≤ 10^8

## 解题思路

### 问题分析

需要为每个选手找到最优的移动和投掷策略，最小化总能量消耗。

### 核心思路

1. 对每个目标，找最近的补给点投掷
2. 选手移动成本与投掷成本的权衡
3. 贪心或 DP 求解

### 算法流程

1. 对每个选手，计算最优策略
2. 选择最近的补给点
3. 最小化 (投掷距离)^2 + 移动成本

### 复杂度分析

- 时间复杂度：O(n * m)
- 空间复杂度：O(n + m)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <cmath>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<long long> x(n);
    for (int i = 0; i < n; i++) cin >> x[i];
    
    long long x0, d;
    cin >> x0 >> d;
    
    int m;
    cin >> m;
    
    for (int j = 0; j < m; j++) {
        long long t;
        cin >> t;
        
        long long ans = 0;
        for (int i = 0; i < n; i++) {
            // 找最近的补给点
            long long best = (long long)1e18;
            for (long long k = -3; k <= 3; k++) {
                long long supply = x0 + k * d;
                long long throw_cost = (x[i] - supply) * (x[i] - supply);
                long long move_cost = abs(k) * t;
                best = min(best, throw_cost + move_cost);
            }
            ans += best;
        }
        
        cout << ans << "\n";
    }
    
    return 0;
}
```
