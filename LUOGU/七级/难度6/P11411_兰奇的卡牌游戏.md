# 兰奇的卡牌游戏

## 题目描述

给定 n 张卡牌，第 i 张卡牌编号为 i，其权值为 ai。玩家希望删去一些卡牌，使得正式游戏阶段的最大得分不超过 k。求最少要删除几张卡片。

## 输入格式

第一行输入两个正整数 n 和 k。
第二行输入 n 个元素，第 i 个元素 ai。

## 输出格式

输出一个正整数，表示最少要删除几张卡片。

## 样例

### 样例 1
输入：
```
3 2
1 2 3
```
输出：
```
0
```

### 样例 2
输入：
```
6 2
3 2 7 99 10 15
```
输出：
```
3
```

## 提示

对于 100% 的数据，1≤n≤2000，1≤ai≤10^9。

## 解题思路

### 问题分析
需要删除最少的卡牌，使得最长递增子序列长度不超过k。

### 核心思路
计算最长递增子序列长度，如果超过k则需要删除多余的部分。

### 算法流程
1. 计算最长递增子序列长度
2. 如果长度超过k，返回长度-k

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 2005;
int n, k;
int a[MAXN];

int main() {
    scanf("%d %d", &n, &k);
    for (int i = 1; i <= n; i++) {
        scanf("%d", &a[i]);
    }
    
    vector<int> lis;
    for (int i = 1; i <= n; i++) {
        auto it = lower_bound(lis.begin(), lis.end(), a[i]);
        if (it == lis.end()) {
            lis.push_back(a[i]);
        } else {
            *it = a[i];
        }
    }
    
    int lis_len = lis.size();
    
    if (lis_len <= k) {
        printf("0\n");
    } else {
        printf("%d\n", lis_len - k);
    }
    
    return 0;
}
```
