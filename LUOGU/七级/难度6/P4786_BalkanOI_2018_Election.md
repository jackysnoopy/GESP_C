# [BalkanOI 2018] Election

## 题目描述
字符串S仅由C和T组成。Q次查询[L,R]，求S[L..R]中至少删除多少字符，使得每个前缀和后缀中C的数量≥T的数量。

## 输入格式
第一行N，第二行字符串S，第三行Q，接下来Q行每行L,R。

## 输出格式
每行一个整数。

## 样例
输入：
```
11
CCCTTTTTTCC
3
1 11
4 9
1 6
```
输出：
```
4
6
3
```

## 提示
N, Q ≤ 5×10^5。

## 解题思路

### 问题分析
子序列问题：使每个前缀和后缀都满足#C≥#T。

### 核心思路
利用前缀和+稀疏表。最小删除数 = max(0, -min_balance) + max(0, max_balance - final_balance)。

### 算法流程
1. 预处理前缀和
2. 建稀疏表查询区间最小/最大值
3. 对每个查询计算答案

### 复杂度分析
- 时间：O(N log N + Q)
- 空间：O(N log N)
