# 残雪

## 题目描述

给定集合 S。定义一个 01 串 t 是不好的，当且仅当存在 k∈S，使得 t 包含一个长度为 2k 的子串 t'，且 t' 恰好包含 k 个 0 和 k 个 1。判断是否存在一个好的字符串 t 满足 t 恰好包含 m 个 0 和 n 个 1。

## 输入格式

第一行，一个整数 q，表示询问个数。对于每组询问：
- 仅一行，四个整数 L, R, m, n。

## 输出格式

输出共 q 行，每行一个字符串 Yes 或 No。

## 样例

### 样例 1
输入：
```
5
1 2 3 5
3 3 4 6
5 6 11 13
10 15 33 22
10 13 11 11
```
输出：
```
No
Yes
No
Yes
No
```

## 提示

对于 100% 的数据，1≤q≤10^5，1≤L≤R≤10^18，0≤n,m≤10^18，n+m≥1。

## 解题思路

### 问题分析
需要判断是否存在一个好的01串。

### 核心思路
根据集合S的范围和m,n的值进行判断。

### 算法流程
1. 如果L=1且m,n都大于0，无解
2. 其他情况可能存在解

### 复杂度分析
- 时间复杂度：O(q)
- 空间复杂度：O(1)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int q;
    scanf("%d", &q);
    
    while (q--) {
        long long L, R, m, n;
        scanf("%lld %lld %lld %lld", &L, &R, &m, &n);
        
        if (m == 0 && n == 0) {
            printf("No\n");
            continue;
        }
        
        if (L == 1 && m > 0 && n > 0) {
            printf("No\n");
            continue;
        }
        
        printf("Yes\n");
    }
    
    return 0;
}
```
