# P5971 [CTSC2009] 移盘子

## 题目描述
三根柱子A、B、C。初始时A柱上有N个盘子，盘子分为三类（1、2、3）。目标是将所有1类盘子移到A柱，2类盘子移到B柱，3类盘子移到C柱。每次只能移动一个盘子到另一柱子的顶部。求最少移动次数。

## 输入格式
- 第一行：整数N，表示盘子数量。
- 第二行：N个整数，表示从上到下的盘子类型。

## 输出格式
一个整数，表示最少移动次数。

## 数据范围
- N ≤ 10^6

## 样例
### 输入
```
5
1 2 1 3 3
```
### 输出
```
8
```

## 解题思路
本题是三类盘子汉诺塔问题的变种，可以使用动态规划解决。

### 状态定义
设 `dp[i][b][c]` 表示处理完前 i 个盘子后，B柱顶盘子类型为 b、C柱顶盘子类型为 c 时的最小移动次数。其中 `b, c ∈ {0, 1, 2, 3}`，0 表示空柱。

### 状态转移
对于第 i 个盘子（类型为 t），我们有两种选择：
1. **移动到B柱**：
   - 若B柱为空（b=0），直接移动，移动次数 +1，新状态为 `(t, c)`。
   - 若B柱非空，需要先将B柱顶盘子移动到C柱，再将第 i 个盘子移动到B柱，移动次数 +2，新状态为 `(t, b)`（因为B柱顶移到了C柱，所以C柱顶变为 b）。
2. **移动到C柱**：
   - 若C柱为空（c=0），直接移动，移动次数 +1，新状态为 `(b, t)`。
   - 若C柱非空，需要先将C柱顶盘子移动到B柱，再将第 i 个盘子移动到C柱，移动次数 +2，新状态为 `(c, t)`（因为C柱顶移到了B柱，所以B柱顶变为 c）。

### 初始化
`dp[0][0][0] = 0`，表示处理0个盘子时，B柱和C柱均为空。

### 空间优化
由于状态只依赖于前一层，可以使用滚动数组将空间复杂度从 O(n) 优化到 O(1)。

### 复杂度
- 时间复杂度：O(n × 4 × 4) = O(n)。
- 空间复杂度：O(1)（滚动数组）。

## 代码说明
代码中使用了滚动数组 `dp[2][4][4]`，通过 `cur` 和 `next` 交替更新。对于每个盘子，遍历所有可能的 `(b, c)` 状态，根据上述转移方程更新下一状态。最终答案为所有可能状态中的最小值。

### 特殊情况处理
- 如果只有一个盘子且类型为1，则不需要移动，直接输出0。

## 注意事项
- 确保数组大小足够，`type` 数组大小为 1000001。
- 使用 `INF` 表示不可达状态，初始化为一个大数（如 1e9）。
- 最终答案需要遍历所有可能的 `(b, c)` 状态取最小值，因为处理完所有盘子后，B柱和C柱的顶部类型可能不为空。
