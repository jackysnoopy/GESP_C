# [BalticOI 2004] Sequence

## 题目描述
给定序列 t_1..t_n，求非递减序列 z_1..z_n 使得 Σ|t_i - z_i| 最小。

## 输入格式
第一行整数 n，接下来 n 行每行一个整数 t_i。

## 输出格式
第一行最小绝对值之和，接下来 n 行每行一个 z_i。

## 样例
输入：
```
7
9
4
8
20
14
15
18
```
输出：
```
13
6
7
8
13
14
15
18
```

## 提示
n ≤ 10^6, 0 ≤ t_i ≤ 2×10^9。

## 解题思路

### 问题分析
经典的等距回归（Isotonic Regression）问题，要求在非递减约束下最小化 L1 距离。

### 核心思路
使用斜率技巧（Slope Trick）/ 优先队列。维护一个分段线性凸函数，每个元素处理时在函数中插入一个"帐篷"函数，然后通过取中位数来维护非递减约束。

### 算法流程
1. 维护一个最大堆，代表函数的"拐点"
2. 对每个 t_i，插入两个 t_i 到堆中
3. 若堆顶大于 t_i，弹出并累加差值到答案
4. 最终的 z_i 由堆中元素确定

### 复杂度分析
- 时间：O(n log n)
- 空间：O(n)
