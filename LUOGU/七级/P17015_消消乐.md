# [GESP202606 七级] 消消乐

## 题目描述

给定一个由 $n$ 个整数构成的数组 $a = [a_1, \ldots, a_n]$。每次你可以对数组 $a$ 进行以下操作，直到数组 $a$ 变为空：

- 指定 $a$ 中的一个元素，获得该元素两侧相邻元素之和的分数，并将该元素从 $a$ 中删去。

特别地，如果相邻元素不存在则该元素的值视为 $0$。例如，对于 $a = [1, 2, 3]$ 可以进行以下操作：

- 指定元素 $2$，获得分数 $1 + 3$，删去 $2$ 后 $a = [1, 3]$；
- 指定元素 $1$，获得分数 $0 + 3$，删去 $1$ 后 $a = [3]$；
- 指定元素 $3$，获得分数 $0 + 0$，删去 $3$ 后 $a$ 变为空。

请问你能获得的分数总和最大是多少？

## 输入格式

第一行，一个正整数 $n$，表示数组长度。

第二行，$n$ 个非负整数 $a_1, \ldots, a_n$，表示数组 $a$ 中的整数。

## 输出格式

输出一行，一个整数，表示能获得的最大分数总和。

## 样例

### 样例 1

**输入**
```
6
1 6 3 2 9 1
```

**输出**
```
55
```

### 样例 2

**输入**
```
5
3 1415 926 53 58
```

**输出**
```
5771
```

## 提示

### 数据范围

对于 $40\%$ 的测试点，保证 $1 \le n \le 50$，$0 \le a_i \le 10^3$。

对于所有测试点，保证 $1 \le n \le 100$，$0 \le a_i \le 10^9$。

## 解题思路

### 问题分析

每次移除一个元素，获得其左右邻居之和的分数。目标是最大化总分数。

### 核心思路

**区间 DP**。在数组两端添加哨兵 $0$，变为 $a' = [0, a_1, a_2, \ldots, a_n, 0]$。

设 $dp[i][j]$ 表示移除 $a'[i+1..j-1]$ 中所有元素能获得的最大分数（$a'[i]$ 和 $a'[j]$ 保留作为边界）。

**状态转移**：枚举 $k$（$i < k < j$）作为最后被移除的元素：
\[
dp[i][j] = \max_{i < k < j} \{ dp[i][k] + dp[k][j] + a'[i] + a'[j] \}
\]

当 $k$ 最后被移除时，它的左右邻居恰好是 $a'[i]$ 和 $a'[j]$（因为 $i$ 到 $k$ 之间和 $k$ 到 $j$ 之间的元素都已被移除）。

**边界条件**：$dp[i][i+1] = 0$（相邻元素之间无需移除）。

### 算法流程

1. 读入 $n$ 和数组 $a$。
2. 构造 $a' = [0, a_1, \ldots, a_n, 0]$，长度 $n+2$。
3. 区间 DP：
   - 枚举区间长度 `len` 从 $2$ 到 $n+1$。
   - 枚举左端点 $i$，右端点 $j = i + len$。
   - 枚举分割点 $k$，$dp[i][j] = \max(dp[i][k] + dp[k][j]) + a'[i] + a'[j]$。
4. 输出 $dp[0][n+1]$。

### 复杂度分析

- **时间复杂度**：$O(n^3)$，三层循环，$n \le 100$ 可接受。
- **空间复杂度**：$O(n^2)$，二维 $dp$ 数组。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<long long> a(n + 2, 0);
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }

    vector<vector<long long>> dp(n + 2, vector<long long>(n + 2, 0));
    for (int len = 2; len <= n + 1; len++) {
        for (int i = 0; i + len <= n + 1; i++) {
            int j = i + len;
            for (int k = i + 1; k < j; k++) {
                dp[i][j] = max(dp[i][j], dp[i][k] + dp[k][j] + a[i] + a[j]);
            }
        }
    }

    cout << dp[0][n + 1] << "\n";
    return 0;
}
```
