# [蓝桥杯 2025 省 B] 水质检测

## 题目描述

给定n个水质检测值，统计有多少个值大于等于阈值k。

## 输入格式

第一行：n, k
第二行：n个整数

## 输出格式

输出满足条件的个数

## 提示

数据范围：1 ≤ n ≤ 1000, 1 ≤ k ≤ 1000

## 解题思路

### 问题分析
给定一个数组和一个阈值，统计数组中大于等于阈值的元素个数。

### 核心思路
遍历数组，统计满足条件的元素个数。

### 算法流程
1. 读入数组和阈值
2. 遍历数组统计
3. 输出结果

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr(nullptr;

    int n, k;
    cin >> n >> k;

    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 统计有多少个值大于等于k
    int count = 0;
    for (int i = 0; i < n; i++) {
        if (a[i] >= k) {
            count++;
        }
    }

    cout << count << "\n";

    return 0;
}
```
