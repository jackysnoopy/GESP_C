# [蓝桥杯 2025 省 Java B] 研发资源分配

## 题目描述

蓝桥科技公司 A、B 两部门竞争新型 AI 芯片研发资源。公司设计了为期 $N$ 天的分配方案：

每天早上 A、B 各自提交一个需求等级（$1$ 到 $N$ 的整数）。提交等级**较高**的部门获得当天资源，资源份额等于当天的日期编号（第 $i$ 天为 $i$ 单位）。若两部门提交等级相同，则当天资源作废，双方均不得。

每个部门必须在 $N$ 天内把 $1$ 到 $N$ 的每个等级各用一次。A 部门安插的间谍提前获知了 B 部门第 $i$ 天提交的等级 $P_i$。已知 B 的提交顺序，求 A 部门总资源份额减去 B 部门总资源份额的差值最大可以是多少。

## 输入格式

第一行一个整数 $N$。
第二行 $N$ 个整数 $P_1,\dots,P_N$，表示 B 部门第 $1$ 到第 $N$ 天提交的需求等级。

## 输出格式

输出一个整数，表示 A 部门总份额减 B 部门总份额的最大差值。

## 提示

- $20\%$ 数据：$1\le N\le 11$。
- $100\%$ 数据：$1\le N\le 10^5$，$P$ 为 $1\dots N$ 的排列。
- 样例：$N=3$，$P=[1,3,2]$。A 取 $[2,1,3]$：第 1 天 A=2>B=1 得 1；第 2 天 A=1<B=3 失 2；第 3 天 A=3>B=2 得 3；差值 $1-2+3=2$。

## 解题思路

### 问题分析

这是经典的"田忌赛马"型贪心。A 要把自己手中的 $N$ 个等级（各用一次）分配到 $N$ 天上，每天与 B 的 $P_d$ 比较：大于则 A 赢得 $d$，小于则 B 赢得 $d$（A 损失 $d$），相等则双方都不得。

关键观察：不同天数的资源价值不同（第 $d$ 天值 $d$）。为使差值最大，应**优先在高价值的天上争取胜利**，其次在赢不了时尽量"打平"以拒绝让对方拿到高价值资源，实在不行才在高价值天认输、把低价值的天让给对方。

由于等级本身就是 $1\dots N$ 的排列，每天比较的是两个排列的对应位，可用类似田忌赛马的贪心：按天数价值从大到小处理，维护 A 手中尚未使用的等级集合。

### 核心思路

按天数 $d$ 从大到小处理（价值高的先决策），用一个 `multiset` 维护 A 当前可用的等级：
1. **能赢则赢**：取 `upper_bound(P_d)`，即最小的、能严格大于 $P_d$ 的等级。若存在，A 赢得该天，`score += d`，消耗该等级。
2. **赢不了则打平**：若等级 $P_d$ 还在手中，就出 $P_d$ 与 B 打平（双方都不得），消耗 $P_d$，避免把高价值天让给 B。
3. **打平也不行则认输**：此时手中所有等级都小于 $P_d$（且 $P_d$ 已用过），只能认输，出**最小**的等级（把大牌留给后面可能赢或打平的天），`score -= d`。

这样在价值最高的天上优先拿到正收益、其次零收益、最差才负收益，且出牌尽量省着用大牌，是全局最优的贪心策略。

### 算法流程

1. 读入 $N$ 与 $P$（下标 $1\dots N$）。
2. 把可用等级 $1\dots N$ 放入 `multiset cards`；按 $d=N,N-1,\dots,1$ 顺序处理。
3. 对每天 $d$：取 $b=P_d$；按"能赢→打平→认输"三段逻辑更新 `score` 并删除相应等级。
4. 输出 `score`。

### 复杂度分析

每天一次 `multiset` 操作 $O(\log N)$，共 $N$ 天。时间 $O(N\log N)$，空间 $O(N)$。

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
using namespace std;
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    vector<int> p(n + 1);
    for (int i = 1; i <= n; i++) cin >> p[i];
    vector<int> order(n);
    for (int i = 0; i < n; i++) order[i] = i + 1;
    sort(order.begin(), order.end(), greater<int>());
    multiset<int> cards;
    for (int i = 1; i <= n; i++) cards.insert(i);
    long long score = 0;
    for (int d : order) {
        int b = p[d];
        auto it = cards.upper_bound(b);
        if (it != cards.end()) {
            score += d;
            cards.erase(it);
        } else {
            auto bit = cards.find(b);
            if (bit != cards.end()) {
                cards.erase(bit);
            } else {
                score -= d;
                cards.erase(cards.begin());
            }
        }
    }
    cout << score << "\n";
    return 0;
}
```
