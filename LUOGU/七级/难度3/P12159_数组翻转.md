# [蓝桥杯 2025 省 Java B] 数组翻转

## 题目描述

给定一个长度为N的数组，将数组元素完全反转（第一个元素变最后一个，最后一个变第一个）。

## 输入格式

第一行：N
第二行：N个整数，表示数组元素

## 输出格式

输出反转后的数组元素，用空格分隔

## 提示

数据范围：1 ≤ N ≤ 1000

## 解题思路

### 问题分析
给定一个数组，需要将其元素顺序完全反转。

### 核心思路
使用标准库的reverse函数直接反转数组，或者使用双指针法交换元素。

### 算法流程
1. 读入数组
2. 使用reverse函数反转数组
3. 输出反转后的数组

### 复杂度分析
- 时间复杂度：O(N)
- 空间复杂度：O(N)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 翻转数组
    reverse(a.begin(), a.end());

    // 输出
    for (int i = 0; i < n; i++) {
        cout << a[i];
        if (i < n - 1) cout << " ";
    }
    cout << "\n";

    return 0;
}
```
