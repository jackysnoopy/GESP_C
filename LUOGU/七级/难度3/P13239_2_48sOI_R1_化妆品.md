# 「2.48sOI R1」化妆品

## 题目描述

商场有 $2n$ 个化妆品，每个提供互不相同的时尚值 $F_i$ 和互不相同的美丽值 $B_i$。Misserina 和 ShenTianYi_ 要进行 $n$ 轮选择，每轮 Misserina 先选、ShenTianYi_ 后选。

- 第 $i$ 轮给定 $Q_i\in\{1,2\}$：Misserina 选择剩余化妆品中该属性（$Q_i=1$ 取时尚值最大，$Q_i=2$ 取美丽值最大）的那个；
- ShenTianYi_ 则选择 Misserina “最不想要”的那个，即同一属性最小的那个。

选完后求两人各自的时尚值之和、美丽值之和。

## 输入格式

第一行 $n$；第二行 $2n$ 个整数 $F_i$；第三行 $2n$ 个整数 $B_i$；第四行 $n$ 个整数 $Q_i$。

## 输出格式

第一行两个整数：Misserina 的时尚值、美丽值；第二行两个整数：ShenTianYi_ 的时尚值、美丽值。

## 提示

- $1\le n\le5\times10^5$，$1\le F_i,B_i\le10^9$，$Q_i\in\{1,2\}$，所有 $F_i$ 互不相同、所有 $B_i$ 互不相同。
- 样例：第 1、2 轮 $Q=1$，Misserina 拿时尚值最大的第 5、4 种；ShenTianYi_ 拿时尚值最小的第 1、3 种；第 3 轮 $Q=2$，Misserina 拿美丽值最大的第 6 种，ShenTianYi_ 拿美丽值最小的第 2 种。最终 Misserina 时尚 $9+8+4=21$、美丽 $4+3+27=34$；ShenTianYi_ 时尚 $1+7+3=11$、美丽 $1665+8888+5=10558$。

## 解题思路

### 问题分析

每一轮只根据单一属性 $Q$ 做选择：Misserina 取该属性剩余最大者，ShenTianYi_ 取该属性剩余最小者，二者被同时移除。难点在于 $n$ 很大（$5\times10^5$），朴素“每轮线性扫描最大/最小并删除”是 $O(n^2)$，无法接受。需要一个能快速查询“当前剩余中的最大/最小”并支持双向删除（因为同一物品要在两个属性结构中同步删除）的数据结构。

### 核心思路

把物品分别按时尚值、美丽值排序得到两个名次数组。维护两棵树状数组（Fenwick）：`bf` 标记“按时尚值名次”仍在场上的物品，`bb` 标记“按美丽值名次”仍在场上的物品。树状数组支持：

- 查询当前在场总数 `total`；
- 二分查找第 $k$ 小在场名次（`findKth`）：$k=1$ 即最小名次，$k=\text{total}$ 即最大名次。

每轮根据 $Q$ 在对应树状数组上取最大、最小名次，得到两个物品 id，分别累加两人的 $F,B$，然后在这两棵树状数组上各删除这两个物品（用预先存好的交叉名次 `frank/brank` 在另一棵上定位）。

### 算法流程

1. 读入 $F,B,Q$。
2. 对 $F$、$B$ 排序得到 `byF,byB`（名次→物品 id），反推 `frank,brank`（物品 id→名次）。
3. 两棵树状数组初始全部置 1。
4. 对每轮 $Q$：
   - $Q=1$：`bf` 上 `rmax=findKth(total)`、`rmin=findKth(1)`；`idM=byF[rmax]`、`idS=byF[rmin]`；累加后从 `bf` 删 `rmax,rmin`，从 `bb` 删 `brank[idM],brank[idS]`。
   - $Q=2$：对称地在 `bb` 上操作，并用 `frank` 在 `bf` 上同步删除。
5. 输出两人累计值。

### 复杂度分析

排序 $O(n\log n)$；每轮 $O(\log n)$（常数次树状数组操作），共 $O(n\log n)$。空间 $O(n)$。$n\le5\times10^5$ 下轻松通过。由于输入量达 $1.5\times10^6$ 个整数，`cin` 解析较慢，故采用自定义快速读入（`fread` 缓冲 + 手写整数解析），实测 $n=5\times10^5$ 约 0.3s。

## 参考代码

```cpp
#include <cstdio>
#include <vector>
#include <algorithm>
using namespace std;

static char ibuf[1<<16];
static int ilen=0, ipos=0;
inline int gc(){
    if(ipos==ilen){ ilen=(int)fread(ibuf,1,sizeof(ibuf),stdin); ipos=0; if(ilen==0) return -1; }
    return ibuf[ipos++];
}
inline long long readInt(){
    int c=gc(); while(c!=-1 && c!='-' && (c<'0'||c>'9')) c=gc();
    if(c==-1) return -1;
    int neg=0; if(c=='-'){neg=1;c=gc();}
    long long x=0; while(c>='0'&&c<='9'){ x=x*10+(c-'0'); c=gc(); }
    return neg?-x:x;
}

struct BIT {
    int n, pw;
    vector<int> t;
    BIT(int n=0):n(n),t(n+2,0){
        pw=1; while(pw*2<=n) pw*=2;
    }
    inline void add(int i,int v){ for(++i; i<=n; i+=i&(-i)) t[i]+=v; }
    inline int findKth(int k){            // 0-based index of k-th present (k>=1)
        int pos=0;
        for(int step=pw; step>0; step>>=1){
            if(pos+step<=n && t[pos+step] < k){ k-=t[pos+step]; pos+=step; }
        }
        return pos;
    }
};

int main(){
    int n = (int)readInt();
    if(n<=0) return 0;
    int m = 2*n;
    vector<long long> F(m), B(m);
    for(int i=0;i<m;++i) F[i]=readInt();
    for(int i=0;i<m;++i) B[i]=readInt();
    vector<int> Q(n);
    for(int i=0;i<n;++i) Q[i]=(int)readInt();

    vector<int> byF(m), byB(m);
    for(int i=0;i<m;++i){ byF[i]=i; byB[i]=i; }
    sort(byF.begin(),byF.end(),[&](int x,int y){ return F[x]<F[y]; });
    sort(byB.begin(),byB.end(),[&](int x,int y){ return B[x]<B[y]; });
    vector<int> frank(m), brank(m);
    for(int r=0;r<m;++r){ frank[byF[r]]=r; brank[byB[r]]=r; }

    BIT bf(m), bb(m);
    for(int i=0;i<m;++i){ bf.add(i,1); bb.add(i,1); }
    int remF=m, remB=m;

    long long MF=0,MB=0,SF=0,SB=0;
    for(int rnd=0; rnd<n; ++rnd){
        if(Q[rnd]==1){
            int rmax = bf.findKth(remF);   // max F remaining
            int rmin = bf.findKth(1);      // min F remaining
            int idM = byF[rmax], idS = byF[rmin];
            MF += F[idM]; MB += B[idM];
            SF += F[idS]; SB += B[idS];
            bf.add(rmax,-1); bb.add(brank[idM],-1);
            bf.add(rmin,-1); bb.add(brank[idS],-1);
            remF-=2; remB-=2;
        } else {
            int rmax = bb.findKth(remB);   // max B remaining
            int rmin = bb.findKth(1);      // min B remaining
            int idM = byB[rmax], idS = byB[rmin];
            MF += F[idM]; MB += B[idM];
            SF += F[idS]; SB += B[idS];
            bb.add(rmax,-1); bf.add(frank[idM],-1);
            bb.add(rmin,-1); bf.add(frank[idS],-1);
            remF-=2; remB-=2;
        }
    }
    printf("%lld %lld\n%lld %lld\n", MF, MB, SF, SB);
    return 0;
}
```
