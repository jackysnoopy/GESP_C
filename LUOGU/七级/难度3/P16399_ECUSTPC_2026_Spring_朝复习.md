# [ECUSTPC 2026 Spring] 朝复习

## 题目描述

:::epigraph
计算机科学：复习 (Computer Science: Go Over, known as ___ ) 是一个在全球流行的事项。
:::

**本题和试题 H 朝复夕没有关系。**

TSUCE(Time-Space Union of Coding Experts) 准备举办 TSUCE Programming Duel Cup，这是两个选手之间的算法竞赛单挑赛！

赛制如下，请留意其中 $n$ 和 $m$ 的含义：

- 每场比赛由两名选手对战，一场比赛有且仅有一名获得胜利的选手，没有平局。
- 一场由一节常规时间比赛 + 若干节加时赛组成（可能只有 0 节加时赛），常规时间包含 $2n$ 轮比赛，一节加时赛包含 $2m$ 轮比赛，一轮比赛有且仅有一名胜者。
- 常规时间内获得 $n+1$ 轮胜利的选手将获得整场比赛的胜利，比赛将立刻结束。
- 常规时间 $2n$ 轮后若未分胜负（比分 $n:n$）后会进入加时赛，单节加时内获得 $m+1$ 轮胜利的选手将获得整场比赛的胜利，比赛将立刻结束。
- 一节加时赛后仍然未分胜负则将进入下一节加时，直到决出胜负。
- 最后的比分为双方在整场比赛中各自赢得的轮数。

遗憾的是，记分员小 T 的数据库坏掉了，他只记得全部 $k$ 场比赛的比分，但他不记得和赛制相关的 $n$ 和 $m$ 了。

请帮助他确定是否存在一组合法的正整数 $n$ 和 $m$ 使得这些比分都是在该赛制下合法的比分。

## 输入格式

第一行输入一个整数 $T \ (1 \le T \le 10^5)$，表示测试数据的数量。

每组测试数据第一行输入一个整数 $k \ (1 \le k \le 10^5)$，表示记录的比赛场数。

随后 $k$ 行输入两个整数 $a$ 和 $b \ (0 \le a, b \le 10^{18}, |a - b| \ge 2)$，表示小 T 数据库中记录的一场比赛的比分。

保证所有测试数据输入中的 $\sum k \le 3 \times 10^5$。

## 输出格式

对于每组测试数据，如果存在一个合法的 $n$ 和 $m$ 使得这些比分都是在该赛制下合法的比分，则输出一行一个字符串 YES，否则输出一行一个字符串 NO.

注意评测时不会区分 YES 和 NO 的大小写，换言之当答案是肯定的时候输出 yes, YES, Yes, YeS 等都会被认为是正确的。

## 提示

### 样例 1 解释

对于第 $3$ 组测试数据，可以发现 $n = 12, m = 3$ 是一个合理的解，这样子比赛会经过如上的过程：

- 第一场 $2:13$，常规时间 B 选手率先拿下 $n + 1 = 13$ 局胜利以结束比赛。
- 第二场 $16:12$，常规时间 $12:12$ 战平，第一节加时 A 选手 $4:0$ 直落对手，拿下了 $m + 1 = 4$ 局胜利。
- 第三场第四场比赛分别以 $9:13$ 和 $13:10$ 在常规时间内结束。
- 第五场比赛常规时间战成 $12:12$，第一节加时和第二节加时双方都以 $3:3$ 战平，比赛以 $18:18$ 进入第三节加时，最终 B 选手在第三节加时 $4:2$ 战胜对手，最终比分 A $20:22$ B.

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> pages(n);
    for (int i = 0; i < n; i++) cin >> pages[i];

    int total = 0;
    for (int p : pages) total += p;

    cout << total << "\n";
    return 0;
}
```
