# [蓝桥杯 2025 省 C/Python A/Java A] 拼好数

## 题目描述

称含有**不少于 6 个数字 6** 的数为"好数"。例如 $666666$、$162636465666$ 是好数，$12366666$ 不是。

给定 $n$ 个正整数 $a_i$，可以把这些数分成若干组拼起来，每组内的数可按任意顺序拼接，但**一组最多 3 个数**。求最多能得到多少个好数。

## 输入格式

第一行一个正整数 $n$。
第二行 $n$ 个正整数 $a_1,\dots,a_n$，空格分隔。

## 输出格式

输出一个整数，表示最多能得到的好数数量。

## 提示

- $20\%$ 数据：$1\le n\le 20$。
- $100\%$ 数据：$1\le n\le 1000$，$0\le a_i\le 10^9$。
- 样例 1：`66 66 66` 三数各含 2 个 6，拼成一组共 6 个 6，得 1 个好数。
- 样例 2：`666666 16166 6696 666 6 6 6`，最多得 2 个好数。

## 解题思路

### 问题分析

一个数拼接后是否为好数，只取决于它（及同组伙伴）含数字 6 的总数是否 $\ge 6$，与 6 的位置无关。因此先把每个数压缩成"含 6 的个数 $c$"。

- 若某个数本身 $c\ge 6$，它单独成一组就是一个好数，直接计入并移除。
- 剩余 $1\le c\le 5$ 的数需要组队：每组至多 3 个数、总 6 数 $\ge 6$ 即算一个好数。问题转化为：用这些 $c$ 值尽量多地凑出"组内求和 $\ge 6$ 且个数 $\le 3$"的组。

这是一个贪心装箱：先拿最大的 $c$ 开组（最接近 6），再用至多 2 个数补足到 6；补的时候优先找一个"刚好够"的数以减少浪费，找不到就用最大的数硬补。

### 核心思路

用 `multiset` 维护剩余的 $c$ 值：
1. 统计每个数含 6 个数 $c$；$c\ge 6$ 的直接 `big++`（自成一好数），$c=0$ 的无用丢弃，$1\le c\le 5$ 的放入 `multiset`。
2. 反复开组：取当前最大的 $c$ 作为组的初始 `sum`，`used=1`。
3. 当 `sum<6` 且 `used<3` 且集合非空时：令 `need=6-sum`，用 `lower_bound(need)` 找最小的 $\ge need$ 的数加上（若存在）；否则加上最大的数。`used++`。
4. 若最终 `sum>=6` 则该组成好数 `groups++`。
5. 答案为 `big + groups`。

先取最大值开组、补足时优先"刚好够"的策略能最大化成组数。

### 算法流程

1. 对每个 $a_i$ 计算含 6 个数 `count_sixes`。
2. $c\ge 6$ 入 `big`；$c>0$ 入 `multiset`。
3. 循环开组：取最大 $c$，再按需补 1～2 个数（优先 `lower_bound(need)`，否则取最大）。
4. `sum>=6` 则 `groups++`；最终输出 `big+groups`。

### 复杂度分析

每个数最多进出 `multiset` 常数次，每次 $O(\log n)$。时间 $O(n\log n)$，空间 $O(n)$。

## 参考代码

```cpp
#include <iostream>
#include <set>
using namespace std;
int count_sixes(long long x) {
    if (x == 0) return 0;
    int c = 0;
    while (x > 0) { if (x % 10 == 6) c++; x /= 10; }
    return c;
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    cin >> n;
    int big = 0;
    multiset<int> ms;
    for (int i = 0; i < n; i++) {
        long long a;
        cin >> a;
        int c = count_sixes(a);
        if (c >= 6) big++;
        else if (c > 0) ms.insert(c);
    }
    int groups = big;
    while (!ms.empty()) {
        auto it = prev(ms.end());
        int sum = *it;
        ms.erase(it);
        int used = 1;
        while (sum < 6 && used < 3 && !ms.empty()) {
            int need = 6 - sum;
            auto fit = ms.lower_bound(need);
            if (fit != ms.end()) { sum += *fit; ms.erase(fit); }
            else { auto lit = prev(ms.end()); sum += *lit; ms.erase(lit); }
            used++;
        }
        if (sum >= 6) groups++;
    }
    cout << groups << "\n";
    return 0;
}
```
