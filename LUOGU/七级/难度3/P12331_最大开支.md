# [蓝桥杯 2023 省 Java B] 最大开支

## 题目描述

小明有 n 件商品要买，有 m 种钱可以花。第 i 件商品价格是 a_i，第 j 种钱的面值是 b_j。
小明可以自由选择怎么花这些钱，要求每种钱只能用一次，且不能换零钱。
求小明最多能花多少钱（如果钱不够支付商品价格，则该种钱不用）。

## 输入格式

第一行：n, m
第二行：n 个整数，表示商品价格
第三行：m 个整数，表示钱的面值

## 输出格式

输出最大开支

## 提示

数据范围：1 ≤ n, m ≤ 100

## 解题思路

### 问题分析
- 有 n 件商品和 m 种钱
- 可以选择购买哪些商品，用哪些钱支付
- 要求最大化总花费（钱的总额 - 商品的总额）

###（核心思路
**贪心策略**：
1. 将商品按价格从小到大排序
2. 将钱按面值从大到小排序
3. 用贵的钱买便宜的商品，这样差值最大

### 算法流程
1. 读入商品价格和钱的面值
2. 排序商品（升序）和钱（降序）
3. 逐对匹配，计算差值累加
4. 输出总差额

### 复杂度分析
- 时间复杂度：O(n log n + m log m)
- 空间复杂度：O(n + m)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m;
    cin >> n >> m;

    vector<int> a(n), b(m);
    for (int i = 0; i < n; i++) cin >> a[i];
    for (int i = 0; i < m; i++) cin >> b[i];

    // 贪心策略：买便宜的商品，花贵的钱
    sort(a.begin(), a.end());
    sort(b.rbegin(), b.rend());

    long long ans = 0;
    int k = min(n, m);
    for (int i = 0; i < k; i++) {
        ans += max(0, b[i] - a[i]);
    }

    cout << ans << "\n";

    return 0;
}
```
