# [蓝桥杯 2024 国 Java A] 修改数位

## 题目描述

给定一个数字字符串，你可以将某些位上的数字修改。
如果一位是0、6或8，可以改为9。求能得到的最大数字。

## 输入格式

一行数字字符串

## 输出格式

输出能得到的最大数字

## 提示

数据范围：字符串长度 ≤ 100

## 解题思路

### 问题分析
给定数字字符串，某些位可以改为9，求能得到的最大值。

### 核心思路
将所有可改的位（0,6,8）改为9，得到最大数字。

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;

    // 将数字修改为最大的可能
    for (int i = 0; i < (int)s.length(); i++) {
        if (s[i] == '0' || s[i] == '6' || s[i] == '8') {
            s[i] = '9';
        }
    }

    cout << s << "\n";

    return 0;
}
```
