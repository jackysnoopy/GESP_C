# 【MX-X25-T2】『FeOI-5』2 的次幂

## 题目描述

给定一个长 $n$ 的非负整数序列 $a$，你可以进行如下形式的操作任意次：

1. 选择一个区间 $[l,r]$（$l$ 和 $r$ 可以相等）；
2. 令 $s=\sum \limits_{i\in [l,r]} a_i$，令 $k$ 为最大的非负整数使得 $2^k\le s$（若 $s=0$ 则 $k=0$）；
3. 获得 $k$ 个金币，并将 $a_{[l,r]}$ 删去。之后 $a_{[1,l-1]}$ 和 $a_{[r+1,n]}$ 会拼接起来，即新的 $a$ 长度为 $n-r+l-1$；

求最多能获得多少个金币。

## 输入格式

第一行两个整数 $c,T$ 分别表示测试点所属的子任务编号（样例保证 $c=0$）和数据组数。

对于每组数据，输入两行：

- 第一行一个整数 $n$；

- 第二行 $n$ 个整数表示序列 $a$；

## 输出格式

对于每组数据，输出一行一个整数表示答案。

## 样例

### 样例输入 1
```
0 1
5
3 2 1 1 4
```

### 样例输出 1
```
5
```

### 样例输入 2
```
0 1
12
1 0 3 2 2 1 4 5 6 8 7 9
```

### 样例输出 2
```
19
```

## 提示

**【样例 1 解释】**

第一次操作：选择区间 $[2,2]$，此时 $s=2,k=1$，获得 $1$ 金币，操作后序列变为 `3 1 1 4`；

第二次操作：选择区间 $[1,3]$，此时 $s=5,k=2$，获得 $2$ 金币，操作后序列变为 `4`；

第三次操作：选择区间 $[1,1]$，此时 $s=4,k=2$，获得 $2$ 金币，操作后序列变为空序列；

总共获得 $5$ 金币，可以证明这是能获得的最多金币数。


**【数据范围】**

对于所有测试数据，$1\le n,\sum n\le 10^6$，$0\le a_i\le 10^{18}$，$1\le T\le 100$。

| 子任务编号 |  $\sum n$  |                 特殊性质                 | 分数 |
| :--------: | :--------: | :--------------------------------------: | :--: |
|    $1$     |  $\le 5$   |                    无                    | $5$  |
|    $2$     | $\le 100$  |                    无                    | $10$ |
|    $3$     | $\le 10^6$ |             $0\le a_i\le 3$              | $10$ |
|    $4$     | $\le 10^6$ | $\forall i$，不存在 $k$ 使得 $a_i=2^k-1$ | $20$ |
|    $5$     | $\le 5000$ |                    无                    | $20$ |
|    $6$     | $\le 10^6$ |                    无                    | $35$ |

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    int ans = 0;
    for (int x : a) {
        while (x > 0 && x % 2 == 0) {
            ans++;
            x /= 2;
        }
    }

    cout << ans << "\n";
    return 0;
}
```

## 解题思路

### 问题分析
给定序列 $a$，每次选择区间 $[l,r]$，计算区间和 $s$，获得 $\lfloor \log_2 s \rfloor$ 个金币，然后删除该区间。目标是最大化总金币数。

### 核心思路
参考代码的核心思想是：对每个元素单独处理，计算其包含的因子 $2$ 的个数。由于每次操作删除区间后剩余部分会拼接，最优策略是将所有元素拆分为 $2$ 的幂次之和，每个 $2$ 的因子贡献 $1$ 个金币。对于每个元素 $x$，不断将 $x$ 除以 $2$ 直到为奇数，每除一次获得 $1$ 个金币，最终求和即为答案。

### 算法流程
1. 读入序列长度 $n$ 和各元素 $a_i$；
2. 对每个元素 $x$，统计其能被 $2$ 整除的次数：
   - 当 $x > 0$ 且 $x$ 为偶数时，金币数加 $1$，$x$ 除以 $2$；
3. 输出总金币数。

### 复杂度分析
时间复杂度：$O(n \log A)$，其中 $A$ 为元素最大值，每个元素最多除以 $2$ 约 $\log_2 A$ 次。
空间复杂度：$O(n)$，存储序列数组。
