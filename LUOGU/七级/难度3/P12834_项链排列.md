# [蓝桥杯 2025 国 B] 项链排列

## 题目描述

小蓝有 $A$ 颗蓝珠（字符 `L`）和 $B$ 颗桥珠（字符 `Q`），用这些珠子串成一条项链（线性字符串）。当相邻两珠种类不同时记一次"变化"。变化次数 = 字符串中相邻且不同字符对的总数。

例如 `LLQLQ` 的变化次数为 $3$。

希望找到一种排列使总变化次数恰好为 $C$，并在所有满足条件的排列中找出**字典序最小**的一个。若不存在则输出 `-1`。

## 输入格式

一行三个整数 $A, B, C$，分别表示蓝珠数量、桥珠数量和目标变化次数。

## 输出格式

输出一个长度为 $A+B$ 的字符串，表示字典序最小的满足条件的排列；不存在则输出 `-1`。

## 提示

**评测用例规模与约定**：
- 20%：$0 \le A,B,C \le 100$，$1 \le A+B \le 200$。
- 100%：$0 \le A,B,C \le 10^6$，$1 \le A+B \le 2\times10^6$。

时间限制 1000ms，内存限制 256MB。

## 解题思路

### 问题分析

线性字符串的变化次数等于"连续同色段（块）数减一"。设块数为 $k$，则变化次数 $C = k-1$，即 $k = C+1$。块交替出现（`L` 块、`Q` 块相间）。问题转化为：用恰好 $A$ 个 `L`、$B$ 个 `Q` 构造 $C+1$ 个交替块，且字典序最小。

### 核心思路

**可行性**：当 $A,B$ 均大于 0 时，最小变化次数为 $1$（两种颜色都出现至少要交替一次）；最大变化次数取决于尽量交替：
- 若 $A \ne B$：$\max C = 2\min(A,B)$；
- 若 $A = B$：$\max C = 2\min(A,B)-1$（首尾不同色时少一次）。

若 $C$ 不在 $[1, \max C]$ 内则无解。只有一种珠子或全无珠子的边界单独处理。

**块数与开头选择**：$k = C+1$ 个交替块。因为 `L` < `Q`，字典序最小的方案应**优先以 `L` 开头**；若以 `L` 开头不可行（`L` 块数需求超过 $A$），则**回退到以 `Q` 开头**。
- 以 `L` 开头：`L` 块数 $lB=\lceil k/2\rceil$，`Q` 块数 $qB=\lfloor k/2\rfloor$；
- 以 `Q` 开头：`L` 块数 $lB=\lfloor k/2\rfloor$，`Q` 块数 $qB=\lceil k/2\rceil$。

要求 $lB \le A$ 且 $qB \le B$，否则该开头不可行。两种开头都不可行才输出 `-1`。

> ⚠️ 仅尝试 `L` 开头会在 $A < B$ 且 $C = 2A$（最大变化次数）时漏掉以 `Q` 开头的解，例如 $A=1,B=2,C=2$ 的答案 `QLQ`。

**字典序最小**：确定开头后，每个块至少 1 颗。把多余的 `L` 全部塞进**第一个 `L` 块**（让前导 `L` 尽可能长，`L` 越早越多越优），把多余的 `Q` 全部塞进**最后一个 `Q` 块**（让靠前的 `Q` 块保持长度 1，使 `L` 尽早出现）。

### 算法流程

1. 边界：$A=B=0$ 时 $C$ 必须为 0；$A=0$ 或 $B=0$ 时 $C$ 必须为 0。
2. 计算 $\min C = 1$，$\max C = 2\min(A,B)$（$A=B$ 时减 1）。越界输出 `-1`。
3. $k = C+1$。先试 `L` 开头，再试 `Q` 开头：
   - 按开头计算 $lB, qB$，若 $lB>A$ 或 $qB>B$ 跳过；
   - $extraL=A-lB$，$extraQ=B-qB$；第一个 `L` 块加 $extraL$，最后一个 `Q` 块加 $extraQ$。
4. 两种都不可行则 `-1`，否则输出构造的串。

### 复杂度分析

- 时间复杂度：$O(A+B)$，即输出串长度，$A+B\le2\times10^6$。
- 空间复杂度：$O(A+B)$。

## ⚠️ 代码勘误

仓库中的 `P12834.cpp` 原始版本**只尝试以 `L` 开头**，在 $A < B$ 且 $C = 2\min(A,B)$ 时会错误地输出 `-1`（例如输入 `1 2 2` 应输出 `QLQ`，原代码输出 `-1`；输入 `2 3 4` 应输出 `QLQLQ`，原代码输出 `-1`）。正确做法是 `L` 开头不可行时回退到 `Q` 开头。下方参考代码为修正版。

## 参考代码

```cpp
#include <iostream>
#include <string>
using namespace std;
int main() {
    int A, B, C;
    cin >> A >> B >> C;
    if (A == 0 && B == 0) { cout << (C == 0 ? "" : "-1") << "\n"; return 0; }
    if (A == 0) {
        if (C != 0) { cout << "-1\n"; return 0; }
        cout << string(B, 'Q') << "\n"; return 0;
    }
    if (B == 0) {
        if (C != 0) { cout << "-1\n"; return 0; }
        cout << string(A, 'L') << "\n"; return 0;
    }
    int minC = 1, maxC = 2 * min(A, B);
    if (A == B) maxC--;
    if (C < minC || C > maxC) { cout << "-1\n"; return 0; }
    int blocks = C + 1;
    bool ok = false;
    string result;
    for (int startL = 1; startL >= 0 && !ok; startL--) {
        int lBlocks, qBlocks;
        if (startL) { lBlocks = (blocks + 1) / 2; qBlocks = blocks / 2; }
        else        { lBlocks = blocks / 2;       qBlocks = (blocks + 1) / 2; }
        if (lBlocks > A || qBlocks > B) continue;
        int extraL = A - lBlocks, extraQ = B - qBlocks;
        int firstL = startL ? 0 : 1;
        int lastQ = startL ? (2 * qBlocks - 1) : (2 * qBlocks - 2);
        string res;
        for (int b = 0; b < blocks; b++) {
            bool isL = startL ? (b % 2 == 0) : (b % 2 == 1);
            if (isL) {
                int cnt = 1 + (b == firstL ? extraL : 0);
                res += string(cnt, 'L');
            } else {
                int cnt = 1 + (b == lastQ ? extraQ : 0);
                res += string(cnt, 'Q');
            }
        }
        result = res; ok = true;
    }
    if (!ok) cout << "-1\n";
    else cout << result << "\n";
    return 0;
}
```
