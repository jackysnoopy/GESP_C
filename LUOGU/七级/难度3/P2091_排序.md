# 排序

## 题目描述

给定n个整数，将它们按升序排序并输出。

## 输入格式

第一行：n
第二行：n个整数

## 输出格式

输出排序后的n个整数，用空格分隔

## 提示

数据范围：1 ≤ n ≤ 1000

## 解题思路

### 问题分析
这是一个基本的排序问题，可以使用C++标准库的sort函数。

### 核心思路
使用STL的sort函数进行排序，然后输出结果。

### 算法流程
1. 读入数组
2. 使用sort函数排序
3. 输出排序后的数组

### 复杂度分析
- 时间复杂度：O(n log n)
- 空间复杂度：O(n)

## 参考代码

\`\`\`cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }

    // 排序
    sort(a.begin(), a.end());

    // 输出排序后的数组
    for (int i = 0; i < n; i++) {
        cout << a[i];
        if (i < n - 1) cout << " ";
    }
    cout << "\n";

    return 0;
}
