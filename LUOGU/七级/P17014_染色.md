# [GESP202606 七级] 染色

## 题目描述

小杨同学有一张包含 $n$ 个结点的无向图 $G$, $G$ 中的结点依次以 $1, 2, \dots, n$ 编号。

小杨同学发现 $G$ 中每个结点的度数都是 $2$。显然 $G$ 中恰好有 $n$ 条边。

小杨同学想为 $G$ 中的结点染色，使得任意一条边两端的结点都有不同的颜色。

小杨同学想知道最少需要多少种颜色才能在满足条件的前提下为 $G$ 染色。

## 输入格式

本题包含多组数据。

第一行，一个正整数 $t$，表示数据组数。

对于每组数据：

第一行，一个正整数 $n$，表示无向图 $G$ 中的结点数。

接下来 $n$ 行，每行两个正整数 $u_i, v_i$，表示一条连接结点 $u_i$ 与 $v_i$ 的无向边，整数之间以空格分隔。

保证 $G$ 中没有重边与自环。

## 输出格式

对于每组数据：输出一行，一个整数，表示在满足条件的前提下为 $G$ 染色需要的最少颜色数。

## 提示

### 数据范围

对于 $40\%$ 的测试点，保证 $\sum n \le 500$，$\sum n$ 指每个输入中多组数据的 $n$ 的总和。

对于所有测试点，保证 $1 \le t \le 100$，$3 \le n \le 10^5$，$\sum n \le 10^5$。保证 $G$ 中没有重边与自环。

## 解题思路

### 问题分析

每个结点度数为 $2$ 的无向图，必然是若干个**不相交的环**的并集。问题转化为：对每个环求最少染色数，取最大值。

### 核心思路

**环的染色规律**：
- **偶数长度的环**：可以交替用 $2$ 种颜色染色（如 $A, B, A, B, \ldots$）。
- **奇数长度的环**：至少需要 $3$ 种颜色（$2$ 种颜色无法使首尾不同色）。

因此，若图中所有环的长度均为偶数，答案为 $2$；若存在至少一个奇数长度的环，答案为 $3$。

### 算法流程

1. 读入 $t$ 组数据。
2. 对每组数据：
   - 构建邻接表。
   - 用 DFS 遍历每个连通分量（即每个环），记录环的长度。
   - 若存在奇数长度的环，输出 $3$；否则输出 $2$。

### 复杂度分析

- **时间复杂度**：$O(\sum n)$，每条边和结点各访问一次。
- **空间复杂度**：$O(\sum n)$，存储邻接表和访问标记。

## 参考代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

vector<vector<int>> adj;
vector<bool> vis;

int dfs(int u) {
    vis[u] = true;
    int cnt = 1;
    for (int v : adj[u]) {
        if (!vis[v]) cnt += dfs(v);
    }
    return cnt;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        adj.assign(n + 1, vector<int>());
        vis.assign(n + 1, false);
        for (int i = 0; i < n; i++) {
            int u, v;
            cin >> u >> v;
            adj[u].push_back(v);
            adj[v].push_back(u);
        }

        int ans = 2;
        for (int i = 1; i <= n; i++) {
            if (!vis[i]) {
                int sz = dfs(i);
                if (sz % 2 == 1) {
                    ans = 3;
                    break;
                }
            }
        }
        cout << ans << "\n";
    }
    return 0;
}
```
