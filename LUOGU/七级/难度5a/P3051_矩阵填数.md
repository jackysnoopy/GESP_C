# P3051 矩阵填数

## 思路

核心算法：DP

## 解题思路

### 问题分析
给定 n 堆物品，每堆有当前数量 a[i] 和目标数量 b[i]。每次操作可以从一堆取一个物品放到相邻堆（环形排列）。求最少操作次数。

### 核心思路
差分 + 前缀和：计算每堆需要的净变化 diff[i] = b[i] - a[i]，然后利用前缀和的绝对值之和得到答案。

### 算法流程
1. 计算 diff[i] = b[i] - a[i]
2. 计算前缀和 prefix[i+1] = prefix[i] + diff[i]
3. 答案 = Σ|prefix[i]| (i = 0 ~ n-1)

### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n), b(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i] >> b[i];
    }
    
    // Calculate net change needed for each pile
    vector<int> diff(n);
    for (int i = 0; i < n; i++) {
        diff[i] = b[i] - a[i];
    }
    
    // Calculate prefix sums of diff
    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; i++) {
        prefix[i + 1] = prefix[i] + diff[i];
    }
    
    // The answer is the sum of absolute values of prefix sums
    // divided by n (for circular distance)
    long long ans = 0;
    for (int i = 0; i < n; i++) {
        ans += abs(prefix[i]);
    }
    
    cout << ans << "\n";
    return 0;
}
```
