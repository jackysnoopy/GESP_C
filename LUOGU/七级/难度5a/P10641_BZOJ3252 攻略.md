# P10641 BZOJ3252 攻略

## 思路

核心算法：贪心+堆

每次选择从根到叶子权值和最大的路径，将路径上节点权值置零，重复 k 次。使用优先队列维护每个节点到叶子的最大路径和，通过惰性删除实现高效更新。

## 代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;

    vector<long long> w(n + 1);
    for (int i = 1; i <= n; i++) cin >> w[i];

    vector<vector<int>> children(n + 1);
    vector<int> parent(n + 1, 0);
    vector<bool> is_child(n + 1, false);
    for (int i = 0; i < n - 1; i++) {
        int u, v;
        cin >> u >> v;
        children[u].push_back(v);
        parent[v] = u;
        is_child[v] = true;
    }

    int root = 1;
    for (int i = 1; i <= n; i++) {
        if (!is_child[i]) {
            root = i;
            break;
        }
    }

    vector<int> order;
    {
        vector<int> q;
        q.push_back(root);
        int head = 0;
        while (head < (int)q.size()) {
            int u = q[head++];
            order.push_back(u);
            for (int v : children[u]) q.push_back(v);
        }
    }

    vector<long long> mx(n + 1);
    vector<int> bc(n + 1, -1);
    vector<priority_queue<pair<long long, int>>> heaps(n + 1);

    for (int i = (int)order.size() - 1; i >= 0; i--) {
        int u = order[i];
        mx[u] = w[u];
        bc[u] = -1;
        for (int v : children[u]) {
            heaps[u].push({mx[v], v});
        }
        if (!heaps[u].empty()) {
            mx[u] = w[u] + heaps[u].top().first;
            bc[u] = heaps[u].top().second;
        }
    }

    long long ans = 0;
    for (int iter = 0; iter < k; iter++) {
        if (mx[root] <= 0) break;
        ans += mx[root];

        vector<int> path;
        int u = root;
        while (u != -1) {
            path.push_back(u);
            w[u] = 0;
            u = bc[u];
        }

        for (int i = (int)path.size() - 1; i >= 0; i--) {
            int node = path[i];
            while (!heaps[node].empty() && mx[heaps[node].top().second] != heaps[node].top().first) {
                heaps[node].pop();
            }
            if (heaps[node].empty()) {
                mx[node] = 0;
                bc[node] = -1;
            } else {
                mx[node] = heaps[node].top().first;
                bc[node] = heaps[node].top().second;
            }
            if (parent[node] != 0) {
                heaps[parent[node]].push({mx[node], node});
            }
        }
    }

    cout << ans << endl;
    return 0;
}
```

## 解题思路

### 问题分析
给定一棵有根树，每个节点有权值。每次操作可以选择从根到某叶子的一条路径，将路径上所有节点权值加到答案中，然后将这些节点权值置零。重复k次操作，求最大答案。

### 核心思路
每次选择从根到叶子权值和最大的路径。使用优先队列（最大堆）维护每个节点到叶子的最大路径和，通过惰性删除实现高效更新。自底向上计算每个节点的mx值（到叶子的最大路径和）和bc值（最优子节点）。每次操作后，沿路径更新所有祖先节点的mx值。

### 算法流程
1. 读入树结构和节点权值，确定根节点
2. BFS确定拓扑序，自底向上预处理：
   - mx[u] = w[u] + max(mx[children])，记录到叶子的最大路径和
   - bc[u] = 最优子节点
   - 使用堆维护子节点的mx值
3. 执行k次操作：
   - 若mx[root] <= 0则停止
   - 累加mx[root]到答案
   - 沿bc路径将所有节点权值置零
   - 自底向上更新路径上所有节点的mx值（惰性删除过期堆顶）
4. 输出答案

### 复杂度分析
- 时间复杂度：O(n + k log n)，预处理O(n)，每次操作O(路径长度 × log n)
- 空间复杂度：O(n)，树结构和堆
