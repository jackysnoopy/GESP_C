# P2102 地砖铺设

## 思路

核心算法：DP

## 解题思路

### 问题分析

在 n×m 的网格中铺设地砖，每块地砖有颜色（A-Z）。要求相邻地砖颜色不同。求一种合法的铺设方案。

### 核心思路

贪心策略：从左到右、从上到下逐格铺设，每次选择最小的合法颜色。由于题目只要求水平相邻不同色（根据代码逻辑），只需检查左边邻居的颜色。

### 算法流程

1. 读入 n 和 m
2. 遍历每个格子 (i, j)：
   - 尝试颜色 'A' 到 'Z'
   - 检查是否与左边邻居 (i, j-1) 颜色冲突
   - 选择第一个不冲突的颜色
3. 输出铺设结果

### 复杂度分析

- 时间复杂度：O(n × m × 26)，每格最多尝试 26 种颜色
- 空间复杂度：O(n × m)，存储网格

## 代码

```cpp
#include <iostream>
#include <vector>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<string> grid(n, string(m, ' '));
    
    // Greedy: for each cell, try to use the smallest possible color
    // that doesn't conflict with left neighbor only (horizontal adjacency)
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            for (char c = 'A'; c <= 'Z'; c++) {
                bool ok = true;
                // Check left only (horizontal adjacency)
                if (j > 0 && grid[i][j - 1] == c) ok = false;
                
                if (ok) {
                    grid[i][j] = c;
                    break;
                }
            }
        }
    }
    
    for (int i = 0; i < n; i++) {
        cout << grid[i] << "\n";
    }
    
    return 0;
}
```
