# P7401 山峰计数

## 题目描述

给定 n 个点的坐标 (x[i], y[i])，其中 x 坐标递增。一个点 i（1 < i < n）被称为"山峰"，如果 y[i] > y[i-1] 且 y[i] > y[i+1]。

求序列中山峰的数量。

## 输入格式

第一行两个整数 n 和 h，分别表示点数和某个参数（本题中 h 未使用）。
接下来 n 行，每行两个整数 x[i] 和 y[i]，表示第 i 个点的坐标。

## 输出格式

一个整数，表示山峰的数量。

## 样例

**样例输入：**
```
5 1
1 1
2 3
3 2
4 4
5 1
```

**样例输出：**
```
2
```

## 提示

- 3 ≤ n ≤ 100000

## 解题思路

### 问题分析

本题是一个简单的线性扫描问题，检查每个中间点是否是局部最大值。

### 核心思路

遍历所有中间点（下标 1 到 n-2），检查是否满足山峰条件：
- y[i] > y[i-1]（比左边高）
- y[i] > y[i+1]（比右边高）

### 算法流程

1. 读入所有点的坐标
2. 遍历 i = 1, 3, 5, ...（每隔一个点检查，步长为 2）
3. 如果 y[i] > y[i-1] 且 y[i] > y[i+1]，则计数加一
4. 输出计数结果

### 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, h;
    cin >> n >> h;
    vector<int> x(n), y(n);
    for (int i = 0; i < n; i++) cin >> x[i] >> y[i];
    int ans = 0;
    for (int i = 1; i < n - 1; i += 2) {
        if (y[i] > y[i-1] && y[i] > y[i+1]) ans++;
    }
    cout << ans << endl;
    return 0;
}
```
