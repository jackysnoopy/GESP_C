# P2511 消灭病毒

## 思路

核心算法：搜索

## 解题思路

### 问题分析
给定 n 段长度 L[i]，将它们分成若干段，使得每段总长度不超过 mid，且段数不超过 m+1。求最小的 mid 值，并统计达到该最小值的方案数（对 10007 取模）。

### 核心思路
二分答案 + DP 计数。二分最小的最大段长度，然后用 DP 统计满足条件的方案数。

### 算法流程
1. 二分查找最小的 mid：
   - check(mid)：贪心地将段拼接，若当前段加上 L[i] 超过 mid 则新开一段，统计需要的段数
   - 若段数 <= m+1，说明 mid 可行
2. 用 DP 统计方案数：
   - dp[i][j] 表示前 i 段分成 j 段且每段不超过 answer 的方案数
   - 用前缀和优化转移，二分找到合法的左边界
3. 累加所有 dp[n][j] (1 ≤ j ≤ m+1) 得到总方案数

### 复杂度分析
- 时间复杂度：O(n × m × log n)
- 空间复杂度：O(n × m)

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MOD = 10007;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<int> L(n);
    long long total = 0;
    for (int i = 0; i < n; i++) {
        cin >> L[i];
        total += L[i];
    }
    
    // Binary search on minimum maximum segment length
    int low = *max_element(L.begin(), L.end());
    int high = total;
    int answer = high;
    
    while (low <= high) {
        int mid = (low + high) / 2;
        
        // Check if we can split into segments with max length <= mid
        int cuts = 0;
        long long cur_sum = 0;
        
        for (int i = 0; i < n; i++) {
            if (cur_sum + L[i] > mid) {
                cuts++;
                cur_sum = L[i];
            } else {
                cur_sum += L[i];
            }
        }
        
        if (cuts <= m) {
            answer = mid;
            high = mid - 1;
        } else {
            low = mid + 1;
        }
    }
    
    // Count number of ways to achieve minimum
    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; i++) {
        prefix[i + 1] = prefix[i] + L[i];
    }
    
    // DP to count ways
    vector<vector<long long>> dp(n + 1, vector<long long>(m + 2, 0));
    vector<vector<long long>> sum_dp(n + 1, vector<long long>(m + 2, 0));
    
    dp[0][0] = 1;
    for (int i = 1; i <= n; i++) {
        sum_dp[0][i] = dp[0][i - 1];
    }
    
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= min(i, m + 1); j++) {
            // Find the range of valid previous positions
            int left = 0, right = i - 1;
            while (left < right) {
                int mid = (left + right) / 2;
                if (prefix[i] - prefix[mid] <= answer) {
                    right = mid;
                } else {
                    left = mid + 1;
                }
            }
            
            if (prefix[i] - prefix[left] <= answer) {
                dp[i][j] = (sum_dp[left][j - 1] - (left > 0 ? sum_dp[left - 1][j - 1] : 0) + MOD) % MOD;
            }
        }
        
        for (int j = 0; j <= m + 1; j++) {
            sum_dp[i][j] = (sum_dp[i - 1][j] + dp[i][j]) % MOD;
        }
    }
    
    long long ways = 0;
    for (int j = 1; j <= m + 1; j++) {
        ways = (ways + dp[n][j]) % MOD;
    }
    
    cout << answer << " " << ways << "\n";
    return 0;
}
```
