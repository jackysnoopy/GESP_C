# B4280 [蓝桥杯青少年组国赛 2023] 数学实验

## 思路

核心算法：区间DP（262144 经典题）

设 `dp[v][i]` 表示从位置 `i` 开始的连续子数组能合并成的最大右端点（合并后值为 `v`）。

- 基础：`dp[a[i]][i] = i`（单个元素本身）
- 转移：`dp[v][i] = dp[v-1][dp[v-1][i]+1]`，即先将 `i..dp[v-1][i]` 合并为 `v-1`，再将 `dp[v-1][i]+1..dp[v-1][dp[v-1][i]+1]` 合并为 `v-1`，最后将两个 `v-1` 合并为 `v`。

由于每个数在 1~80 之间，最多能合并到约 80+18=98，枚举 `v` 从 1 到 100，用滚动数组优化空间。

时间复杂度 O(100N)，空间 O(N)。

## 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    vector<int> a(n + 1);
    for (int i = 1; i <= n; i++) cin >> a[i];
    
    const int MAXV = 100;
    vector<vector<int>> dp(2, vector<int>(n + 1, 0));
    
    int ans = 0;
    for (int v = 1; v <= MAXV; v++) {
        int cur = v & 1;
        int pre = 1 - cur;
        
        for (int i = 1; i <= n; i++) {
            dp[cur][i] = (a[i] == v) ? i : 0;
        }
        
        for (int i = 1; i <= n; i++) {
            if (dp[pre][i] > 0) {
                int j = dp[pre][i] + 1;
                if (j <= n && dp[pre][j] > 0) {
                    dp[cur][i] = max(dp[cur][i], dp[pre][j]);
                }
            }
            if (dp[cur][i] > 0) ans = v;
        }
    }
    
    cout << ans << endl;
    return 0;
}
```

## 解题思路

### 问题分析

给定 n 个数的序列，相邻两个相同的数可以合并为原数+1。求通过合并能得到的最大数值。

### 核心思路

动态规划（262144 经典题）。设 dp[v][i] 表示从位置 i 开始的连续子数组能合并成的最大右端点（合并后值为 v）。转移方程：dp[v][i] = dp[v-1][dp[v-1][i]+1]。

### 算法流程

1. 初始化 dp[a[i]][i] = i（单个元素）
2. 枚举值 v 从 1 到 100：
   - 对每个位置 i，若 dp[v-1][i] > 0：
     - 检查 j = dp[v-1][i]+1 处能否合并出 v-1
     - 若能则 dp[v][i] = dp[v-1][j]
3. 记录最大的 v 使得存在 dp[v][i] > 0
4. 输出最大值

### 复杂度分析

- 时间复杂度：O(100×n)，枚举值 × 遍历位置
- 空间复杂度：O(n)，滚动数组优化
