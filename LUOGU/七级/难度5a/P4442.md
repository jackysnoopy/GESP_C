# P4442 迷宫最短路

## 题目描述

给定一个 n×m 的网格迷宫，包含以下元素：
- `C`：起点
- `F`：终点
- `#`：墙壁（不可通过）
- 其他字符：空地（可通过）

求从起点到终点的最短路径长度。如果无法到达，输出 "nemoguce"。

## 输入格式

第一行两个整数 n 和 m，表示网格的行数和列数。
接下来 n 行，每行 m 个字符，表示网格。

## 输出格式

一个整数表示最短路径长度，或 "nemoguce" 表示不可达。

## 样例

**样例输入：**
```
5 5
.....
.C#..
..#..
..#F.
.....
```

**样例输出：**
```
8
```

## 提示

- 1 ≤ n, m ≤ 1000

## 解题思路

### 问题分析

本题是经典的网格最短路径问题，使用 BFS 求解。

### 核心思路

使用**广度优先搜索（BFS）**：
1. 从起点 C 开始，向四个方向扩展
2. 跳过墙壁 `#` 和越界位置
3. 记录每个位置的最短距离
4. 到达终点 F 时输出距离

### 算法流程

1. 读入网格，找到起点 C 和终点 F
2. BFS 从起点开始扩展
3. 对于每个位置，尝试上下左右四个方向
4. 如果到达终点，输出距离
5. 如果 BFS 结束未到达终点，输出 "nemoguce"

### 复杂度分析

- 时间复杂度：O(n × m)
- 空间复杂度：O(n × m)

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <string>
using namespace std;

const int dx[] = {0, 0, 1, -1};
const int dy[] = {1, -1, 0, 0};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, m;
    cin >> n >> m;
    vector<string> grid(n);
    int sx, sy, fx, fy;
    for (int i = 0; i < n; i++) {
        cin >> grid[i];
        for (int j = 0; j < m; j++) {
            if (grid[i][j] == 'C') { sx = i; sy = j; }
            else if (grid[i][j] == 'F') { fx = i; fy = j; }
        }
    }
    vector<vector<int>> dist(n, vector<int>(m, -1));
    queue<pair<int, int>> q;
    q.push({sx, sy});
    dist[sx][sy] = 0;
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        if (x == fx && y == fy) { cout << dist[x][y] << endl; return 0; }
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d], ny = y + dy[d];
            if (nx >= 0 && nx < n && ny >= 0 && ny < m && grid[nx][ny] != '#' && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                q.push({nx, ny});
            }
        }
    }
    cout << "nemoguce" << endl;
    return 0;
}
```
