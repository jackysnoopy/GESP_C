# [NOIP 2013 提高组] 货车运输

## 题目描述

A 国有 $n$ 座城市，编号从 $1$ 到 $n$，城市之间有 $m$ 条双向道路。每一条道路对车辆都有重量限制，简称限重。

现在有 $q$ 辆货车在运输货物，司机们想知道每辆车在不超过车辆限重的情况下，最多能运多重的货物。

## 输入格式

第一行有两个用一个空格隔开的整数 $n,m$，表示 A 国有 $n$ 座城市和 $m$ 条道路。  

接下来 $m$ 行每行三个整数 $x, y, z$，每两个整数之间用一个空格隔开，表示从 $x$ 号城市到 $y$ 号城市有一条限重为 $z$ 的道路。    
注意：$x \neq y$，两座城市之间可能有多条道路。

接下来一行有一个整数 $q$，表示有 $q$ 辆货车需要运货。

接下来 $q$ 行，每行两个整数 $x,y$，之间用一个空格隔开，表示一辆货车需要从 $x$ 城市运输货物到 $y$ 城市，保证 $x \neq y$。

## 输出格式

共有 $q$ 行，每行一个整数，表示对于每一辆货车，它的最大载重是多少。  
如果货车不能到达目的地，输出 $-1$。

## 样例

Sample 1:
Input:
4 3
1 2 4
2 3 3
3 1 1
3
1 3
1 4
1 3
Output:
3
-1
3

## 提示

对于 $30\%$ 的数据，$1 \le n < 1000$，$1 \le m < 10,000$，$1\le q< 1000$；

对于 $60\%$ 的数据，$1 \le n < 1000$，$1 \le m < 5\times 10^4$，$1 \le q< 1000$；

对于 $100\%$ 的数据，$1 \le n < 10^4$，$1 \le m < 5\times 10^4$，$1 \le q< 3\times 10^4 $，$0 \le z \le 10^5$。

## 解题思路

### 问题分析
- 给定一个带权无向图，每条边有重量限制。
- 查询从城市 $x$ 到城市 $y$ 的路径上，最大能通过的重量（即路径上最小边权的最大值）。
- 如果 $x$ 和 $y$ 不连通，输出 $-1$。

### 核心思路
- 最大生成树 + LCA（最近公共祖先）。
- 最大生成树保证了任意两点间路径上的最小边权最大。
- 在最大生成树上使用倍增法求LCA，同时维护路径上的最小边权。

### 算法流程
1. 使用Kruskal算法构建最大生成树。
2. 在最大生成树上进行DFS，预处理每个节点的父节点、深度和到父节点的边权。
3. 使用倍增法预处理LCA和路径最小边权。
4. 对于每个查询 $(x, y)$：
   - 如果 $x$ 和 $y$ 不连通，输出 $-1$。
   - 否则，使用LCA查询路径上的最小边权。

### 复杂度分析
- 时间复杂度：$O((N + M) \log N + Q \log N)$。
- 空间复杂度：$O(N \log N)$。

## 参考代码

```cpp

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 10005;
const int MAXM = 50005;
const int LOG = 15;

struct Edge {
    int u, v, w;
};

int n, m, q;
Edge edges[MAXM];
int parent[MAXN], depth[MAXN], minEdge[MAXN][LOG];
bool visited[MAXN];
vector<pair<int, int>> tree[MAXN];

int find(int x) {
    return parent[x] == x ? x : parent[x] = find(parent[x]);
}

void kruskal() {
    for (int i = 1; i <= n; i++) parent[i] = i;
    sort(edges, edges + m, [](const Edge& a, const Edge& b) {
        return a.w > b.w;
    });
    
    int cnt = 0;
    for (int i = 0; i < m && cnt < n - 1; i++) {
        int u = find(edges[i].u);
        int v = find(edges[i].v);
        if (u != v) {
            parent[u] = v;
            tree[edges[i].u].push_back({edges[i].v, edges[i].w});
            tree[edges[i].v].push_back({edges[i].u, edges[i].w});
            cnt++;
        }
    }
}

void dfs(int u, int p, int w, int d) {
    visited[u] = true;
    parent[u] = p;
    depth[u] = d;
    minEdge[u][0] = w;
    
    for (int i = 1; i < LOG; i++) {
        minEdge[u][i] = min(minEdge[u][i-1], minEdge[parent[u]][i-1]);
        parent[u] = parent[parent[u]];
    }
    
    for (auto& [v, w2] : tree[u]) {
        if (!visited[v]) {
            dfs(v, u, w2, d + 1);
        }
    }
}

int lca(int x, int y) {
    if (depth[x] < depth[y]) swap(x, y);
    
    int diff = depth[x] - depth[y];
    int ans = 1e9;
    
    for (int i = 0; i < LOG; i++) {
        if ((diff >> i) & 1) {
            ans = min(ans, minEdge[x][i]);
            x = parent[x];
        }
    }
    
    if (x == y) return ans;
    
    for (int i = LOG - 1; i >= 0; i--) {
        if (parent[x][i] != parent[y][i]) {
            ans = min(ans, min(minEdge[x][i], minEdge[y][i]));
            x = parent[x][i];
            y = parent[y][i];
        }
    }
    
    ans = min(ans, min(minEdge[x][0], minEdge[y][0]));
    return ans;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        cin >> edges[i].u >> edges[i].v >> edges[i].w;
    }
    
    kruskal();
    
    for (int i = 1; i <= n; i++) {
        if (!visited[i]) {
            dfs(i, 0, 1e9, 0);
        }
    }
    
    cin >> q;
    while (q--) {
        int x, y;
        cin >> x >> y;
        if (find(x) != find(y)) {
            cout << "-1\n";
        } else {
            cout << lca(x, y) << "\n";
        }
    }
    
    return 0;
}

```
