# [GKS 2015 #D] Virtual Rabbit

## 题目描述

虚拟兔子若 X 秒未喂食会死。Alice 有可用时间段（起床到上班、下班到睡觉）。求最小喂食次数使兔子存活到第 D 天。

## 输入格式

T 个测试用例。每个：G W H B X（时间）和 D（天数）。

## 输出格式

Case #x: y（y 为最小次数或 -1）。

## 样例

输入：
```
3
08:00:00
09:00:00
18:00:00
22:00:00
12:00:00
100
...
```
输出：
```
Case #1: 200
Case #2: -1
Case #3: 86401
```

## 提示

- D 可达 10^14

## 解题思路

### 核心思路

贪心：每天在可用窗口末尾喂食，最大化每次喂食覆盖时间。

### 复杂度分析

- 时间：O(D) 或 O(1)（大 D 时用公式）
- 空间：O(1)
