# [2018 KAIST RUN Spring] Touch The Sky

## 题目描述

N 个气球，每个有高度限制 L_i 和容量 D_i。从高度 0 出发，每次使用一个气球（当前高度 <= L_i 时才能使用），高度增加 D_i。最大化使用气球数。

## 输入格式

第一行 N。
接下来 N 行每行 L_i D_i。

## 输出格式

一个整数。

## 样例

输入：
```
3
1 4
1 5
9 2
```
输出：
```
2
```

## 提示

- 1 <= N <= 250,000

## 解题思路

### 核心思路

贪心 + 优先队列。按 L_i 排序，维护小根堆，优先使用 D_i 最小的气球。

### 复杂度分析

- 时间：O(N log N)
- 空间：O(N)
