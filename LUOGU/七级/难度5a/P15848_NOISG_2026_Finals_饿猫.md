# [NOISG 2026 Finals] 饿猫 / Famished Cats

## 题目描述

n 栋猫屋排成一行，每栋有燃料 f[i]。卡车从食物银行出发，初始燃料 x。距离 d[i]。可以交换第 i 和第 n-i+1 栋的燃料。求最远能到第几栋以及最少交换次数。

## 输入格式

第一行 n x。
第二行 d[1]..d[n]。
第三行 f[1]..f[n]。

## 输出格式

两个整数：最远到达的栋号 D 和最少交换次数 S。

## 样例

输入：
```
6 1
1 1 3 1 1 6
1 1 1 4 3 2
```
输出：
```
5 1
```

## 提示

- 2 <= n <= 500,000，n 为偶数

## 解题思路

### 问题分析

每次到达一栋房可以获得燃料。可以交换对称位置的燃料。

### 核心思路

贪心模拟：按顺序遍历房屋，对于每对对称位置 (i, n-i+1)，选择燃料更多的那个。

### 算法流程

1. 从左到右遍历
2. 对于每对 (i, n-i+1)，选择 max(f[i], f[n-i+1])
3. 如果选择了交换后的位置，交换次数+1
4. 燃料不足时停止

### 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)
