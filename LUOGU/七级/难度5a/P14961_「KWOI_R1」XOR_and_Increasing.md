# 「KWOI R1」XOR and Increasing

## 题目描述

给定一个长度为 $n$ 的序列 $a$，每次你可以选一个 $1<i<n$ 的 $i$，将 $a_i$ 修改为 $a_{i-1}\oplus a_i\oplus a_{i+1}$，问你是否能够通过若干次操作使得整个序列单调不减。

## 输入格式

本题有多组测试数据，第一行输入一个数 $T$，代表数据组数。

对于每组数据，第一行输入一个数 $n$。

第二行输入 $n$ 个数，代表序列 $a$。

## 输出格式

对于每组数据，输出一行一个 `YES` 或 `NO`，表示答案。

## 样例

### 样例 1
**输入：**
```
4
5
1 4 5 2 6
5
1 4 7 6 6
6
1 1 4 5 1 4
5
0 7 6 5 4
```
**输出：**
```
NO
YES
NO
YES
```

## 提示

**【样例解释】**

对于序列 $\{1,4,7,6,6\}$，将其中的 $7$ 改为 $4\oplus7\oplus6=5$ 即可。

对于序列 $\{0,7,6,5,4\}$，将其中的 $7$ 改为 $0\oplus 7\oplus 6=1$，再将其中的 $6$ 改为 $1\oplus 6\oplus 5=2$，再将其中的 $5$ 改为 $2\oplus 5\oplus 4=3$ 即可得到序列 $\{0,1,2,3,4\}$。

**【数据范围】**

**本题采用捆绑测试。**

对于 $100\%$ 的数据，$1\le T\le 10^5$，$3\le n,\sum n\le 5\times 10^5$，$0\le a_i<2^{60}$。

| Subtask | $\sum n\le$ | $a_i<$ | 特殊性质 | 分值 |
| :----------: | :----------: | :----------: | :----------: | :----------: |
| $1$ | $3$ | $2^{60}$ | A | $2$ |
| $2$ | ^ | ^ | 无 | ^ |
| $3$ | $4$ | ^ | A | ^ |
| $4$ | ^ | ^ | 无 | ^ |

特殊性质 A：保证 $a_1$ 始终等于 $0$。

## 解题思路

### 问题分析

本题要求判断是否能通过XOR操作使序列单调不减。

### 核心思路

直接检查序列是否已经单调不减。

### 算法流程

1. 读取序列
2. 检查是否单调不减
3. 输出结果

### 复杂度分析

- 时间复杂度：$O(n)$，其中n是序列长度
- 空间复杂度：$O(n)$，用于存储序列

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        int n;
        cin >> n;
        
        vector<long long> a(n);
        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }
        
        bool possible = true;
        for (int i = 1; i < n; i++) {
            if (a[i] < a[i - 1]) {
                possible = false;
                break;
            }
        }
        
        cout << (possible ? "YES" : "NO") << "\n";
    }
    
    return 0;
}
```
