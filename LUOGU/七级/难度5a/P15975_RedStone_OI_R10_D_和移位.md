# 「RedStone OI R10 D」和移位

## 题目描述

有根树，每个节点有权值 a_i。f(p) 为子树 p 中所有节点权值的 GCD。F = sum f(p)。可以最多一次操作：将一棵子树移到另一个位置。最大化 F'。

## 输入格式

第一行 n。
第二行 a_1..a_n。
接下来 n-1 行每行 u_i v_i（u_i 是 v_i 的父节点）。

## 输出格式

一个整数。

## 样例

输入：
```
5
6 10 15 14 21
1 2
1 3
2 4
3 5
```
输出：
```
53
```

## 提示

- n <= 2*10^5

## 解题思路

### 核心思路

枚举要移动的子树和目标位置，利用 GCD 的性质加速计算。

### 算法流程

1. 计算初始 F
2. 对每个可能的移动，计算 F 的变化量
3. 取最大值

### 复杂度分析

- 暴力：O(n^3)
- 优化后：O(n log^2 n)
