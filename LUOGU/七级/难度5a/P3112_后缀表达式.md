# P3112 后缀表达式

## 思路

核心算法：栈

## 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

struct Cow {
    int height, weight, strength;
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, h;
    cin >> n >> h;
    
    vector<Cow> cows(n);
    for (int i = 0; i < n; i++) {
        cin >> cows[i].height >> cows[i].weight >> cows[i].strength;
    }
    
    // Sort by strength + weight (larger first)
    sort(cows.begin(), cows.end(), [](const Cow& a, const Cow& b) {
        return a.strength + a.weight > b.strength + b.weight;
    });
    
    // DP: dp[i][j] = maximum safety factor using first i cows with total weight j
    vector<vector<int>> dp(n + 1, vector<int>(n * 1000 + 1, -1));
    dp[0][0] = 1e9;
    
    int total_height = 0;
    int best_safety = -1;
    
    for (int i = 0; i < n; i++) {
        for (int j = n * 1000; j >= 0; j--) {
            if (dp[i][j] < 0) continue;
            
            // Don't take this cow
            dp[i + 1][j] = max(dp[i + 1][j], dp[i][j]);
            
            // Take this cow
            if (j + cows[i].weight <= cows[i].strength) {
                int new_safety = min(dp[i][j], cows[i].strength - j - cows[i].weight);
                dp[i + 1][j + cows[i].weight] = max(dp[i + 1][j + cows[i].weight], new_safety);
            }
        }
        
        // Check if we can reach height h
        if (dp[i + 1][0] >= 0) {
            int height_sum = 0;
            for (int k = 0; k <= i; k++) {
                height_sum += cows[k].height;
            }
            if (height_sum >= h) {
                best_safety = max(best_safety, dp[i + 1][0]);
            }
        }
    }
    
    if (best_safety < 0) {
        cout << "Mark is too tall\n";
    } else {
        cout << best_safety << "\n";
    }
    
    return 0;
}
```

## 解题思路

### 问题分析
有n头牛，每头牛有三个属性：身高h、体重w、承重力s（即能承受的上方总重量）。牛可以叠成一摞，但要求最底下的牛承重力 ≥ 上方所有牛的体重之和。同时要求总高度达到h，求最大安全系数（即所有牛的s - 上方体重之和的最小值）。

### 核心思路
这是一个带约束的背包问题。按(s+w)从大到小排序，确保强壮的牛在下方。使用二维DP：dp[i][j]表示考虑前i头牛、总体重为j时的最大安全系数。状态转移时，考虑放或不放当前牛，若放则需检查当前牛的承重力是否足够。

### 算法流程
1. 读入n头牛的(h, w, s)和目标高度h
2. 按s+w从大到小排序（确保强壮的牛先放）
3. 初始化dp[0][0] = 1e9（无穷大安全系数），其余为-1（不可达）
4. 遍历每头牛：
   - 不放：dp[i+1][j] = max(dp[i+1][j], dp[i][j])
   - 放：若j+w ≤ s（承重力足够），则dp[i+1][j+w] = max(..., min(dp[i][j], s-j-w))
5. 检查dp[i+1][0]对应的累计高度是否≥h，更新最大安全系数
6. 若无法达到高度h，输出"Mark is too tall"；否则输出最大安全系数

### 复杂度分析
- 时间复杂度：O(n × n × max_weight)，其中max_weight为单牛最大体重
- 空间复杂度：O(n × n × max_weight)，DP数组
