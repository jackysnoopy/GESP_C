# P3686 [CERC2016] 爵士之旅 Jazz Journey

## 题目描述

Ivan 正在为他的爵士乐队计划一场规模盛大的欧洲巡演。在欧洲一共有 $n$ 个城市，编号依次为 $1 \sim n$。Ivan 计划举办 $d$ 场演出，分别在城市 $a_1$，$a_2$，$\dots$，$a_d$，并且严格遵循这个顺序。

有两种可以购买的机票：
1. 从 $a$ 到 $b$ 的单程票
2. 从 $a$ 到 $b$ 的双程票

给定可以购买的机票集合，每种机票都是无限量供应的。请帮助 Ivan 找到一种最省钱的方案。

## 输入格式

第一行包含两个正整数 $n,d$，分别表示城市的个数和巡演的次数。

第二行包含 $d$ 个正整数 $a_1,a_2,\dots,a_d$，依次表示巡演计划中每一场所在的城市。

接下来一行包含一个正整数 $m$，表示机票的种类数。

接下来 $m$ 行，每行描述一种机票。

## 输出格式

输出一行一个整数，即完成巡演所需支付的最小机票总费用。

## 样例

### 样例1
**输入：**
```
2 5
1 2 1 2 1
4
1 2 R 6
1 2 O 3
2 1 O 3
1 2 R 5
```

**输出：**
```
10
```

## 解题思路

这是一道图论+最短路题目。

**算法**：
1. 对每段路程使用Dijkstra算法求最短路
2. 累加所有路程的最小花费

**时间复杂度**：$O(d \cdot (n + m) \log n)$

## 相关题目

- Dijkstra算法
- 最短路
