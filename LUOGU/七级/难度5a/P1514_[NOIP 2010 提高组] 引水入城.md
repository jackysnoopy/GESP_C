# P1514 [NOIP 2010 提高组] 引水入城

## 思路

核心算法：BFS+贪心覆盖

## 代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
#include <cstring>
using namespace std;

const int MAXN = 510;

int n, m;
int h[MAXN][MAXN];
bool vis[MAXN][MAXN];
int cover[MAXN][MAXN];
int maxr[MAXN];

int dx[] = {0, 0, 1, -1};
int dy[] = {1, -1, 0, 0};

void bfs(int sx, int sy, int id) {
    queue<pair<int, int>> q;
    q.push({sx, sy});
    vis[sx][sy] = true;
    cover[sx][sy] = id;
    maxr[id] = max(maxr[id], sx);
    
    while (!q.empty()) {
        int x = q.front().first;
        int y = q.front().second;
        q.pop();
        
        for (int d = 0; d < 4; d++) {
            int nx = x + dx[d];
            int ny = y + dy[d];
            if (nx < 1 || nx > n || ny < 1 || ny > m) continue;
            if (vis[nx][ny]) continue;
            if (h[nx][ny] >= h[x][y]) continue;
            
            vis[nx][ny] = true;
            cover[nx][ny] = id;
            maxr[id] = max(maxr[id], nx);
            q.push({nx, ny});
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            cin >> h[i][j];
        }
    }
    
    memset(vis, false, sizeof(vis));
    memset(cover, 0, sizeof(cover));
    memset(maxr, 0, sizeof(maxr));
    
    int id = 0;
    for (int j = 1; j <= m; j++) {
        if (!vis[1][j]) {
            id++;
            bfs(1, j, id);
        }
    }
    
    bool ok = true;
    for (int j = 1; j <= m; j++) {
        if (cover[n][j] == 0) {
            ok = false;
            break;
        }
    }
    
    if (!ok) {
        int cnt = 0;
        for (int j = 1; j <= m; j++) {
            if (cover[n][j] == 0) cnt++;
        }
        cout << "0\n" << cnt << "\n";
    } else {
        vector<pair<int, int>> intervals;
        for (int i = 1; i <= id; i++) {
            if (maxr[i] == n) {
                int l = m + 1, r = 0;
                for (int j = 1; j <= m; j++) {
                    if (cover[n][j] == i) {
                        l = min(l, j);
                        r = max(r, j);
                    }
                }
                intervals.push_back({l, r});
            }
        }
        
        sort(intervals.begin(), intervals.end());
        
        int ans = 0;
        int cur = 0;
        int idx = 0;
        while (cur < m) {
            int nxt = cur;
            while (idx < (int)intervals.size() && intervals[idx].first <= cur + 1) {
                nxt = max(nxt, intervals[idx].second);
                idx++;
            }
            if (nxt == cur) {
                cout << "0\n" << m - cur << "\n";
                return 0;
            }
            cur = nxt;
            ans++;
        }
        
        cout << "1\n" << ans << "\n";
    }
    
    return 0;
}
```

## 解题思路

### 问题分析

n×m 的网格，第 1 行是水源，第 n 行是城市。水只能从高处流向低处（严格递减）。问能否从第 1 行的某些点引水覆盖第 n 行所有城市，若能则求最少水源数，若不能则求无法覆盖的城市数。

### 核心思路

BFS + 贪心区间覆盖。从第 1 行每个点做 BFS 找到能到达的区域，记录每个水源能覆盖第 n 行的区间，然后用区间覆盖贪心求最少水源数。

### 算法流程

1. 从第 1 行每个未访问的点做 BFS，标记每个点属于哪个水源的覆盖区域
2. 检查第 n 行所有城市是否都被覆盖
3. 若未全覆盖：输出 0 和未覆盖城市数
4. 若全覆盖：对每个能到达第 n 行的水源，记录其覆盖的区间 [l, r]
5. 按左端点排序，贪心选择最少区间覆盖 [1, m]

### 复杂度分析

- 时间复杂度：O(nm)，BFS 遍历每个点最多一次
- 空间复杂度：O(nm)
