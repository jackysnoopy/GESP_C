# P3147 [USACO16OPEN] 262144 P

## 思路

核心算法：区间DP

## 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    // dp[i][j] = maximum value achievable by merging elements from i to j
    vector<vector<int>> dp(n, vector<int>(n, 0));
    
    // Base case
    for (int i = 0; i < n; i++) {
        dp[i][i] = a[i];
    }
    
    int ans = *max_element(a.begin(), a.end());
    
    // Fill dp table
    for (int len = 2; len <= n; len++) {
        for (int i = 0; i + len <= n; i++) {
            int j = i + len - 1;
            
            for (int k = i; k < j; k++) {
                if (dp[i][k] == dp[k+1][j] && dp[i][k] > 0) {
                    dp[i][j] = max(dp[i][j], dp[i][k] + 1);
                }
            }
            
            ans = max(ans, dp[i][j]);
        }
    }
    
    cout << ans << "\n";
    return 0;
}
```

## 解题思路

### 问题分析

给定 n 个数的序列，相邻两个相同的数可以合并为原数+1。求通过合并能得到的最大数值。

### 核心思路

区间 DP。dp[i][j] 表示将区间 [i,j] 内的数合并后能得到的最大值。若存在分割点 k 使得 dp[i][k] = dp[k+1][j] 且均大于 0，则 dp[i][j] = dp[i][k] + 1。

### 算法流程

1. 初始化 dp[i][i] = a[i]
2. 按区间长度从小到大枚举 len = 2~n
3. 对每个区间 [i,j]，枚举分割点 k：
   - 若 dp[i][k] = dp[k+1][j] > 0，则 dp[i][j] = dp[i][k] + 1
4. 更新全局最大值 ans
5. 输出 ans

### 复杂度分析

- 时间复杂度：O(n³)，三重循环
- 空间复杂度：O(n²)
