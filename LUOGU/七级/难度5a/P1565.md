# P1565 牛宫

## 题目分析
- 给一个 `N×M` 的整数矩阵，求**面积最大的子矩形**，使其元素**总和 ≥ 0**（平均海拔在海平面之上 ⇔ sum ≥ 0）。
- `N,M ≤ 200`，元素绝对值最大 2×10^9。

## 解题思路
### 二维转一维
枚举上行 `u`、下行 `d`：
- 计算每列在区间 `[u,d]` 的列和 `s[j] = Σ_{i=u..d} a[i][j]`（用列前缀和 `O(1)` 求）。
- 在数组 `s[1..m]` 中找**最长的连续子段 `[l..r]` 使 `Σs[l..r] ≥ 0`**，宽 = `r-l+1`，面积 = `宽 × 高`。

### 最长非负和子段
设前缀和 `P[j] = Σ_{k=1..j} s[k]`，`P[0]=0`。
要找最长 `[l..r]` 满足 `P[r] - P[l-1] ≥ 0`，即 `P[r] ≥ P[l-1]`。
记 `i = l-1, j = r`，找最大的 `j - i` 满足 `P[j] ≥ P[i]`。

经典 **单调栈 O(M)**：
1. 从左到右扫描，把 `P[j] < 栈顶 P` 的下标依次入栈，得到 **严格递减的下标栈**（候选左端点 `i`）。
2. 从右往左扫描 `j = M..1`：只要栈顶 `P[stk[top]] ≤ P[j]`，更新答案 `j - stk[top]`，然后弹栈。
3. 这样每个候选左端点最多被使用一次，每次弹栈即获得最大可能的右端点。

### 复杂度
- 上下行枚举 `O(N^2)` = 4×10^4
- 每对内部 `O(M)` = 200
- 总 `O(N^2 · M) ≈ 8×10^6`，远低于 1s。

## 实现细节
- 列前缀 `col[i][j]`，便于 `O(1)` 取列段和。
- 用 `long long` 计算前缀和（最大可能 200×200×2e9 ≈ 8e13）。
- 注意答案是面积（一个非负整数），可在 int 范围（≤ 200×200=40000）内。

## 样例验证
3×2 矩阵：
```
4   0
-10 8
-2 -2
```
最大子矩形：`(1,1)`～`(1,2)`：4+0=4 ≥ 0，面积 2；
`(2,2)`：8 ≥ 0，面积 1；
`(1,1)`～`(2,2)`：4+0-10+8=2 ≥ 0，面积 4。
输出 `4`，与样例一致。
