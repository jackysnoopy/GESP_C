# P3672 小清新签到题

## 题目描述

给定自然数 $n$、$k$、$x$，你要求出第 $k$ 小的长度为 $n$ 的逆序对对数为 $x$ 的 $1\sim n$ 的排列 $a_1,a_2 ... a_n$。

注：逆序对为满足 $i<j$、$a_i>a_j$ 的 $(i,j)$。比较为字典序比较，即比较从前往后第一个不同的位置。第 $k$ 小从 $1$ 开始标号。

## 输入格式

一行三个自然数 $n$、$k$、$x$。

## 输出格式

输出满足条件的排列，一行n个数，用空格分隔。

## 样例

### 样例1
**输入：**
```
3 2 2
```

**输出：**
```
3 1 2
```

### 样例2
**输入：**
```
10 6 4
```

**输出：**
```
1 2 3 4 5 7 6 10 9 8
```

## 解题思路

这是一道排列组合题目。

**算法**：
1. 使用组合数计算每种选择的方案数
2. 贪心确定每个位置的数字
3. 考虑逆序对的限制

**时间复杂度**：$O(n^2)$
