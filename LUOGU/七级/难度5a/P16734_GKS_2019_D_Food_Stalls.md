# [GKS 2019 #D] Food Stalls

## 题目描述

在 N 个位置建 K 个摊位和 1 个仓库。位置 X[i]，建造费 C[i]。仓库在 j 时，摊位 i 额外费用 |X[j]-X[i]|。求最小总费用。

## 输入格式

T 个测试用例。每个：K N，然后 X 数组和 C 数组。

## 输出格式

Case #x: y。

## 样例

输入：
```
3
2 4
1 2 3 10
100 70 80 20
...
```
输出：
```
Case #1: 178
Case #2: 62
Case #3: 82
```

## 提示

- N 可达 10^5（最多 5 个大测试用例）

## 解题思路

### 核心思路

枚举仓库位置，计算每个摊位费用 = C[i] + |X[w]-X[i]|，选 K 个最小的。

### 复杂度分析

- 时间：O(N^2)（小数据），O(N^2 log N)（大数据需优化）
- 空间：O(N)
