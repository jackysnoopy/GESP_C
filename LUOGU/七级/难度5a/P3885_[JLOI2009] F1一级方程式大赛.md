# P3885 [JLOI2009] F1一级方程式大赛

## 题目描述

我们知道，在 F1 大赛中，如果赛车的其他参数都相同，赛车的速度主要取决于其载油量。载油太多就会降低赛车的速度，也会增加汽油的消耗速度；但载油太少就需要在比赛中增加进站加油的次数。

智囊团的一个关键任务就是在每次比赛前确定舒马赫的初始载油量以及进站加油的方案，使得舒马赫完成所有赛程的总时间尽量短。

## 输入格式

输入数据包括以下 $7$ 个数：
- 比赛的总圈数 $n$
- 理论上，没有油的空车跑一圈所需的时间
- 每增加 $1$ 公升汽油，赛车跑一圈所增加的时间
- 理论上，没有油的空车跑一圈所消耗的油量
- 每增加 $1$ 公升汽油，赛车跑一圈所增加的油量损耗
- 每次进站所需花费的时间
- 每次进站后，每加 $1$ 公升汽油，所需花费的时间

## 输出格式

- 第一行包括三个数：最短时间、初始载油量、进站加油次数
- 接下来是 $m$ 行，每行包括两个数：第 $i$ 次进站加油时已经完成的圈数、第 $i$ 次进站加油所加的油量

## 样例

### 样例1
**输入：**
```
3 100 0 10 0 20 0
```

**输出：**
```
300.000 30.000 0
```

## 解题思路

这是一道优化+模拟的题目。

**算法**：
1. 枚举不同的初始油量
2. 模拟每圈的运行情况
3. 找到最优方案

**时间复杂度**：$O(n \cdot fuel\_range)$
