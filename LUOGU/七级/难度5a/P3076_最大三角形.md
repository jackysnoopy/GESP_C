# P3076 最大三角形

## 思路

核心算法：几何

## 解题思路

### 问题分析

根据代码逻辑，本题涉及 n 头牛和 m 个初始位置，每头牛有起始位置和目标位置。需要计算将所有牛从起始位置移动到目标位置的总移动距离。

### 核心思路

贪心策略：按起始位置排序后依次处理每头牛。维护当前位置 cur_pos，对每头牛计算从当前位置到其起始位置的距离，再加上从起始位置到目标位置的距离。

### 算法流程

1. 读入 n 头牛的起始位置和目标位置
2. 按起始位置排序
3. 初始化 total = m（初始位置），cur_pos = 0
4. 遍历每头牛：
   - 计算从 cur_pos 到 start 的距离
   - 计算从 start 到 end 的距离
   - 更新 cur_pos = end
5. 输出总移动距离

### 复杂度分析

- 时间复杂度：O(n log n)，排序复杂度
- 空间复杂度：O(n)，存储牛的位置信息

## 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<pair<int, int>> cows(n);
    for (int i = 0; i < n; i++) {
        cin >> cows[i].first >> cows[i].second;
    }
    
    // Sort by start position
    sort(cows.begin(), cows.end());
    
    long long total = m;
    int cur_pos = 0;
    
    for (int i = 0; i < n; i++) {
        int start = cows[i].first;
        int end = cows[i].second;
        
        if (start >= cur_pos) {
            // Move to start, then to end
            total += (start - cur_pos);
            total += abs(end - start);
            cur_pos = end;
        } else {
            // Move to start, then to end
            total += (start - cur_pos);
            total += abs(end - start);
            cur_pos = end;
        }
    }
    
    cout << total << "\n";
    return 0;
}
```
