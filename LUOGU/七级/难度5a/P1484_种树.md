# P1484 种树

## 思路

核心算法：反悔贪心

## 解题思路

### 问题分析

有 n 个位置可以种树，每个位置有一个收益值 a[i]（可能为负）。选择 k 个位置种树，要求任意两个被选位置不相邻。求最大总收益。

### 核心思路

反悔贪心：每次选择当前最大收益的位置，同时标记其左右邻居不可选。关键操作是"反悔"——当选择位置 idx 后，将其值替换为 `a[l[idx]] + a[r[idx]] - a[idx]`，这相当于撤销之前的某个选择，允许后续重新决策。

### 算法流程

1. 用链表维护左右邻居关系，用优先队列维护可选位置
2. 每次弹出最大值位置 idx：
   - 若已访问或值≤0则跳过
   - 累加收益，标记 idx 及其左右邻居为已访问
   - 更新 idx 的值为 `a[l[idx]] + a[r[idx]] - a[idx]`
   - 重新链接链表，跳过已访问的邻居
   - 若新值>0则重新入队
3. 重复直到选够 k 个或队列为空

### 复杂度分析

- 时间复杂度：O(n log n)，每个元素最多入队出队一次
- 空间复杂度：O(n)，链表和优先队列

## 代码

```cpp
#include <iostream>
#include <queue>
#include <vector>
#include <algorithm>
using namespace std;

const int MAXN = 300010;

int n, k;
int a[MAXN], l[MAXN], r[MAXN];
bool vis[MAXN];
long long ans;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n >> k;
    
    priority_queue<pair<int, int>> pq;
    
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
        l[i] = i - 1;
        r[i] = i + 1;
        if (a[i] > 0) {
            pq.push({a[i], i});
        }
    }
    
    r[0] = 1;
    l[n + 1] = n;
    
    int cnt = 0;
    while (cnt < k && !pq.empty()) {
        int val = pq.top().first;
        int idx = pq.top().second;
        pq.pop();
        
        if (vis[idx]) continue;
        if (val <= 0) break;
        
        ans += val;
        vis[idx] = true;
        vis[l[idx]] = true;
        vis[r[idx]] = true;
        
        int new_val = a[l[idx]] + a[r[idx]] - a[idx];
        a[idx] = new_val;
        
        l[idx] = l[l[idx]];
        r[idx] = r[r[idx]];
        r[l[idx]] = idx;
        l[r[idx]] = idx;
        
        if (new_val > 0) {
            pq.push({new_val, idx});
        }
        
        cnt++;
    }
    
    cout << ans << "\n";
    return 0;
}```
