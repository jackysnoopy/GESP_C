# 生成序列

## 题目描述
**提示：请注意本题中 $n$ 和 $q$ 的范围。**

约定 $\text{len}(a)$ 表示序列 $a$ 的长度，$a_{[l,r]}$ 表示将序列 $a$ 中下标在 $[l,r]$ 之间的元素按顺序排列组成的序列。

初始，小 L 有一个空序列 $b$。小 L 可以按顺序执行以下操作若干次，使得序列 $b$ 最终变为目标序列 $a$：

- 生成一个序列 $c$，满足 $\text{len}(b)\leq\text{len}(c)$。
- 若存在 $1\leq i\leq\text{len}(b)$，满足 $b_i=c_i$，那么可以获得 $1$ 的收益，否则获得收益为 $0$。
- 将序列 $c$ 拼接在序列 $b$ 的后面。

小 L 想知道，在达成目标的前提下，最多可以获得多少收益。记 $f(a)$ 为 $a$ 为目标序列时该问题的答案。

为了防止你诈骗小 L，小 L 会先给你一个数组 $a$，并进行 $q$ 次询问，每次会给你两个整数 $l,r$，你需要回答 $f(a_{[l,r]})$ 的值。

## 输入格式
第一行包含两个整数 $n,q$，分别表示序列 $a$ 的长度和询问次数。

第二行包含 $n$ 个非负整数 $a_i$。

接下来 $q$ 行，每行两个整数 $l,r$，表示询问区间。

## 输出格式
输出 $q$ 行，第 $i$ 行表示第 $i$ 次询问的答案。

## 样例
### 样例1
输入：
```
5 3
1 2 3 4 5
1 5
2 4
1 3
```
输出：
```
4
2
2
```

## 提示
### 样例 1 解释

对于询问 $(1,5)$，目标序列是 $[1,2,3,4,5]$。一种可能的操作序列：
1. 生成序列 $[1,2,3,4,5]$，获得收益 $0$（因为 $b$ 为空）。
2. 生成序列 $[1,2,3,4,5]$，获得收益 $5$（因为 $b=[1,2,3,4,5]$，$c=[1,2,3,4,5]$，所有位置都相同）。
但输出是4，所以可能我的理解有误。

### 数据范围

- $1 \leq n, q \leq 10^5$
- $0 \leq a_i \leq 10^9$
- $1 \leq l \leq r \leq n$

## 解题思路

### 问题分析
这是一个关于序列生成和收益最大化的问题。我们需要通过生成序列来构造目标序列，并最大化收益。

### 核心思路
1. **贪心策略**：尽可能复用已有序列的前缀。
2. **动态规划**：记录每个位置的最大收益。
3. **区间查询**：处理多次区间询问。

### 算法流程
1. 预处理每个位置的最大收益。
2. 对于每次询问，计算区间的最大收益。

### 复杂度分析
- 时间复杂度：$O(n + q)$。
- 空间复杂度：$O(n)$。
