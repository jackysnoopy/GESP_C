# [SEATST 2026] 车辆集结 / Car Gathering

## 题目描述

N 辆车，位置 X[i] 和油耗 C[i]（均排序）。位置和油耗的配对未知。求整数点 p 使最坏情况总燃料成本最小。

## 输入格式

第一行 N。
第二行 X[0]..X[N-1]。
第三行 C[0]..C[N-1]。

## 输出格式

一个整数 p。

## 样例

输入：
```
3
-1 2 3
1 1 2
```
输出：
```
1
```

## 提示

- 1 <= N <= 10,000,000

## 解题思路

### 核心思路

由重排不等式，最坏情况是将最大距离配最大油耗。worst(p) = sum sorted(|x[i]-p|) * sorted(c)。这是关于 p 的凸函数，用三分搜索。

### 复杂度分析

- 时间：O(N log N)
- 空间：O(N)
