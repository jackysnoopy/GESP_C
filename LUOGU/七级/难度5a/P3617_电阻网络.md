# P3617 电阻网络

## 题目描述

电路都带有正、负极接点，正极在左，负极在右。电路分为以下几类：
- 单独的一个 $1\Omega$ 电阻是电路
- 如果 $A$ 和 $B$ 都是电路，可以串联
- 如果 $A$ 和 $B$ 都是电路，可以并联

现在给出一个电路，求它正负极之间的电阻。

## 输入格式

第一行为两个正整数 $n,m$，分别代表接点数和电阻数。

接下来 $m$ 行，每行三个整数 $a_i,b_i,c_i$，代表这个电阻连接了 $a_i$ 与 $b_i$ 接点，其阻值为 $c_i$。

## 输出格式

输出一个实数，表示总的电阻值，保留三位小数输出。

## 样例

### 样例1
**输入：**
```
7 7
1 2 0
1 3 0
2 4 1
3 5 1
4 6 0
5 6 0
6 7 1
```

**输出：**
```
1.500
```

## 解题思路

这是一道图论题目，可以使用Dijkstra算法求最短路。

**算法**：
1. 将电阻网络建模为图
2. 使用Dijkstra算法求从正极到负极的最短路
3. 最短路长度即为总电阻

**时间复杂度**：$O((n+m) \log n)$
