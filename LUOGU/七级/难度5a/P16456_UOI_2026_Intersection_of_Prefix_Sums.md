# [UOI 2026] Intersection of Prefix Sums

## 题目描述

给定数组 a 和整数 x，重排数组使没有任何非空前缀和等于 x。输出 YES/NO 和排列。

## 输入格式

第一行 t（测试用例数）。
每个用例：第一行 n x，第二行 a_1..a_n。

## 输出格式

YES/NO，若 YES 再输出排列。

## 样例

输入：
```
2
6 4
1 1 2 2 2 -3
3 7
7 7 7
```
输出：
```
YES
2 1 2 1 -3 2
NO
```

## 提示

- n <= 10^5

## 解题思路

### 核心思路

贪心构造：选择一个不等于 x 的元素开头，然后逐步选择不会使前缀和等于 x 的元素。

### 复杂度分析

- 时间：O(n^2)（暴力匹配）
- 空间：O(n)
