# 【MX-X19-T4】「FeOI Round 4.5」Supernova

## 题目描述
遥给了你三个非负整数 $x, y, p$。你可以以任意顺序执行以下两种操作任意次（也可以不操作）：

1. 令 $x \gets (x + 1) \mathbin{\vert} p$。
2. 令 $x \gets x \mathbin{\vert} (p + 1)$。

这里 $\mathbin{\vert}$ 表示按位或运算，$a \gets b$ 表示把 $b$ 的值赋给 $a$。

遥想知道，是否能通过操作使 $x$ 变为 $y$？如果可以，她还想知道所需的最小操作次数。

请你帮忙解决她的问题。

## 输入格式
**本题输入包含多组数据。**

第一行，一个整数 $T$，表示数据组数。对于每组数据：

- 仅一行，三个非负整数 $x, y, p$。

## 输出格式
对于每组测试数据，输出一行，如果有解，输出一个非负整数，表示最小操作次数，否则无解时输出 `-1`。

## 样例
### 样例1
输入：
```
3
0 1 0
1 2 1
3 5 2
```
输出：
```
1
1
-1
```

## 提示
**【样例解释】**

- 第一组数据：执行操作1，$x \gets (0+1) \mathbin{|} 0 = 1$，得到 $y=1$。
- 第二组数据：执行操作2，$x \gets 1 \mathbin{|} (1+1) = 1 \mathbin{|} 2 = 3$，得到 $y=3$，但题目要求变为2，所以实际应为操作1：$x \gets (1+1) \mathbin{|} 1 = 2 \mathbin{|} 1 = 3$，也不是2。这里可能需要重新理解。
- 第三组数据：无法通过操作使 $x=3$ 变为 $y=5$。

**【数据范围】**

- $1 \leq T \leq 10^5$
- $0 \leq x, y, p < 2^{30}$

## 解题思路

### 问题分析
这是一个关于位运算和操作最小化的问题。我们需要通过给定的两种操作，将x变为y，并求最小操作次数。

### 核心思路
1. **位运算分析**：分析两种操作对x的二进制表示的影响。
2. **贪心策略**：从高位到低位考虑，尽量减少操作次数。
3. **BFS搜索**：使用广度优先搜索寻找最小操作次数。

### 算法流程
1. 检查y是否可达：如果y的某些位在x和p中都无法设置，则无解。
2. 使用BFS搜索最小操作次数。
3. 或者使用贪心算法直接计算。

### 复杂度分析
- 时间复杂度：$O(T \log (\max(x, y, p)))$ 或 $O(T)$，取决于具体实现。
- 空间复杂度：$O(1)$。
