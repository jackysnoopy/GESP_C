# [COCI 2025/2026 #6] 教室 / Učionica

## 题目描述

n×m 网格教室，部分座位被占用（有学生身高），部分空闲。k 个朋友想坐同一行连续 k 个座位。每个朋友要求前方（同行同列更小行号）所有学生严格比自己矮。求合法座位集合数。

## 输入格式

第一行 n m k。
第二行 k 个朋友身高。
接下来 n 行每行 m 个数（0=空，>0=学生身高）。

## 输出格式

一个整数。

## 样例

输入：
```
3 4 2
2 6
0 0 3 1
8 0 0 0
0 0 1 0
```
输出：
```
3
```

## 提示

- 1 <= n, m <= 2000

## 解题思路

### 核心思路

1. 预处理每列每行前方最大身高
2. 对每行每个长度为 k 的窗口，排序朋友身高和前方最大身高，检查是否每个朋友都更高

### 复杂度分析

- 时间：O(n*m*k*log k)
- 空间：O(n*m)
