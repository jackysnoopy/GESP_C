# P3812 【模板】线性基

## 题目描述

给定 $n$ 个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。

## 输入格式

第一行一个数 $n$，表示元素个数。

接下来一行 $n$ 个数。

## 输出格式

输出一行一个整数，表示答案。

## 样例

### 样例1
**输入：**
```
2
1 1
```

**输出：**
```
1
```

### 样例2
**输入：**
```
4
1 5 9 4
```

**输出：**
```
13
```

## 解题思路

这是一道经典的线性基模板题。

**算法**：
1. 使用线性基维护所有数的异或空间
2. 从高位到低位贪心选择

**时间复杂度**：$O(n \log S)$
