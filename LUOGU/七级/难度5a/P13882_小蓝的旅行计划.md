# [蓝桥杯 2023 省 Java A] 小蓝的旅行计划

## 题目描述
小蓝正计划进行一次漫长的旅行。小蓝计划开车完成这次旅行。显然他在途中需要加油，否则可能无法完成这次旅行。

小蓝要依次经过 $n$ 个地点，其中从第 $i-1$ 个地点到达第 $i$ 个地点需要消耗 $Dis_i$ 升油。小蓝经过的每个地点都有一个加油站，但每个加油站的规定也不同。在第 $i$ 个加油站加 1 升油需要 $Cost_i$ 的费用，且在这个加油站最多只能加 $Lim_i$ 升油。

小蓝的车的油箱也有容量限制，他的车上最多只能装载 $m$ 升油。

一开始小蓝的油箱是满的，请问小蓝需要准备多少钱才能顺利完成他的旅行计划。如果小蓝按给定条件无论准备多少钱都不能完成他的旅行计划，请输出 $-1$。

## 输入格式
输入的第一行包含两个整数 $n, m$，用一个空格分隔。

接下来 $n$ 行每行包含 3 个整数 $Dis_i, Cost_i, Lim_i$，相邻整数之间使用一个空格分隔。

## 输出格式
输出一行包含一个整数表示答案。

## 样例
### 样例1
输入：
```
4 5
2 9 2
4 5 6
3 2 2
4 1 3
```
输出：
```
38
```

## 提示
**【评测用例规模与约定】**

对于 $30\%$ 的评测用例，$n, Dis_i, Cost_i, Lim_i, m \leq 300$；

对于 $60\%$ 的评测用例，$n, Dis_i, Cost_i, Lim_i, m \leq 5000$；

对于所有评测用例，$1 \leq n \leq 2 \times 10^5$，$1 \leq Dis_i, Lim_i, m \leq 10^9$，$1 \leq Cost_i \leq 40000$。

## 解题思路

### 问题分析
这是一个关于旅行加油和费用最小化的问题。我们需要在满足油箱容量限制和加油站加油限制的前提下，找到完成旅行的最小费用。

### 核心思路
1. **贪心策略**：在每个加油站，我们应该优先选择价格便宜的油。
2. **动态规划**：记录当前油量和已花费的费用。
3. **优先队列**：使用优先队列来管理可选的油。

### 算法流程
1. 从起点开始，油箱是满的。
2. 对于每个加油站，考虑加油的选择。
3. 使用贪心或动态规划找到最小费用。
4. 检查是否能完成旅行。

### 复杂度分析
- 时间复杂度：$O(n \log n)$ 或 $O(n^2)$，取决于具体实现。
- 空间复杂度：$O(n)$。
