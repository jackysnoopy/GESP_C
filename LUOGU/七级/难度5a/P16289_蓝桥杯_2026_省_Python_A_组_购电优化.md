# [蓝桥杯 2026 省 Python A 组] 购电优化

## 题目描述

n 个时段，每个需 w_i 电量。电池容量 k。第 i 时段购电价格 c_i。求最小总花费。

## 输入格式

第一行 n k。
第二行 w_1..w_n。
第三行 c_1..c_n。

## 输出格式

一个整数。

## 样例

输入：
```
3 1
1 2 1
20 25 100
```
输出：
```
90
```

## 提示

- 1 <= n <= 5*10^5

## 解题思路

### 核心思路

贪心：优先使用电池中已购电量，不足时按当前价格购买。电池满时优先使用最贵的电。

### 复杂度分析

- 时间：O(n log n)
- 空间：O(n)
