# P3141 标号计数

## 思路

核心算法：计数

## 代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

struct Edge {
    int u, v, w;
};

int parent[500005];

int find(int x) {
    return parent[x] == x ? x : parent[x] = find(parent[x]);
}

bool unite(int x, int y) {
    x = find(x);
    y = find(y);
    if (x == y) return false;
    parent[x] = y;
    return true;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int A, B, n, m;
    cin >> A >> B >> n >> m;
    
    vector<int> a(n), b(m);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    for (int i = 0; i < m; i++) {
        cin >> b[i];
    }
    
    // Sort fence positions
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());
    
    // Create edges for vertical fences
    vector<Edge> edges;
    for (int i = 0; i < n; i++) {
        int cost = (i == 0) ? a[0] : a[i] - a[i-1];
        edges.push_back({i, i + 1, cost});
    }
    edges.push_back({n, n + 1, A - a[n-1]});
    
    // Create edges for horizontal fences
    for (int i = 0; i < m; i++) {
        int cost = (i == 0) ? b[0] : b[i] - b[i-1];
        edges.push_back({n + 1 + i, n + 1 + i + 1, cost});
    }
    edges.push_back({n + m + 1, n + m + 2, B - b[m-1]});
    
    // Kruskal's algorithm
    for (int i = 0; i <= n + m + 2; i++) {
        parent[i] = i;
    }
    
    sort(edges.begin(), edges.end(), [](const Edge& a, const Edge& b) {
        return a.w < b.w;
    });
    
    long long total = 0;
    for (const Edge& e : edges) {
        if (unite(e.u, e.v)) {
            total += e.w;
        }
    }
    
    cout << total << "\n";
    return 0;
}
```

## 解题思路

### 问题分析

A×B 的网格上有 n 条竖栅栏和 m 条横栅栏，将网格分成若干区域。需要用栅栏把所有区域连通，求最小总长度。

### 核心思路

最小生成树（Kruskal）。将相邻区域之间的栅栏长度作为边权，用并查集合并连通分量，贪心选择最短边。

### 算法流程

1. 排序竖栅栏位置 a 和横栅栏位置 b
2. 建边：
   - 竖栅栏相邻间隙作为边，权值为间距
   - 横栅栏相邻间隙作为边，权值为间距
3. 按边权排序，用 Kruskal 算法求最小生成树
4. 输出总权值

### 复杂度分析

- 时间复杂度：O((n+m) log(n+m))，排序 + 并查集
- 空间复杂度：O(n+m)
