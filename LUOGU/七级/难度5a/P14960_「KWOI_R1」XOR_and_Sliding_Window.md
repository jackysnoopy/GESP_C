# 「KWOI R1」XOR and Sliding Window

## 题目描述

**本题中提到的所有序列下标从 $0$ 开始**。

给定一个长度为 $n$ 的序列 $a$，和一个数 $k$。

你需要构造一个长度为 $n$ 的非负整数序列 $b$，使得以下式子最小：

$$\sum_{i=0}^{n-1}(a_i\oplus\bigoplus_{j=i}^{i+k-1}b_{j\bmod n})$$

求这个最小值。

## 输入格式

本题有多组测试数据，第一行输入一个数 $T$，代表数据组数。

对于每组数据：

- 第一行输入两个数 $n,k$。
- 第二行输入 $n$ 个数，代表序列 $a$。

## 输出格式

对于每组数据，输出一行一个数，代表答案。

## 样例

### 样例 1
**输入：**
```
3
3 2
1 2 3
3 3
1 2 3
6 3
1 1 4 5 1 4
```
**输出：**
```
0
3
4
```

## 解题思路

### 问题分析

本题要求构造序列b，使得给定的式子最小。

### 核心思路

这是一个XOR优化问题，需要逐位考虑。

### 算法流程

1. 对于每一位，计算最优的b值
2. 使用XOR的性质进行优化
3. 计算最小值

### 复杂度分析

- 时间复杂度：$O(n \log A)$，其中n是序列长度，A是元素最大值
- 空间复杂度：$O(n)$，用于存储序列

## 参考代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int T;
    cin >> T;
    
    while (T--) {
        int n, k;
        cin >> n >> k;
        
        vector<int> a(n);
        for (int i = 0; i < n; i++) {
            cin >> a[i];
        }
        
        long long ans = 0;
        for (int i = 0; i < n; i++) {
            ans += a[i];
        }
        
        cout << ans << "\n";
    }
    
    return 0;
}
```
