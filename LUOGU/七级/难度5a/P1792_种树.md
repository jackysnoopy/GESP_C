# [国家集训队] 种树

## 题目描述

A 城市有一个巨大的圆形广场，为了绿化环境和净化空气，市政![]()府决定沿圆形广场外圈种一圈树。

园林部门得到指令后，初步规划出 $n$ 个种树的位置，顺时针编号 $1$ 到 $n$。并且每个位置都有一个美观度 $A_i$，如果在这里种树就可以得到这 $A_i$ 的美观度。但由于 A 城市土壤肥力欠佳，两棵树决不能种在相邻的位置（$i$ 号位置和 $i+1$ 号位置叫相邻位置。值得注意的是 $1$ 号和 $n$ 号也算相邻位置）。

最终市![]()政![]()府给园林部门提供了 $m$ 棵树苗并要求全部种上，请你帮忙设计种树方案使得美观度总和最大。如果无法将 $m$ 棵树苗全部种上，给出无解信息。

## 输入格式

输入的第一行包含两个正整数 $n,m$。

第二行 $n$ 个整数，第 $i$ 个代表 $A_i$。

## 输出格式

输出一个整数，表示最佳植树方案可以得到的美观度。如果无解输出 `Error!`。

## 样例

Sample 1:
Input:
7 3
1 2 3 4 5 6 7
Output:
15

Sample 2:
Input:
7 4
1 2 3 4 5 6 7

Output:
Error!

## 解题思路

### 问题分析
- 在一个圆形广场上种树，有 $n$ 个位置，每个位置有一个美观度 $A_i$。
- 两棵树不能种在相邻的位置（包括 $1$ 号和 $n$ 号相邻）。
- 需要种 $m$ 棵树，使得美观度总和最大。
- 如果无法种 $m$ 棵树（即 $n < 2m$），输出 `Error!`。

### 核心思路
- 这是一个环形选择问题，类似于“打家劫舍 II”，但需要选择 $m$ 个不相邻的位置。
- 可以使用贪心+反悔堆（可反悔贪心）来解决。
- 将环拆成链，复制一份，然后使用堆维护可选位置，每次选择最大值，然后反悔（将当前节点与左右邻居合并）。

### 算法流程
1. 判断是否无解：如果 $n < 2m$，输出 `Error!`。
2. 将环拆成链，复制一份，得到长度为 $2n$ 的数组。
3. 初始化一个最大堆，存储每个位置的美观度和索引。
4. 初始化 `used` 数组标记是否已使用，`left` 和 `right` 数组维护双向链表。
5. 循环 $m$ 次：
   - 从堆中取出最大值对应的索引。
   - 如果该索引已被使用或超过 $n$，跳过。
   - 将美观度加入答案。
   - 反悔操作：将当前节点与左右邻居合并，更新堆。
6. 如果选择的节点数不足 $m$，输出 `Error!`；否则输出答案。

### 复杂度分析
- 时间复杂度：$O(n \log n)$，每个节点最多被合并一次，堆操作为 $O(\log n)$。
- 空间复杂度：$O(n)$，存储数组和堆。

## 参考代码

```cpp

#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    if (m == 0) {
        cout << 0 << "\n";
        return 0;
    }
    
    if (n < 2 * m) {
        cout << "Error!\n";
        return 0;
    }
    
    // 贪心+反悔堆
    // 将环拆成链，复制一份
    vector<int> b(2 * n);
    for (int i = 0; i < n; i++) {
        b[i] = b[i + n] = a[i];
    }
    
    // 使用堆维护可选位置
    priority_queue<pair<int, int>> pq;
    vector<bool> used(2 * n, false);
    vector<int> left(2 * n), right(2 * n);
    
    for (int i = 0; i < 2 * n; i++) {
        left[i] = i - 1;
        right[i] = i + 1;
        pq.push({b[i], i});
    }
    
    int ans = 0;
    int cnt = 0;
    
    while (cnt < m && !pq.empty()) {
        auto [val, idx] = pq.top();
        pq.pop();
        
        if (used[idx]) continue;
        if (idx >= n) continue; // 只取前n个
        
        ans += val;
        cnt++;
        
        // 反悔：将当前节点与左右邻居合并
        int l = left[idx], r = right[idx];
        if (l >= 0 && r < 2 * n) {
            used[l] = true;
            used[r] = true;
            
            int new_val = b[l] + b[r] - b[idx];
            b[idx] = new_val;
            
            left[idx] = left[l];
            right[idx] = right[r];
            if (left[idx] >= 0) right[left[idx]] = idx;
            if (right[idx] < 2 * n) left[right[idx]] = idx;
            
            pq.push({new_val, idx});
        } else {
            used[idx] = true;
        }
    }
    
    if (cnt < m) {
        cout << "Error!\n";
    } else {
        cout << ans << "\n";
    }
    
    return 0;
}

```
