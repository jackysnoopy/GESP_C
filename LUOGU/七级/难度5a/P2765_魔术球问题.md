# P2765 魔术球问题

## 思路

核心算法：贪心+回溯

## 解题思路

### 问题分析
魔术球问题：有 n 根柱子，按顺序将编号为 1, 2, 3, ... 的球放入柱子中。若相邻两个球编号之和为完全平方数，则可放在同一根柱子上。求最多能放多少个球，并输出放置方案。

### 核心思路
贪心策略：依次尝试将每个球放到已有柱子的顶部（若满足完全平方数条件），否则使用新的空柱子。如果所有柱子都已满且无法放置，则停止。

### 算法流程
1. 初始化 n 根空柱子
2. 从球 1 开始依次尝试放置：
   - 遍历所有柱子，找到顶部球编号与当前球编号之和为完全平方数的柱子，放入
   - 若无可用柱子，找一根空柱子放入
   - 若无空柱子，停止放置
3. 输出最多放置的球数及每根柱子的球

### 复杂度分析
- 时间复杂度：O(N × M)，N 为球数（最多约 1000），M 为柱子数
- 空间复杂度：O(N × M)

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>
using namespace std;

bool isSquare(int x) {
    int s = sqrt(x);
    return s * s == x;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;
    
    // Greedy: try to place each ball on an existing pillar
    // If not possible, create a new pillar
    
    vector<vector<int>> pillars(n);
    
    for (int ball = 1; ball <= 1000; ball++) {
        bool placed = false;
        
        for (int i = 0; i < n; i++) {
            if (!pillars[i].empty() && isSquare(pillars[i].back() + ball)) {
                pillars[i].push_back(ball);
                placed = true;
                break;
            }
        }
        
        if (!placed) {
            // Find an empty pillar
            for (int i = 0; i < n; i++) {
                if (pillars[i].empty()) {
                    pillars[i].push_back(ball);
                    placed = true;
                    break;
                }
            }
        }
        
        if (!placed) {
            // All pillars are full, we can't place more balls
            break;
        }
    }
    
    // Find the maximum number of balls placed
    int max_balls = 0;
    for (int i = 0; i < n; i++) {
        max_balls = max(max_balls, (int)pillars[i].size());
    }
    
    cout << max_balls << "\n";
    
    // Output the placement
    for (int i = 0; i < n; i++) {
        if (!pillars[i].empty()) {
            for (int j = 0; j < (int)pillars[i].size(); j++) {
                cout << pillars[i][j] << (j == (int)pillars[i].size() - 1 ? "\n" : " ");
            }
        } else {
            cout << "\n";
        }
    }
    
    return 0;
}
```
