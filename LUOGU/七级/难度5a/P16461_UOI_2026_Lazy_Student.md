# [UOI 2026] Lazy Student

## 题目描述

n×m 矩阵，最多 k 次行内交换，使列和字典序最小。

## 输入格式

第一行 n m k。
接下来 n 行每行 m 个数。

## 输出格式

m 个整数：最小列和数组。

## 样例

输入：
```
3 3 1
7 1 3
6 8 4
5 2 9
```
输出：
```
12 17 16
```

## 提示

- 1 <= n, m <= 1000

## 解题思路

### 核心思路

贪心：从左到右处理每列，每行将最小未使用元素换到当前列。

### 复杂度分析

- 时间：O(n*m*k)
- 空间：O(n*m)
