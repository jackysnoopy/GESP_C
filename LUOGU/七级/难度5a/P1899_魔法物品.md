# 魔法物品

## 题目描述

有两种类型的物品：普通物品和魔法物品。普通物品没有魔法属性，而魔法物品拥有一些魔法属性。每种普通物品有一个价值 $P$，但每种魔法物品有两种价值：鉴定前的价值 $P_1$ 和鉴定后的价值 $P_2$（当然，$P_2$ 总是大于 $P_1$）。

为了鉴定一个魔法物品，你需要购买一个鉴定卷轴，用它来鉴定魔法物品。鉴定完一件魔法物品以后，鉴定卷轴便会消失。每个鉴定卷轴将会消耗 $P_i$ 元钱。如果没有足够的钱，你将无法购买任何鉴定卷轴。    
现在，你正在一个集市中，同时拥有很多物品。你知道每件物品的价值并且想要出售全部物品。那么，你最多能够获得多少钱呢？

你可以假定：

- 开始的时候你有足够的钱来购买鉴定卷轴。

## 输入格式

第一行有两个整数 $N$ 和 $P_i$（$1 \le P_i \le 5000$），表示你拥有的物品数和一个鉴定卷轴的价格。  
接下来 $N$ 行，每行给出一件物品的价格。  
对于每件普通物品行仅有一个整数 $P$（$1 \le P \le 10000$）。  
对于每件魔法物品,那一行将会有两个整数 $P_1$ 和 $P_2$（$1 \le P_1 < P_2 \le 10000$）。

## 输出格式

输出一个整数表示你最多能够获得多少钱。

## 样例

Sample 1:
Input:
2 10
10
20 100
Output:
100

## 提示

对于 $30\%$ 的数据，$1\le N \le 50$。  
对于 $100\%$ 的数据，$1 \le N \le 1000$。

## 解题思路

### 问题分析
- 有两种物品：普通物品和魔法物品。
- 普通物品直接出售，魔法物品可以鉴定后出售（价值更高）。
- 鉴定需要消耗卷轴，每个卷轴价格为 $P_i$。
- 目标是最大化总收益。

### 核心思路
- 这是一个背包问题的变种。
- 对于魔法物品，鉴定的收益是 $P_2 - P_1$，成本是卷轴价格 $P_i$。
- 贪心策略：优先鉴定收益最大的魔法物品。
- 使用动态规划计算最优的卷轴使用方案。

### 算法流程
1. 分离普通物品和魔法物品。
2. 计算普通物品的总价值。
3. 对魔法物品按收益 $(P_2 - P_1)$ 降序排序。
4. 使用01背包计算使用 $j$ 个卷轴的最大额外收益。
5. 找到最优的卷轴使用数量，使得 $dp[j] - j \times P_i$ 最大。

### 复杂度分析
- 时间复杂度：$O(N \log N + N \times P_i)$，排序 $O(N \log N)$，背包 $O(N \times P_i)$。
- 空间复杂度：$O(P_i)$。

## 参考代码

```cpp

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, pi;
    cin >> n >> pi;
    
    vector<int> normal;
    vector<pair<int, int>> magic;
    
    for (int i = 0; i < n; i++) {
        int type;
        cin >> type;
        if (cin.peek() == '\n' || cin.peek() == EOF) {
            // 普通物品
            normal.push_back(type);
        } else {
            // 魔法物品
            int p2;
            cin >> p2;
            magic.push_back({type, p2});
        }
    }
    
    // 计算普通物品的总价值
    int normal_sum = 0;
    for (int v : normal) {
        normal_sum += v;
    }
    
    // 对魔法物品按p2-p1降序排序（贪心策略）
    sort(magic.begin(), magic.end(), [](const pair<int, int>& a, const pair<int, int>& b) {
        return (a.second - a.first) > (b.second - b.first);
    });
    
    int m = magic.size();
    // dp[j] 表示使用j个卷轴能获得的最大额外收益
    vector<int> dp(pi + 1, 0);
    
    for (int i = 0; i < m; i++) {
        int p1 = magic[i].first;
        int p2 = magic[i].second;
        int gain = p2 - p1;
        
        // 逆序更新（01背包）
        for (int j = pi; j >= 1; j--) {
            if (j >= 1) {
                dp[j] = max(dp[j], dp[j - 1] + gain);
            }
        }
    }
    
    // 找到最优的卷轴使用数量
    int best = 0;
    for (int j = 0; j <= pi; j++) {
        best = max(best, dp[j] - j * pi);
    }
    
    cout << normal_sum + best << "\n";
    return 0;
}

```
