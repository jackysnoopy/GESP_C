# P9166 - 省选联考 2023 火车站

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

n 个火车站，m 条轨道覆盖区间。从 x 出发，沿轨道单向行驶，可在重叠站换轨。求可能到达的终点。

### 核心思路

区间合并 + 连通性分析。将覆盖 x 的轨道合并为连通分量。向左可达分量内所有左端点，向右可达所有右端点。

### 复杂度分析

时间复杂度：O(m log m)，空间复杂度：O(m)

### 关键技巧

按左端点排序后合并重叠区间。注意火车只能单向行驶，向左和向右分别计算可达端点。
