# [NOIP 2007 普及组] 纪念品分组

## 题目描述

元旦快到了，校学生会让乐乐负责新年晚会的纪念品发放工作。为使得参加晚会的同学所获得的纪念品价值相对均衡，他要把购来的纪念品根据价格进行分组，但每组最多只能包括两件纪念品，并且每组纪念品的价格之和不能超过一个给定的整数。为了保证在尽量短的时间内发完所有纪念品，乐乐希望分组的数目最少。

你的任务是写一个程序，找出所有分组方案中分组数最少的一种，输出最少的分组数目。

## 输入格式

共 $n+2$ 行：

第一行包括一个整数 $w$，为每组纪念品价格之和的上限。

第二行为一个整数 $n$，表示购来的纪念品的总件数 $G$。

第 $3\sim n+2$ 行每行包含一个正整数 $P_i$ 表示所对应纪念品的价格。

## 输出格式

一个整数，即最少的分组数目。

## 样例

### 样例输入
```
100 
9 
90 
20 
20 
30 
50 
60 
70 
80 
90
```

### 样例输出
```
6
```

## 提示

$50\%$ 的数据满足：$1\le n\le15$。

$100\%$ 的数据满足：$1\le n\le3	imes10^4$，$80\le w\le200$，$5 \le  P_i  \le  w$。

## 解题思路

### 问题分析

纪念品分组问题：n 件纪念品，每组总价值不超过 W，最少分几组。

### 核心思路

贪心双指针：排序后，最重的和最轻的尝试放一组。若不能则最重的单独一组。

### 算法流程

1. 将纪念品按价格排序
2. 双指针：l=0, r=n-1
3. 若 a[l]+a[r] ≤ W，l++, r--（一组两个）
4. 否则 r--（最重的单独一组）
5. 统计组数

### 复杂度分析

- 时间复杂度：O(N·log(N))
- 空间复杂度：O(N)

### 关键技巧

- 排序 + 双指针是此类问题的经典解法
- 最轻的总是配最重的，这样最省空间
