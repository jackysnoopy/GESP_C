# P8818 - CSP-S 2022 策略游戏

## 题目描述

参见洛谷原题。

## 解题思路

### 问题分析

矩阵 C = A × B^T。小 L 选行，小 Q 选列。L 想最大化得分，Q 想最小化。q 次区间查询。

### 核心思路

博弈论 + 稀疏表。对 B 预处理区间 max/min。对 A 预处理区间 max、min、maxNeg、minPos。根据 A[x] 的正负号确定 Q 的最优策略，L 再选择最优的 x。

### 复杂度分析

时间复杂度：O((n+m) log(n+m) + q)，空间复杂度：O((n+m) log(n+m))

### 关键技巧

关键：A[x]>0 时 Q 选 minB，A[x]<0 时 Q 选 maxB，A[x]=0 时得分 0。
