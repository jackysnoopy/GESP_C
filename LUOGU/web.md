# Web 前端上机操作题（50 题）

> 适用对象：Web 前端开发学习者  
> 技术栈：HTML5 + CSS3 + JavaScript（ES6+）  
> 建议环境：VS Code + Chrome 浏览器  
> 提交要求：每题提交可运行的 HTML 文件（可内联 CSS/JS 或外链）

---

## 第一部分：HTML 与 CSS 基础（第 1-15 题）

### 题目 1：个人简历页面
**难度：★☆☆☆☆**

创建一个语义化的个人简历页面，要求：
- 使用 `<header>`、`<nav>`、`<main>`、`<section>`、`<footer>` 等语义化标签
- 包含个人信息、教育背景、工作经历、技能特长四个板块
- 使用 `<ul>`/`<ol>` 列表展示内容
- 添加一张个人头像图片（可用占位图）
- 页面标题使用 `<h1>` ~ `<h3>` 正确嵌套

### 题目 2：响应式导航栏
**难度：★★☆☆☆**

- 移动端（≤768px）：汉堡菜单按钮，点击展开/收起导航
- 导航项包含：首页、产品、关于我们、联系我们
